# Clawpack V2 - Product Requirements

## What
Unified AI Agent Ecosystem - 15 specialized agents working as one CLI tool.

## Who For
Developers who want AI assistance with coding, legal research, medical info, translation, math, blockchain, and more.

## Success Criteria
- All agents work from single command: `python3 clawpack.py`
- Each agent can run standalone: `python3 claw.py <agent>`
- Shared memory across sessions
- 23+ LLMs available (Ollama, OpenRouter, Anthropic)
