# TX Blockchain Introduction

TX is a 3rd-generation layer-1 enterprise-grade blockchain built for decentralized applications with ISO20022 compatibility, IBC interoperability, and Smart Tokens.

## Key Features
- **WASM Smart Contracts** - Write in Rust, C++, JavaScript, Go
- **Smart Tokens** - Advanced token features (minting, burning, freezing, clawback)
- **IBC Compatible** - Cross-chain communication
- **Low Cost** - Deterministic gas fees
- **Fast** - Low-latency PoS consensus
- **Green** - Energy-efficient Bonded PoS

## Native Token
**$TX** is used for:
- Transaction fees
- Staking
- Governance
- DEX trading
