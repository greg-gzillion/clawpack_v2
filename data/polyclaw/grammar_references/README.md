# Grammar Reference by Language

## Formality Levels

### Japanese
- **Casual**: 食べる (taberu) - plain form
- **Polite**: 食べます (tabemasu) - masu form
- **Honorific**: 召し上がる (meshiagaru) - respectful

### French
- **Informal** (tu): Tu parles français?
- **Formal** (vous): Vous parlez français?

### German
- **Informal** (du): Du sprichst Deutsch
- **Formal** (Sie): Sie sprechen Deutsch

### Korean
- **Informal** (반말): 먹어 (meogeo)
- **Polite** (존댓말): 먹어요 (meogeoyo)
- **Formal** (격식체): 먹습니다 (meogseumnida)

## Word Order Variations
- **English/Spanish/French**: Subject-Verb-Object (SVO)
- **Japanese/Korean**: Subject-Object-Verb (SOV)
- **Arabic/Gaelic**: Verb-Subject-Object (VSO)

## Gendered Languages
- **Spanish**: el (masculine), la (feminine)
- **French**: le (masculine), la (feminine)
- **German**: der (masculine), die (feminine), das (neuter)
- **Russian**: он (masculine), она (feminine), оно (neuter)
