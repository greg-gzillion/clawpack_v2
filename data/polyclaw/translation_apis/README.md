# Translation APIs Reference

## Free APIs
- **LibreTranslate** - Open source, self-hostable
- **Argos Translate** - Open source neural machine translation
- **Apertium** - Rule-based translation for related languages

## Paid APIs
- **Google Cloud Translation** - 100+ languages, $20/1M characters
- **DeepL API** - High quality, supports 30+ languages
- **Microsoft Translator** - 100+ languages, $10/1M characters
- **Amazon Translate** - 75+ languages, $15/1M characters

## Language Detection
- **Compact Language Detector (CLD2)** - Google's language detection
- **WhatTheLang** - Fast language detection API
- **LangDetect** - Java/Python library for 50+ languages

## Translation Best Practices
1. Always specify source language when possible
2. Use language codes (ISO 639-1)
3. Consider formality levels (tu/vous in French, you formal/informal)
4. Handle regional variations (pt-PT vs pt-BR)
5. Implement fallback chains for reliability
