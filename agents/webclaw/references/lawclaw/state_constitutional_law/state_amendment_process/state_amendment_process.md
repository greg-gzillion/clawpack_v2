# State Constitutional Law - State Constitutional Amendment Process

## Amendment Methods Overview
| Method | Description | States | URL |
|--------|-------------|--------|-----|
| Legislative Proposal | Legislature proposes amendment; voter ratification | All 50 states | https://www.ncsl.org/about-state-legislatures/amending-state-constitutions |
| Constitutional Convention | Convention proposes amendments or new constitution | 44 states provide for conventions | https://www.ncsl.org/about-state-legislatures/state-constitutional-conventions |
| Constitutional Initiative | Citizens propose amendments directly to voters | 18 states | https://www.ncsl.org/elections-and-campaigns/initiative-and-referendum-processes |
| Constitutional Commission | Commission studies and recommends amendments | Florida, others | https://www.ncsl.org/about-state-legislatures/constitutional-amendment-processes |

## Legislative Proposal Requirements
| Requirement | Typical Provision | URL |
|-------------|------------------|-----|
| Supermajority Vote | Two-thirds of each house | https://www.ncsl.org/about-state-legislatures/amending-state-constitutions |
| Single Session vs. Two Sessions | Most states require single session; some require two | https://www.ncsl.org/about-state-legislatures/amending-state-constitutions |
| Publication Requirement | Amendment text published before election | https://www.ncsl.org/about-state-legislatures/amending-state-constitutions |
| Voter Ratification | Majority vote required in most states | https://www.ncsl.org/about-state-legislatures/amending-state-constitutions |
