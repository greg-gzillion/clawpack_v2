# Alternative Dispute Resolution (ADR) - Overview

## What is ADR?
Alternative Dispute Resolution encompasses methods of resolving disputes outside of traditional litigation, including arbitration, mediation, negotiation, and conciliation.

## Primary ADR Methods
| Method | Description | Binding? |
|--------|-------------|----------|
| **Arbitration** | Neutral third party renders decision | Usually binding |
| **Mediation** | Facilitator helps parties reach agreement | Non-binding |
| **Negotiation** | Direct party-to-party discussion | Binding if agreement reached |
| **Conciliation** | Similar to mediation, more evaluative | Non-binding |
| **Neutral Evaluation** | Expert evaluates case strengths | Non-binding |
| **Mini-Trial** | Abbreviated presentation to decision-makers | Non-binding |
| **Summary Jury Trial** | Mock jury provides advisory verdict | Non-binding |
