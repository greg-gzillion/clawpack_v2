# Workers' Compensation Law - Disability Benefits

## Types of Disability Benefits
| Type | Description | URL |
|------|-------------|-----|
| Temporary Total Disability (TTD) | Unable to work at all during recovery | https://www.law.cornell.edu/wex/temporary_total_disability |
| Temporary Partial Disability (TPD) | Able to work with restrictions at reduced wages | https://www.law.cornell.edu/wex/temporary_partial_disability |
| Permanent Total Disability (PTD) | Permanently unable to work | https://www.law.cornell.edu/wex/permanent_total_disability |
| Permanent Partial Disability (PPD) | Permanent impairment but able to work | https://www.law.cornell.edu/wex/permanent_partial_disability |

## Temporary Total Disability (TTD)
| Topic | Description | URL |
|-------|-------------|-----|
| Eligibility | Unable to work due to injury | Varies by state |
| Benefit Rate | 2/3 of average weekly wage (typical) | Varies by state |
| Minimum/Maximum | Subject to state minimum and maximum | Varies by state |
| Waiting Period | 3-7 days before benefits begin | Varies by state |
| Retroactive Period | Benefits paid retroactively if disability exceeds threshold (e.g., 14-21 days) | Varies by state |
| Duration | Until maximum medical improvement (MMI) | Varies by state |
| Cap on Duration | Some states cap TTD (e.g., 104 weeks, 500 weeks) | Varies by state |

## Temporary Partial Disability (TPD)
| Topic | Description | URL |
|-------|-------------|-----|
| Eligibility | Working with restrictions at reduced wages | Varies by state |
| Benefit Rate | 2/3 of difference between pre-injury and post-injury wages | Varies by state |
| Duration | Until MMI or return to full duty | Varies by state |

## Permanent Partial Disability (PPD)
| Topic | Description | URL |
|-------|-------------|-----|
| Definition | Permanent impairment but able to work | https://www.law.cornell.edu/wex/permanent_partial_disability |
| Impairment Rating | Physician assigns percentage using AMA Guides | https://www.ama-assn.org/topics/ama-guides |
| Scheduled Injuries | Specific awards for loss of body parts (e.g., hand, foot, eye) | Varies by state |
| Non-Scheduled Injuries | Spine; head; internal organs | Varies by state |
| Wage Loss Approach | Based on actual loss of earning capacity | Varies by state |
| Impairment-Based Approach | Based solely on impairment rating | Varies by state |
| Bifurcated Approach | Combination of impairment and wage loss | Varies by state |

## Permanent Total Disability (PTD)
| Topic | Description | URL |
|-------|-------------|-----|
| Definition | Permanently unable to work in any capacity | https://www.law.cornell.edu/wex/permanent_total_disability |
| Benefit Rate | 2/3 of average weekly wage | Varies by state |
| Duration | Lifetime (or statutory cap in some states) | Varies by state |
| Presumption | Certain injuries presumed PTD (loss of both eyes, both hands, etc.) | Varies by state |
| Odd Lot Doctrine | Worker unemployable in competitive labor market | https://www.law.cornell.edu/wex/odd_lot_doctrine |

## Average Weekly Wage (AWW) Calculation
| Method | Description | URL |
|--------|-------------|-----|
| 13-Week Average | Actual earnings for 13 weeks before injury | Varies by state |
| 52-Week Average | Actual earnings for 52 weeks before injury | Varies by state |
| Similar Employee | If insufficient work history | Varies by state |
| Multiple Jobs | Concurrent employment may be included | Varies by state |
| Fringe Benefits | Some states include value of benefits | Varies by state |

## Maximum Medical Improvement (MMI)
| Topic | Description | URL |
|-------|-------------|-----|
| Definition | Condition stabilized; no further improvement expected | https://www.law.cornell.edu/wex/maximum_medical_improvement |
| Significance | Transition from temporary to permanent benefits | Varies by state |
| Determination | Treating physician or IME | Varies by state |
