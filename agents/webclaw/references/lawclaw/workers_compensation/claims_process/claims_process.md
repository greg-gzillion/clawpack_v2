# Workers' Compensation Law - Claims Process

## Notice of Injury
| Requirement | Description | URL |
|-------------|-------------|-----|
| Employee Notice to Employer | Required within statutory period (e.g., 30 days) | Varies by state |
| Written Notice | Many states require written notice | Varies by state |
| Actual Knowledge | Employer's actual knowledge may excuse formal notice | Varies by state |
| Late Notice | May bar claim unless excused | Varies by state |
| Employer Notice to Insurer | Employer must notify insurer promptly | Varies by state |

## Filing a Claim
| Requirement | Description | URL |
|-------------|-------------|-----|
| Claim Form | State-specific form | Varies by state |
| Filing Deadline | Statute of limitations (typically 1-3 years) | Varies by state |
| Discovery Rule | Limitations runs from discovery of injury/disease | Varies by state |
| Occupational Disease | Special rules for latent conditions | Varies by state |
| Tolling | May be tolled for minority or incapacity | Varies by state |

## Claim Administration
| Step | Description | URL |
|------|-------------|-----|
| First Report of Injury | Employer files with state agency and insurer | Varies by state |
| Claim Investigation | Insurer investigates compensability | Varies by state |
| Acceptance/Denial | Insurer accepts or denies claim within statutory period | Varies by state |
| Payment of Benefits | Benefits begin if accepted | Varies by state |
| Ongoing Management | Medical and disability management | Varies by state |

## Claim Denial and Dispute Resolution
| Step | Description | URL |
|------|-------------|-----|
| Notice of Denial | Insurer issues written denial | Varies by state |
| Informal Conference | Many states require informal resolution attempt | Varies by state |
| Mediation | Voluntary or mandatory mediation | Varies by state |
| Formal Hearing | Administrative hearing before workers' comp judge | Varies by state |
| Appeal | Appeal to board/commission; then appellate court | Varies by state |

| Feature | Description | URL |

## Settlements
| Type | Description | URL |
|------|-------------|-----|
| Compromise and Release (C&R) | Full and final settlement of all benefits | Varies by state |
| Stipulated Award | Ongoing benefits with agreed terms | Varies by state |
| Structured Settlement | Periodic payments over time | Varies by state |
| Medicare Set-Aside (MSA) | Funds set aside for future Medicare-covered care | https://www.cms.gov/Medicare/Coordination-of-Benefits-and-Recovery/Workers-Compensation-Medicare-Set-Aside-Arrangements/WCMSA-Overview |
