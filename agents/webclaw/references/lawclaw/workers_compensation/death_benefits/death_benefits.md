# Workers' Compensation Law - Death Benefits

## Death Benefits Overview
| Topic | Description | URL |
|-------|-------------|-----|
| Eligibility | Death resulting from work-related injury or illness | Varies by state |
| Dependents | Surviving spouse; children; other dependents | Varies by state |
| Benefit Amount | Percentage of employee's average weekly wage | Varies by state |
| Duration | Varies; often until remarriage or age 18 | Varies by state |
| Maximum | Subject to state maximum | Varies by state |

## Eligible Dependents
| Dependent | Presumption | URL |
|-----------|-------------|-----|
| Surviving Spouse | Conclusive presumption of dependency | Varies by state |
| Children Under 18 | Conclusive presumption | Varies by state |
| Children 18-23 (Full-Time Students) | Conclusive presumption in many states | Varies by state |
| Disabled Adult Children | May remain eligible | Varies by state |
| Parents | If actually dependent | Varies by state |
| Siblings | If actually dependent | Varies by state |
| Grandchildren | If actually dependent | Varies by state |

## Death Benefit Amounts
| Benefit | Description | URL |
|---------|-------------|-----|
| Weekly Benefit | Percentage of AWW (typically 66 2/3%) | Varies by state |
| Minimum/Maximum | Subject to state minimum and maximum | Varies by state |
| Total Maximum | Most states cap total death benefits | Varies by state |
| Cost of Living Adjustments | Some states provide COLAs | Varies by state |

## Duration of Death Benefits
| Dependent | Duration | URL |
|-----------|----------|-----|
| Surviving Spouse | Lifetime or until remarriage | Varies by state |
| Remarriage | Lump sum settlement (e.g., 2 years benefits) | Varies by state |
| Children | Until age 18 (or 23 if full-time student) | Varies by state |
| Disabled Children | Lifetime if disability continues | Varies by state |

## Burial and Funeral Expenses
| Benefit | Description | URL |
|---------|-------------|-----|
| Burial Allowance | Typically $2,000 - $12,000 | Varies by state |
| Funeral Expenses | Actual reasonable expenses | Varies by state |
| Transportation | Transport of remains | Varies by state |

## Death Benefits in Special Circumstances
| Circumstance | Effect | URL |
|--------------|--------|-----|
| Death After PTD Award | Dependents may continue receiving benefits | Varies by state |
| Death from Unrelated Cause | Death benefits not payable | Varies by state |
| Suicide | May be compensable if caused by work injury | Varies by state |
| No Dependents | Estate may receive burial expenses only | Varies by state |
