|--------|--------|---------|

## Short Citation Forms
| Type | Format | Example |
|------|--------|---------|
| Id. | Id. at [Page] | Id. at 495 |
| Supra | [Author], supra note [Number], at [Page] | Doe, supra note 5, at 123 |
| Infra | See infra [Section/Page] | See infra Part III |
| Short Case Name | [Short Name], [Volume] [Reporter] at [Page] | Brown, 347 U.S. at 495 |

## Signals (Bluebook Rule 1.2)
| Signal | Meaning | Usage |
|--------|---------|-------|
| [No signal] | Directly states proposition | The cited authority directly supports |
| E.g., | Examples | Authority is one of several examples |
| See | Supports proposition | Authority supports but not directly |
| See also | Additional support | Additional authority supporting |
| Cf. | Compare | Authority supports by analogy |
| Contra | Contrary | Authority directly contradicts |
| But see | But see | Authority contradicts in part |
| See generally | Background | Authority provides background |
