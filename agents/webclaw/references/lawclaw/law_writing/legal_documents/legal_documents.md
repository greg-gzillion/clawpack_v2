## IRAC/CRAC Structure
| Acronym | Meaning | Description |
|---------|---------|-------------|
| I | Issue | State the legal issue |
| R | Rule | State the applicable legal rule |
| A | Analysis | Apply rule to facts |
| C | Conclusion | Conclude on the issue |
| C | Conclusion | (CRAC: Conclusion first) |
| R | Rule | State the rule |
| A | Application | Apply rule to facts |
| C | Conclusion | Restate conclusion |
