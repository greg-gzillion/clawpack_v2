## Legalese to Avoid
| Legalese | Plain Alternative |
|----------|------------------|
| Hereinafter | (Define term and use consistently) |
| Aforesaid | The, that, those |
| Wherein | In which |
| Hereby | By this |
| Thereof | Of it |
| Witnesseth | (Omit entirely) |
| Now therefore | (Omit entirely) |
| In the event that | If |
| Prior to | Before |
| Subsequent to | After |
| Pursuant to | Under, according to |
| In accordance with | Under, following |
| For the purpose of | To |
| In order to | To |
| With respect to | About, for |
| By means of | By |
| In the nature of | Like |
| In connection with | With, about |

## Plain Language Techniques
| Technique | Before | After |
|-----------|--------|-------|
| Active Voice | The report was submitted by the agency | The agency submitted the report |
| Shorter Sentences | (45-word sentence) | (Two 20-word sentences) |
| Remove Redundancy | Null and void and of no further force or effect | Void |
| Positive Form | Do not fail to submit | Submit |
| Strong Verbs | Make a determination | Determine |
| Personal Pronouns | The claimant must file the claimant's appeal | You must file your appeal |
