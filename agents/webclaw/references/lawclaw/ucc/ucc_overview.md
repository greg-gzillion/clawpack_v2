# Uniform Commercial Code (UCC) - Overview

## What is the Uniform Commercial Code?
The Uniform Commercial Code (UCC) is a comprehensive set of laws governing commercial transactions in the United States. It is a model code drafted by the Uniform Law Commission (ULC) and the American Law Institute (ALI) that has been adopted, with some variations, by all 50 states, the District of Columbia, and U.S. territories.
