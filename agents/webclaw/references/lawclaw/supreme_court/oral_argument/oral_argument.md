# Supreme Court - Oral Argument

## Oral Argument Procedures
| Feature | Description | URL |
|---------|-------------|-----|
| Time Allotment | 30 minutes per side (typical); extended in major cases | https://www.supremecourt.gov/oral_arguments/oral_arguments.aspx |
| Argument Dates | October through April; two weeks per month | https://www.supremecourt.gov/oral_arguments/argument_calendars.aspx |
| Seating | First-come, first-served for public; bar members section | https://www.supremecourt.gov/oral_arguments/visitingthecourt.aspx |
| Hot Bench | Justices actively question counsel | https://www.supremecourt.gov/oral_arguments/oral_arguments.aspx |
| Audio Recordings | Released at end of argument week | https://www.supremecourt.gov/oral_arguments/argument_audio.aspx |
| Transcripts | Available same day | https://www.supremecourt.gov/oral_arguments/argument_transcript.aspx |
| Livestreaming | Audio livestreamed since 2020 | https://www.supremecourt.gov/oral_arguments/live.aspx |
