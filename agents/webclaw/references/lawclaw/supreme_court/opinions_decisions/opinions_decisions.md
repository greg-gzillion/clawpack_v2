# Supreme Court - Opinions and Decisions

## Types of Opinions
| Type | Description | URL |
|------|-------------|-----|
| Majority Opinion | Opinion of the Court; binding precedent | https://www.supremecourt.gov/opinions/opinions.aspx |
| Concurring Opinion | Agrees with result; different reasoning | https://www.supremecourt.gov/opinions/opinions.aspx |
| Dissenting Opinion | Disagrees with majority | https://www.supremecourt.gov/opinions/opinions.aspx |
| Per Curiam Opinion | Unsigned opinion for the Court | https://www.supremecourt.gov/opinions/opinions.aspx |
| Plurality Opinion | Opinion with most votes but not majority | https://www.law.cornell.edu/wex/plurality_opinion |
| Summary Disposition | Decided without full briefing or argument | https://www.supremecourt.gov/opinions/opinions.aspx |

## Opinion Drafting Process
| Step | Description | URL |
|------|-------------|-----|
| Initial Draft | Assigned Justice drafts opinion | https://www.supremecourt.gov/about/procedures.aspx |
| Circulation | Draft circulated to all chambers | https://www.supremecourt.gov/about/procedures.aspx |
| Comments and Responses | Justices respond with join memos, suggestions, or dissents | https://www.supremecourt.gov/about/procedures.aspx |
| Revisions | Author revises based on feedback | https://www.supremecourt.gov/about/procedures.aspx |
| Joining | Justices formally join opinions | https://www.supremecourt.gov/about/procedures.aspx |
| Release | Opinion announced from bench or released online | https://www.supremecourt.gov/opinions/opinions.aspx |
