# Texas Constitution of 1876

## Overview
| Feature | Description |
|---------|-------------|
| Current Constitution | Constitution of Texas (1876) |
| Previous Constitutions | 1827, 1836, 1845, 1861, 1866, 1869 |
| Amendments | 500+ amendments |
| Length | ~86,000 words (second longest) |

## Unique Features
- **Second longest constitution in U.S. (after Alabama)**
- Strong homestead protections (Art. XVI, § 50)
- No state income tax (constitutional prohibition)
- Plural executive (multiple elected officials)
