# North Dakota Constitution of 1889

## Overview
| Feature | Description |
|---------|-------------|
| Current Constitution | Constitution of North Dakota (1889) |
| Previous Constitutions | None (statehood 1889) |
| Amendments | 150+ amendments |
| Length | ~15,000 words |

## Unique Features
- State-owned bank (Bank of North Dakota)
- State-owned mill (North Dakota Mill and Elevator)
- Strong initiative and referendum powers
