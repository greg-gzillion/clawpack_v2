# Alaska Constitution of 1956

## Overview
| Feature | Description |
|---------|-------------|
| Current Constitution | Constitution of Alaska (1956) |
| Previous Constitutions | None (statehood 1959) |
| Amendments | 28 amendments |
| Length | ~15,000 words |

## Unique Features
- Article VIII: Strong natural resource protections
- Article XI: Robust initiative and referendum powers
- Right to privacy explicitly protected (Art. I, § 22)
- Permanent Fund established by constitutional amendment (1976)
