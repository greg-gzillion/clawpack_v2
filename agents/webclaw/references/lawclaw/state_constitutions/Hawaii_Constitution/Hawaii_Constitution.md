# Hawaii Constitution of 1950

## Overview
| Feature | Description |
|---------|-------------|
| Current Constitution | Constitution of Hawaii (1950) |
| Previous Constitutions | None (statehood 1959) |
| Amendments | 100+ amendments |
| Length | ~30,000 words |

## Unique Features
- Office of Hawaiian Affairs (Art. XII)
- Explicit right to privacy (Art. I, § 6)
- Strong conservation provisions (Art. XI)
- Public employees' collective bargaining (Art. XIII)
