# Montana Constitution of 1972

## Overview
| Feature | Description |
|---------|-------------|
| Current Constitution | Constitution of Montana (1972) |
| Previous Constitutions | 1889 |
| Amendments | 30+ amendments |
| Length | ~13,000 words |

## Unique Features
- **Explicit right to clean and healthful environment (Art. II, § 3; Art. IX)**
- Right to privacy (Art. II, § 10)
- Strong environmental protections
- Initiative and referendum powers
