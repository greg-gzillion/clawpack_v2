# Virginia Constitution of 1971

## Overview
| Feature | Description |
|---------|-------------|
| Current Constitution | Constitution of Virginia (1971) |
| Previous Constitutions | 1776, 1830, 1851, 1864, 1870, 1902 |
| Amendments | 40+ amendments |
| Length | ~22,000 words |

## Unique Features
- Dillon Rule state (limited local government power)
- Strong conservation provisions (Art. XI)
- Independent cities system
