# Louisiana Constitution of 1974 and Civil Code

## Overview
| Feature | Description |
|---------|-------------|
| Current Constitution | Constitution of Louisiana (1974) |
| Previous Constitutions | 1812, 1845, 1852, 1864, 1868, 1879, 1898, 1913, 1921 |
| Legal System | Civil Law (based on French/Spanish codes) |
| Civil Code | Louisiana Civil Code (separate from Constitution) |
| Amendments | 200+ amendments |
| Length | ~70,000 words |

## Unique Features
- **Only civil law jurisdiction in United States**
- Based on French and Spanish civil law traditions
- Civil Code is separate from Constitution
- Napoleonic Code influence
- Parishes instead of counties
- Civil law property concepts (usufruct, naked ownership, predial servitudes)
