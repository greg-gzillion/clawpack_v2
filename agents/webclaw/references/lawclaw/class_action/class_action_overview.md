# Class Action Law - Overview

## What is a Class Action?
A class action is a procedural device that allows one or more plaintiffs to file and prosecute a lawsuit on behalf of a larger group or "class." Class actions promote judicial economy by consolidating numerous similar individual claims into a single proceeding.

## Purposes of Class Actions
| Purpose | Description |
|---------|-------------|
| Judicial Economy | Avoid duplicative litigation of common issues |
| Access to Justice | Enable small claims that would be uneconomical to bring individually |
| Deterrence | Deter wrongful conduct that causes widespread but small individual harm |
| Consistency | Ensure uniform treatment of similarly situated claimants |

## Types of Class Actions
| Type | FRCP Rule | Description |
|------|-----------|-------------|
| (b)(1) | Limited Fund/Potential Incompatibility | Individual actions would create inconsistent standards or impair other class members' interests |
| (b)(2) | Injunctive/Declaratory Relief | Party opposing class acted on grounds generally applicable to class |
| (b)(3) | Damages Class | Common questions predominate; class action superior to other methods |
