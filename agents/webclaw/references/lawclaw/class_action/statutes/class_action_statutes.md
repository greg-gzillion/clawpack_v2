## CAFA Jurisdiction Requirements
| Requirement | Threshold | URL |
|-------------|-----------|-----|
| Minimal Diversity | Any class member diverse from any defendant | https://www.law.cornell.edu/uscode/text/28/1332#d_2 |
| Amount in Controversy | Aggregate claims exceed $5,000,000 | https://www.law.cornell.edu/uscode/text/28/1332#d_2 |
| Numerosity | 100+ class members | https://www.law.cornell.edu/uscode/text/28/1332#d_5_B |

## CAFA Exceptions
| Exception | Description | URL |
|-----------|-------------|-----|
| Local Controversy | >2/3 of class from state; significant relief from in-state defendant | https://www.law.cornell.edu/uscode/text/28/1332#d_4_A |
| Home State Exception | Primary defendants from same state; 2/3+ class from that state | https://www.law.cornell.edu/uscode/text/28/1332#d_4_B |
| Securities Class Actions | Excluded from CAFA | https://www.law.cornell.edu/uscode/text/28/1332#d_9_A |
| Corporate Governance | Internal affairs of corporations | https://www.law.cornell.edu/uscode/text/28/1332#d_9_B |
