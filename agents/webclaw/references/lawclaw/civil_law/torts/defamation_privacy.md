# Tort Law - Defamation and Privacy Torts

## Defamation - Overview
Defamation is a false statement that harms reputation.

- **Cornell LII - Defamation**: https://www.law.cornell.edu/wex/defamation

### Libel vs. Slander
| Type | Description | URL |
|------|-------------|-----|
| Libel | Written or permanent defamation | https://www.law.cornell.edu/wex/libel |
| Slander | Spoken or transient defamation | https://www.law.cornell.edu/wex/slander |
| Libel Per Se | Defamatory on its face | https://www.law.cornell.edu/wex/libel_per_se |
| Slander Per Se | Serious categories (crime, disease, unchastity, business) | https://www.law.cornell.edu/wex/slander_per_se |

## Defamation Elements
| Element | Description | URL |
|---------|-------------|-----|
| False Statement | Statement of fact, not opinion | https://www.law.cornell.edu/wex/defamation#elements |
| Publication | Communicated to third party | https://www.law.cornell.edu/wex/publication_defamation |

## Constitutional Limits
