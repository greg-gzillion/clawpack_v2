# State Statutory Law - Overview

## What is State Statutory Law?
State statutory law consists of laws enacted by state legislatures and signed by state governors. Each state has its own statutory code, organized by subject matter, that governs conduct within that state. State statutes address areas of traditional state concern including criminal law, family law, property, contracts, torts, and business regulation.
