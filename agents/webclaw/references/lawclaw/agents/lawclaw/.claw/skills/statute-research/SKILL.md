# Statute Research Skill

## Trigger
/research [statute_name] in [jurisdiction]

## Sources
- U.S. Code (USC)
- Code of Federal Regulations (CFR)
- State codes (by state)
- Uniform Law Commission (ULC)

## Workflow
1. Identify jurisdiction and statute name
2. Search relevant legal database
3. Extract key provisions
4. Summarize in plain English
5. Provide citation

## Output Format
============================================================
STATUTE RESEARCH: [statute_name]
============================================================

JURISDICTION: [state/federal]

CITATION: [citation]

KEY PROVISIONS:
- [provision 1]
- [provision 2]

SUMMARY: [plain English explanation]

EFFECTIVE DATE: [date]
============================================================
