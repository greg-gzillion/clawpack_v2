# Landlord-Tenant Law - Security Deposits

## Security Deposit Basics
| Topic | Description | URL |
|-------|-------------|-----|
| Purpose | Security for landlord against tenant defaults | https://www.law.cornell.edu/wex/security_deposit |
| Maximum Amount | State law limits (1-3 months' rent) | State law |
| Separate Account Requirement | Many states require segregated account | State law |
| Interest Requirement | Some states/cities require interest payments | State law |
| Receipt Requirement | Written receipt upon payment | State law |

## Permitted Uses of Security Deposit
| Use | Description | URL |
|-----|-------------|-----|
| Unpaid Rent | Accrued rent obligations | Varies |
| Damage Beyond Normal Wear and Tear | Repair of tenant-caused damage | Varies |
| Cleaning | Excessive cleaning beyond normal | Varies |
| Unpaid Utilities | If lease requires tenant payment | Varies |
| Early Termination Fees | If lease provides | Varies |
| Abandoned Property Removal | Cost of removal and storage | State law |

## Normal Wear and Tear vs. Damage
| Normal Wear and Tear | Damage |
|---------------------|--------|
| Faded paint | Holes in walls |
| Worn carpet (traffic patterns) | Stains or burns in carpet |
| Loose door handles | Broken door handles |
| Scuffed floors | Gouged floors |
| Faded curtains | Missing or torn curtains |
| Minor nail holes | Large holes requiring patching |
| Worn appliance finish | Broken appliances |
| Dusty blinds | Broken/missing blinds |

## Security Deposit Return Process
| Step | Description | URL |
|------|-------------|-----|
| Move-Out Inspection | Initial and final inspection rights | State law |
| Tenant's Right to Be Present | Some states allow tenant participation | State law |
| Deadline for Return | Statutory deadline (14-60 days) | State law |
| Itemized Statement | Written list of deductions | State law |
| Supporting Documentation | Receipts for repairs/cleaning | State law |
| Forwarding Address | Tenant must provide forwarding address | State law |
| Failure to Return Timely | Penalties; forfeiture of right to withhold | State law |

## Penalties for Wrongful Withholding
| Penalty | Description | URL |
|---------|-------------|-----|
| Statutory Damages | 2-3 times deposit amount | State law |
| Attorney's Fees | Prevailing tenant may recover fees | State law |
| Court Costs | Filing and service fees | State law |
| Interest | Pre-judgment and post-judgment interest | State law |
| Forfeiture of Claims | Landlord may lose right to assert damages | State law |

## Security Deposit State Law Variations
| State | Maximum | Return Deadline | Interest Required | Separate Account |
|-------|---------|-----------------|-------------------|------------------|
| California | 2 months (3 if furnished) | 21 days | No | No |
| New York | 1 month | 14 days | Yes (buildings with 6+ units) | Yes |
| Texas | No statutory limit | 30 days | No | No |
| Florida | No statutory limit | 15-60 days | Yes (or 5% simple interest) | Yes |
| Illinois | No statutory limit | 30-45 days | Yes (buildings with 25+ units) | Yes |
| Massachusetts | 1 month | 30 days | Yes | Yes |
| Washington | No statutory limit | 21 days | No | Yes |
| New Jersey | 1.5 months | 30 days | Yes | Yes |
