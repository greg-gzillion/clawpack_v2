# State Case Law - State Court Structure

## Typical State Court Hierarchy
| Level | Court Type | Description | URL |
|-------|------------|-------------|-----|
| Highest Court | State Supreme Court | Final appellate authority; discretionary review in most states | https://www.ncsc.org/information-and-resources/state-court-structures |
| Intermediate Appellate Court | Court of Appeals | Mandatory review of trial court decisions | https://www.ncsc.org/information-and-resources/state-court-structures |
| Trial Courts of General Jurisdiction | Superior/District/Circuit Court | Civil and criminal matters; unlimited jurisdiction | https://www.ncsc.org/information-and-resources/state-court-structures |
| Trial Courts of Limited Jurisdiction | Municipal/County/Justice Courts | Minor civil; traffic; small claims; misdemeanors | https://www.ncsc.org/information-and-resources/state-court-structures |
| Specialty Courts | Family/Drug/Veterans Courts | Problem-solving courts with specialized dockets | https://www.ncsc.org/services-and-experts/areas-of-expertise/problem-solving-courts |

## State Supreme Courts
| Feature | Description | URL |
|---------|-------------|-----|
| Composition | 5-9 justices typically | https://www.ncsc.org/information-and-resources/state-court-structures |
| Jurisdiction | Discretionary review in most states; mandatory for death penalty and certain cases | https://www.ncsc.org/information-and-resources/state-court-structures |
| Selection Methods | Election (partisan/nonpartisan); gubernatorial appointment; legislative appointment; merit selection | https://www.judicialselection.com |
| Term Length | 6-14 years typically; some states have life tenure with retention elections | https://www.ncsc.org/information-and-resources/state-court-structures |

## State Intermediate Appellate Courts
| Feature | Description | URL |
|---------|-------------|-----|

| States without IACs | Delaware, Maine, Montana, Nevada, New Hampshire, Rhode Island, South Dakota, Vermont, West Virginia, Wyoming (Note: Nevada and West Virginia have IACs; some lists vary) | https://www.ncsc.org/information-and-resources/state-court-structures |
| Jurisdiction | Mandatory review of final trial court judgments | https://www.ncsc.org/information-and-resources/state-court-structures |
| Panel Decisions | Usually 3-judge panels; en banc review available | https://www.ncsc.org/information-and-resources/state-court-structures |

## Judicial Selection Methods by State
| Method | Description | Example States | URL |
|--------|-------------|----------------|-----|
| Partisan Election | Candidates run with party affiliation | Alabama, Illinois, Louisiana, New Mexico, Pennsylvania, Texas | https://www.judicialselection.com |
| Nonpartisan Election | Candidates run without party affiliation | California, Florida, Michigan, Minnesota, Ohio, Washington | https://www.judicialselection.com |
| Gubernatorial Appointment | Governor appoints (often with commission recommendation) | California (appellate), Maine, New Jersey | https://www.judicialselection.com |
| Legislative Appointment | Legislature selects judges | South Carolina, Virginia | https://www.judicialselection.com |
| Merit Selection (Missouri Plan) | Commission nominates; governor appoints; retention election | Arizona, Colorado, Indiana, Iowa, Kansas, Missouri, Nebraska | https://www.judicialselection.com |
