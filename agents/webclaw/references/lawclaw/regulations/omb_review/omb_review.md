## EO 12866 Review Process
| Step | Description | URL |
|------|-------------|-----|
| Agency Submission | Agency submits draft rule and RIA to OIRA | https://www.whitehouse.gov/omb/information-regulatory-affairs/ |
| OIRA Review | 90-day review period (extendable) | https://www.archives.gov/federal-register/executive-orders/1993.html |
| Return Letter | OIRA may return rule for reconsideration | https://www.whitehouse.gov/omb/information-regulatory-affairs/ |
| Interagency Review | Coordination with other affected agencies | https://www.whitehouse.gov/omb/information-regulatory-affairs/ |
| Disclosure | OIRA meetings and communications disclosed | https://www.reginfo.gov |
