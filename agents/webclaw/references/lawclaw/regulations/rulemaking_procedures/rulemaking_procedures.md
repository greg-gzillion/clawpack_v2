# Regulations and Administrative Law - Rulemaking Procedures

## The Rulemaking Process
| Step | Description | URL |
|------|-------------|-----|
| 1. Regulatory Agenda | Agency publishes semi-annual regulatory agenda | https://www.reginfo.gov/public/do/eAgendaMain |
| 2. Advanced Notice of Proposed Rulemaking (ANPRM) | Optional; solicits early public input | https://www.regulations.gov |
| 3. Notice of Proposed Rulemaking (NPRM) | Required under APA ｧ 553; published in Federal Register | https://www.federalregister.gov |
| 4. Public Comment Period | Typically 30-60 days; may be extended | https://www.regulations.gov |
| 5. Agency Consideration | Agency reviews and responds to comments | https://www.acus.gov |
| 6. Final Rule | Published in Federal Register with effective date | https://www.federalregister.gov |
| 7. Judicial Review | Potential challenge in federal court | https://www.law.cornell.edu/uscode/text/5/702 |

## Notice of Proposed Rulemaking Requirements
| Element | Description | URL |
|---------|-------------|-----|
| Statement of Time, Place, Nature | Public participation procedures | https://www.law.cornell.edu/uscode/text/5/553 |
| Reference to Legal Authority | Statutory authority for rule | https://www.law.cornell.edu/uscode/text/5/553 |
| Terms or Substance | Description of proposed rule or issues involved | https://www.law.cornell.edu/uscode/text/5/553 |
| Logical Outgrowth Test | Final rule must be logical outgrowth of proposal | https://www.law.cornell.edu/wex/logical_outgrowth_test |

## Comment Period Requirements
| Requirement | Description | URL |
|-------------|-------------|-----|
| Minimum Period | Generally 30-60 days | https://www.regulations.gov |
| Extension | May be extended for complex rules | https://www.acus.gov |
| Reopening | Substantial changes may require reopening | https://www.law.cornell.edu/wex/logical_outgrowth_test |
| Electronic Comments | Accepted through Regulations.gov | https://www.regulations.gov |

## Final Rule Requirements
| Requirement | Description | URL |
|-------------|-------------|-----|
| Concise General Statement | Basis and purpose of rule | https://www.law.cornell.edu/uscode/text/5/553 |
| Response to Comments | Address significant comments | https://www.law.cornell.edu/wex/response_to_comments |
| Effective Date | At least 30 days after publication | https://www.law.cornell.edu/uscode/text/5/553 |

## Exceptions to Notice and Comment
| Exception | Description | URL |
|-----------|-------------|-----|
| Interpretative Rules | Clarify existing law without binding effect | https://www.law.cornell.edu/uscode/text/5/553 |
| General Statements of Policy | Non-binding guidance | https://www.law.cornell.edu/uscode/text/5/553 |
| Rules of Agency Organization, Procedure, or Practice | Internal agency procedures | https://www.law.cornell.edu/uscode/text/5/553 |
| Good Cause | Notice and comment impracticable, unnecessary, or contrary to public interest | https://www.law.cornell.edu/uscode/text/5/553 |
| Military or Foreign Affairs | Exempt from APA requirements | https://www.law.cornell.edu/uscode/text/5/553 |

## Direct Final Rulemaking
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Rule published as final with notice that it will become effective unless adverse comment received | https://www.acus.gov |
| Purpose | Expedite non-controversial rules | https://www.acus.gov |
| Adverse Comment | Withdrawal and reproposal if adverse comment received | https://www.acus.gov |
| Notice Requirement | Must publish notice of direct final rulemaking | https://www.federalregister.gov |

## Interim Final Rulemaking
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Rule effective immediately with post-promulgation comment period | https://www.acus.gov |
| Good Cause Required | Must satisfy APA ｧ 553 good cause exception | https://www.law.cornell.edu/uscode/text/5/553 |
| Post-Promulgation Comments | Agency considers comments and may revise rule | https://www.acus.gov |
| Common Uses | Emergencies; statutory deadlines | https://www.acus.gov |

## Negotiated Rulemaking (Reg-Neg)
