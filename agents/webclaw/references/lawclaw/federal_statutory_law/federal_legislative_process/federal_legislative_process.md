## How a Bill Becomes Law
| Stage | Description | URL |
|-------|-------------|-----|
| 1. Bill Introduction | Member introduces bill in House or Senate | https://www.congress.gov/legislative-process |
| 2. Committee Referral | Bill referred to committee(s) with jurisdiction | https://www.congress.gov/legislative-process/committee-consideration |
| 3. Committee Action | Hearings; markup; committee vote | https://www.congress.gov/legislative-process/committee-consideration |
| 4. Floor Action | Debate; amendments; passage by one chamber | https://www.congress.gov/legislative-process/house-floor |
| 5. Second Chamber | Process repeated in other chamber | https://www.congress.gov/legislative-process/senate-floor |
| 6. Conference Committee | Resolves differences between versions | https://www.congress.gov/legislative-process/resolving-differences |
| 7. Final Passage | Both chambers approve identical version | https://www.congress.gov/legislative-process/presidential-action |
| 8. Presidential Action | Sign; veto; pocket veto; become law without signature | https://www.congress.gov/legislative-process/presidential-action |
| 9. Veto Override | 2/3 of both chambers required | https://www.congress.gov/legislative-process/presidential-action |
| 10. Publication | Slip law; Statutes at Large; U.S. Code | https://www.govinfo.gov |

## Types of Legislation
