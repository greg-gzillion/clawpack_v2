# Property Law - Adverse Possession

## Elements of Adverse Possession
| Element | Description | URL |
|---------|-------------|-----|
| Actual | Physical occupation of property | https://www.law.cornell.edu/wex/adverse_possession |
| Open and Notorious | Visible to true owner | https://www.law.cornell.edu/wex/adverse_possession |
| Exclusive | Not shared with true owner or public | https://www.law.cornell.edu/wex/adverse_possession |
| Hostile / Adverse | Without permission; under claim of right | https://www.law.cornell.edu/wex/adverse_possession |
| Continuous | Uninterrupted for statutory period | https://www.law.cornell.edu/wex/adverse_possession |
| Statutory Period | Varies by state (typically 5-20 years) | https://www.ncsl.org/environment-and-natural-resources/adverse-possession |

## Hostility Standards
| Standard | Description | URL |
|----------|-------------|-----|
| Objective Test (Majority) | State of mind irrelevant; possession without permission is adverse | https://www.law.cornell.edu/wex/adverse_possession |
| Good Faith Test | Possessor must believe in good faith that they own property | https://www.law.cornell.edu/wex/adverse_possession |
| Aggressive Trespass Test | Possessor must know they do not own property but intend to claim it | https://www.law.cornell.edu/wex/adverse_possession |
