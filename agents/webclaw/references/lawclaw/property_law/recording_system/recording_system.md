# Property Law - Recording System

## Types of Recording Acts
| Type | Description | URL |
|------|-------------|-----|
| Race Statute | First to record prevails regardless of notice | https://www.law.cornell.edu/wex/race_statute |
| Notice Statute | Subsequent purchaser without notice prevails over unrecorded prior interest | https://www.law.cornell.edu/wex/notice_statute |
| Race-Notice Statute | Subsequent purchaser without notice who records first prevails | https://www.law.cornell.edu/wex/race_notice_statute |
| Pure Race | First to record wins (minority) | https://www.law.cornell.edu/wex/race_statute |
