## Private Rights of Action
| Statute | Private Right? | Damages | URL |
|---------|---------------|---------|-----|
| FTC Act | No (FTC only) | N/A | https://www.law.cornell.edu/uscode/text/15/45 |
| FCRA | Yes | Actual, statutory ($100-$1,000), punitive, attorney's fees | https://www.law.cornell.edu/uscode/text/15/1681n |
| FDCPA | Yes | Actual, statutory ($1,000), attorney's fees | https://www.law.cornell.edu/uscode/text/15/1692k |
| TCPA | Yes | $500 per violation ($1,500 willful) | https://www.law.cornell.edu/uscode/text/47/227#b_3 |
| TILA | Yes | Actual, statutory, attorney's fees | https://www.law.cornell.edu/uscode/text/15/1640 |
| State UDAP | Yes (varies) | Varies by state | Varies |

## Statutory Damages Summary
| Statute | Statutory Damages | Willful Violation |
|---------|-------------------|-------------------|
| FCRA | $100-$1,000 per violation | Actual + punitive |
| FDCPA | Up to $1,000 per action | N/A |
| TCPA | $500 per call | $1,500 per call |
| TILA | $400-$4,000 per violation | Actual + statutory |
| CAN-SPAM | No private right | FTC enforcement only |
