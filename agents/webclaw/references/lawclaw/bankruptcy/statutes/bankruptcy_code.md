# Bankruptcy Code - 11 U.S.C. Key Provisions

## Overview
The Bankruptcy Code is codified at Title 11 of the United States Code. It provides the federal statutory framework for bankruptcy proceedings.

## Chapter Structure
| Chapter | Sections | Subject |
|---------|----------|---------|
| **1** | §§ 101-112 | General Provisions |
| **3** | §§ 301-366 | Case Administration |
| **5** | §§ 501-562 | Creditors, Debtors, and Estate |
| **7** | §§ 701-784 | Liquidation |
| **9** | §§ 901-946 | Municipality Bankruptcy |
| **11** | §§ 1101-1195 | Reorganization |
| **12** | §§ 1201-1232 | Family Farmer/Fisherman |
| **13** | §§ 1301-1330 | Individual Debt Adjustment |
| **15** | §§ 1501-1532 | Ancillary and Cross-Border Cases |

## Key Definitions (§ 101)
| Term | Definition |
|------|------------|
| **Automatic Stay** | § 362 injunction against collection actions |
| **Claim** | Right to payment, secured or unsecured |
| **Debtor** | Person/business filing bankruptcy |
| **Discharge** | Release of debtor from personal liability |
| **Estate** | All legal/equitable interests of debtor at filing |
| **Exemptions** | Property debtor may keep |
| **Insider** | Director, officer, relative, controlling person |

## Key Provisions
| Section | Topic | Summary |
|---------|-------|---------|
| **§ 362** | Automatic Stay | Stops collection actions upon filing |
| **§ 363** | Use/Sale of Estate Property | Court approval required outside ordinary course |
| **§ 365** | Executory Contracts | Debtor may assume or reject |
| **§ 507** | Priority Claims | Order of payment (administrative, wages, taxes) |
| **§ 523** | Exceptions to Discharge | Debts that survive bankruptcy (fraud, taxes, student loans) |
| **§ 524** | Discharge Injunction | Permanent injunction against collection |
| **§ 547** | Preferences | Trustee may recover preferential transfers (90 days pre-petition) |
| **§ 548** | Fraudulent Transfers | Trustee may avoid transfers made with intent to hinder/delay/defraud |

## Bankruptcy Rules
- **Federal Rules of Bankruptcy Procedure**: Procedural rules governing bankruptcy cases
- **Official Bankruptcy Forms**: Mandatory forms for petitions, schedules, statements
- **Local Bankruptcy Rules**: District-specific procedural requirements
