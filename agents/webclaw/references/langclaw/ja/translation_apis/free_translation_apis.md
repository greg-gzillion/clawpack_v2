# Free Translation APIs for ja

## Free Tier APIs
- [MyMemory API](https://api.mymemory.translated.net/get?q=TEXT&langpair=en|ja)
  - Free: 1000 requests/day
  - No API key required for basic use
  
- [LibreTranslate](https://libretranslate.com/?source=en&target=ja)
  - Open source
  - Self-hostable
  
- [Lingva Translate](https://lingva.ml/en/ja/)
  - Privacy-focused Google Translate alternative

## Rate-Limited Free APIs
- [Google Translate (Unofficial)](https://translate.google.com/?sl=en&tl=ja)
- [Yandex Translate](https://translate.yandex.com/?lang=en-ja)
- [Bing Translator](https://www.bing.com/translator?from=en&to=ja)
