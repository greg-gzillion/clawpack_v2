# Endocrinology - Overview

## What is Endocrinology?
Endocrinology is the medical specialty concerned with the endocrine system, which consists of glands that secrete hormones directly into the bloodstream to regulate metabolism, growth and development, tissue function, sleep, mood, and reproduction.

## Major Endocrine Glands and Hormones
| Gland | Hormones | Primary Function |
|-------|----------|------------------|
| Hypothalamus | TRH, CRH, GnRH, GHRH, Somatostatin, Dopamine | Regulates pituitary |
| Anterior Pituitary | TSH, ACTH, LH, FSH, GH, Prolactin | Tropic hormones |
| Posterior Pituitary | ADH (Vasopressin), Oxytocin | Water balance, Labor |
| Thyroid | T4, T3, Calcitonin | Metabolism, Calcium |
| Parathyroid | PTH | Calcium homeostasis |
| Adrenal Cortex | Cortisol, Aldosterone, Androgens | Stress, BP, Sex hormones |
| Adrenal Medulla | Epinephrine, Norepinephrine | Fight or flight |
| Pancreas (Endocrine) | Insulin, Glucagon, Somatostatin | Glucose homeostasis |
| Ovaries | Estrogen, Progesterone | Reproduction |
| Testes | Testosterone | Reproduction |
| Pineal Gland | Melatonin | Sleep-wake cycle |

## Hormone Classification
| Class | Examples | Mechanism |
|-------|----------|-----------|
| Peptide/Protein | Insulin, GH, PTH, ACTH | Cell surface receptors |
| Steroid | Cortisol, Aldosterone, Testosterone, Estrogen | Nuclear receptors |
| Amine | Thyroid hormones, Catecholamines | Nuclear (T4/T3) or Surface (catecholamines) |
| Eicosanoid | Prostaglandins, Leukotrienes | Cell surface receptors |

## Feedback Loops
| Type | Description | Example |
|------|-------------|---------|
| Negative Feedback | Hormone inhibits its own stimulation | Cortisol inhibits CRH/ACTH |
| Positive Feedback | Hormone stimulates its own production | Oxytocin during labor |
| Ultra-short Loop | Hormone inhibits its own releasing factor | ACTH inhibits CRH |
