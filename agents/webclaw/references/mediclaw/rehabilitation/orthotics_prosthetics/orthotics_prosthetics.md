# Rehabilitation Medicine - Orthotics and Prosthetics

## Lower Extremity Orthotics
| Orthosis | Description | Indications | URL |
|----------|-------------|-------------|-----|
| **AFO (Ankle-Foot Orthosis)** | **Controls ankle and foot** | **Foot drop**, **Ankle instability**, **Mild-moderate spasticity** | https://www.aaos.org |
| **KAFO (Knee-Ankle-Foot Orthosis)** | **Controls knee, Ankle, Foot** | **Quadriceps weakness**, **Knee instability** | https://www.aaos.org |
| **HKAFO (Hip-Knee-Ankle-Foot Orthosis)** | **Controls hip, Knee, Ankle, Foot** | **Hip instability**, **Paraplegia** | https://www.aaos.org |
| **RGO (Reciprocating Gait Orthosis)** | **Allows reciprocal gait** via mechanical linkage | **Paraplegia** (T9-L2) | https://www.aaos.org |
| **SMO (Supramalleolar Orthosis)** | **Controls foot** (subtalar, Midtarsal) | **Pes planus**, **Mild ankle instability** | https://www.aaos.org |

## Upper Extremity Orthotics
| Orthosis | Description | Indications | URL |
|----------|-------------|-------------|-----|
| **Wrist Cock-Up Splint** | **Maintains wrist in neutral/extension** | **Carpal tunnel syndrome**, **Wrist drop** | https://www.assh.org/handcare |
| **Thumb Spica Splint** | **Immobilizes thumb** | **De Quervain tenosynovitis**, **Thumb CMC arthritis**, **Scaphoid fracture** | https://www.assh.org/handcare |
| **Resting Hand Splint** | **Maintains hand in functional position** | **Spasticity**, **Contracture prevention** | https://www.assh.org/handcare |
| **Anti-Claw Orthosis** | **Prevents MCP hyperextension** | **Ulnar nerve palsy**, **Median/ulnar nerve palsy** | https://www.assh.org/handcare |
