# Rehabilitation Medicine - Traumatic Brain Injury Rehabilitation

## TBI Rehabilitation - Rancho Los Amigos Scale
| Level | Description | Rehabilitation Focus | URL |
|-------|-------------|---------------------|-----|
| **I** | **No Response** | **Prevent complications**, **Sensory stimulation** | https://www.biausa.org |
| **II** | **Generalized Response** | **Sensory stimulation**, **Prevent complications** | https://www.biausa.org |
| **III** | **Localized Response** | **Sensory stimulation**, **Early mobilization** | https://www.biausa.org |
| **IV** | **Confused/Agitated** | **Structure**, **Routine**, **Minimize stimulation**, **Safety** | https://www.biausa.org |
| **V** | **Confused/Inappropriate** | **Orientation**, **Cueing**, **Structure** | https://www.biausa.org |
| **VI** | **Confused/Appropriate** | **Cognitive rehabilitation**, **Increasing independence** | https://www.biausa.org |
| **VII** | **Automatic/Appropriate** | **Community reintegration**, **Return to work/school** | https://www.biausa.org |
| **VIII** | **Purposeful/Appropriate** | **Return to work/school**, **Long-term follow-up** | https://www.biausa.org |
