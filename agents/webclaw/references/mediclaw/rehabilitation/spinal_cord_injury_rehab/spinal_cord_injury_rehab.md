# Rehabilitation Medicine - Spinal Cord Injury Rehabilitation

## SCI - ASIA Impairment Scale (AIS)
| Grade | Description | URL |
|-------|-------------|-----|
| **A** | **Complete**: No sensory or motor function in S4-S5 | https://www.asia-spinalinjury.org |
| **B** | **Sensory Incomplete**: Sensory but no motor function below level, Including S4-S5 | https://www.asia-spinalinjury.org |
| **C** | **Motor Incomplete**: Motor function below level, >50% key muscles <3/5 | https://www.asia-spinalinjury.org |
| **D** | **Motor Incomplete**: Motor function below level, =50% key muscles =3/5 | https://www.asia-spinalinjury.org |
| **E** | **Normal**: Normal sensory and motor function | https://www.asia-spinalinjury.org |

## SCI - Functional Goals by Level
| Level | Key Muscles | Functional Goals | Equipment | URL |
|-------|-------------|------------------|-----------|-----|
| **C1-C4** | **Neck**, **Diaphragm** (C3-C5) | **Ventilator-dependent** (C1-C3), **Power wheelchair** (chin/head control), **ECU**, **Total assist** | Power wheelchair, Ventilator, ECU | https://www.nscisc.uab.edu |
| **C5** | **Elbow flexion** (biceps, Brachialis) | **Power wheelchair** (hand control), **Assist with feeding/grooming** (adaptive) | Power wheelchair, Mobile arm support | https://www.nscisc.uab.edu |
| **C6** | **Wrist extension** (ECRL, ECRB) | **Tenodesis grasp**, **Independent feeding/grooming**, **Manual wheelchair** (short) | Manual wheelchair, Tenodesis splint | https://www.nscisc.uab.edu |
| **C7-C8** | **Elbow extension** (triceps), **Finger flexion** | **Independent transfers**, **Manual wheelchair** | Manual wheelchair | https://www.nscisc.uab.edu |
| **T1-T12** | **Intercostals**, **Abdominals** | **Independent with wheelchair**, **Standing frame**, **Therapeutic ambulation** | Manual wheelchair, Standing frame, KAFO | https://www.nscisc.uab.edu |
| **L1-L5** | **Hip flexors**, **Knee extensors**, **Ankle dorsiflexors** | **Community ambulation** (with braces/crutches), **Bowel/bladder management** | AFO, Crutches, Walker | https://www.nscisc.uab.edu |

## SCI - Medical Complications and Management
| Complication | Management | URL |
|--------------|------------|-----|
| **Neurogenic Bladder** | **Clean intermittent catheterization (CIC)** , **Anticholinergics**, **Botulinum toxin**, **Suprapubic catheter** | https://www.nscisc.uab.edu |
| **Neurogenic Bowel** | **Bowel program** (timed, Suppository, Digital stimulation), **Dietary fiber/fluids** | https://www.nscisc.uab.edu |
| **Spasticity** | **Stretching**, **Oral medications** (baclofen, Tizanidine), **Botulinum toxin**, **Intrathecal baclofen** | https://www.nscisc.uab.edu |
| **Autonomic Dysreflexia** | **Identify and remove trigger** (bladder, Bowel, Skin), **Elevate head**, **Antihypertensives** | https://www.nscisc.uab.edu |
| **Pressure Injuries** | **Pressure relief**, **Proper cushion/mattress**, **Skin checks**, **Wound care** | https://www.nscisc.uab.edu |
| **Heterotopic Ossification** | **ROM**, **NSAIDs**, **Bisphosphonates**, **Radiation**, **Surgical excision** | https://www.nscisc.uab.edu |
| **Osteoporosis** | **Weight-bearing** (standing frame), **Calcium/Vitamin D**, **Bisphosphonates** | https://www.nscisc.uab.edu |
| **Venous Thromboembolism** | **Prophylaxis** (LMWH), **Early mobilization**, **Compression** | https://www.nscisc.uab.edu |
