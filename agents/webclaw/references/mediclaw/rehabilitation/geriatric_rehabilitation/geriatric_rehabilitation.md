# Rehabilitation Medicine - Geriatric Rehabilitation

## Geriatric Rehabilitation - Principles
| Principle | Description | URL |
|-----------|-------------|-----|
| **Comprehensive Geriatric Assessment** | **Multidimensional evaluation** (medical, Functional, Cognitive, Social) | https://www.nia.nih.gov |
| **Frailty Assessment** | **Fried criteria**, **Clinical Frailty Scale** | https://www.nia.nih.gov/health/frailty |
| **Fall Prevention** | **Multifactorial interventions** (exercise, Medication review, Vision, Home safety) | https://www.cdc.gov/steadi |
| **Interdisciplinary Team** | **Physiatrist**, **Geriatrician**, **PT/OT/SLP**, **Pharmacist**, **Social worker** | https://www.americangeriatrics.org |

## Geriatric Rehabilitation - Key Considerations
| Consideration | Management | URL |
|---------------|------------|-----|
| **Polypharmacy** | **Medication review**, **Deprescribe** when appropriate | https://deprescribing.org |
| **Cognitive Impairment** | **Simplify instructions**, **Use cues**, **Involve caregiver** | https://www.alz.org |
| **Sensory Impairment** | **Ensure glasses/hearing aids**, **Adequate lighting** | https://www.nia.nih.gov |
| **Nutrition** | **Adequate protein** (1.2-1.5 g/kg), **Calories**, **Hydration** | https://www.nia.nih.gov |
| **Skin Integrity** | **Pressure relief**, **Skin checks**, **Moisture management** | https://www.nia.nih.gov |
| **Bowel/Bladder** | **Timed voiding**, **Bowel program**, **Avoid constipation** | https://www.nia.nih.gov |
