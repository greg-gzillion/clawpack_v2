# Nuclear Medicine - Overview

## What is Nuclear Medicine?
Nuclear medicine is a medical specialty that uses radioactive tracers (radiopharmaceuticals) to diagnose and treat disease. It provides unique functional and molecular information that complements anatomic imaging modalities like CT and MRI.

| Core Concept | Description | URL |
|--------------|-------------|-----|
| **Radiopharmaceutical** | Radioactive compound (radionuclide + pharmaceutical) administered to patient | https://www.snmmi.org/About-SNMMI/Content.aspx?ItemNumber=6433 |
| **Tracer Principle** | Administering minute amounts of radioactive material to track physiologic processes | https://www.nibib.nih.gov/science-education/science-topics/nuclear-medicine |
| **Functional Imaging** | Assesses organ function and physiology rather than just anatomy | https://www.radiologyinfo.org/en/info/gennuclear |
| **Theranostics** | Combining diagnostic imaging and targeted radionuclide therapy using the same molecular target | https://www.snmmi.org/Clinical-Practice/Theranostics |

| **Iodine-123 (I-123)** | **13.2 hours** | Electron capture (gamma 159 keV) | **Thyroid imaging** (superior to I-131) | https://www.nrc.gov/materials/miau/med-use-toolkit/i-123.html |

| **Radium-223 (Ra-223)** | **11.4 days** | Alpha emission | **Bone metastases (prostate cancer)** | https://www.fda.gov/drugs/resources-information-approved-drugs/radium-ra-223-dichloride |
