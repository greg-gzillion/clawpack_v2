# Nuclear Medicine - Pulmonary Imaging

## Ventilation/Perfusion (V/Q) Scan
| Component | Description | URL |
|-----------|-------------|-----|
| **Indication** | **Pulmonary embolism** (especially with contraindication to CTPA, Renal failure, Pregnancy) | https://www.radiologyinfo.org/en/info/lung-ventilation-perfusion |
| **Perfusion Agent** | **Tc-99m MAA** (Macroaggregated Albumin) - 200,000-700,000 particles | https://www.radiologyinfo.org/en/info/lung-ventilation-perfusion |
| **Ventilation Agents** | **Xe-133 gas**, **Tc-99m DTPA aerosol**, **Technegas** (Tc-99m labeled carbon nanoparticles) | https://www.radiologyinfo.org/en/info/lung-ventilation-perfusion |

## PIOPED II (Modified) Interpretation Criteria
| Category | Description | PE Probability | URL |
|----------|-------------|---------------|-----|
| **Normal** | No perfusion defects | **<2%** | https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3037485/ |
| **Very Low Probability** | Nonsegmental defects, Defects smaller than CXR opacity | **<5%** | https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3037485/ |
| **Low Probability** | Nonsegmental defects, Perfusion defect substantially smaller than CXR opacity | **<20%** | https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3037485/ |
| **Intermediate Probability** | One segmental mismatch, Difficult to categorize | **20-80%** | https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3037485/ |
| **High Probability** | **=2 large segmental mismatches** | **>80%** | https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3037485/ |

## V/Q Scan Patterns
| Pattern | Perfusion | Ventilation | Interpretation | URL |
|---------|-----------|-------------|----------------|-----|
| **Mismatch** | **Defect** | **Normal** | **PE** (most likely) | https://www.radiologyinfo.org/en/info/lung-ventilation-perfusion |
| **Match** | **Defect** | **Defect** | **Airway disease** (COPD, Asthma), **Pneumonia** | https://www.radiologyinfo.org/en/info/lung-ventilation-perfusion |
| **Reverse Mismatch** | **Normal** | **Defect** | **Mucus plug**, **Tumor**, **Radiation** | https://www.radiologyinfo.org/en/info/lung-ventilation-perfusion |

## Quantitative Lung Perfusion
| Component | Description | URL |
|-----------|-------------|-----|
| **Indication** | **Preoperative assessment for lung resection** (predict postoperative FEV1) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Method** | **Tc-99m MAA** perfusion scan with **ROI analysis** of each lung/lobe | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Calculation** | **Predicted Postoperative FEV1 = Preoperative FEV1 ﾗ (1 - Fraction of lung to be resected)** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Threshold** | **ppoFEV1 >40%** or **ppoDLCO >40%** = Acceptable risk | https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3747617/ |
