# Nuclear Medicine - Infection and Inflammation Imaging

## Gallium-67 Scintigraphy
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **Binds to transferrin**, **Lactoferrin** (in neutrophils), **Siderophores** (bacteria) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Normal Biodistribution** | **Liver**, **Spleen**, **Bone marrow**, **Bowel** (variable), **Lacrimal/Salivary glands**, **Breasts** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Imaging Time** | **48-72 hours** post-injection | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Indications** | **Fever of unknown origin (FUO)** , **Sarcoidosis**, **Interstitial lung disease**, **Spinal infection** (when MRI contraindicated) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Panda Sign** | **Bilateral lacrimal and parotid uptake** = **Sarcoidosis** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Lambda Sign** | **Bilateral hilar and right paratracheal uptake** = **Sarcoidosis** | https://www.snmmi.org/Clinical-Practice/Guidelines |

## Labeled Leukocyte (WBC) Scintigraphy
| Component | Description | URL |
|-----------|-------------|-----|
| **Radiopharmaceuticals** | **In-111 Oxine** or **Tc-99m HMPAO** labeled autologous WBCs | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Indications** | **Prosthetic joint infection**, **Diabetic foot osteomyelitis**, **Vascular graft infection**, **Abdominal abscess** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Imaging Time** | **In-111**: 18-24h; **Tc-99m**: 4h and 24h | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Sensitivity/Specificity** | **80-90% / 80-90%** for musculoskeletal infection | https://www.snmmi.org/Clinical-Practice/Guidelines |

## WBC/Marrow Scintigraphy
| Component | Description | URL |
|-----------|-------------|-----|
| **Principle** | **Compare WBC uptake to bone marrow uptake** (Tc-99m Sulfur Colloid) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **WBC/Marrow Match** | **Normal marrow** or **Aseptic loosening** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **WBC/Marrow Mismatch** | **Infection** (WBC uptake without marrow uptake) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Indication** | **Prosthetic joint infection** (especially hip/knee) | https://www.snmmi.org/Clinical-Practice/Guidelines |
