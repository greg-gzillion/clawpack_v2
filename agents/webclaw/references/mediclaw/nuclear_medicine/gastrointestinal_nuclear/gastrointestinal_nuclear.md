# Nuclear Medicine - Gastrointestinal Imaging

## Hepatobiliary Iminodiacetic Acid (HIDA) Scan
| Component | Description | URL |
|-----------|-------------|-----|
| **Radiopharmaceutical** | **Tc-99m Mebrofenin (Choletec)** or **Tc-99m Disofenin (Hepatolite)** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Indications** | **Acute cholecystitis**, **Chronic cholecystitis**, **Biliary leak**, **Biliary atresia**, **Sphincter of Oddi dysfunction** | https://www.snmmi.org/Clinical-Practice/Guidelines |

## HIDA Scan Patterns
| Pattern | Findings | Interpretation | URL |
|---------|----------|----------------|-----|
| **Normal** | **Gallbladder visualized within 60 min**, Bowel activity | **Patent cystic duct** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Acute Cholecystitis** | **Non-visualization of gallbladder at 4 hours** (or after morphine augmentation) | **Cystic duct obstruction** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Chronic Cholecystitis** | **Delayed gallbladder visualization** (60 min - 4 hours) | **Partial cystic duct obstruction** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Biliary Leak** | **Extravasation of tracer** (peritoneal cavity, Drain) | **Postoperative leak**, **Trauma** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Biliary Atresia** | **Good hepatic uptake but no bowel activity at 24h** | **Neonatal cholestasis** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Sphincter of Oddi Dysfunction** | **Delayed biliary-to-bowel transit** (>60 min) | **Functional obstruction** | https://www.snmmi.org/Clinical-Practice/Guidelines |

## HIDA with CCK (Cholecystokinin)
| Component | Description | URL |
|-----------|-------------|-----|
| **Indication** | **Calculate gallbladder ejection fraction (GBEF)** for **functional gallbladder disorder** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Method** | **Infuse CCK** over 30-60 min after gallbladder filling | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Normal GBEF** | **>35-38%** (varies by protocol) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Abnormal GBEF** | **<35%** suggests **biliary dyskinesia** (may benefit from cholecystectomy) | https://www.snmmi.org/Clinical-Practice/Guidelines |

## Gastric Emptying Study
| Component | Description | URL |
|-----------|-------------|-----|
| **Radiopharmaceutical** | **Tc-99m Sulfur Colloid** in **standardized meal** (Egg Beaters, Toast, Jam, Water) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Indications** | **Gastroparesis**, **Dumping syndrome**, **Functional dyspepsia** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Protocol** | **Imaging at 0, 1, 2, 4 hours** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Normal Values** | **<10% retention at 4 hours** (or <60% at 2h, <30% at 4h) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Gastroparesis** | **>10% retention at 4 hours** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Dumping Syndrome** | **Rapid emptying** (>30% at 1h) | https://www.snmmi.org/Clinical-Practice/Guidelines |

## GI Bleeding Scintigraphy
| Component | Description | URL |
|-----------|-------------|-----|
| **Radiopharmaceutical** | **Tc-99m labeled RBCs** (in vivo or in vitro labeling) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Indication** | **Localize active GI bleeding** (especially obscure or intermittent) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Sensitivity** | **0.1 mL/min** (more sensitive than angiography) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Localization** | **SPECT/CT improves anatomic localization** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Positive Study** | **Focal activity that moves with bowel peristalsis** | https://www.snmmi.org/Clinical-Practice/Guidelines |

## Meckel Diverticulum Scan
| Component | Description | URL |
|-----------|-------------|-----|
| **Radiopharmaceutical** | **Tc-99m Pertechnetate** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Indication** | **Suspected Meckel diverticulum** (painless rectal bleeding in child) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Mechanism** | **Uptake by ectopic gastric mucosa** in diverticulum | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Preparation** | **H2 blocker or PPI** (to enhance uptake), **Foley catheter** (avoid bladder activity) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Positive Study** | **Focal activity in right lower quadrant** that appears **simultaneously with gastric mucosa** | https://www.snmmi.org/Clinical-Practice/Guidelines |

## Liver-Spleen Scintigraphy
| Component | Description | URL |
|-----------|-------------|-----|
| **Radiopharmaceutical** | **Tc-99m Sulfur Colloid** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Indications** | **Focal nodular hyperplasia** (FNH), **Splenic tissue** (accessory spleen, Splenosis), **Diffuse liver disease** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **FNH** | **Normal or increased uptake** (contains Kupffer cells) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Adenoma/HCC/Metastasis** | **Photopenic** (lack Kupffer cells) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Splenosis** | **Focal uptake in ectopic splenic tissue** | https://www.snmmi.org/Clinical-Practice/Guidelines |
