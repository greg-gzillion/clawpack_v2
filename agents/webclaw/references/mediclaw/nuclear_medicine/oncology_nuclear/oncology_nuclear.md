# Nuclear Medicine - Oncology Imaging

## F-18 FDG PET/CT
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **Glucose analog** taken up by GLUT transporters, Phosphorylated by hexokinase, **Trapped intracellularly** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Patient Preparation** | **Fasting 4-6 hours**, **Blood glucose <200 mg/dL** (preferably <150) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Normal Biodistribution** | **Brain**, **Myocardium** (variable), **Liver**, **Spleen**, **GI tract** (variable), **Kidneys/Bladder** (excretion), **Brown fat** (neck/supraclavicular), **Bone marrow** (mild) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Deauville Criteria (Lymphoma)** | **1**: = Background; **2**: = Mediastinum; **3**: > Mediastinum but = Liver; **4**: Moderately > Liver; **5**: Markedly > Liver and/or new lesions | https://www.nccn.org/guidelines/guidelines-detail?category=1&id=1480 |
| **PERCIST Criteria** | **PET Response Criteria in Solid Tumors** (quantitative response assessment) | https://www.ncbi.nlm.nih.gov/pmc/articles/PMC2743118/ |

## False Positives on FDG PET
| Finding | Mechanism | URL |
|---------|-----------|-----|
| **Infection/Inflammation** | **Activated inflammatory cells** (macrophages, neutrophils) have ? GLUT expression | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Brown Fat** | **Cold-induced sympathetic activation** ? ? FDG uptake in supraclavicular/paraspinal fat | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Postoperative Changes** | **Inflammatory response** (up to 6-12 months) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Post-Radiation Changes** | **Inflammation** (up to 3-6 months, may persist longer) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Thymic Rebound** | **Post-chemotherapy thymic hyperplasia** (children/young adults) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Muscle Activity** | **Recent exercise**, **Tension**, **Hyperinsulinemia** | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Bone Marrow** | **G-CSF therapy**, **Anemia**, **Infection** | https://www.snmmi.org/Clinical-Practice/Guidelines |

## False Negatives on FDG PET
| Finding | Mechanism | URL |
|---------|-----------|-----|
| **Hyperglycemia** | **Competitive inhibition** of FDG uptake | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Small Lesions** | **Below spatial resolution** (<6-8 mm) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Low-Grade Tumors** | **Low metabolic activity** (carcinoid, MALT lymphoma, Low-grade sarcoma) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Mucinous Tumors** | **Hypocellular**, **Abundant mucin** (mucinous adenocarcinoma) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Renal Cell Carcinoma** | **Variable FDG uptake** (often low) | https://www.snmmi.org/Clinical-Practice/Guidelines |
| **Hepatocellular Carcinoma** | **Variable FDG uptake** (well-differentiated often low) | https://www.snmmi.org/Clinical-Practice/Guidelines |
