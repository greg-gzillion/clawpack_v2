# Nuclear Medicine - Instrumentation

## Gamma Camera Components
| Component | Function | URL |
|-----------|----------|-----|
| **Collimator** | **Absorbs scattered gamma rays**, Allows only perpendicular rays to reach crystal | https://www.nibib.nih.gov/science-education/science-topics/nuclear-medicine |
| **Scintillation Crystal** | **Converts gamma rays to visible light** (NaI(Tl) most common) | https://www.nibib.nih.gov/science-education/science-topics/nuclear-medicine |
| **Photomultiplier Tubes (PMTs)** | **Convert light to electrical signal**, Amplify signal | https://www.nibib.nih.gov/science-education/science-topics/nuclear-medicine |
| **Pulse Height Analyzer (PHA)** | **Selects photopeak**, Rejects scatter | https://www.nibib.nih.gov/science-education/science-topics/nuclear-medicine |
| **Positioning Circuitry** | **Determines location** of each scintillation event | https://www.nibib.nih.gov/science-education/science-topics/nuclear-medicine |

## PET (Positron Emission Tomography)
| Component | Description | URL |
|-----------|-------------|-----|
| **Principle** | **Positron emission ? Annihilation ? Two 511 keV photons at 180ｰ** | https://www.radiologyinfo.org/en/info/pet |

## Quality Control Tests
| Test | Frequency | Purpose | URL |
|------|-----------|---------|-----|
