## Constraint-Induced Movement Therapy (CIMT)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Restraint of unaffected arm** to **force use of affected arm** | https://www.stroke.org/en/life-after-stroke/recovery/physical-recovery |
| **Indications** | **Some active wrist/finger extension**, **Minimal cognitive impairment**, **Good endurance** | https://www.stroke.org/en/life-after-stroke/recovery/physical-recovery |
| **Protocol** | **Restraint 90% of waking hours** + **6h/day shaping** x2-3 weeks | https://www.stroke.org/en/life-after-stroke/recovery/physical-recovery |
| **Modified CIMT** | **Less intensive** (2-3h/day) - more feasible clinically | https://www.stroke.org/en/life-after-stroke/recovery/physical-recovery |
