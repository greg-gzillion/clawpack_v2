# Physical Medicine - Physiatry Principles

## The ICF Model (International Classification of Functioning, Disability and Health)
| Component | Description | Example | URL |
|-----------|-------------|---------|-----|
| **Body Functions and Structures** | **Physiologic and anatomic** aspects | **Muscle weakness**, **Joint contracture** | https://www.who.int/standards/classifications/international-classification-of-functioning-disability-and-health |
| **Activity** | **Execution of tasks or actions** | **Difficulty walking**, **Difficulty dressing** | https://www.who.int/standards/classifications/international-classification-of-functioning-disability-and-health |
| **Participation** | **Involvement in life situations** | **Unable to work**, **Social isolation** | https://www.who.int/standards/classifications/international-classification-of-functioning-disability-and-health |
| **Contextual Factors** | **Environmental and personal factors** | **Accessibility of home**, **Social support**, **Coping style** | https://www.who.int/standards/classifications/international-classification-of-functioning-disability-and-health |

## Functional Assessment Tools
| Domain | Tool | Description | URL |
|--------|------|-------------|-----|
| **Global Function** | **Functional Independence Measure (FIM)** | **18 items** (13 motor, 5 cognitive), Score 18-126 | https://www.udsmr.org/WebFIM |
| **Global Function** | **Barthel Index** | **10 items** (ADLs, mobility), Score 0-100 | https://www.mdcalc.com/calc/10454/barthel-index-activities-daily-living |
| **Cognition** | **Mini-Mental State Exam (MMSE)** | **30-point** cognitive screen | https://www.parinc.com/Products/Pkey/237 |
| **Cognition** | **Montreal Cognitive Assessment (MoCA)** | **30-point** cognitive screen (more sensitive for executive function) | https://www.mocatest.org |
| **Mobility** | **Timed Up and Go (TUG)** | **Time to rise, walk 3m, return, sit** (=12 sec = fall risk) | https://www.cdc.gov/steadi |
| **Mobility** | **6-Minute Walk Test** | **Distance walked in 6 min** (functional capacity) | https://www.atsjournals.org/doi/full/10.1164/ajrccm.166.1.at1102 |
| **Balance** | **Berg Balance Scale** | **14 items**, Score 0-56 (=45 = fall risk) | https://www.sralab.org/rehabilitation-measures/berg-balance-scale |

## Discharge Planning
| Disposition | Description | URL |
|-------------|-------------|-----|
| **Home** | **Independent** or with **family support**, **Home health services** | https://www.cms.gov/medicare-coverage-database |
| **Home with Outpatient Therapy** | **Continued therapy** in outpatient setting | https://www.cms.gov/medicare-coverage-database |
| **Home Health** | **Skilled nursing**, **PT/OT/SLP** in home (homebound required) | https://www.cms.gov/medicare-coverage-database |
| **Skilled Nursing Facility (SNF)** | **Subacute rehabilitation** (=3h/day therapy, 5-7 days/week) | https://www.cms.gov/medicare-coverage-database |
| **Inpatient Rehabilitation Facility (IRF)** | **Acute rehabilitation** (=3h/day therapy, 5-7 days/week, Physician oversight) | https://www.cms.gov/medicare-coverage-database |
| **Long-Term Acute Care (LTACH)** | **Medically complex**, **Prolonged ventilator weaning** | https://www.cms.gov/medicare-coverage-database |

## IRF Admission Criteria (Medicare 60% Rule)
| Criterion | Description | URL |
|-----------|-------------|-----|
| **60% Rule** | **=60% of patients** must have **1 of 13 qualifying conditions** (stroke, SCI, TBI, Amputation, Hip fracture, etc.) | https://www.cms.gov/medicare-coverage-database |
| **Medical Necessity** | **Require intensive rehabilitation** (=3h/day therapy) | https://www.cms.gov/medicare-coverage-database |
| **Reasonable Expectation** | **Significant functional improvement** expected | https://www.cms.gov/medicare-coverage-database |
| **Medical Supervision** | **24/7 rehabilitation nursing + Daily physician oversight** | https://www.cms.gov/medicare-coverage-database |
