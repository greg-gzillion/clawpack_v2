# Neurology - Neuromuscular Disorders

## Myasthenia Gravis (MG)
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathophysiology** | **Anti-AChR antibodies** (85%), **Anti-MuSK antibodies** (5-10%), **Anti-LRP4 antibodies** ? **Impaired neuromuscular transmission** | https://www.ninds.nih.gov/health-information/disorders/myasthenia-gravis |
| **Clinical Features** | **Fatigable weakness** (worse with repetition, Improves with rest), **Ptosis**, **Diplopia**, **Dysphagia**, **Dysarthria**, **Proximal limb weakness** | https://www.ninds.nih.gov/health-information/disorders/myasthenia-gravis |
| **Diagnosis** | **Ice pack test** (improvement in ptosis), **Edrophonium test** (historical), **Repetitive nerve stimulation** (decremental response), **Single-fiber EMG** (increased jitter), **Anti-AChR/MuSK antibodies** | https://www.ninds.nih.gov/health-information/disorders/myasthenia-gravis |
| **Treatment (Symptomatic)** | **Pyridostigmine** (acetylcholinesterase inhibitor) | https://www.ninds.nih.gov/health-information/disorders/myasthenia-gravis |
| **Treatment (Immunomodulatory)** | **Corticosteroids**, **Mycophenolate**, **Azathioprine**, **Rituximab** (MuSK+), **Eculizumab** (refractory AChR+), **Thymectomy** (AChR+, non-thymomatous) | https://www.ninds.nih.gov/health-information/disorders/myasthenia-gravis |

## Myasthenic Crisis
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Respiratory failure** requiring mechanical ventilation | https://www.ninds.nih.gov/health-information/disorders/myasthenia-gravis |
| **Triggers** | **Infection**, **Surgery**, **Medications** (aminoglycosides, Fluoroquinolones, Beta-blockers, Magnesium), **Pregnancy**, **Tapering immunosuppression** | https://www.ninds.nih.gov/health-information/disorders/myasthenia-gravis |
| **Management** | **Plasmapheresis (PLEX)** or **IVIG**, **Hold pyridostigmine** (initially), **Mechanical ventilation**, **Treat triggers** | https://www.ninds.nih.gov/health-information/disorders/myasthenia-gravis |

## Lambert-Eaton Myasthenic Syndrome (LEMS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathophysiology** | **Anti-VGCC antibodies** ? **Impaired presynaptic ACh release** | https://www.ninds.nih.gov/health-information/disorders/lambert-eaton-myasthenic-syndrome |
| **Association** | **Small cell lung cancer** (50-60%) | https://www.ninds.nih.gov/health-information/disorders/lambert-eaton-myasthenic-syndrome |
| **Clinical Features** | **Proximal weakness** (legs > arms), **Hyporeflexia** (improves with exercise), **Autonomic dysfunction** (dry mouth, Constipation, Erectile dysfunction), **Ptosis** (less prominent than MG) | https://www.ninds.nih.gov/health-information/disorders/lambert-eaton-myasthenic-syndrome |
| **Diagnosis** | **Repetitive nerve stimulation** (**incremental** response at high frequency), **Anti-VGCC antibodies** | https://www.ninds.nih.gov/health-information/disorders/lambert-eaton-myasthenic-syndrome |
| **Treatment** | **3,4-Diaminopyridine (3,4-DAP)** (Amifampridine), **Pyridostigmine**, **Immunosuppression**, **Treat underlying cancer** | https://www.ninds.nih.gov/health-information/disorders/lambert-eaton-myasthenic-syndrome |

## Inflammatory Myopathies
| Disease | Clinical Features | Antibodies | Treatment | URL |
|---------|-------------------|------------|-----------|-----|
| **Dermatomyositis (DM)** | **Proximal weakness**, **Heliotrope rash**, **Gottron papules**, **Shawl sign**, **V-sign** | **Anti-Mi-2**, Anti-TIF1?, Anti-MDA5, Anti-NXP2 | **Corticosteroids**, **IVIG**, Mycophenolate, Azathioprine, Rituximab | https://www.ninds.nih.gov/health-information/disorders/inflammatory-myopathies |
| **Polymyositis (PM)** | **Proximal weakness** without rash | **Anti-Jo-1** (antisynthetase), Anti-SRP | Same as DM | https://www.ninds.nih.gov/health-information/disorders/inflammatory-myopathies |
| **Antisynthetase Syndrome** | Myositis, **ILD**, **Arthritis**, **Raynaud**, **Mechanic's hands** | **Anti-Jo-1**, Anti-PL-7, Anti-PL-12 | Same as DM | https://www.ninds.nih.gov/health-information/disorders/inflammatory-myopathies |
| **Immune-Mediated Necrotizing Myopathy (IMNM)** | **Severe weakness**, **Very high CK** (>5000) | **Anti-SRP**, **Anti-HMGCR** (statin-associated) | **Corticosteroids**, **IVIG**, Rituximab, **Stop statin** | https://www.ninds.nih.gov/health-information/disorders/inflammatory-myopathies |
| **Inclusion Body Myositis (IBM)** | **Asymmetric weakness** (finger flexors, Quadriceps), **Dysphagia**, **Refractory to immunosuppression** | Anti-cN1A | **Supportive**; No proven therapy | https://www.ninds.nih.gov/health-information/disorders/inclusion-body-myositis |

## Muscular Dystrophies
| Disease | Genetics | Onset | Clinical Features | URL |
|---------|----------|-------|-------------------|-----|
| **Duchenne Muscular Dystrophy (DMD)** | **X-linked recessive**, **DMD gene** (dystrophin) | Early childhood | **Progressive proximal weakness**, **Gowers sign**, **Calf pseudohypertrophy**, **Cardiomyopathy**, Loss of ambulation by age 12 | https://www.ninds.nih.gov/health-information/disorders/muscular-dystrophy |
| **Becker Muscular Dystrophy (BMD)** | **X-linked recessive**, **DMD gene** (partially functional dystrophin) | Adolescence/adulthood | **Milder**, Slower progression than DMD | https://www.ninds.nih.gov/health-information/disorders/muscular-dystrophy |
| **Myotonic Dystrophy Type 1 (DM1)** | **Autosomal dominant**, **DMPK gene** (CTG repeat) | Variable | **Myotonia**, **Distal weakness**, **Cataracts**, **Cardiac conduction defects**, **Frontal balding**, **Insulin resistance** | https://www.ninds.nih.gov/health-information/disorders/myotonic-dystrophy |
| **Facioscapulohumeral Dystrophy (FSHD)** | **Autosomal dominant**, **DUX4** | Adolescence/adulthood | **Facial weakness**, **Scapular winging**, **Humeral weakness**, Asymmetric | https://www.ninds.nih.gov/health-information/disorders/facioscapulohumeral-muscular-dystrophy |
| **Limb-Girdle Muscular Dystrophy (LGMD)** | Multiple genes | Variable | **Proximal weakness** (shoulder and hip girdle) | https://www.ninds.nih.gov/health-information/disorders/limb-girdle-muscular-dystrophy |
