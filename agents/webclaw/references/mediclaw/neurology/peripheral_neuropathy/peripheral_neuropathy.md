# Neurology - Peripheral Neuropathy

## Peripheral Neuropathy - Classification
| Type | Affected Fibers | Clinical Features | URL |
|------|-----------------|-------------------|-----|
| **Distal Symmetric Polyneuropathy** | **Length-dependent** | **Stocking-glove** sensory loss, Distal weakness | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Mononeuropathy** | Single nerve | Focal sensory/motor deficit in nerve distribution | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Mononeuritis Multiplex** | Multiple non-contiguous nerves | **Asymmetric**, Stepwise involvement; **Vasculitis** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Radiculopathy** | Nerve root | Dermatomal sensory loss, Myotomal weakness | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Plexopathy** | Brachial or lumbosacral plexus | Regional sensorimotor deficits | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Neuronopathy** | Dorsal root ganglion or anterior horn cell | **Sensory ataxia** (sensory neuronopathy) or **Pure motor** (motor neuronopathy) | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |

## Peripheral Neuropathy - Etiologies
| Category | Causes | URL |
|----------|--------|-----|
| **Metabolic/Endocrine** | **Diabetes** (most common), **Hypothyroidism**, **Uremia**, **Vitamin B12 deficiency** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Toxic** | **Alcohol**, **Chemotherapy** (platinum, Taxanes, Vincristine), **Heavy metals** (lead, Arsenic, Mercury) | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Infectious** | **HIV**, **Leprosy**, **Lyme disease**, **VZV**, **Hepatitis C** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Inflammatory** | **Guillain-Barré syndrome**, **CIDP**, **Vasculitis**, **Sarcoidosis** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Hereditary** | **Charcot-Marie-Tooth (CMT)** , **Hereditary sensory and autonomic neuropathy (HSAN)** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Paraproteinemic** | **Monoclonal gammopathy** (MGUS, Multiple myeloma, Waldenström), **Amyloidosis** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Idiopathic** | Up to 30% despite extensive evaluation | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |

| Component | Description | URL |
|-----------|-------------|-----|
| **Epidemiology** | **Most common cause of peripheral neuropathy**; Affects 50% of diabetics | https://www.niddk.nih.gov/health-information/diabetes/overview/preventing-problems/nerve-damage-diabetic-neuropathies |
| **Types** | **Distal symmetric polyneuropathy** (most common), **Autonomic neuropathy**, **Mononeuropathy** (cranial III, VI, VII), **Radiculoplexopathy** (diabetic amyotrophy) | https://www.niddk.nih.gov/health-information/diabetes/overview/preventing-problems/nerve-damage-diabetic-neuropathies |
| **Pathophysiology** | **Hyperglycemia** ? **Polyol pathway**, **AGE formation**, **Oxidative stress**, **Microvascular disease** | https://www.niddk.nih.gov/health-information/diabetes/overview/preventing-problems/nerve-damage-diabetic-neuropathies |
| **Treatment** | **Glycemic control** (most important), **Pain management**: **Gabapentinoids**, **SNRIs** (duloxetine), **TCAs**, **Topical capsaicin/lidocaine** | https://www.niddk.nih.gov/health-information/diabetes/overview/preventing-problems/nerve-damage-diabetic-neuropathies |

## Guillain-Barré Syndrome (GBS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathophysiology** | **Acute immune-mediated demyelinating polyradiculoneuropathy**; Often post-infectious (**Campylobacter jejuni**, CMV, EBV, Zika) | https://www.ninds.nih.gov/health-information/disorders/guillain-barre-syndrome |
| **Clinical Features** | **Ascending symmetric weakness**, **Areflexia**, **Paresthesias**, **Autonomic dysfunction** (tachycardia, BP lability), **Respiratory failure** (20-30%) | https://www.ninds.nih.gov/health-information/disorders/guillain-barre-syndrome |
| **Variants** | **AIDP** (most common - 90%), **AMAN** (pure motor), **AMSAN** (motor + sensory), **Miller Fisher syndrome** (ophthalmoplegia, Ataxia, Areflexia) | https://www.ninds.nih.gov/health-information/disorders/guillain-barre-syndrome |
| **Diagnosis** | **CSF**: **Albuminocytologic dissociation** (elevated protein, Normal WBC), **EMG/NCS**: Demyelinating features | https://www.ninds.nih.gov/health-information/disorders/guillain-barre-syndrome |
| **Treatment** | **IVIG** (0.4 g/kg/day x 5 days) OR **Plasmapheresis**, **Supportive care**, **DVT prophylaxis**, **Monitor respiratory function** (FVC, NIF) | https://www.ninds.nih.gov/health-information/disorders/guillain-barre-syndrome |

## Chronic Inflammatory Demyelinating Polyneuropathy (CIDP)
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathophysiology** | **Chronic immune-mediated demyelinating polyneuropathy** | https://www.ninds.nih.gov/health-information/disorders/cidp |
| **Clinical Features** | **Progressive or relapsing symmetric proximal and distal weakness**, **Sensory loss**, **Areflexia**, Duration >8 weeks | https://www.ninds.nih.gov/health-information/disorders/cidp |
| **Diagnosis** | **CSF**: Albuminocytologic dissociation, **EMG/NCS**: Demyelinating features (conduction block, Temporal dispersion, Prolonged latencies) | https://www.ninds.nih.gov/health-information/disorders/cidp |
| **Treatment** | **IVIG**, **Plasmapheresis**, **Corticosteroids**, **Immunosuppressants** (mycophenolate, Azathioprine, Rituximab) | https://www.ninds.nih.gov/health-information/disorders/cidp |

## Charcot-Marie-Tooth (CMT) Disease
| Component | Description | URL |
|-----------|-------------|-----|
| **Epidemiology** | **Most common inherited neuropathy** (1 in 2,500) | https://www.ninds.nih.gov/health-information/disorders/charcot-marie-tooth-disease |
| **Genetics** | **CMT1A** (PMP22 duplication - most common), CMT1B (MPZ), CMTX1 (GJB1), CMT2 (multiple genes) | https://www.ninds.nih.gov/health-information/disorders/charcot-marie-tooth-disease |
| **Clinical Features** | **Distal weakness** (peroneal - foot drop, Steppage gait), **Distal atrophy** ("stork legs", "inverted champagne bottle"), **Pes cavus**, **Hammer toes**, **Hyporeflexia** | https://www.ninds.nih.gov/health-information/disorders/charcot-marie-tooth-disease |
| **Diagnosis** | **EMG/NCS**, **Genetic testing** | https://www.ninds.nih.gov/health-information/disorders/charcot-marie-tooth-disease |
| **Treatment** | **Supportive** (PT/OT, Orthotics, Bracing, Surgery for deformities) | https://www.ninds.nih.gov/health-information/disorders/charcot-marie-tooth-disease |
