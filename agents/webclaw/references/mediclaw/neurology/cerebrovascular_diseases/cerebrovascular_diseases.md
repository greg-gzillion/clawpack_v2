## Intracerebral Hemorrhage (ICH)
| Component | Description | URL |
|-----------|-------------|-----|
| **Etiology** | **Hypertension** (most common), **Cerebral amyloid angiopathy**, **Vascular malformation**, **Coagulopathy**, **Tumor** | https://www.stroke.org/en/about-stroke/types-of-stroke/hemorrhagic-stroke |
| **Locations** | **Deep** (basal ganglia, Thalamus, Pons, Cerebellum) - hypertensive; **Lobar** - amyloid angiopathy | https://www.stroke.org/en/about-stroke/types-of-stroke/hemorrhagic-stroke |
| **Imaging** | **Non-contrast CT** (hyperdense), **CTA** (spot sign), **MRI** (T2*/SWI for microbleeds) | https://www.stroke.org/en/about-stroke/types-of-stroke/hemorrhagic-stroke |
| **Blood Pressure Management** | **Acute**: SBP 140-160 mmHg (avoid <140); **Long-term**: <130/80 mmHg | https://www.stroke.org/en/professionals/stroke-resource-library/guidelines-and-quality |
| **Coagulopathy Reversal** | **Warfarin**: 4F-PCC + Vitamin K; **DOACs**: Andexanet alfa (Xa), Idarucizumab (dabigatran); **Antiplatelet**: No routine platelet transfusion | https://www.stroke.org/en/professionals/stroke-resource-library/guidelines-and-quality |
| **Surgical Management** | **Cerebellar ICH >3 cm** or **brainstem compression/hydrocephalus** ? **Evacuation** | https://www.stroke.org/en/professionals/stroke-resource-library/guidelines-and-quality |

## Transient Ischemic Attack (TIA)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Transient focal neurologic deficit **without infarction** | https://www.stroke.org/en/about-stroke/types-of-stroke/tia-transient-ischemic-attack |
| **ABCD2 Score** | Age =60 (1), BP =140/90 (1), Clinical: Unilateral weakness (2) or Speech impairment (1), Duration =60 min (2) or 10-59 min (1), Diabetes (1) | https://www.stroke.org/en/about-stroke/types-of-stroke/tia-transient-ischemic-attack |
| **Risk Stratification** | **ABCD2 =4** or **ABCD2 0-3 + high-risk features** ? **Admit** | https://www.stroke.org/en/about-stroke/types-of-stroke/tia-transient-ischemic-attack |
| **Evaluation** | **Brain MRI** (DWI), **Vascular imaging** (CTA/MRA carotid and intracranial), **Cardiac** (ECG, Telemetry, Echo), **Labs** | https://www.stroke.org/en/about-stroke/types-of-stroke/tia-transient-ischemic-attack |
| **Management** | **Dual antiplatelet** (aspirin + clopidogrel) x 21 days then single agent, **High-intensity statin**, **Risk factor control** | https://www.stroke.org/en/about-stroke/types-of-stroke/tia-transient-ischemic-attack |
