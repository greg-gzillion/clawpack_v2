|-----------|----------------------------------|------------------|-----|

## Benign Paroxysmal Positional Vertigo (BPPV)
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathophysiology** | **Otoconia** displaced into **semicircular canals** (posterior > horizontal > anterior) | https://www.ninds.nih.gov/health-information/disorders/benign-paroxysmal-positional-vertigo |
| **Clinical Features** | **Brief (<1 min) episodes of vertigo** triggered by **head position changes** (rolling over, Looking up) | https://www.ninds.nih.gov/health-information/disorders/benign-paroxysmal-positional-vertigo |
| **Diagnosis** | **Dix-Hallpike maneuver** (posterior canal): Upbeating torsional nystagmus, Latency, Fatigability | https://www.ninds.nih.gov/health-information/disorders/benign-paroxysmal-positional-vertigo |
| **Treatment** | **Canalith repositioning maneuvers**: **Epley** (posterior), **Barbecue roll** (horizontal), **Semont** | https://www.ninds.nih.gov/health-information/disorders/benign-paroxysmal-positional-vertigo |

## Vestibular Neuritis
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathophysiology** | **Viral inflammation** of vestibular nerve (often HSV-1) | https://www.ninds.nih.gov/health-information/disorders/vestibular-neuritis |
| **Clinical Features** | **Acute onset severe vertigo** (days), **Nausea/Vomiting**, **Imbalance**, **No hearing loss** | https://www.ninds.nih.gov/health-information/disorders/vestibular-neuritis |
| **Diagnosis** | **HINTS**: Abnormal head impulse, Unidirectional nystagmus, No skew; **Normal hearing** | https://www.ninds.nih.gov/health-information/disorders/vestibular-neuritis |
| **Treatment (Acute)** | **Corticosteroids** (methylprednisolone), **Vestibular suppressants** (meclizine, Diazepam, Ondansetron - short-term only) | https://www.ninds.nih.gov/health-information/disorders/vestibular-neuritis |
| **Treatment (Recovery)** | **Vestibular rehabilitation** (early mobilization, Gaze stabilization exercises) | https://www.ninds.nih.gov/health-information/disorders/vestibular-neuritis |

## Méničre Disease
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathophysiology** | **Endolymphatic hydrops** (excess endolymph in inner ear) | https://www.ninds.nih.gov/health-information/disorders/meniere-disease |
| **Clinical Features** | **Episodic vertigo** (hours), **Fluctuating hearing loss**, **Tinnitus**, **Aural fullness** | https://www.ninds.nih.gov/health-information/disorders/meniere-disease |
| **Diagnosis** | **Audiometry** (low-frequency sensorineural hearing loss), **Clinical criteria** | https://www.ninds.nih.gov/health-information/disorders/meniere-disease |
| **Treatment (Acute)** | **Vestibular suppressants** (meclizine, Diazepam), **Antiemetics** | https://www.ninds.nih.gov/health-information/disorders/meniere-disease |
| **Treatment (Prophylaxis)** | **Low-salt diet**, **Diuretics** (HCTZ), **Betahistine** (not FDA-approved), **Intratympanic steroids** or **gentamicin**, **Surgery** (refractory) | https://www.ninds.nih.gov/health-information/disorders/meniere-disease |
