## Obstructive Sleep Apnea (OSA)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Repetitive upper airway collapse** during sleep ? **Apneas/Hypopneas** ? **Intermittent hypoxia**, **Sleep fragmentation** | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **Risk Factors** | **Obesity**, **Male sex**, **Age >50**, **Craniofacial abnormalities**, **Large neck circumference** (>17" men, >16" women) | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **Symptoms** | **Loud snoring**, **Witnessed apneas**, **Excessive daytime sleepiness**, **Morning headaches**, **Non-restorative sleep** | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **Screening** | **STOP-BANG**: Snoring, Tired, Observed apnea, Pressure (HTN), BMI >35, Age >50, Neck >40 cm, Gender (male) | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **Diagnosis** | **Polysomnography (PSG)** or **Home sleep apnea test (HSAT)** ; **AHI =5** (mild 5-14, Moderate 15-29, Severe =30) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Treatment** | **CPAP** (first-line), **Oral appliance**, **Positional therapy**, **Weight loss**, **Upper airway surgery** (UPPP, MMA), **Hypoglossal nerve stimulation** (Inspire) | https://www.nhlbi.nih.gov/health/sleep-apnea/treatment |

## Narcolepsy
| Type | Features | Diagnosis | URL |
|------|----------|-----------|-----|
| **Type 1 (with Cataplexy)** | **Excessive daytime sleepiness** + **Cataplexy** (sudden loss of muscle tone with emotion) + **Low/absent CSF hypocretin-1** | **PSG + MSLT**: Mean sleep latency <8 min + =2 SOREMPs | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Type 2 (without Cataplexy)** | Excessive daytime sleepiness, Normal CSF hypocretin-1 | Same PSG/MSLT criteria | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Other Features** | **Sleep paralysis**, **Hypnagogic/Hypnopompic hallucinations**, **Fragmented nighttime sleep** | | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Treatment (Sleepiness)** | **Modafinil**, **Armodafinil**, **Solriamfetol**, **Pitolisant**, **Traditional stimulants** (methylphenidate, Amphetamines) | | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Treatment (Cataplexy)** | **Sodium oxybate**, **Pitolisant**, **Venlafaxine**, **Fluoxetine**, **Clomipramine** | | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |

## REM Sleep Behavior Disorder (RBD)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Loss of normal REM atonia** ? **Dream enactment behavior** (talking, Shouting, Flailing, Punching, Kicking) | https://www.ninds.nih.gov/health-information/disorders/rem-sleep-behavior-disorder |
| **Associations** | **Synucleinopathies**: **Parkinson disease**, **Dementia with Lewy bodies**, **Multiple system atrophy** (80%+ develop within 10-15 years) | https://www.ninds.nih.gov/health-information/disorders/rem-sleep-behavior-disorder |
| **Diagnosis** | **Polysomnography** with **REM without atonia** | https://www.ninds.nih.gov/health-information/disorders/rem-sleep-behavior-disorder |
| **Treatment** | **Melatonin** (high-dose, 3-15 mg), **Clonazepam** (caution in elderly, OSA, Cognitive impairment), **Environmental safety** (padding, Remove weapons) | https://www.ninds.nih.gov/health-information/disorders/rem-sleep-behavior-disorder |

## Restless Legs Syndrome (RLS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Urge to move legs**, Worse at rest/in evening, Relieved by movement | https://www.ninds.nih.gov/health-information/disorders/restless-legs-syndrome |
| **Associations** | **Iron deficiency** (low ferritin), **Pregnancy**, **CKD**, **Peripheral neuropathy**, **Medications** (antidepressants, Antihistamines, Dopamine antagonists) | https://www.ninds.nih.gov/health-information/disorders/restless-legs-syndrome |
| **Diagnosis** | **Clinical** (IRLSSG criteria); **Ferritin** (goal >75 ng/mL) | https://www.ninds.nih.gov/health-information/disorders/restless-legs-syndrome |
| **Treatment** | **Iron supplementation** (if ferritin <75), **Gabapentinoids** (gabapentin enacarbil, Pregabalin), **Dopamine agonists** (pramipexole, Ropinirole - caution with augmentation), **Opioids** (refractory) | https://www.ninds.nih.gov/health-information/disorders/restless-legs-syndrome |
