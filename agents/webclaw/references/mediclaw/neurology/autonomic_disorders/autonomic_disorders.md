## Orthostatic Hypotension (OH)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **SBP decrease =20 mmHg** or **DBP decrease =10 mmHg** within 3 minutes of standing | https://www.ninds.nih.gov/health-information/disorders/orthostatic-hypotension |
| **Neurogenic OH** | **Autonomic failure** (Parkinson disease, MSA, Pure autonomic failure, Diabetes, Amyloidosis) | https://www.ninds.nih.gov/health-information/disorders/orthostatic-hypotension |
| **Non-Neurogenic OH** | **Hypovolemia**, **Medications** (antihypertensives, Diuretics, Antidepressants, Alpha-blockers), **Deconditioning** | https://www.ninds.nih.gov/health-information/disorders/orthostatic-hypotension |
| **Symptoms** | **Dizziness**, **Lightheadedness**, **Vision changes**, **Weakness**, **Syncope**, **Coat-hanger pain** (neck/shoulders) | https://www.ninds.nih.gov/health-information/disorders/orthostatic-hypotension |
| **Evaluation** | **Orthostatic vitals**, **Tilt table test**, **Autonomic reflex screen** (Valsalva, Deep breathing, QSART) | https://www.ninds.nih.gov/health-information/disorders/orthostatic-hypotension |
| **Treatment** | **Non-pharmacologic**: Hydration, Salt, Compression stockings, Slow positional changes, Elevate head of bed; **Pharmacologic**: **Midodrine**, **Fludrocortisone**, **Droxidopa** | https://www.ninds.nih.gov/health-information/disorders/orthostatic-hypotension |

## Postural Orthostatic Tachycardia Syndrome (POTS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **HR increase =30 bpm** (=40 bpm in adolescents) within 10 minutes of standing, **Without OH**, Associated with symptoms of orthostatic intolerance | https://www.ninds.nih.gov/health-information/disorders/postural-tachycardia-syndrome |
| **Epidemiology** | **Young women** (5:1), Often post-viral, Post-concussion, Post-pregnancy | https://www.ninds.nih.gov/health-information/disorders/postural-tachycardia-syndrome |
| **Symptoms** | **Palpitations**, **Lightheadedness**, **Fatigue**, **Brain fog**, **Nausea**, **Exercise intolerance**, **Heat intolerance** | https://www.ninds.nih.gov/health-information/disorders/postural-tachycardia-syndrome |
| **Treatment** | **Non-pharmacologic**: Hydration, Salt, Compression stockings, **Exercise** (reconditioning); **Pharmacologic**: Beta-blockers, Ivabradine, Midodrine, Fludrocortisone | https://www.ninds.nih.gov/health-information/disorders/postural-tachycardia-syndrome |

## Autonomic Dysreflexia
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Life-threatening sympathetic surge** in response to noxious stimuli **below level of SCI** (usually =T6) | https://www.ninds.nih.gov/health-information/disorders/autonomic-dysreflexia |
| **Triggers** | **Bladder distention** (most common), **Bowel impaction**, **Pressure injury**, **UTI**, **Fracture**, **Surgery**, **Labor** | https://www.ninds.nih.gov/health-information/disorders/autonomic-dysreflexia |
| **Symptoms** | **Severe hypertension**, **Bradycardia** (reflex), **Headache**, **Flushing/sweating above lesion**, **Piloerection**, **Nasal congestion**, **Blurred vision** | https://www.ninds.nih.gov/health-information/disorders/autonomic-dysreflexia |
| **Management** | **Sit upright**, **Remove trigger** (check Foley, Bowel), **Rapid-acting antihypertensive** (nifedipine, Nitroglycerin, Hydralazine) if BP remains elevated | https://www.ninds.nih.gov/health-information/disorders/autonomic-dysreflexia |

## Pure Autonomic Failure (PAF)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Isolated autonomic failure** without other neurologic deficits; **Synucleinopathy** | https://www.ninds.nih.gov/health-information/disorders/pure-autonomic-failure |
| **Clinical Features** | **Orthostatic hypotension**, **Urinary dysfunction**, **Constipation**, **Erectile dysfunction**, **Anhidrosis** | https://www.ninds.nih.gov/health-information/disorders/pure-autonomic-failure |
| **Diagnosis** | **Autonomic testing** (severe adrenergic and cardiovagal impairment), **Normal neurologic exam** (no parkinsonism, Cerebellar signs), **Normal cardiac MIBG** (distinguishes from MSA) | https://www.ninds.nih.gov/health-information/disorders/pure-autonomic-failure |
| **Prognosis** | **Slow progression**; Some evolve to MSA or DLB | https://www.ninds.nih.gov/health-information/disorders/pure-autonomic-failure |
