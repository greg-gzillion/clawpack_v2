## Febrile Seizures
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Seizure in child **6 months - 5 years** with **fever =38°C**, No CNS infection, No prior afebrile seizures | https://www.ninds.nih.gov/health-information/disorders/febrile-seizures |
| **Simple Febrile Seizure** | **Generalized**, **<15 minutes**, **Once per 24 hours**, **No focal features** | https://www.ninds.nih.gov/health-information/disorders/febrile-seizures |
| **Complex Febrile Seizure** | **Focal**, **>15 minutes**, **Recurs within 24 hours** | https://www.ninds.nih.gov/health-information/disorders/febrile-seizures |
| **Evaluation** | **LP** if signs of meningitis; **Neuroimaging/EEG** not routinely indicated for simple febrile seizures | https://www.ninds.nih.gov/health-information/disorders/febrile-seizures |
| **Management** | **Reassurance**, **Antipyretics** (do not prevent seizures), **Rescue benzodiazepines** for prolonged seizures; **No prophylactic ASMs** | https://www.ninds.nih.gov/health-information/disorders/febrile-seizures |
| **Prognosis** | **Excellent**; 30% recurrence; <5% risk of epilepsy | https://www.ninds.nih.gov/health-information/disorders/febrile-seizures |

## Infantile Spasms (West Syndrome)
| Component | Description | URL |
|-----------|-------------|-----|
| **Clinical Triad** | **Epileptic spasms** (flexion, Extension, Mixed), **Hypsarrhythmia** on EEG, **Developmental regression** | https://www.ninds.nih.gov/health-information/disorders/infantile-spasms |
| **Onset** | **3-12 months** (peak 4-7 months) | https://www.ninds.nih.gov/health-information/disorders/infantile-spasms |
| **Etiology** | **Structural** (cortical dysplasia, Tuberous sclerosis), **Genetic** (CDKL5, ARX, STXBP1), **Metabolic**, **Unknown** | https://www.ninds.nih.gov/health-information/disorders/infantile-spasms |
| **Treatment** | **ACTH** (first-line), **Vigabatrin** (especially tuberous sclerosis), **Corticosteroids** | https://www.ninds.nih.gov/health-information/disorders/infantile-spasms |
| **Prognosis** | **Poor**; 70-90% have intellectual disability; Better if early treatment and normal development prior to onset | https://www.ninds.nih.gov/health-information/disorders/infantile-spasms |

## Pediatric Stroke
| Component | Description | URL |
|-----------|-------------|-----|
| **Types** | **Arterial ischemic stroke (AIS)** , **Cerebral sinovenous thrombosis (CSVT)** , **Hemorrhagic stroke** | https://www.stroke.org/en/about-stroke/stroke-in-children |
| **Risk Factors** | **Cardiac disease** (congenital, Acquired), **Arteriopathy** (focal cerebral arteriopathy, Moyamoya), **Thrombophilia**, **Sickle cell disease**, **Infection** (varicella, Meningitis) | https://www.stroke.org/en/about-stroke/stroke-in-children |
| **Clinical Features** | **Focal neurologic deficit**, **Seizures** (common), **Altered mental status**, **Headache** | https://www.stroke.org/en/about-stroke/stroke-in-children |
| **Diagnosis** | **MRI brain with MRA** (preferred), **Echocardiography**, **Thrombophilia evaluation** | https://www.stroke.org/en/about-stroke/stroke-in-children |
| **Treatment (AIS)** | **Aspirin** (initial), **Anticoagulation** (cardioembolic, Dissection), **Consider thrombectomy** in select cases | https://www.stroke.org/en/about-stroke/stroke-in-children |

## Cerebral Palsy (CP)
| Type | Clinical Features | URL |
|------|-------------------|-----|
| **Spastic CP** (80%) | **Hypertonia**, **Hyperreflexia**, **Clonus**, **Scissoring gait**; **Hemiplegic**, **Diplegic** (legs > arms), **Quadriplegic** | https://www.ninds.nih.gov/health-information/disorders/cerebral-palsy |
| **Dyskinetic CP** (10-15%) | **Chorea**, **Athetosis**, **Dystonia**; Often due to **kernicterus** or **hypoxic-ischemic injury** | https://www.ninds.nih.gov/health-information/disorders/cerebral-palsy |
| **Ataxic CP** (5%) | **Cerebellar signs** (ataxia, Dysmetria, Intention tremor) | https://www.ninds.nih.gov/health-information/disorders/cerebral-palsy |
| **Management** | **Multidisciplinary**: PT/OT/SLP, **Spasticity management** (baclofen, Botulinum toxin, Selective dorsal rhizotomy), **Seizure management**, **Orthopedic surgery** | https://www.ninds.nih.gov/health-information/disorders/cerebral-palsy |
