# Neurology - Dementia and Cognitive Disorders

## Mild Cognitive Impairment (MCI)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Objective cognitive impairment** (1-1.5 SD below mean) with **preserved functional independence** | https://www.alz.org/alzheimers-dementia/what-is-dementia/related_conditions/mild-cognitive-impairment |
| **Subtypes** | **Amnestic MCI** (memory predominant, Higher AD risk), **Non-amnestic MCI** (executive, Language, Visuospatial) | https://www.alz.org/alzheimers-dementia/what-is-dementia/related_conditions/mild-cognitive-impairment |
| **Progression** | **10-15% per year** progress to dementia (vs. 1-2% in general population) | https://www.alz.org/alzheimers-dementia/what-is-dementia/related_conditions/mild-cognitive-impairment |
| **Management** | **Cognitive stimulation**, **Physical exercise**, **Cardiovascular risk factor control**, **No FDA-approved medications** | https://www.alz.org/alzheimers-dementia/what-is-dementia/related_conditions/mild-cognitive-impairment |

## Alzheimer Disease (AD)
| Component | Description | URL |
|-----------|-------------|-----|
| **Epidemiology** | **Most common cause of dementia** (60-80%); Age >65 | https://www.ninds.nih.gov/health-information/disorders/alzheimers-disease |
| **Pathophysiology** | **Amyloid plaques** (Aß), **Neurofibrillary tangles** (hyperphosphorylated tau), **Neuronal loss** | https://www.ninds.nih.gov/health-information/disorders/alzheimers-disease |
| **Genetics** | **APOE e4** (risk factor), **APP**, **PSEN1**, **PSEN2** (early-onset autosomal dominant) | https://www.ninds.nih.gov/health-information/disorders/alzheimers-disease |
| **Clinical Features** | **Progressive episodic memory loss**, **Anosognosia**, **Visuospatial deficits**, **Language impairment**, **Executive dysfunction** | https://www.ninds.nih.gov/health-information/disorders/alzheimers-disease |
| **Atypical Variants** | **Posterior cortical atrophy** (visuospatial), **Logopenic variant PPA** (language), **Frontal variant** (executive/behavioral) | https://www.ninds.nih.gov/health-information/disorders/alzheimers-disease |

## Dementia with Lewy Bodies (DLB)
| Component | Description | URL |
|-----------|-------------|-----|
| **Core Features** | **Fluctuating cognition**, **Recurrent visual hallucinations**, **REM sleep behavior disorder**, **Parkinsonism** | https://www.ninds.nih.gov/health-information/disorders/lewy-body-dementia |
| **Supportive Features** | Severe neuroleptic sensitivity, Falls/Syncope, Autonomic dysfunction, Depression, Delusions | https://www.ninds.nih.gov/health-information/disorders/lewy-body-dementia |
| **Diagnostic Criteria** | Dementia + =2 core features OR 1 core + =1 indicative biomarker (DaTscan, MIBG) | https://www.ninds.nih.gov/health-information/disorders/lewy-body-dementia |
| **Treatment** | **Cholinesterase inhibitors** (rivastigmine FDA-approved); **AVOID antipsychotics** (neuroleptic sensitivity); Treat RBD (melatonin, Clonazepam); Carbidopa/levodopa for parkinsonism | https://www.ninds.nih.gov/health-information/disorders/lewy-body-dementia |

## Frontotemporal Dementia (FTD)
| Variant | Presentation | Pathology | URL |
|---------|--------------|-----------|-----|
| **Behavioral variant FTD (bvFTD)** | **Disinhibition**, **Apathy**, **Loss of empathy**, **Compulsive behaviors**, **Hyperorality**, Executive dysfunction | tau (40%), TDP-43 (50%), FUS | https://www.ninds.nih.gov/health-information/disorders/frontotemporal-dementia |
| **Semantic variant PPA** | **Fluent aphasia**, **Loss of word meaning**, Impaired object knowledge | TDP-43 (most) | https://www.ninds.nih.gov/health-information/disorders/frontotemporal-dementia |
| **Nonfluent/Agrammatic variant PPA** | **Effortful speech**, **Agrammatism**, **Apraxia of speech** | tau (most) | https://www.ninds.nih.gov/health-information/disorders/frontotemporal-dementia |
| **Treatment** | Supportive; **SSRIs** for behavioral symptoms; **Avoid cholinesterase inhibitors** (may worsen) | | https://www.ninds.nih.gov/health-information/disorders/frontotemporal-dementia |

## Vascular Dementia (VaD)
| Component | Description | URL |
|-----------|-------------|-----|
| **Epidemiology** | **Second most common cause** (10-20%) | https://www.ninds.nih.gov/health-information/disorders/vascular-dementia |
| **Subtypes** | **Multi-infarct dementia**, **Strategic infarct dementia**, **Subcortical ischemic vascular dementia** (Binswanger), **Cerebral amyloid angiopathy** | https://www.ninds.nih.gov/health-information/disorders/vascular-dementia |
| **Clinical Features** | **Executive dysfunction**, **Psychomotor slowing**, **Gait disturbance**, **Urinary symptoms**, Focal neurologic deficits | https://www.ninds.nih.gov/health-information/disorders/vascular-dementia |
| **Imaging** | MRI: **White matter hyperintensities**, **Lacunar infarcts**, **Microbleeds** | https://www.ninds.nih.gov/health-information/disorders/vascular-dementia |
| **Treatment** | **Risk factor control** (BP, Lipids, Glucose, Smoking cessation), Antiplatelet therapy, Cholinesterase inhibitors (modest benefit) | https://www.ninds.nih.gov/health-information/disorders/vascular-dementia |

## Normal Pressure Hydrocephalus (NPH)
| Component | Description | URL |
|-----------|-------------|-----|
| **Classic Triad (Hakim Triad)** | **Gait disturbance** (magnetic gait, Wide-based, Shuffling), **Cognitive impairment** (executive, Bradyphrenia), **Urinary incontinence** | https://www.ninds.nih.gov/health-information/disorders/normal-pressure-hydrocephalus |
| **Imaging** | **Ventriculomegaly** (Evans index >0.3) out of proportion to atrophy; **DESH** (disproportionately enlarged subarachnoid space hydrocephalus) | https://www.ninds.nih.gov/health-information/disorders/normal-pressure-hydrocephalus |
| **Diagnostic Test** | **High-volume LP** (30-50 mL CSF removal) ? Assess gait/cognition improvement | https://www.ninds.nih.gov/health-information/disorders/normal-pressure-hydrocephalus |
| **Treatment** | **Ventriculoperitoneal (VP) shunt** | https://www.ninds.nih.gov/health-information/disorders/normal-pressure-hydrocephalus |
