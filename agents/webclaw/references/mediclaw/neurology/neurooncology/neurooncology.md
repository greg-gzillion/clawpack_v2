## Glioblastoma - Management
| Component | Description | URL |
|-----------|-------------|-----|
| **Standard of Care** | **Maximal safe resection** ? **Radiation** (60 Gy in 30 fractions) + **Concurrent Temozolomide** ? **Adjuvant Temozolomide** x 6-12 cycles | https://www.cancer.gov/types/brain/hp/adult-brain-treatment-pdq |
| **Tumor Treating Fields (TTFields)** | **Optune** (alternating electric fields); Improves survival when added to maintenance temozolomide | https://www.fda.gov/medical-devices |
| **Recurrent Disease** | **Bevacizumab** (anti-VEGF), **Lomustine**, **Re-irradiation**, **Clinical trials** | https://www.cancer.gov/types/brain/hp/adult-brain-treatment-pdq |

## Brain Metastases
| Component | Description | URL |
|-----------|-------------|-----|
| **Epidemiology** | **Most common intracranial tumor**; 10-30% of cancer patients | https://www.cancer.gov/types/brain |
| **Common Primary Sites** | **Lung** (50%), **Breast** (15-20%), **Melanoma** (10%), **Renal**, **Colorectal** | https://www.cancer.gov/types/brain |
| **Clinical Features** | **Headache**, **Focal neurologic deficits**, **Seizures**, **Cognitive changes** | https://www.cancer.gov/types/brain |
| **Diagnosis** | **MRI brain with contrast** (multiple ring-enhancing lesions at gray-white junction) | https://www.cancer.gov/types/brain |
| **Treatment** | **Stereotactic radiosurgery (SRS)** (1-4 lesions), **Whole brain radiation** (multiple lesions, Poor prognosis), **Surgery** (single large lesion, Mass effect, Tissue diagnosis), **Systemic therapy** (targeted, Immunotherapy) | https://www.cancer.gov/types/brain/hp/adult-brain-treatment-pdq |

## Leptomeningeal Carcinomatosis
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Diffuse infiltration of leptomeninges** by malignant cells | https://www.cancer.gov/types/brain |
| **Common Primary Sites** | **Breast**, **Lung**, **Melanoma**, **GI**, **Lymphoma/Leukemia** | https://www.cancer.gov/types/brain |
| **Clinical Features** | **Multilevel neurologic deficits** (cranial nerves, Radiculopathy, Myelopathy), **Headache**, **Nausea/Vomiting**, **Altered mental status**, **Seizures** | https://www.cancer.gov/types/brain |
| **Diagnosis** | **MRI brain/spine with contrast** (leptomeningeal enhancement, Cranial nerve enhancement, Nodules), **LP** (elevated opening pressure, Pleocytosis, Low glucose, High protein, **Positive cytology**) | https://www.cancer.gov/types/brain |
| **Treatment** | **Intrathecal chemotherapy** (methotrexate, Cytarabine, Thiotepa), **Radiation** (symptomatic sites), **Systemic therapy**, **Palliative care** | https://www.cancer.gov/types/brain/hp/adult-brain-treatment-pdq |
