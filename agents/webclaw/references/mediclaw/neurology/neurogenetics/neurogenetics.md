## Hereditary Spastic Paraplegia (HSP)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Progressive spasticity and weakness** of lower extremities | https://www.ninds.nih.gov/health-information/disorders/hereditary-spastic-paraplegia |
| **Genetics** | **Autosomal dominant** (70-80%, SPG4 most common), **Autosomal recessive**, **X-linked** | https://www.ninds.nih.gov/health-information/disorders/hereditary-spastic-paraplegia |
| **Pure HSP** | **Isolated spastic paraparesis**, Hyperreflexia, Babinski, Urinary urgency | https://www.ninds.nih.gov/health-information/disorders/hereditary-spastic-paraplegia |
| **Complex HSP** | Spasticity + **Additional features**: Peripheral neuropathy, Ataxia, Cognitive impairment, Optic atrophy, Retinopathy | https://www.ninds.nih.gov/health-information/disorders/hereditary-spastic-paraplegia |
| **Treatment** | **Symptomatic**: PT/OT, Baclofen, Tizanidine, Botulinum toxin, Anticholinergics (urinary) | https://www.ninds.nih.gov/health-information/disorders/hereditary-spastic-paraplegia |

## Mitochondrial Disorders - Neurologic Features
| Syndrome | Key Neurologic Features | Other Features | Genetics | URL |
|----------|-------------------------|----------------|----------|-----|
| **MELAS** | **Stroke-like episodes**, Seizures, Migraine, Dementia | **Lactic acidosis**, Short stature, Hearing loss, Diabetes | **mtDNA** (m.3243A>G) | https://www.ninds.nih.gov/health-information/disorders/mitochondrial-disorders |
| **MERRF** | **Myoclonus**, **Epilepsy**, Ataxia | **Ragged red fibers** | **mtDNA** (m.8344A>G) | https://www.ninds.nih.gov/health-information/disorders/mitochondrial-disorders |

| **Leigh Syndrome** | **Progressive neurodegeneration**, Hypotonia, Seizures, Respiratory failure | **Bilateral basal ganglia lesions** | mtDNA or nDNA | https://www.ninds.nih.gov/health-information/disorders/mitochondrial-disorders |
| **NARP** | **Neuropathy**, **Ataxia**, **Retinitis pigmentosa** | Developmental delay | **mtDNA** (MT-ATP6) | https://www.ninds.nih.gov/health-information/disorders/mitochondrial-disorders |

## Genetic Testing in Neurology
| Test | Indications | URL |
|------|-------------|-----|
| **Karyotype** | Suspected aneuploidy, Large structural variants | https://www.genome.gov/genetics-glossary/Karyotype |
| **Chromosomal Microarray (CMA)** | **First-line for unexplained developmental delay, Autism, Congenital anomalies** | https://www.genome.gov/genetics-glossary/Microarray |
| **Single-Gene Testing** | High clinical suspicion for specific disorder (e.g., Huntington disease, Duchenne MD) | https://www.genome.gov/genetics-glossary/Gene-Testing |
| **Gene Panel** | Heterogeneous disorders (e.g., Epilepsy, CMT, HSP, Mitochondrial) | https://www.genome.gov/genetics-glossary/Gene-Panel |
| **Whole Exome Sequencing (WES)** | Unexplained neurologic disorder with suspected genetic etiology | https://www.genome.gov/genetics-glossary/Whole-Exome-Sequencing |
| **Whole Genome Sequencing (WGS)** | WES-negative, Suspected non-coding variants | https://www.genome.gov/genetics-glossary/Whole-Genome-Sequencing |
