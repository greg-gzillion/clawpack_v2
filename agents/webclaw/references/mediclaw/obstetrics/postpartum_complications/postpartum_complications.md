# Obstetrics - Postpartum Complications

## Postpartum Hemorrhage (See Obstetric Emergencies)

## Postpartum Endometritis
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Infection of endometrial cavity** (usually polymicrobial) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2018/12/postpartum-hemorrhage |
| **Risk Factors** | **Cesarean delivery** (especially after labor/ROM), **Chorioamnionitis**, **Prolonged ROM**, **Multiple vaginal exams** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2018/12/postpartum-hemorrhage |
| **Symptoms** | **Fever**, **Uterine tenderness**, **Foul lochia**, **Tachycardia** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2018/12/postpartum-hemorrhage |
| **Treatment** | **Clindamycin 900 mg IV q8h + Gentamicin 1.5 mg/kg IV q8h** (or 5 mg/kg daily) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2018/12/postpartum-hemorrhage |

## Wound Complications
| Complication | Management | URL |
|--------------|------------|-----|
| **Wound Infection** | **Open incision**, **Drainage**, **Antibiotics** (cephalexin + metronidazole) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2019/02/cesarean-delivery |
| **Wound Dehiscence** | **Wound care**, **Secondary intention** or **Delayed closure** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2019/02/cesarean-delivery |
| **Fascial Dehiscence** | **Surgical repair** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2019/02/cesarean-delivery |
| **Necrotizing Fasciitis** | **Emergent surgical debridement**, **Broad-spectrum antibiotics** | https://www.cdc.gov/groupastrep/diseases-hcp/necrotizing-fasciitis.html |

## Venous Thromboembolism (VTE)
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **Risk Factors** | **Cesarean delivery**, **Obesity**, **Preeclampsia**, **Infection**, **Hemorrhage**, **Immobility** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2018/07/thromboembolism-in-pregnancy |
| **Prophylaxis** | **Mechanical** (SCDs) for all cesareans; **LMWH** for high-risk | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2018/07/thromboembolism-in-pregnancy |
| **Diagnosis** | **Compression ultrasound** (DVT), **CT-PA** or **V/Q scan** (PE) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2018/07/thromboembolism-in-pregnancy |
| **Treatment** | **LMWH** (enoxaparin 1 mg/kg BID or 1.5 mg/kg daily) or **Unfractionated heparin** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2018/07/thromboembolism-in-pregnancy |
| **Duration** | **=6 weeks postpartum** (minimum 3-6 months total) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2018/07/thromboembolism-in-pregnancy |

## Mastitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Infection of breast tissue** (usually S. aureus) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/02/breastfeeding |
| **Symptoms** | **Fever**, **Breast pain**, **Erythema**, **Induration** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/02/breastfeeding |
| **Treatment** | **Continue breastfeeding/pumping** (empty breast), **Dicloxacillin** or **Cephalexin** x10-14 days, **Analgesics** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/02/breastfeeding |
| **Breast Abscess** | **Ultrasound**, **Needle aspiration** or **Incision and drainage** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/02/breastfeeding |

## Postpartum Preeclampsia
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **New-onset preeclampsia** presenting **postpartum** (usually within 48h, up to 6 weeks) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Symptoms** | **Headache**, **Visual changes**, **Hypertension**, **Proteinuria**, **Seizures** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Treatment** | **Magnesium sulfate** for severe features/seizures, **Antihypertensives** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
