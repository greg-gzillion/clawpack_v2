# Obstetrics - Hypertensive Disorders of Pregnancy

## Classification
| Disorder | Diagnostic Criteria | URL |
|----------|---------------------|-----|
| **Chronic Hypertension** | **HTN diagnosed before pregnancy or <20 weeks** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2019/06/chronic-hypertension-in-pregnancy |
| **Gestational Hypertension** | **New-onset HTN =20 weeks** without proteinuria or end-organ dysfunction | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Preeclampsia** | **New-onset HTN =20 weeks** + **Proteinuria** OR **End-organ dysfunction** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Chronic Hypertension with Superimposed Preeclampsia** | Chronic HTN + **New-onset proteinuria** or **Worsening HTN** or **End-organ dysfunction** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Eclampsia** | **Seizures** in setting of preeclampsia (not attributable to other cause) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **HELLP Syndrome** | **H**emolysis, **E**levated **L**iver enzymes, **L**ow **P**latelets | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |

## Diagnostic Criteria for Preeclampsia (ACOG)
| Criterion | Threshold | URL |
|-----------|-----------|-----|
| **Blood Pressure** | **=140/90 mmHg** on **2 occasions =4 hours apart** (or =160/110 confirmed within minutes) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **AND =1 of the following:** | | |
| **Proteinuria** | **=300 mg/24h**, **Protein/Cr ratio =0.3**, **Dipstick =2+** (if quantitative unavailable) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Thrombocytopenia** | **Platelets <100,000/ÁL** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Renal Insufficiency** | **Cr =1.1 mg/dL** or **Doubling of baseline** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Liver Involvement** | **Transaminases =2x ULN** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Pulmonary Edema** | Clinical or radiographic | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Neurologic Symptoms** | **New-onset headache** unresponsive to medication, **Visual disturbances** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |

## Preeclampsia with Severe Features
| Criterion | Threshold | URL |
|-----------|-----------|-----|
| **Severe HTN** | **=160/110 mmHg** on 2 occasions =4 hours apart (or confirmed within minutes) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Thrombocytopenia** | **Platelets <100,000/ÁL** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Renal Insufficiency** | **Cr =1.1 mg/dL** or **Doubling of baseline** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Liver Involvement** | **Transaminases =2x ULN** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Pulmonary Edema** | Clinical or radiographic | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Neurologic Symptoms** | **Persistent headache**, **Visual disturbances**, **Altered mental status** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |

## Management of Hypertensive Disorders
| Condition | Delivery Timing | URL |
|-----------|-----------------|-----|
| **Chronic Hypertension** | **37-39 weeks** (individualize) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2019/06/chronic-hypertension-in-pregnancy |
| **Gestational Hypertension** | **37 weeks** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Preeclampsia (No Severe Features)** | **37 weeks** (expectant management 34-36 weeks if stable) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Preeclampsia with Severe Features** | **=34 weeks**: **Delivery at diagnosis**; **<34 weeks**: **Expectant management** (if stable) with corticosteroids | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **HELLP Syndrome** | **=34 weeks**: **Delivery at diagnosis**; **<34 weeks**: **Corticosteroids ? Delivery in 24-48h** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Eclampsia** | **Delivery after stabilization** (regardless of gestational age) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |

## Magnesium Sulfate for Seizure Prophylaxis
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | **Preeclampsia with severe features**, **Eclampsia** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Loading Dose** | **4-6 g IV over 20-30 minutes** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Maintenance Dose** | **1-2 g/hour IV** (continue 24h postpartum) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Therapeutic Level** | **4-8 mg/dL** (4-7 mEq/L) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Monitoring** | **Patellar reflexes**, **Respiratory rate**, **Urine output** (=30 mL/h) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Toxicity** | **Loss of reflexes** (8-10 mg/dL), **Respiratory depression** (10-15 mg/dL), **Cardiac arrest** (>15 mg/dL) | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
| **Reversal** | **Calcium gluconate 1 g IV** over 5-10 min | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2020/06/gestational-hypertension-and-preeclampsia |
