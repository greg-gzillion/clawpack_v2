## Neuraxial Analgesia (Epidural/Spinal)
| Component | Description | URL |
|-----------|-------------|-----|
| **Epidural Analgesia** | **Local anesthetic + Opioid** (e.g., Bupivacaine + Fentanyl) via epidural catheter | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2019/03/obstetric-analgesia-and-anesthesia |
| **Combined Spinal-Epidural (CSE)** | **Spinal injection** for rapid onset + **Epidural catheter** for prolonged labor | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2019/03/obstetric-analgesia-and-anesthesia |
| **Contraindications** | **Coagulopathy**, **Thrombocytopenia** (<70-100K), **Infection at site**, **Elevated ICP**, **Patient refusal** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2019/03/obstetric-analgesia-and-anesthesia |
| **Complications** | **Hypotension** (prevent with IV fluid bolus, Left uterine displacement), **Post-dural puncture headache**, **High block**, **Local anesthetic toxicity** | https://www.acog.org/clinical/clinical-guidance/practice-bulletin/articles/2019/03/obstetric-analgesia-and-anesthesia |
