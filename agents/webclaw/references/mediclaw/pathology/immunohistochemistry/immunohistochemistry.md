# Pathology - Immunohistochemistry (IHC)

## Overview
Immunohistochemistry uses antibodies to detect specific antigens in tissue sections, allowing for precise identification of cell lineage, differentiation, and molecular alterations.

| Component | Description | URL |
|-----------|-------------|-----|
| **Principle** | Antibody-antigen binding visualized by chromogen (DAB) or fluorophore | https://www.cap.org/member-resources/pathology-overview/immunohistochemistry |
| **Applications** | Tumor diagnosis, Subclassification, Prognosis, Predictive markers, Infectious agents | https://www.amp.org/education/ihc-resources/ |
| **Controls** | Positive and negative controls essential for valid interpretation | https://www.cap.org/member-resources/pathology-overview/immunohistochemistry |
