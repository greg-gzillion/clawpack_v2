# Pathology - Overview

## What is Pathology?
Pathology is the medical specialty that provides the scientific foundation for all medical practice. It bridges basic science and clinical medicine by studying the structural and functional changes in cells, tissues, and organs that underlie disease.

| Branch | Description | URL |
|--------|-------------|-----|
| **Anatomic Pathology** | Diagnosis of disease based on gross, microscopic, and molecular examination of tissues | https://www.cap.org/member-resources/pathology-overview |

| Type | Morphology | Mechanism | Example | URL |

| **Necrosis** | Cell swelling, Organelle breakdown, Membrane rupture | **ATP depletion**, Membrane injury | **Coagulative**: Ischemia (except brain); **Liquefactive**: Brain; **Caseous**: TB | https://www.ncbi.nlm.nih.gov/books/NBK53449/ |
