## Movement Terms
| Term | Definition | Example |
|------|------------|---------|
| Flexion | Decreasing angle | Bending elbow |
| Extension | Increasing angle | Straightening elbow |
| Abduction | Moving away from midline | Raising arm laterally |
| Adduction | Moving toward midline | Lowering arm |
| Rotation | Turning around axis | Turning head |
| Circumduction | Circular movement | Arm circles |
| Pronation | Palm facing down/back | Turning palm down |
| Supination | Palm facing up/forward | Turning palm up |
| Inversion | Sole facing inward | Turning ankle in |
| Eversion | Sole facing outward | Turning ankle out |
| Dorsiflexion | Toes toward shin | Flexing foot up |
| Plantarflexion | Toes pointing down | Pointing toes |
| Protraction | Moving forward | Jutting jaw forward |
| Retraction | Moving backward | Pulling jaw back |
| Elevation | Lifting upward | Shrugging shoulders |
| Depression | Lowering downward | Lowering shoulders |
| Opposition | Thumb to other fingers | Pinching |
