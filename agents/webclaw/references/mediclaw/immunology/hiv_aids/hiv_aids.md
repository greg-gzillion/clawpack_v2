# Immunology - HIV/AIDS

## HIV Virology and Pathogenesis
| Component | Description | URL |
|-----------|-------------|-----|
| **Virus** | **Human Immunodeficiency Virus (HIV)** - Retrovirus (Lentivirus) | https://www.hiv.gov |
| **Types** | **HIV-1** (global), **HIV-2** (West Africa, Slower progression, NNRTI-resistant) | |
| **Target Cell** | **CD4+ T cells**, Macrophages, Dendritic cells | |
| **Viral Entry** | gp120 binds **CD4** ? Conformational change ? Binds **co-receptor** (CCR5 or CXCR4) ? gp41 mediates fusion | |
| **Key Enzymes** | **Reverse Transcriptase** (RNA ? DNA), **Integrase** (integrates viral DNA into host genome), **Protease** (cleaves viral polyproteins) | |
| **Pathogenesis** | **Progressive depletion of CD4+ T cells** ? **Immunodeficiency** ? Opportunistic infections and malignancies | |

|------|---------------|---------|-------|-----|

| **3rd Generation Antibody Immunoassay** | **22-50 days** | HIV-1/2 antibodies (IgG, IgM) | Older test | |

| **CD4 Count** | - | CD4+ T cells/無 | **Immune status**, OI risk stratification | |
| **CD4 Percentage** | - | % of total lymphocytes | More stable than absolute CD4 count | |

## HIV Disease Staging (CDC)
| Stage | CD4 Count | CD4 Percentage | Clinical | URL |
|-------|-----------|----------------|----------|-----|
| **Stage 1** | =500 cells/無 | =26% | No AIDS-defining condition | https://www.cdc.gov/hiv/basics |
| **Stage 2** | 200-499 cells/無 | 14-25% | No AIDS-defining condition | |
| **Stage 3 (AIDS)** | <200 cells/無 | <14% | **OR** **AIDS-defining condition** regardless of CD4 | |

## Antiretroviral Therapy (ART) - Drug Classes
| Class | Examples | Mechanism | Key Side Effects | URL |
|-------|----------|-----------|------------------|-----|
| **NRTIs** | Tenofovir (TAF/TDF), Emtricitabine, Abacavir, Lamivudine | **Nucleoside reverse transcriptase inhibitors** (chain terminators) | **TDF**: Renal toxicity, Bone loss; **TAF**: Weight gain, Lipids; **Abacavir**: **HLA-B*5701 hypersensitivity** | https://clinicalinfo.hiv.gov |
| **NNRTIs** | Efavirenz, Rilpivirine, Doravirine, Etravirine | **Non-nucleoside reverse transcriptase inhibitors** (allosteric inhibition) | **Efavirenz**: CNS toxicity, Rash; **Rilpivirine**: QTc prolongation | |
| **PIs** | Darunavir, Atazanavir, Lopinavir | **Protease inhibitors** (prevent viral maturation) | **GI intolerance**, **Hyperlipidemia**, **Lipodystrophy**, **Ritonavir/cobicistat boosting** | |
| **INSTIs** | **Bictegravir**, **Dolutegravir**, **Raltegravir**, **Cabotegravir** | **Integrase strand transfer inhibitors** | **Dolutegravir**: Insomnia, **Neural tube defects** (controversial), **Weight gain** | |
| **Entry Inhibitors** | Maraviroc | **CCR5 antagonist** (blocks viral entry) | **Requires CCR5 tropism testing** | |
| **Fusion Inhibitors** | Enfuvirtide (T-20) | Binds gp41 (prevents fusion) | Injection site reactions (rarely used) | |
| **Attachment Inhibitors** | Fostemsavir | Binds gp120 (prevents CD4 binding) | For heavily treatment-experienced | |
| **Post-Attachment Inhibitors** | Ibalizumab | Anti-CD4 monoclonal antibody | IV infusion; For heavily treatment-experienced | |
| **Capsid Inhibitors** | **Lenacapavir** | Disrupts capsid assembly/disassembly | **Twice-yearly** injection; For heavily treatment-experienced | |

## HIV Prevention
| Strategy | Intervention | Efficacy | URL |
|----------|--------------|----------|-----|
| **PrEP (Pre-Exposure Prophylaxis)** | **Daily oral TDF/FTC (Truvada)** or **TAF/FTC (Descovy)**; **Cabotegravir IM q2 months** | **>99%** with adherence | https://www.cdc.gov/hiv/prevention |
| **PEP (Post-Exposure Prophylaxis)** | **TDF/FTC + Raltegravir** or **Dolutegravir** x 28 days | **>80%** if started within 72 hours | |
| **Treatment as Prevention (U=U)** | **Undetectable = Untransmittable** (HIV RNA <200 copies/mL) | **100%** (no transmissions in PARTNER studies) | https://www.preventionaccess.org |
| **Condoms** | Consistent and correct use | 70-90% | |
| **Circumcision** | Voluntary medical male circumcision | 50-60% (heterosexual) | |
| **Harm Reduction** | Needle exchange, Opioid agonist therapy | Reduces transmission | |

## Opportunistic Infection Prophylaxis

|------------|-------------|-----|

| **CD4 <100 + Positive Toxoplasma IgG** | **TMP-SMX DS daily** | |
| **CD4 <50 + Positive MAC culture** (or not on ART) | **Azithromycin 1200 mg weekly** | |
| **CD4 <100 + Positive TST/IGRA** | **Isoniazid x 9 months** (or alternative LTBI regimen) | |

## HIV and Pregnancy
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **ART** | **All pregnant women should receive ART** regardless of CD4 or viral load | https://clinicalinfo.hiv.gov/perinatal |
| **Preferred Regimens** | **Dolutegravir + TAF/FTC** or **TDF/FTC**; **Raltegravir + TAF/FTC** | |
| **Intrapartum** | IV Zidovudine if HIV RNA >1000 near delivery | |
| **Neonatal Prophylaxis** | **Zidovudine x 4-6 weeks**; Additional agents if high-risk | |
| **Breastfeeding** | **NOT recommended** in US (safe alternatives available); **U=U does NOT apply to breastfeeding** | |
| **Mode of Delivery** | **Cesarean at 38 weeks** if HIV RNA >1000; Vaginal delivery if HIV RNA <1000 | |
