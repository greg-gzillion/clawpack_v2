# Immunology - Complement System

## Complement Pathways Overview
| Pathway | Activators | URL |
|---------|------------|-----|
| **Classical** | **Antigen-antibody complexes** (IgM, IgG1, IgG3), CRP, Apoptotic cells | https://www.niaid.nih.gov |
| **Lectin** | **Mannose residues** on microbial surfaces (MBL, Ficolins) | |
| **Alternative** | **Spontaneous C3 hydrolysis** (C3 tickover), Microbial surfaces | |

## Classical Pathway
| Step | Component | Action | URL |
|------|-----------|--------|-----|
| 1 | **C1q** | Binds Fc region of IgM or IgG (=2 IgG molecules) | https://www.niaid.nih.gov |
| 2 | **C1r, C1s** | Activated ? Cleave C4 and C2 | |
| 3 | **C4b2a** | **C3 convertase** (classical/lectin) | |
| 4 | **C4b2a3b** | **C5 convertase** (classical/lectin) | |

## Lectin Pathway
| Step | Component | Action | URL |
|------|-----------|--------|-----|
| 1 | **MBL, Ficolins** | Bind mannose residues on microbial surfaces | https://www.niaid.nih.gov |
| 2 | **MASP-1, MASP-2** | Activated ? Cleave C4 and C2 | |
| 3 | **C4b2a** | C3 convertase (same as classical) | |
| 4 | **C4b2a3b** | C5 convertase (same as classical) | |

## Alternative Pathway
| Step | Component | Action | URL |
|------|-----------|--------|-----|
| 1 | **C3** | Spontaneous hydrolysis ? C3(H2O) | https://www.niaid.nih.gov |
| 2 | **Factor B** | Binds C3(H2O) ? Cleaved by **Factor D** ? **C3(H2O)Bb** (initial C3 convertase) | |
| 3 | **C3b** | Binds microbial surfaces ? Binds Factor B ? Cleaved by Factor D ? **C3bBb** (C3 convertase) | |
| 4 | **Properdin (Factor P)** | **Stabilizes** C3bBb | |
| 5 | **C3bBb3b** | C5 convertase | |

## Terminal Pathway (Common)
| Step | Component | Action | URL |
|------|-----------|--------|-----|
| 1 | **C5 convertase** | Cleaves C5 ? **C5a** (anaphylatoxin) + **C5b** | https://www.niaid.nih.gov |
| 2 | **C5b** | Binds C6, C7, C8 ? Inserts into membrane | |
| 3 | **C9** | Multiple C9 molecules polymerize ? **Membrane Attack Complex (MAC)** | |
| 4 | **MAC** | Pore formation ? **Osmotic lysis** | |

## Complement Effector Functions
| Function | Mediators | Mechanism | URL |
|----------|-----------|-----------|-----|
| **Opsonization** | **C3b, C4b** | Bind complement receptors (CR1) on phagocytes ? Enhanced phagocytosis | https://www.niaid.nih.gov |
| **Anaphylatoxins** | **C3a, C4a, C5a** | Mast cell degranulation ? Increased vascular permeability, Smooth muscle contraction | |
| **Chemotaxis** | **C5a** | Attracts neutrophils, Macrophages to site of infection | |
| **Lysis** | **MAC (C5b-9)** | Direct killing of Gram-negative bacteria, Enveloped viruses | |
| **Immune Complex Clearance** | **C3b, C4b** | Bind to immune complexes ? Transported by RBCs (CR1) ? Cleared by spleen/liver | |

| Type | C1-INH Level | C1-INH Function | C4 | C1q | URL |
|------|--------------|----------------|----|-----|-----|
| **Type I** (85%) | **Low** | Low | Low | Normal | https://www.aaaai.org/HAE |
| **Type II** (15%) | Normal/elevated | **Low** | Low | Normal | |
| **Type III** (rare) | Normal | Normal | Normal | Normal | |
