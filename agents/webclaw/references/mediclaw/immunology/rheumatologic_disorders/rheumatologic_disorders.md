## Rheumatoid Arthritis (RA)
| Component | Description | URL |
|-----------|-------------|-----|
| Pathophysiology | **HLA-DR4** (shared epitope) + Environmental trigger ? **Citrullination of proteins** ? **Anti-CCP antibodies** ? **Synovial inflammation**, **Pannus formation**, **Cartilage/bone erosion** | https://www.rheumatology.org |
| Clinical Features | **Symmetric polyarthritis** (MCP, PIP, Wrists, MTP), **Morning stiffness >1 hour**, **Rheumatoid nodules**, **Extra-articular** (ILD, Vasculitis, Felty syndrome, Sj鐷ren) | |
| Diagnostic Criteria (ACR/EULAR 2010) | **Joint distribution** (0-5), **Serology** (RF, Anti-CCP) (0-3), **Acute phase reactants** (CRP, ESR) (0-1), **Duration** (=6 weeks) (0-1); **Score =6 = Definite RA** | |
| Laboratory | **RF** (70-80% sensitive, 80% specific), **Anti-CCP** (60-70% sensitive, **95-98% specific**), **Elevated ESR/CRP** | |
| Imaging | X-ray (erosions, Joint space narrowing, Periarticular osteopenia), **Ultrasound** (synovitis, Erosions), **MRI** (bone marrow edema) | |
| Treatment | **Treat-to-target** (remission or low disease activity); **Methotrexate** (first-line), **Hydroxychloroquine**, **Sulfasalazine**, **Leflunomide**; **Biologics** (TNF inhibitors, IL-6R inhibitors, Abatacept, Rituximab), **JAK inhibitors** (tofacitinib, Baricitinib, Upadacitinib) | |

## Systemic Lupus Erythematosus (SLE)
| Component | Description | URL |
|-----------|-------------|-----|
| Pathophysiology | **Loss of tolerance** to nuclear antigens (dsDNA, Sm, RNP) ? **Immune complex deposition** ? **Complement activation** ? **Multi-organ inflammation** | https://www.lupus.org |
| Clinical Features | **Malar rash**, **Discoid rash**, **Photosensitivity**, **Oral ulcers**, **Arthritis**, **Serositis** (pleuritis, Pericarditis), **Renal** (lupus nephritis), **Neurologic** (seizures, Psychosis), **Hematologic** (cytopenias) | |
| Diagnostic Criteria (ACR/EULAR 2019) | **ANA =1:80** (entry criterion) + =10 points from clinical and immunologic domains; **Clinical**: Constitutional, Hematologic, Neuropsychiatric, Mucocutaneous, Serosal, Musculoskeletal, Renal; **Immunologic**: Anti-dsDNA, Anti-Sm, Complement (C3/C4), Antiphospholipid antibodies | |
| Laboratory | **ANA** (98% sensitive), **Anti-dsDNA** (specific, Correlates with activity), **Anti-Sm** (specific), **Low C3/C4** (active disease), **Direct Coombs** (AIHA), **Antiphospholipid antibodies** | |
| Lupus Nephritis | **Class I** (minimal mesangial), **II** (mesangial proliferative), **III** (focal proliferative), **IV** (diffuse proliferative), **V** (membranous), **VI** (advanced sclerosing) | https://www.kidney.org |
| Treatment | **Hydroxychloroquine** (all patients), **Corticosteroids**, **Immunosuppressants** (Mycophenolate, Azathioprine, Cyclophosphamide, Tacrolimus), **Biologics** (Belimumab, Anifrolumab, Rituximab), **Voclosporin** (lupus nephritis) | |
