## Cytokine Release Syndrome (CRS)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Systemic inflammatory response due to massive cytokine release (IL-6, IFN-?, TNF-a) | https://www.fda.gov |
| Causes | **CAR T-cell therapy**, **Bispecific antibodies**, **Stem cell transplant**, **COVID-19** | |
| Clinical Features | Fever, Hypotension, Hypoxia, Capillary leak, Organ dysfunction | |
| Grading (ASTCT) | Grade 1: Fever; Grade 2: Hypotension not requiring pressors, Hypoxia requiring low-flow O2; Grade 3: Pressors required, High-flow O2; Grade 4: Life-threatening | |
| Management | **Tocilizumab** (anti-IL-6R) ± **Corticosteroids** | |
