## T Cell Development and Maturation
| Stage | Location | Process | URL |
|-------|----------|---------|-----|
| **Lymphoid Progenitor** | Bone marrow | Commitment to T cell lineage | https://www.niaid.nih.gov |
| **Double Negative (DN)** | Thymus (cortex) | No CD4 or CD8; TCR rearrangement (ß chain) | |
| **Double Positive (DP)** | Thymus (cortex) | CD4+ CD8+; TCR rearrangement (a chain) | |
| **Positive Selection** | Thymus (cortex) | T cells that recognize **self-MHC** survive; Others undergo apoptosis | |
| **Negative Selection** | Thymus (medulla) | T cells that bind **self-antigen + self-MHC too strongly** undergo apoptosis (**central tolerance**) | |
| **Single Positive (SP)** | Thymus (medulla) | CD4+ or CD8+ mature T cells exit to periphery | |
