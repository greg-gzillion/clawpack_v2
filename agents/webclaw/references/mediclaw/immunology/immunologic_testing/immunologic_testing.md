# Immunology - Immunologic Testing

## Antibody Detection Tests
| Test | Method | Clinical Use | Interpretation | URL |
|------|--------|--------------|----------------|-----|
| **ELISA** | Antigen-coated plate + Patient serum + Enzyme-linked anti-human Ig + Substrate | **HIV screening**, **Hepatitis**, **Autoantibodies** | **Optical density**; Quantitative | https://www.aaaai.org |
| **Western Blot** | Antigens separated by electrophoresis ? Transferred to membrane ? Incubated with patient serum | **Confirmatory testing** (HIV, Lyme) | **Band pattern** | |
| **Immunofluorescence (IFA)** | Patient serum + Tissue substrate + Fluorescent anti-human Ig | **ANA**, **ANCA**, **Anti-GBM**, **Anti-DSG** | **Pattern** (ANA: homogeneous, Speckled, Nucleolar, Centromere), **Titer** | |
| **Multiplex Bead Assay** | Beads coated with multiple antigens + Patient serum + Fluorescent anti-human Ig | **ANA profile**, **Vaccine responses** | **Mean fluorescence intensity (MFI)** | |
| **Complement Fixation** | Patient serum + Antigen + Complement ? Add sensitized RBCs ? Lysis? | **Coccidioidomycosis**, **Histoplasmosis** | **Titer** | |
| **Immunodiffusion** | Antigen and antibody diffuse through gel ? Precipitin line | **Fungal serologies**, **Autoantibodies** | **Presence/absence of precipitin line** | |

## Cellular Immunity Tests
| Test | Method | Clinical Use | Interpretation | URL |
|------|--------|--------------|----------------|-----|
| **Flow Cytometry** | Fluorescent-labeled antibodies bind cell surface/intracellular markers ? Cells analyzed by laser | **Immunophenotyping** (CD4, CD8, CD19, CD20, CD3, CD56), **Leukemia/Lymphoma**, **PNH** | **Percentage and absolute count** of cell populations | https://www.ash.org |
| **Lymphocyte Proliferation Assay** | Patient lymphocytes + Mitogen (PHA, PWM) or Antigen (tetanus, Candida) ? Measure 3H-thymidine incorporation | **T cell function** (SCID, CVID) | **Stimulation index** (counts with mitogen / counts without) | |
| **Delayed-Type Hypersensitivity (DTH) Skin Testing** | Intradermal injection of antigens (Candida, Tetanus, Mumps) ? Measure induration at 48-72 hours | **T cell function** (anergy panel) | **=5 mm induration = Positive** | |
| **IFN-? Release Assay (IGRA)** | Whole blood + TB antigens (ESAT-6, CFP-10) ? Measure IFN-? production | **Latent TB infection** | **Positive = TB infection** | https://www.cdc.gov/tb |
| **T Cell Receptor Excision Circle (TREC) Assay** | PCR for TRECs (byproduct of TCR rearrangement) on newborn dried blood spot | **Newborn screening for SCID** | **Low/absent TRECs = SCID** | https://primaryimmune.org |
| **Kappa-Deleting Recombination Excision Circle (KREC) Assay** | PCR for KRECs (byproduct of BCR rearrangement) | **Newborn screening for B cell defects** | **Low/absent KRECs = B cell deficiency** | |

## Neutrophil Function Tests
| Test | Method | Clinical Use | Interpretation | URL |
|------|--------|--------------|----------------|-----|
| **Dihydrorhodamine (DHR) Test** | Patient neutrophils + DHR + Stimulus (PMA) ? Measure fluorescence by flow cytometry | **Chronic Granulomatous Disease (CGD)** | **Absent/reduced fluorescence = CGD**; **Carriers** have bimodal pattern | https://primaryimmune.org |
| **Nitroblue Tetrazolium (NBT) Test** | Patient neutrophils + NBT + Stimulus ? Microscopy for blue formazan | **CGD** (historical) | **<10% positive neutrophils = CGD** | |
| **Neutrophil Oxidative Burst** | Patient neutrophils + Stimulus ? Measure superoxide production | **CGD** | **Absent/reduced superoxide = CGD** | |
| **Neutrophil Chemotaxis Assay** | Patient neutrophils placed in Boyden chamber with chemoattractant ? Measure migration | **Leukocyte adhesion deficiency**, **Job syndrome** | **Impaired migration** | |

## Complement Tests
| Test | Method | Clinical Use | Interpretation | URL |
|------|--------|--------------|----------------|-----|
| **CH50 (Total Hemolytic Complement)** | Patient serum + Antibody-coated sheep RBCs ? Measure lysis | **Screening for complement deficiency** | **Low/absent CH50 = Complement deficiency** (classical pathway) | https://www.aaaai.org |
| **AH50 (Alternative Pathway Hemolytic Complement)** | Patient serum + Rabbit RBCs ? Measure lysis | **Screening for alternative pathway deficiency** | **Low AH50 + Normal CH50 = Alternative pathway deficiency** | |
| **C3, C4 Levels** | Nephelometry | **Complement consumption** (SLE, C1-INH deficiency) | **Low C3, C4 = Immune complex disease**; **Low C4, Normal C3 = C1-INH deficiency** | |
| **C1-INH Level and Function** | Nephelometry (level), Chromogenic assay (function) | **Hereditary angioedema** | **Type I**: Low level, Low function; **Type II**: Normal/elevated level, **Low function** | |
| **C1q Level** | ELISA | **C1q deficiency** (SLE-like) | **Low/absent C1q** | |
| **Mannose-Binding Lectin (MBL) Level** | ELISA | **Recurrent infections** | **Low MBL** (common variant) | |

## Immunoglobulin and Antibody Response Tests
| Test | Method | Clinical Use | Interpretation | URL |
|------|--------|--------------|----------------|-----|
| **Serum Immunoglobulins (IgG, IgA, IgM, IgE)** | Nephelometry | **Humoral immunodeficiency** | **Low levels** = Hypogammaglobulinemia | https://primaryimmune.org |
| **IgG Subclasses** | Nephelometry | **IgG subclass deficiency** | **Low IgG1-4** (especially IgG2, IgG3) | |
| **Specific Antibody Titers** | ELISA | **Response to vaccination or infection** | **Protective titers** (tetanus >0.1 IU/mL, Diphtheria >0.1 IU/mL, Measles >120 mIU/mL, Rubella >10 IU/mL, VZV >1:8) | |
| **Antibody Response to Polysaccharide Vaccines** | Measure pre- and post-vaccination (4-8 weeks) titers to PPSV23 | **Specific antibody deficiency (SAD)** | **Normal**: =70% of serotypes with =2-fold increase and =1.3 mcg/mL | |
| **Antibody Response to Protein Vaccines** | Measure pre- and post-vaccination titers to tetanus/diphtheria | **T cell-dependent antibody response** | **=4-fold increase** or protective titer | |
| **Serum Protein Electrophoresis (SPEP)** | Separation of serum proteins by charge | **Monoclonal gammopathy** | **M-spike** = Monoclonal gammopathy; **Polyclonal hypergammaglobulinemia** = Chronic infection/inflammation | |

## Allergy Testing
| Test | Method | Clinical Use | Interpretation | URL |
|------|--------|--------------|----------------|-----|
| **Skin Prick Testing (SPT)** | Allergen extract placed on skin ? Prick ? Measure wheal/flare at 15-20 min | **IgE-mediated allergy** (aeroallergens, Foods, Venoms, Drugs) | **Wheal =3 mm > negative control** = Positive; **High NPV** | https://www.aaaai.org/allergy-testing |
| **Intradermal Testing** | Allergen injected intradermally ? Measure wheal/flare at 15-20 min | **Drug allergy** (penicillin), **Venom allergy**, **Aeroallergens** (if SPT negative) | **More sensitive than SPT**; **Higher false-positive rate** | |
| **Serum Specific IgE (sIgE)** | Immunoassay for allergen-specific IgE | **IgE-mediated allergy** (when SPT contraindicated or unavailable) | **Quantitative** (kU/L); **Lower sensitivity than SPT** | |
| **Component-Resolved Diagnostics (CRD)** | sIgE to specific allergen components | **Distinguish true allergy from cross-reactivity** (e.g., Ara h 2 for peanut, Cor a 9 for hazelnut) | **Component-specific interpretation** | |
| **Patch Testing** | Allergens applied to back under occlusion ? Read at 48, 72, 96 hours | **Type IV hypersensitivity** (contact dermatitis) | **Erythema, Induration, Vesicles** = Positive | https://www.aad.org |
| **Drug Provocation Testing** | Graded challenge with suspected drug under medical supervision | **Confirm/exclude drug allergy** when history unclear | **Development of objective symptoms** = Positive | |
| **Basophil Activation Test (BAT)** | Patient basophils + Allergen ? Measure CD63/CD203c by flow cytometry | **IgE-mediated allergy** (research/specialized centers) | **% activated basophils** | |
