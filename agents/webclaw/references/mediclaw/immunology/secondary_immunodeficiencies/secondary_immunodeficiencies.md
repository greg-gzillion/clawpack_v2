# Immunology - Secondary Immunodeficiencies

## Causes of Secondary Immunodeficiency
| Category | Examples | Mechanism | URL |
|----------|----------|-----------|-----|
| **Infections** | **HIV**, Measles, Influenza, TB | Depletion of CD4+ T cells, Immune suppression | https://www.hiv.gov |
| **Malignancy** | **CLL**, **Multiple myeloma**, **Lymphoma** | Impaired humoral immunity, Hypogammaglobulinemia | https://www.lls.org |
| **Medications** | **Corticosteroids**, **Chemotherapy**, **Immunosuppressants** (Cyclosporine, Tacrolimus, Mycophenolate, Rituximab, BTK inhibitors), **Anticonvulsants** | T cell suppression, B cell depletion, Hypogammaglobulinemia | https://www.fda.gov |
| **Protein-Losing States** | **Nephrotic syndrome**, **Protein-losing enteropathy**, **Severe burns** | Loss of immunoglobulins | |
| **Metabolic** | **Diabetes mellitus**, **Uremia**, **Liver cirrhosis** | Impaired phagocytosis, T cell dysfunction | |
| **Malnutrition** | Protein-calorie malnutrition, Zinc deficiency | Impaired cell-mediated immunity | |
| **Splenectomy / Asplenia** | Surgical splenectomy, Functional asplenia (sickle cell) | **Increased risk of encapsulated bacterial infections** | https://www.cdc.gov/vaccines |
| **Extremes of Age** | Neonates, Elderly | Immature immune system, Immunosenescence | |
| **Trauma/Surgery** | Major trauma, Major surgery, Burns | Immune suppression | |

## Immunosuppressive Medications - Mechanisms
| Drug Class | Examples | Primary Effect | Infectious Risk | URL |
|------------|----------|----------------|-----------------|-----|
| **Corticosteroids** | Prednisone, Methylprednisolone | **T cell suppression**, Impaired phagocytosis | Bacterial, Fungal, Viral (HSV, VZV), **PJP** (if >20 mg/day x >4 weeks) | https://www.rheumatology.org |
| **Calcineurin Inhibitors** | Cyclosporine, Tacrolimus | **Inhibit T cell activation** (block IL-2 production) | Viral (CMV, EBV, BK), Fungal | |
| **Antimetabolites** | Mycophenolate mofetil, Azathioprine | **Inhibit lymphocyte proliferation** | Viral (CMV, BK), Bacterial | |
| **mTOR Inhibitors** | Sirolimus, Everolimus | **Inhibit T cell proliferation** | Impaired wound healing, Pneumonitis | |
| **Anti-CD20** | **Rituximab**, Ocrelizumab, Ofatumumab | **B cell depletion** ? **Hypogammaglobulinemia** | Bacterial, Viral (HBV reactivation), **PML** (rare) | |
| **BTK Inhibitors** | Ibrutinib, Acalabrutinib, Zanubrutinib | **B cell signaling inhibition** | Bacterial, Fungal (Aspergillus), Viral | |
| **Anti-TNF** | Infliximab, Adalimumab, Etanercept | **Neutralize TNF-a** | **TB reactivation**, Fungal (histoplasmosis), Bacterial | |
| **JAK Inhibitors** | Tofacitinib, Baricitinib, Upadacitinib | **Inhibit JAK-STAT signaling** | **Herpes zoster**, Bacterial, Opportunistic | |
| **Anti-IL-6R** | Tocilizumab, Sarilumab | **Block IL-6 signaling** | Bacterial, Viral (HSV, VZV) | |
| **Anti-IL-17** | Secukinumab, Ixekizumab | **Block IL-17** | **Mucocutaneous candidiasis** | |
| **Anti-IL-23** | Ustekinumab, Guselkumab | **Block IL-23** | Low infection risk | |

## Opportunistic Infections in Immunocompromised Hosts
| Pathogen | At-Risk Population | Prophylaxis | Treatment | URL |
|----------|---------------------|-------------|-----------|-----|
| **Pneumocystis jirovecii (PJP)** | **Corticosteroids** (>20 mg prednisone x >4 weeks), **T cell defects**, **ALL**, **CLL**, **HSCT** | **TMP-SMX** DS daily or 3x/week; Alternatives: Dapsone, Atovaquone, Pentamidine | **TMP-SMX** (high-dose), **Corticosteroids** (if hypoxic) | https://www.cdc.gov |
| **Toxoplasma gondii** | **HSCT**, **Solid organ transplant**, **HIV** (CD4 <100), **T cell defects** | **TMP-SMX** (same as PJP) | **Pyrimethamine + Sulfadiazine + Leucovorin** | |
| **Cytomegalovirus (CMV)** | **HSCT**, **Solid organ transplant**, **HIV** (CD4 <50) | **Letermovir** (HSCT), **Valganciclovir** (high-risk transplant) | **Valganciclovir**, **Ganciclovir**, **Foscarnet**, **Cidofovir** | |
| **Herpes Simplex Virus (HSV)** | **HSCT**, **Chemotherapy**, **HIV** | **Acyclovir**, **Valacyclovir** | **Acyclovir**, **Valacyclovir**, **Famciclovir** | |
| **Varicella Zoster Virus (VZV)** | **HSCT**, **JAK inhibitors**, **Corticosteroids** | **Acyclovir**, **Valacyclovir** (post-exposure or prolonged) | **Acyclovir**, **Valacyclovir** | |
| **Epstein-Barr Virus (EBV)** | **HSCT**, **Solid organ transplant** | Monitor **EBV PCR**; **Rituximab** preemptively | **Reduce immunosuppression**, **Rituximab** | |
| **BK Virus** | **HSCT** (hemorrhagic cystitis), **Kidney transplant** (nephropathy) | Monitor **BK PCR** (urine/blood) | **Reduce immunosuppression**, **Cidofovir**, **Leflunomide** | |
| **Mycobacterium tuberculosis (TB)** | **Anti-TNF therapy**, **HIV**, **Corticosteroids** | **Screening** (IGRA or TST) before immunosuppression; **Treatment of LTBI** | **RIPE** (Rifampin, Isoniazid, Pyrazinamide, Ethambutol) | https://www.cdc.gov/tb |
| **Nocardia** | **CGD**, **Corticosteroids**, **Solid organ transplant** | **TMP-SMX** (if high-risk) | **TMP-SMX**, **Carbapenems**, **Ceftriaxone**, **Linezolid** | |
| **Cryptococcus neoformans** | **HIV** (CD4 <100), **Corticosteroids**, **Solid organ transplant** | Not routinely recommended | **Amphotericin B + Flucytosine** (induction) ? **Fluconazole** (consolidation/maintenance) | |
| **Histoplasma capsulatum** | **Anti-TNF therapy**, **HIV** (CD4 <150), **Endemic areas** | **Itraconazole** (if high-risk in endemic area) | **Liposomal Amphotericin B** (severe) ? **Itraconazole** | |
| **Candida** | **Neutropenia**, **HSCT**, **ICU** | **Fluconazole** (high-risk HSCT, ICU) | **Echinocandins**, **Fluconazole**, **Amphotericin B** | |
| **Aspergillus** | **Prolonged neutropenia**, **HSCT**, **Corticosteroids** | **Posaconazole**, **Voriconazole** (high-risk HSCT/AML) | **Voriconazole**, **Isavuconazole**, **Liposomal Amphotericin B** | |

## Immunization in Immunocompromised Hosts
| Vaccine Type | Recommendation | URL |
|--------------|----------------|-----|
| **Inactivated Vaccines** | **Generally safe** (may have reduced immunogenicity); **Administer =2 weeks before immunosuppression** if possible | https://www.cdc.gov/vaccines |
| **Live Vaccines** | **CONTRAINDICATED** during significant immunosuppression; Administer =4 weeks before immunosuppression | |
| **Influenza** | **Inactivated** (annual); **Avoid live attenuated (LAIV)** | |
| **Pneumococcal** | **PCV20** or **PCV15 + PPSV23** | |
| **Herpes Zoster** | **Recombinant zoster vaccine (RZV, Shingrix)**; **Avoid live zoster vaccine (ZVL)** | |
| **HPV** | Recommended through age 26 (shared decision-making 27-45) | |
| **Hepatitis B** | Check serologies; Vaccinate if non-immune | |
| **COVID-19** | **mRNA or protein subunit**; Follow current guidelines | |
| **Post-HSCT Revaccination** | Begin **6-12 months** post-transplant; Follow specific schedules | https://www.cdc.gov/vaccines/hcp/acip-recs |
