# Immunology - Hypersensitivity Reactions

## Gell and Coombs Classification
| Type | Alternative Name | Mechanism | Onset | Examples | URL |
|------|------------------|-----------|-------|----------|-----|
| **Type I** | **Immediate** | **IgE-mediated** mast cell/basophil degranulation | Minutes | Anaphylaxis, Allergic rhinitis, Asthma, Urticaria, Angioedema | https://www.aaaai.org |
| **Type II** | **Cytotoxic** | **IgG/IgM against cell surface antigens** ? Complement/FcR-mediated destruction | Hours to days | AIHA, ITP, Goodpasture syndrome, Pemphigus vulgaris, HTR | |
| **Type III** | **Immune Complex** | **Antigen-antibody complexes** deposit in tissues ? Complement/FcR activation | 1-3 weeks | Serum sickness, Arthus reaction, SLE, Post-streptococcal GN, HSP/IgA vasculitis | |
| **Type IV** | **Delayed-Type** | **T cell-mediated** (CD4+ Th1/Th17 or CD8+ cytotoxic) | 2-3 days | Contact dermatitis, PPD test, Celiac disease, Drug rash, GVHD | |

## Type I Hypersensitivity (Immediate)
| Phase | Mediators | Clinical Effects | URL |
|-------|-----------|------------------|-----|
| **Sensitization** | Th2 cells ? IL-4, IL-13 ? **IgE class switching** ? IgE binds **FceRI** on mast cells/basophils | Asymptomatic | https://www.aaaai.org |
| **Early Phase (Minutes)** | Cross-linking of IgE ? **Histamine**, **Tryptase**, **Leukotrienes**, **Prostaglandins** | Vasodilation, Bronchoconstriction, Edema, Pruritus | |
| **Late Phase (4-6 Hours)** | **Eosinophils**, **Neutrophils**, **Th2 cytokines** (IL-4, IL-5, IL-13) | Persistent inflammation, Tissue remodeling | |

## Anaphylaxis - Diagnostic Criteria
| Criterion | Description | URL |
|-----------|-------------|-----|
| **1. Acute onset** (minutes-hours) with **skin/mucosal involvement** (urticaria, Angioedema, Flushing) **AND** =1 of: | Respiratory compromise (dyspnea, Wheeze, Stridor, Hypoxia) OR Hypotension OR End-organ dysfunction | https://www.aaaai.org/anaphylaxis |
| **2. =2 of the following** after exposure to a **likely allergen**: | Skin/mucosal involvement, Respiratory compromise, Hypotension, Persistent GI symptoms (cramping, Vomiting) | |
| **3. Hypotension** after exposure to a **known allergen**: | Adults: SBP <90 mmHg or >30% decrease; Children: Age-specific low BP | |

## Anaphylaxis - Management
| Step | Intervention | URL |
|------|--------------|-----|
| **1. Epinephrine** | **IM epinephrine 0.3-0.5 mg (1:1000) into anterolateral thigh**; Repeat q5-15min as needed; **DO NOT DELAY** | https://www.aaaai.org/anaphylaxis |
| **2. Positioning** | **Supine with legs elevated** (if tolerated); **Avoid upright** (risk of empty ventricle syndrome) | |
| **3. Oxygen** | High-flow O2 | |
| **4. IV Fluids** | 1-2 L crystalloid (adults); 20-30 mL/kg (children) | |
| **5. Adjunctive** | **H1 antihistamine** (Diphenhydramine 25-50 mg IV), **H2 antihistamine** (Famotidine 20 mg IV), **Corticosteroids** (Methylprednisolone 125 mg IV - no acute benefit, May prevent biphasic) | |
| **6. Refractory** | **IV epinephrine infusion**, **Glucagon** (if on beta-blockers) | |
| **7. Biphasic Anaphylaxis** | 1-20% recur within 72 hours; **Observe 4-24 hours** | |

| **Receptor Blockade** | **Myasthenia gravis** (AChR antibodies), **Pemphigus vulgaris** (desmoglein antibodies) | |

## Type III Hypersensitivity (Immune Complex)
| Disease | Antigen | Immune Complex Deposition | URL |
|---------|---------|--------------------------|-----|
| **Serum Sickness** | Foreign protein (e.g., ATG, Rituximab) | Blood vessels, Joints, Kidneys | https://www.rheumatology.org |
| **Arthus Reaction** | Local antigen (e.g., Vaccine) | Local blood vessels | |
| **SLE** | Nuclear antigens (dsDNA, Sm) | **Kidneys** (glomerulonephritis), **Skin**, **Joints**, **Serosa** | |
| **Post-Streptococcal GN** | Streptococcal antigens | **Kidneys** (subepithelial "humps") | |
| **IgA Vasculitis (HSP)** | Unknown (often post-infectious) | **Skin**, **GI**, **Joints**, **Kidneys** (IgA deposition) | |
| **Hypersensitivity Pneumonitis** | Inhaled antigens (mold, Bird proteins) | **Lungs** (alveoli) | |

## Drug Hypersensitivity Reactions
| Reaction | Type | Timing | Clinical Features | Offending Drugs | Management | URL |
|----------|------|--------|-------------------|-----------------|------------|-----|
| **Anaphylaxis** | I | Minutes-hours | Urticaria, Angioedema, Bronchospasm, Hypotension | Beta-lactams, NSAIDs, Biologics | **Epinephrine** | https://www.aaaai.org/drug-allergy |
| **DRESS** | IVb | 2-6 weeks | **Fever**, **Eosinophilia**, **Rash**, **Organ involvement** (liver, Kidney, Lung, Heart) | **Aromatic anticonvulsants** (phenytoin, Carbamazepine, Phenobarbital), **Allopurinol**, **Sulfonamides**, **Minocycline** | **Stop drug**, **Corticosteroids** | |
| **SJS/TEN** | IVc | 1-4 weeks | **Mucosal erosions**, **Epidermal detachment** (<10% SJS, 10-30% overlap, >30% TEN), **Positive Nikolsky sign** | **Aromatic anticonvulsants**, **Allopurinol**, **Sulfonamides**, **NSAIDs**, **Antibiotics** | **Stop drug**, **Supportive care** (burn unit), **IVIG**, **Cyclosporine**, **Etanercept** | |
| **AGEP** | IVd | 1-2 weeks | **Non-follicular pustules** on erythematous base, Fever, Leukocytosis | **Antibiotics** (aminopenicillins, Macrolides), **CCBs**, **Antimalarials** | **Stop drug**, **Supportive care**, **Topical steroids** | |
| **Fixed Drug Eruption** | Unknown | 30 min - 8 hours | **Well-demarcated**, **Round**, **Erythematous/violaceous plaque**; **Recurs at same site** | **TMP-SMX**, **NSAIDs**, **Tetracyclines**, **Acetaminophen** | **Stop drug**, **Topical steroids** | |
