## Alloantigen Recognition Pathways
| Pathway | Mechanism | T Cells Activated | Role in Rejection | URL |
|---------|-----------|-------------------|-------------------|-----|
| **Direct Allorecognition** | Recipient T cells recognize **intact donor MHC molecules** on donor APCs | **High frequency** | **Dominant in acute rejection** | https://www.niaid.nih.gov |
| **Indirect Allorecognition** | Recipient APCs process and present **donor peptides** on **self-MHC** | Lower frequency | **Dominant in chronic rejection** | |
| **Semi-Direct Allorecognition** | Recipient APCs acquire donor MHC molecules via trogocytosis | | | |

## Types of Graft Rejection
| Type | Timing | Mechanism | Histology | Prevention/Treatment | URL |
|------|--------|-----------|-----------|----------------------|-----|
| **Hyperacute** | **Minutes to hours** | **Pre-existing anti-donor antibodies** (ABO, HLA) ? Complement activation ? Thrombosis | **Hemorrhage, Thrombosis, Necrosis** | **Crossmatch**, ABO compatibility; **Irreversible** ? Graft removal | https://www.unos.org |
| **Acute Cellular** | Days to weeks (can occur anytime) | **T cell-mediated** (CD4+ and CD8+) ? Infiltration and injury | **Lymphocytic infiltrate**, Tubulitis, Endothelialitis | **Corticosteroids**, **Thymoglobulin**, **Calcineurin inhibitors** | |
| **Acute Antibody-Mediated (ABMR)** | Days to weeks | **Donor-specific antibodies (DSA)** ? Complement activation, ADCC | **C4d deposition**, **Microvascular inflammation** (glomerulitis, Peritubular capillaritis) | **Plasmapheresis**, **IVIG**, **Rituximab**, **Eculizumab**, **Bortezomib** | |
| **Chronic Rejection** | Months to years | **Multifactorial** (indirect allorecognition, DSA, Ischemia, CNI toxicity) | **Fibrosis**, **Graft vasculopathy** (intimal hyperplasia), Glomerulosclerosis | **No effective treatment**; Optimize immunosuppression, Manage risk factors | |

| Class | Examples | Mechanism | Key Side Effects | URL |
|-------|----------|-----------|------------------|-----|

| **Antimetabolites** | **Mycophenolate mofetil (MMF)**, Azathioprine | Inhibit purine synthesis ? ? Lymphocyte proliferation | **GI intolerance**, **Bone marrow suppression**, **Teratogenicity** | |
| **mTOR Inhibitors** | **Sirolimus**, Everolimus | Inhibit mTOR ? ? T cell proliferation | **Impaired wound healing**, **Hyperlipidemia**, **Proteinuria**, **Pneumonitis**, **Mouth ulcers** | |
| **Corticosteroids** | Prednisone, Methylprednisolone | Multiple (? cytokines, ? adhesion molecules, ? T cell activation) | **Hyperglycemia**, **Osteoporosis**, **Weight gain**, **Cataracts**, **Avascular necrosis** | |
| **Co-Stimulation Blockers** | **Belatacept** (CTLA4-Ig) | Blocks CD28-CD80/86 co-stimulation | **Less nephrotoxicity**, **Higher acute rejection** (early), **PTLD** | |
| **Anti-CD25** | **Basiliximab**, Daclizumab | Blocks IL-2 receptor (CD25) on activated T cells | **Induction therapy**; Well-tolerated | |
| **Anti-CD52** | **Alemtuzumab** | Depletes CD52+ cells (T cells, B cells, NK cells, Monocytes) | **Prolonged lymphopenia**, **Autoimmunity**, **Infection** | |
| **Polyclonal Antibodies** | **Anti-thymocyte globulin (ATG)** | Depletes T cells | **Cytokine release syndrome**, **Thrombocytopenia**, **Serum sickness** | |

| **Proteasome Inhibitors** | **Bortezomib**, Carfilzomib | Depletes plasma cells (reduces DSA) | **Peripheral neuropathy**, **Thrombocytopenia** | |

## Hematopoietic Stem Cell Transplantation (HSCT) Immunology
| Component | Description | URL |
|-----------|-------------|-----|
| **Graft-versus-Host Disease (GVHD)** | **Donor T cells attack recipient tissues** | https://www.cibmtr.org |
| **Graft-versus-Leukemia (GVL)** | **Donor T cells attack residual malignant cells** (beneficial) | |
| **Conditioning Regimens** | **Myeloablative** (maximize tumor kill, Higher toxicity) vs **Reduced Intensity** (lower toxicity, Relies on GVL) | |
| **HLA Matching** | **10/10** (HLA-A, B, C, DRB1, DQB1) or **8/8** (A, B, C, DRB1) | |
| **GVHD Prophylaxis** | **Tacrolimus + Methotrexate**; **Post-transplant cyclophosphamide** (haploidentical) | |
| **Acute GVHD** | **Skin** (rash), **GI** (diarrhea), **Liver** (cholestasis) | |
| **Chronic GVHD** | **Skin** (scleroderma), **GI** (dysphagia), **Lung** (BOS), **Eyes** (dry eyes) | |

## Crossmatch Testing
| Test | Method | Detects | Clinical Significance | URL |
|------|--------|---------|----------------------|-----|
| **Complement-Dependent Cytotoxicity (CDC) Crossmatch** | Recipient serum + Donor lymphocytes + Complement ? Lysis | **Pre-existing anti-donor HLA antibodies** | **Positive = Contraindication to transplant** (hyperacute rejection risk) | https://www.unos.org |
| **Flow Cytometry Crossmatch** | Recipient serum + Donor lymphocytes + Fluorescent anti-IgG | **Anti-donor HLA antibodies** (more sensitive than CDC) | **Positive** ? Increased rejection risk; May proceed with desensitization | |
| **Virtual Crossmatch** | Recipient HLA antibody profile vs. Donor HLA typing | **Predicted compatibility** | Used for organ allocation | |
