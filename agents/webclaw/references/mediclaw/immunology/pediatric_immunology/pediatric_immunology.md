# Immunology - Pediatric Immunology

## Development of the Immune System
| Age | Immune Milestones | Clinical Implications | URL |
|-----|-------------------|----------------------|-----|
| **Birth** | **Maternal IgG** (placental transfer), **Immature T cells**, **Low complement activity**, **Impaired polysaccharide response** | **Protection against encapsulated bacteria limited**; **Susceptibility to viral and encapsulated bacterial infections** | https://www.aap.org |
| **2-6 months** | **Maternal IgG wanes**, **Naive immune system** | **Physiologic nadir of IgG** (~400 mg/dL at 3-6 months); **Increased infection risk** | |
| **6-12 months** | **Endogenous IgG production increases**, **IgM near adult levels** | **Improved response to protein antigens** | |
| **1-2 years** | **IgA production increases**, **Response to polysaccharide antigens remains poor** | **Unable to respond to polysaccharide vaccines** (PPSV23 ineffective) | |
| **2-5 years** | **IgG, IgA reach near-adult levels**, **Polysaccharide response improves** | **Can respond to conjugate vaccines** | |
| **5-10 years** | **Full polysaccharide response** | **PPSV23 effective** | |
| **Adolescence** | **Adult immune function** | | |

## Approach to the Child with Recurrent Infections
| Feature | Normal Child | Immunodeficiency | URL |
|---------|--------------|------------------|-----|
| **Frequency** | 6-8 URIs/year (more in daycare) | **Persistent, Severe, Unusual** | https://www.aaaai.org/recurrent-infections |
| **Pathogens** | Common respiratory viruses | **Opportunistic**, **Invasive encapsulated bacteria**, **Fungi** | |
| **Response to Antibiotics** | Good | **Poor**, **Prolonged courses needed** | |
| **Growth** | Normal | **Failure to thrive** | |
| **Family History** | Usually negative | **Positive for PID**, **Consanguinity** | |
| **Physical Exam** | Normal | **Absent tonsils** (XLA), **Dysmorphic features**, **Eczema**, **Petechiae**, **Lymphadenopathy/Hepatosplenomegaly** | |

## Evaluation of Suspected PID in Children
| Test | Indication | Interpretation | URL |
|------|------------|----------------|-----|
| **CBC with Differential** | All | **Lymphopenia** (SCID), **Neutropenia** (congenital neutropenia), **Thrombocytopenia with small platelets** (WAS) | https://primaryimmune.org |
| **Immunoglobulin Levels (IgG, IgA, IgM, IgE)** | All | **Low IgG/A/M** (CVID, XLA), **Elevated IgE** (Hyper-IgE syndrome), **Elevated IgM** (Hyper-IgM) | |
| **Vaccine Responses** | Age =6 months | **Impaired response to tetanus/diphtheria** (T cell defect), **Impaired response to pneumococcal** (B cell/Asplenia) | |
| **Lymphocyte Subsets (CD3, CD4, CD8, CD19, CD16/56)** | All | **Low T cells** (SCID, DiGeorge), **Low B cells** (XLA), **Low NK cells** | |
| **TREC/KREC** | Newborn screening | **Low TREC** (SCID), **Low KREC** (B cell defects) | |
| **DHR Test** | Recurrent catalase-positive infections, Granulomas | **Abnormal** (CGD) | |
| **CH50** | Recurrent Neisseria, Autoimmunity | **Low/absent** (Complement deficiency) | |
| **HIV Testing** | Risk factors | **Positive** | https://www.hiv.gov |

## Kawasaki Disease
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | **Acute, Self-limited vasculitis** of medium-sized arteries, primarily affecting children <5 years | https://www.heart.org/kawasaki |
| Pathophysiology | Unknown trigger (likely infectious) in genetically susceptible host ? **Immune activation** ? **Vasculitis** | |
| Diagnostic Criteria | **Fever =5 days** + **=4 of 5**: 1. Bilateral conjunctival injection, 2. Oral mucous membrane changes, 3. Polymorphous rash, 4. Cervical lymphadenopathy (>1.5 cm), 5. Extremity changes (erythema/edema ? desquamation) | |
| Incomplete Kawasaki | Fever =5 days + <4 criteria + **Laboratory/echo findings** (CRP =3, ESR =40, Albumin =3, Anemia, Elevated ALT, Sterile pyuria, Coronary artery abnormalities) | |
| Complications | **Coronary artery aneurysms** (25% untreated, 5% treated), Myocarditis, Pericarditis, Valvular regurgitation, Arrhythmias | |
| Treatment | **IVIG 2 g/kg x 1** (within 10 days of fever onset) + **High-dose aspirin** (80-100 mg/kg/day divided QID) until afebrile, Then **low-dose aspirin** (3-5 mg/kg/day) for 6-8 weeks | |
| Refractory Kawasaki | **Second dose IVIG**, **Corticosteroids**, **Infliximab**, **Cyclosporine**, **Anakinra** | |

## Multisystem Inflammatory Syndrome in Children (MIS-C)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | **Hyperinflammatory syndrome** occurring 2-6 weeks after **SARS-CoV-2 infection** in children | https://www.cdc.gov/mis-c |
| Clinical Features | **Fever =3 days**, **Elevated inflammatory markers**, **Multi-organ involvement** (cardiac, GI, Mucocutaneous, Neurologic, Hematologic), **No alternative diagnosis** | |
| Laboratory | **Elevated CRP, ESR, Ferritin, D-dimer**, **Lymphopenia**, **Thrombocytopenia**, **Elevated BNP/Troponin** | |
| Diagnosis | **Positive SARS-CoV-2 PCR or serology** OR **COVID-19 exposure within 4 weeks** | |
| Treatment | **IVIG 2 g/kg**, **Corticosteroids**, **Biologics** (anakinra, Infliximab) for refractory; **Antiplatelet/Anticoagulation** | |
