# Immunology - Overview

## What is Immunology?
Immunology is the study of the immune system, including its physiological functioning, pathological conditions (autoimmunity, immunodeficiency, hypersensitivity), and therapeutic applications (vaccines, immunotherapy).

## Components of the Immune System
| Component | Primary Function |
|-----------|------------------|
| **Innate Immunity** | Immediate, Non-specific defense (barriers, Phagocytes, NK cells, Complement) |
| **Adaptive Immunity** | Delayed, Antigen-specific response (B cells, T cells, Antibodies) |
| **Humoral Immunity** | Antibody-mediated (B cells, Plasma cells) |
| **Cell-Mediated Immunity** | T cell-mediated (CD4+ helper, CD8+ cytotoxic) |
