# Immunology - Allergic Disorders

## Allergic Rhinitis
| Component | Description | URL |
|-----------|-------------|-----|
| Classification | **Intermittent** (<4 days/week or <4 weeks/year) vs **Persistent** (=4 days/week and =4 weeks/year); **Mild** vs **Moderate-Severe** | https://www.aaaai.org/allergic-rhinitis |
| Pathophysiology | Allergen exposure ? IgE cross-linking on mast cells ? **Histamine, Leukotrienes, Prostaglandins** ? **Early phase** (sneezing, Rhinorrhea, Pruritus) and **Late phase** (nasal congestion, Hyperreactivity) | |
| Common Allergens | **Seasonal**: Tree pollen (spring), Grass pollen (summer), Weed pollen (fall), Mold spores; **Perennial**: Dust mites, Cockroach, Animal dander, Mold | |
| Diagnosis | Clinical history + **Skin prick testing** or **Serum specific IgE** | |
| Non-Pharmacologic | **Allergen avoidance** (HEPA filters, Dust mite covers, Pet restrictions) | |
| Pharmacologic | **Intranasal corticosteroids** (first-line for persistent/mod-severe), **Oral/Intranasal antihistamines** (second-generation preferred), **Leukotriene receptor antagonists** (montelukast), **Intranasal antihistamines** (azelastine), **Intranasal cromolyn**, **Oral decongestants** (short-term) | |
| Immunotherapy | **Subcutaneous immunotherapy (SCIT)** or **Sublingual immunotherapy (SLIT)** for moderate-severe, Refractory disease | |

## Asthma (Allergic and Non-Allergic)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Chronic inflammatory airway disease characterized by **reversible airflow obstruction**, **Airway hyperresponsiveness**, and **Airway inflammation** | https://www.aaaai.org/asthma |
| Pathophysiology | **Th2-high**: IL-4, IL-5, IL-13 ? **Eosinophilic inflammation**, **IgE production**; **Th2-low**: Neutrophilic, Paucigranulocytic | |
| Diagnosis | **Spirometry**: **FEV1/FVC <0.70** with **=12% and =200 mL improvement in FEV1** after bronchodilator; **Bronchoprovocation testing** (methacholine challenge); **Peak flow variability** | |
| Classification | **Intermittent**: =2 days/week symptoms, =2 nights/month, Normal FEV1; **Mild Persistent**: >2 days/week but not daily, 3-4 nights/month; **Moderate Persistent**: Daily symptoms, >1 night/week; **Severe Persistent**: Continuous symptoms, Frequent nights | |
| Stepwise Management | **Step 1**: PRN SABA; **Step 2**: Low-dose ICS; **Step 3**: Low-dose ICS/LABA; **Step 4**: Medium-dose ICS/LABA; **Step 5**: High-dose ICS/LABA + LAMA; **Step 6**: High-dose ICS/LABA + LAMA + Oral steroids/Biologics | https://www.nhlbi.nih.gov/asthma-guidelines |
| Biologics (Severe Asthma) | **Omalizumab** (anti-IgE), **Mepolizumab, Reslizumab** (anti-IL-5), **Benralizumab** (anti-IL-5Ra), **Dupilumab** (anti-IL-4Ra), **Tezepelumab** (anti-TSLP) | |

## Atopic Dermatitis (Eczema)
| Component | Description | URL |
|-----------|-------------|-----|
| Pathophysiology | **Skin barrier dysfunction** (filaggrin mutations), **Immune dysregulation** (Th2/Th22-predominant inflammation), **Microbiome alterations** (S. aureus colonization) | https://nationaleczema.org |
| Clinical Features | **Pruritic, Erythematous, Scaly plaques**; **Infantile** (face, Extensor surfaces), **Childhood** (flexural), **Adult** (flexural, Head/neck, Hands) | |
| Diagnosis | **Clinical** (Hanifin-Rajka criteria); **Rule out other causes** (contact dermatitis, Scabies, Psoriasis, Cutaneous T-cell lymphoma) | |
| Management | **Emollients** (liberal use), **Topical corticosteroids** (first-line for flares), **Topical calcineurin inhibitors** (tacrolimus, Pimecrolimus - steroid-sparing), **Topical PDE4 inhibitor** (crisaborole), **Phototherapy** (narrowband UVB), **Systemic immunosuppressants** (cyclosporine, Methotrexate, Mycophenolate), **Biologics** (dupilumab, Tralokinumab), **JAK inhibitors** (upadacitinib, Abrocitinib) | |

## Urticaria (Hives) and Angioedema
| Type | Features | Management | URL |
|------|----------|------------|-----|
| **Acute Urticaria** | <6 weeks duration; Often triggered by infections, Foods, Drugs | **Second-generation H1 antihistamines** (cetirizine, Loratadine, Fexofenadine); Avoid triggers | https://www.aaaai.org/urticaria |
| **Chronic Spontaneous Urticaria (CSU)** | =6 weeks duration; No identifiable trigger; Often autoimmune (anti-FceRI, Anti-IgE) | **Second-generation H1 antihistamines** (up to 4x dose); **Omalizumab** (anti-IgE); **Cyclosporine** | |
| **Chronic Inducible Urticaria** | Triggered by specific stimuli (dermatographism, Cold, Heat, Solar, Pressure, Vibratory, Cholinergic, Aquagenic) | **Avoid triggers**, **Second-generation H1 antihistamines** | |
| **Angioedema (Histamine-Mediated)** | Associated with urticaria; Rapid onset (minutes-hours); Responds to antihistamines, Steroids, Epinephrine | **Same as urticaria**; **Epinephrine** if severe | |
| **Angioedema (Bradykinin-Mediated)** | **NO urticaria**; Slower onset (hours-days); **ACE inhibitor-induced** or **Hereditary (C1-INH deficiency)**; Does NOT respond to antihistamines/steroids/epinephrine | **ACE-I**: Discontinue drug; **HAE**: **C1-INH concentrate**, **Icatibant**, **Ecallantide** | |

## Food Allergy
| Component | Description | URL |
|-----------|-------------|-----|
| IgE-Mediated | **Immediate** (minutes-2 hours): Urticaria, Angioedema, Anaphylaxis, Oral allergy syndrome | **Skin prick testing**, **Serum specific IgE**, **Oral food challenge** | https://www.foodallergy.org |
| Non-IgE-Mediated | **Delayed** (hours-days): **Food protein-induced enterocolitis syndrome (FPIES)**, **Food protein-induced allergic proctocolitis (FPIAP)**, **Eosinophilic esophagitis (EoE)** | Clinical history, Elimination diet, Endoscopy (EoE) | |
| Mixed IgE/Non-IgE | **Atopic dermatitis**, **Eosinophilic gastroenteritis** | | |
| Common Allergens | **Children**: Milk, Egg, Peanut, Tree nuts, Soy, Wheat, Fish, Shellfish; **Adults**: Peanut, Tree nuts, Fish, Shellfish, Fruits/Vegetables (oral allergy syndrome) | | |
| Management | **Strict avoidance**; **Epinephrine auto-injector** for IgE-mediated; **Emergency action plan**; **Oral immunotherapy (OIT)** for peanut (Palforzia), Milk, Egg; **Anti-IgE (omalizumab)** for multiple food allergies | |
| Oral Allergy Syndrome | **Pollen-food allergy syndrome**: Raw fruits/vegetables ? Oropharyngeal pruritus, Mild angioedema; Due to cross-reactive pollen allergens (e.g., Birch pollen ? Apple, Peach, Carrot); **Cooked forms tolerated** | |

## Drug Allergy
| Component | Description | URL |
|-----------|-------------|-----|
| **Penicillin Allergy** | **Most commonly reported drug allergy**; **90-95% are NOT truly allergic**; **Skin testing** (major and minor determinants) + **Graded challenge** | https://www.aaaai.org/drug-allergy |
| **Penicillin Skin Testing** | **Major determinant** (benzylpenicilloyl polylysine/Pre-Pen), **Minor determinant mixture** (penicillin G, Penicilloate, Penilloate); **Negative predictive value >99%** | |
| **Cephalosporin Cross-Reactivity** | **~2% cross-reactivity** (historically overestimated); **Low risk** if penicillin skin test negative; **Avoid cephalosporins with similar R1 side chain** (e.g., Ceftriaxone and Cefotaxime) | |
| **Sulfonamide Allergy** | **Antimicrobial sulfonamides** (TMP-SMX) vs **Non-antimicrobial sulfonamides** (furosemide, Celecoxib) - **No cross-reactivity**; **Desensitization** if TMP-SMX required | |
| **NSAID Hypersensitivity** | **Cross-reactive** (COX-1 inhibition ? Pseudoallergic) vs **Selective** (IgE-mediated to single NSAID) | |
| **Radiocontrast Hypersensitivity** | **Pretreatment** (prednisone + Diphenhydramine ｱ Ephedrine) for prior reactions; **Low-osmolality contrast** | |
