## Intravenous Immunoglobulin (IVIG)
| Component | Description | URL |
|-----------|-------------|-----|
| Composition | Pooled IgG from **thousands of healthy donors** (>95% IgG, Trace IgA, IgM) | https://www.aaaai.org/ivig |
| Mechanisms | **Fc receptor blockade** (ITP), **Neutralization of autoantibodies**, **Inhibition of complement deposition**, **Anti-idiotypic antibodies**, **Modulation of cytokine production**, **Enhanced Treg function** | |
| FDA-Approved Indications | **Primary immunodeficiency** (CVID, XLA), **ITP**, **CIDP**, **Kawasaki disease**, **Multifocal motor neuropathy**, **Guillain-Barré syndrome**, **Myasthenia gravis** | |
| Off-Label Uses | **Autoimmune hemolytic anemia**, **Dermatomyositis**, **Stiff person syndrome**, **Necrotizing autoimmune myopathy**, **Toxic epidermal necrolysis (TEN)**, **Hemophagocytic lymphohistiocytosis (HLH)** | |
| Dosing | **PID**: 400-600 mg/kg q3-4 weeks; **Autoimmune/Inflammatory**: **2 g/kg** divided over 2-5 days (often 1 g/kg/day x 2 days) | |
| Adverse Effects | **Headache** (common), **Aseptic meningitis** (rare), **Thrombosis** (risk factors: age >65, Immobility, Hyperviscosity), **Acute kidney injury** (sucrose-containing preparations), **Hemolytic anemia** (due to anti-A/B antibodies), **Anaphylaxis** (IgA deficiency with anti-IgA) | |

## IVIG vs SCIG
| Feature | IVIG | SCIG | URL |
|---------|------|------|-----|
| **Route** | Intravenous (q3-4 weeks) | Subcutaneous (weekly or more frequent) | https://primaryimmune.org |
| **Peak/Trough Levels** | High peak, Low trough | **Stable levels** (less fluctuation) | |
| **Systemic Side Effects** | More common (headache, Flu-like) | **Less common** | |
| **Local Side Effects** | Rare | **Common** (swelling, Erythema, Pruritus at infusion site) | |
| **Convenience** | Requires IV access, Infusion center | **Self-administered at home** | |
| **Dosing** | 400-600 mg/kg q3-4 weeks (PID) | 100-200 mg/kg weekly (PID) | |
| **Contraindications** | **IgA deficiency with anti-IgA antibodies** (use IgA-depleted product) | Same | |
