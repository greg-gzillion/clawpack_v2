# Immunology - Innate Immunity

## Innate vs Adaptive Immunity
| Feature | Innate Immunity | Adaptive Immunity | URL |
|---------|----------------|-------------------|-----|
| **Response Time** | Minutes to hours | Days to weeks | https://www.niaid.nih.gov |
| **Specificity** | Limited (PAMPs) | Highly specific (antigens) | |
| **Memory** | None | **Immunologic memory** | |
| **Components** | Barriers, Phagocytes, NK cells, Complement, Cytokines | B cells, T cells, Antibodies | |
| **Receptors** | Pattern Recognition Receptors (PRRs) | BCR, TCR | |
