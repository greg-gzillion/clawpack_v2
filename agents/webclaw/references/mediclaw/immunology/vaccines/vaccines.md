# Immunology - Vaccines

## Types of Vaccines
| Type | Mechanism | Examples | Advantages | Disadvantages | URL |
|------|-----------|----------|------------|---------------|-----|
| **Live Attenuated** | Weakened pathogen replicates in host ? Induces robust cellular and humoral immunity | **MMR**, **Varicella**, **Rotavirus**, **Intranasal influenza (LAIV)**, **Yellow fever**, **Oral polio (OPV)** | **Strong, Long-lasting immunity**; Single dose often sufficient | **Contraindicated in immunocompromised**, Pregnancy; Risk of reversion to virulence (rare) | https://www.cdc.gov/vaccines |
| **Inactivated/Killed** | Pathogen killed by heat or chemicals ? Cannot replicate | **Inactivated polio (IPV)**, **Hepatitis A**, **Rabies**, **Whole-cell pertussis (wP)** | **Safe in immunocompromised**; Stable | **Weaker immune response**; Requires multiple doses and boosters; **Th2-biased response** | |
| **Subunit/Conjugate** | Purified antigens (proteins, Polysaccharides) | **Hepatitis B** (HBsAg), **HPV** (L1 protein), **Acellular pertussis (aP)** | **Safe**; Well-defined antigens | Weaker immune response; Requires adjuvants and multiple doses | |
| **Polysaccharide** | Purified bacterial capsular polysaccharides ? **T-independent response** | **PPSV23 (Pneumovax)** | Broad serotype coverage | **Poor response in children <2 years**; No memory; Hyporesponsiveness with repeated doses | |
| **Polysaccharide-Conjugate** | Polysaccharide conjugated to protein carrier ? **T-dependent response** | **PCV13, PCV15, PCV20**, **Hib**, **MenACWY**, **MenB** | **Effective in infants**; **Induces memory**; **Reduces carriage** | More expensive; Limited serotype coverage | |
| **Toxoid** | Inactivated bacterial toxins | **Tetanus toxoid**, **Diphtheria toxoid** | Safe; Induces neutralizing antibodies | Requires boosters | |
| **mRNA** | Lipid nanoparticle-encapsulated mRNA encoding viral protein ? Host cells produce antigen | **COVID-19 (Pfizer-BioNTech, Moderna)** | **Rapid development**, **Highly effective**, **Safe** | **Cold chain requirements**, **Reactogenicity** | |
| **Viral Vector** | Replication-incompetent virus delivers gene encoding antigen | **COVID-19 (Janssen, AstraZeneca)**, **Ebola**, **RSV (investigational)** | Strong cellular and humoral immunity | **Pre-existing immunity to vector**, **Thrombosis with thrombocytopenia (VITT)** | |

## Vaccination in Special Populations
| Population | Live Vaccines | Inactivated Vaccines | Special Recommendations | URL |
|------------|---------------|----------------------|------------------------|-----|
| **Pregnancy** | **CONTRAINDICATED** | **Safe** | **Tdap** (27-36 weeks each pregnancy), **Influenza** (inactivated), **COVID-19**, **RSV** (32-36 weeks, seasonal) | https://www.cdc.gov/vaccines/pregnancy |
| **Immunocompromised** | **CONTRAINDICATED** (exceptions: CD4 =200, No other immunosuppression) | **Safe** (may have reduced immunogenicity) | **Avoid live vaccines**; **Pneumococcal**, **Influenza**, **COVID-19**, **Shingrix** (=19 years) | https://www.cdc.gov/vaccines/hcp/acip-recs |
| **Asplenia** | Safe | Safe | **Pneumococcal**, **Meningococcal ACWY and B**, **Hib**, **Influenza** | |
| **HSCT Recipients** | **Contraindicated for =2 years** | **Revaccinate** starting 6-12 months post-HSCT | Follow specific revaccination schedule | |
| **HIV** | **CD4 <200**: CONTRAINDICATED; **CD4 =200**: Generally safe (except LAIV) | Safe | **Pneumococcal**, **Influenza**, **Hepatitis B**, **HPV**, **Meningococcal**, **Shingrix** (=19 years) | |
| **Chronic Liver Disease** | Safe | Safe | **Hepatitis A**, **Hepatitis B**, **Pneumococcal**, **Influenza** | |
| **Chronic Kidney Disease** | Safe | Safe | **Hepatitis B** (high-dose), **Pneumococcal**, **Influenza** | |

## Vaccine Adverse Events
| Adverse Event | Associated Vaccines | Management | URL |
|---------------|---------------------|------------|-----|
| **Anaphylaxis** | Any (rare: ~1 per million doses) | **Epinephrine**; Observe 15-30 min post-vaccination | https://www.cdc.gov/vaccinesafety |
| **Syncope (Vasovagal)** | Any (especially adolescents) | **Supine position**, Recovery; **Observe 15 min** | |
| **Local Reactions** | Most | Ice, Analgesics | |
| **Fever** | Most (especially infants) | Antipyretics | |
| **Febrile Seizures** | MMR, Varicella, DTaP | **Antipyretics** for fever; Reassurance | |
| **Thrombocytopenia** | MMR (transient, Mild) | Self-limited | |
| **Arthus Reaction** | Tetanus toxoid (with frequent boosters) | Avoid over-immunization | |
| **Guillain-Barré Syndrome** | Influenza (1-2 per million), Tetanus toxoid | **Not a contraindication** to future vaccination (except if GBS within 6 weeks of prior dose) | |
| **Thrombosis with Thrombocytopenia (VITT)** | **Adenoviral vector COVID-19 vaccines** (Janssen, AstraZeneca) | **Alternative vaccine preferred**; **Avoid heparin** (if occurs), **IVIG**, **Non-heparin anticoagulant** | |
