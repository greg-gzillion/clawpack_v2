# Immunology - Tumor Immunology and Immunotherapy

## Cancer Immunoediting
| Phase | Description | URL |
|-------|-------------|-----|
| **Elimination** | Immune system (CD8+ T cells, NK cells, IFN-?) recognizes and destroys transformed cells | https://www.cancer.gov/immunotherapy |
| **Equilibrium** | Immune system controls tumor growth but does not eliminate; **Immunologic sculpting** (selection of less immunogenic clones) | |
| **Escape** | Tumor evades immune system ? Clinical progression | |

## Mechanisms of Tumor Immune Evasion
| Mechanism | Description | Examples | URL |
|-----------|-------------|----------|-----|
| **Loss of Antigen Presentation** | Downregulation of MHC class I, TAP, ﾟ2-microglobulin | Many cancers | https://www.cancer.gov |
| **Immune Checkpoint Activation** | Upregulation of **PD-L1**, **CTLA-4 ligands**, **LAG-3 ligands** | PD-L1 expression in melanoma, NSCLC, RCC | |
| **Treg Recruitment** | Tumor secretes CCL22 ? Recruits Tregs ? Immune suppression | Many cancers | |
| **Myeloid-Derived Suppressor Cells (MDSCs)** | Immature myeloid cells ? Suppress T cell and NK cell function | Many cancers | |
| **Immunosuppressive Cytokines** | **TGF-ﾟ**, **IL-10**, **VEGF** | Many cancers | |
| **Tumor-Associated Macrophages (TAMs)** | **M2 phenotype** ? Promote tumor growth, Angiogenesis, Immune suppression | Many cancers | |
| **Indoleamine 2,3-Dioxygenase (IDO)** | Depletes tryptophan ? T cell suppression | Many cancers | |

## Immune-Related Adverse Events (irAEs)
| irAE | Incidence | Management | URL |
|------|-----------|------------|-----|
| **Rash/Pruritus** | 30-50% | **Topical steroids**, Antihistamines; **Hold ICI** if severe; Systemic steroids if Grade 3-4 | https://www.asco.org/immunotherapy |
| **Colitis/Diarrhea** | 10-30% (higher with CTLA-4) | **Corticosteroids** (methylprednisolone 1-2 mg/kg/day); **Infliximab** if refractory; **Rule out infection** | |
| **Hepatitis** | 5-15% | **Corticosteroids**; **Mycophenolate** if refractory; **Avoid infliximab** (hepatotoxic) | |
| **Pneumonitis** | 2-10% (higher with PD-1, Combination) | **Corticosteroids** (high-dose); **Infliximab**, **Cyclophosphamide**, **IVIG** if refractory; **Hold ICI** | |
| **Hypophysitis** | 5-10% (higher with CTLA-4) | **Hormone replacement** (levothyroxine, Hydrocortisone, Testosterone/Estrogen); **High-dose steroids** if severe (mass effect) | |
| **Thyroiditis** | 5-15% | **Levothyroxine** for hypothyroidism; **Beta-blockers** for hyperthyroid phase; **Steroids rarely needed** | |
| **Myocarditis** | <1% (high mortality) | **High-dose corticosteroids**, **Abatacept**, **Anti-thymocyte globulin**, **Mycophenolate**; **Permanent discontinuation** | |
| **Arthralgia/Myalgia** | 10-20% | **NSAIDs**, **Low-dose corticosteroids**; **Hydroxychloroquine**, **Methotrexate** if persistent | |

## irAE Management by Grade
| Grade | Definition | Management | URL |
|-------|------------|------------|-----|
| **Grade 1** | Mild, Asymptomatic | **Continue ICI**, Symptomatic treatment | https://www.nccn.org/immunotherapy-toxicity |
| **Grade 2** | Moderate, Limiting ADLs | **Hold ICI**, **Corticosteroids** (0.5-1 mg/kg/day) | |
| **Grade 3** | Severe, Medically significant | **Hold ICI**, **Corticosteroids** (1-2 mg/kg/day), Consider **inpatient** | |
| **Grade 4** | Life-threatening | **Permanent discontinuation**, **High-dose corticosteroids** (methylprednisolone 1-2 mg/kg/day or pulse), **Additional immunosuppression** | |

## Other Cancer Immunotherapies
| Therapy | Mechanism | Examples | Indications | URL |
|---------|-----------|----------|-------------|-----|
| **CAR T-Cell Therapy** | Autologous T cells engineered to express **chimeric antigen receptor (CAR)** targeting tumor antigen | **CD19 CAR-T** (Axi-cel, Tisa-cel, Brexu-cel, Liso-cel), **BCMA CAR-T** (Ide-cel, Cilta-cel) | B-ALL, DLBCL, FL, MCL, Multiple myeloma | https://www.cancer.gov/car-t |
| **Bispecific T Cell Engagers (BiTEs)** | Bispecific antibodies that bind **CD3** (T cells) and **tumor antigen** | **Blinatumomab** (CD19 x CD3), **Teclistamab** (BCMA x CD3), **Elranatamab** (BCMA x CD3), **Talquetamab** (GPRC5D x CD3), **Epcoritamab** (CD20 x CD3), **Glofitamab** (CD20 x CD3) | B-ALL, Multiple myeloma, DLBCL | |
| **Antibody-Drug Conjugates (ADCs)** | Monoclonal antibody linked to cytotoxic payload | **Trastuzumab deruxtecan** (HER2), **Trastuzumab emtansine** (HER2), **Brentuximab vedotin** (CD30), **Enfortumab vedotin** (Nectin-4), **Sacituzumab govitecan** (Trop-2) | Breast, Gastric, Hodgkin, Urothelial, TNBC | |
| **Monoclonal Antibodies** | Direct tumor targeting or ADCC | **Rituximab** (CD20), **Daratumumab** (CD38), **Elotuzumab** (SLAMF7), **Cetuximab** (EGFR), **Trastuzumab** (HER2) | Lymphoma, CLL, Myeloma, CRC, HNSCC, Breast | |
| **Oncolytic Viruses** | Genetically modified viruses that selectively replicate in tumor cells | **Talimogene laherparepvec (T-VEC)** | Melanoma | |
| **Cancer Vaccines** | Stimulate immune response against tumor antigens | **Sipuleucel-T** (Provenge) | Prostate cancer | |
