# First Aid - Allergic Reactions and Anaphylaxis

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Allergic Reaction** | **Immune response** to allergen (localized or systemic) | https://www.redcross.org/take-a-class/first-aid/allergic-reactions |
| **Anaphylaxis** | **Severe, life-threatening systemic allergic reaction** | https://www.aaaai.org/conditions-treatments/allergies/anaphylaxis |

## Signs and Symptoms of Anaphylaxis
| System | Symptoms | URL |
|--------|----------|-----|
| **Skin/Mucosa** | **Urticaria** (hives), **Angioedema** (swelling of lips/tongue/face), **Flushing**, **Pruritus** | https://www.aaaai.org/conditions-treatments/allergies/anaphylaxis |
| **Respiratory** | **Dyspnea**, **Wheezing**, **Stridor**, **Hoarseness**, **Throat tightness** | https://www.aaaai.org/conditions-treatments/allergies/anaphylaxis |
| **Cardiovascular** | **Tachycardia**, **Hypotension**, **Syncope**, **Shock** | https://www.aaaai.org/conditions-treatments/allergies/anaphylaxis |
| **Gastrointestinal** | **Nausea**, **Vomiting**, **Diarrhea**, **Abdominal pain** | https://www.aaaai.org/conditions-treatments/allergies/anaphylaxis |

## First Aid for Anaphylaxis
| Step | Action | URL |
|------|--------|-----|
| **1** | **CALL 911** immediately | https://www.redcross.org/take-a-class/first-aid/anaphylaxis |
| **2** | **If victim has epinephrine auto-injector (EpiPen)** : **Administer immediately** into **mid-outer thigh** (through clothing if needed), **Hold for 3 seconds** | https://www.redcross.org/take-a-class/first-aid/anaphylaxis |
| **3** | **Position**: **Supine with legs elevated** (if breathing comfortably), **Sitting upright** (if respiratory distress) | https://www.redcross.org/take-a-class/first-aid/anaphylaxis |
| **4** | **Monitor ABCs**, **Start CPR if needed** | https://www.redcross.org/take-a-class/first-aid/anaphylaxis |
| **5** | **If no improvement in 5-15 minutes**, **Administer second dose** of epinephrine | https://www.redcross.org/take-a-class/first-aid/anaphylaxis |
| **6** | **Do NOT give anything by mouth** | https://www.redcross.org/take-a-class/first-aid/anaphylaxis |

## Epinephrine Auto-Injector (EpiPen) Instructions
| Step | Action | URL |
|------|--------|-----|
| **1** | **Remove safety cap** (blue side to sky, orange to thigh) | https://www.epipen.com |
| **2** | **Hold firmly** with fist (do NOT place thumb over end) | https://www.epipen.com |
| **3** | **Jab firmly** into **mid-outer thigh** (90° angle) | https://www.epipen.com |
| **4** | **Hold for 3 seconds** (count slowly) | https://www.epipen.com |
| **5** | **Remove and massage** injection site | https://www.epipen.com |
| **6** | **Note time** of administration | https://www.epipen.com |
| **7** | **Give used injector to EMS** | https://www.epipen.com |
