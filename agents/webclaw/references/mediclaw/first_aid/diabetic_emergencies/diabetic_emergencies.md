# First Aid - Diabetic Emergencies

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Hypoglycemia** | **Low blood sugar** (<70 mg/dL) | https://www.redcross.org/take-a-class/first-aid/diabetic-emergencies |
| **Hyperglycemia** | **High blood sugar** (usually >250 mg/dL) | https://www.redcross.org/take-a-class/first-aid/diabetic-emergencies |

## Hypoglycemia vs Hyperglycemia
| Feature | Hypoglycemia | Hyperglycemia | URL |
|---------|--------------|---------------|-----|
| **Onset** | **Rapid** (minutes-hours) | **Gradual** (hours-days) | https://www.redcross.org/take-a-class/first-aid/diabetic-emergencies |
| **Skin** | **Pale**, **Cool**, **Clammy** | **Flushed**, **Warm**, **Dry** | https://www.redcross.org/take-a-class/first-aid/diabetic-emergencies |
| **Breathing** | **Normal or Shallow** | **Deep, rapid** (Kussmaul), **Fruity breath** | https://www.redcross.org/take-a-class/first-aid/diabetic-emergencies |
| **Mental Status** | **Irritable**, **Confused**, **Seizures**, **Coma** | **Lethargic**, **Confused**, **Coma** | https://www.redcross.org/take-a-class/first-aid/diabetic-emergencies |
| **Hydration** | **Normal** | **Dehydrated** | https://www.redcross.org/take-a-class/first-aid/diabetic-emergencies |

## Hypoglycemia First Aid
| Step | Action | URL |
|------|--------|-----|
| **1** | **If conscious and able to swallow**: **Give 15-20g fast-acting sugar** (4 oz juice/soda, 1 tbsp sugar/honey, 3-4 glucose tablets) | https://www.redcross.org/take-a-class/first-aid/diabetic-emergencies |
| **2** | **Recheck in 15 minutes**; **Repeat if still symptomatic** | https://www.redcross.org/take-a-class/first-aid/diabetic-emergencies |
| **3** | **Once improved**, **Give snack with protein/complex carbs** | https://www.redcross.org/take-a-class/first-aid/diabetic-emergencies |
| **4** | **If unconscious or unable to swallow**: **CALL 911**, **Place in recovery position**, **Do NOT give anything by mouth** | https://www.redcross.org/take-a-class/first-aid/diabetic-emergencies |

## Hyperglycemia First Aid
| Step | Action | URL |
|------|--------|-----|
| **1** | **If unsure whether hypo or hyperglycemia**: **Give sugar** (if hypoglycemic, Helps; if hyperglycemic, Minimal harm) | https://www.redcross.org/take-a-class/first-aid/diabetic-emergencies |
| **2** | **If known hyperglycemia**: **Encourage sugar-free fluids**, **Assist with insulin** (if victim able), **Seek medical care** | https://www.redcross.org/take-a-class/first-aid/diabetic-emergencies |
| **3** | **If altered mental status/unconscious**: **CALL 911** | https://www.redcross.org/take-a-class/first-aid/diabetic-emergencies |
