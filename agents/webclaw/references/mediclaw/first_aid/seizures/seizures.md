# First Aid - Seizures

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Abnormal, excessive synchronous neuronal activity** in brain | https://www.epilepsy.com/recognition/first-aid |
| **Common Causes** | **Epilepsy**, **Febrile** (children), **Head trauma**, **Hypoglycemia**, **Drug/alcohol withdrawal**, **Stroke**, **Infection** | https://www.epilepsy.com/recognition/first-aid |

## Generalized Tonic-Clonic Seizure First Aid
| Step | Action | URL |
|------|--------|-----|
| **1** | **Stay calm**, **Note time** (seizure start) | https://www.epilepsy.com/recognition/first-aid |
| **2** | **Protect from injury**: **Clear area**, **Place something soft under head**, **Remove glasses** | https://www.epilepsy.com/recognition/first-aid |
| **3** | **Loosen tight clothing** around neck | https://www.epilepsy.com/recognition/first-aid |
| **4** | **Do NOT restrain** or **hold down** | https://www.epilepsy.com/recognition/first-aid |
| **5** | **Do NOT put anything in mouth** | https://www.epilepsy.com/recognition/first-aid |
| **6** | **After seizure stops**: **Place in recovery position**, **Check breathing** | https://www.epilepsy.com/recognition/first-aid |
| **7** | **Stay with victim** until fully alert | https://www.epilepsy.com/recognition/first-aid |

## When to Call 911 for Seizure
| Indication | URL |
|------------|-----|
| **Seizure lasts >5 minutes** | https://www.epilepsy.com/recognition/first-aid |
| **Repeated seizures** without recovery between | https://www.epilepsy.com/recognition/first-aid |
| **First seizure** (no known epilepsy) | https://www.epilepsy.com/recognition/first-aid |
| **Difficulty breathing** after seizure | https://www.epilepsy.com/recognition/first-aid |
| **Seizure in water** | https://www.epilepsy.com/recognition/first-aid |

| **Pregnancy** | https://www.epilepsy.com/recognition/first-aid |
