# First Aid - Temperature-Related Emergencies

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Heat Emergencies** | **Heat cramps**, **Heat exhaustion**, **Heat stroke** (see Environmental Emergencies) | https://www.cdc.gov/niosh/topics/heatstress/heatrelillness.html |
| **Cold Emergencies** | **Frostnip**, **Frostbite**, **Hypothermia** (see Environmental Emergencies) | https://www.cdc.gov/niosh/topics/coldstress/coldrelatedillnesses.html |

## Preventing Heat-Related Emergencies
| Recommendation | URL |
|----------------|-----|
| **Hydrate** (water, sports drinks) | https://www.cdc.gov/niosh/topics/heatstress/heatrecommendations.html |
| **Wear lightweight, light-colored, loose-fitting clothing** | https://www.cdc.gov/niosh/topics/heatstress/heatrecommendations.html |
| **Limit outdoor activity** during hottest part of day | https://www.cdc.gov/niosh/topics/heatstress/heatrecommendations.html |
| **Take frequent breaks** in shade/air conditioning | https://www.cdc.gov/niosh/topics/heatstress/heatrecommendations.html |
| **Never leave children/pets in parked car** | https://www.cdc.gov/niosh/topics/heatstress/heatrecommendations.html |

## Preventing Cold-Related Emergencies
| Recommendation | URL |
|----------------|-----|
| **Dress in layers** | https://www.cdc.gov/niosh/topics/coldstress/recommendations.html |
| **Cover head, ears, hands, feet** | https://www.cdc.gov/niosh/topics/coldstress/recommendations.html |
| **Stay dry** (wet clothing loses insulation) | https://www.cdc.gov/niosh/topics/coldstress/recommendations.html |
| **Avoid alcohol** (increases heat loss) | https://www.cdc.gov/niosh/topics/coldstress/recommendations.html |
