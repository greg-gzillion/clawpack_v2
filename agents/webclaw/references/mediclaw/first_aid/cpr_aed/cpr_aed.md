# First Aid - CPR and AED

## Automated External Defibrillator (AED)
| Component | Description | URL |
|-----------|-------------|-----|
| **Purpose** | **Analyzes heart rhythm** and **delivers shock** to restore normal rhythm in **VF/pulseless VT** | https://cpr.heart.org/en/cpr-courses-and-kits/aed-implementation |
| **Indications** | **Unresponsive**, **Not breathing normally**, **No pulse** | https://cpr.heart.org/en/cpr-courses-and-kits/aed-implementation |
| **Pediatric Pads** | **Use for children <8 years or <55 lbs** (if available); **Adult pads acceptable** if pediatric unavailable | https://cpr.heart.org/en/cpr-courses-and-kits/aed-implementation |

## AED Steps (AHA)
| Step | Action | URL |
|------|--------|-----|
| **1** | **Turn ON AED** (follow voice/visual prompts) | https://cpr.heart.org/en/cpr-courses-and-kits/aed-implementation |
| **2** | **Attach pads** to bare chest (upper right chest, lower left ribs) | https://cpr.heart.org/en/cpr-courses-and-kits/aed-implementation |
| **3** | **Plug in connector** (if required) | https://cpr.heart.org/en/cpr-courses-and-kits/aed-implementation |
| **4** | **Clear victim** (no one touching) during analysis | https://cpr.heart.org/en/cpr-courses-and-kits/aed-implementation |
| **5** | **If shock advised**: **Clear victim** ? **Press shock button** | https://cpr.heart.org/en/cpr-courses-and-kits/aed-implementation |
| **6** | **Immediately resume CPR** (starting with compressions) for 2 minutes | https://cpr.heart.org/en/cpr-courses-and-kits/aed-implementation |
| **7** | **Repeat analysis** every 2 minutes | https://cpr.heart.org/en/cpr-courses-and-kits/aed-implementation |

## Special AED Considerations
| Situation | Action | URL |
|-----------|--------|-----|
| **Water** | **Remove from water**, **Dry chest** before pad placement | https://cpr.heart.org/en/cpr-courses-and-kits/aed-implementation |
| **Medication Patch** | **Remove patch**, **Wipe area** before pad placement | https://cpr.heart.org/en/cpr-courses-and-kits/aed-implementation |
| **Implanted Device (Pacemaker/ICD)** | **Place pad =1 inch away** from device | https://cpr.heart.org/en/cpr-courses-and-kits/aed-implementation |
| **Hairy Chest** | **Shave chest** if excessive hair prevents pad adhesion | https://cpr.heart.org/en/cpr-courses-and-kits/aed-implementation |
| **Pregnancy** | **Use AED as usual** (benefits outweigh risks) | https://cpr.heart.org/en/cpr-courses-and-kits/aed-implementation |

## CPR with Advanced Airway
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **Compressions** | **Continuous** at 100-120/min | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
| **Ventilations** | **1 breath every 6 seconds** (10 breaths/min) | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
| **No Pause for Breaths** | **Asynchronous** ventilations | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
