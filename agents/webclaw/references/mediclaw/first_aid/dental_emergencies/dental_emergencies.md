# First Aid - Dental Emergencies

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **General Principle** | **Control bleeding**, **Manage pain**, **Preserve tooth** (if avulsed), **Seek dental care** | https://www.redcross.org/take-a-class/first-aid/dental-emergencies |

## Specific Dental Emergencies
| Emergency | First Aid | URL |
|-----------|-----------|-----|
| **Toothache** | **Rinse mouth with warm water**, **Floss** (remove debris), **Apply cold compress** (external), **OTC pain reliever**, **See dentist** | https://www.ada.org/resources/community/dental-emergencies |
| **Avulsed (Knocked-Out) Permanent Tooth** | **Hold by crown** (not root), **Rinse gently** (do NOT scrub), **Reimplant in socket** (if possible), **If not, place in milk/saline/saliva**, **See dentist IMMEDIATELY** (<30 min best) | https://www.ada.org/resources/community/dental-emergencies |
| **Avulsed Primary (Baby) Tooth** | **Do NOT reimplant** (damages permanent tooth bud), **Control bleeding**, **See dentist** | https://www.ada.org/resources/community/dental-emergencies |
| **Chipped/Broken Tooth** | **Rinse mouth**, **Save fragments**, **Apply cold compress** (external), **See dentist** | https://www.ada.org/resources/community/dental-emergencies |
| **Bitten Lip/Tongue** | **Apply direct pressure** with gauze, **Cold compress**, **If bleeding persists >15 min**, **Seek medical care** | https://www.redcross.org/take-a-class/first-aid/dental-emergencies |
| **Object Stuck Between Teeth** | **Try to remove with floss**, **Do NOT use sharp objects**, **See dentist** if unable | https://www.ada.org/resources/community/dental-emergencies |
| **Lost Filling/Crown** | **Temporary dental cement** (OTC), **Sugarless gum**, **See dentist** | https://www.ada.org/resources/community/dental-emergencies |
