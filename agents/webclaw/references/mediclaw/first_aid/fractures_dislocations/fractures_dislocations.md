# First Aid - Fractures and Dislocations

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Fracture** | **Break in bone** (closed = skin intact; open = skin broken) | https://www.redcross.org/take-a-class/first-aid/fractures |
| **Dislocation** | **Displacement of bone from joint** | https://www.redcross.org/take-a-class/first-aid/dislocations |

## Signs and Symptoms
| Sign/Symptom | Description | URL |
|--------------|-------------|-----|
| **Pain** | **Localized**, **Worse with movement** | https://www.redcross.org/take-a-class/first-aid/fractures |
| **Swelling/Bruising** | **Rapid onset** | https://www.redcross.org/take-a-class/first-aid/fractures |
| **Deformity** | **Visible angulation**, **Shortening**, **Rotation** | https://www.redcross.org/take-a-class/first-aid/fractures |
| **Loss of Function** | **Inability to move** or **bear weight** | https://www.redcross.org/take-a-class/first-aid/fractures |
| **Crepitus** | **Grating sensation** with movement | https://www.redcross.org/take-a-class/first-aid/fractures |
| **Exposed Bone** | **Open fracture** (emergency) | https://www.redcross.org/take-a-class/first-aid/fractures |

## General Fracture Care
| Step | Action | URL |
|------|--------|-----|
| **1** | **Assess for life-threatening injuries first** (ABCs) | https://www.redcross.org/take-a-class/first-aid/fractures |
| **2** | **Control bleeding** (for open fractures: cover with sterile dressing, Do NOT push bone back in) | https://www.redcross.org/take-a-class/first-aid/fractures |
| **3** | **Immobilize** the injured area (include joints above and below) | https://www.redcross.org/take-a-class/first-aid/fractures |
| **4** | **Check CSM** before and after splinting: **C**irculation (pulses, capillary refill), **S**ensation, **M**ovement | https://www.redcross.org/take-a-class/first-aid/fractures |
| **5** | **Apply ice** (20 min on, 20 min off) to reduce swelling | https://www.redcross.org/take-a-class/first-aid/fractures |
| **6** | **Elevate** extremity above heart level | https://www.redcross.org/take-a-class/first-aid/fractures |
| **7** | **Seek medical care** | https://www.redcross.org/take-a-class/first-aid/fractures |

## Splinting Principles
| Principle | Description | URL |
|-----------|-------------|-----|
| **Immobilize Joints** | **Include joints above and below** fracture | https://www.redcross.org/take-a-class/first-aid/splinting |
| **Pad for Comfort** | **Pad bony prominences** | https://www.redcross.org/take-a-class/first-aid/splinting |
| **Secure Snugly** | **Not too tight** (check CSM after) | https://www.redcross.org/take-a-class/first-aid/splinting |
| **Splint in Position Found** | **Do NOT straighten** angulated fractures (except if no distal pulse) | https://www.redcross.org/take-a-class/first-aid/splinting |
| **Improvised Splints** | **Magazines**, **Cardboard**, **Pillows**, **Blankets**, **Anatomic** (tape to adjacent finger/leg) | https://www.redcross.org/take-a-class/first-aid/splinting |

## Specific Fractures
| Fracture | Special Considerations | URL |
|----------|----------------------|-----|
| **Clavicle** | **Sling and swathe** | https://www.redcross.org/take-a-class/first-aid/fractures |
| **Humerus** | **Sling and swathe**, **Do NOT attempt to straighten** | https://www.redcross.org/take-a-class/first-aid/fractures |
| **Forearm/Wrist** | **Splint in position found**, **Sling** | https://www.redcross.org/take-a-class/first-aid/fractures |
| **Femur** | **Traction splint** (if trained), **Otherwise immobilize as is**, **Call 911** | https://www.redcross.org/take-a-class/first-aid/fractures |
| **Pelvis** | **Do NOT move**, **Call 911**, **Treat for shock** | https://www.redcross.org/take-a-class/first-aid/fractures |
| **Ribs** | **Encourage deep breathing** (prevent pneumonia), **Do NOT wrap tightly** | https://www.redcross.org/take-a-class/first-aid/fractures |
