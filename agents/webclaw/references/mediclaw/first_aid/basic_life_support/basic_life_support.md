## Adult BLS Algorithm (AHA 2020)
| Step | Action | URL |
|------|--------|-----|
| **1** | **Verify scene safety** | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
| **2** | **Check responsiveness**: "Are you okay?" Shout, Tap shoulder | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
| **3** | **If unresponsive**: **Call 911** (or send someone), **Get AED** | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
| **4** | **Check breathing and pulse simultaneously** (5-10 seconds) | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
| **5** | **If no breathing/pulse**: **Start CPR** (30 compressions : 2 breaths) | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
| **6** | **Use AED as soon as available** | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |

## High-Quality CPR Components (AHA 2020)
| Component | Adult | Child (1-puberty) | Infant (<1 year) | URL |
|-----------|-------|------------------|------------------|-----|
| **Compression Depth** | **=2 inches (5 cm)** | **=1/3 AP diameter** (~2 inches) | **=1/3 AP diameter** (~1.5 inches) | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
| **Compression Rate** | **100-120/min** | **100-120/min** | **100-120/min** | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
| **Compression:Ventilation Ratio** | **30:2** (1 rescuer), **30:2** (2 rescuers) | **30:2** (1 rescuer), **15:2** (2 rescuers) | **30:2** (1 rescuer), **15:2** (2 rescuers) | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
| **Chest Recoil** | **Full recoil** between compressions | **Full recoil** | **Full recoil** | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
| **Minimize Interruptions** | **<10 seconds** | **<10 seconds** | **<10 seconds** | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
| **Hand Placement** | **Lower half of sternum** | **Lower half of sternum** | **Just below nipple line** (2 fingers or 2 thumbs) | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |

## Hands-Only CPR
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | **Adult cardiac arrest** (out-of-hospital), **Untrained rescuer** or **Unwilling to give rescue breaths** | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
| **Technique** | **Continuous chest compressions** at 100-120/min, **No rescue breaths** | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
| **Efficacy** | **Equivalent to conventional CPR** for first few minutes of cardiac arrest | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |
| **Not Recommended For** | **Children/Infants**, **Drowning**, **Drug overdose**, **Respiratory arrest** | https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr |

## Recovery Position
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | **Unresponsive but breathing normally**, **No suspected spinal injury** | https://www.redcross.org/take-a-class/first-aid/first-aid-steps |

| **Technique** | **Roll onto side**, **Extend neck** (if no spinal injury), **Bend top leg**, **Place top hand under cheek** | https://www.redcross.org/take-a-class/first-aid/first-aid-steps |
