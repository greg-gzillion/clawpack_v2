# First Aid - Head and Neck Injuries

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Head Injury** | **Trauma to scalp, skull, or brain** | https://www.redcross.org/take-a-class/first-aid/head-injuries |
| **Spinal Injury** | **Damage to spinal column or cord** | https://www.redcross.org/take-a-class/first-aid/spinal-injuries |
| **Assume Spinal Injury If** | **Mechanism concerning** (fall >3x height, Diving, Motor vehicle crash), **Altered mental status**, **Neck pain**, **Neurologic deficit**, **Intoxication** | https://www.redcross.org/take-a-class/first-aid/spinal-injuries |

## Concussion (Mild TBI)
| Sign/Symptom | Description | URL |
|--------------|-------------|-----|
| **Physical** | **Headache**, **Dizziness**, **Nausea/Vomiting**, **Balance problems**, **Visual changes**, **Sensitivity to light/noise** | https://www.cdc.gov/headsup/basics/concussion_symptoms.html |
| **Cognitive** | **Confusion**, **Feeling "foggy"**, **Memory problems**, **Slowed processing** | https://www.cdc.gov/headsup/basics/concussion_symptoms.html |
| **Emotional** | **Irritability**, **Sadness**, **Anxiety** | https://www.cdc.gov/headsup/basics/concussion_symptoms.html |
| **Sleep** | **Drowsiness**, **Sleeping more/less**, **Difficulty falling asleep** | https://www.cdc.gov/headsup/basics/concussion_symptoms.html |

## Spinal Motion Restriction
| Step | Action | URL |
|------|--------|-----|
| **1** | **Instruct victim NOT to move** | https://www.redcross.org/take-a-class/first-aid/spinal-injuries |
| **2** | **Manual stabilization**: Place hands on both sides of head to prevent movement | https://www.redcross.org/take-a-class/first-aid/spinal-injuries |
| **3** | **Maintain stabilization** until EMS arrives | https://www.redcross.org/take-a-class/first-aid/spinal-injuries |
| **4** | **Only move victim if**: **Scene unsafe**, **Need to perform CPR**, **Need to clear airway** | https://www.redcross.org/take-a-class/first-aid/spinal-injuries |
| **5** | **If must move**: **Log roll** with multiple rescuers maintaining spinal alignment | https://www.redcross.org/take-a-class/first-aid/spinal-injuries |

## Helmet Removal
| Principle | Description | URL |
|-----------|-------------|-----|
| **General Rule** | **Leave helmet on** unless it interferes with airway/breathing | https://www.redcross.org/take-a-class/first-aid/helmet-removal |
| **Remove If** | **Facemask cannot be removed** for CPR, **Helmet loose** and cannot be stabilized, **Vomiting** with airway compromise | https://www.redcross.org/take-a-class/first-aid/helmet-removal |
| **Technique** | **Two rescuers**: One stabilizes head/neck, Other removes helmet by pulling apart laterally | https://www.redcross.org/take-a-class/first-aid/helmet-removal |
