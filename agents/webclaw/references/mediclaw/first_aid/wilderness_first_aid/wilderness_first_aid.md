## Wilderness-Specific Considerations
| Consideration | Description | URL |
|---------------|-------------|-----|
| **Hypothermia Prevention** | **Keep warm/dry**, **Insulate from ground**, **Hypothermia wrap** | https://wms.org/mountain-medicine/wilderness-first-aid |
| **Hydration/Nutrition** | **Maintain fluid and calorie intake** | https://wms.org/mountain-medicine/wilderness-first-aid |
| **Wound Care** | **Irrigate with potable water**, **Delayed closure** (>6-12h leave open) | https://wms.org/mountain-medicine/wilderness-first-aid |
| **Improvised Splints** | **Use available materials** (pads, Trekking poles) | https://wms.org/mountain-medicine/wilderness-first-aid |
| **Altitude Illness** | **Descend if AMS/HACE/HAPE** | https://wwwnc.cdc.gov/travel/page/altitude-illness |
