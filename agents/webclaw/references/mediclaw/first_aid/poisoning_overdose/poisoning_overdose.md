# First Aid - Poisoning and Overdose

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Poisoning** | **Harmful effects** from **ingestion**, **inhalation**, **injection**, or **skin contact** with toxic substance | https://www.poison.org |
| **Poison Help Line** | **1-800-222-1222** (24/7, free, confidential) | https://www.poison.org |

## General Poisoning First Aid
| Step | Action | URL |
|------|--------|-----|
| **1** | **Assess scene safety** (protect yourself) | https://www.redcross.org/take-a-class/first-aid/poisoning |
| **2** | **Check responsiveness**, **ABCs** | https://www.redcross.org/take-a-class/first-aid/poisoning |
| **3** | **Call Poison Help** (1-800-222-1222) or **911** (if unresponsive) | https://www.redcross.org/take-a-class/first-aid/poisoning |
| **4** | **Follow instructions** from Poison Control | https://www.redcross.org/take-a-class/first-aid/poisoning |
| **5** | **Do NOT induce vomiting** unless instructed by Poison Control | https://www.redcross.org/take-a-class/first-aid/poisoning |
| **6** | **Do NOT give anything to drink** unless instructed | https://www.redcross.org/take-a-class/first-aid/poisoning |

## Inhaled Poison (e.g., Carbon Monoxide)
| Step | Action | URL |
|------|--------|-----|
| **1** | **Get victim to fresh air immediately** (protect yourself) | https://www.redcross.org/take-a-class/first-aid/poisoning |
| **2** | **Call 911** | https://www.redcross.org/take-a-class/first-aid/poisoning |
| **3** | **Check ABCs**, **Start CPR if needed** | https://www.redcross.org/take-a-class/first-aid/poisoning |

## Skin/Eye Exposure
| Step | Action | URL |
|------|--------|-----|
| **1** | **Remove contaminated clothing** | https://www.redcross.org/take-a-class/first-aid/poisoning |
| **2** | **Brush off dry powder** (before irrigation) | https://www.redcross.org/take-a-class/first-aid/poisoning |
| **3** | **Flush skin/eyes** with **copious water** x15-20 minutes | https://www.redcross.org/take-a-class/first-aid/poisoning |
| **4** | **Call Poison Help** | https://www.redcross.org/take-a-class/first-aid/poisoning |

## Opioid Overdose
| Sign | Action | URL |
|------|--------|-----|
| **Pinpoint pupils** | **CALL 911** | https://www.cdc.gov/stop-overdose/caring/naloxone.html |
| **Unresponsive** | **Check ABCs**, **Start CPR if no pulse/breathing** | https://www.cdc.gov/stop-overdose/caring/naloxone.html |
| **Respiratory depression** (slow/shallow) | **Administer Naloxone (Narcan)** if available: **1 spray in nostril**, **Repeat q2-3min** | https://www.cdc.gov/stop-overdose/caring/naloxone.html |
| **Cyanosis** | **Place in recovery position** if breathing resumes | https://www.cdc.gov/stop-overdose/caring/naloxone.html |
