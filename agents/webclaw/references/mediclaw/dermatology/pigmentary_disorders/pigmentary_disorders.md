# Dermatology - Pigmentary Disorders

## Vitiligo
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Acquired depigmentation due to autoimmune destruction of melanocytes | https://www.ncbi.nlm.nih.gov/books/NBK559149/ |
| Epidemiology | 0.5-2% worldwide; Equal gender; Any age (peak 10-30 years) | https://www.aad.org |
| Types | Segmental (unilateral, early onset, rapid stabilization); Non-segmental (bilateral, symmetric, progressive) | https://www.aad.org |
| Clinical Features | Well-demarcated, chalky-white macules/patches; Wood's lamp: bright white fluorescence | https://www.aad.org |
| Koebner Phenomenon | Lesions develop at sites of trauma | https://www.aad.org |
| Associated Conditions | Autoimmune thyroid disease (15-25%), alopecia areata, pernicious anemia, Addison disease | https://www.aad.org |

## Vitiligo Treatment
| Extent | Treatment | URL |
|--------|-----------|-----|
| Limited (<10% BSA) | Topical corticosteroids; Topical calcineurin inhibitors (Tacrolimus, Pimecrolimus); Topical JAK inhibitors (Ruxolitinib) | https://www.aad.org/guidelines |
| Moderate (10-20% BSA) | Phototherapy (NB-UVB); Excimer laser; Topical JAK inhibitors | https://www.aad.org/guidelines |
| Extensive (>20% BSA) | NB-UVB; Oral JAK inhibitors (Upadacitinib - off-label); Depigmentation (Monobenzone) if universal vitiligo | https://www.aad.org/guidelines |
| Stable, Refractory | Surgical grafting (punch, blister, melanocyte-keratinocyte transplantation) | https://www.aad.org/guidelines |

## Melasma
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Acquired hyperpigmentation of sun-exposed skin | https://www.ncbi.nlm.nih.gov/books/NBK459271/ |
| Epidemiology | Women (90%); Fitzpatrick skin types III-V; Pregnancy, OCPs | https://www.aad.org |
| Clinical Features | Irregular, light-to-dark brown macules/patches; Centrofacial, malar, mandibular patterns | https://www.aad.org |
| Wood's Lamp | Epidermal (accentuated), Dermal (not accentuated), Mixed | https://www.aad.org |
| Pathophysiology | UV exposure; Hormonal influences; Genetic predisposition | https://www.ncbi.nlm.nih.gov/books/NBK459271/ |

## Melasma Treatment
| Component | Agents | URL |
|-----------|--------|-----|
| Sun Protection | Broad-spectrum SPF =30; Visible light protection (tinted sunscreen with iron oxides) | https://www.aad.org |
| Topical Therapy | Hydroquinone 4% (first-line); Triple combination cream (Hydroquinone 4% + Tretinoin 0.05% + Fluocinolone acetonide 0.01%); Azelaic acid; Kojic acid; Tranexamic acid | https://www.aad.org |
| Procedures | Chemical peels (glycolic acid, salicylic acid, TCA); Laser (low-fluence Q-switched Nd:YAG, fractional); Microneedling | https://www.aad.org |
| Oral Therapy | Tranexamic acid (off-label for refractory cases) | https://www.aad.org |

## Post-Inflammatory Hyperpigmentation (PIH)
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Hyperpigmentation following cutaneous inflammation or injury | https://www.ncbi.nlm.nih.gov/books/NBK559150/ |
| Causes | Acne, eczema, psoriasis, trauma, burns, laser/peel complications | https://www.aad.org |
| Predisposition | Fitzpatrick skin types III-VI | https://www.aad.org |
| Treatment | Sun protection; Topical hydroquinone, azelaic acid, retinoids, kojic acid; Chemical peels; Laser (low-fluence Q-switched, fractional) | https://www.aad.org |
| Prevention | Treat underlying condition early; Sun protection; Consider pretreatment with topical retinoids/hydroquinone before procedures | https://www.aad.org |
