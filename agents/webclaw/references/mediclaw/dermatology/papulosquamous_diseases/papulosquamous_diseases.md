# Dermatology - Papulosquamous Diseases

## Psoriasis
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Chronic, immune-mediated inflammatory disease with well-demarcated erythematous plaques and silvery scale | https://www.ncbi.nlm.nih.gov/books/NBK448074/ |
| Epidemiology | 2-3% of population; Bimodal onset (20-30 and 50-60 years) | https://www.aad.org |
| Genetics | HLA-Cw6 (strongest association); PSORS1 locus | https://www.ncbi.nlm.nih.gov/books/NBK448074/ |
| Pathophysiology | Th17/IL-23 axis; Keratinocyte hyperproliferation | https://www.ncbi.nlm.nih.gov/books/NBK448074/ |

## Psoriasis Subtypes
| Type | Clinical Features | % of Cases | URL |
|------|-------------------|------------|-----|
| Chronic Plaque Psoriasis | Well-demarcated erythematous plaques with silvery scale; Extensor surfaces, scalp, sacrum | 80-90% | https://www.aad.org |
| Guttate Psoriasis | Small, drop-like papules; Trunk, proximal extremities; Post-streptococcal | 8% | https://www.aad.org |
| Pustular Psoriasis | Sterile pustules; Generalized (von Zumbusch) or localized (palmoplantar) | <5% | https://www.aad.org |
| Erythrodermic Psoriasis | Generalized erythema and scaling; >90% BSA | 1-2% | https://www.aad.org |
| Inverse Psoriasis | Erythematous plaques in flexural areas; Minimal scale | 3-7% | https://www.aad.org |
| Nail Psoriasis | Pitting, oil spots, onycholysis, subungual hyperkeratosis | 50% | https://www.aad.org |
| Psoriatic Arthritis | Inflammatory arthritis; Dactylitis, enthesitis | 30% | https://www.aad.org |

## Lichen Planus
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Inflammatory disorder affecting skin, mucous membranes, hair, nails | https://www.ncbi.nlm.nih.gov/books/NBK448126/ |
| Clinical Features | 6 P's: Purple, Pruritic, Polygonal, Planar (flat-topped), Papules, Plaques; Wickham striae | https://www.ncbi.nlm.nih.gov/books/NBK448126/ |
| Distribution | Flexor wrists, ankles, lower back, shins, genitals | https://www.ncbi.nlm.nih.gov/books/NBK448126/ |
| Oral Lichen Planus | Reticular (Wickham striae) or erosive; Buccal mucosa, tongue, gingiva | https://www.ncbi.nlm.nih.gov/books/NBK448126/ |
| Associations | Hepatitis C; Drug-induced (ACE-I, NSAIDs, beta-blockers, antimalarials) | https://www.ncbi.nlm.nih.gov/books/NBK448126/ |
| Treatment | Topical corticosteroids (high-potency for skin; medium-high for oral); Intralesional corticosteroids; Systemic corticosteroids; Phototherapy; Oral retinoids | https://www.ncbi.nlm.nih.gov/books/NBK448126/ |
