# Dermatology - Autoimmune Blistering Diseases

## Pemphigus Vulgaris
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Autoimmune intraepidermal blistering disease targeting desmoglein 3 (mucosal) and desmoglein 1 (cutaneous) | https://www.ncbi.nlm.nih.gov/books/NBK560860/ |
| Pathophysiology | IgG autoantibodies against desmogleins ? acantholysis | https://www.ncbi.nlm.nih.gov/books/NBK560860/ |
| Clinical Features | Flaccid bullae on non-inflammatory base; Erosions; Positive Nikolsky sign; Mucosal involvement (90%) | https://www.aad.org |
| Histopathology | Suprabasal acantholysis; "Tombstone" pattern of basal cells | https://www.aad.org |
| Direct Immunofluorescence (DIF) | IgG and C3 in intercellular spaces (chicken wire pattern) | https://www.aad.org |
| Indirect Immunofluorescence (IIF) | Circulating anti-desmoglein antibodies | https://www.aad.org |
| ELISA | Anti-Dsg3 (mucosal); Anti-Dsg1 (cutaneous) | https://www.aad.org |

## Pemphigus Vulgaris Treatment
| Severity | First-Line | Second-Line/Adjuvant | URL |
|----------|------------|----------------------|-----|

| Severe | Systemic corticosteroids (Prednisone 1-1.5 mg/kg/day) + Rituximab | IVIG; Plasmapheresis; Immunoadsorption | https://www.aad.org/guidelines |
| Refractory | Rituximab; IVIG; Cyclophosphamide | | https://www.aad.org/guidelines |

## Bullous Pemphigoid
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Autoimmune subepidermal blistering disease targeting BP180 and BP230 | https://www.ncbi.nlm.nih.gov/books/NBK448096/ |
| Epidemiology | Elderly (>60 years); Most common autoimmune blistering disease | https://www.aad.org |
| Clinical Features | Tense bullae on erythematous or urticarial base; Intense pruritus; Negative Nikolsky | https://www.aad.org |
| Histopathology | Subepidermal blister with eosinophils | https://www.aad.org |
| DIF | Linear IgG and C3 at basement membrane zone | https://www.aad.org |
| IIF (Salt-Split Skin) | IgG on epidermal side (roof) | https://www.aad.org |
| ELISA | Anti-BP180, Anti-BP230 antibodies | https://www.aad.org |

## Bullous Pemphigoid Treatment
| Severity | Treatment | URL |
|----------|-----------|-----|
| Localized | Superpotent topical corticosteroids (Clobetasol) | https://www.aad.org/guidelines |
| Mild-Moderate | Topical corticosteroids ± Doxycycline/Niacinamide | https://www.aad.org/guidelines |
| Severe | Systemic corticosteroids (Prednisone 0.5 mg/kg/day) ± Steroid-sparing agents (Azathioprine, Mycophenolate, Methotrexate) | https://www.aad.org/guidelines |
| Refractory | Rituximab; IVIG; Omalizumab; Dupilumab | https://www.aad.org/guidelines |
