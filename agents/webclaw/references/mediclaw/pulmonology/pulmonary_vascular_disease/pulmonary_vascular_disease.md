## Pulmonary Arteriovenous Malformation (PAVM)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Abnormal communication between pulmonary artery and vein** | https://www.nhlbi.nih.gov/health/arteriovenous-malformations |
| **Associations** | **Hereditary Hemorrhagic Telangiectasia (HHT)** (70-90%) | https://www.cdc.gov/hht/index.html |
| **Clinical Features** | **Dyspnea**, **Hypoxemia** (orthodeoxia), **Hemoptysis**, **Paradoxical emboli** (stroke, Brain abscess), **Epistaxis**, **Telangiectasias** | https://www.nhlbi.nih.gov/health/arteriovenous-malformations |
| **Diagnosis** | **Contrast echocardiography**, **CT pulmonary angiography** | https://www.nhlbi.nih.gov/health/arteriovenous-malformations |
| **Treatment** | **Embolization** for feeding artery =2-3 mm | https://www.nhlbi.nih.gov/health/arteriovenous-malformations |

## Hepatopulmonary Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Triad**: **Liver disease** + **Intrapulmonary vascular dilation** + **Hypoxemia** | https://www.niddk.nih.gov/health-information/liver-disease/hepatopulmonary-syndrome |
| **Pathophysiology** | **Increased nitric oxide** ? **Intrapulmonary vasodilation** ? **V/Q mismatch**, **Diffusion-perfusion impairment** | https://www.niddk.nih.gov/health-information/liver-disease/hepatopulmonary-syndrome |
| **Clinical Features** | **Dyspnea**, **Platypnea** (worse upright), **Orthodeoxia** (desaturation upright), **Clubbing**, **Spider angiomata** | https://www.niddk.nih.gov/health-information/liver-disease/hepatopulmonary-syndrome |
| **Diagnosis** | **Contrast echocardiography** (delayed appearance of bubbles in left heart after 3-6 cardiac cycles) | https://www.niddk.nih.gov/health-information/liver-disease/hepatopulmonary-syndrome |
| **Treatment** | **Liver transplantation** (only definitive treatment), **Oxygen** | https://www.niddk.nih.gov/health-information/liver-disease/hepatopulmonary-syndrome |
