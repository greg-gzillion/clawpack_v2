## Idiopathic Pulmonary Fibrosis (IPF)
| Component | Description | URL |
|-----------|-------------|-----|
| **Epidemiology** | **Older adults** (>60 years), **Male > Female** | https://www.nhlbi.nih.gov/health/idiopathic-pulmonary-fibrosis |
| **Clinical Features** | **Progressive dyspnea**, **Dry cough**, **Velcro crackles**, **Clubbing** | https://www.nhlbi.nih.gov/health/idiopathic-pulmonary-fibrosis |
| **HRCT** | **UIP pattern**: **Honeycombing**, **Traction bronchiectasis**, **Basal and subpleural predominance** | https://www.nhlbi.nih.gov/health/idiopathic-pulmonary-fibrosis |
| **Diagnosis** | **HRCT** (UIP pattern) or **Surgical lung biopsy** (probable UIP, Indeterminate) | https://www.nhlbi.nih.gov/health/idiopathic-pulmonary-fibrosis |
| **Treatment** | **Antifibrotics**: **Pirfenidone**, **Nintedanib** (slow progression) | https://www.nhlbi.nih.gov/health/idiopathic-pulmonary-fibrosis |
| **Supportive Care** | **Oxygen**, **Pulmonary rehabilitation**, **Vaccinations**, **Treat GERD**, **Lung transplant** | https://www.nhlbi.nih.gov/health/idiopathic-pulmonary-fibrosis |

## Hypersensitivity Pneumonitis (HP)
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathophysiology** | **Immune-mediated response** to inhaled antigens | https://www.nhlbi.nih.gov/health/hypersensitivity-pneumonitis |
| **Common Antigens** | **Bird proteins** (pigeons, Parakeets), **Mold** (farmer's lung), **Hot tub lung** (Mycobacterium), **Metalworking fluids** | https://www.nhlbi.nih.gov/health/hypersensitivity-pneumonitis |
| **Non-Fibrotic HP** | **GGO**, **Centrilobular nodules**, **Mosaic attenuation**, **Air trapping** | https://www.nhlbi.nih.gov/health/hypersensitivity-pneumonitis |
| **Fibrotic HP** | **Reticulation**, **Traction bronchiectasis**, **Honeycombing**, **Upper/mid-lung predominance** | https://www.nhlbi.nih.gov/health/hypersensitivity-pneumonitis |
| **Treatment** | **Antigen avoidance** (critical), **Corticosteroids** (non-fibrotic), **Antifibrotics** (nintedanib for progressive fibrotic) | https://www.nhlbi.nih.gov/health/hypersensitivity-pneumonitis |

## Progressive Pulmonary Fibrosis (PPF)
| Criteria | Description | URL |
|----------|-------------|-----|
| **Definition** | **ILD other than IPF** with **progressive fibrosis** despite treatment | https://www.thoracic.org/statements/interstitial-lung-disease.php |
| **=2 of 3 Criteria** | **Worsening respiratory symptoms**, **Physiologic decline** (FVC =5% or DLCO =10% in 24 months), **Radiologic progression** | https://www.thoracic.org/statements/interstitial-lung-disease.php |
| **Treatment** | **Nintedanib** (FDA-approved for PPF) | https://www.fda.gov/drugs |
