# Pulmonology - Pulmonary Function Tests (PFTs)

## Spirometry - Key Parameters
| Parameter | Definition | Normal Value | URL |
|-----------|------------|--------------|-----|
| **FVC** | Forced Vital Capacity | **>80% predicted** | https://www.nhlbi.nih.gov/health/lung-tests |
| **FEV1** | Forced Expiratory Volume in 1 second | **>80% predicted** | https://www.nhlbi.nih.gov/health/lung-tests |
| **FEV1/FVC Ratio** | **>0.70** (or >LLN) | **>70%** | https://www.nhlbi.nih.gov/health/lung-tests |

## Bronchodilator Response
| Response | Definition | Significance | URL |
|----------|------------|--------------|-----|
| **Significant** | **FEV1 or FVC increase =12% AND =200 mL** | Suggests **asthma** (not diagnostic) | https://www.nhlbi.nih.gov/health/lung-tests |
| **Volume Response** | **FVC increase =12% AND =200 mL** | Suggests **air trapping** | https://www.nhlbi.nih.gov/health/lung-tests |
