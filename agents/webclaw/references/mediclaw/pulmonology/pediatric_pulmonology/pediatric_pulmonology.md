## Bronchiolitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Etiology** | **RSV** (most common), Rhinovirus, Human metapneumovirus | https://www.cdc.gov/rsv/index.html |
| **Clinical Features** | **Wheezing**, **Crackles**, **Respiratory distress** (infants <2 years) | https://www.cdc.gov/rsv/index.html |
| **Treatment** | **Supportive** (suctioning, Hydration, Oxygen); **AVOID** bronchodilators, Steroids, Antibiotics | https://www.cdc.gov/rsv/index.html |
| **Prevention** | **Palivizumab** for high-risk infants (<29 weeks GA, CHD, CLD) | https://www.cdc.gov/rsv/index.html |

## Croup (Laryngotracheobronchitis)
| Component | Description | URL |
|-----------|-------------|-----|
| **Etiology** | **Parainfluenza** (most common), RSV, Influenza | https://www.cdc.gov/parainfluenza/index.html |
| **Clinical Features** | **Barking cough**, **Stridor**, Hoarseness, Fever | https://www.cdc.gov/parainfluenza/index.html |
| **Treatment** | **Dexamethasone** 0.6 mg/kg PO/IM; **Nebulized epinephrine** (moderate-severe) | https://www.cdc.gov/parainfluenza/index.html |
