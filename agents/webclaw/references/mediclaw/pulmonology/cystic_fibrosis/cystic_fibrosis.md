# Pulmonology - Cystic Fibrosis (CF)

## Cystic Fibrosis - Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Genetics** | **Autosomal recessive**, **CFTR gene** (most common ?F508) | https://www.nhlbi.nih.gov/health/cystic-fibrosis |
| **Pathophysiology** | **Defective CFTR** ? **Impaired chloride and bicarbonate transport** ? **Dehydrated, Acidic airway surface liquid** ? **Mucus plugging**, **Infection**, **Inflammation** | https://www.nhlbi.nih.gov/health/cystic-fibrosis |
| **Newborn Screening** | **Elevated immunoreactive trypsinogen (IRT)** ? **CFTR mutation panel** ? **Sweat chloride test** | https://www.cdc.gov/newbornscreening/index.html |

## Cystic Fibrosis - Diagnosis
| Test | Diagnostic Value | URL |
|------|------------------|-----|
| **Sweat Chloride** | **=60 mmol/L** (diagnostic), **30-59** (intermediate), **<30** (normal) | https://www.nhlbi.nih.gov/health/cystic-fibrosis |
| **CFTR Genetic Testing** | **Two disease-causing mutations** | https://www.nhlbi.nih.gov/health/cystic-fibrosis |
| **Nasal Potential Difference** | Abnormal chloride transport | https://www.nhlbi.nih.gov/health/cystic-fibrosis |

## Cystic Fibrosis - Pulmonary Management
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **Airway Clearance** | **Chest physiotherapy**, **PEP**, **Oscillatory PEP**, **High-frequency chest wall oscillation**, **Aerobic exercise** | https://www.nhlbi.nih.gov/health/cystic-fibrosis |
| **Mucoactive Agents** | **Dornase alfa** (Pulmozyme), **Hypertonic saline** (7%) | https://www.nhlbi.nih.gov/health/cystic-fibrosis |
| **Bronchodilators** | Before airway clearance | https://www.nhlbi.nih.gov/health/cystic-fibrosis |
| **Antibiotics** | **Chronic suppression** (inhaled tobramycin, Aztreonam, Colistin; Oral azithromycin), **Acute exacerbation** (IV antibiotics) | https://www.nhlbi.nih.gov/health/cystic-fibrosis |
| **CFTR Modulators** | **Ivacaftor** (G551D, Others), **Lumacaftor/Ivacaftor** (F508del homozygous), **Tezacaftor/Ivacaftor** (F508del, Residual), **Elexacaftor/Tezacaftor/Ivacaftor** (F508del - most) | https://www.nhlbi.nih.gov/health/cystic-fibrosis |
