## Chest CT - Patterns
| Pattern | Description | Differential | URL |
|---------|-------------|--------------|-----|
| **Ground-Glass Opacity (GGO)** | Hazy increased attenuation with visible vessels | **Infection** (PJP, Viral), **ILD** (NSIP, DIP), **Edema**, **Hemorrhage** | https://www.acr.org/Clinical-Resources/ACR-Appropriateness-Criteria |
| **Consolidation** | Opacification obscuring vessels | **Pneumonia**, **Edema**, **Hemorrhage**, **Malignancy** | https://www.acr.org/Clinical-Resources/ACR-Appropriateness-Criteria |
| **Reticular** | Linear opacities | **ILD** (UIP, NSIP), **Edema** | https://www.acr.org/Clinical-Resources/ACR-Appropriateness-Criteria |
| **Nodular** | Small rounded opacities | **Miliary TB**, **Metastases**, **Sarcoidosis**, **Hypersensitivity pneumonitis** | https://www.acr.org/Clinical-Resources/ACR-Appropriateness-Criteria |
| **Honeycombing** | Clustered cystic airspaces | **UIP** (IPF, CTD-ILD), **End-stage ILD** | https://www.acr.org/Clinical-Resources/ACR-Appropriateness-Criteria |
| **Tree-in-Bud** | Centrilobular nodules + branching opacities | **Infection** (TB, MAC, Aspiration), **Cystic fibrosis**, **ABPA** | https://www.acr.org/Clinical-Resources/ACR-Appropriateness-Criteria |
| **Mosaic Attenuation** | Patchwork of different densities | **Small airways disease**, **Chronic PE**, **Hypersensitivity pneumonitis** | https://www.acr.org/Clinical-Resources/ACR-Appropriateness-Criteria |

## Interstitial Lung Disease - UIP Pattern (ATS/ERS/JRS/ALAT)
| Category | CT Features | URL |
|----------|-------------|-----|
| **UIP Pattern** | **Honeycombing** ± Traction bronchiectasis, **Basal and subpleural predominance**, **No inconsistent features** | https://www.thoracic.org/statements/interstitial-lung-disease.php |
| **Probable UIP** | **Reticular pattern** + Traction bronchiectasis, **Basal and subpleural**, **No honeycombing**, No inconsistent features | https://www.thoracic.org/statements/interstitial-lung-disease.php |
| **Indeterminate** | Fibrosis not meeting UIP/probable UIP, No inconsistent features | https://www.thoracic.org/statements/interstitial-lung-disease.php |
| **Alternative Diagnosis** | **Cysts**, **Mosaic attenuation**, **GGO predominance**, **Nodules**, **Pleural plaques**, **Upper/mid-lung predominance** | https://www.thoracic.org/statements/interstitial-lung-disease.php |
