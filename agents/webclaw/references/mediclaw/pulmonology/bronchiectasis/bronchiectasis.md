# Pulmonology - Bronchiectasis

## Bronchiectasis - Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Permanent, Irreversible dilation of bronchi** due to **destruction of elastic and muscular components** of bronchial wall | https://www.nhlbi.nih.gov/health/bronchiectasis |
| **Pathophysiology** | **"Vicious cycle"**: Infection ? Inflammation ? Airway damage ? Impaired mucociliary clearance ? Infection | https://www.nhlbi.nih.gov/health/bronchiectasis |
| **Etiology** | **Idiopathic** (50%), **Post-infectious** (TB, NTM, Pertussis), **CF**, **ABPA**, **PCD**, **Immunodeficiency**, **Autoimmune** (RA, Sjögren), **Aspiration**, **Alpha-1** | https://www.nhlbi.nih.gov/health/bronchiectasis |

## Bronchiectasis - Diagnosis
| Modality | Findings | URL |
|----------|----------|-----|
| **High-Resolution CT (HRCT)** | **Bronchial dilation** (internal diameter > adjacent pulmonary artery), **Lack of tapering**, **Bronchi visible within 1 cm of pleura**, **Tram-track sign**, **Signet ring sign** | https://www.nhlbi.nih.gov/health/bronchiectasis |
| **Spirometry** | **Obstructive** pattern (most common), Restrictive, Mixed | https://www.nhlbi.nih.gov/health/bronchiectasis |
| **Sputum Culture** | **H. influenzae**, **P. aeruginosa**, **S. aureus**, **NTM** | https://www.nhlbi.nih.gov/health/bronchiectasis |

## Bronchiectasis - Management
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **Airway Clearance** | **Chest physiotherapy**, **PEP**, **Oscillatory PEP** (Flutter, Acapella), **High-frequency chest wall oscillation** (Vest), **Aerobic exercise** | https://www.nhlbi.nih.gov/health/bronchiectasis |
| **Mucoactive Agents** | **Hypertonic saline**, **Mannitol**, **Dornase alfa** (CF only) | https://www.nhlbi.nih.gov/health/bronchiectasis |
| **Bronchodilators** | If **reversible airflow obstruction** | https://www.nhlbi.nih.gov/health/bronchiectasis |
| **Inhaled Corticosteroids** | Not routinely; If **asthma/ABPA** | https://www.nhlbi.nih.gov/health/bronchiectasis |
| **Antibiotics (Acute Exacerbation)** | 14 days (based on prior sputum cultures) | https://www.nhlbi.nih.gov/health/bronchiectasis |
| **Antibiotics (Chronic Suppression)** | **Inhaled** (tobramycin, Aztreonam, Colistin) or **Oral macrolides** (azithromycin) for **=3 exacerbations/year** | https://www.nhlbi.nih.gov/health/bronchiectasis |
| **NTM Treatment** | **MAC**: Azithromycin + Rifampin + Ethambutol; **M. abscessus**: Macrolide + Amikacin + Cefoxitin/Imipenem | https://www.cdc.gov/nontuberculous-mycobacteria/index.html |
| **Surgery** | **Localized disease** with **refractory symptoms** or **massive hemoptysis** | https://www.nhlbi.nih.gov/health/bronchiectasis |
