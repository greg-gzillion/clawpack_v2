# Pulmonology - Sleep Apnea

## Obstructive Sleep Apnea (OSA)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Repetitive upper airway collapse** during sleep ? **Apneas/Hypopneas** ? **Intermittent hypoxia**, **Sleep fragmentation** | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **Risk Factors** | **Obesity**, **Male**, **Age >50**, **Craniofacial abnormalities**, **Large neck circumference** (>17" men, >16" women) | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **Symptoms** | **Loud snoring**, **Witnessed apneas**, **Excessive daytime sleepiness**, **Morning headaches**, **Nocturia** | https://www.nhlbi.nih.gov/health/sleep-apnea |

## OSA - Diagnosis
| Parameter | Definition | URL |
|-----------|------------|-----|
| **Apnea** | **Cessation of airflow =10 seconds** | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **Hypopnea** | **=30% reduction in airflow** + **=3% oxygen desaturation** or **arousal** | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **AHI** | Apnea-Hypopnea Index (events/hour) | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **OSA Severity** | **Mild**: AHI 5-14; **Moderate**: 15-29; **Severe**: =30 | https://www.nhlbi.nih.gov/health/sleep-apnea |
