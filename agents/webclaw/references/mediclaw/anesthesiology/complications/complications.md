## Malignant Hyperthermia (MH)
| Feature | Description | URL |
|---------|-------------|-----|
| Pathophysiology | Ryanodine receptor mutation ? uncontrolled Ca2+ release | https://www.mhaus.org |
| Triggers | Succinylcholine; volatile anesthetics (ALL except N2O) | https://www.mhaus.org |
| Safe Agents | Propofol; opioids; N2O; non-depolarizers; local anesthetics | https://www.mhaus.org |
| Early Signs | Increased ETCO2; tachycardia; tachypnea; masseter spasm | https://www.mhaus.org |
| Late Signs | Hyperthermia; hyperkalemia; rhabdomyolysis; DIC | https://www.mhaus.org |
| Treatment | **Dantrolene 2.5 mg/kg** (repeat up to 10 mg/kg); stop triggers; hyperventilate; cool; treat hyperkalemia | https://www.mhaus.org |
| MH Hotline | 1-800-MH-HYPER (1-800-644-9737) | https://www.mhaus.org |
