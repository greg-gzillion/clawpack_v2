# Toxicology - Caustic Injuries

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Acids** | **Coagulative necrosis** (eschar formation limits depth) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Alkalis** | **Liquefactive necrosis** (deeper penetration, More severe injury) | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Clinical Features
| Location | Findings | URL |
|----------|----------|-----|
| **Oropharyngeal** | **Burns**, **Erythema**, **Edema**, **Drooling**, **Stridor** | https://www.acmt.net/Resources/Clinical-Resources.aspx |

| **Gastric** | **Epigastric pain**, **Nausea**, **Vomiting**, **Hematemesis** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Systemic** | **Perforation**, **Mediastinitis**, **Peritonitis**, **Metabolic acidosis** | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Management
| Intervention | Recommendation | URL |
|--------------|----------------|-----|
| **Airway** | **Early intubation** if signs of airway compromise | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Decontamination** | **DO NOT induce emesis**, **DO NOT give activated charcoal**, **DO NOT attempt neutralization** (exothermic reaction) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Dilution** | **Small volume of water/milk** (if able to swallow, No perforation) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Endoscopy** | **Within 12-24h** (unless asymptomatic, No oral burns) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Corticosteroids** | **Controversial** (may ? stricture formation); Consider for **Grade IIb** circumferential burns | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Antibiotics** | **Only if perforation or infection** | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Button Battery Ingestion
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **Liquefactive necrosis** within **2 hours** (electrical current, Alkali leakage, Pressure necrosis) | https://www.poison.org/battery |
| **Location** | **Esophageal** = **Emergency** (risk of perforation, Tracheoesophageal fistula, Aortoesophageal fistula) | https://www.poison.org/battery |
| **Management** | **Emergent endoscopic removal** if esophageal; **Honey** (if >1 year, Not allergic) may slow injury while awaiting endoscopy | https://www.poison.org/battery |

## Hydrofluoric Acid
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **Fluoride ion** binds **calcium and magnesium** ? **Hypocalcemia**, **Hypomagnesemia**, **Hyperkalemia**, **Cellular necrosis** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Clinical Effects** | **Severe pain** (often delayed), **Deep tissue destruction**, **Cardiac arrhythmias** (QTc prolongation, Torsades) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Calcium gluconate** (topical gel, Subcutaneous infiltration, Intra-arterial, IV), **Cardiac monitoring**, **Correct hypocalcemia/hypomagnesemia** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
