## Specific Opioid Considerations
| Opioid | Unique Features | URL |
|--------|-----------------|-----|
| **Fentanyl** | **Rapid onset**, **High potency** (100x morphine), **Chest wall rigidity** ("wooden chest") | https://www.cdc.gov/opioids/basics/fentanyl.html |
| **Methadone** | **Long half-life** (8-59h), **QTc prolongation**, **Delayed respiratory depression** | https://www.cdc.gov/opioids/basics/methadone.html |
| **Buprenorphine** | **Partial agonist**, **High binding affinity**, **May require higher naloxone doses** | https://www.samhsa.gov/medications-substance-use-disorders/medications-counseling-related-conditions/buprenorphine |
| **Tramadol** | **SNRI activity** ? **Seizures**, **Serotonin syndrome** | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/tramadol |
| **Loperamide** | **P-glycoprotein substrate**; **High doses** ? **Cardiotoxicity** (QTc prolongation, TdP) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/loperamide |

## Naloxone
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **Competitive ｵ-opioid receptor antagonist** | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/naloxone |
| **Indications** | **Opioid overdose** (respiratory depression, CNS depression) | https://www.cdc.gov/stop-overdose/caring/naloxone.html |
| **Dose (IV/IM/IN)** | **0.04-0.4 mg** (repeat q2-3min); **Higher doses** (2-10 mg) for fentanyl, buprenorphine | https://www.cdc.gov/stop-overdose/caring/naloxone.html |
| **Duration** | **30-90 minutes** (shorter than most opioids) | https://www.cdc.gov/stop-overdose/caring/naloxone.html |
| **Goal** | **Restore adequate ventilation** (NOT full consciousness) | https://www.cdc.gov/stop-overdose/caring/naloxone.html |
| **Adverse Effects** | **Precipitated withdrawal** (nausea, vomiting, agitation, Tachycardia, Hypertension) | https://www.cdc.gov/stop-overdose/caring/naloxone.html |
| **Continuous Infusion** | **2/3 of effective bolus dose per hour** (if recurrent respiratory depression) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
