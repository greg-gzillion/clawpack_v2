# Toxicology - Anticholinergic Toxicity

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **Muscarinic acetylcholine receptor antagonism** (central and peripheral) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Sources** | **Antihistamines** (diphenhydramine), **TCAs**, **Antipsychotics**, **Atropine**, **Scopolamine**, **Jimson weed** (Datura), **Belladonna**, **Overactive bladder medications** (oxybutynin) | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Specific Agents
| Agent | Unique Features | URL |
|-------|-----------------|-----|
| **Diphenhydramine** | **Sodium channel blockade** (wide QRS at high doses), **Seizures**, **Rhabdomyolysis** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **TCAs** | **Sodium channel blockade** (wide QRS), **Seizures**, **Hypotension** (see Antidepressant Toxicity) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Jimson Weed (Datura)** | **Severe anticholinergic delirium**, **Visual hallucinations**, **Amnesia** | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Management
| Intervention | Indication | URL |
|--------------|------------|-----|
| **Supportive Care** | **Most patients** (agitation controlled with benzodiazepines) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Benzodiazepines** | **Agitation**, **Seizures** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Active Cooling** | **Hyperthermia** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Physostigmine** | **Severe anticholinergic delirium** (agitation not controlled with benzodiazepines), **Seizures refractory to benzodiazepines** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Physostigmine Contraindications** | **Wide QRS**, **Bradycardia**, **Asthma**, **Intestinal obstruction**, **Peripheral vascular disease** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Sodium Bicarbonate** | **Wide QRS** (diphenhydramine, TCA) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Avoid Antipsychotics** | **Lower seizure threshold**, **Worsen anticholinergic effects**, **Impaired thermoregulation** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
