# Toxicology - Heavy Metal Toxicity

## Lead
| Component | Description | URL |
|-----------|-------------|-----|
| **Sources** | **Old paint** (pre-1978), **Contaminated soil**, **Water pipes**, **Imported pottery**, **Traditional remedies**, **Occupational** (battery manufacturing, Construction) | https://www.atsdr.cdc.gov/toxprofiles/tp13.pdf |
| **Clinical Effects (Children)** | **Neurodevelopmental delay**, **? IQ**, **Behavioral problems**, **Anemia**, **Abdominal pain** | https://www.cdc.gov/nceh/lead/prevention/health-effects.htm |
| **Clinical Effects (Adults)** | **Hypertension**, **Nephropathy**, **Peripheral neuropathy** (wrist drop), **Anemia**, **Abdominal pain**, **Gout** | https://www.cdc.gov/nceh/lead/prevention/health-effects.htm |
| **Diagnosis** | **Blood lead level (BLL)** (reference value <3.5 mcg/dL) | https://www.cdc.gov/nceh/lead/prevention/screening-testing.htm |
| **Management** | **Remove source**, **Chelation** (BLL =45 mcg/dL): **Succimer** (DMSA) PO, **Calcium EDTA** IV, **BAL** (if encephalopathy) | https://www.cdc.gov/nceh/lead/prevention/management.htm |

## Arsenic
| Component | Description | URL |
|-----------|-------------|-----|
| **Sources** | **Groundwater** (wells), **Pesticides**, **Pressure-treated wood**, **Traditional remedies** | https://www.atsdr.cdc.gov/toxprofiles/tp2.pdf |
| **Acute Toxicity** | **Garlic breath**, **GI hemorrhage**, **QTc prolongation**, **Torsades**, **Shock** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Chronic Toxicity** | **Hyperkeratosis** (palms/soles), **Mees lines** (nails), **Peripheral neuropathy**, **Skin cancer**, **Bladder/lung cancer** | https://www.atsdr.cdc.gov/toxprofiles/tp2.pdf |
| **Diagnosis** | **24-hour urine arsenic** (speciated - organic vs inorganic) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Remove source**, **Chelation**: **Dimercaprol (BAL)** , **Succimer (DMSA)** | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Mercury
| Form | Sources | Clinical Effects | URL |
|------|---------|------------------|-----|
| **Elemental (Hg0)** | **Thermometers**, **Fluorescent bulbs**, **Dental amalgam** | **Inhalation**: **Pneumonitis**, **Tremor**, **Erethism** (shyness, Irritability), **Gingivitis** | https://www.atsdr.cdc.gov/toxprofiles/tp46.pdf |
| **Inorganic (Hg2+)** | **Skin-lightening creams**, **Traditional remedies** | **GI corrosion**, **Renal failure** | https://www.atsdr.cdc.gov/toxprofiles/tp46.pdf |
| **Organic (Methylmercury)** | **Fish** (bioaccumulation) | **Neurotoxicity** (ataxia, Dysarthria, Visual field constriction, Paresthesias) | https://www.atsdr.cdc.gov/toxprofiles/tp46.pdf |
| **Diagnosis** | **24-hour urine mercury** (elemental/inorganic), **Whole blood mercury** (organic) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Remove source**, **Chelation**: **Succimer (DMSA)** , **Dimercaprol (BAL)** | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Iron
| Component | Description | URL |
|-----------|-------------|-----|
| **Toxic Dose** | **>20 mg/kg elemental iron** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Stages** | **Stage 1** (0.5-6h): GI (vomiting, diarrhea, GI bleed); **Stage 2** (6-24h): Apparent improvement; **Stage 3** (12-48h): Shock, Liver failure, Metabolic acidosis; **Stage 4** (2-5d): Hepatic necrosis; **Stage 5** (2-6w): Gastric outlet obstruction | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Diagnosis** | **Serum iron** (peak 4-6h post-ingestion), **Abdominal X-ray** (radiopaque tablets) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Whole bowel irrigation**, **Deferoxamine** (indications: Iron >500 mcg/dL, Metabolic acidosis, Shock, Altered mental status) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
