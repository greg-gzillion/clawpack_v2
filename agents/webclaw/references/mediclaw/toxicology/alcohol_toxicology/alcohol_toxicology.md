# Toxicology - Alcohol Toxicology

## Ethanol (Ethyl Alcohol)
| Component | Description | URL |
|-----------|-------------|-----|
| **Toxic Dose** | **Variable** (depends on tolerance) | https://www.atsdr.cdc.gov/toxprofiles/tp191.pdf |
| **Metabolism** | **Zero-order kinetics** (~15-20 mg/dL/h) | https://www.atsdr.cdc.gov/toxprofiles/tp191.pdf |
| **Clinical Stages** | **Legal limit**: 80 mg/dL (0.08%); **Euphoria**: 20-50 mg/dL; **Excitement**: 50-150 mg/dL; **Confusion**: 150-250 mg/dL; **Stupor**: 250-400 mg/dL; **Coma**: >400 mg/dL | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Supportive care**, **Thiamine** (prevent Wernicke), **Glucose** (hypoglycemia) | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Alcohol Withdrawal
| Component | Description | URL |
|-----------|-------------|-----|
| **Onset** | **6-12h** (tremulousness), **12-48h** (seizures), **48-96h** (delirium tremens) | https://www.niaaa.nih.gov |
| **CIWA-Ar Protocol** | **Symptom-triggered benzodiazepines** (diazepam, lorazepam, chlordiazepoxide) | https://www.mdcalc.com/calc/10433/ciwa-ar-alcohol-withdrawal |
| **Delirium Tremens (DTs)** | **Delirium**, **Agitation**, **Hallucinations**, **Autonomic instability** (fever, tachycardia, hypertension) | https://www.niaaa.nih.gov |
| **DTs Management** | **ICU**, **High-dose benzodiazepines**, **Phenobarbital** (refractory) | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Methanol
| Component | Description | URL |
|-----------|-------------|-----|
| **Sources** | **Windshield washer fluid**, **Moonshine**, **Solvents**, **Varnishes** | https://www.atsdr.cdc.gov/toxprofiles/tp119.pdf |
| **Toxic Dose** | **~10 mL** can cause blindness; **~30 mL** can be lethal | https://www.atsdr.cdc.gov/toxprofiles/tp119.pdf |
| **Metabolism** | **Methanol ? Formaldehyde ? Formic acid** (via alcohol dehydrogenase) | https://www.atsdr.cdc.gov/toxprofiles/tp119.pdf |
| **Latent Period** | **12-24h** (if no ethanol co-ingestion) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Symptoms** | **Visual disturbances** ("snowfield vision"), **Headache**, **Nausea**, **Abdominal pain**, **Metabolic acidosis**, **Optic neuropathy**, **Putamen necrosis** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Labs** | **? Anion gap metabolic acidosis**, **? Osmolal gap**, **Methanol level** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Treatment** | **Fomepizole** or **Ethanol** (inhibits alcohol dehydrogenase), **Hemodialysis** (severe) | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Ethylene Glycol
| Component | Description | URL |
|-----------|-------------|-----|
| **Sources** | **Antifreeze**, **De-icing solutions**, **Brake fluid** | https://www.atsdr.cdc.gov/toxprofiles/tp96.pdf |
| **Toxic Dose** | **~30 mL** can be lethal | https://www.atsdr.cdc.gov/toxprofiles/tp96.pdf |
| **Metabolism** | **Ethylene glycol ? Glycoaldehyde ? Glycolic acid ? Oxalic acid** (via alcohol dehydrogenase) | https://www.atsdr.cdc.gov/toxprofiles/tp96.pdf |
| **Stages** | **Stage 1** (0.5-12h): CNS depression, Inebriation; **Stage 2** (12-24h): Cardiopulmonary (tachycardia, Hypertension, CHF); **Stage 3** (24-72h): Renal failure (flank pain, Oliguria, Calcium oxalate crystals) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Labs** | **? Anion gap metabolic acidosis**, **? Osmolal gap**, **Hypocalcemia** (calcium oxalate precipitation), **Ethylene glycol level** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Urinalysis** | **Calcium oxalate crystals** (envelope-shaped) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Treatment** | **Fomepizole** or **Ethanol**, **Hemodialysis** (severe) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
