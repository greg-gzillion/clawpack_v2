# Toxicology - Sedative-Hypnotic Toxicity

## Benzodiazepines
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **GABA-A receptor agonist** (? frequency of chloride channel opening) | https://www.dea.gov/factsheets/benzodiazepines |
| **Clinical Effects** | **CNS depression**, **Respiratory depression** (mild unless co-ingestants), **Hypotension** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Toxidrome** | **Sedative-hypnotic toxidrome** (CNS depression, Normal/miotic pupils, Normal vital signs) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Flumazenil** | **GABA-A receptor antagonist**; **Use with CAUTION** (precipitates withdrawal/seizures in chronic users, TCA co-ingestion) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/flumazenil |
| **Flumazenil Indications** | **Isolated benzodiazepine overdose** in **non-habituated patient** (e.g., Pediatric ingestion, Procedural sedation reversal) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Flumazenil Contraindications** | **Chronic benzodiazepine use**, **Seizure history**, **Co-ingestion of proconvulsant** (TCA, bupropion), **Unknown ingestion**, **Wide QRS** | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Barbiturates
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **GABA-A receptor agonist** (? duration of chloride channel opening) | https://www.dea.gov/factsheets/barbiturates |
| **Clinical Effects** | **Profound CNS depression**, **Respiratory depression**, **Hypotension**, **Hypothermia**, **Coma** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Toxic Dose** | **10x hypnotic dose** can be lethal | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Supportive care**, **Multidose activated charcoal** (phenobarbital), **Urine alkalinization** (phenobarbital), **Hemodialysis** (severe) | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Z-Drugs (Zolpidem, Eszopiclone, Zaleplon)
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **Selective GABA-A a1 receptor agonist** | https://www.fda.gov/drugs |
| **Clinical Effects** | **Similar to benzodiazepines**; **Complex sleep behaviors** (sleepwalking, Sleep driving) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/zolpidem |
| **Management** | **Supportive care**; **Flumazenil** (same precautions as benzodiazepines) | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Baclofen
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **GABA-B receptor agonist** | https://www.fda.gov/drugs |
| **Clinical Effects** | **CNS depression**, **Respiratory depression**, **Hypotension**, **Bradycardia**, **Hypothermia**, **Seizures** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Supportive care**, **Hemodialysis** (severe) | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Gamma-Hydroxybutyrate (GHB)
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **GABA-B receptor agonist**, **GHB receptor agonist** | https://www.dea.gov/factsheets/ghb-gamma-hydroxybutyric-acid |
| **Clinical Effects** | **Rapid onset** CNS depression, **Abrupt awakening** (often with agitation), **Bradycardia**, **Hypotension**, **Hypothermia**, **Myoclonus** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Withdrawal** | **Severe** (delirium, Autonomic instability, Rhabdomyolysis) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Supportive care**; **Benzodiazepines** (withdrawal) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
