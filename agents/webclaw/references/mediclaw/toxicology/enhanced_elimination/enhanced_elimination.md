## Urine Alkalinization Protocol
| Step | Procedure | URL |
|------|-----------|-----|
| **1** | **Sodium bicarbonate** 1-2 mEq/kg IV bolus | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **2** | **Sodium bicarbonate infusion** (150 mEq in 1L D5W) at 150-250 mL/h | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **3** | **Target urine pH =7.5** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **4** | **Monitor serum pH** (goal <7.55), **Potassium** (replace if <4.0 mEq/L) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **5** | **Contraindications**: **Volume overload**, **Hypokalemia**, **Cerebral edema**, **Pulmonary edema** | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Indications for Hemodialysis in Toxicology (I STUMBLE)
| Mnemonic | Indication | URL |
|----------|------------|-----|
| **I** | **Isopropanol** (severe poisoning) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **S** | **Salicylates** (severe poisoning: level >100 mg/dL acute, >60 mg/dL chronic, Altered mental status, Pulmonary edema, Refractory acidosis) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **T** | **Theophylline** (severe poisoning) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **U** | **Uremia** (from toxin-induced renal failure) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **M** | **Methanol** (level >50 mg/dL, Acidosis, Visual symptoms) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **B** | **Barbiturates** (severe phenobarbital poisoning) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **L** | **Lithium** (level >4 mEq/L acute, >2.5 mEq/L chronic, Neurologic symptoms, Renal failure) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **E** | **Ethylene Glycol** (level >50 mg/dL, Acidosis, Renal failure) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
