# Toxicology - Anticonvulsant Toxicity

## Phenytoin
| Component | Description | URL |
|-----------|-------------|-----|
| **Toxic Level** | **>20 mcg/mL** (total); **>2 mcg/mL** (free) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Kinetics** | **Zero-order** at toxic levels | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Clinical Effects** | **Nystagmus** (early), **Ataxia**, **Dysarthria**, **Sedation**, **Coma** (severe) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Cardiovascular** | **IV phenytoin**: **Hypotension**, **Bradycardia**, **Arrhythmias** (propylene glycol diluent) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Supportive care**, **Multidose activated charcoal** (controversial), **Hemodialysis** NOT effective | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Carbamazepine
| Component | Description | URL |
|-----------|-------------|-----|
| **Toxic Level** | **>12 mcg/mL** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Clinical Effects** | **Anticholinergic**, **Sodium channel blockade** (wide QRS), **Seizures**, **Coma** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Multidose activated charcoal** (enhances elimination), **Sodium bicarbonate** (wide QRS), **Hemodialysis** (severe) | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Valproic Acid
| Component | Description | URL |
|-----------|-------------|-----|
| **Toxic Level** | **>120 mcg/mL** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Clinical Effects** | **CNS depression**, **Cerebral edema**, **Hyperammonemia** (with normal LFTs), **Hepatotoxicity**, **Thrombocytopenia**, **Hypernatremia** (IV formulation) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **L-Carnitine** (100 mg/kg IV load ? 50 mg/kg q8h), **Hemodialysis** (severe) | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Lamotrigine
| Component | Description | URL |
|-----------|-------------|-----|
| **Clinical Effects** | **Sodium channel blockade** (wide QRS), **Seizures**, **Sedation**, **Ataxia** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Sodium bicarbonate** (wide QRS), **Supportive care** | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Lithium
| Component | Description | URL |
|-----------|-------------|-----|
| **Therapeutic Level** | **0.6-1.2 mEq/L** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Toxic Level (Mild)** | **1.5-2.5 mEq/L** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Toxic Level (Moderate)** | **2.5-3.5 mEq/L** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Toxic Level (Severe)** | **>3.5 mEq/L** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Acute Toxicity** | **GI symptoms** (nausea, vomiting, diarrhea) predominate; **Lower mortality** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Chronic Toxicity** | **Neurologic symptoms** (tremor, ataxia, confusion, Seizures, Coma) predominate; **Higher mortality** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Risk Factors** | **Dehydration**, **NSAIDs**, **Thiazides**, **ACEi/ARBs**, **Renal failure** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **IV fluids** (NS), **Hemodialysis** (indicated for: level >4 mEq/L acute, >2.5 mEq/L chronic, Neurologic symptoms, Renal failure) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
