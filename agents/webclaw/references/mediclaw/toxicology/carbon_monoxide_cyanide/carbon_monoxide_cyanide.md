# Toxicology - Carbon Monoxide and Cyanide

## Carbon Monoxide (CO)
| Component | Description | URL |
|-----------|-------------|-----|
| **Sources** | **Incomplete combustion** (fires, Furnaces, Generators, Vehicles) | https://www.cdc.gov/co/index.html |
| **Mechanism** | **Binds hemoglobin with 250x affinity of oxygen** ? **? O2 delivery**, **Shifts oxyhemoglobin curve left** ? **? O2 release**, **Binds cytochrome c oxidase** ? **Cellular hypoxia** | https://www.atsdr.cdc.gov/toxprofiles/tp201.pdf |
| **Symptoms** | **Headache**, **Nausea**, **Dizziness**, **Confusion**, **Syncope**, **Seizures**, **Coma** | https://www.cdc.gov/co/symptoms.html |
| **Diagnosis** | **COHb level** (venous or arterial) | https://www.cdc.gov/co/faqs.htm |
| **COHb Levels** | **<10%**: Smokers, Urban dwellers; **10-20%**: Headache, Nausea; **20-30%**: Confusion, Dizziness; **30-40%**: Syncope, Tachycardia; **40-50%**: Coma, Seizures; **>50%**: Often fatal | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Pulse Oximetry** | **Falsely normal/elevated** (cannot distinguish COHb from O2Hb) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **100% O2** (non-rebreather), **Hyperbaric oxygen (HBO)** (indications: COHb >25%, Loss of consciousness, Neurologic symptoms, Cardiac ischemia, Pregnancy with COHb >15%) | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Cyanide
| Component | Description | URL |
|-----------|-------------|-----|
| **Sources** | **Smoke inhalation** (most common), **Industrial exposures** (mining, electroplating), **Nitroprusside**, **Amygdalin** (apricot pits, Bitter almonds) | https://www.atsdr.cdc.gov/toxprofiles/tp8.pdf |
| **Mechanism** | **Binds cytochrome c oxidase** (complex IV) ? **Inhibits oxidative phosphorylation** ? **Cellular hypoxia**, **Anaerobic metabolism** ? **Lactic acidosis** | https://www.atsdr.cdc.gov/toxprofiles/tp8.pdf |
| **Symptoms** | **Rapid onset**: **Headache**, **Confusion**, **Seizures**, **Coma**, **Cardiovascular collapse** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Signs** | **Bitter almond breath** (only 40-60% can detect), **Cherry-red skin** (late), **? Anion gap metabolic acidosis**, **? Lactate**, **Normal PaO2 with ? AV O2 difference** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Hydroxocobalamin (Cyanokit)** 5 g IV (preferred), **Sodium thiosulfate** 12.5 g IV (adjunct), **Amyl nitrite + Sodium nitrite + Sodium thiosulfate** (historical) | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Smoke Inhalation
| Component | Management | URL |
|-----------|------------|-----|
| **CO** | **100% O2**, **HBO** (if indicated) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Cyanide** | **Hydroxocobalamin** (empiric if severe lactic acidosis) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Airway Injury** | **Early intubation** (edema peaks 12-24h), **Bronchoscopy** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Pulmonary Injury** | **Supportive care**, **Low tidal volume ventilation** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
