# Toxicology - Antidepressant Toxicity

## Tricyclic Antidepressants (TCAs)
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanisms** | **Sodium channel blockade** (wide QRS), **Anticholinergic**, **Alpha-adrenergic blockade** (hypotension), **GABA-A antagonism** (seizures), **Potassium channel blockade** (QTc prolongation) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Clinical Effects** | **Rapid deterioration** (can progress from alert to comatose/seizing within 1-2h) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **ECG Findings** | **Wide QRS** (>100 ms), **Terminal R wave in aVR** (>3 mm), **R/S ratio in aVR >0.7**, **QTc prolongation** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Wide QRS** | **Predicts seizures** (>100 ms) and **ventricular arrhythmias** (>160 ms) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Sodium bicarbonate** (target pH 7.50-7.55), **IV fluids**, **Vasopressors** (norepinephrine), **Benzodiazepines** (seizures), **Avoid physostigmine**, **Avoid class IA/IC/III antiarrhythmics** | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## SSRIs and SNRIs
| Component | Description | URL |
|-----------|-------------|-----|
| **Toxicity** | **Generally safer** than TCAs; **Serotonin syndrome** (especially with co-ingestants) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Clinical Effects** | **Nausea**, **Vomiting**, **Sedation**, **Tachycardia**, **Serotonin syndrome** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Citalopram/Escitalopram** | **QTc prolongation** (dose-dependent) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/citalopram |
| **Management** | **Supportive care**, **Benzodiazepines** (serotonin syndrome), **Cyproheptadine** (serotonin syndrome) | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Bupropion
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **NDRI** (norepinephrine-dopamine reuptake inhibitor) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/bupropion |
| **Clinical Effects** | **Seizures** (dose-dependent), **Tachycardia**, **Hypertension**, **Wide QRS** (high-dose), **Agitation** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Benzodiazepines** (seizures, Agitation), **Sodium bicarbonate** (wide QRS), **Supportive care** | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Mirtazapine
| Component | Description | URL |
|-----------|-------------|-----|
| **Toxicity** | **Generally safe**; **Sedation**, **Tachycardia**, **Mild hypotension** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Supportive care** | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Venlafaxine
| Component | Description | URL |
|-----------|-------------|-----|
| **Toxicity** | **Serotonin syndrome**, **Seizures**, **Tachycardia**, **Hypertension** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Benzodiazepines** (seizures, Agitation), **Supportive care** | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## MAOIs
| Component | Description | URL |
|-----------|-------------|-----|
| **Toxicity** | **Delayed onset** (6-24h), **Severe serotonin syndrome**, **Hypertensive crisis** (tyramine interaction), **Hyperthermia**, **Rigidity** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Aggressive cooling**, **Benzodiazepines**, **Cyproheptadine**, **Antihypertensives** (phentolamine, nitroprusside) | https://www.acmt.net/Resources/Clinical-Resources.aspx |

## Serotonin Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **Excess serotonin** (5-HT2A agonism) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Triad** | **Neuromuscular hyperactivity** (clonus, hyperreflexia, tremor), **Autonomic instability** (tachycardia, hypertension, hyperthermia), **Altered mental status** (agitation, confusion) | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Hunter Criteria** | **Clonus** (spontaneous, inducible, ocular) + **Hyperreflexia/Tremor** + **Agitation/Diaphoresis** | https://www.mdcalc.com/calc/10450/hunter-serotonin-toxicity-criteria |
| **Culprit Drugs** | **SSRIs**, **SNRIs**, **MAOIs**, **Tramadol**, **Linezolid**, **Methylene blue**, **St. John's Wort**, **MDMA** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
| **Management** | **Discontinue serotonergic agents**, **Benzodiazepines**, **Cyproheptadine** (12 mg load ? 2 mg q2h), **Active cooling**, **IV fluids** | https://www.acmt.net/Resources/Clinical-Resources.aspx |
