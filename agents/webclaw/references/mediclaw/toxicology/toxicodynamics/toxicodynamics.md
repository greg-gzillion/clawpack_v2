# Toxicology - Toxicodynamics

## Overview
Toxicodynamics is the study of **what a toxicant does to the body** (mechanisms of action, dose-response relationships).

| Mechanism | Description | Examples | URL |
|-----------|-------------|----------|-----|
| **Receptor Interactions** | **Agonism** or **Antagonism** at receptors | **Opioids** (ｵ-opioid agonist), **Atropine** (muscarinic antagonist) | https://www.atsdr.cdc.gov/training/toxmanual/modules/3/lecturenotes.html |
| **Enzyme Inhibition** | **Blocking enzyme activity** | **Organophosphates** (acetylcholinesterase), **Cyanide** (cytochrome c oxidase), **Acetaminophen** (cyclooxygenase) | https://www.atsdr.cdc.gov/training/toxmanual/modules/3/lecturenotes.html |
| **Ion Channel Modulation** | **Opening or blocking** ion channels | **Local anesthetics** (Na+ channel block), **Calcium channel blockers** (L-type Ca2+ channel block) | https://www.atsdr.cdc.gov/training/toxmanual/modules/3/lecturenotes.html |
| **Oxidative Stress** | **Reactive oxygen species (ROS)** damage lipids, proteins, DNA | **Acetaminophen** (NAPQI), **Paraquat**, **Carbon tetrachloride** | https://www.atsdr.cdc.gov/training/toxmanual/modules/3/lecturenotes.html |
| **DNA Damage** | **Direct DNA damage** or **Interference with replication/repair** | **Alkylating agents** (cyclophosphamide), **Heavy metals** (arsenic, chromium) | https://www.atsdr.cdc.gov/training/toxmanual/modules/3/lecturenotes.html |

| Type | Description | Example | URL |
