# Oncology - Oncologic Emergencies

## Tumor Lysis Syndrome (TLS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Massive release of intracellular contents (K+, PO4, Uric acid) from lysed tumor cells | https://www.cancer.gov/about-cancer/treatment/side-effects/tumor-lysis-syndrome |
| **Risk Factors** | High tumor burden, Chemosensitive tumors (Burkitt, ALL, AML), Pre-existing renal impairment, Dehydration | https://www.cancer.gov/about-cancer/treatment/side-effects/tumor-lysis-syndrome |
| **Prophylaxis** | **Aggressive hydration**, **Allopurinol** (low/intermediate risk), **Rasburicase** (high risk) | https://www.cancer.gov/about-cancer/treatment/side-effects/tumor-lysis-syndrome |
| **Treatment** | **Rasburicase**, Manage hyperkalemia, **Dialysis** if refractory | https://www.cancer.gov/about-cancer/treatment/side-effects/tumor-lysis-syndrome |

## Hypercalcemia of Malignancy
| Component | Description | URL |
|-----------|-------------|-----|
| **Causes** | **PTHrP** (squamous, Renal, Breast), **Osteolytic metastases** (breast, Lung, Multiple myeloma), **Calcitriol** (lymphoma) | https://www.cancer.gov/about-cancer/treatment/side-effects/hypercalcemia |
| **Treatment** | **Aggressive IVF**, **Calcitonin** (rapid onset), **Bisphosphonates** (Zoledronic acid, Pamidronate), **Denosumab** | https://www.cancer.gov/about-cancer/treatment/side-effects/hypercalcemia |

## Spinal Cord Compression
| Component | Description | URL |
|-----------|-------------|-----|
| **Presentation** | **Back pain** (90%), Weakness, Sensory level, Bowel/bladder dysfunction | https://www.cancer.gov/about-cancer/treatment/side-effects/spinal-cord-compression |
| **Diagnosis** | **Emergent MRI spine** | https://www.cancer.gov/about-cancer/treatment/side-effects/spinal-cord-compression |
| **Treatment** | **Dexamethasone 10 mg IV**, **Radiation** or **Surgery** (if rapid progression, Instability) | https://www.cancer.gov/about-cancer/treatment/side-effects/spinal-cord-compression |

## Superior Vena Cava (SVC) Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| **Presentation** | Facial swelling/plethora, Arm swelling, Dyspnea, Chest wall collateral veins | https://www.cancer.gov/about-cancer/treatment/side-effects/svc-syndrome |
| **Diagnosis** | **CT chest with contrast** | https://www.cancer.gov/about-cancer/treatment/side-effects/svc-syndrome |
| **Treatment** | **Elevate head**, **Diuretics**, **Steroids**, **Radiation** or **Chemotherapy** or **Stenting** | https://www.cancer.gov/about-cancer/treatment/side-effects/svc-syndrome |

## Febrile Neutropenia

|-----------|-------------|-----|
