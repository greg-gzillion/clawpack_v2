# Oncology - Palliative Care in Oncology

## Palliative Care - Principles
| Principle | Description | URL |
|-----------|-------------|-----|
| **Definition** | Interdisciplinary care focused on improving **quality of life** for patients with **serious illness** at any stage | https://www.cancer.gov/about-cancer/advanced-cancer/care-choices/palliative-care-fact-sheet |
| **Timing** | **Early integration** with oncology care (improves outcomes) | https://www.cancer.gov/about-cancer/advanced-cancer/care-choices/palliative-care-fact-sheet |
| **Domains** | **Physical** (pain, Symptoms), **Psychological**, **Social**, **Spiritual** | https://www.cancer.gov/about-cancer/advanced-cancer/care-choices/palliative-care-fact-sheet |

## Opioid Prescribing in Cancer Pain
| Principle | Description | URL |
|-----------|-------------|-----|
| **Route** | **Oral** preferred; Transdermal (stable pain); Subcutaneous/IV (unable to take PO) | https://www.cancer.gov/about-cancer/treatment/side-effects/pain |
| **Scheduled + PRN** | **Long-acting scheduled** + **Short-acting PRN** (10-20% of total daily dose) | https://www.cancer.gov/about-cancer/treatment/side-effects/pain |
| **Dose Escalation** | Increase by **25-50%** for uncontrolled pain | https://www.cancer.gov/about-cancer/treatment/side-effects/pain |
| **Opioid Rotation** | If toxicity or inadequate analgesia; Reduce new opioid by **25-50%** for incomplete cross-tolerance | https://www.cancer.gov/about-cancer/treatment/side-effects/pain |
| **Constipation** | **Scheduled stimulant laxative** (senna) + Stool softener (docusate) | https://www.cancer.gov/about-cancer/treatment/side-effects/constipation |
| **Naloxone Co-Prescribing** | Consider for all patients on chronic opioids | https://www.cancer.gov/about-cancer/treatment/side-effects/pain |

## Common Symptoms in Advanced Cancer
| Symptom | Management | URL |
|---------|------------|-----|
| **Dyspnea** | **Opioids** (first-line), **Fan**, **Positioning**, **Oxygen** (if hypoxic), **Benzodiazepines** (anxiety) | https://www.cancer.gov/about-cancer/treatment/side-effects/breathing-problems |
| **Nausea/Vomiting** | Identify cause; **Haloperidol**, **Metoclopramide**, **Ondansetron**, **Prochlorperazine**, **Dexamethasone** | https://www.cancer.gov/about-cancer/treatment/side-effects/nausea |
| **Delirium** | **Haloperidol**, **Quetiapine**, **Olanzapine**; **AVOID benzodiazepines** | https://www.cancer.gov/about-cancer/treatment/side-effects/delirium |
| **Fatigue** | **Exercise**, **Energy conservation**, **Psychostimulants** (methylphenidate) for selected patients | https://www.cancer.gov/about-cancer/treatment/side-effects/fatigue |
| **Anorexia/Cachexia** | **Corticosteroids**, **Megestrol acetate**, **Olanzapine**, **Mirtazapine** | https://www.cancer.gov/about-cancer/treatment/side-effects/appetite-loss |
