# Oncology - Surgical Oncology

## Surgical Oncology - Principles
| Principle | Description | URL |
|-----------|-------------|-----|
| **Curative Intent** | Complete resection with **negative margins (R0)** | https://www.cancer.gov/about-cancer/treatment/types/surgery |
| **Staging** | Lymph node evaluation, Peritoneal washings, Biopsy of suspicious areas | https://www.cancer.gov/about-cancer/treatment/types/surgery |
| **Palliative Intent** | Relieve symptoms (obstruction, Bleeding, Pain) | https://www.cancer.gov/about-cancer/treatment/types/surgery |
| **Debulking** | Remove as much tumor as possible (ovarian, Neuroendocrine) | https://www.cancer.gov/about-cancer/treatment/types/surgery |

## Sentinel Lymph Node Biopsy (SLNB)
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | **Breast cancer** (clinically node-negative), **Melanoma** (=T1b), **Vulvar cancer**, **Penile cancer**, **Merkel cell carcinoma** | https://www.cancer.gov/about-cancer/treatment/types/surgery/sentinel-node-biopsy |
| **Technique** | Injection of **radioactive tracer** and/or **blue dye** ? Identify first draining lymph node(s) ? Remove for pathologic evaluation | https://www.cancer.gov/about-cancer/treatment/types/surgery/sentinel-node-biopsy |
| **Positive SLN** | **Completion lymph node dissection** (or observation in select cases - breast, Melanoma) | https://www.cancer.gov/about-cancer/treatment/types/surgery/sentinel-node-biopsy |

## Common Oncologic Surgeries
| Surgery | Description | Cancers | URL |
|---------|-------------|---------|-----|
| **Lobectomy** | Removal of lung lobe | Early NSCLC | https://www.cancer.gov/types/lung/patient/lung-treatment-pdq |
| **Whipple Procedure (Pancreaticoduodenectomy)** | Removal of pancreatic head, Duodenum, Gallbladder, CBD ± Distal stomach | Pancreatic head, Distal CBD, Ampullary, Duodenal | https://www.cancer.gov/types/pancreatic/patient/pancreatic-treatment-pdq |
| **Radical Cystectomy** | Removal of bladder, Prostate/seminal vesicles (men), Uterus/ovaries/anterior vagina (women) + PLND | Muscle-invasive bladder cancer | https://www.cancer.gov/types/bladder/patient/bladder-treatment-pdq |
| **Radical Prostatectomy** | Removal of prostate + seminal vesicles ± PLND | Localized prostate cancer | https://www.cancer.gov/types/prostate/patient/prostate-treatment-pdq |
| **Total Mesorectal Excision (TME)** | En bloc resection of rectum and mesorectum | Rectal cancer | https://www.cancer.gov/types/colorectal/patient/rectal-treatment-pdq |
| **Cytoreductive Surgery + HIPEC** | Maximal tumor debulking + Heated intraperitoneal chemotherapy | Peritoneal carcinomatosis (ovarian, Colorectal, Appendiceal, Mesothelioma) | https://www.cancer.gov/about-cancer/treatment/types/surgery/hipec |
