# Oncology - Hereditary Cancer Syndromes

## Major Hereditary Cancer Syndromes
| Syndrome | Gene(s) | Key Cancers | URL |
|----------|---------|-------------|-----|
| **Hereditary Breast and Ovarian Cancer (HBOC)** | **BRCA1**, **BRCA2**, **PALB2** | Breast, Ovarian, Pancreatic, Prostate | https://www.cancer.gov/about-cancer/causes-prevention/genetics/brca-fact-sheet |
| **Lynch Syndrome** | **MLH1**, **MSH2**, **MSH6**, **PMS2**, **EPCAM** | Colorectal, Endometrial, Ovarian, Gastric, Urothelial, Pancreatic | https://www.cancer.gov/about-cancer/causes-prevention/genetics/lynch-syndrome |
| **Familial Adenomatous Polyposis (FAP)** | **APC** | Colorectal (>100 adenomas), Duodenal, Desmoid, Thyroid | https://www.cancer.gov/about-cancer/causes-prevention/genetics/fap |
| **Li-Fraumeni Syndrome** | **TP53** | Breast, Sarcoma, Brain, Adrenocortical, Leukemia | https://www.cancer.gov/about-cancer/causes-prevention/genetics/li-fraumeni-syndrome |
| **Cowden Syndrome** | **PTEN** | Breast, Thyroid, Endometrial, Renal | https://www.cancer.gov/about-cancer/causes-prevention/genetics/cowden-syndrome |
| **Peutz-Jeghers Syndrome** | **STK11** | GI, Breast, Pancreatic, Ovarian, Cervical, Testicular | https://www.cancer.gov/about-cancer/causes-prevention/genetics/peutz-jeghers |
| **Von Hippel-Lindau** | **VHL** | RCC, Hemangioblastoma, Pheochromocytoma | https://www.cancer.gov/about-cancer/causes-prevention/genetics/vhl |
| **Multiple Endocrine Neoplasia Type 1** | **MEN1** | Parathyroid, Pancreatic NET, Pituitary | https://www.cancer.gov/about-cancer/causes-prevention/genetics/men1 |
| **Multiple Endocrine Neoplasia Type 2** | **RET** | Medullary thyroid cancer, Pheochromocytoma, Parathyroid | https://www.cancer.gov/about-cancer/causes-prevention/genetics/men2 |
| **Hereditary Diffuse Gastric Cancer** | **CDH1** | Diffuse gastric cancer, Lobular breast cancer | https://www.cancer.gov/about-cancer/causes-prevention/genetics/hdgc-fact-sheet |
| **Familial Melanoma** | **CDKN2A**, **CDK4** | Melanoma, Pancreatic | https://www.cancer.gov/about-cancer/causes-prevention/genetics/familial-melanoma |
| **Juvenile Polyposis Syndrome** | **SMAD4**, **BMPR1A** | Colorectal, Gastric | https://www.cancer.gov/about-cancer/causes-prevention/genetics/juvenile-polyposis |

## Genetic Testing - Indications
| Clinical Scenario | Recommendation | URL |
|------------------|----------------|-----|
| **Breast cancer =50** | Offer BRCA1/2 testing | https://www.nccn.org/guidelines/genetics |
| **Triple-negative breast cancer =60** | Offer BRCA1/2 testing | https://www.nccn.org/guidelines/genetics |
| **Ovarian/Fallopian tube/Peritoneal cancer** | Offer BRCA1/2, Lynch testing | https://www.nccn.org/guidelines/genetics |
| **Colorectal cancer =50** | Offer Lynch testing (MSI/IHC) | https://www.nccn.org/guidelines/genetics |
| **Endometrial cancer =50** | Offer Lynch testing | https://www.nccn.org/guidelines/genetics |
| **=10 colorectal adenomas** | Offer FAP/MAP testing | https://www.nccn.org/guidelines/genetics |
| **Medullary thyroid cancer** | Offer RET testing | https://www.nccn.org/guidelines/genetics |
| **Pheochromocytoma/Paraganglioma** | Offer SDHx, VHL, RET testing | https://www.nccn.org/guidelines/genetics |
| **Male breast cancer** | Offer BRCA1/2 testing | https://www.nccn.org/guidelines/genetics |
| **Ashkenazi Jewish ancestry + Breast/ovarian/pancreatic** | Offer BRCA1/2 testing | https://www.nccn.org/guidelines/genetics |
