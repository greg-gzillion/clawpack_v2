# Cardiology - Heart Failure

## Definition and Classification
| Type | Definition | URL |
|------|------------|-----|
| Heart Failure with Reduced EF (HFrEF) | LVEF =40% | https://www.acc.org/guidelines |
| Heart Failure with Improved EF (HFimpEF) | Previous LVEF =40%; follow-up LVEF >40% | https://www.acc.org/guidelines |
| Heart Failure with Mildly Reduced EF (HFmrEF) | LVEF 41-49% | https://www.acc.org/guidelines |
| Heart Failure with Preserved EF (HFpEF) | LVEF =50% with evidence of increased filling pressures | https://www.acc.org/guidelines |

## NYHA Functional Classification
| Class | Description | URL |
|-------|-------------|-----|
| Class I | No limitation of physical activity | https://www.heart.org |
| Class II | Slight limitation; comfortable at rest; ordinary activity causes symptoms | https://www.heart.org |
| Class III | Marked limitation; comfortable at rest; less than ordinary activity causes symptoms | https://www.heart.org |
| Class IV | Unable to carry on any physical activity without discomfort; symptoms at rest | https://www.heart.org |

## ACC/AHA Stages of Heart Failure
| Stage | Description | URL |
|-------|-------------|-----|
| Stage A | At risk for HF but without structural heart disease or symptoms | https://www.acc.org/guidelines |
| Stage B | Structural heart disease but without signs/symptoms of HF | https://www.acc.org/guidelines |
| Stage C | Structural heart disease with prior or current symptoms of HF | https://www.acc.org/guidelines |
| Stage D | Refractory HF requiring specialized interventions | https://www.acc.org/guidelines |
