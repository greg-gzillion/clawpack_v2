## Coarctation of Aorta
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Congenital narrowing of aorta (typically juxtaductal) | https://www.acc.org/guidelines |
| Presentation | Upper extremity HTN; lower extremity hypotension; radial-femoral delay | https://www.acc.org/guidelines |
| Imaging | Echo (suprasternal notch); CT/MR angiography | https://www.acc.org/guidelines |
| Intervention Threshold | Peak gradient =20 mmHg or gradient <20 mmHg with evidence of collateral flow and HTN | https://www.acc.org/guidelines |
| Treatment | Stenting (adults); Surgery (infants) | https://www.acc.org/guidelines |
| Follow-up | Lifelong; monitor for recoarctation; aneurysm; HTN | https://www.acc.org/guidelines |

## Tetralogy of Fallot (TOF) - Repaired
| Feature | Description | URL |
|---------|-------------|-----|
| Components | VSD; RVOT obstruction; Overriding aorta; RV hypertrophy | https://www.acc.org/guidelines |
| Long-Term Issues | Pulmonary regurgitation (most common); RV dilation; RV dysfunction; Arrhythmias; Sudden death | https://www.acc.org/guidelines |
| PVR Indications | Severe PR with symptoms; Severe PR with RV dilation (RVEDV >160 mL/m2 or RVESV >80 mL/m2); Moderate-severe PR with other lesions | https://www.acc.org/guidelines |
| Surveillance | Echo; Cardiac MRI; Holter; Exercise testing | https://www.acc.org/guidelines |

## Ebstein Anomaly
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Apical displacement of tricuspid valve leaflets; atrialized RV | https://www.acc.org/guidelines |
| Associations | ASD/PFO (80-90%); WPW syndrome (10-20%) | https://www.acc.org/guidelines |
| Management | Surgery if symptomatic or progressive RV dysfunction | https://www.acc.org/guidelines |
| Avoid | Pregnancy if cyanotic or symptomatic | https://www.acc.org/guidelines |

## Congenitally Corrected Transposition of Great Arteries (ccTGA)
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | AV and VA discordance (RA ? LV ? PA; LA ? RV ? Aorta) | https://www.acc.org/guidelines |
| Associated Lesions | VSD (70%); Pulmonary stenosis (40%); Systemic AV valve (tricuspid) regurgitation | https://www.acc.org/guidelines |
| Long-Term Issues | Systemic RV failure; TR; Heart block | https://www.acc.org/guidelines |
| Management | GDMT for systemic RV failure; Pacemaker for heart block; Tricuspid valve surgery | https://www.acc.org/guidelines |

## Fontan Circulation
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Total cavopulmonary connection for single ventricle physiology | https://www.acc.org/guidelines |
| Long-Term Complications | Arrhythmias; Thromboembolism; Fontan-associated liver disease (FALD); Protein-losing enteropathy (PLE); Plastic bronchitis; Heart failure | https://www.acc.org/guidelines |
| Anticoagulation | Warfarin or ASA (individualized) | https://www.acc.org/guidelines |
| Surveillance | Echo; Cardiac MRI; Liver imaging; Exercise testing | https://www.acc.org/guidelines |

## Eisenmenger Syndrome
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Pulmonary hypertension with reversed/ bidirectional shunt due to uncorrected CHD | https://www.acc.org/guidelines |
| Management | Avoid volume depletion; Phlebotomy only for hyperviscosity symptoms; Pulmonary vasodilators (Bosentan; Sildenafil); Avoid pregnancy | https://www.acc.org/guidelines |
| Prognosis | Guarded; median survival 40-50 years | https://www.acc.org/guidelines |
