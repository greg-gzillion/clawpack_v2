# Cardiology - Pericardial Disease

## Acute Pericarditis
| Feature | Description | URL |
|---------|-------------|-----|
| Diagnostic Criteria | =2 of: Chest pain; Pericardial friction rub; ECG changes (diffuse ST elevation; PR depression); Pericardial effusion | https://www.acc.org/guidelines |
| Chest Pain | Sharp; pleuritic; positional (improves leaning forward) | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| ECG | Stage I: Diffuse ST elevation; PR depression; Stage II: Normalization; Stage III: T-wave inversion; Stage IV: Normalization | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Treatment | NSAIDs (Ibuprofen 600-800 mg TID) + Colchicine 0.5-1 mg daily for 3 months; Treat underlying cause | https://www.acc.org/guidelines |
| Avoid | Corticosteroids (unless refractory or autoimmune) | https://www.acc.org/guidelines |

## Recurrent Pericarditis
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Recurrence after symptom-free interval of =4-6 weeks | https://www.acc.org/guidelines |
| Treatment | NSAIDs + Colchicine; Consider Anakinra (IL-1 inhibitor) if refractory; Avoid steroids | https://www.acc.org/guidelines |

## Cardiac Tamponade
| Feature | Description | URL |
|---------|-------------|-----|
| Pathophysiology | Pericardial pressure exceeds cardiac filling pressures ? decreased CO | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Beck's Triad | Hypotension; Muffled heart sounds; JVD | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Pulsus Paradoxus | SBP decrease >10 mmHg during inspiration | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Echo Findings | RA/RV diastolic collapse; IVC plethora; Respiratory variation in MV/TV inflow | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Treatment | Pericardiocentesis; Pericardial window | https://www.acc.org/guidelines |

## Constrictive Pericarditis
| Feature | Description | URL |
|---------|-------------|-----|
| Pathophysiology | Thickened, non-compliant pericardium restricts diastolic filling | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Causes | Idiopathic; Post-cardiac surgery; Radiation; Tuberculosis | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Physical Exam | Kussmaul's sign (JVP rises with inspiration); Pericardial knock | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Echo Findings | Septal bounce; Respiratory variation in MV inflow; Preserved medial e'; Annulus reversus | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Hemodynamics | Equalization of diastolic pressures; Dip-and-plateau ("square root") sign | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Treatment | Pericardiectomy | https://www.acc.org/guidelines |

## Constrictive Pericarditis vs. Restrictive Cardiomyopathy
| Parameter | Constrictive Pericarditis | Restrictive Cardiomyopathy | URL |
|-----------|--------------------------|---------------------------|-----|
| Septal Bounce | Present | Absent | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Medial e' | Preserved (>7-9 cm/s) | Reduced (<7 cm/s) | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Annulus Reversus | Present | Absent | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Hepatic Vein Flow | Expiratory diastolic reversal | Inspiratory diastolic reversal | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| BNP | Normal-mildly elevated | Markedly elevated | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
