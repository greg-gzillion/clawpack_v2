# Cardiology - Cardiomyopathies

## Dilated Cardiomyopathy (DCM)
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | LV dilation and systolic dysfunction (LVEF <50%) not explained by CAD or loading conditions | https://www.acc.org/guidelines |
| Causes | Idiopathic (50%); Genetic (TTN, LMNA, MYH7); Myocarditis; Alcohol; Chemotherapy; Peripartum | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Diagnosis | Echo; Cardiac MRI (LGE); Genetic testing; Endomyocardial biopsy (selected) | https://www.acc.org/guidelines |
| Management | GDMT for HFrEF; ICD if LVEF =35%; CRT if LBBB with QRS =150 ms | https://www.acc.org/guidelines |

## Hypertrophic Cardiomyopathy (HCM)
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | LV wall thickness =15 mm (or =13 mm with family history) not explained by loading conditions | https://www.acc.org/guidelines |
| Genetics | Autosomal dominant; sarcomeric protein mutations (MYH7, MYBPC3 most common) | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Pathophysiology | LVOT obstruction (70%); Diastolic dysfunction; Mitral regurgitation (SAM) | https://www.acc.org/guidelines |
| Symptoms | Dyspnea; angina; syncope; sudden cardiac death | https://www.acc.org/guidelines |
| Diagnosis | Echo (asymmetric septal hypertrophy; SAM; LVOT gradient); Cardiac MRI | https://www.acc.org/guidelines |

## HCM Management
| Treatment | Indications | URL |
|-----------|-------------|-----|
| Beta-Blockers | Symptomatic (first-line) | https://www.acc.org/guidelines |
| Non-Dihydropyridine CCB | Symptomatic if beta-blockers ineffective/contraindicated | https://www.acc.org/guidelines |
| Disopyramide | Refractory symptoms | https://www.acc.org/guidelines |
| Mavacamten | Symptomatic obstructive HCM | https://www.acc.org/guidelines |
| Septal Reduction Therapy | Severe symptoms despite medical therapy; LVOT gradient =50 mmHg | https://www.acc.org/guidelines |
| ICD | Primary prevention (5-year SCD risk =4-6% or major risk factors); Secondary prevention | https://www.acc.org/guidelines |
| Avoid | Vasodilators; Dehydration; Competitive sports | https://www.acc.org/guidelines |

## Arrhythmogenic Right Ventricular Cardiomyopathy (ARVC)
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Progressive fibrofatty replacement of RV myocardium | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Genetics | Desmosomal mutations (PKP2, DSP, DSG2, DSC2, JUP) | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Diagnosis | Modified Task Force Criteria (Echo; CMR; ECG; Holter; Family history; Genetics) | https://www.acc.org/guidelines |
| ECG Findings | Epsilon wave; T-wave inversion V1-V3; Prolonged S-wave upstroke | https://www.acc.org/guidelines |
| Management | Beta-blockers; ICD (high-risk); Exercise restriction | https://www.acc.org/guidelines |

## Restrictive Cardiomyopathy
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Restrictive filling with normal/reduced diastolic volumes; normal wall thickness; preserved EF | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Causes | Amyloidosis; Sarcoidosis; Hemochromatosis; Endomyocardial fibrosis; Radiation | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Diagnosis | Echo (restrictive filling; biatrial enlargement); Cardiac MRI; Biopsy | https://www.acc.org/guidelines |
| Management | Treat underlying cause; Diuretics for congestion | https://www.acc.org/guidelines |

## Takotsubo Cardiomyopathy (Stress Cardiomyopathy)
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Transient LV dysfunction (apical ballooning) triggered by emotional/physical stress | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Presentation | Chest pain; dyspnea; ECG changes (ST elevation); elevated troponin | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Diagnosis | Mayo Criteria: Transient wall motion abnormality; Absence of obstructive CAD; New ECG changes/troponin; No pheochromocytoma/myocarditis | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Management | Supportive; Beta-blockers; ACE-I; Avoid catecholamines | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Prognosis | Generally excellent; recovery within weeks | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |

## Peripartum Cardiomyopathy
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | LVEF <45% developing in last month of pregnancy or within 5 months postpartum | https://www.acc.org/guidelines |
| Risk Factors | Advanced maternal age; Multiparity; Multiple gestation; Preeclampsia; African ancestry | https://www.acc.org/guidelines |
| Management | GDMT (avoid ACE-I/ARB during pregnancy); Anticoagulation; Early delivery if severe | https://www.acc.org/guidelines |
| Prognosis | 50-60% recover LV function; 10-20% mortality | https://www.acc.org/guidelines |
