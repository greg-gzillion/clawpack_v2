# Geriatrics - Renal Aging and Disease

## Renal Changes with Aging
| Change | Clinical Implication | URL |
|--------|---------------------|-----|
| Decreased GFR | ~1 mL/min/year decline after age 40; Reduced drug clearance | https://www.kidney.org |
| Decreased renal blood flow | Increased ischemic injury risk | https://www.asn-online.org |
| Impaired sodium conservation | Hypovolemia risk with salt restriction, Diuretics | |
| Impaired free water excretion | Hyponatremia risk | https://www.niddk.nih.gov |
| Impaired potassium excretion | Hyperkalemia risk with ACE-I, ARB, MRA, K+ supplements | |
| Decreased renin and aldosterone | Type IV renal tubular acidosis (hyperkalemic, normal AG metabolic acidosis) | |
| Decreased vitamin D activation | Osteoporosis, Secondary hyperparathyroidism | https://www.bonehealthandosteoporosis.org |
| Decreased erythropoietin | Anemia of chronic kidney disease | |

## Estimating GFR in Older Adults
| Equation | Variables | Considerations | URL |
|----------|-----------|----------------|-----|
| **CKD-EPI (2021)** | Cr, Age, Sex | **Race-free**; More accurate at higher GFR | https://www.kidney.org/professionals/kdoqi/gfr_calculator |
| **MDRD** | Cr, Age, Sex, Race | Less accurate at GFR >60 | |
| **Cockcroft-Gault** | Cr, Age, Weight, Sex | **Used for drug dosing**; Overestimates in frail/elderly | |
| **Cystatin C** | Cystatin C, Age, Sex | Less influenced by muscle mass; **Preferred in sarcopenia** | |

## Chronic Kidney Disease (CKD) Staging (KDIGO)
| Stage | GFR (mL/min/1.73mý) | Description | Management | URL |
|-------|---------------------|-------------|------------|-----|
| G1 | =90 | Kidney damage with normal GFR | Treat underlying cause, CVD risk reduction | https://kdigo.org |
| G2 | 60-89 | Mildly decreased GFR | Same as G1 | |
| G3a | 45-59 | Mildly to moderately decreased | Monitor progression, Complications | |
| G3b | 30-44 | Moderately to severely decreased | Prepare for renal replacement therapy | |
| G4 | 15-29 | Severely decreased | Renal replacement planning | |
| G5 | <15 | Kidney failure | Dialysis or transplant | |

## CKD Management in Older Adults
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **BP Control** | <130/80 mmHg (if tolerated); ACE-I or ARB first-line if albuminuria | https://kdigo.org |
| **Glycemic Control** | Individualized HbA1c (7-8% for most with CKD) | https://www.diabetes.org |
| **SGLT2 Inhibitors** | **Strongly recommended** for CKD with albuminuria (regardless of diabetes) | https://kdigo.org/diabetes-guideline |
| **Statins** | Moderate-high intensity for age 50+ with CKD (not on dialysis) | https://www.acc.org |
| **Anemia Management** | Iron supplementation; ESA if Hb <10 g/dL (target 10-11.5 g/dL) | |
| **Mineral Bone Disorder** | Phosphate binders, Vitamin D, Calcimimetics | |
| **Acidosis Management** | Sodium bicarbonate if HCO3 <22 mmol/L | |
| **Medication Dose Adjustment** | **Critical**; Adjust for GFR (especially DOACs, Antibiotics, Opioids, Gabapentinoids) | https://www.fda.gov |

## Acute Kidney Injury (AKI) in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| Increased Susceptibility | Decreased renal reserve, Polypharmacy, Comorbidities | https://www.asn-online.org |
| Common Causes | **Prerenal** (dehydration, diuretics, ACE-I/ARB, NSAIDs), **Intrinsic** (ATN, AIN), **Postrenal** (BPH, Obstruction) | https://www.kidney.org |
| Drug-Induced AKI | **NSAIDs** (avoid), **ACE-I/ARB** (hold during illness), **Diuretics**, **Aminoglycosides**, **Contrast** | |
| Prevention | Avoid nephrotoxins, Maintain hydration, Hold ACE-I/ARB/diuretics during illness | |
| Management | Treat underlying cause, Fluid resuscitation, Avoid further injury, Dialysis if indicated (AEIOU) | |

## Hyponatremia in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| Prevalence | 10-20% in community; 30-40% in hospitalized/nursing home | https://www.niddk.nih.gov |
| Risk Factors | **Thiazide diuretics**, **SSRIs**, SIADH, Hypothyroidism, Adrenal insufficiency, CHF, Cirrhosis | |
| Symptoms | Confusion, Falls, Gait instability, Seizures, Coma | |
| Diagnosis | Assess volume status, Urine osmolality, Urine sodium | |
| SIADH | Euvolemic, Urine osmolality >100, Urine Na >40 | |
| Management (Acute, Symptomatic) | **Hypertonic saline (3%)**; Correct slowly (=8 mEq/L/24h) | https://www.kidney.org |
| Management (Chronic, Asymptomatic) | Fluid restriction, Salt tablets, Loop diuretics, Vaptans (tolvaptan) | |
| Thiazide-Induced | Discontinue thiazide, Replete sodium cautiously | |

## Hypernatremia in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| Causes | Dehydration (decreased thirst, Impaired access to water), Diabetes insipidus, Osmotic diuresis | https://www.niddk.nih.gov |
| Risk Factors | Dementia, Functional impairment, Febrile illness, Tube feeding | |
| Management | **Free water deficit** = 0.5 x weight (kg) x (Na/140 - 1) for women; 0.6 for men | |
| Correction Rate | Correct half of deficit over 24 hours; =0.5 mEq/L/hr | |

## Benign Prostatic Hyperplasia (BPH) and Urinary Retention
| Component | Description | URL |
|-----------|-------------|-----|
| Prevalence | 50% of men =60; 90% of men =85 | https://www.auanet.org |
| Symptoms | Hesitancy, Weak stream, Incomplete emptying, Nocturia, Frequency | |
| AUA Symptom Score | 0-7 Mild; 8-19 Moderate; 20-35 Severe | |
| Medical Therapy | **Alpha-blockers** (Tamsulosin, Alfuzosin); **5-ARI** (Finasteride, Dutasteride); Combination | |
| Acute Urinary Retention | **Bladder catheterization**; Alpha-blocker; Monitor for post-obstructive diuresis | |
| Surgical Options | TURP, Laser prostatectomy, Urolift | |
