# Geriatrics - Pain Management

## Pain in Older Adults - Epidemiology
| Statistic | Value | URL |
|-----------|-------|-----|
| Persistent pain in community-dwelling | 25-50% | https://www.americangeriatrics.org |
| Persistent pain in long-term care | 50-80% | https://paltc.org |
| Common causes | Osteoarthritis, Low back pain, Neuropathy, Cancer, Post-herpetic neuralgia | |
| Consequences | Depression, Sleep disturbance, Functional decline, Falls, Social isolation, Polypharmacy | |
| **Underassessment** | **Cognitive impairment, Sensory impairment, Belief that pain is normal aging** | |

## Pain Assessment in Older Adults
| Population | Assessment Tools | URL |
|------------|------------------|-----|
| **Cognitively Intact** | **Numeric Rating Scale (NRS)** 0-10; Verbal Descriptor Scale; **Pain Thermometer** | https://www.iasp-pain.org |
| **Cognitive Impairment (Mild-Moderate)** | **Faces Pain Scale-Revised (FPS-R)**, **Iowa Pain Thermometer**, Verbal Descriptor Scale | |
| **Severe Cognitive Impairment/Non-Verbal** | **PAINAD** (Pain Assessment in Advanced Dementia), **PACSLAC**, **DOLOPLUS-2** | https://www.alz.org |
| **Key Behaviors Suggesting Pain** | Facial grimacing, Moaning, Guarding, Rigidity, Agitation, Resistance to care, Decreased appetite, Sleep disturbance | |

## PAINAD Scale (Pain Assessment in Advanced Dementia)
| Item | Scoring (0, 1, 2) | URL |
|------|------------------|-----|
| Breathing | Normal; Occasional labored; Noisy labored/hyperventilation | https://www.geriatricpain.org |
| Negative Vocalization | None; Occasional moan/groan; Repeated calling out/Loud moaning | |
| Facial Expression | Smiling/Neutral; Sad/Frightened/Frowning; Grimacing | |
| Body Language | Relaxed; Tense/Distressed/Fidgeting; Rigid/Fists clenched/Knees pulled up/Pushing | |
| Consolability | No need to console; Distracted/Reassured; Unable to console | |
| **Interpretation** | **Total score 0-10**; =2 suggests pain | |

## Pain Management - General Principles
| Principle | Description | URL |
|-----------|-------------|-----|
| **Start low, go slow** | Lower initial doses; Slower titration | https://www.americangeriatrics.org/pain-guidelines |
| **Multimodal approach** | Combine pharmacologic and non-pharmacologic interventions | |
| **Non-pharmacologic first** | Physical therapy, Heat/Cold, Massage, Acupuncture, CBT, Exercise | |
| **Avoid NSAIDs** | **Use with extreme caution** (GI bleeding, AKI, Hypertension, CHF exacerbation) | https://www.americangeriatrics.org/beers |
| **Avoid muscle relaxants** | Cyclobenzaprine, Methocarbamol (anticholinergic, Sedation, Falls) | |
| **Reassess regularly** | Pain intensity, Functional status, Adverse effects | |

## Pharmacologic Pain Management
| Drug Class | Examples | Considerations in Older Adults | URL |
|------------|----------|-------------------------------|-----|
| **Acetaminophen** | Tylenol | **First-line for mild-moderate pain** (OA, Low back pain); Max 3g/day (frail) or 2g/day (liver disease) | https://www.fda.gov |
| **Topical NSAIDs** | Diclofenac gel | **Preferred over oral NSAIDs**; Local effect, Minimal systemic absorption | |
| **Oral NSAIDs** | Ibuprofen, Naproxen, Celecoxib | **Avoid if possible**; If necessary: Lowest dose, Shortest duration, With PPI | |
| **Tramadol** | Tramadol | Weak opioid + SNRI; **Increased fall risk**; Seizure risk; Serotonin syndrome | |
| **Gabapentinoids** | Gabapentin, Pregabalin | **First-line for neuropathic pain**; **Reduce dose in CKD**; Dizziness, Somnolence, Edema, **Increased fall risk** | |
| **SNRIs** | Duloxetine | **First-line for neuropathic pain**, OA, Low back pain; Monitor for hyponatremia | |
| **TCAs** | Nortriptyline, Amitriptyline | **Avoid amitriptyline** (Beers); Nortriptyline (lower anticholinergic) for neuropathic pain | |
| **Opioids** | Morphine, Oxycodone, Hydromorphone | **Last resort** for non-cancer pain; **Start low, go slow**; **Bowel regimen** required; **Increased fall risk**, Delirium, Respiratory depression | |
| **Topical Lidocaine** | Lidocaine 5% patch | **First-line for localized neuropathic pain** (post-herpetic neuralgia); Well-tolerated | |
| **Topical Capsaicin** | Capsaicin 8% patch | Neuropathic pain; Burning sensation initially | |

## Opioid Prescribing in Older Adults
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **Indication** | **Severe pain unresponsive to non-opioid therapies**; Cancer pain; End-of-life pain | https://www.cdc.gov/opioids |
| **Risk Assessment** | Assess for misuse risk (opioid use disorder, Aberrant behaviors) | |
| **Informed Consent** | Discuss risks (falls, Delirium, Respiratory depression, Constipation, Tolerance, Dependence) | |
| **Start Low, Go Slow** | **50% reduction** in starting dose; Titrate slowly | |
| **Avoid Codeine, Tramadol** | Variable metabolism; Limited efficacy | |
| **Preferred Opioids** | **Morphine, Oxycodone, Hydromorphone** (immediate-release initially) | |
| **Avoid Meperidine** | Neurotoxic metabolite (normeperidine); Seizure risk | |
| **Bowel Regimen** | **Scheduled stimulant laxative** (senna) + Stool softener (docusate) | |
| **Naloxone Co-Prescribing** | Consider for all patients on chronic opioids | |
| **Monitoring** | Pain relief, Function, Adverse effects, Misuse (PMP check, Urine drug screen) | |
| **Tapering** | **Do not abruptly stop**; Taper by 10% per month if stable | |
