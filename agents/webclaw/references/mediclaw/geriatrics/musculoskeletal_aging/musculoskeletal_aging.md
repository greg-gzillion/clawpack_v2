## Osteoarthritis (OA) in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | Most common joint disorder; 50% of adults =65 have radiographic OA | https://www.rheumatology.org |
| Common Sites | Knees, Hips, Hands (DIP, PIP, First CMC), Spine | |
| Risk Factors | Age, Obesity, Female sex, Genetics, Joint injury, Occupation | |
| Diagnosis | Clinical (pain, stiffness <30 min, crepitus, bony enlargement); X-ray (joint space narrowing, osteophytes, subchondral sclerosis) | |
| Non-Pharmacologic | **Exercise** (aerobic, strengthening, aquatic), **Weight loss**, Physical therapy, Assistive devices, Bracing, Tai Chi | https://www.arthritis.org |
| Pharmacologic | **Topical NSAIDs** (first-line), **Oral NSAIDs** (lowest effective dose, shortest duration, with PPI), **Acetaminophen** (modest benefit), **Duloxetine**, **Intra-articular corticosteroids** (short-term relief) | https://www.acrannualmeeting.org |
| **AVOID** | Opioids (chronic), Glucosamine/Chondroitin (no benefit), Hyaluronic acid injections (controversial) | |
| Surgical | **Total joint arthroplasty** for severe, refractory OA; Assess surgical risk, Frailty, Cognitive function | https://www.aaos.org |

## Polymyalgia Rheumatica (PMR)
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | Almost exclusively >50 years; Peak incidence 70-80 years; Female > Male | https://www.rheumatology.org |
| Presentation | **Bilateral shoulder and hip girdle pain/stiffness**, Morning stiffness >45 min, Constitutional symptoms (fatigue, weight loss, low-grade fever) | |
| Diagnosis | **Elevated ESR (often >40) and/or CRP**; Clinical diagnosis; Response to steroids | |
| Imaging | Ultrasound/MRI: Subacromial/subdeltoid bursitis, Hip synovitis | |
| Treatment | **Prednisone 12.5-25 mg daily** (rapid, dramatic improvement); Taper slowly over 12-18 months | |
| Monitoring | Symptoms, ESR/CRP; Watch for **Giant Cell Arteritis** (15-20% have or develop GCA) | |

## Giant Cell Arteritis (GCA) / Temporal Arteritis
| Component | Description | URL |
|-----------|-------------|-----|
| Presentation | **New headache** (temporal), Scalp tenderness, **Jaw claudication**, **Visual symptoms** (amaurosis fugax, diplopia, vision loss), PMR symptoms (50%) | https://www.rheumatology.org |
| Diagnosis | **Elevated ESR (often >50) and CRP**; **Temporal artery biopsy** (gold standard, but may be negative); Temporal artery ultrasound (halo sign) | |
| Treatment | **High-dose steroids** (Prednisone 40-60 mg daily or Methylprednisolone 1g IV for vision symptoms); **Do NOT delay steroids for biopsy** | |
| Steroid-Sparing | **Tocilizumab** (IL-6 inhibitor) FDA-approved; Methotrexate | |
| Complications | **Permanent vision loss** (anterior ischemic optic neuropathy), Aortic aneurysm, Stroke | |

## Gout and Pseudogout in Older Adults
| Condition | Features | Management | URL |
|-----------|----------|------------|-----|
| **Gout** | Acute monoarthritis (first MTP, ankle, knee), Tophi, Hyperuricemia | Acute: **NSAIDs** (caution in CKD), **Colchicine** (low dose 0.6 mg BID, reduce in CKD), **Corticosteroids** (oral or intra-articular); Chronic: **Allopurinol** or **Febuxostat** (goal UA <6) | https://www.rheumatology.org |
| **Calcium Pyrophosphate Deposition (CPPD/Pseudogout)** | Acute monoarthritis (knee, wrist), Chondrocalcinosis on X-ray | Acute: Similar to gout; Chronic: Colchicine prophylaxis, Treat underlying metabolic disease | |

## Rotator Cuff Disease
| Component | Description | URL |
|-----------|-------------|-----|
| Spectrum | Tendinopathy, Partial tear, Full-thickness tear, Cuff tear arthropathy | https://www.aaos.org |
| Risk Factors | Age (50% of asymptomatic =60 have tears), Repetitive overhead activity, Trauma | |
| Presentation | Pain with overhead activity, Night pain, Weakness (abduction, external rotation) | |
| Diagnosis | Physical exam (Empty can, Drop arm, External rotation lag), Ultrasound, MRI | |
| Treatment | **Physical therapy** (first-line for most), NSAIDs, Subacromial corticosteroid injection; **Surgery** for acute traumatic tear in active patient, Failed conservative management | |

## Spinal Stenosis
| Component | Description | URL |
|-----------|-------------|-----|
| Pathophysiology | Degenerative narrowing of spinal canal (central) or neural foramina | https://www.spine.org |
| Presentation | **Neurogenic claudication**: Buttock/leg pain, Numbness, Weakness with standing/walking; **Relieved by flexion** (leaning forward, sitting, shopping cart sign) | |
| Diagnosis | **MRI lumbar spine** (central canal stenosis, foraminal stenosis) | |
| Conservative | Physical therapy, NSAIDs, Epidural steroid injections (short-term relief) | |
| Surgical | **Laminectomy** for severe, refractory symptoms; Assess surgical risk | |

## Hip Fracture in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | >300,000 annually in US; 95% due to falls; 25% mortality at 1 year | https://www.cdc.gov/falls/hip-fractures |
| Types | **Femoral neck** (intracapsular, risk of AVN), **Intertrochanteric** (extracapsular), **Subtrochanteric** | |
| Management | **Surgery within 24-48 hours** (reduces mortality, complications); **Geriatric co-management** (orthogeriatrics) | https://www.aaos.org |
| Femoral Neck | Nondisplaced: Internal fixation; Displaced: **Hemiarthroplasty or THA** (older adults) | |
| Intertrochanteric | **Cephalomedullary nail** or Sliding hip screw | |
| Post-op | **Early mobilization**, **Delirium prevention**, **Osteoporosis treatment**, **Fall prevention** | https://www.fallsfree.org |
