# Geriatrics - Urinary and Fecal Incontinence

## Urinary Incontinence - Epidemiology and Impact
| Statistic | Value | URL |
|-----------|-------|-----|
| Prevalence in community-dwelling older adults | 15-30% (women), 5-15% (men) | https://www.nafc.org |
| Prevalence in long-term care | 50-70% | https://paltc.org |
| Consequences | Falls, Fractures, Pressure injuries, UTI, Depression, Social isolation, Caregiver burden, Institutionalization | https://www.americangeriatrics.org |
| **Underreporting** | **Only 25-50% discuss with provider** (embarrassment, Belief it's normal aging) | |

## Types of Urinary Incontinence
| Type | Mechanism | Key Features | URL |
|------|-----------|--------------|-----|
| **Urge Incontinence** | Detrusor overactivity | Sudden strong urge to void, Frequency, Nocturia, "Key-in-the-door" syndrome | https://www.auanet.org |
| **Stress Incontinence** | Urethral hypermobility, Sphincter deficiency | Leakage with cough, Laugh, Sneeze, Lifting, Exercise | |
| **Mixed Incontinence** | Combination of urge and stress | Most common in older women | |
| **Overflow Incontinence** | Bladder outlet obstruction (BPH) OR Detrusor underactivity | Frequent/small volumes, Sensation of incomplete emptying, Post-void dribbling | |
| **Functional Incontinence** | Impaired cognition, Mobility, Dexterity | Unable to reach toilet in time despite normal bladder function | |
| **Transient Incontinence** | Reversible causes | **DIAPPERS**: Delirium, Infection (UTI), Atrophic vaginitis/Urethritis, Pharmaceuticals, Psychological, Endocrine, Restricted mobility, Stool impaction | |

## Evaluation of Urinary Incontinence
| Component | Assessment | URL |
|-----------|------------|-----|
| History | Onset, Frequency, Severity, Triggers, Fluid intake, Voiding diary, Medication review, Obstetric/surgical history | https://www.auanet.org/guidelines/incontinence |
| Physical Exam | Abdominal (distention, masses), Pelvic (prolapse, atrophic vaginitis, Prostate exam), Rectal (tone, impaction), Neurologic (lower extremity, perineal sensation, Reflexes) | |
| **Cough Stress Test** | Observe for leakage with cough (full bladder) | |
| **Post-Void Residual (PVR)** | **Bladder scan** (preferred) or Catheterization; Normal <50 mL; >200 mL = Abnormal | |
| Urinalysis | Rule out infection, Hematuria | |
| Urodynamics | Not routinely indicated; Consider for complex, Refractory, Pre-surgical | |

## Management of Urinary Incontinence
| Type | Non-Pharmacologic | Pharmacologic | Procedural/Surgical | URL |
|------|------------------|---------------|---------------------|-----|
| **Urge Incontinence** | **Bladder training**, Pelvic floor muscle exercises, Fluid management, Avoid bladder irritants | **Antimuscarinics**: Oxybutynin, Tolterodine, Solifenacin, Trospium, Darifenacin; **Beta-3 agonist**: **Mirabegron**, Vibegron (preferred in older adults - less anticholinergic) | **Sacral neuromodulation**, **Percutaneous tibial nerve stimulation (PTNS)**, **Intradetrusor onabotulinumtoxinA (Botox)** | https://www.auanet.org |
| **Stress Incontinence** | **Pelvic floor muscle exercises (Kegels)**, Weight loss, Pessary | **Duloxetine** (off-label, modest benefit) | **Mid-urethral sling**, Urethral bulking agents | |
| **Overflow Incontinence** | **Intermittent self-catheterization**, Indwelling catheter (last resort) | **Alpha-blockers** (Tamsulosin), **5-ARI** (Finasteride) for BPH; **Bethanechol** (limited evidence) | Surgical relief of obstruction (TURP) | |
| **Functional Incontinence** | Scheduled toileting, Prompted voiding, Environmental modifications, Mobility aids | N/A | N/A | |

## Antimuscarinics in Older Adults - Caution
| Drug | Considerations | URL |
|------|----------------|-----|
| **Oxybutynin** | **High anticholinergic burden**; Avoid in older adults (Beers Criteria) | https://www.americangeriatrics.org/beers |
| **Tolterodine** | Moderate anticholinergic; Caution | |
| **Solifenacin, Darifenacin, Trospium** | More bladder-selective; Lower CNS penetration; Preferred if antimuscarinic needed | |
| **Mirabegron, Vibegron** | **Beta-3 agonists**; **Preferred in older adults** (no anticholinergic effects); Monitor BP (mirabegron can increase) | |
| **All Antimuscarinics** | Risk of cognitive impairment, Delirium, Constipation, Dry mouth, Blurred vision, Falls | |

## Fecal Incontinence
| Component | Description | URL |
|-----------|-------------|-----|
| Prevalence | 7-15% of community-dwelling older adults; 30-50% in long-term care | https://gi.org |
| Risk Factors | Age, Female sex, Diarrhea, Fecal urgency, Sphincter injury (obstetric, surgical), Neurologic disease, Cognitive impairment, Functional impairment | |
| Evaluation | History (frequency, Consistency, Urgency), Rectal exam (sphincter tone, Impaction, Mass), **Anorectal manometry**, **Endoanal ultrasound** | |
| Management | **Treat underlying cause** (diarrhea, Constipation/impaction); **Fiber supplementation**, **Loperamide** (if diarrhea); **Pelvic floor rehabilitation/Biofeedback**; **Surgical repair** (sphincteroplasty) | |
| Fecal Impaction | Common cause of overflow incontinence; **Disimpaction** + Bowel regimen | |
