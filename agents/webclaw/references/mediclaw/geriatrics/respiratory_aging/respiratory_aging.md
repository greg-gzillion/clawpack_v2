# Geriatrics - Respiratory Aging and Disease

## Respiratory Changes with Aging
| Change | Clinical Implication | URL |
|--------|---------------------|-----|
| Decreased lung elastic recoil | Increased residual volume, Increased FRC | https://www.lung.org |
| Chest wall stiffening | Increased work of breathing, Kyphosis | https://www.thoracic.org |
| Decreased respiratory muscle strength | Reduced cough effectiveness, Aspiration risk | https://www.nhlbi.nih.gov |
| Decreased mucociliary clearance | Increased pneumonia risk | https://www.atsjournals.org |
| Decreased ventilatory response to hypoxia/hypercapnia | Blunted respiratory drive, Sleep-disordered breathing | https://www.sleepfoundation.org |
| V/Q mismatch | Decreased PaO2 (PaO2 = 100 - 0.3 x age) | https://www.thoracic.org |
| Decreased FEV1 | ~25-30 mL/year decline after age 35 | https://www.lung.org |

## Chronic Obstructive Pulmonary Disease (COPD) in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| Prevalence | 10-15% in =65 years; Underdiagnosed | https://goldcopd.org |
| Diagnosis | Spirometry: **FEV1/FVC <0.70** (post-bronchodilator) | https://www.thoracic.org |
| GOLD Stages | 1: FEV1 =80%; 2: 50-79%; 3: 30-49%; 4: <30% | |
| GOLD Groups | A: Low symptoms, Low exacerbations; B: High symptoms, Low exacerbations; E: High exacerbations (=2 or =1 hospitalization) | https://goldcopd.org/2024-gold-report |
| Treatment (Group A) | Bronchodilator (LAMA or LABA) | |
| Treatment (Group B) | LABA + LAMA | |
| Treatment (Group E) | LABA + LAMA; Consider ICS if blood eosinophils =300 | |
| Inhaler Technique | **Critical in older adults**; Assess cognition, Dexterity, Coordination | https://www.lung.org/inhaler |
| Oxygen Therapy | PaO2 =55 mmHg or SpO2 =88%; PaO2 56-59 + Cor pulmonale or Hct >55% | https://www.thoracic.org |
| Pulmonary Rehabilitation | **Strongly recommended**; Improves dyspnea, Exercise capacity, Quality of life | https://www.lung.org/pulmonary-rehabilitation |

## Pneumonia in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| Atypical Presentation | **Afebrile**, Confusion/Delirium, Falls, Functional decline, Tachypnea | https://www.idsociety.org |
| Risk Factors | Aspiration, Impaired cough, Immunosenescence, Comorbidities | https://www.thoracic.org |
| Community-Acquired Pneumonia (CAP) | S. pneumoniae, H. influenzae, Respiratory viruses, Atypicals | |
| Healthcare-Associated Pneumonia | S. aureus (including MRSA), Gram-negatives | |
| Aspiration Pneumonia | Oral anaerobes, Gram-negatives | |
| CURB-65 Score | Confusion, Uremia (BUN >19), RR =30, BP <90/60, Age =65 | https://www.mdcalc.com |
| Treatment (CAP - Outpatient) | Amoxicillin OR Doxycycline OR Macrolide (if low resistance) | https://www.idsociety.org/pneumonia-guidelines |
| Treatment (CAP - Inpatient) | Ceftriaxone + Azithromycin OR Levofloxacin monotherapy | |
| Treatment (Aspiration) | Ampicillin-sulbactam OR Ceftriaxone + Metronidazole | |
| Prevention | **Pneumococcal vaccines** (PCV20 or PCV15 + PPSV23), **Annual influenza vaccine** | https://www.cdc.gov/vaccines |

## Interstitial Lung Disease (ILD) in Older Adults
| Type | Features | Management | URL |
|------|----------|------------|-----|
| **Idiopathic Pulmonary Fibrosis (IPF)** | Progressive dyspnea, Dry cough, Velcro crackles, Clubbing, **HRCT: UIP pattern** (honeycombing, traction bronchiectasis) | **Antifibrotics** (Pirfenidone, Nintedanib); Lung transplant in select patients | https://www.thoracic.org |
| **Hypersensitivity Pneumonitis** | Exposure history (birds, mold), Ground-glass opacities, Centrilobular nodules | Antigen avoidance; Corticosteroids | |
| **Connective Tissue Disease-ILD** | RA, SSc, Myositis; Often NSIP pattern | Immunosuppression; Antifibrotics | https://www.rheumatology.org |
| **Drug-Induced ILD** | Amiodarone, Methotrexate, Nitrofurantoin, Chemotherapy | Discontinue offending drug | |

## Obstructive Sleep Apnea (OSA) in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| Prevalence | 30-50% in =65 years (often underdiagnosed) | https://www.sleepfoundation.org |
| Screening | **STOP-BANG**: Snoring, Tired, Observed apnea, Pressure (HTN), BMI >35, Age >50, Neck >40 cm, Gender (male) | https://www.aasm.org |
| Consequences | Hypertension, Atrial fibrillation, Stroke, Cognitive impairment, Falls | https://www.heart.org |
| Diagnosis | Home sleep apnea test (HSAT) or Polysomnography | |
| Treatment | **CPAP** (first-line); Oral appliance; Positional therapy | |
| CPAP Adherence | Education, Mask fitting, Humidification, Follow-up | |

## Pulmonary Embolism (PE) in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| Atypical Presentation | Syncope, Delirium, Functional decline, Tachypnea only | https://www.heart.org |
| Risk Stratification | **sPESI** (Simplified Pulmonary Embolism Severity Index): Age >80, Cancer, Cardiopulmonary disease, HR =110, SBP <100, SpO2 <90% | https://www.mdcalc.com |
| Diagnosis | Wells score + D-dimer (age-adjusted: age x 10 if >50); CTA chest | |
| Treatment (Low-risk, sPESI 0) | Outpatient anticoagulation (DOAC preferred) | https://www.chestnet.org |
| Treatment (High-risk, Hypotension) | **Thrombolytics** (tPA) OR Embolectomy | |

## Lung Cancer in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | Median age at diagnosis is 70 years | https://www.cancer.gov |
| Screening | **Low-dose CT chest annually** for age 50-80, =20 pack-year smoking history, Current smoker or quit <15 years | https://www.uspreventiveservicestaskforce.org |
| Histology | Adenocarcinoma (most common), Squamous cell, Small cell | |
| Molecular Testing | EGFR, ALK, ROS1, BRAF, MET, RET, KRAS, NTRK, PD-L1 | |
| Treatment | Surgery, SBRT, Chemotherapy, Immunotherapy, Targeted therapy | |
| Geriatric Assessment | **Essential before treatment**; Assess frailty, Cognition, Function, Comorbidities | https://www.mycarg.org |
