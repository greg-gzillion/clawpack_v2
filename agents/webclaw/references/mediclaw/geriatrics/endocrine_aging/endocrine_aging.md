# Geriatrics - Endocrine Aging and Disorders

## Endocrine Changes with Aging
| Change | Clinical Implication | URL |
|--------|---------------------|-----|
| Insulin resistance | Type 2 diabetes risk | https://www.diabetes.org |
| Decreased T4 clearance | Lower levothyroxine requirements (~25% lower) | https://www.thyroid.org |
| Decreased testosterone (men) | Sarcopenia, Osteoporosis, Libido | https://www.endocrine.org |
| Menopause (women) | Estrogen deficiency effects | https://www.menopause.org |
| Decreased growth hormone/IGF-1 | Sarcopenia | |
| Decreased vitamin D activation | Osteoporosis, Falls, Muscle weakness | https://ods.od.nih.gov |
| Increased PTH | Secondary hyperparathyroidism (due to vitamin D deficiency, CKD) | https://www.bonehealthandosteoporosis.org |

## Diabetes Mellitus in Older Adults
| Component | Recommendation | URL |
|-----------|----------------|-----|
| Prevalence | 25-30% of adults =65 years | https://www.diabetes.org |
| Diagnostic Criteria | Same as younger adults (FPG =126, HbA1c =6.5%, OGTT =200) | |
| Glycemic Goals | **Individualize based on health status** | https://www.americangeriatrics.org |
| Healthy (few comorbidities, intact cognition/function) | HbA1c <7.0-7.5% | |
| Intermediate (multiple comorbidities, mild-moderate impairment) | HbA1c <8.0% | |
| Poor health (end-stage disease, severe impairment, limited life expectancy) | HbA1c <8.5% (avoid symptomatic hyperglycemia) | |
| **Avoid Hypoglycemia** | **Critical**; Increased falls, Cognitive impairment, Arrhythmias, Mortality | https://www.endocrine.org/hypoglycemia |
| Deintensification | Reduce/stop sulfonylureas, Insulin when HbA1c <6.5% or life expectancy limited | https://www.choosingwisely.org |

## Diabetes Medications in Older Adults
| Drug Class | Considerations in Older Adults | URL |
|------------|-------------------------------|-----|
| **Metformin** | **First-line**; Monitor B12; Avoid if eGFR <30 | https://www.diabetes.org |
| **SGLT2 Inhibitors** | **CV and renal benefits**; Risk of UTI, Dehydration, Euglycemic DKA, Amputation (canagliflozin) | |
| **GLP-1 Agonists** | **Weight loss, CV benefits**; GI side effects (nausea); Weekly injections improve adherence | |
| **DPP-4 Inhibitors** | Well-tolerated; Modest efficacy; Safe in CKD | |
| **Sulfonylureas** | **AVOID glyburide** (prolonged hypoglycemia); Glimepiride/Glipizide with caution | https://www.americangeriatrics.org/beers |
| **TZDs** | **AVOID** (fluid retention, CHF, Fractures) | |
| **Insulin** | Basal insulin preferred (glargine, degludec); Avoid sliding scale; Simplify regimen | |

## Thyroid Disorders in Older Adults
| Disorder | Presentation | Management | URL |
|----------|--------------|------------|-----|
| **Hypothyroidism** | Atypical: Fatigue, Cognitive impairment, Depression, Constipation, Bradycardia | Levothyroxine; **Start low, go slow** (25-50 mcg/day); Target TSH 4-6 (higher if frail) | https://www.thyroid.org |
| **Subclinical Hypothyroidism** | TSH 4.5-10, Normal FT4 | Treat if TSH >10, Symptoms, Goiter, TPO+, Pregnancy; Observe if asymptomatic, >80 years | |
| **Hyperthyroidism** | Apathetic hyperthyroidism: Weight loss, Atrial fibrillation, Weakness, Depression | Methimazole (preferred); RAI; Surgery; Beta-blockers for symptoms | |
| **Subclinical Hyperthyroidism** | TSH <0.1-0.4, Normal FT4 | Treat if TSH <0.1, Age >65, CV disease, Osteoporosis | |
| **Amiodarone-Induced** | Type 1 (Jod-Basedow): Treat with methimazole; Type 2 (Thyroiditis): Treat with steroids | | |

## Osteoporosis in Older Adults
| Component | Recommendation | URL |
|-----------|----------------|-----|
| Screening | **All women =65**; **Men =70**; Younger if risk factors (fragility fracture, steroid use) | https://www.bonehealthandosteoporosis.org |
| Diagnosis | DXA: T-score = -2.5 = Osteoporosis; -1.0 to -2.4 = Osteopenia | |
| FRAX Tool | 10-year fracture risk; Treatment if =3% hip or =20% major osteoporotic fracture | https://frax.shef.ac.uk |
| Calcium | 1200 mg/day total (diet + supplements); Calcium citrate preferred (less GI, no acid needed) | https://ods.od.nih.gov |
| Vitamin D | 800-2000 IU/day; Goal 25(OH)D =30 ng/mL | |
| Bisphosphonates | **First-line** (Alendronate, Risedronate, Zoledronic acid); Drug holiday after 3-5 years if low risk | https://www.endocrine.org/osteoporosis-guidelines |
| Denosumab | Alternative; **Do not stop abruptly** (rebound fractures) | |
| Anabolic Agents | Teriparatide, Abaloparatide, Romosozumab for severe osteoporosis | |
| Fall Prevention | **Critical**; Multifactorial interventions | https://www.cdc.gov/steadi |

## Vitamin B12 Deficiency in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| Prevalence | 10-20% of older adults | https://ods.od.nih.gov |
| Causes | **Food-cobalamin malabsorption** (atrophic gastritis, PPI use, Metformin), Pernicious anemia, Ileal disease/resection | |
| Symptoms | Macrocytic anemia, Peripheral neuropathy, Subacute combined degeneration (dorsal column), Cognitive impairment, Gait ataxia, Glossitis | |
| Diagnosis | Serum B12 <200 pg/mL; Borderline 200-300: Check **MMA** and **homocysteine** (elevated in deficiency) | |
| Treatment | **IM B12** 1000 mcg daily x 1 week ? weekly x 4 weeks ? monthly (if neurologic symptoms or malabsorption); **High-dose oral** 1000-2000 mcg daily (if no malabsorption) | |

## Testosterone Deficiency in Older Men
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Symptoms + Low testosterone (<300 ng/dL on 2 morning measurements) | https://www.endocrine.org/testosterone-guidelines |
| Symptoms | Decreased libido, Erectile dysfunction, Fatigue, Depression, Sarcopenia, Osteoporosis | |
| Diagnosis | Total testosterone (morning), Free testosterone (if low-normal total), LH, FSH, Prolactin | |
| Treatment | **Only if confirmed hypogonadism + symptoms**; Not for "anti-aging" | |
| Contraindications | Prostate cancer, Breast cancer, Severe BPH, Hct >50%, Untreated OSA, Severe CHF | |
| Monitoring | Testosterone (mid-normal), Hct (target <54%), PSA, DRE | |
| Risks | Erythrocytosis, Worsening OSA, Possible CV risk (controversial) | |
