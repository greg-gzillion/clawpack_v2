# Geriatrics - Delirium

## Definition and Epidemiology
| Component | Description | URL |
|-----------|-------------|-----|
| Definition (DSM-5) | Acute disturbance in attention and awareness, Change in cognition, Fluctuating course, Not due to pre-existing neurocognitive disorder, Evidence of medical cause | https://www.psychiatry.org |
| Prevalence | 10-30% of hospitalized older adults; 50-80% in ICU; 20-60% post-hip fracture | https://www.americangeriatrics.org |
| Significance | Increased mortality, Longer hospital stay, Functional decline, Institutionalization, Persistent cognitive impairment | https://www.nia.nih.gov |

## Delirium Subtypes
| Subtype | Characteristics | Recognition | URL |
|---------|-----------------|-------------|-----|
| **Hyperactive** | Agitation, Restlessness, Hallucinations, Combativeness | Easily recognized | https://www.americandeliriumsociety.org |
| **Hypoactive** | Lethargy, Decreased responsiveness, "Quiet" | **Often missed** (most common in older adults) | |
| **Mixed** | Fluctuates between hyperactive and hypoactive | Variable recognition | |

## Risk Factors for Delirium
| Category | Risk Factors | URL |
|----------|--------------|-----|
| **Predisposing (Baseline)** | Age >65, Pre-existing cognitive impairment/dementia, Sensory impairment, Functional impairment, Comorbidity burden, History of delirium, Frailty | https://www.americangeriatrics.org |
| **Precipitating (Acute)** | Infection, Medications, Dehydration, Electrolyte abnormalities, Surgery, Pain, Hypoxia, Urinary retention, Constipation, Sleep deprivation, Immobilization, Lines/catheters | https://www.hospitalelderlifeprogram.org |

## Common Precipitating Medications in Delirium
| Drug Class | Examples | Mechanism | URL |
|------------|----------|-----------|-----|
| **Anticholinergics** | Diphenhydramine, Oxybutynin, Amitriptyline, Promethazine | Central anticholinergic effects | https://www.americangeriatrics.org/beers |
| **Benzodiazepines** | Lorazepam, Alprazolam, Clonazepam | GABA agonism | |
| **Nonbenzodiazepine Hypnotics** | Zolpidem, Eszopiclone | GABA agonism | |
| **Opioids** | Morphine, Oxycodone, Hydromorphone | CNS effects | |
| **Corticosteroids** | Prednisone, Methylprednisolone | Psychiatric effects | |
| **Antipsychotics** | Haloperidol, Risperidone, Olanzapine | Anticholinergic, Dopamine blockade | |
| **Anticonvulsants** | Phenytoin, Topiramate, Levetiracetam | CNS effects | |
| **H2 Blockers** | Ranitidine, Famotidine | CNS penetration | |

|------|------|-------------|-----|

## Delirium Evaluation (DELIRIUM Mnemonic)
| Letter | Etiology | Evaluation | URL |
|--------|----------|------------|-----|
| **D**rugs | Medications (new, changed, anticholinergic burden) | Medication review, Deprescribe | https://deprescribing.org |
| **E**lectrolytes/Endocrine | Hyponatremia, Hypercalcemia, Hypo/Hyperglycemia, Hypoxia, Thyroid | BMP, Glucose, SpO2, TSH | |
| **L**ack of drugs | Withdrawal (alcohol, benzodiazepines, opioids) | CIWA, COWS | |
| **I**nfection | UTI, Pneumonia, Sepsis, Cellulitis | UA, CXR, Blood cultures, Exam | https://www.idsociety.org |
| **R**educed sensory input | Vision/hearing impairment | Ensure glasses, Hearing aids, Pocket talkers | |
| **I**ntracranial | Stroke, SDH, Seizure (non-convulsive), Meningitis | Neuro exam, Consider CT head, EEG | https://www.stroke.org |
| **U**rinary/Fecal | Urinary retention, Constipation, Fecal impaction | Bladder scan, Rectal exam | |
| **M**yocardial/Pulmonary | MI, CHF, PE, COPD exacerbation | ECG, Troponin, CXR, BNP | https://www.heart.org |
| **M**etabolic/Miscellaneous | Pain, Sleep deprivation, Immobility, Lines/restraints | Pain assessment, Non-pharmacologic sleep, Early mobility | |

## Non-Pharmacologic Prevention and Management (HELP - Hospital Elder Life Program)
| Intervention | Description | URL |
|--------------|-------------|-----|
| **Orientation** | Clocks, Calendars, Staff introductions, Reorientation | https://www.hospitalelderlifeprogram.org |
| **Sensory Optimization** | Ensure glasses, Hearing aids, Adequate lighting | |
| **Sleep Hygiene** | Minimize nighttime interruptions, Warm milk/herbal tea, Relaxation, Avoid caffeine | |
| **Early Mobility** | Ambulate 3x daily, Range of motion exercises | |
| **Hydration** | Encourage oral fluids; Monitor I/Os | |
| **Nutrition** | Assist with meals; Dentures in place | |
| **Medication Review** | Avoid deliriogenic medications | |
| **Pain Management** | Scheduled acetaminophen; Avoid opioids if possible | |
| **Family/Caregiver Involvement** | Extended visiting hours, Caregiver presence | |
| **Avoid Restraints** | Use alternatives (sitters, low beds, bed alarms) | |

## Pharmacologic Management of Severe Agitation
| Drug | Dose | Indications | Precautions | URL |
|------|------|-------------|-------------|-----|
| **Haloperidol** | 0.25-0.5 mg PO/IV/IM; Repeat q30-60 min | Severe agitation threatening safety | **ECG monitoring** (QTc prolongation); EPS; Avoid in Parkinson/LBD | https://www.psychiatry.org |
| **Quetiapine** | 12.5-25 mg PO | Agitation, Sleep disturbance | Sedation; Orthostasis; Metabolic effects | |
| **Risperidone** | 0.25-0.5 mg PO | Agitation, Psychosis | Higher EPS risk; Cerebrovascular risk in dementia | |
| **Olanzapine** | 2.5-5 mg PO/IM | Agitation | Sedation; Metabolic; Anticholinergic | |
| **Dexmedetomidine** | IV infusion | ICU delirium | Bradycardia; Hypotension | https://www.sccm.org |
| **AVOID Benzodiazepines** | Unless alcohol/benzodiazepine withdrawal or catatonia | Worsens delirium in older adults | |

## Delirium Prognosis and Follow-up
| Outcome | Data | URL |
|---------|------|-----|
| Resolution | 50% resolve by discharge; 30% persist at 1 month; 20% at 6 months | https://www.americangeriatrics.org |
| Mortality | 2-3x increased mortality in-hospital and 1-year | |
| Cognitive Decline | Accelerated cognitive decline; Increased dementia risk | https://www.alz.org |
| Functional Decline | Loss of ADLs; Increased nursing home placement | |
| Follow-up | Cognitive assessment post-discharge; Medication review; Caregiver education | |
