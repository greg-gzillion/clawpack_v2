# Geriatrics - Sensory Impairment

## Vision Impairment in Older Adults
| Statistic | Value | URL |
|-----------|-------|-----|
| Prevalence | 1 in 3 adults =65 has vision-impairing eye disease | https://www.nei.nih.gov |
| Leading Causes | Cataracts, Age-related macular degeneration, Glaucoma, Diabetic retinopathy | |
| Consequences | Falls, Functional decline, Depression, Social isolation, Cognitive decline, Driving cessation | https://www.aoa.org |

## Age-Related Cataracts
| Component | Description | URL |
|-----------|-------------|-----|
| Pathophysiology | Opacification of crystalline lens | https://www.aao.org |
| Types | Nuclear sclerotic (most common), Cortical, Posterior subcapsular | |
| Symptoms | Gradual painless vision loss, Glare, Halos around lights, Decreased contrast sensitivity, Difficulty with night driving | |
| Diagnosis | Slit lamp examination | |
| Treatment | **Cataract surgery** (phacoemulsification + IOL implantation); Indicated when symptoms affect quality of life | |
| Timing | No need to wait for "ripe" cataract; Elective procedure | |

## Age-Related Macular Degeneration (AMD)
| Type | Features | Management | URL |
|------|----------|------------|-----|
| **Dry AMD (Non-exudative)** | **Drusen**, Retinal pigment epithelium changes, Geographic atrophy (advanced) | **AREDS2 vitamins** (Vitamins C/E, Zinc, Copper, Lutein, Zeaxanthin) for intermediate/advanced; Smoking cessation; Monitor for conversion to wet | https://www.aao.org/amd |
| **Wet AMD (Exudative/Neovascular)** | Choroidal neovascularization, Subretinal fluid/hemorrhage, Rapid vision loss | **Anti-VEGF injections** (Ranibizumab, Aflibercept, Bevacizumab, Faricimab) | |
| Symptoms | Central vision loss, Metamorphopsia (distortion), Scotoma | | |
| Screening | **Amsler grid** home monitoring | | |

## Glaucoma
| Type | Features | Management | URL |
|------|----------|------------|-----|
| **Primary Open-Angle Glaucoma (POAG)** | Insidious, Asymptomatic early, Optic nerve cupping, Visual field loss (peripheral ? central) | **Prostaglandin analogs** (Latanoprost, Bimatoprost, Travoprost); Beta-blockers (Timolol); Alpha-agonists (Brimonidine); CAIs (Dorzolamide); SLT; Surgery | https://www.aao.org/glaucoma |
| **Primary Angle-Closure Glaucoma (PACG)** | Acute: Eye pain, Headache, N/V, Halos, Mid-dilated fixed pupil, Cloudy cornea, Elevated IOP; Chronic: Asymptomatic | Acute: **Ophthalmologic emergency**; Timolol, Pilocarpine, Acetazolamide, Mannitol; **Laser peripheral iridotomy** | |
| Screening | **All adults =40 should have comprehensive eye exam**; More frequent if risk factors (family history, African American/Hispanic, Diabetes, High myopia) | | |

## Diabetic Retinopathy
| Stage | Features | Management | URL |
|-------|----------|------------|-----|
| **Non-Proliferative (NPDR)** | Microaneurysms, Dot-blot hemorrhages, Hard exudates, Cotton-wool spots, Venous beading | Glycemic control, BP control, Lipid control; **Anti-VEGF** for DME | https://www.aao.org/diabetic-retinopathy |
| **Proliferative (PDR)** | **Neovascularization** (disc or elsewhere), Vitreous hemorrhage, Tractional retinal detachment | **Panretinal photocoagulation (PRP)**, Anti-VEGF, Vitrectomy | |
| Screening | **Annual dilated eye exam** for all diabetics; More frequent if retinopathy present | | |

## Hearing Impairment (Presbycusis)
| Component | Description | URL |
|-----------|-------------|-----|
| Prevalence | 1 in 3 adults 65-74; 1 in 2 adults =75 | https://www.nidcd.nih.gov |
| Pathophysiology | Age-related sensorineural hearing loss (loss of cochlear hair cells, Auditory nerve degeneration) | |
| Symptoms | Difficulty understanding speech (especially in noise), Tinnitus, Social withdrawal | |
| Consequences | **Cognitive decline** (dose-response), **Dementia risk** (2-5x increased), Depression, Falls, Functional decline | https://www.hopkinsmedicine.org/aging/hearing-loss-dementia |
| Screening | **Audiometry** (gold standard); Whisper test, Finger rub, Hearing Handicap Inventory for Elderly (HHIE-S) | |
| Treatment | **Hearing aids** (first-line); Cochlear implants for severe-profound loss; Assistive listening devices; Communication strategies | https://www.hearingloss.org |
| Barriers | Cost, Stigma, Perceived lack of benefit, Dexterity issues | |

## Hearing Aid Considerations in Older Adults
| Consideration | Recommendation | URL |
|---------------|----------------|-----|
| Assessment | Comprehensive audiologic evaluation | https://www.audiology.org |
| Device Selection | Behind-the-ear (BTE) often preferred (easier handling, better battery life) | |
| Features | Telecoil, Bluetooth connectivity, Rechargeable batteries | |
| Training | **Essential**; Multiple sessions; Include caregiver | |
| Over-the-Counter (OTC) Hearing Aids | FDA-approved for mild-moderate hearing loss; May benefit some older adults | https://www.fda.gov/medical-devices/hearing-aids/otc-hearing-aids |

## Dual Sensory Impairment (Vision + Hearing)
| Component | Description | URL |
|-----------|-------------|-----|
| Prevalence | 10-20% of adults =80 | https://www.afb.org |
| Consequences | **Synergistic negative effects**: Severe functional impairment, Social isolation, Depression, Cognitive decline, Mortality | |
| Management | **Interdisciplinary approach**; Optimize both vision and hearing; Environmental modifications; Communication training; Support services | https://www.helenkeller.org |

## Olfactory and Gustatory Impairment
| Sense | Changes with Aging | Consequences | URL |
|-------|-------------------|--------------|-----|
| **Olfaction (Smell)** | Decreased olfactory receptor neurons, Reduced mucus production | **Anorexia of aging**, Weight loss, **Safety risk** (gas leak, smoke), Decreased quality of life | https://www.nia.nih.gov/health/smell-and-taste |
| **Gustation (Taste)** | Decreased taste buds, Medications, Dry mouth | Reduced food enjoyment, Anorexia, Weight loss, Increased salt/sugar intake | |
| Olfaction and Neurodegeneration | **Hyposmia is early feature of Parkinson disease and Alzheimer disease** | |
