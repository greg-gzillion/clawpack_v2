# Geriatrics - Cardiovascular Aging and Disease

## Cardiovascular Changes with Aging
| Change | Clinical Implication | URL |
|--------|---------------------|-----|
| Increased arterial stiffness | Systolic hypertension, Widened pulse pressure, LVH | https://www.nhlbi.nih.gov |
| Decreased baroreceptor sensitivity | Orthostatic hypotension, Postprandial hypotension | https://www.nia.nih.gov |
| Decreased maximal heart rate | Reduced exercise capacity (HRmax = 220 - age) | https://www.heart.org |
| Conduction system fibrosis | Atrial fibrillation, Sick sinus syndrome, AV block | |
| Valvular calcification | Aortic stenosis, Mitral annular calcification | https://www.acc.org |
| Left ventricular hypertrophy | Diastolic dysfunction, HFpEF | |
| Decreased beta-adrenergic responsiveness | Blunted response to beta-agonists | |

## Hypertension in Older Adults
| Guideline | Target BP | URL |
|-----------|-----------|-----|
| **ACC/AHA 2017** | <130/80 mmHg for most (including =65 years) | https://www.acc.org/guidelines/hypertension |
| **ACP/AAFP 2017** | <150/90 mmHg for =60 years (less aggressive) | https://www.acponline.org |
| **JNC 8 (2014)** | <150/90 mmHg for =60 years without CKD/DM | |
| **SPRINT Trial** | Intensive (<120 mmHg) reduced CV events but increased AKI, syncope | https://www.sprinttrial.org |
| **Individualize** | Consider frailty, Comorbidities, Life expectancy, Polypharmacy | https://www.americangeriatrics.org |

## Hypertension Treatment in Older Adults
| Drug Class | Examples | Considerations | URL |
|------------|----------|----------------|-----|
| **Thiazide Diuretics** | Chlorthalidone, Hydrochlorothiazide | **First-line**; Monitor for hyponatremia, Hypokalemia, Gout | https://www.acc.org |
| **ACE Inhibitors** | Lisinopril, Enalapril, Ramipril | **First-line**; Monitor Cr, K+; Cough; Avoid in bilateral renal artery stenosis | |
| **ARBs** | Losartan, Valsartan, Candesartan | Alternative if ACE-I cough | |
| **Calcium Channel Blockers** | Amlodipine | **First-line**; Well-tolerated; Edema | |
| **Beta-Blockers** | Metoprolol, Atenolol | **Not first-line** (less stroke protection); Indicated in CAD, HFrEF, AF | |
| **Avoid** | Central alpha-agonists (clonidine), Alpha-blockers (orthostasis) | | |

## Orthostatic Hypotension (OH)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | SBP decrease =20 mmHg or DBP decrease =10 mmHg within 3 minutes of standing | https://www.heart.org |
| Prevalence | 10-30% of community-dwelling; 50-60% in long-term care | https://www.nia.nih.gov |
| Initial OH | BP drop within 15 seconds of standing | |
| Delayed OH | BP drop after 3 minutes | |
| Causes | Hypovolemia, Medications (diuretics, alpha-blockers, TCAs, Levodopa), Autonomic dysfunction (DM, Parkinson, MSA), Deconditioning | |
| Management | Review medications, Hydration, Compression stockings, Slow positional changes, Salt supplementation (if not contraindicated), Midodrine, Fludrocortisone | |

## Heart Failure in Older Adults
| Feature | HFrEF (EF =40%) | HFpEF (EF =50%) | HFmrEF (EF 41-49%) | URL |
|---------|-----------------|-----------------|---------------------|-----|
| Prevalence in older adults | 40-50% | **50-60%** (most common in elderly) | 10-20% | https://www.heart.org |
| Key pathophysiology | Systolic dysfunction | Diastolic dysfunction, Increased LV stiffness | Mixed | |
| Risk factors | CAD, MI | Hypertension, Age, Obesity, DM, AF | | |
| Diagnosis | Echo showing reduced EF | Echo showing preserved EF + Signs of HF + Elevated BNP/NT-proBNP or diastolic dysfunction | | |
| Treatment (GDMT) | **ARNI/ACE-I/ARB + Beta-blocker + MRA + SGLT2i** | **SGLT2i** (empagliflozin, dapagliflozin); Diuretics for congestion; Treat comorbidities | | |

## GDMT for HFrEF in Older Adults
| Drug Class | Examples | Starting Dose | Target Dose | Considerations | URL |
|------------|----------|---------------|-------------|----------------|-----|
| **ARNI** | Sacubitril/Valsartan | 24/26 mg BID | 97/103 mg BID | Monitor BP, Cr, K+; Avoid with ACE-I (36h washout) | https://www.acc.org/heart-failure-guidelines |
| **ACE-I** | Lisinopril, Enalapril | Low dose | Goal dose or maximally tolerated | Alternative if ARNI not available/affordable | |
| **Beta-Blocker** | Carvedilol, Metoprolol succinate, Bisoprolol | Low dose | Goal dose or maximally tolerated | **Start low, go slow**; May worsen symptoms initially | |
| **MRA** | Spironolactone, Eplerenone | 12.5-25 mg daily | 25-50 mg daily | Monitor Cr, K+ (risk of hyperkalemia); Avoid if CrCl <30 or K+ >5.0 | |
| **SGLT2i** | Empagliflozin, Dapagliflozin | 10 mg daily | 10 mg daily | **Add to GDMT**; Reduces HF hospitalizations; Euglycemic DKA risk | |

## Atrial Fibrillation in Older Adults
| Component | Recommendation | URL |
|-----------|----------------|-----|
| Prevalence | 10% in =80 years | https://www.heart.org |
| Stroke Risk | **CHA2DS2-VASc score** (=2 in men, =3 in women ? Anticoagulation) | |
| Bleeding Risk | **HAS-BLED score** (not a contraindication to anticoagulation; identifies modifiable risks) | |
| Rate Control | Beta-blockers, CCBs (diltiazem, verapamil), Digoxin; Goal HR <110 bpm (RACE II) | |
| Rhythm Control | Antiarrhythmics (amiodarone, flecainide, sotalol); Catheter ablation | |
| Anticoagulation | **DOACs preferred over warfarin** (except mechanical valve, mitral stenosis, CrCl <15) | https://www.acc.org/guidelines/atrial-fibrillation |
| Left Atrial Appendage Occlusion | Alternative for high stroke risk + contraindication to long-term anticoagulation | |

## Aortic Stenosis in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| Prevalence | 2-5% in =65 years; 10% in =80 years | https://www.heart.org |
| Etiology | Degenerative calcific (most common), Bicuspid valve, Rheumatic | |
| Symptoms | Angina, Syncope, **Heart failure** (dyspnea) | |
| Physical Exam | Systolic ejection murmur (crescendo-decrescendo) at RUSB radiating to carotids, Pulsus parvus et tardus, **S4**, Paradoxical split S2 | |
| Echo | Aortic valve area <1.0 cm˛ (severe), Mean gradient >40 mmHg, Peak velocity >4 m/s | |
| Asymptomatic Severe AS | Watchful waiting; Consider intervention if very severe (Vmax >5 m/s), Rapid progression, Elevated BNP | |
| Symptomatic Severe AS | **AVR (TAVR or SAVR) is only effective treatment**; 50% 2-year mortality without AVR | |
| TAVR vs SAVR | TAVR preferred in =80 years or high surgical risk | https://www.acc.org |

## Peripheral Arterial Disease (PAD) in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| Prevalence | 10-20% in =70 years | https://www.heart.org |
| Risk Factors | Smoking, Diabetes, Hypertension, Hyperlipidemia, CKD | |
| Symptoms | Intermittent claudication, Atypical leg pain, Critical limb ischemia (rest pain, tissue loss) | |
| Diagnosis | **Ankle-Brachial Index (ABI)** =0.90; Toe-Brachial Index if non-compressible vessels | |
| Medical Therapy | **Antiplatelet** (aspirin or clopidogrel), **High-intensity statin**, Smoking cessation, Supervised exercise therapy, Cilostazol | |
| Revascularization | For lifestyle-limiting claudication despite medical therapy, or critical limb ischemia | |
