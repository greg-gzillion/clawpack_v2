# Geriatrics - Nutrition and Aging

## Nutritional Changes with Aging
| Change | Clinical Implication | URL |
|--------|---------------------|-----|
| Decreased energy requirements | Lower caloric needs (due to decreased lean body mass, Activity) | https://www.nia.nih.gov/health/nutrition |
| Increased protein requirements | **1.2-1.5 g/kg/day** (to prevent sarcopenia) | https://www.nutritioncare.org |
| Decreased thirst sensation | Dehydration risk | |
| Altered taste and smell | Anorexia of aging, Weight loss | |
| Delayed gastric emptying | Early satiety, Postprandial hypotension | |
| Decreased vitamin D synthesis | Vitamin D deficiency, Osteoporosis, Falls | https://ods.od.nih.gov |
| Decreased B12 absorption | B12 deficiency (food-cobalamin malabsorption) | |
| Dental issues | Difficulty chewing, Food avoidance | |

## Malnutrition in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| Prevalence | 5-10% community-dwelling; 20-50% hospitalized; 30-60% long-term care | https://www.defeatmalnutrition.today |
| Consequences | Sarcopenia, Frailty, Falls, Pressure injuries, Infections, Hospital readmissions, Mortality | |
| Screening Tools | **MNA-SF** (Mini Nutritional Assessment-Short Form), **MST** (Malnutrition Screening Tool), **MUST** | https://www.mna-elderly.com |
| Diagnosis (GLIM Criteria) | At least 1 phenotypic (weight loss, Low BMI, Reduced muscle mass) + 1 etiologic (reduced intake, Inflammation) | https://www.espen.org |
| Management | **Food first** approach; Oral nutritional supplements (ONS); Enteral nutrition (if unable to meet needs orally); Parenteral nutrition (if GI tract non-functional) | |

## Mini Nutritional Assessment - Short Form (MNA-SF)
| Item | Scoring | URL |
|------|---------|-----|
| Food intake decline in past 3 months | 0 = Severe; 1 = Moderate; 2 = No | https://www.mna-elderly.com |
| Weight loss in past 3 months | 0 = >3 kg; 1 = Does not know; 2 = 1-3 kg; 3 = No | |
| Mobility | 0 = Bed/chair bound; 1 = Able to get out but not go out; 2 = Goes out | |
| Psychological stress/acute disease | 0 = Yes; 2 = No | |
| Neuropsychological problems | 0 = Severe dementia/depression; 1 = Mild; 2 = None | |
| BMI | 0 = <19; 1 = 19-21; 2 = 21-23; 3 = =23 | |
| **Interpretation** | 12-14 = Normal; 8-11 = At risk; 0-7 = Malnourished | |

## Protein Requirements in Older Adults
| Population | Recommendation | URL |
|------------|----------------|-----|
| **Healthy older adults** | 1.0-1.2 g/kg/day | https://www.nutritioncare.org |
| **Acute/chronic illness** | 1.2-1.5 g/kg/day | |
| **Severe illness/injury** | Up to 2.0 g/kg/day | |
| **Per meal distribution** | 25-30g high-quality protein per meal (to maximize muscle protein synthesis) | |
| **Leucine content** | 2.5-3g leucine per meal (whey protein, Animal proteins) | |

## Vitamin D and Calcium
| Nutrient | Recommendation | URL |
|----------|----------------|-----|
| **Vitamin D** | **800-2000 IU/day**; Goal 25(OH)D =30 ng/mL | https://ods.od.nih.gov |
| **Calcium** | 1200 mg/day total (diet + supplements); Calcium citrate preferred | |
| **Role of Vitamin D** | Calcium absorption, Bone health, **Muscle function**, **Fall prevention**, Immune function | |
| **Deficiency** | Osteomalacia, Osteoporosis, Muscle weakness, Falls, Fractures | |

## Vitamin B12
| Component | Description | URL |
|-----------|-------------|-----|
| Recommendation | 2.4 mcg/day (diet); **Higher supplementation often needed** due to malabsorption | https://ods.od.nih.gov |
| At-Risk Populations | PPI/H2RA use, Metformin use, Atrophic gastritis, Pernicious anemia, Vegan/vegetarian, Gastric bypass | |
| Screening | **B12 level**; If borderline (200-300 pg/mL), check **MMA** and **homocysteine** | |
| Treatment | **Oral B12** 1000-2000 mcg daily (if no malabsorption); **IM B12** 1000 mcg monthly (if malabsorption or neurologic symptoms) | |

## Fiber and Fluids
| Nutrient | Recommendation | URL |
|----------|----------------|-----|
| **Fiber** | 25-30 g/day (women), 30-38 g/day (men); Gradual increase to avoid bloating | https://www.nutrition.gov |
| **Fluids** | 1.5-2.0 L/day (unless fluid restriction); 30 mL/kg/day | |
| **Constipation** | Fiber + Adequate fluids + Physical activity | |

## Oral Nutritional Supplements (ONS)
| Indication | Evidence | URL |
|------------|----------|-----|
| **Malnutrition or at risk** | **Strong evidence** for reduced mortality, Complications, Readmissions | https://www.nutritioncare.org |
| **Hip fracture** | Reduced complications, Improved functional outcomes | |
| **Pressure injuries** | Improved healing | |
| **Pre-surgery** | Reduced postoperative complications | |
| **Timing** | Between meals (not with meals) to avoid appetite suppression | |

## Enteral Nutrition (Tube Feeding) in Advanced Dementia
| Consideration | Recommendation | URL |
|---------------|----------------|-----|
| **Evidence** | **Does NOT** improve survival, Prevent aspiration, Heal pressure injuries, Improve function, or Improve comfort | https://www.americangeriatrics.org |
| **Harms** | Agitation, Restraints, Aspiration, Diarrhea, Pressure injuries, Increased healthcare utilization | |
| **Guidelines** | **Discourage** tube feeding in advanced dementia; **Encourage** careful hand feeding (assisted oral feeding) | https://www.choosingwisely.org |
| **Goals of Care Discussion** | Discuss goals, Values, Prognosis with family; Emphasize comfort-focused care | |
