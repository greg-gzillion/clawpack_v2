# Geriatrics - Polypharmacy and Deprescribing

## Definitions and Epidemiology
| Term | Definition | URL |
|------|------------|-----|
| Polypharmacy | Use of =5 medications concurrently | https://www.cdc.gov/medicationsafety |
| Hyperpolypharmacy | Use of =10 medications concurrently | https://www.nia.nih.gov |
| Prevalence | 44% of older adults take =5 medications; 12% take =10 | https://www.cdc.gov/nchs |
| Consequences | ADEs, Falls, Cognitive impairment, Hospitalization, Mortality | https://www.ahrq.gov |
| Potentially Inappropriate Medications (PIMs) | Medications where risks outweigh benefits in older adults | https://www.americangeriatrics.org |

## Beers Criteria - Key Potentially Inappropriate Medications
| Drug Class | Examples | Rationale | Alternatives | URL |
|------------|----------|-----------|--------------|-----|
| **Anticholinergics** | Diphenhydramine, Oxybutynin, Amitriptyline | Confusion, Constipation, Falls | Cetirizine, Mirabegron, Duloxetine | https://geriatricscareonline.org |
| **Benzodiazepines** | Lorazepam, Alprazolam, Clonazepam | Falls, Cognitive impairment, Dependence | CBT, SSRIs, Melatonin | https://www.americangeriatrics.org |
| **Nonbenzodiazepine Hypnotics** | Zolpidem, Eszopiclone | Falls, Delirium, Complex sleep behaviors | CBT-I, Melatonin, Trazodone | https://www.aasm.org |
| **NSAIDs** | Ibuprofen, Naproxen, Indomethacin | GI bleeding, AKI, Hypertension | Acetaminophen, Topical NSAIDs, Celecoxib (caution) | https://www.rheumatology.org |
| **PPIs >8 weeks** | Omeprazole, Pantoprazole | C. diff, Fractures, B12 deficiency, CKD | H2RAs, Deprescribe if no indication | https://deprescribing.org |
| **Sulfonylureas** | Glyburide, Glimepiride | **Severe prolonged hypoglycemia** | Metformin, DPP-4 inhibitors, SGLT2 inhibitors | https://www.diabetes.org |
| **Sliding Scale Insulin** | Regular insulin | Hypoglycemia without benefit | Basal insulin, Basal-bolus regimen | https://www.diabetes.org |
| **Skeletal Muscle Relaxants** | Cyclobenzaprine, Methocarbamol | Sedation, Falls, Anticholinergic effects | Physical therapy, Acetaminophen | https://www.aaos.org |
| **TCAs** | Amitriptyline, Nortriptyline | Anticholinergic, Orthostasis, Cardiac toxicity | SSRIs, SNRIs, Gabapentinoids | https://www.psychiatry.org |
| **Antipsychotics (for dementia)** | Risperidone, Olanzapine, Quetiapine | **Increased mortality**, CVA, Falls | Non-pharmacologic interventions | https://www.alz.org |

## Beers Criteria - Drug-Disease Interactions
| Disease/Condition | Drug Class to Avoid | Rationale | URL |
|-------------------|---------------------|-----------|-----|
| Heart Failure | NSAIDs, COX-2 inhibitors, CCBs (non-dihydropyridine), TZDs | Fluid retention, Exacerbation | https://www.heart.org |
| Dementia/Cognitive Impairment | Anticholinergics, Benzodiazepines, Antipsychotics | Worsening cognition | https://www.alz.org |
| History of Falls/Fractures | Benzodiazepines, Antipsychotics, Anticonvulsants, Opioids | Increased fall risk | https://www.cdc.gov/steadi |
| Parkinson Disease | Antipsychotics (except quetiapine/clozapine), Metoclopramide | Worsening parkinsonism | https://www.parkinson.org |
| CKD Stage 4-5 (CrCl <30) | NSAIDs, Nitrofurantoin, Bisphosphonates (some) | Nephrotoxicity, Accumulation | https://www.kidney.org |
| Urinary Incontinence | Alpha-blockers (in women), Anticholinergics (in dementia) | Worsening incontinence | https://www.nafc.org |
| Constipation | Anticholinergics, Opioids, CCBs, Iron | Worsening constipation | https://gi.org |

## Deprescribing Approach (5-Step Framework)
| Step | Action | URL |
|------|--------|-----|
| 1. Comprehensive Medication Review | Identify all medications (prescription, OTC, supplements) | https://deprescribing.org |
| 2. Assess Each Medication | Indication, Effectiveness, Safety, Patient goals | https://www.ahrq.gov |
| 3. Prioritize Deprescribing | Highest risk, Lowest benefit medications first | https://www.americangeriatrics.org |
| 4. Plan and Implement | Taper when needed, Monitor for withdrawal | https://deprescribing.org |
| 5. Monitor and Follow-up | Assess for benefit/harm, Adjust as needed | https://www.nia.nih.gov |

## Deprescribing Algorithms (Bruyère Research Institute)
| Drug Class | Tapering Protocol | Monitoring | URL |
|------------|-------------------|------------|-----|
| **PPIs** | Reduce dose by 50% weekly or stop; Consider on-demand use | Symptom recurrence | https://deprescribing.org/resources/deprescribing-guidelines-algorithms/ |
| **Benzodiazepines** | Reduce dose by 25% every 2 weeks; Slower if long-term | Withdrawal symptoms, Sleep | |
| **Antipsychotics (BPSD)** | Reduce dose by 25-50% every 1-2 weeks | Behavioral symptoms | |
| **Antihyperglycemics** | Reduce sulfonylureas/insulin by 25-50% when starting SGLT2/GLP-1 | Hypoglycemia | |
| **Cholinesterase Inhibitors** | Taper over 4 weeks (if no benefit or severe dementia) | Cognitive/functional decline | |
