# Geriatrics - Depression and Mental Health

## Late-Life Depression - Epidemiology
| Statistic | Value | URL |
|-----------|-------|-----|
| Major depression in community-dwelling older adults | 1-5% | https://www.nimh.nih.gov/health/statistics/major-depression |
| Clinically significant depressive symptoms | 10-15% | https://www.aagponline.org |
| Depression in hospitalized older adults | 10-35% | https://www.americangeriatrics.org |
| Depression in long-term care | 15-40% | https://paltc.org |
| Suicide rate in older adults | Highest of any age group (especially white men =85) | https://www.cdc.gov/suicide |
| Gender difference | Women > Men (community); Men = Women (institutionalized) | |

|-------|----------|--------|-----|
| 0-4 | None-minimal | None | https://www.phqscreeners.com |
| 5-9 | Mild | Watchful waiting; Repeat PHQ-9 | |

## Differential Diagnosis of Late-Life Depression
| Condition | Distinguishing Features | Evaluation | URL |
|-----------|------------------------|------------|-----|
| **Grief/Bereavement** | Waves of grief, Yearning, Preoccupation with deceased, Retains self-esteem | Clinical history | https://www.hospicefoundation.org |
| **Bipolar Disorder** | History of mania/hypomania, Early onset, Family history | Mood disorder questionnaire | https://www.dbsalliance.org |
| **Dementia** | Cognitive symptoms precede mood, Anosognosia, "Don't care" rather than distress | Cognitive testing, Collateral history | https://www.alz.org |
| **Delirium** | Acute onset, Fluctuating course, Inattention | CAM | https://www.hospitalelderlifeprogram.org |
| **Anxiety Disorders** | Primary worry, Panic attacks, Avoidance | GAD-7 | https://adaa.org |
| **Substance Use** | Alcohol, Benzodiazepines, Opioids | CAGE, AUDIT-C | https://www.samhsa.gov |
| **Hypothyroidism** | Fatigue, Weight gain, Cold intolerance, Constipation | TSH | https://www.thyroid.org |
| **Vitamin B12 Deficiency** | Macrocytic anemia, Peripheral neuropathy, Gait ataxia | B12, MMA, Homocysteine | |

## Pharmacologic Treatment of Late-Life Depression
| Drug Class | Examples | Starting Dose | Considerations | URL |
|------------|----------|---------------|----------------|-----|
| **SSRIs (first-line)** | Sertraline, Escitalopram, Citalopram | Half usual adult dose; Slow titration | **Sertraline** (best evidence, fewer drug interactions); **Citalopram** (max 20 mg if >60 due to QTc); **Escitalopram** (well-tolerated) | https://www.aagponline.org |
| **SNRIs** | Duloxetine, Venlafaxine | Low dose; Slow titration | Duloxetine: Pain syndromes; Venlafaxine: Higher doses increase BP | |
| **Bupropion** | Bupropion SR/XL | 100-150 mg daily | Energizing; Avoid in seizure disorder, Eating disorders | |
| **Mirtazapine** | Mirtazapine | 7.5-15 mg qHS | Sedating, Appetite stimulant; Weight gain | |
| **Vortioxetine** | Vortioxetine | 5 mg daily | Cognitive benefits in some studies | |
| **AVOID** | Amitriptyline, Nortriptyline, Paroxetine | - | Anticholinergic, Orthostasis, Cardiac toxicity | https://www.americangeriatrics.org/beers |

## Treatment Considerations in Late-Life Depression
| Consideration | Recommendation | URL |
|---------------|----------------|-----|
| **Start low, go slow** | Half usual adult starting dose; Titrate every 2-4 weeks | https://www.aagponline.org |
| **Treat to remission** | Goal is full symptom resolution (not just improvement) | |
| **Duration of treatment** | First episode: 6-12 months after remission; Recurrent: =2 years or indefinite | |
| **Monitor for SIADH** | SSRIs can cause hyponatremia; Check sodium 2-4 weeks after starting | |
| **Monitor for falls** | SSRIs increase fall risk (mechanism unclear) | https://www.cdc.gov/steadi |
| **Monitor for GI bleeding** | SSRIs increase risk (especially with NSAIDs, antiplatelets) | https://gi.org |
| **Avoid benzodiazepines** | Increased falls, Cognitive impairment, Dependence | |
