# Geriatrics - Pressure Injuries

## Pressure Injury - Definition and Staging (NPIAP)
| Stage | Description | URL |
|-------|-------------|-----|
| **Stage 1** | **Non-blanchable erythema** of intact skin | https://npiap.com |
| **Stage 2** | **Partial-thickness skin loss** with exposed dermis; Blister (intact or ruptured); Shallow open ulcer with red/pink wound bed | |
| **Stage 3** | **Full-thickness skin loss**; Subcutaneous fat visible; No bone, tendon, or muscle exposure; Undermining and tunneling may be present | |
| **Stage 4** | **Full-thickness skin and tissue loss**; Bone, tendon, or muscle exposed; Undermining and tunneling common | |
| **Unstageable** | Full-thickness loss with **slough or eschar** obscuring wound base | |
| **Deep Tissue Pressure Injury (DTPI)** | Persistent **non-blanchable deep red, maroon, or purple** discoloration; Intact or non-intact skin | |

## Pressure Injury - Risk Factors
| Category | Risk Factors | URL |
|----------|--------------|-----|
| **Immobility** | Bedbound, Chairbound, Limited ability to reposition | https://www.ahrq.gov/patient-safety |
| **Sensory Impairment** | Unable to feel pain/discomfort (spinal cord injury, Neuropathy) | |
| **Moisture** | Urinary/fecal incontinence, Perspiration | |
| **Nutrition** | Malnutrition, Dehydration, Low BMI, Hypoalbuminemia | |
| **Perfusion** | Hypotension, Vasopressors, PAD, Anemia | |
| **Friction/Shear** | Dragging during transfers, Head of bed >30 degrees | |
| **Comorbidities** | Diabetes, CKD, Dementia, Frailty | |

## Pressure Injury - Risk Assessment Scales
| Scale | Components | Interpretation | URL |
|-------|------------|----------------|-----|
| **Braden Scale** | Sensory perception, Moisture, Activity, Mobility, Nutrition, Friction/Shear | **=18 = At risk**; 15-18 = Mild risk; 13-14 = Moderate; 10-12 = High; =9 = Very high | https://www.bradenscale.com |
| **Norton Scale** | Physical condition, Mental condition, Activity, Mobility, Incontinence | =14 = At risk | |

## Pressure Injury - Prevention
| Intervention | Description | URL |
|--------------|-------------|-----|
| **Risk Assessment** | **Braden Scale on admission** and daily | https://www.ahrq.gov/patient-safety/settings/hospital/resource/pressure-injury |
| **Skin Inspection** | Daily full skin assessment; Focus on bony prominences (sacrum, Heels, Ischium, Trochanters, Elbows, Occiput) | |
| **Repositioning** | **Every 2 hours** in bed; **Every 15-30 minutes** in chair (self-repositioning if able) | |
| **Support Surfaces** | High-density foam mattress; Alternating pressure mattress; Air-fluidized bed for high-risk | |
| **Moisture Management** | Prompt incontinence care; Barrier creams; Moisture-wicking underpads | |
| **Nutrition** | Adequate protein (1.2-1.5 g/kg/day), Calories, Hydration, Vitamin C, Zinc | https://www.nutritioncare.org |
| **Friction/Shear Reduction** | Use lift sheets; Keep head of bed =30 degrees (unless contraindicated); Heel offloading devices | |

## Pressure Injury - Treatment
| Component | Description | URL |
|-----------|-------------|-----|
| **Wound Assessment** | Size (length x width x depth), Location, Stage, Exudate, Odor, Wound bed (granulation, Slough, Eschar), Periwound skin, Undermining/Tunneling | https://www.wocn.org |
| **Cleansing** | Normal saline or potable water; Avoid antiseptics (cytotoxic) | |
| **Debridement** | Remove non-viable tissue (autolytic, Enzymatic, Mechanical, Sharp); **Do not debride stable eschar on heels** (unless infection) | |
| **Dressing Selection** | Stage 2: Moisture-retentive (hydrocolloid, Foam, Transparent film); Stage 3/4: Absorptive (foam, Alginate, Hydrofiber); Infected: Silver, Cadexomer iodine | |
| **Infection Control** | Consider topical antimicrobials (silver); Systemic antibiotics only if cellulitis, Osteomyelitis, Sepsis | |
| **Pain Management** | Assess pain; Topical opioids (morphine gel) for dressing changes; Systemic analgesics | |
| **Nutritional Support** | Aggressive nutritional intervention; Consider oral supplements, Enteral nutrition | |
| **Surgical Management** | Flap reconstruction for large Stage 3/4; Consider overall prognosis and goals of care | |

## Pressure Injury - Documentation and Quality
| Component | Description | URL |
|-----------|-------------|-----|
| **Present on Admission (POA)** | **Critical** for quality reporting and reimbursement | https://www.cms.gov |
| **Hospital-Acquired Pressure Injury (HAPI)** | Stage 3, 4, and Unstageable are **never events** (non-reimbursable) | |
| **Photography** | Standardized photos with consent; Include measuring device | |
| **Interdisciplinary Team** | Wound care specialist, Nutrition, PT/OT, Nursing, Provider | |
