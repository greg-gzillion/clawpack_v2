## Parkinson Disease (PD)
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | 1-2% of adults =65; Mean onset 60 years | https://www.parkinson.org |
| Pathology | Loss of dopaminergic neurons in substantia nigra; **Lewy bodies** (alpha-synuclein) | |
| Cardinal Motor Features | **TRAP**: **T**remor (resting, pill-rolling), **R**igidity (cogwheel), **A**kinesia/Bradykinesia, **P**ostural instability (late) | |
| Non-Motor Features | Hyposmia, REM sleep behavior disorder, Constipation, Depression, Anxiety, Cognitive impairment, Autonomic dysfunction (orthostasis, urinary) | |
| Diagnosis | Clinical; DaTscan (abnormal in PD, normal in essential tremor/drug-induced) | |
| Treatment | **Carbidopa/Levodopa** (most effective); Dopamine agonists (pramipexole, ropinirole); MAO-B inhibitors (rasagiline, selegiline); COMT inhibitors (entacapone); Deep brain stimulation (DBS) | https://www.movementdisorders.org |
| Geriatric Considerations | **Avoid anticholinergics** (benztropine, trihexyphenidyl); Monitor for orthostasis, Hallucinations, Impulse control disorders; PT/OT/SLP essential | |

## Parkinsonism - Differential Diagnosis
| Condition | Features | URL |
|-----------|----------|-----|
| **Drug-Induced Parkinsonism** | Symmetric, Rapid onset, **Dopamine-blocking drugs** (antipsychotics, antiemetics) | https://www.movementdisorders.org |
| **Vascular Parkinsonism** | Lower body predominant, Gait disturbance, Postural instability, MRI: White matter disease | |
| **Dementia with Lewy Bodies (DLB)** | Dementia BEFORE or within 1 year of parkinsonism, Fluctuating cognition, Visual hallucinations, REM sleep behavior disorder | https://www.lbda.org |
| **Progressive Supranuclear Palsy (PSP)** | Early falls, Vertical gaze palsy, Axial rigidity, Poor response to levodopa | https://www.psp.org |
| **Multiple System Atrophy (MSA)** | Autonomic failure (orthostasis, incontinence), Cerebellar signs (MSA-C) or Parkinsonism (MSA-P), Poor levodopa response | |
| **Normal Pressure Hydrocephalus (NPH)** | Triad: Gait apraxia, Cognitive impairment, Urinary incontinence; Ventriculomegaly | https://www.hydroassoc.org |

## Essential Tremor (ET)
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | Most common movement disorder; 5% of adults =65 | https://www.essentialtremor.org |
| Presentation | **Bilateral action/postural tremor** (hands, head, voice); **Improves with alcohol**; No other neurologic signs | |
| Diagnosis | Clinical; Distinguish from PD (resting tremor, bradykinesia, rigidity) | |
| Treatment | **Propranolol**, **Primidone** (first-line); Gabapentin, Topiramate, Benzodiazepines; **DBS** or Focused ultrasound for refractory | |

## Peripheral Neuropathy in Older Adults
| Cause | Features | Diagnosis | URL |
|-------|----------|-----------|-----|
| **Diabetic** | Distal symmetric, Sensory > Motor, Burning pain, Numbness | Clinical; EMG/NCS | https://www.diabetes.org |
| **B12 Deficiency** | Subacute combined degeneration (dorsal column), Gait ataxia, Romberg positive | B12, MMA, Homocysteine | |
| **Idiopathic** | Common in older adults; Slow progression | Diagnosis of exclusion | |
| **Chemotherapy-Induced** | History of neurotoxic chemotherapy (platinum, taxanes, vincristine) | Clinical | |
| **Alcohol-Related** | Nutritional deficiency, Direct toxicity | Clinical, Labs | |
| **CIDP** | Symmetric proximal and distal weakness, Sensory loss, Areflexia | EMG/NCS, CSF (elevated protein) | https://www.gbs-cidp.org |
| Management | Treat underlying cause; Symptomatic: **Gabapentinoids** (gabapentin, pregabalin), **SNRIs** (duloxetine), **TCAs** (nortriptyline), Topicals (capsaicin, lidocaine) | |

## Seizures and Epilepsy in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | Highest incidence in =65 years | https://www.epilepsy.com |
| Common Causes | **Stroke** (most common), Neurodegenerative disease, Tumor, Trauma, Metabolic | |
| Presentation | Often focal seizures with impaired awareness; Prolonged post-ictal state | |
| Diagnosis | **EEG** (may be normal interictally); MRI brain | |
| Treatment | **Levetiracetam**, **Lamotrigine** (first-line in elderly); Avoid enzyme-inducers (phenytoin, carbamazepine, phenobarbital) | https://www.aan.com |
| Geriatric Considerations | Lower doses, Slower titration, Monitor for hyponatremia (carbamazepine, oxcarbazepine), Osteoporosis (enzyme-inducers), Drug interactions | |
