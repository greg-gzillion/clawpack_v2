# Geriatrics - Frailty and Sarcopenia

## Frailty - Definition and Epidemiology
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | State of increased vulnerability to stressors due to decreased physiologic reserve | https://www.nia.nih.gov/research/dgc/frailty |
| Prevalence | 10-15% of community-dwelling adults =65; 25-50% in those =85 | https://www.americangeriatrics.org |
| Significance | Increased risk of falls, hospitalization, disability, mortality | https://pubmed.ncbi.nlm.nih.gov |

## Fried Frailty Phenotype
| Criterion | Definition | URL |
|-----------|------------|-----|
| **Weight Loss** | Unintentional weight loss =10 lbs or =5% in past year | https://www.ncbi.nlm.nih.gov/pmc/articles/PMC5390682/ |
| **Exhaustion** | Self-reported fatigue (CES-D questions) | https://www.americangeriatrics.org |
| **Weakness** | Grip strength (lowest 20% by gender/BMI) | https://www.nia.nih.gov |
| **Slowness** | Walking speed (lowest 20% by gender/height) | https://www.cdc.gov/steadi |
| **Low Physical Activity** | Kcal/week (lowest 20% by gender) | https://www.nia.nih.gov |
| **Frailty** | =3 criteria present | |
| **Pre-frailty** | 1-2 criteria present | |
| **Robust** | 0 criteria present | |

## Frailty Index (Rockwood Accumulation of Deficits Model)
| Component | Description | URL |
|-----------|-------------|-----|
| Calculation | Number of deficits / Total number of items assessed (usually 30-70 items) | https://www.dal.ca/sites/gmr/our-tools.html |
| Cutoff | >0.25 = Frail; >0.36 = Severely frail | https://pubmed.ncbi.nlm.nih.gov |
| Items | Symptoms, Signs, Diseases, Disabilities, Abnormal labs | https://www.dal.ca |
| Advantages | Continuous measure; Sensitive to change | |
| Limitations | Requires comprehensive assessment | |

## Clinical Frailty Scale (CFS)
| Score | Description | URL |
|-------|-------------|-----|
| 1 | Very Fit - Robust, active, energetic | https://www.dal.ca/sites/gmr/our-tools/clinical-frailty-scale.html |
| 2 | Well - No active disease, less fit than category 1 | |
| 3 | Managing Well - Medical problems controlled | |
| 4 | Vulnerable - Not dependent, but symptoms limit activities | |
| 5 | Mildly Frail - Limited dependence for IADLs | |
| 6 | Moderately Frail - Help with IADLs and some ADLs | |
| 7 | Severely Frail - Completely dependent for ADLs | |
| 8 | Very Severely Frail - Approaching end of life | |
| 9 | Terminally Ill - Life expectancy <6 months | |

## Sarcopenia - Definition and Diagnosis (EWGSOP2 Criteria)
| Criterion | Definition | URL |
|-----------|------------|-----|
| **Probable Sarcopenia** | Low muscle strength | https://www.ewgsop.org |
| **Confirmed Sarcopenia** | Low muscle strength + Low muscle quantity/quality | |
| **Severe Sarcopenia** | Low muscle strength + Low muscle quantity + Low physical performance | |
| Muscle Strength | Grip strength <27 kg (men), <16 kg (women); Chair stand >15 sec for 5 rises | https://www.ncbi.nlm.nih.gov/pmc/articles/PMC6322506/ |
| Muscle Quantity | DXA: ASM/height2 <7.0 kg/m2 (men), <5.5 kg/m2 (women); BIA, CT, MRI | |
| Physical Performance | Gait speed =0.8 m/s; SPPB =8; TUG =20 sec; 400m walk =6 min | |

## Sarcopenia - SARC-F Screening Questionnaire
| Component | Question | Scoring |
|-----------|----------|---------|
| **S**trength | How much difficulty lifting 10 lbs? | None=0, Some=1, A lot/Unable=2 |
| **A**ssistance walking | How much difficulty walking across room? | None=0, Some=1, A lot/Unable=2 |
| **R**ise from chair | How much difficulty transferring? | None=0, Some=1, A lot/Unable=2 |
| **C**limb stairs | How much difficulty climbing 10 stairs? | None=0, Some=1, A lot/Unable=2 |
| **F**alls | How many falls in past year? | None=0, 1-3=1, =4=2 |
| Interpretation | =4 = Predictive of sarcopenia and poor outcomes | |
| Reference | https://www.ncbi.nlm.nih.gov/pmc/articles/PMC4899100/ |

## Management of Frailty and Sarcopenia
| Intervention | Evidence | Recommendations | URL |
|--------------|----------|-----------------|-----|
| **Resistance Exercise** | Strong | 2-3 sessions/week; Progressive overload | https://go4life.nia.nih.gov |
| **Protein Intake** | Moderate-Strong | 1.2-1.5 g/kg/day; 25-30g per meal | https://www.nutritioncare.org |
| **Vitamin D** | Moderate | 800-2000 IU/day if deficient | https://ods.od.nih.gov |
| **Medication Review** | Strong | Deprescribe anticholinergics, sedatives | https://deprescribing.org |
| **Multicomponent Intervention** | Strong | Exercise + Nutrition + Cognitive training + Social engagement | https://www.nia.nih.gov |
| **Treat Underlying Conditions** | Strong | Manage chronic diseases, depression | https://www.americangeriatrics.org |

## Exercise Prescription for Frail Older Adults
| Component | Frequency | Intensity | Examples | URL |
|-----------|-----------|-----------|----------|-----|
| Aerobic | 3-5 days/week | Moderate (RPE 5-6/10) | Walking, Cycling, Swimming | https://go4life.nia.nih.gov |
| Resistance | 2-3 days/week | 60-80% 1RM | Weight machines, Bands, Body weight | https://www.acsm.org |
| Balance | 2-3 days/week | Progress difficulty | Tai Chi, Heel-toe walk, Single-leg stance | https://www.cdc.gov/steadi |
| Flexibility | Daily | Gentle stretch | Yoga, Stretching | https://go4life.nia.nih.gov |

## Protein Supplementation in Sarcopenia
| Recommendation | Details | URL |
|----------------|---------|-----|
| Total Daily Protein | 1.2-1.5 g/kg/day (up to 2.0 g/kg in severe sarcopenia) | https://www.nutritioncare.org |
| Per Meal Distribution | 25-30g high-quality protein per meal | https://www.americangeriatrics.org |
| Leucine Content | 2.5-3g leucine per meal to stimulate MPS | https://ods.od.nih.gov |
| Timing | Within 1-2 hours post-exercise | https://www.acsm.org |
| Sources | Whey, Casein, Soy, Animal proteins | https://www.nutrition.gov |

## Anorexia of Aging
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Age-related decrease in appetite and food intake | https://www.nia.nih.gov/health/anorexia-aging |
| Causes | Sensory decline, Delayed gastric emptying, Hormonal changes, Medications, Social isolation | https://pubmed.ncbi.nlm.nih.gov |
| Consequences | Weight loss, Sarcopenia, Frailty, Increased mortality | https://www.americangeriatrics.org |
| Management | Small frequent meals, Enhance flavor, Social dining, Address contributing factors | https://www.nutritioncare.org |
