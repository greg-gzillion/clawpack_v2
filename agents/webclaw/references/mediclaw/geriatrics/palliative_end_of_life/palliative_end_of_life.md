# Geriatrics - Palliative and End-of-Life Care

## Definitions
| Term | Definition | URL |
|------|------------|-----|
| **Palliative Care** | Interdisciplinary care focused on improving quality of life for patients with **serious illness at any stage**, Regardless of prognosis; Provided **alongside curative treatment** | https://www.capc.org |
| **Hospice Care** | Palliative care for patients with **life expectancy =6 months** (if disease runs its natural course); Focus on **comfort**, Not cure | https://www.nhpco.org |
| **End-of-Life Care** | Care during the final weeks to days of life | |

## Hospice Eligibility (Medicare Hospice Benefit)
| Diagnosis | General Guidelines | URL |
|-----------|---------------------|-----|
| **Dementia** | **FAST 7C** (incontinence, Unable to ambulate, Minimal verbal communication) + One of: Aspiration pneumonia, Pyelonephritis, Sepsis, Stage 3-4 pressure injury, Weight loss >10% in 6 months, Albumin <2.5 | https://www.cms.gov/medicare/hospice |
| **Heart Failure** | **NYHA Class IV** + **EF =20%** (or preserved EF with symptoms) + Optimal medical therapy + Not candidate for advanced therapies | |
| **COPD** | **FEV1 <30%** + Cor pulmonale + Hypoxemia (SpO2 =88% on room air or O2 dependent) + Hypercapnia (pCO2 =50) + Frequent hospitalizations | |
| **Cancer** | Metastatic or locally advanced + Declining functional status + Refractory to treatment or not pursuing treatment | |
| **Stroke** | **PPS =40%** + Poor nutritional status (albumin <2.5, Weight loss) + Aspiration pneumonia or pyelonephritis | |
| **General Decline** | **PPS =40%** + Weight loss + Albumin <2.5 + Functional decline + Frequent infections/hospitalizations | |

## Symptom Management at End of Life
| Symptom | Management | URL |
|---------|------------|-----|
| **Pain** | **Opioids** (morphine, Hydromorphone, Oxycodone); Scheduled long-acting + PRN short-acting; **Start low, go slow**; Monitor for constipation, Delirium | https://www.capc.org |
| **Dyspnea** | **Opioids** (first-line, low dose); Fan, Positioning, Oxygen (only if hypoxic), Benzodiazepines (if anxiety) | |
| **Nausea/Vomiting** | Identify cause; **Haloperidol**, **Metoclopramide**, **Ondansetron**, **Prochlorperazine**, **Dexamethasone** | |
| **Delirium** | **Haloperidol** (first-line); **Quetiapine**, **Olanzapine**; **AVOID benzodiazepines** (unless alcohol withdrawal) | |
| **Terminal Secretions ("Death Rattle")** | **Glycopyrrolate**, **Scopolamine**, **Atropine**; Positioning (lateral); Suctioning (gentle) | |
| **Constipation** | **Scheduled stimulant laxative** (senna) for all patients on opioids | |
| **Anxiety** | **Benzodiazepines** (lorazepam); Non-pharmacologic (presence, Reassurance) | |

## Opioid Prescribing in Palliative Care
| Principle | Description | URL |
|-----------|-------------|-----|
| **Route** | Oral preferred; **Subcutaneous** (alternative to IV); Transdermal (stable pain); Avoid IM | https://www.capc.org |
| **Scheduled + PRN** | Long-acting scheduled + Short-acting PRN for breakthrough (10-20% of total daily dose) | |
| **Dose Escalation** | Increase by 25-50% for uncontrolled pain | |
| **Renal Impairment** | **Hydromorphone or Fentanyl preferred** (avoid morphine - metabolite accumulation) | |
| **Opioid Rotation** | If toxicity or inadequate analgesia; Use equianalgesic table; Reduce new opioid by 25-50% for incomplete cross-tolerance | |
| **Constipation** | **Always prescribe bowel regimen** | |

## Recognizing the Dying Patient - Signs of Imminent Death (Days to Hours)
| Sign | Description | URL |
|------|-------------|-----|
| **Decreased Level of Consciousness** | Lethargy ? Obtundation ? Coma | https://www.nhpco.org |
| **Decreased Oral Intake** | Anorexia, Dysphagia | |
| **Terminal Secretions** | "Death rattle" | |
| **Changes in Breathing** | **Cheyne-Stokes respiration**, Apnea, Irregular pattern | |
| **Changes in Perfusion** | **Mottling** of extremities, Cool skin, Cyanosis | |
| **Changes in Urine Output** | Oliguria ? Anuria | |
| **"Terminal Rally"** | Brief period of lucidity/energy before death | |
