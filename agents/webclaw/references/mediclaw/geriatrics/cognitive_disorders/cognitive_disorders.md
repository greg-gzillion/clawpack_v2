# Geriatrics - Cognitive Disorders

## Spectrum of Cognitive Impairment
| Condition | Definition | Progression | URL |
|-----------|------------|-------------|-----|
| Subjective Cognitive Decline (SCD) | Self-reported cognitive concerns, normal testing | 10-15% progress to MCI/dementia per year | https://www.alz.org |
| Mild Cognitive Impairment (MCI) | Objective cognitive impairment, preserved function | 10-15% progress to dementia per year | https://www.nia.nih.gov/health/mild-cognitive-impairment |
| Dementia | Cognitive impairment affecting multiple domains, functional decline | Progressive | https://www.alz.org |
| Major Neurocognitive Disorder | DSM-5 term for dementia | Progressive | https://www.psychiatry.org |

## Alzheimer Disease (AD)
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | Most common cause of dementia (60-80%) | https://www.alz.org/facts |
| Pathophysiology | Amyloid plaques (Aﾟ), Neurofibrillary tangles (hyperphosphorylated tau), Neuronal loss | https://www.nia.nih.gov/health/alzheimers-disease |
| Genetics | **APOE e4** (risk factor), **APP, PSEN1, PSEN2** (early-onset autosomal dominant) | https://www.genome.gov |
| Typical Presentation | Progressive episodic memory loss, Anosognosia, Visuospatial deficits, Language impairment | https://www.alz.org |
| Atypical Variants | Posterior cortical atrophy (visuospatial), Logopenic variant PPA (language), Frontal variant (executive/behavioral) | https://memory.ucsf.edu |

## Vascular Dementia (VaD)
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | Second most common cause (10-20%) | https://www.stroke.org |
| Risk Factors | Hypertension, Diabetes, Hyperlipidemia, Smoking, Atrial fibrillation | https://www.heart.org |
| Subtypes | Multi-infarct dementia, Strategic infarct dementia, Subcortical ischemic vascular dementia (Binswanger), Cerebral amyloid angiopathy | https://www.aan.com |
| Presentation | Executive dysfunction, Psychomotor slowing, Gait disturbance (frontal gait), Urinary symptoms, Focal neurologic deficits | |
| Imaging | MRI: White matter hyperintensities, Lacunar infarcts, Microbleeds | https://www.acr.org |
| Treatment | **Risk factor control** (BP, lipids, glucose, smoking cessation), Antiplatelet therapy, Cholinesterase inhibitors (modest benefit) | https://www.stroke.org |

## Dementia with Lewy Bodies (DLB)
| Component | Description | URL |
|-----------|-------------|-----|
| Core Features | Fluctuating cognition, Recurrent visual hallucinations, **REM sleep behavior disorder**, Parkinsonism | https://www.lbda.org |
| Supportive Features | Severe neuroleptic sensitivity, Falls/Syncope, Autonomic dysfunction, Depression, Delusions | |
| Diagnostic Criteria | Dementia + =2 core features OR 1 core + =1 indicative biomarker (DaTscan, MIBG) | https://www.ncbi.nlm.nih.gov/pmc/articles/PMC5396457/ |
| Treatment | **Cholinesterase inhibitors** (rivastigmine FDA-approved); **AVOID antipsychotics** (neuroleptic sensitivity); Treat RBD (melatonin, clonazepam); Carbidopa/levodopa for parkinsonism | |
| Prognosis | Faster progression than AD; Median survival 5-7 years | |

## Frontotemporal Dementia (FTD)
| Variant | Presentation | Pathology | URL |
|---------|--------------|-----------|-----|
| **Behavioral variant FTD (bvFTD)** | Disinhibition, Apathy, Loss of empathy, Compulsive behaviors, Hyperorality, Executive dysfunction | tau (40%), TDP-43 (50%), FUS | https://www.theaftd.org |
| **Semantic variant PPA** | Fluent aphasia, Loss of word meaning, Impaired object knowledge | TDP-43 (most) | https://memory.ucsf.edu |
| **Nonfluent/Agrammatic variant PPA** | Effortful speech, Agrammatism, Apraxia of speech | tau (most) | |
| Genetics | **C9orf72** (most common), **MAPT**, **GRN** | | https://www.genome.gov |
| Treatment | Supportive; SSRIs for behavioral symptoms; Avoid cholinesterase inhibitors (may worsen) | | |

## Normal Pressure Hydrocephalus (NPH)
| Component | Description | URL |
|-----------|-------------|-----|
| Classic Triad (Hakim's Triad) | **Gait disturbance** (magnetic gait, wide-based, shuffling), **Cognitive impairment** (executive, bradyphrenia), **Urinary incontinence** | https://www.hydroassoc.org |
| Imaging | Ventriculomegaly (Evans index >0.3) out of proportion to atrophy; DESH (disproportionately enlarged subarachnoid space hydrocephalus) | https://www.acr.org |
| Diagnostic Test | **High-volume LP** (30-50 mL CSF removal) ? Assess gait/cognition improvement | https://www.aan.com |
| Treatment | **Ventriculoperitoneal (VP) shunt** | https://www.aans.org |
| Prognosis | Gait most likely to improve; Cognitive and urinary symptoms less predictable | |

## Rapidly Progressive Dementias
| Cause | Key Features | Diagnostic Tests | URL |
|-------|--------------|------------------|-----|
| **Creutzfeldt-Jakob Disease (CJD)** | Rapid progression (weeks-months), Myoclonus, Ataxia, Visual changes, Mutism | MRI (DWI cortical ribboning, basal ganglia), CSF 14-3-3, RT-QuIC | https://www.cjdfoundation.org |
| **Autoimmune Encephalitis** | Subacute onset, Seizures, Psychiatric symptoms, Movement disorders | CSF (pleocytosis, oligoclonal bands), Antibodies (NMDA, LGI1, etc.) | https://www.aealliance.org |
| **Neurosyphilis** | Variable (meningitis, meningovascular, general paresis, tabes dorsalis) | Serum RPR/VDRL, CSF VDRL | https://www.cdc.gov/std/syphilis |
| **HIV-Associated Neurocognitive Disorder (HAND)** | Subcortical dementia, Motor slowing | HIV viral load, CD4 count | https://www.hiv.gov |
| **Limbic Encephalitis** | Subacute memory loss, Seizures, Psychiatric symptoms | MRI (T2 hyperintensity mesial temporal), Antibodies (Hu, Ma2, VGKC) | |
