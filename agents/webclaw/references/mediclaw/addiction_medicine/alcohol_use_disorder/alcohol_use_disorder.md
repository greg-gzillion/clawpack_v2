# Addiction Medicine - Alcohol Use Disorder (AUD)

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Problematic pattern of alcohol use** leading to clinically significant impairment or distress | https://www.niaaa.nih.gov/alcohol-health/overview-alcohol-consumption/alcohol-use-disorders |
| **Prevalence** | **~14%** of US adults (lifetime); **~6%** (past year) | https://www.niaaa.nih.gov/alcohol-health/overview-alcohol-consumption/alcohol-use-disorders |
| **Low-Risk Drinking Limits** | **Men**: =14 drinks/week AND =4 drinks/day; **Women**: =7 drinks/week AND =3 drinks/day | https://www.niaaa.nih.gov |

## Alcohol Withdrawal
| Stage | Timing | Symptoms | URL |
|-------|--------|----------|-----|
| **Stage 1 (Minor)** | **6-12h** | **Tremor**, **Anxiety**, **Insomnia**, **Nausea**, **Diaphoresis**, **Tachycardia** | https://www.niaaa.nih.gov |
| **Stage 2 (Alcoholic Hallucinosis)** | **12-24h** | **Visual/auditory/tactile hallucinations** (clear sensorium) | https://www.niaaa.nih.gov |
| **Stage 3 (Withdrawal Seizures)** | **24-48h** | **Generalized tonic-clonic seizures** | https://www.niaaa.nih.gov |
| **Stage 4 (Delirium Tremens)** | **48-96h** | **Delirium**, **Agitation**, **Hallucinations**, **Autonomic instability** (fever, tachycardia, hypertension), **Mortality 5-15% untreated** | https://www.niaaa.nih.gov |

## CIWA-Ar Protocol
| Component | Description | URL |
|-----------|-------------|-----|
| **Purpose** | **Symptom-triggered benzodiazepine dosing** for alcohol withdrawal | https://www.mdcalc.com/calc/10433/ciwa-ar-alcohol-withdrawal |
| **Scoring** | **10 items** (nausea, tremor, anxiety, agitation, etc.), Score 0-67 | https://www.mdcalc.com/calc/10433/ciwa-ar-alcohol-withdrawal |
| **Mild (<8)** | **Supportive care**, **No benzodiazepines** | https://www.mdcalc.com/calc/10433/ciwa-ar-alcohol-withdrawal |
| **Moderate (8-15)** | **Benzodiazepines** (diazepam 10 mg or lorazepam 2 mg) | https://www.mdcalc.com/calc/10433/ciwa-ar-alcohol-withdrawal |
| **Severe (>15)** | **Higher-dose benzodiazepines**, **Consider ICU** | https://www.mdcalc.com/calc/10433/ciwa-ar-alcohol-withdrawal |

## Wernicke-Korsakoff Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| **Wernicke Encephalopathy** | **Acute, reversible**: **Confusion**, **Ataxia**, **Ophthalmoplegia** (classic triad present in only 10-30%) | https://www.ninds.nih.gov/health-information/disorders/wernicke-korsakoff-syndrome |
| **Korsakoff Syndrome** | **Chronic, irreversible**: **Anterograde amnesia**, **Confabulation** | https://www.ninds.nih.gov/health-information/disorders/wernicke-korsakoff-syndrome |
| **Treatment** | **High-dose thiamine** (500 mg IV TID x3-5 days ? 250 mg daily) | https://www.ninds.nih.gov/health-information/disorders/wernicke-korsakoff-syndrome |
| **Prevention** | **Thiamine supplementation** in all patients at risk for deficiency | https://www.ninds.nih.gov/health-information/disorders/wernicke-korsakoff-syndrome |
