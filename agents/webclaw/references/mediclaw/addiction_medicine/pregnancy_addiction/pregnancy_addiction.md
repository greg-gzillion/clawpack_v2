# Addiction Medicine - Addiction in Pregnancy

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Prevalence** | **~5-10%** of pregnant women use illicit substances | https://www.samhsa.gov/data/report/2021-nsduh-annual-national-report |
| **Screening** | **Universal screening** recommended (4Ps, 5Ps, SURP-P) | https://www.acog.org/clinical/clinical-guidance/committee-opinion/articles/2017/07/opioid-use-and-opioid-use-disorder-in-pregnancy |
| **Stigma** | **Avoid punitive approaches**; **Engage in compassionate care** | https://www.acog.org/clinical/clinical-guidance/committee-opinion/articles/2017/07/opioid-use-and-opioid-use-disorder-in-pregnancy |

## Opioid Use Disorder in Pregnancy
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **Treatment** | **MOUD is standard of care** (buprenorphine or methadone) | https://www.acog.org/clinical/clinical-guidance/committee-opinion/articles/2017/07/opioid-use-and-opioid-use-disorder-in-pregnancy |
| **Buprenorphine** | **Preferred** (lower risk of neonatal abstinence syndrome, Office-based) | https://www.samhsa.gov/medications-substance-use-disorders |
| **Methadone** | **Alternative** (better retention for some) | https://www.samhsa.gov/medications-substance-use-disorders |
| **Detoxification** | **NOT recommended** (high relapse rate, Fetal distress) | https://www.acog.org/clinical/clinical-guidance/committee-opinion/articles/2017/07/opioid-use-and-opioid-use-disorder-in-pregnancy |

## Neonatal Abstinence Syndrome (NAS) / Neonatal Opioid Withdrawal Syndrome (NOWS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Withdrawal in newborn** after in-utero opioid exposure | https://www.cdc.gov/pregnancy/opioids/index.html |
| **Onset** | **24-72h** (heroin), **3-5 days** (buprenorphine), **5-7 days** (methadone) | https://www.cdc.gov/pregnancy/opioids/index.html |
| **Assessment** | **Finnegan Neonatal Abstinence Scoring System** or **Eat, Sleep, Console (ESC)** | https://www.cdc.gov/pregnancy/opioids/index.html |
| **Treatment** | **Non-pharmacologic** (rooming-in, Breastfeeding), **Morphine** or **Methadone** (if severe) | https://www.cdc.gov/pregnancy/opioids/index.html |

## Alcohol Use in Pregnancy
| Component | Description | URL |
|-----------|-------------|-----|

| **Prevention** | **No safe amount of alcohol in pregnancy** | https://www.cdc.gov/ncbddd/fasd/alcohol-use.html |
