# Addiction Medicine - Screening and Assessment

## Screening Tools
| Tool | Substance | Description | URL |
|------|-----------|-------------|-----|
| **Single Question Screen** | **Alcohol/Drugs** | **"How many times in the past year have you had 5+ (men) or 4+ (women) drinks in a day?"** OR **"How many times in the past year have you used an illegal drug or prescription medication for non-medical reasons?"** | https://www.samhsa.gov/sbirt |
| **AUDIT-C** | **Alcohol** | **3-item** screen for unhealthy alcohol use | https://www.niaaa.nih.gov/alcohol-health/screening-and-assessment-tools |
| **AUDIT** | **Alcohol** | **10-item** assessment (score =8 = hazardous use) | https://www.niaaa.nih.gov/alcohol-health/screening-and-assessment-tools |
| **DAST-10** | **Drugs** | **10-item** Drug Abuse Screening Test | https://www.drugabuse.gov |
| **CRAFFT** | **Adolescents** | **6-item** screen for alcohol/drugs | https://crafft.org |
| **TAPS** | **Tobacco, Alcohol, Prescription drugs, Substances** | **Brief, validated** screening | https://www.drugabuse.gov/taps/ |

## ASAM Six Dimensions Assessment
| Dimension | Description | URL |
|-----------|-------------|-----|
| **1. Acute Intoxication/Withdrawal** | **Current intoxication or withdrawal potential** | https://www.asam.org/quality-care/clinical-guidelines/asam-criteria |
| **2. Biomedical Conditions** | **Physical health conditions and complications** | https://www.asam.org/quality-care/clinical-guidelines/asam-criteria |
| **3. Emotional/Behavioral/Cognitive** | **Mental health conditions**, **Cognitive functioning** | https://www.asam.org/quality-care/clinical-guidelines/asam-criteria |

| **6. Recovery Environment** | **Living situation**, **Employment**, **Legal issues**, **Family support** | https://www.asam.org/quality-care/clinical-guidelines/asam-criteria |

## Stages of Change (Transtheoretical Model)
| Stage | Description | Intervention | URL |
|-------|-------------|--------------|-----|
| **Precontemplation** | **Not considering change** | **Raise awareness**, **Express concern** | https://www.samhsa.gov/sbirt |
| **Contemplation** | **Considering change** | **Explore ambivalence**, **Discuss pros/cons** | https://www.samhsa.gov/sbirt |
| **Preparation** | **Planning change** | **Develop action plan**, **Set specific goals** | https://www.samhsa.gov/sbirt |
| **Action** | **Actively changing** | **Provide support**, **Reinforce success** | https://www.samhsa.gov/sbirt |
| **Maintenance** | **Sustained change** | **Relapse prevention**, **Address triggers** | https://www.samhsa.gov/sbirt |
