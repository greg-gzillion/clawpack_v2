## Acute Intoxication
| Symptom | Description | URL |
|---------|-------------|-----|
| **Cardiovascular** | **Tachycardia**, **Hypertension**, **Arrhythmias**, **Vasospasm** (cocaine) | https://www.nida.nih.gov/research-topics/stimulants |
| **CNS** | **Euphoria**, **Hypervigilance**, **Agitation**, **Paranoia**, **Hallucinations**, **Seizures** | https://www.nida.nih.gov/research-topics/stimulants |
| **Other** | **Mydriasis**, **Diaphoresis**, **Hyperthermia**, **Bruxism** | https://www.nida.nih.gov/research-topics/stimulants |
| **Management** | **Benzodiazepines** (first-line), **Active cooling**, **IV fluids**, **Avoid beta-blockers** (cocaine - unopposed alpha) | https://www.nida.nih.gov/research-topics/stimulants |

## Withdrawal
| Phase | Symptoms | Duration | URL |
|-------|----------|----------|-----|
| **Acute (Crash)** | **Fatigue**, **Hypersomnia**, **Depression**, **Anxiety**, **Increased appetite** | **1-3 days** | https://www.nida.nih.gov/research-topics/stimulants |
| **Subacute** | **Anhedonia**, **Dysphoria**, **Craving**, **Irritability** | **1-3 weeks** | https://www.nida.nih.gov/research-topics/stimulants |
| **Protracted** | **Intermittent craving**, **Mood instability** | **Weeks-months** | https://www.nida.nih.gov/research-topics/stimulants |

## Excited Delirium
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Extreme agitation**, **Aggression**, **Hyperthermia**, **Superhuman strength**, **Inability to fatigue** | https://www.nida.nih.gov/research-topics/stimulants |
| **Associated Substances** | **Cocaine**, **Methamphetamine**, **Synthetic cathinones ("bath salts")** | https://www.nida.nih.gov/research-topics/stimulants |
| **Complications** | **Metabolic acidosis**, **Rhabdomyolysis**, **Hyperkalemia**, **Cardiac arrest** | https://www.nida.nih.gov/research-topics/stimulants |
