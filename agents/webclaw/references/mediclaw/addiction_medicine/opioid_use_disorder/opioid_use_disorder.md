# Addiction Medicine - Opioid Use Disorder (OUD)

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Problematic pattern of opioid use** leading to clinically significant impairment or distress | https://www.samhsa.gov/medications-substance-use-disorders |
| **Prevalence** | **~2-3 million** Americans with OUD | https://www.cdc.gov/opioids/data/index.html |
| **Overdose Deaths** | **>80,000** annually in US (predominantly fentanyl) | https://www.cdc.gov/opioids/data/index.html |

## Medications for Opioid Use Disorder (MOUD)
| Medication | Setting | Dose | Considerations | URL |
|------------|---------|------|----------------|-----|
| **Buprenorphine** | **Office-based** (waivered) | **8-24 mg SL daily** | **First-line**; **Precipitates withdrawal** if given too soon (COWS =8-12) | https://www.samhsa.gov/medications-substance-use-disorders |
| **Methadone** | **OTP only** | **80-120 mg daily** | **Most effective** for retention; **QTc monitoring**; **Respiratory depression risk** | https://www.samhsa.gov/medications-substance-use-disorders |
| **Naltrexone** | **Office-based** | **380 mg IM monthly** | Requires **7-14 days opioid-free**; **Not first-line**; **High attrition** | https://www.samhsa.gov/medications-substance-use-disorders |

## Buprenorphine Initiation
| Protocol | Description | URL |
|----------|-------------|-----|
| **Standard Induction** | **Wait for COWS =8-12**, **Start 2-4 mg**, **Reassess q1-2h**, **Titrate to 8-16 mg day 1** | https://www.samhsa.gov/medications-substance-use-disorders |
| **Microdosing (Bernese Method)** | **Start 0.5 mg while continuing full agonist**, **Gradually increase buprenorphine** while **tapering full agonist** | https://www.samhsa.gov/medications-substance-use-disorders |
| **Home Induction** | **Appropriate for motivated, stable patients** with clear instructions | https://www.samhsa.gov/medications-substance-use-disorders |

## Opioid Overdose and Naloxone
| Component | Description | URL |
|-----------|-------------|-----|
| **Signs** | **Unresponsiveness**, **Respiratory depression**, **Pinpoint pupils** | https://www.cdc.gov/stop-overdose/caring/naloxone.html |
| **Naloxone Formulations** | **Narcan Nasal Spray** (4 mg), **IM/IV** (0.4 mg) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/naloxone |
| **Administration** | **1 dose q2-3min** until responsive or EMS arrival | https://www.cdc.gov/stop-overdose/caring/naloxone.html |
