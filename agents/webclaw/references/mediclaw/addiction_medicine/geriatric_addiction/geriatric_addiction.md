# Addiction Medicine - Geriatric Addiction

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Prevalence** | **~3-5%** of older adults have SUD; **Increasing** as population ages | https://www.samhsa.gov/data/report/2021-nsduh-annual-national-report |
| **Unique Challenges** | **Polypharmacy**, **Medical comorbidities**, **Cognitive impairment**, **Social isolation**, **Ageism** | https://www.niaaa.nih.gov/alcohol-health/special-populations-co-occurring-disorders/older-adults |

## Alcohol Use in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| **Low-Risk Limits** | **Same as younger adults** (=7 drinks/week, =3/day women; =14/week, =4/day men) | https://www.niaaa.nih.gov |
| **Physiologic Changes** | **? Lean body mass**, **? Hepatic metabolism**, **? CNS sensitivity** ? **Higher BAC at same dose** | https://www.niaaa.nih.gov/alcohol-health/special-populations-co-occurring-disorders/older-adults |
| **Atypical Presentation** | **Falls**, **Cognitive impairment**, **Depression**, **Malnutrition**, **Self-neglect** | https://www.niaaa.nih.gov/alcohol-health/special-populations-co-occurring-disorders/older-adults |
| **Screening** | **SMAST-G** (Short Michigan Alcoholism Screening Test-Geriatric) | https://www.niaaa.nih.gov/alcohol-health/special-populations-co-occurring-disorders/older-adults |

## Prescription Drug Misuse in Older Adults
| Substance | Considerations | URL |
|-----------|----------------|-----|
| **Opioids** | **Chronic pain common**, **Risk of falls**, **Cognitive impairment**, **Respiratory depression** | https://www.nida.nih.gov/research-topics/older-adults |
| **Benzodiazepines** | **Avoid** (Beers Criteria) - **Falls**, **Cognitive impairment**, **Delirium** | https://dcri.org/beers-criteria/ |
| **Sleep Medications** | **Avoid nonbenzodiazepine hypnotics** (zolpidem) - **Falls**, **Delirium** | https://dcri.org/beers-criteria/ |
