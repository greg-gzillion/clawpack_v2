# Forensic Medicine - Forensic Toxicology

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Detection and interpretation of drugs and poisons** in biological specimens for medicolegal purposes | https://www.aafs.org/forensic-science-forensic-toxicology |
| **Specimens** | **Blood**, **Urine**, **Vitreous humor**, **Bile**, **Liver**, **Gastric contents**, **Hair**, **Nails** | https://www.aafs.org/forensic-science-forensic-toxicology |

## Specimen Selection
| Specimen | Advantages | Disadvantages | URL |
|----------|------------|---------------|-----|
| **Blood** | **Quantitative**, **Correlates with effects** | **Invasive**, **Limited volume** | https://www.aafs.org/forensic-science-forensic-toxicology |
| **Urine** | **Non-invasive**, **Longer detection window**, **Concentrated** | **Qualitative only**, **Easily adulterated** | https://www.aafs.org/forensic-science-forensic-toxicology |
| **Vitreous Humor** | **Protected from putrefaction**, **Stable** | **Limited volume** | https://www.aafs.org/forensic-science-forensic-toxicology |
| **Bile** | **Concentrated**, **Detects drugs not in blood** | **Qualitative** | https://www.aafs.org/forensic-science-forensic-toxicology |
| **Liver** | **Large quantity**, **High concentrations** | **Qualitative**, **Postmortem redistribution** | https://www.aafs.org/forensic-science-forensic-toxicology |
| **Hair** | **Long detection window** (months), **Segmental analysis** | **External contamination**, **Hair color bias** | https://www.aafs.org/forensic-science-forensic-toxicology |

## Postmortem Redistribution
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Movement of drugs from tissues into blood** after death | https://www.aafs.org/forensic-science-forensic-toxicology |
| **High Redistribution** | **Lipophilic**, **High Vd**, **Basic drugs** (TCAs, Amphetamines, Opioids, Digoxin) | https://www.aafs.org/forensic-science-forensic-toxicology |
| **Low Redistribution** | **Hydrophilic**, **Low Vd**, **Acidic drugs** (Salicylates, Acetaminophen, Alcohols) | https://www.aafs.org/forensic-science-forensic-toxicology |
| **Preferred Site** | **Peripheral blood** (femoral) > Central blood (heart) | https://www.aafs.org/forensic-science-forensic-toxicology |
