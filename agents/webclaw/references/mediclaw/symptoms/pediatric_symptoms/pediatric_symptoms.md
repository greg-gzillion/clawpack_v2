# Symptoms - Pediatric Symptoms

## Fever Without a Source
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Fever =38°C (100.4°F)** without localizing signs | https://www.healthychildren.org/English/health-issues/conditions/fever/Pages/Fever-Without-Fear.aspx |
| **Neonate (<28 days)** | **High risk of serious bacterial infection (SBI)** ; **Full sepsis workup + Empiric antibiotics** | https://www.aap.org/en/patient-care/fever/ |
| **Infant 29-90 days** | **Risk stratification** (Rochester, Philadelphia, PECARN criteria) | https://www.aap.org/en/patient-care/fever/ |
| **Older Infant/Child** | **Viral URI** (most common), **Otitis media**, **UTI**, **Pneumonia**, **Occult bacteremia** (rare in vaccinated) | https://www.aap.org/en/patient-care/fever/ |
| **Red Flags** | **Age <28 days**, **Ill-appearing**, **Lethargy/Irritability**, **Poor feeding**, **Petechiae**, **Bulging fontanelle** | https://www.healthychildren.org/English/health-issues/conditions/fever/Pages/Fever-Without-Fear.aspx |

## Acute Otitis Media (AOM)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Middle ear infection** with **bulging TM**, **Erythema**, **Pain** | https://www.healthychildren.org/English/health-issues/conditions/ear-nose-throat/Pages/Ear-Infection-Information.aspx |
| **Diagnostic Criteria** | **Bulging TM** (most specific), **Impaired mobility**, **Otorrhea**, **Erythema** (less specific) | https://www.aap.org/en/patient-care/acute-otitis-media/ |
| **Observation Criteria** | **=6 months**, **Unilateral**, **Mild symptoms**, **No otorrhea**, **Reliable follow-up** | https://www.aap.org/en/patient-care/acute-otitis-media/ |
| **Treatment** | **Amoxicillin** 80-90 mg/kg/day (first-line), **Amoxicillin-clavulanate** (treatment failure) | https://www.aap.org/en/patient-care/acute-otitis-media/ |
| **Red Flags** | **Age <6 months**, **Bilateral disease**, **Severe symptoms**, **Immunocompromise**, **Anatomic abnormalities** | https://www.healthychildren.org/English/health-issues/conditions/ear-nose-throat/Pages/Ear-Infection-Information.aspx |
