## Sore Throat (Pharyngitis)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Pain, scratchiness, or irritation of the throat** | https://www.mayoclinic.org/diseases-conditions/sore-throat/symptoms-causes/syc-20351635 |
| **Differential Diagnosis** | **Viral** (most common: rhinovirus, adenovirus, EBV), **Bacterial** (GAS - strep throat), **Fungal** (Candida), **Non-infectious** (GERD, Allergies, Dry air, Irritants) | https://www.merckmanuals.com/professional/ear,-nose,-and-throat-disorders/approach-to-the-patient-with-nasal-and-pharyngeal-symptoms/sore-throat |
| **Centor Criteria** | **Fever**, **Tonsillar exudates**, **Tender anterior cervical LAD**, **Absence of cough** (score =3 ? test for GAS) | https://www.mdcalc.com/calc/104/centor-score-modified-mcisaac-strep-pharyngitis |
| **Red Flags** | **Drooling**, **Stridor**, **Trismus**, **Muffled voice** ("hot potato"), **Unilateral tonsillar swelling** (peritonsillar abscess) | https://www.mayoclinic.org/diseases-conditions/sore-throat/symptoms-causes/syc-20351635 |

| Component | Description | URL |

## Tinnitus
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Perception of sound without external source** (ringing, buzzing, hissing) | https://www.mayoclinic.org/diseases-conditions/tinnitus/symptoms-causes/syc-20350156 |
| **Subjective** | **Most common**; Audible only to patient | https://www.nidcd.nih.gov/health/tinnitus |
| **Objective** | **Rare**; Audible to examiner (vascular, Muscular) | https://www.nidcd.nih.gov/health/tinnitus |
| **Differential Diagnosis** | **Sensorineural hearing loss** (presbycusis, Noise-induced), **Meniere disease**, **Acoustic neuroma**, **Ototoxic medications** (NSAIDs, Aspirin, Aminoglycosides), **Vascular** (pulsatile), **TMJ disorder** | https://www.merckmanuals.com/professional/ear,-nose,-and-throat-disorders/approach-to-the-patient-with-ear-problems/tinnitus |
| **Red Flags** | **Unilateral tinnitus**, **Pulsatile tinnitus**, **Associated neurologic symptoms**, **Sudden onset** | https://www.mayoclinic.org/diseases-conditions/tinnitus/symptoms-causes/syc-20350156 |
