# Symptoms - General and Constitutional Symptoms

## Fever
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Body temperature =38.0°C (100.4°F)** | https://www.mayoclinic.org/diseases-conditions/fever/symptoms-causes/syc-20352759 |
| **Differential Diagnosis** | **Infection** (viral, bacterial), **Inflammatory** (RA, SLE, IBD), **Malignancy** (lymphoma, Renal cell carcinoma, Leukemia), **Drug fever**, **Endocrine** (thyroid storm, Adrenal insufficiency), **VTE** | https://www.merckmanuals.com/professional/infectious-diseases/biology-of-infectious-disease/fever |
| **Fever of Unknown Origin (FUO)** | **Fever =38.3°C** for **=3 weeks** without diagnosis despite **1 week of inpatient investigation** | https://www.merckmanuals.com/professional/infectious-diseases/biology-of-infectious-disease/fever-of-unknown-origin-fuo |
| **Red Flags** | **Altered mental status**, **Neck stiffness**, **Hypotension**, **Immunocompromise**, **Recent travel**, **Prosthetic material** | https://www.mayoclinic.org/diseases-conditions/fever/symptoms-causes/syc-20352759 |

## Fatigue
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Persistent, overwhelming tiredness** not relieved by rest | https://www.mayoclinic.org/symptoms/fatigue/basics/definition/sym-20050894 |
| **Differential Diagnosis** | **Anemia**, **Thyroid disease** (hypothyroidism), **Sleep disorders** (OSA, Insomnia), **Depression/Anxiety**, **Chronic fatigue syndrome**, **Malignancy**, **Heart failure**, **CKD**, **Liver disease**, **Autoimmune** (SLE, RA), **Medications** | https://www.merckmanuals.com/professional/special-subjects/nonspecific-symptoms/fatigue |
| **Red Flags** | **Unintentional weight loss**, **Lymphadenopathy**, **Fever**, **Night sweats**, **Focal neurologic deficits**, **Dyspnea** | https://www.mayoclinic.org/symptoms/fatigue/basics/when-to-see-doctor/sym-20050894 |

## Unintentional Weight Loss
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Loss of >5% body weight over 6-12 months** without intentional effort | https://www.mayoclinic.org/symptoms/unexplained-weight-loss/basics/definition/sym-20050700 |
| **Differential Diagnosis** | **Malignancy** (GI, Lung, Lymphoma), **Endocrine** (Hyperthyroidism, Diabetes, Adrenal insufficiency), **GI** (Malabsorption, IBD, Celiac), **Chronic infection** (HIV, TB), **Psychiatric** (Depression, Eating disorder), **Heart failure**, **COPD**, **CKD** | https://www.merckmanuals.com/professional/nutritional-disorders/nutritional-support/undernutrition |
| **Red Flags** | **Age >50**, **Smoking history**, **Family history of malignancy**, **Dysphagia**, **Early satiety**, **Abdominal pain**, **Change in bowel habits**, **Lymphadenopathy** | https://www.mayoclinic.org/symptoms/unexplained-weight-loss/basics/when-to-see-doctor/sym-20050700 |

## Night Sweats
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Drenching sweats** requiring change of bedclothes, **Not related to environmental temperature** | https://www.mayoclinic.org/symptoms/night-sweats/basics/definition/sym-20050768 |
| **Differential Diagnosis** | **Malignancy** (lymphoma - classic B symptom), **Infection** (TB, HIV, Endocarditis, Osteomyelitis), **Endocrine** (Menopause, Hyperthyroidism, Pheochromocytoma), **Medications** (antidepressants, Antipyretics), **Anxiety**, **OSA**, **GERD** | https://www.merckmanuals.com/professional/pulmonary-disorders/symptoms-of-pulmonary-disorders/night-sweats |
| **B Symptoms (Lymphoma)** | **Fever**, **Night sweats**, **Unintentional weight loss** | https://www.cancer.gov/types/lymphoma |

## Lymphadenopathy
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Enlarged lymph nodes** (>1 cm) | https://www.mayoclinic.org/diseases-conditions/swollen-lymph-nodes/symptoms-causes/syc-20353902 |
| **Localized** | **Infection** (bacterial, viral), **Malignancy** (metastatic, Lymphoma) | https://www.merckmanuals.com/professional/cardiovascular-disorders/lymphatic-disorders/lymphadenopathy |
| **Generalized** | **Infection** (EBV, CMV, HIV, TB), **Autoimmune** (SLE, RA, Sarcoidosis), **Malignancy** (lymphoma, Leukemia), **Medications** (phenytoin, allopurinol) | https://www.merckmanuals.com/professional/cardiovascular-disorders/lymphatic-disorders/lymphadenopathy |
| **Red Flags** | **Fixed, hard, non-tender nodes**, **>2 cm**, **Supraclavicular location** (Virchow node), **Constitutional symptoms**, **Rapid enlargement** | https://www.mayoclinic.org/diseases-conditions/swollen-lymph-nodes/symptoms-causes/syc-20353902 |
