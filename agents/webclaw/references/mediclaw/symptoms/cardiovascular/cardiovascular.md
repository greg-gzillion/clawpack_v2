## Palpitations
| Component | Description | URL |
|-----------|-------------|-----|

| **Cardiac Causes** | **Arrhythmias** (AF, SVT, PVCs, VT), **Valvular disease**, **Cardiomyopathy**, **Pacemaker** | https://www.merckmanuals.com/professional/cardiovascular-disorders/symptoms-of-cardiovascular-disorders/palpitations |

## Syncope
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Transient loss of consciousness** due to **global cerebral hypoperfusion**, with **rapid, spontaneous recovery** | https://www.mayoclinic.org/diseases-conditions/syncope/symptoms-causes/syc-20352247 |
| **Reflex (Neurocardiogenic)** | **Vasovagal** (most common), **Situational** (micturition, cough, defecation), **Carotid sinus hypersensitivity** | https://www.merckmanuals.com/professional/cardiovascular-disorders/symptoms-of-cardiovascular-disorders/syncope |
| **Orthostatic Hypotension** | **? BP with standing** (volume depletion, Medications, Autonomic dysfunction) | https://www.merckmanuals.com/professional/cardiovascular-disorders/symptoms-of-cardiovascular-disorders/syncope |
| **Cardiac** | **Arrhythmias** (bradycardia, tachycardia), **Structural** (AS, HCM, PE, Tamponade) | https://www.merckmanuals.com/professional/cardiovascular-disorders/symptoms-of-cardiovascular-disorders/syncope |
| **Red Flags** | **Syncope during exertion**, **Family history of sudden death**, **Abnormal ECG**, **No prodrome**, **Chest pain**, **Palpitations** | https://www.mayoclinic.org/diseases-conditions/syncope/symptoms-causes/syc-20352247 |

## Edema
| Component | Description | URL |

| Component | Description | URL |
|-----------|-------------|-----|
