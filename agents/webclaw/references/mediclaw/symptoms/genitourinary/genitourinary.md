## Hematuria
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Blood in urine** (gross or microscopic) | https://www.mayoclinic.org/diseases-conditions/blood-in-urine/symptoms-causes/syc-20353432 |
| **Gross Hematuria** | **Visible blood** (pink, red, cola-colored) | https://www.auanet.org/guidelines/hematuria |
| **Microscopic Hematuria** | **=3 RBCs/HPF** on urinalysis | https://www.auanet.org/guidelines/hematuria |
| **Differential Diagnosis** | **UTI**, **Nephrolithiasis**, **Malignancy** (bladder, kidney, prostate), **BPH**, **Glomerulonephritis**, **Trauma**, **Strenuous exercise** | https://www.merckmanuals.com/professional/genitourinary-disorders/symptoms-of-genitourinary-disorders/hematuria |
| **Red Flags** | **Gross hematuria**, **Age >35**, **Smoking history**, **Occupational exposures** (dyes, rubber), **Analgesic abuse**, **Pelvic radiation** | https://www.mayoclinic.org/diseases-conditions/blood-in-urine/symptoms-causes/syc-20353432 |
