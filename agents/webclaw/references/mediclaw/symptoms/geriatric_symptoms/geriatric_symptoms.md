# Symptoms - Geriatric Symptoms

## Falls
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Unintentional coming to rest on ground or lower level** | https://www.cdc.gov/falls/index.html |
| **Intrinsic Risk Factors** | **Age**, **Prior falls**, **Gait/Balance impairment**, **Muscle weakness**, **Visual impairment**, **Cognitive impairment**, **Orthostatic hypotension**, **Polypharmacy** | https://www.cdc.gov/steadi |
| **Extrinsic Risk Factors** | **Environmental hazards** (rugs, poor lighting, Clutter), **Improper footwear** | https://www.cdc.gov/steadi |
| **Screening** | **"Have you fallen in the past year?"**, **Timed Up and Go (TUG)** (=12 sec = increased risk) | https://www.cdc.gov/steadi |
| **Interventions** | **Exercise** (balance, strength), **Medication review**, **Vitamin D**, **Home safety**, **Vision correction** | https://www.cdc.gov/steadi |
| **Red Flags** | **Syncope**, **Head trauma**, **Anticoagulation**, **Inability to get up** | https://www.cdc.gov/falls/index.html |

## Delirium
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Acute, fluctuating disturbance in attention and cognition** | https://www.nia.nih.gov/health/delirium |
| **Features** | **Acute onset**, **Fluctuating course**, **Inattention**, **Disorganized thinking**, **Altered level of consciousness** | https://www.merckmanuals.com/professional/neurologic-disorders/delirium-and-dementia/delirium |
| **Causes (I WATCH DEATH)** | **I**nfection, **W**ithdrawal, **A**cute metabolic, **T**rauma, **C**NS pathology, **H**ypoxia, **D**eficiencies, **E**ndocrine, **A**cute vascular, **T**oxins/drugs, **H**eavy metals | https://www.merckmanuals.com/professional/neurologic-disorders/delirium-and-dementia/delirium |
| **Screening** | **Confusion Assessment Method (CAM)** | https://www.hospitalelderlifeprogram.org/delirium-instruments |
| **Management** | **Treat underlying cause**, **Environmental modifications**, **Avoid restraints**, **Antipsychotics** (low-dose for severe agitation) | https://www.nia.nih.gov/health/delirium |
| **Red Flags** | **Hypoactive delirium** (often missed), **Refractory agitation** | https://www.nia.nih.gov/health/delirium |
