## Pruritus (Itching)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Unpleasant sensation provoking urge to scratch** | https://www.mayoclinic.org/diseases-conditions/itchy-skin/symptoms-causes/syc-20355006 |
| **Dermatologic Causes** | **Atopic dermatitis**, **Urticaria**, **Psoriasis**, **Scabies**, **Contact dermatitis**, **Xerosis** (dry skin) | https://www.merckmanuals.com/professional/dermatologic-disorders/approach-to-the-dermatologic-patient/itching |
| **Systemic Causes** | **Cholestasis** (PBC, ICP), **CKD** (uremic pruritus), **Hematologic** (polycythemia vera, Lymphoma, Iron deficiency), **Thyroid disease**, **Malignancy** | https://www.merckmanuals.com/professional/dermatologic-disorders/approach-to-the-dermatologic-patient/itching |
| **Red Flags** | **Generalized pruritus without rash**, **Weight loss**, **Night sweats**, **Jaundice**, **Lymphadenopathy** | https://www.mayoclinic.org/diseases-conditions/itchy-skin/symptoms-causes/syc-20355006 |
