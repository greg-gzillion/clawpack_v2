# Neurosurgery - Spinal Cord Injury (SCI)

## ASIA Impairment Scale (AIS)
| Grade | Definition | URL |
|-------|------------|-----|
| **A** | **Complete**: No sensory or motor function in S4-S5 | https://www.asia-spinalinjury.org/learning/asia-iscos-exam-sheet/ |
| **B** | **Sensory Incomplete**: Sensory but no motor function below level, **including S4-S5** | https://www.asia-spinalinjury.org/learning/asia-iscos-exam-sheet/ |
| **C** | **Motor Incomplete**: Motor function below level, **>50% key muscles grade <3** | https://www.asia-spinalinjury.org/learning/asia-iscos-exam-sheet/ |
| **D** | **Motor Incomplete**: Motor function below level, **=50% key muscles grade =3** | https://www.asia-spinalinjury.org/learning/asia-iscos-exam-sheet/ |
| **E** | **Normal**: Normal motor and sensory function | https://www.asia-spinalinjury.org/learning/asia-iscos-exam-sheet/ |

| Level | Key Motor Function | Functional Outcome | URL |
|-------|-------------------|-------------------|-----|
| **C1-C4** | **Diaphragm** (C3-5), Neck muscles | **Ventilator dependent** (complete above C3), Power wheelchair with chin/head control | https://www.asia-spinalinjury.org/learning/asia-iscos-exam-sheet/ |
| **C5** | **Deltoid**, **Biceps** | **Elbow flexion**, Power wheelchair with hand controls, **Requires assistance for ADLs** | https://www.asia-spinalinjury.org/learning/asia-iscos-exam-sheet/ |
| **C6** | **Wrist extension**, Biceps | **Tenodesis grasp**, Manual wheelchair, **Independent feeding/grooming** | https://www.asia-spinalinjury.org/learning/asia-iscos-exam-sheet/ |
| **C7** | **Triceps**, **Wrist flexion**, **Finger extension** | **Transfers**, **Independent ADLs**, Manual wheelchair | https://www.asia-spinalinjury.org/learning/asia-iscos-exam-sheet/ |
| **C8** | **Finger flexion** | Improved hand function | https://www.asia-spinalinjury.org/learning/asia-iscos-exam-sheet/ |
| **T1-T12** | Intercostals, Abdominals | **Independent with manual wheelchair**, Standing frame, Brace walking (T10-L2) | https://www.asia-spinalinjury.org/learning/asia-iscos-exam-sheet/ |

## Spinal Shock vs Neurogenic Shock
| Feature | Spinal Shock | Neurogenic Shock | URL |
|---------|--------------|------------------|-----|
| **Definition** | **Transient loss of all neurologic function** below injury | **Loss of sympathetic tone** ? Vasodilation, Hypotension | https://www.ncbi.nlm.nih.gov/books/NBK537247/ |
| **Onset** | Immediate | Minutes-hours | https://www.ncbi.nlm.nih.gov/books/NBK537247/ |
| **Duration** | Days-weeks | Days-weeks | https://www.ncbi.nlm.nih.gov/books/NBK537247/ |
| **Resolution** | **Bulbocavernosus reflex** returns first | Gradual | https://www.ncbi.nlm.nih.gov/books/NBK537247/ |
| **Motor** | **Flaccid paralysis**, **Areflexia** | Variable (depends on injury) | https://www.ncbi.nlm.nih.gov/books/NBK537247/ |
| **Cardiovascular** | Normal BP/HR | **Hypotension**, **Bradycardia** | https://www.ncbi.nlm.nih.gov/books/NBK537247/ |
| **Treatment** | Supportive | **IV fluids**, **Vasopressors** (phenylephrine, norepinephrine) | https://www.ncbi.nlm.nih.gov/books/NBK537247/ |

## Acute SCI Management
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **MAP Goal** | **85-90 mmHg** for first 7 days (AANS/CNS Guidelines) | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Spinal-Cord-Injury |
| **Steroids** | **NOT recommended** (AANS/CNS 2013) due to lack of benefit and increased complications | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Spinal-Cord-Injury |
| **Early Surgery** | **Within 24 hours** if indicated | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Spinal-Cord-Injury |
| **Venous Thromboembolism Prophylaxis** | **LMWH** within 72 hours (if no bleeding) + Mechanical | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Spinal-Cord-Injury |

## Autonomic Dysreflexia
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Life-threatening sympathetic surge in response to noxious stimuli **below level of SCI** (usually =T6) | https://www.ninds.nih.gov/health-information/disorders/autonomic-dysreflexia |
| **Triggers** | **Bladder distention** (most common), Bowel impaction, UTI, Pressure injury, Ingrown toenail | https://www.ninds.nih.gov/health-information/disorders/autonomic-dysreflexia |
| **Symptoms** | **Severe hypertension**, **Bradycardia** (reflex), **Headache**, **Flushing/sweating above lesion** | https://www.ninds.nih.gov/health-information/disorders/autonomic-dysreflexia |
| **Management** | **Sit upright**, **Remove trigger** (check Foley, Perform CIC), **Rapid-acting antihypertensive** (nifedipine, nitroglycerin paste) if BP remains elevated | https://www.ninds.nih.gov/health-information/disorders/autonomic-dysreflexia |
