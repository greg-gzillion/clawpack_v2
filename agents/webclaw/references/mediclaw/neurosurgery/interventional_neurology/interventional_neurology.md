| Technique | Description | URL |
|-----------|-------------|-----|
| **Coil Embolization** | **Detachable platinum coils** delivered into aneurysm sac | https://www.snisonline.org |

## Acute Ischemic Stroke Intervention
| Component | Description | URL |
|-----------|-------------|-----|
| **IV tPA Window** | **<4.5 hours** from last known well | https://www.heart.org/en/professional/quality-improvement/get-with-the-guidelines/get-with-the-guidelines-stroke |
| **Mechanical Thrombectomy Window** | **<24 hours** for LVO with perfusion mismatch (DAWN/DEFUSE-3 criteria) | https://www.heart.org/en/professional/quality-improvement/get-with-the-guidelines/get-with-the-guidelines-stroke |
| **DAWN Trial** | Thrombectomy **6-24 hours** for patients with **clinical-imaging mismatch** | https://www.nejm.org/doi/full/10.1056/NEJMoa1706442 |
| **DEFUSE 3 Trial** | Thrombectomy **6-16 hours** with **perfusion mismatch** | https://www.nejm.org/doi/full/10.1056/NEJMoa1713973 |
| **Techniques** | **Stent retriever**, **Aspiration catheter**, **Combined** | https://www.snisonline.org |

## Carotid Artery Stenting (CAS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | **Symptomatic =50%** or **Asymptomatic =70%** stenosis, **High surgical risk** for CEA | https://www.snisonline.org |
| **Embolic Protection** | **Distal filter** or **Proximal balloon occlusion** | https://www.snisonline.org |
| **Dual Antiplatelet Therapy** | **Aspirin + Clopidogrel** pre- and post-procedure | https://www.snisonline.org |
| **Complications** | **Stroke**, **Hyperperfusion syndrome**, **Bradycardia/Hypotension** (carotid sinus stimulation) | https://www.snisonline.org |

## Arteriovenous Malformation (AVM) Embolization
| Component | Description | URL |
|-----------|-------------|-----|
| **Goal** | **Preoperative** (reduce flow), **Preradiosurgical** (reduce volume), **Palliative** (symptom control), **Curative** (rare) | https://www.snisonline.org |
| **Embolic Agents** | **n-BCA** (glue), **Onyx** (EVOH copolymer), **PVA particles**, **Coils** | https://www.snisonline.org |
| **Complications** | **Hemorrhage** (periprocedural rupture, Post-embolization edema/hemorrhage), **Ischemia** (non-target embolization) | https://www.snisonline.org |
