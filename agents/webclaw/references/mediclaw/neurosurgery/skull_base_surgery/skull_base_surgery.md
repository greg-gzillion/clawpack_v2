# Neurosurgery - Skull Base Surgery

## Common Skull Base Approaches
| Approach | Indications | Key Anatomy | URL |
|----------|-------------|-------------|-----|
| **Pterional (Frontotemporal)** | **Anterior circulation aneurysms**, **Suprasellar tumors**, **Anterior temporal lesions** | Sylvian fissure, ICA, MCA, ACA, Optic nerve | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Skull-Base-Tumors |
| **Retrosigmoid** | **Cerebellopontine angle tumors** (vestibular schwannoma, Meningioma), **Trigeminal neuralgia (MVD)** | CN VII-VIII, CN V, AICA, Sigmoid sinus | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Skull-Base-Tumors |
| **Translabyrinthine** | **Vestibular schwannoma** with **no serviceable hearing** | Labyrinth, IAC, CN VII-VIII | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Skull-Base-Tumors |
| **Middle Fossa** | **Small intracanalicular vestibular schwannoma** (hearing preservation), **Facial nerve decompression** | IAC, Geniculate ganglion, Superior semicircular canal | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Skull-Base-Tumors |
| **Transsphenoidal** | **Pituitary adenoma**, **Craniopharyngioma**, **Rathke cleft cyst** | Sphenoid sinus, Sella turcica, Pituitary gland | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Pituitary-Tumors |
| **Endoscopic Endonasal** | **Expanded indications**: Olfactory groove meningioma, Planum sphenoidale meningioma, Clival chordoma | Nasal cavity, Sphenoid sinus, Sella, Clivus | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Endoscopic-Skull-Base-Surgery |
| **Orbitozygomatic** | **Large/complex** anterior/middle fossa lesions, **Basilar apex aneurysms** | Orbit, Zygoma, Sylvian fissure | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Skull-Base-Tumors |
| **Far Lateral** | **Foramen magnum lesions**, **Lower clival lesions**, **PICA aneurysms** | Vertebral artery, PICA, C1 lateral mass, Occipital condyle | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Skull-Base-Tumors |

## Pituitary Adenoma
| Type | Hormone | Clinical Features | Treatment | URL |
|------|----------|-------------------|-----------|-----|
| **Prolactinoma** | **Prolactin** | **Galactorrhea**, **Amenorrhea**, Hypogonadism | **Dopamine agonists** (cabergoline); Surgery if refractory | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Pituitary-Tumors |
| **Acromegaly** | **GH** | Enlarged hands/feet, Coarse facies, Prognathism, **? IGF-1** | **Surgery** (transsphenoidal), Somatostatin analogs, Pegvisomant | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Pituitary-Tumors |
| **Cushing Disease** | **ACTH** | **Central obesity**, Moon facies, Buffalo hump, Striae, HTN, Diabetes | **Surgery** (transsphenoidal) | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Pituitary-Tumors |
| **Non-Functioning** | None | **Mass effect** (headache, Visual field deficit, Hypopituitarism) | **Surgery** if symptomatic or growing | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Pituitary-Tumors |
| **TSH-oma** | **TSH** | **Hyperthyroidism** with **inappropriately normal/elevated TSH** | **Surgery** | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Pituitary-Tumors |

## Craniopharyngioma
| Component | Description | URL |
|-----------|-------------|-----|
| **Origin** | **Rathke pouch remnants** | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Craniopharyngioma |
| **Location** | **Suprasellar** (often with intrasellar extension) | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Craniopharyngioma |
| **Histology** | **Adamantinomatous** (children, CTNNB1 mutation) vs **Papillary** (adults, BRAF V600E) | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Craniopharyngioma |
| **Imaging** | **Cystic + Solid + Calcification** | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Craniopharyngioma |
| **Symptoms** | **Visual field deficit** (bitemporal hemianopia), **Endocrine dysfunction** (GH deficiency, Hypothyroidism, Adrenal insufficiency, DI), **Hydrocephalus** | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Craniopharyngioma |
| **Treatment** | **Gross total resection** (if safe) vs **Subtotal resection + Radiation** | https://www.aans.org/Patients/Neurosurgical-Conditions-and-Treatments/Craniopharyngioma |
