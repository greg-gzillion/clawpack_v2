# Gynecology - Endometriosis and Adenomyosis

## Endometriosis - Overview
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Presence of endometrial-like tissue outside the uterus | https://www.acog.org/endometriosis |
| Prevalence | 10% of reproductive-age women; 50-70% of women with infertility/chronic pelvic pain | |
| Pathophysiology | **Retrograde menstruation** (Sampson's theory), Coelomic metaplasia, Lymphatic/vascular spread, Immune dysfunction, Genetic predisposition | |
| Common Sites | Ovaries (endometriomas), Posterior cul-de-sac, Uterosacral ligaments, Broad ligament, Rectovaginal septum, Bladder, Bowel | |

## Endometriosis - Diagnosis
| Method | Description | URL |
|--------|-------------|-----|
| **History and Physical** | Cyclic pain, Nodularity on rectovaginal exam, Fixed retroverted uterus, Adnexal tenderness/mass | https://www.asrm.org |
| **Transvaginal Ultrasound** | Detects **endometriomas** (ground-glass appearance), Deep infiltrating endometriosis (limited sensitivity) | |
| **MRI** | Useful for deep infiltrating endometriosis, Pre-operative planning | |
| **Laparoscopy with Biopsy** | **Gold standard for diagnosis**; Direct visualization (powder-burn lesions, Clear vesicles, Red flame lesions, White plaques) | |
| **Biomarkers** | **CA-125** (elevated, but poor sensitivity/specificity); Not recommended for diagnosis | |

## Endometriosis - Staging (rASRM Classification)
| Stage | Score | Description | URL |
|-------|-------|-------------|-----|
| Stage I (Minimal) | 1-5 | Superficial implants, Filmy adhesions | https://www.asrm.org |
| Stage II (Mild) | 6-15 | Deeper implants, Dense adhesions | |
| Stage III (Moderate) | 16-40 | Endometriomas, Cul-de-sac obliteration | |
| Stage IV (Severe) | >40 | Large endometriomas, Extensive adhesions, Organ involvement | |
| **Note** | **Stage does not correlate with pain severity or fertility prognosis** | |

## Endometriosis - Medical Management
| Treatment | Mechanism | Indications | Side Effects | URL |
|-----------|-----------|--------------|--------------|-----|
| **NSAIDs** | Prostaglandin inhibition | Mild pain, Dysmenorrhea | GI, Renal | https://www.acog.org/endometriosis-treatment |
| **Combined OCPs** (continuous) | Suppresses ovulation, Decidualizes endometrium | First-line for pain | Breakthrough bleeding, VTE risk | |
| **Progestins** (Norethindrone, DMPA, LNg-IUD) | Decidualizes/atrophies endometrium | First-line for pain | Irregular bleeding, Weight gain (DMPA) | |
| **GnRH Agonists** (Leuprolide) | Suppresses ovarian function (pseudo-menopause) | Second-line, Pre-surgical | **Vasomotor symptoms**, **Bone loss** (add-back therapy required) | |
| **GnRH Antagonists** (Elagolix, Relugolix) | Dose-dependent ovarian suppression | Second-line | Hot flashes, Bone loss | |
| **Aromatase Inhibitors** (Letrozole) | Reduces peripheral estrogen production | Refractory pain (often with progestin) | Bone loss, Ovarian cysts | |
| **Danazol** | Androgenic, Suppresses ovulation | Historical | **Androgenic side effects** (acne, Hirsutism, Voice changes) | |

## Endometriosis - Surgical Management
| Procedure | Description | Indications | URL |
|-----------|-------------|-------------|-----|
| **Diagnostic Laparoscopy** | Visualize and biopsy lesions | Diagnosis, Staging | https://www.aagl.org |
| **Laparoscopic Excision/Ablation** | Remove or destroy endometriosis lesions | **Pain relief**, Improved fertility | |
| **Ovarian Cystectomy** | Remove endometrioma | Symptomatic endometrioma, Infertility | |
| **Lysis of Adhesions** | Restore normal anatomy | Pain, Infertility | |
| **Presacral Neurectomy** | Interrupt uterine nerve fibers | Midline dysmenorrhea (controversial) | |
| **Hysterectomy ｱ BSO** | Remove uterus ｱ ovaries | **Definitive treatment** for completed childbearing | |
| **Postoperative Hormonal Suppression** | LNg-IUD, OCPs, Progestins | **Reduces recurrence** | |

## Endometriosis and Infertility
| Consideration | Recommendation | URL |
|---------------|----------------|-----|
| **Mechanisms** | Distorted anatomy, Impaired oocyte quality, Reduced endometrial receptivity, Pro-inflammatory peritoneal environment | https://www.asrm.org/endometriosis-infertility |
| **Surgical Treatment** | **Laparoscopic excision/ablation improves spontaneous pregnancy rates** | |
| **Endometrioma Management** | **Avoid cystectomy if possible** (reduces ovarian reserve); Consider drainage or ablation | |
| **ART** | **IVF** is effective; Consider GnRH agonist suppression for 2-3 months before IVF | |

## Adenomyosis
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Presence of endometrial glands and stroma within the myometrium | https://www.acog.org/adenomyosis |
| Pathophysiology | Invagination of endometrium into myometrium (junction zone disruption) | |
| Presentation | **Heavy menstrual bleeding**, **Dysmenorrhea**, Enlarged "boggy" uterus, Chronic pelvic pain | |
| Diagnosis | **Transvaginal ultrasound** (heterogeneous myometrium, Myometrial cysts, Poorly defined junctional zone); **MRI** (junction zone >12 mm) | |
| Medical Management | **LNg-IUD** (first-line), OCPs, Progestins, GnRH agonists | |
| Surgical Management | **Hysterectomy** (definitive); Uterine artery embolization, Endometrial ablation (limited efficacy) | |
