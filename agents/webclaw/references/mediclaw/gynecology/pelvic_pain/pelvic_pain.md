# Gynecology - Pelvic Pain

| Category | Causes | URL |
|----------|--------|-----|
| **Gynecologic** | Ectopic pregnancy, Ovarian torsion, Ruptured ovarian cyst, Tubo-ovarian abscess, PID, Degenerating fibroid, Dysmenorrhea | https://www.acog.org/pelvic-pain |
| **Gastrointestinal** | Appendicitis, Diverticulitis, Bowel obstruction, Inflammatory bowel disease, Irritable bowel syndrome | |
| **Urologic** | UTI/Pyelonephritis, Nephrolithiasis, Interstitial cystitis | |
| **Musculoskeletal** | Abdominal wall pain (Carnett's sign), Pelvic floor myalgia, Hernia | |
| **Other** | Porphyria, Sickle cell crisis, Herpes zoster | |

## Chronic Pelvic Pain (CPP)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Non-cyclic pelvic pain lasting =6 months; Associated with functional disability or treatment seeking | https://www.acog.org/chronic-pelvic-pain |
| Prevalence | 15-20% of women | |
| Common Causes | **Endometriosis** (70%), Adhesions, Adenomyosis, Fibroids, Pelvic congestion syndrome, Interstitial cystitis, IBS, Pelvic floor myofascial pain, Central sensitization | |
| Evaluation | Comprehensive history, Physical exam (abdominal, Pelvic, Pelvic floor), Imaging (TVUS), Consider laparoscopy | |
| Management | **Multidisciplinary approach**: Pain management, Physical therapy, Behavioral health, Hormonal suppression, Surgery | |

## Ovarian Cysts
| Type | Features | Management | URL |
|------|----------|------------|-----|
| **Functional Cyst (Follicular/Corpus Luteum)** | Most common, <10 cm, Unilocular, Anechoic | Expectant management; OCPs for suppression | https://www.acog.org/ovarian-cysts |
| **Hemorrhagic Cyst** | Acute pain, Internal echoes, Retracting clot | Expectant; Analgesia | |
| **Endometrioma** | Homogeneous low-level echoes, "Ground glass" | Hormonal suppression; Surgery if symptomatic | |
| **Dermoid (Mature Teratoma)** | Heterogeneous, Echogenic focus, Shadowing, Fluid-fluid levels | Surgical removal (risk of torsion) | |
| **Cystadenoma** | Unilocular or multilocular, Thin septations | Monitor or remove if large/symptomatic | |
| **Concerning Features** | **>10 cm**, **Solid components**, **Papillary projections**, **Thick septations**, **Ascites**, **Bilateral** | **Refer to gynecologic oncology** | |

## Ovarian Torsion
| Component | Description | URL |
|-----------|-------------|-----|
| Presentation | **Sudden severe unilateral pelvic pain**, Nausea/Vomiting, Adnexal tenderness | https://www.acog.org/ovarian-torsion |
| Risk Factors | Ovarian mass (=5 cm), Pregnancy, Ovulation induction | |
| Diagnosis | **Pelvic ultrasound with Doppler** (decreased/absent flow, Ovarian enlargement, Edema); **Note: Doppler can be normal** | |
| Management | **Emergent surgical detorsion** (preserve ovary even if necrotic-appearing); Oophoropexy if recurrent | |
| Fertility Preservation | **Detorsion does not increase thromboembolism risk**; Ovarian function often preserved | |

## Pelvic Congestion Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| Pathophysiology | Incompetent ovarian veins ? Venous reflux ? Pelvic varicosities | https://www.sirweb.org |
| Presentation | **Dull, aching pelvic pain**; **Worsens with prolonged standing**, **End of day**; **Improves with lying down**; Dyspareunia; Postcoital pain | |
| Diagnosis | **Transvaginal ultrasound** (dilated pelvic veins >5 mm, Reversed flow), **MR venography**, **Venography** (gold standard) | |
| Management | **Ovarian vein embolization** (interventional radiology); Hormonal suppression; Hysterectomy with ovarian vein ligation | |
