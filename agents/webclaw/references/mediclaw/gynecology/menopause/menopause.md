# Gynecology - Menopause

## Menopause - Definitions and Staging (STRAW+10)
| Stage | Definition | Hormonal Changes | URL |
|-------|------------|------------------|-----|
| **Reproductive** | Regular menses | Normal FSH, Estradiol | https://www.menopause.org/straw |
| **Menopausal Transition (Early)** | Variable cycle length (>7 days difference) | FSH may be elevated, Estradiol normal/high | |
| **Menopausal Transition (Late)** | =2 skipped cycles and =60 days amenorrhea | FSH =25 IU/L, Estradiol variable | |
| **Menopause** | **12 months of amenorrhea** | FSH >40 IU/L, Estradiol low | |
| **Postmenopause (Early)** | 1-6 years after menopause | FSH high, Estradiol very low | |
| **Postmenopause (Late)** | >6 years after menopause | FSH high (may decline), Estradiol very low | |

## Menopause - Clinical Presentation
| Symptom | Frequency | Duration | Management | URL |
|---------|-----------|----------|------------|-----|
| **Vasomotor Symptoms (Hot Flashes, Night Sweats)** | 75-80% | Median 7-10 years (longer in Black women) | **Hormone therapy** (most effective), **Non-hormonal** (SSRIs/SNRIs, Gabapentin, Clonidine, Fezolinetant), CBT, Lifestyle | https://www.menopause.org |
| **Genitourinary Syndrome of Menopause (GSM)** | 50-70% | Progressive | **Vaginal estrogen**, Vaginal DHEA, Ospemifene, Lubricants/Moisturizers | |
| **Sleep Disturbance** | 40-60% | Variable | Treat vasomotor symptoms, CBT-I, Exercise, Melatonin | |
| **Mood Changes** | 20-40% | Variable | Hormone therapy (perimenopause), SSRIs/SNRIs, CBT | |
| **Cognitive Changes** | Common | Variable | Reassurance; Evaluate for other causes | |
| **Arthralgias** | Common | Variable | Exercise, NSAIDs, Weight loss | |

## Menopausal Hormone Therapy (MHT) - Indications
| Indication | Recommendation | URL |
|------------|----------------|-----|
| **Vasomotor Symptoms** | **First-line treatment**; Most effective therapy | https://www.menopause.org/hormone-therapy |
| **Genitourinary Syndrome of Menopause (GSM)** | **Low-dose vaginal estrogen** (first-line); Ospemifene, Vaginal DHEA | |
| **Prevention of Bone Loss/Fractures** | Effective but **not first-line** (bisphosphonates preferred); Consider if also treating vasomotor symptoms | |
| **Primary Prevention of CVD** | **NOT indicated** | |
| **Primary Prevention of Dementia** | **NOT indicated** (may increase risk if started after age 65) | |

## Menopausal Hormone Therapy - Regimens
| Regimen | Indication | Examples | URL |
|---------|------------|----------|-----|
| **Estrogen Only** | **Hysterectomized women** | Oral estradiol, Transdermal estradiol, Vaginal ring | https://www.menopause.org |
| **Estrogen + Progestogen (Continuous-Combined)** | **Intact uterus** (prevents endometrial hyperplasia) | Oral estradiol + Norethindrone, Oral estradiol + Micronized progesterone, Transdermal estradiol + Oral progesterone | |
| **Estrogen + Progestogen (Cyclic)** | Intact uterus, Perimenopause | Oral estradiol daily + Progesterone 12-14 days/month | |
| **Progestogen Only** | Contraindication to estrogen | Norethindrone, DMPA, Megestrol acetate | |
| **Tissue-Selective Estrogen Complex (TSEC)** | Intact uterus | **Bazedoxifene + Conjugated estrogens (Duavee)** | |
| **Vaginal Estrogen** | GSM (minimal systemic absorption) | Estradiol cream, Estradiol tablet, Estradiol ring | |

## Menopausal Hormone Therapy - Risks and Benefits
| Outcome | Estrogen Only | Estrogen + Progestogen | URL |
|---------|---------------|------------------------|-----|
| **Vasomotor Symptoms** | **95% reduction** | **95% reduction** | https://www.menopause.org |
| **Osteoporotic Fractures** | **30-40% reduction** | **30-40% reduction** | |
| **Breast Cancer** | **No increased risk** (WHI) or slight decrease | **Slight increased risk** (~1 additional case per 1000 women per year after 5+ years) | |
| **Endometrial Cancer** | **Increased risk** (if uterus present) | **No increased risk** | |
| **VTE** | **Increased risk** (oral > transdermal) | **Increased risk** | |
| **Stroke** | **Increased risk** | **Increased risk** | |
| **Coronary Heart Disease** | Decreased if started <60 or <10 years since menopause | Neutral | |
| **Colorectal Cancer** | **Decreased risk** | **Decreased risk** | |
| **All-Cause Mortality** | **Decreased** if started <60 | Neutral/Decreased | |

## Non-Hormonal Treatments for Vasomotor Symptoms
| Treatment | Mechanism | Efficacy | Side Effects | URL |
|-----------|-----------|----------|--------------|-----|
| **SSRIs/SNRIs** | Paroxetine (FDA-approved), Venlafaxine, Escitalopram, Citalopram | Moderate (50-60% reduction) | Nausea, Sexual dysfunction, Withdrawal | https://www.menopause.org/nonhormonal |
| **Gabapentinoids** | Gabapentin, Pregabalin | Moderate | Dizziness, Somnolence, Edema | |
| **Clonidine** | Central alpha-2 agonist | Modest | Hypotension, Dry mouth | |
| **Fezolinetant (Veozah)** | **NK3 receptor antagonist** | Moderate (50-60% reduction) | **Liver enzyme elevation** (monitor LFTs) | |
| **CBT** | Cognitive restructuring, Paced breathing | Moderate | None | |
| **Hypnosis** | Clinical hypnosis | Moderate | None | |
