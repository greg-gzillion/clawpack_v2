# Gynecology - Cervical Disorders

## Cervical Cancer Screening Guidelines
| Organization | Age 21-29 | Age 30-65 | Age >65 | URL |
|--------------|-----------|-----------|---------|-----|
| **ACS (2020)** | Pap q3 years | **HPV primary q5 years** (preferred) OR HPV/Pap co-test q5 years OR Pap q3 years | Stop if adequate negative screening | https://www.cancer.org/cervical-cancer-screening |
| **USPSTF (2018)** | Pap q3 years | HPV/Pap co-test q5 years OR Pap q3 years OR HPV primary q5 years | Stop if adequate negative screening | https://www.uspreventiveservicestaskforce.org |
| **ASCCP (2019)** | Pap q3 years | HPV/Pap co-test q5 years OR Pap q3 years | Stop if adequate negative screening | https://www.asccp.org |
| **ACOG (2021)** | Pap q3 years | HPV/Pap co-test q5 years OR Pap q3 years | Stop if adequate negative screening | https://www.acog.org/cervical-cancer-screening |

## When to Stop Screening
| Criteria | Recommendation | URL |
|----------|----------------|-----|
| **Age >65** | Stop if **adequate negative screening** (=3 consecutive negative Paps or =2 negative HPV/co-tests in past 10 years, most recent within 5 years) AND no history of CIN2+ in past 25 years | https://www.asccp.org |
| **Post-Hysterectomy** | Stop if **total hysterectomy for benign disease** AND no history of CIN2+ | |
| **Supracervical Hysterectomy** | Continue screening per age guidelines | |
| **History of CIN2+** | Continue screening for at least 25 years (even if >65) | |
| **Immunocompromised** | Continue screening regardless of age | |

## Abnormal Cervical Cancer Screening - ASCCP Risk-Based Management (2019)
| Result | Management | URL |
|--------|------------|-----|
| **HPV Negative, Pap NILM** | Routine screening in 5 years | https://www.asccp.org/guidelines |
| **HPV Positive, Pap NILM** | **Option 1**: HPV genotyping (if HPV 16/18+ ? Colposcopy; if other 12 types ? Repeat co-test in 1 year) **Option 2**: Repeat co-test in 1 year | |
| **ASC-US, HPV Negative** | Routine screening in 3 years | |
| **ASC-US, HPV Positive** | **Colposcopy** | |
| **LSIL** | **Colposcopy** (immediate if HPV unknown or HPV+; If HPV- and age >30, Repeat co-test in 1 year) | |
| **ASC-H** | **Colposcopy** | |
| **HSIL** | **Colposcopy** (or expedited treatment - see below) | |
| **AGC** | **Colposcopy + Endometrial sampling** (if age =35 or risk factors) | |
| **AIS** | **Diagnostic excisional procedure** (CKC preferred) | |

## Colposcopy - Indications and Technique
| Component | Description | URL |
|-----------|-------------|-----|
| Indications | Abnormal cervical screening, Abnormal-appearing cervix, Postcoital bleeding, Persistent ASC-US/LSIL | https://www.asccp.org/colposcopy |
| Procedure | 3-5% acetic acid ? Identify acetowhite lesions ? Lugol's iodine (Schiller test) ? **Biopsy all visible lesions** | |
| Endocervical Curettage (ECC) | **Recommended for all non-pregnant patients** (except age <30 with LSIL); **Not performed in pregnancy** | |
| Findings | Acetowhite epithelium, Mosaicism, Punctation, Atypical vessels | |
| Inadequate Colposcopy | **Squamocolumnar junction not fully visualized** ? **Diagnostic excisional procedure** (if HSIL or AGC) | |

## Cervical Intraepithelial Neoplasia (CIN) - Management
| Diagnosis | Management | URL |
|----------|------------|-----|
| **CIN 1** | **Observation** (spontaneous regression in 60-80%); Repeat co-test in 1 year; If persists >2 years ? Treatment or continued observation acceptable | https://www.asccp.org |
| **CIN 2** | **Observation acceptable** for age <25 (high regression rate); **Treatment preferred** for age =25 (LEEP, Cryotherapy, Laser) | |
| **CIN 3** | **Treatment recommended** (LEEP, CKC, Cryotherapy, Laser) | |
| **AIS** | **Diagnostic excisional procedure** (CKC preferred) followed by hysterectomy (if completed childbearing) | |

## Expedited Treatment (See and Treat)
| Criteria | Recommendation | URL |
|----------|----------------|-----|
| **HSIL Cytology** | Expedited treatment acceptable (LEEP) without colposcopic biopsy if **age =25** (shared decision-making) | https://www.asccp.org |
| **Age <25** | Colposcopy with biopsy preferred (avoid overtreatment) | |
| **Pregnancy** | **Colposcopy only**; Biopsy if concern for cancer; Treatment deferred until postpartum | |

## Cervical Cancer - Staging (FIGO 2018)
| Stage | Description | Treatment | URL |
|-------|-------------|-----------|-----|
| **IA1** | Stromal invasion =3 mm depth, =7 mm width | **Cone biopsy** (fertility-sparing) OR Simple hysterectomy | https://www.nccn.org/cervical |
| **IA2** | Invasion >3-5 mm depth, =7 mm width | **Modified radical hysterectomy + PLND** OR Radical trachelectomy + PLND (fertility-sparing) | |
| **IB1** | Tumor =2 cm | **Radical hysterectomy + PLND** OR Radical trachelectomy + PLND (fertility-sparing) | |
| **IB2** | Tumor >2-4 cm | **Radical hysterectomy + PLND** ± Adjuvant therapy | |
| **IB3** | Tumor >4 cm | **Chemoradiation** (preferred) OR Neoadjuvant chemotherapy + Surgery | |
| **IIA-IVA** | Locally advanced | **Definitive chemoradiation** (Cisplatin + Radiation + Brachytherapy) | |
| **IVB** | Metastatic | **Palliative chemotherapy** ± Bevacizumab ± Pembrolizumab (PD-L1+) | |

## Cervical Polyps
| Component | Description | URL |
|-----------|-------------|-----|
| Presentation | Asymptomatic, Postcoital bleeding, Intermenstrual bleeding | https://www.acog.org/cervical-polyps |
| Diagnosis | Speculum exam (smooth, Red, Friable polyp protruding from cervical os) | |
| Management | **Polypectomy** (twist with ring forceps); Send for pathology (rule out malignancy) | |
| Pathology | **Benign** (>99%); Rarely malignant (adenocarcinoma, Sarcoma) | |
