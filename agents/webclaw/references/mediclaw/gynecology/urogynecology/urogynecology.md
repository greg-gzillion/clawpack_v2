# Gynecology - Urogynecology and Pelvic Floor Disorders

## Pelvic Organ Prolapse (POP) - Overview
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Descent of pelvic organs into or through the vagina | https://www.augs.org |
| Prevalence | 30-50% of women on exam; 3-6% symptomatic | |
| Risk Factors | **Vaginal delivery** (strongest), **Parity**, **Age**, **Obesity**, **Chronic constipation**, **Heavy lifting**, **Connective tissue disorders**, **Hysterectomy** | |
| Protective Factors | **Cesarean delivery**, **Estrogen** | |

## Pelvic Organ Prolapse - POP-Q Staging
| Stage | Definition | URL |
|-------|------------|-----|
| **Stage 0** | No prolapse | https://www.augs.org/POPQ |
| **Stage I** | Most distal portion >1 cm above hymen | |
| **Stage II** | Most distal portion =1 cm above to =1 cm below hymen | |
| **Stage III** | Most distal portion >1 cm below hymen but =2 cm less than total vaginal length | |
| **Stage IV** | Complete eversion (procidentia) | |

## Pelvic Organ Prolapse - Compartments
| Compartment | Organs | Symptoms | URL |
|-------------|--------|----------|-----|
| **Anterior** | Bladder (cystocele), Urethra (urethrocele) | Vaginal bulge, Urinary symptoms (incomplete emptying, Retention) | https://www.augs.org |
| **Apical** | Uterus (procidentia), Vaginal cuff (post-hysterectomy), Enterocele | Vaginal bulge, Pelvic pressure | |
| **Posterior** | Rectum (rectocele), Small bowel (enterocele) | Vaginal bulge, **Splinting** to defecate, Constipation | |

## Pelvic Organ Prolapse - Management
| Treatment | Indications | Description | URL |
|-----------|-------------|-------------|-----|
| **Observation** | Asymptomatic or mild symptoms | Reassurance, Pelvic floor muscle exercises | https://www.augs.org |
| **Pessary** | **First-line for symptomatic POP** | Ring, Gellhorn, Cube, Donut; Fitted in office; Removed/cleaned periodically | |
| **Pelvic Floor Physical Therapy** | Mild-moderate POP | Strengthen pelvic floor muscles | |
| **Surgery - Obliterative (Colpocleisis)** | **Advanced POP, No desire for future vaginal intercourse** | **High success (90-95%)**, Low morbidity | |
| **Surgery - Reconstructive** | Desire for vaginal function | Native tissue repair (anterior/posterior colporrhaphy, Uterosacral ligament suspension, Sacrospinous ligament fixation); **Mesh sacrocolpopexy** (apical prolapse) | |

## Urinary Incontinence - Types
| Type | Mechanism | Symptoms | URL |
|------|-----------|----------|-----|
| **Stress Incontinence (SUI)** | Urethral hypermobility, Intrinsic sphincter deficiency | Leakage with cough, Sneeze, Laugh, Exercise, Lifting | https://www.augs.org/incontinence |
| **Urgency Incontinence (UUI)** | Detrusor overactivity | Sudden strong urge, Frequency, Nocturia, "Key-in-the-door" syndrome | |
| **Mixed Incontinence** | Combination of SUI and UUI | Both symptoms | |
| **Overflow Incontinence** | Detrusor underactivity, Bladder outlet obstruction | Frequent small volumes, Incomplete emptying, Post-void dribbling | |

## Stress Urinary Incontinence - Evaluation
| Component | Description | URL |
|-----------|-------------|-----|
| History | Leakage triggers, Severity (pad usage), Impact on quality of life | https://www.augs.org/SUI |
| Physical Exam | **Cough stress test** (full bladder), **Q-tip test** (urethral hypermobility), POP-Q | |
| **Post-Void Residual (PVR)** | Bladder scan; Normal <50 mL; >150 mL = Abnormal | |
| Urinalysis | Rule out infection, Hematuria | |
| Urodynamics | Not routinely indicated; Consider for complex, Prior surgery, Neurologic disease, Elevated PVR | |

## Stress Urinary Incontinence - Management
| Treatment | Description | Efficacy | URL |
|-----------|-------------|----------|-----|
| **Pelvic Floor Muscle Exercises (Kegels)** | **First-line**; Supervised by pelvic floor PT | 50-75% improvement | https://www.augs.org |
| **Pessary** | Incontinence ring, Dish with knob | 50-60% satisfaction | |
| **Urethral Bulking Agents** | Injectable (Collagen, Silicone) | Modest, Temporary | |
| **Mid-Urethral Sling** | **Gold standard surgical treatment**; Retropubic (TVT) or Transobturator (TOT) | **80-90% cure rate** | |
| **Autologous Fascial Sling** | Alternative to synthetic mesh | 80-90% cure | |
| **Burch Colposuspension** | Open or laparoscopic | 70-85% cure | |

## Urgency Urinary Incontinence / Overactive Bladder (OAB) - Management
| Treatment | Description | URL |
|-----------|-------------|-----|
| **Behavioral** | **Bladder training**, Timed voiding, Fluid management, Avoid bladder irritants (caffeine, Alcohol) | https://www.auanet.org/OAB |
| **Pelvic Floor PT** | Urge suppression techniques | |
| **Medications** | **Mirabegron/Vibegron** (beta-3 agonists - preferred in elderly), **Antimuscarinics** (Oxybutynin - avoid in elderly, Tolterodine, Solifenacin, Darifenacin, Trospium) | |
| **PTNS** | Percutaneous tibial nerve stimulation (weekly x 12 weeks) | |
| **Sacral Neuromodulation** | InterStim device | |
| **Intradetrusor Botox** | OnabotulinumtoxinA injections | |

## Interstitial Cystitis / Bladder Pain Syndrome (IC/BPS)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Chronic bladder pain/pressure with urinary symptoms lasting >6 weeks without identifiable cause | https://www.ichelp.org |
| Presentation | Bladder pain (relieved by voiding), Urinary urgency/frequency, Dyspareunia | |
| Diagnosis | **Clinical** (rule out infection, Malignancy, Other causes); Cystoscopy with hydrodistention (Hunner's lesions in 10-20%) | |
| Management | **Multimodal**: Dietary modification (avoid triggers), Pelvic floor PT, Amitriptyline, Pentosan polysulfate (Elmiron), Hydroxyzine, Bladder instillations (DMSO, Heparin, Lidocaine), Cyclosporine | |
