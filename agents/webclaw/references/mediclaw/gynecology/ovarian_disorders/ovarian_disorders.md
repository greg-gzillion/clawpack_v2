## Ovarian Tumors - Classification
| Category | Examples | Malignant Potential | URL |
|----------|----------|---------------------|-----|
| **Epithelial Tumors** (60-70%) | Serous, Mucinous, Endometrioid, Clear cell, Brenner | Variable | https://www.sgo.org |
| **Germ Cell Tumors** (15-20%) | Mature teratoma (dermoid), Dysgerminoma, Yolk sac tumor, Choriocarcinoma | Variable | |
| **Sex Cord-Stromal Tumors** (5-10%) | Granulosa cell, Thecoma, Fibroma, Sertoli-Leydig cell | Usually low malignant potential | |
| **Metastatic Tumors** (5%) | Krukenberg (GI), Breast, Endometrial | High | |

## Benign Ovarian Tumors
| Tumor | Characteristics | Management | URL |
|-------|-----------------|------------|-----|
| **Functional Cyst** | Follicular, Corpus luteum; <10 cm; Unilocular | Expectant management; OCPs for suppression | https://www.acog.org/ovarian-masses |
| **Mature Cystic Teratoma (Dermoid)** | **Most common ovarian neoplasm in reproductive age**; Ectodermal derivatives (hair, Teeth, Sebum); Bilateral in 10-15% | Surgical removal (cystectomy); Risk of torsion | |
| **Serous Cystadenoma** | Unilocular, Thin-walled, Serous fluid | Monitor or remove if large/symptomatic | |
| **Mucinous Cystadenoma** | Multilocular, Thick mucinous fluid, Can be very large | Surgical removal (risk of rupture, Pseudomyxoma peritonei) | |
| **Endometrioma** | "Ground glass" appearance on US | Hormonal suppression; Cystectomy if symptomatic | |
| **Fibroma/Thecoma** | Solid, Hypoechoic | Remove if large/symptomatic; **Meigs syndrome** (ascites, Pleural effusion) | |
| **Struma Ovarii** | Monodermal teratoma (thyroid tissue) | Surgical removal; **Can cause hyperthyroidism** | |

## Ovarian Cancer - Epidemiology and Risk Factors
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | 5th leading cause of cancer death in women; Lifetime risk 1.3% | https://www.cancer.gov/types/ovarian |
| Median Age at Diagnosis | 63 years | |
| **Protective Factors** | **OCP use** (30-50% risk reduction with =5 years use), **Multiparity**, **Breastfeeding**, **Tubal ligation**, **Salpingectomy**, **Risk-reducing BSO** (BRCA mutation carriers) | |
| **Risk Factors** | **BRCA1/2 mutations** (lifetime risk 40-50% BRCA1, 20-30% BRCA2), **Lynch syndrome**, Family history, **Nulliparity**, **Endometriosis** (clear cell, Endometrioid subtypes), Postmenopausal hormone therapy | |

## Risk-Reducing Salpingo-Oophorectomy (RRSO)
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **BRCA1** | **Age 35-40** (after childbearing) | https://www.nccn.org/ovarian-cancer |
| **BRCA2** | **Age 40-45** (later onset) | |
| **Lynch Syndrome** | Consider at time of hysterectomy (age 40-45) | |
| **Benefits** | **80-90% reduction in ovarian cancer risk**; 50% reduction in breast cancer risk (premenopausal RRSO) | |
| **Risks** | Surgical menopause, Vasomotor symptoms, Osteoporosis, Cardiovascular disease | |
| **Hormone Therapy After RRSO** | **Safe and recommended** (unless personal history of breast cancer) | |

## Ovarian Cancer - Clinical Presentation
| Symptom | Frequency | URL |
|---------|-----------|-----|
| **Abdominal bloating/increased girth** | 70-80% | https://www.sgo.org/ovarian-cancer-symptoms |
| **Pelvic/abdominal pain** | 50-60% | |
| **Early satiety/Difficulty eating** | 40-50% | |
| **Urinary urgency/frequency** | 30-40% | |
| **Symptoms are frequent (>12 days/month) and recent onset (<1 year)** | | |
| **Ovarian Cancer Symptom Index** | Any of above symptoms >12 days/month for <1 year ? **Evaluate** | |

## Ovarian Cancer - Diagnosis and Evaluation
| Modality | Findings | URL |
|----------|----------|-----|
| **Pelvic Ultrasound** | Complex mass, Solid components, Papillary projections, Thick septations (>3 mm), Ascites | https://www.acog.org/ovarian-mass |
| **CA-125** | Elevated (>35 U/mL) in 80% of epithelial ovarian cancers; **Low sensitivity/specificity** (elevated in endometriosis, Fibroids, PID, Pregnancy) | |
| **CT Abdomen/Pelvis** | Staging; Evaluate for omental caking, Ascites, Lymphadenopathy, Metastases | |
| **Surgical Staging** | **Definitive diagnosis and treatment**; Total hysterectomy, BSO, Omentectomy, Lymphadenectomy, Peritoneal biopsies | |

## Ovarian Cancer - Treatment Overview
| Stage | Treatment | URL |
|-------|-----------|-----|
| **Stage I** | Surgical staging; Adjuvant chemotherapy for high-risk (Grade 3, Clear cell, Stage IC) | https://www.nccn.org/ovarian |
| **Stage II-IV** | **Primary debulking surgery** (optimal: <1 cm residual disease) followed by **Platinum-based chemotherapy** (Carboplatin + Paclitaxel); OR **Neoadjuvant chemotherapy** followed by interval debulking | |
| **Maintenance Therapy** | **PARP inhibitors** (Olaparib, Niraparib, Rucaparib) for BRCA-mutated or HRD-positive; **Bevacizumab** | |
| **Recurrent Disease** | Platinum-sensitive (>6 months): Re-treat with platinum doublet ± maintenance; Platinum-resistant: Non-platinum agents (Pegylated liposomal doxorubicin, Gemcitabine, Topotecan) | |
