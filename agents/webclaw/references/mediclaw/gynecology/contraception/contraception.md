## Intrauterine Devices (IUDs) - Comparison
| Feature | Levonorgestrel IUD | Copper IUD | URL |
|---------|---------------------|------------|-----|
| Brands | Mirena (52 mg), Kyleena (19.5 mg), Skyla (13.5 mg), Liletta (52 mg) | Paragard | https://www.acog.org/IUD |
| Duration | 3-8 years (depending on brand) | 10 years | |
| Mechanism | Thickens cervical mucus, Thins endometrium | Copper toxic to sperm | |
| Bleeding Pattern | **Lighter menses** (80-95% reduction); Amenorrhea in 20-50% | **Heavier menses** (may increase 50%) | |
| Non-Contraceptive Benefits | Treatment of HMB, Endometriosis, Endometrial protection | Emergency contraception (99% effective within 5 days) | |
| Insertion Pain Management | Lidocaine paracervical block, NSAIDs, Misoprostol (controversial) | Same | |

## Emergency Contraception (EC)
| Method | Timing | Effectiveness | Mechanism | URL |
|--------|--------|---------------|-----------|-----|
| **Copper IUD** | Within 5 days | **>99%** | Copper toxic to sperm, Prevents implantation | https://www.acog.org/emergency-contraception |
| **Ulipristal Acetate (ella)** | Within 5 days | 98% (does not decline over 5 days) | Progesterone receptor modulator (delays ovulation) | |
| **Levonorgestrel 1.5 mg** | Within 3 days | 89% (declines with time) | Delays ovulation | |
| **Yuzpe Regimen** | Within 3 days | 75% | Combined OCPs | |
| **BMI Considerations** | Levonorgestrel less effective if BMI =25; Ulipristal less effective if BMI =30-35; Copper IUD unaffected by BMI | | |
