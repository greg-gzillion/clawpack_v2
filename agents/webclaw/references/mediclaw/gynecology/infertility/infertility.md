# Gynecology - Infertility

## Infertility - Definitions and Epidemiology
| Term | Definition | URL |
|------|------------|-----|
| **Infertility** | Failure to achieve pregnancy after **12 months** of regular unprotected intercourse (or **6 months** if age =35) | https://www.asrm.org/infertility |
| **Primary Infertility** | No prior pregnancies | |
| **Secondary Infertility** | Prior pregnancy but unable to conceive again | |
| **Fecundability** | Probability of pregnancy per menstrual cycle (20-25% in healthy couples) | |
| **Prevalence** | 10-15% of couples | |

## Infertility - Etiology
| Cause | Frequency | Evaluation | URL |
|-------|-----------|------------|-----|
| **Ovulatory Dysfunction** | 25-30% | Menstrual history, Ovulation testing (OPK, BBT, Progesterone), Labs (TSH, Prolactin, FSH, AMH) | https://www.asrm.org/infertility-evaluation |
| **Male Factor** | 25-30% | **Semen analysis** (volume, Count, Motility, Morphology) | |
| **Tubal Factor** | 10-15% | **Hysterosalpingography (HSG)** (assess tubal patency, Uterine cavity) | |
| **Endometriosis** | 5-10% | Laparoscopy (gold standard) | |
| **Uterine Factor** | 5% | HSG, Saline infusion sonography (SIS), Hysteroscopy | |
| **Unexplained** | 15-20% | All testing normal | |

## Infertility - Basic Evaluation
| Test | Timing | Purpose | URL |
|------|--------|---------|-----|
| **Semen Analysis** | After 2-5 days abstinence | Assess male factor | https://www.asrm.org |
| **Ovulation Testing** | Mid-luteal (day 21 of 28-day cycle) | **Progesterone >3 ng/mL** confirms ovulation | |
| **Ovarian Reserve** | Cycle day 2-4 | **AMH** (any day), **FSH + Estradiol**, **Antral follicle count (AFC)** | |
| **HSG** | Cycle day 6-12 | Tubal patency, Uterine cavity | |
| **TSH, Prolactin** | Any day | Rule out thyroid disease, Hyperprolactinemia | |

## Ovarian Reserve Testing
| Test | Normal | Diminished Ovarian Reserve | URL |
|------|--------|---------------------------|-----|
| **AMH** | >1.0 ng/mL | <1.0 ng/mL | https://www.asrm.org/ovarian-reserve |
| **FSH (Cycle Day 2-4)** | <10 IU/L | >10 IU/L | |
| **Estradiol (Cycle Day 2-4)** | <80 pg/mL | >80 pg/mL (premature follicular recruitment) | |
| **Antral Follicle Count (AFC)** | >10 | <5-7 | |

## Ovulation Induction
| Medication | Mechanism | Indications | Monitoring | Risks | URL |
|------------|-----------|--------------|------------|-------|-----|
| **Letrozole** | **Aromatase inhibitor** (reduces estrogen ? Increased FSH) | **First-line for PCOS**, Unexplained | Ultrasound, Progesterone | **Lower multiple pregnancy rate** (5-10%), Ovarian cysts | https://www.asrm.org/ovulation-induction |
| **Clomiphene Citrate** | **SERM** (blocks estrogen receptor ? Increased FSH) | Anovulation, Unexplained | Ultrasound, Progesterone | Multiple pregnancy (8-10%), **Anti-estrogenic effects** (thin endometrium, Hostile mucus) | |
| **Gonadotropins** | FSH ± LH | Hypogonadotropic hypogonadism, PCOS (second-line), Controlled ovarian hyperstimulation (IVF) | **Close monitoring** (ultrasound, Estradiol) | **Multiple pregnancy (15-20%)**, **OHSS** | |
| **Metformin** | Insulin sensitizer | **PCOS** (adjunct) | None | GI side effects | |

## Tubal Factor Infertility
| Finding on HSG | Management | URL |
|----------------|------------|-----|
| **Proximal Tubal Occlusion** | **Hysteroscopic tubal cannulation**; IVF if unsuccessful | https://www.asrm.org/tubal-factor |
| **Distal Tubal Occlusion / Hydrosalpinx** | **Salpingectomy or proximal tubal occlusion** before IVF (improves IVF success); **Salpingostomy** (if mild disease and desire for spontaneous conception) | |
| **Tubal Ligation (Desiring Reversal)** | **Tubal anastomosis** (if age <40, Adequate tubal length, Normal semen analysis) OR **IVF** | |

## Unexplained Infertility - Management
| Treatment | Description | URL |
|-----------|-------------|-----|
| **Expectant Management** | Reasonable if age <35, Short duration of infertility | https://www.asrm.org/unexplained-infertility |
| **Ovulation Induction + IUI** | Letrozole or Clomiphene + Intrauterine insemination (3-4 cycles) | |
| **Gonadotropins + IUI** | Higher pregnancy rates but higher multiple pregnancy/OHSS risk | |
| **IVF** | **Most effective**; Proceed after 3-4 failed IUI cycles or if age >38 | |

## Assisted Reproductive Technology (ART)
| Procedure | Description | Indications | URL |
|-----------|-------------|-------------|-----|
| **Intrauterine Insemination (IUI)** | Washed sperm placed into uterine cavity | Mild male factor, Unexplained infertility, Cervical factor | https://www.sart.org |
| **In Vitro Fertilization (IVF)** | Oocyte retrieval, Fertilization in vitro, Embryo transfer | Tubal factor, Severe male factor, Endometriosis, Failed IUI, Genetic testing | |
| **Intracytoplasmic Sperm Injection (ICSI)** | Single sperm injected into oocyte | **Severe male factor**, Prior fertilization failure | |
| **Preimplantation Genetic Testing (PGT)** | Biopsy of embryo for genetic analysis | **Aneuploidy screening (PGT-A)**, **Monogenic disorders (PGT-M)**, **Chromosomal rearrangements (PGT-SR)** | |
| **Frozen Embryo Transfer (FET)** | Transfer of cryopreserved embryo | **Preferred over fresh transfer** (lower OHSS risk, Better endometrial receptivity) | |

## Ovarian Hyperstimulation Syndrome (OHSS)
| Severity | Features | Management | URL |
|----------|----------|------------|-----|
| **Mild** | Abdominal bloating, Mild pain, Ovaries <8 cm | **Outpatient**, Oral fluids, NSAIDs, Monitor | https://www.asrm.org/OHSS |
| **Moderate** | Nausea/Vomiting, Ascites on US, Ovaries 8-12 cm | Outpatient or inpatient; IV fluids, Monitor I/Os, Weight, Labs | |
| **Severe** | Tense ascites, Oliguria, Hemoconcentration (Hct >45%), Hypovolemia, Thromboembolism | **Inpatient**, **Aggressive IV fluids**, **Anticoagulation**, **Paracentesis** if respiratory compromise | |
| **Critical** | Anasarca, ARDS, Renal failure, Thromboembolism | **ICU** | |
| **Prevention** | **GnRH agonist trigger** (instead of hCG), **Freeze-all** (no fresh transfer), **Cabergoline** | | |
