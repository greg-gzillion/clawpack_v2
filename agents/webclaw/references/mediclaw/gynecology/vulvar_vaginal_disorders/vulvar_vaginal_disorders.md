## Vulvovaginal Candidiasis (VVC)
| Type | Presentation | Treatment | URL |
|------|--------------|-----------|-----|
| **Uncomplicated VVC** | Mild-moderate symptoms, <4 episodes/year, Immunocompetent, C. albicans | **Fluconazole 150 mg PO x 1**; Topical azoles (1-7 days) | https://www.cdc.gov/std/treatment-guidelines/candidiasis |
| **Complicated VVC** | Severe symptoms, =4 episodes/year, Non-albicans species, Immunocompromised, Uncontrolled diabetes, Pregnancy | **Fluconazole 150 mg q72h x 2-3 doses**; Topical azoles x 7-14 days | |
| **Recurrent VVC (RVVC)** | =4 episodes/year | **Fluconazole 150 mg weekly x 6 months** (maintenance) | |

## Atrophic Vaginitis (Genitourinary Syndrome of Menopause - GSM)
| Component | Description | URL |
|-----------|-------------|-----|
| Pathophysiology | Estrogen deficiency ? Thinning of vaginal epithelium, Loss of rugae, Decreased lubrication, Increased pH | https://www.menopause.org/gsm |
| Symptoms | Vaginal dryness, Dyspareunia, Irritation, Burning, Discharge, Urinary urgency, Recurrent UTI | |
| Diagnosis | Clinical (pH >5, Increased parabasal cells on microscopy) | |
| Treatment | **Vaginal estrogen** (cream, Tablet, Ring) - **first-line**; **Vaginal DHEA** (Prasterone); **Ospemifene** (oral SERM); **Lubricants/Moisturizers** | |
| Safety of Vaginal Estrogen | **Minimal systemic absorption**; Safe in breast cancer survivors (discuss with oncologist) | |

## Lichen Sclerosus
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | Bimodal (prepubertal girls, Postmenopausal women); Autoimmune association | https://www.aad.org/lichen-sclerosus |
| Presentation | Intense pruritus, Dyspareunia, White atrophic plaques ("parchment-like"), Figure-of-eight distribution (vulva and perianal), Fusion of labia, Phimosis of clitoris | |
| Diagnosis | **Clinical**; Biopsy if atypical or treatment-resistant | |
| Treatment | **Ultrapotent topical steroid** (Clobetasol 0.05% ointment daily x 4-12 weeks then maintenance 1-2x/week) | |
| Complications | **Scarring**, **Sexual dysfunction**, **Squamous cell carcinoma** (2-6% lifetime risk) | |
| Follow-up | **Annual surveillance** (even if asymptomatic) | |

## Lichen Planus
| Component | Description | URL |
|-----------|-------------|-----|
| Types | Erosive (most common genital), Papulosquamous, Hypertrophic | https://www.aad.org/lichen-planus |
| Presentation | Painful erosions, White reticulate papules (Wickham striae), Vaginal adhesions, Desquamative vaginitis | |
| Associations | Oral lichen planus (50-70%), Hepatitis C | |
| Treatment | **Ultrapotent topical steroids** (Clobetasol); **Tacrolimus** (second-line); Systemic steroids/immunosuppressants for severe disease | |
| Complications | Scarring, Vaginal stenosis, **Squamous cell carcinoma** (rare) | |

## Vulvodynia
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | **Vulvar pain of at least 3 months duration without clear identifiable cause** | https://www.nva.org |
| Types | **Localized provoked vestibulodynia** (most common - pain with touch/penetration), **Generalized** (spontaneous or provoked) | |
| Pathophysiology | **Neuropathic pain** (peripheral and central sensitization), Pelvic floor hypertonicity, Psychosocial factors | |
| Diagnosis | **Cotton-swab test** (map areas of tenderness); Rule out other causes (infections, Dermatoses) | |
| Treatment | **Multidisciplinary**: Pelvic floor physical therapy, **Cognitive behavioral therapy (CBT)**, Topical lidocaine, Oral medications (TCAs, SNRIs, Gabapentinoids), **Vestibulectomy** (for refractory localized vestibulodynia) | |

## Bartholin's Gland Cyst and Abscess
| Component | Description | URL |
|-----------|-------------|-----|
| Anatomy | Bartholin glands at 4 and 8 o'clock positions of vaginal introitus | https://www.acog.org/bartholin-cyst |
| Cyst | Asymptomatic swelling; **Observation** (if asymptomatic); Marsupialization or Word catheter (if symptomatic) | |
| Abscess | Painful, Erythematous, Fluctuant mass; **Incision and drainage + Word catheter placement** (prevents recurrence); Antibiotics only if surrounding cellulitis or high-risk (immunocompromised) | |
| Word Catheter | Insert into cavity, Inflate balloon, Leave in place 4-6 weeks (allows epithelialization) | |
| Marsupialization | Create permanent opening; For recurrent cysts/abscesses | |

## Vulvar Cancer
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | Rare (4% of gynecologic cancers); Mean age 65-70 | https://www.sgo.org/vulvar-cancer |
| Types | **Squamous cell carcinoma** (90%), Melanoma, Basal cell, Adenocarcinoma, Sarcoma | |
| Risk Factors | **HPV infection** (HPV 16/18 - younger patients), **Lichen sclerosus** (older patients), Smoking, Immunosuppression | |
| Presentation | Pruritus, Mass, Ulcer, Bleeding, Pain | |
| Diagnosis | **Biopsy** (punch biopsy of any suspicious lesion) | |
| Treatment | **Radical local excision + Inguinofemoral lymphadenectomy** (or SLNB); **Chemoradiation** for advanced disease | |

## Vaginal Cancer
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | Very rare (1-2% of gynecologic cancers) | https://www.sgo.org/vaginal-cancer |
| Types | **Squamous cell carcinoma** (80-90%), Adenocarcinoma (DES-related clear cell), Melanoma, Sarcoma | |
| Risk Factors | **HPV infection**, **DES exposure in utero** (clear cell adenocarcinoma), Prior cervical cancer, Chronic irritation (pessary) | |
| Presentation | Vaginal bleeding, Discharge, Mass, Pain, Urinary symptoms | |
| Diagnosis | **Biopsy** (colposcopy-directed) | |
| Treatment | **Radiation therapy** (primary); Surgery for early-stage or radiation failure | |
