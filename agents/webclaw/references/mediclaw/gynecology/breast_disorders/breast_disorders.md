## Common Benign Breast Conditions
| Condition | Presentation | Management | URL |
|-----------|--------------|------------|-----|
| **Fibrocystic Changes** | Bilateral nodularity, Cyclic pain | Reassurance, Supportive bra, NSAIDs, OCPs | https://www.acog.org/benign-breast-conditions |
| **Fibroadenoma** | **Most common breast tumor in young women**; Firm, Rubbery, Mobile, Well-circumscribed | Observation (if <2-3 cm, Typical features); Excision if large, Growing, Atypical features | |
| **Phyllodes Tumor** | Rapidly growing, Large, Firm mass | **Surgical excision with wide margins** (can be malignant) | |
| **Simple Cyst** | Smooth, Mobile, Tender, Aspirate fluid | Reassurance; Aspiration if symptomatic | |
| **Complex Cyst** | Thick walls, Septations, Solid components | **Biopsy** (rule out malignancy) | |
| **Intraductal Papilloma** | **Bloody nipple discharge**, Subareolar mass | **Microdochectomy** (excision of involved duct) | |
| **Fat Necrosis** | History of trauma/surgery, Firm, Irregular mass | Core needle biopsy (can mimic cancer) | |
| **Mastitis** | Erythema, Warmth, Tenderness, Fever | **Antibiotics** (Dicloxacillin, Cephalexin); Continue breastfeeding; Drain abscess if present | |

|------|--------|------------|-----|
