# Gynecology - Uterine Fibroids (Leiomyomas)

## Uterine Fibroids - Overview
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Benign monoclonal smooth muscle tumors of the myometrium | https://www.acog.org/fibroids |
| Prevalence | 20-50% of reproductive-age women; 70-80% by age 50; **Most common pelvic tumor in women** | |
| Risk Factors | **African American race** (3-4x increased risk), Age (peak 30-40s), Early menarche, Obesity, Nulliparity, Family history | |
| Protective Factors | Multiparity, Late menarche, Smoking, OCP use | |
| Genetics | **MED12 mutations** (70%), HMGA2 rearrangements, FH deficiency | |

## Fibroids - Clinical Presentation
| Symptom | Frequency | Mechanism | URL |
|---------|-----------|-----------|-----|
| **Heavy Menstrual Bleeding** | 30-50% | Increased endometrial surface area, Impaired myometrial contraction, Venous congestion | https://www.acog.org/fibroids |
| **Pelvic Pressure/Pain** | 20-30% | Mass effect, Degeneration, Torsion (pedunculated) | |
| **Anemia** | Common | Chronic blood loss | |
| **Urinary Symptoms** | Frequency, Urgency, Retention | Compression of bladder/urethra | |
| **Constipation** | Less common | Compression of rectosigmoid | |
| **Infertility/Recurrent Pregnancy Loss** | Variable | Distorted cavity, Impaired implantation, Tubal obstruction | |
| **Asymptomatic** | 50-60% | | |

## Fibroids - Diagnostic Evaluation
| Modality | Findings | Utility | URL |
|----------|----------|---------|-----|
| **Transvaginal Ultrasound** | Hypoechoic well-circumscribed masses, Acoustic shadowing | **First-line imaging** | https://www.acog.org/fibroids |
| **Saline Infusion Sonohysterography (SIS)** | Delineates submucosal component | **Best for intracavitary lesions** | |
| **Hysteroscopy** | Direct visualization of cavity | Diagnostic and therapeutic | |
| **MRI** | Precise mapping of size, Number, Location | **Pre-procedural planning** (UAE, MRgFUS, Myomectomy) | |
| **Endometrial Biopsy** | Rule out hyperplasia/malignancy | If AUB and risk factors | |

## Fibroids - Medical Management
| Treatment | Mechanism | Indications | Side Effects | URL |
|-----------|-----------|--------------|--------------|-----|
| **LNg-IUD** | Endometrial atrophy | HMB (if cavity not distorted) | Irregular spotting | https://www.acog.org/fibroids |
| **Tranexamic Acid** | Antifibrinolytic | HMB (cyclic) | VTE risk (low) | |
| **NSAIDs** | Prostaglandin inhibition | Dysmenorrhea, HMB | GI, Renal | |
| **Combined OCPs** | Endometrial atrophy | HMB, Dysmenorrhea | VTE risk | |
| **Oral Progestins** | Endometrial atrophy | HMB | Breakthrough bleeding | |
| **GnRH Agonists** (Leuprolide) | Hypoestrogenism (shrink fibroids 30-50%) | **Pre-operative** (3-6 months) | Vasomotor symptoms, **Bone loss** (short-term only) | |
| **GnRH Antagonists** (Elagolix, Relugolix, Linzagolix) | Dose-dependent estrogen suppression | **Pre-operative or long-term** (with add-back) | Hot flashes, Bone loss (dose-dependent) | |
| **Selective Progesterone Receptor Modulators** (Ulipristal - EU only) | Progesterone antagonist | HMB, Fibroid volume reduction | **Liver injury** (rare) | |

## Fibroids - Surgical and Procedural Management
| Procedure | Description | Indications | Considerations | URL |
|-----------|-------------|-------------|----------------|-----|
| **Myomectomy (Hysteroscopic)** | Resection of submucosal fibroids | Type 0, 1, 2 fibroids, Infertility | Preserves fertility; Risk of fluid overload | https://www.aagl.org |
| **Myomectomy (Abdominal/Laparoscopic/Robotic)** | Enucleation of fibroids | Intramural/subserosal fibroids, Infertility, Symptomatic | Preserves fertility; Risk of adhesions, Recurrence (15-30% at 5 years) | |
| **Hysterectomy** | Removal of uterus | **Definitive treatment**, Completed childbearing, Severe symptoms | Most common indication for hysterectomy | |
| **Uterine Artery Embolization (UAE)** | Occlusion of uterine arteries ? Fibroid infarction | Symptomatic fibroids, Completed childbearing, Avoid surgery | **Post-embolization syndrome** (pain, Fever); 15-30% require reintervention; **Pregnancy possible but higher complication rate** | https://www.sirweb.org |
| **Radiofrequency Ablation (Acessa, Sonata)** | Thermal ablation of fibroids | Symptomatic fibroids, Fertility preservation | Outpatient; Less data on fertility outcomes | |
| **MRI-Guided Focused Ultrasound (MRgFUS)** | Thermal ablation via ultrasound | Symptomatic fibroids, Avoid invasive procedure | Limited by fibroid location, Size; Not widely available | |

## Fibroids and Pregnancy
| Consideration | Management | URL |
|---------------|------------|-----|
| **Prevalence in Pregnancy** | 10-30% (most asymptomatic or minimally symptomatic) | https://www.acog.org/pregnancy-fibroids |
| **Complications** | **Pain** (red degeneration - 5-10%), **Preterm labor**, **Malpresentation**, **Placental abruption**, **Postpartum hemorrhage**, **Cesarean delivery** | |
| **Red Degeneration** | Acute pain due to hemorrhagic infarction (usually second trimester); **Conservative management** (NSAIDs, Opioids, Rest) | |
| **Myomectomy During Pregnancy** | **Contraindicated** (except pedunculated fibroid with torsion) | |
| **Myomectomy at Cesarean** | **Generally avoided** (increased hemorrhage risk); May be considered for pedunculated fibroids | |
