# Gynecology - Pediatric and Adolescent Gynecology

## Pubertal Development - Tanner Staging
| Stage | Breast (Female) | Pubic Hair (Female) | Age Range | URL |
|-------|-----------------|---------------------|-----------|-----|
| **1** | Prepubertal (no glandular tissue) | No pubic hair | <9 years | https://www.acog.org/adolescent-gynecology |
| **2** | **Breast bud** (thelarche) | Sparse, Long, Slightly pigmented | 9-11 years | |
| **3** | Enlargement of breast and areola | Darker, Coarser, Curling | 11-13 years | |
| **4** | Areola forms secondary mound | Adult-type but limited distribution | 12-14 years | |
| **5** | **Mature breast** (areola recesses) | Adult distribution (medial thighs) | 13-16 years | |
| **Menarche** | Occurs ~2-3 years after thelarche | Average age 12.5 years (range 10-15) | | |

## Precocious Puberty
| Type | Definition | Causes | Evaluation | Treatment | URL |
|------|------------|--------|------------|-----------|-----|
| **Central (GnRH-dependent)** | Breast development <8 years, Pubic hair <8 years | **Idiopathic** (80-90% of girls), CNS lesions (hamartoma, Tumor, Trauma), Genetic mutations | **LH, FSH, Estradiol**, **Bone age**, **Pelvic US**, **Brain MRI** | **GnRH agonist** (Leuprolide) | https://www.endocrine.org/precocious-puberty |
| **Peripheral (GnRH-independent)** | Signs of puberty without GnRH activation | **McCune-Albright syndrome** (cafķ-au-lait spots, Polyostotic fibrous dysplasia), **Ovarian cyst/tumor**, **Adrenal tumor**, **Exogenous estrogen** | LH/FSH (low), Estradiol (high), Pelvic US, Bone age | Treat underlying cause; **Aromatase inhibitors**, **Tamoxifen** | |

## Delayed Puberty
| Definition | Causes | Evaluation | Treatment | URL |
|------------|--------|------------|-----------|-----|
| **No breast development by age 13** | **Constitutional delay** (most common), **Hypogonadotropic hypogonadism** (Kallmann syndrome, Eating disorder, Chronic illness), **Hypergonadotropic hypogonadism** (Turner syndrome, POI, Chemo/radiation) | **FSH, LH, Estradiol**, **TSH, Prolactin**, **Bone age**, **Karyotype**, **Pelvic US**, **Brain MRI** | **Constitutional**: Reassurance or short course estrogen; **Hypogonadism**: Estrogen replacement (gradual titration) | https://www.acog.org/delayed-puberty |
| **No menses by age 15** (or 3 years after thelarche) | Same as above + **Outflow tract obstruction** (imperforate hymen, Transverse vaginal septum, M³llerian agenesis), **Androgen insensitivity** | Add pelvic US, MRI if indicated | Treat underlying cause | |

## Adolescent Heavy Menstrual Bleeding
| Component | Description | URL |
|-----------|-------------|-----|
| Common Causes | **Anovulation** (immature HPO axis - most common), **Bleeding disorders** (von Willebrand disease - 10-20%, Platelet disorders, Factor deficiencies), **PCOS**, **Thyroid disease** | https://www.acog.org/adolescent-HMB |
| Evaluation | **CBC**, **Coagulation studies** (von Willebrand panel), **TSH**, **Prolactin**, **Pelvic US** (if indicated) | |
| Management (Acute) | **High-dose OCPs** (monophasic 30-35 mcg ethinyl estradiol: 1 pill TID x 7 days), **Tranexamic acid**, **IV estrogen** (Premarin 25 mg IV q4-6h) | |
| Management (Chronic) | **Combined OCPs**, **LNg-IUD**, **Tranexamic acid**, **Iron supplementation** | |

## Ovarian Masses in Adolescents
| Type | Frequency | Management | URL |
|------|-----------|------------|-----|
| **Functional Cyst** | Most common | Expectant management | https://www.acog.org/adolescent-ovarian-mass |
| **Mature Teratoma (Dermoid)** | Most common neoplasm | Ovarian cystectomy (fertility-sparing) | |
| **Germ Cell Tumors** | Dysgerminoma, Yolk sac, Immature teratoma | **Fertility-sparing surgery** + Chemotherapy | |
| **Sex Cord-Stromal Tumors** | Granulosa cell, Sertoli-Leydig | Fertility-sparing surgery | |
| **Torsion** | **Surgical emergency** | **Detorsion** (preserve ovary) | |

## M³llerian Anomalies
| Anomaly | Description | Presentation | Management | URL |
|---------|-------------|--------------|------------|-----|
| **M³llerian Agenesis (MRKH)** | Absent uterus and upper vagina; Normal ovaries | **Primary amenorrhea**, Normal breast development, **46,XX** | **Vaginal dilators** (first-line), Vaginoplasty; Psychological support | https://www.acog.org/mullerian-anomalies |
| **Imperforate Hymen** | Failure of hymenal canalization | Cyclic pelvic pain, **Primary amenorrhea**, Bulging bluish membrane | **Hymenectomy** | |
| **Transverse Vaginal Septum** | Incomplete canalization | Similar to imperforate hymen | **Septum resection** | |
| **Unicornuate Uterus** | Single horn | **Pregnancy complications** (preterm, Malpresentation) | Monitor pregnancy | |
| **Didelphys Uterus** | Two separate uteri, Cervices, ▒ Longitudinal vaginal septum | **Dysmenorrhea** (if obstructed hemivagina), Pregnancy complications | Resection of vaginal septum if symptomatic | |
| **Bicornuate Uterus** | Heart-shaped uterus | Pregnancy complications | Usually no intervention | |
| **Septate Uterus** | **Most common**; Fibrous septum | **Recurrent pregnancy loss** | **Hysteroscopic septum resection** | |

## Labial Adhesions
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Fusion of labia minora in prepubertal girls | https://www.acog.org/labial-adhesions |
| Etiology | Hypoestrogenism + Irritation (diaper dermatitis, Poor hygiene) | |
| Presentation | Asymptomatic (most), Urinary symptoms (post-void dribbling, UTI), Vulvar irritation | |
| Management | **Expectant** (most resolve spontaneously at puberty); **Topical estrogen cream** (0.01% estradiol BID x 2-6 weeks); **Topical betamethasone** (alternative); **Surgical separation** (if urinary obstruction or failed medical) | |
