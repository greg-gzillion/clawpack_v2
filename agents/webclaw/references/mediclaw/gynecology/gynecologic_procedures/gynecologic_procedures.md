# Gynecology - Gynecologic Procedures

## Endometrial Biopsy (EMB)
| Component | Description | URL |
|-----------|-------------|-----|
| Indications | **Abnormal uterine bleeding** (especially age =45 or risk factors), **Endometrial surveillance** (PCOS, Tamoxifen), **Infertility evaluation** | https://www.acog.org/endometrial-biopsy |
| Contraindications | Pregnancy, Active pelvic infection, Cervical cancer | |
| Technique | **Pipelle** (preferred - less painful, Adequate sample); Tenaculum on anterior lip of cervix; Sound uterus; Insert Pipelle to fundus; Withdraw plunger while rotating | |
| Complications | Cramping, Bleeding, Infection, **Uterine perforation** (rare) | |
| Adequacy | **Insufficient sample in 10-20%** ? Requires D&C/hysteroscopy | |

## LEEP (Loop Electrosurgical Excision Procedure)
| Component | Description | URL |
|-----------|-------------|-----|
| Indications | **CIN 2/3** (treatment), **Discordant cytology/colposcopy**, **AIS** (diagnostic CKC preferred) | https://www.asccp.org/LEEP |
| Technique | Colposcopy to identify lesion; Local anesthesia (lidocaine with epinephrine); Select appropriate loop size; Excise lesion with 3-5 mm margin; Ball electrode for hemostasis | |
| Complications | Bleeding (immediate or delayed 7-10 days), Infection, Cervical stenosis, **Preterm birth** (increased risk with larger/deeper excisions) | |
| Follow-Up | **Co-test at 6 months** | |

## Cold Knife Cone (CKC)
| Component | Description | URL |
|-----------|-------------|-----|
| Indications | **AIS** (preferred), **Microinvasive cervical cancer** (fertility-sparing), **Inadequate LEEP**, **Suspected invasive cancer** | https://www.sgo.org/CKC |
| Advantages | **Clear margins**, No thermal artifact | |
| Disadvantages | **Higher complication rate** (bleeding, Cervical stenosis, Preterm birth) | |

## Hysteroscopy
| Component | Description | URL |
|-----------|-------------|-----|
| Indications | **Abnormal uterine bleeding**, **Endometrial polyp/fibroid**, **Retained IUD**, **Infertility**, **Recurrent pregnancy loss**, **Asherman syndrome** | https://www.aagl.org/hysteroscopy |
| Diagnostic Hysteroscopy | Office-based; Smaller scope; No anesthesia or local | |
| Operative Hysteroscopy | OR-based; Larger scope; Anesthesia; Resection of polyps, Fibroids, Adhesions, Septum | |
| Complications | **Fluid overload** (monitor fluid deficit), Uterine perforation, Bleeding, Infection | |
| Distention Media | **Normal saline** (monopolar), **Glycine/Sorbitol** (bipolar) | |

## Dilation and Curettage (D&C)
| Component | Description | URL |
|-----------|-------------|-----|
| Indications | **Early pregnancy loss**, **Retained products of conception**, **Abnormal uterine bleeding** (when EMB inadequate), **Endometrial dating** | https://www.acog.org/DandC |
| Technique | Paracervical block or general anesthesia; Dilate cervix with Hegar dilators; **Sharp curettage** or **Suction curettage** | |
| Complications | Uterine perforation, Bleeding, Infection, **Asherman syndrome** (intrauterine adhesions) | |

## Hysterectomy - Routes
| Route | Indications | Advantages | Disadvantages | URL |
|-------|-------------|------------|---------------|-----|
| **Vaginal** | **Uterovaginal prolapse**, Small uterus, Benign disease | **Shortest recovery**, **Least complications**, **No visible scars** | Limited visualization, Cannot address adnexal pathology | https://www.acog.org/hysterectomy |
| **Laparoscopic** | Benign disease, Endometriosis, Adnexal pathology | Faster recovery than abdominal, Better visualization | Longer OR time, Requires expertise | |
| **Robotic** | Similar to laparoscopic | 3D visualization, Wristed instruments | **Higher cost**, Longer OR time | |
| **Abdominal** | **Large uterus**, **Suspected malignancy**, **Extensive adhesions**, **Inability to perform minimally invasive** | **Best visualization**, **Can address any pathology** | **Longest recovery**, **Highest complication rate** | |

## Hysterectomy - Types
| Type | Definition | Indications | URL |
|------|------------|-------------|-----|
| **Total Hysterectomy** | Removal of uterus and cervix | Most common | https://www.acog.org/hysterectomy |
| **Supracervical Hysterectomy** | Removal of uterine corpus, Leave cervix | Benign disease, Normal Pap/HPV | |
| **Radical Hysterectomy** | Removal of uterus, Cervix, Parametria, Upper vagina | **Cervical cancer** | |
| **BSO** | Bilateral salpingo-oophorectomy | **Ovarian cancer risk reduction**, **Postmenopausal**, **Endometriosis** (definitive) | |
| **Salpingectomy** | Removal of fallopian tubes | **Ovarian cancer prevention** (recommended at time of hysterectomy or sterilization) | |
