# Gynecology - Early Pregnancy

## Early Pregnancy Loss (Miscarriage)
| Type | Definition | Management | URL |
|------|------------|------------|-----|
| **Threatened Abortion** | Vaginal bleeding, Closed cervix, IUP on US | **Expectant management**; RhoGAM if Rh negative | https://www.acog.org/early-pregnancy-loss |
| **Inevitable Abortion** | Vaginal bleeding, Open cervix | Expectant, Medical (Misoprostol), or Surgical (D&C) | |
| **Incomplete Abortion** | Partial passage of POC, Open cervix | **Misoprostol** or **D&C** | |
| **Complete Abortion** | Complete passage of POC, Closed cervix | **Expectant** (confirm with US) | |
| **Missed Abortion** | Non-viable IUP, Closed cervix | Expectant, Medical (Misoprostol), or Surgical (D&C) | |
| **Septic Abortion** | Fever, Uterine tenderness, Purulent discharge | **IV antibiotics**, **Emergent D&C** | |
| **Recurrent Pregnancy Loss** | =2 consecutive pregnancy losses | **Comprehensive evaluation** (karyotype, Uterine cavity, APS, Endocrine) | |

## Early Pregnancy Loss - Management Options
| Option | Description | Success Rate | Risks | URL |
|--------|-------------|--------------|-------|-----|
| **Expectant Management** | Await spontaneous passage | 50-80% (varies by type) | Prolonged bleeding, Incomplete passage, Emotional distress | https://www.acog.org/early-pregnancy-loss |
| **Medical Management** | **Misoprostol 800 mcg vaginally** (single dose; repeat if needed) | 70-90% | Pain, Bleeding, Incomplete passage (need for D&C) | |
| **Surgical Management** | **D&C** (suction curettage) | >95% | Anesthesia risks, Uterine perforation, **Asherman syndrome** (rare) | |

## Ectopic Pregnancy
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Implantation of embryo outside the endometrial cavity | https://www.acog.org/ectopic-pregnancy |
| Locations | **Fallopian tube** (95%), Interstitial/Cornual (2-3%), Cesarean scar, Cervical, Ovarian, Abdominal | |
| Risk Factors | **Prior ectopic**, **PID**, **Tubal surgery**, **IUD**, **Smoking**, **ART**, **Prior tubal ligation** | |
| Presentation | **Abdominal pain** (90%), **Vaginal bleeding** (50-80%), **Amenorrhea**, Adnexal tenderness, Shoulder pain (hemoperitoneum) | |
| Diagnosis | **Positive hCG** + **Empty uterus on TVUS** + **Adnexal mass** (or free fluid); **Discriminatory zone**: hCG >1500-2000 ? IUP should be visible | |
| **Pregnancy of Unknown Location (PUL)** | Positive hCG, No IUP or ectopic on US ? **Serial hCG** (48 hours); Abnormal rise (<53% increase) or plateau ? Suspect ectopic | |

## Ectopic Pregnancy - Management
| Treatment | Criteria | Success Rate | Follow-Up | URL |
|-----------|----------|--------------|-----------|-----|
| **Expectant Management** | Asymptomatic, hCG <200 and declining | Variable | Serial hCG until <5 | https://www.acog.org/ectopic-pregnancy |
| **Methotrexate (Single-Dose)** | **Hemodynamically stable**, **hCG <5000**, **No fetal cardiac activity**, **Ectopic <3.5 cm**, **No contraindications**, **Able to follow up** | 80-90% (single dose); 90-95% (multi-dose) | Serial hCG days 4 and 7 (=15% decline between days 4-7 = success); Weekly until <5 | |
| **Methotrexate (Two-Dose)** | hCG 5000-10,000, Larger size | Higher success | Same | |
| **Surgery (Salpingostomy)** | Desire for future fertility, Unruptured | 85-90% tubal preservation | Serial hCG (risk of persistent trophoblast 5-15%) | |
| **Surgery (Salpingectomy)** | **Hemodynamically unstable**, **Ruptured**, Completed childbearing, Abnormal contralateral tube | >95% | Usually no follow-up needed | |

## Recurrent Pregnancy Loss (RPL)
| Component | Evaluation | Management | URL |
|-----------|------------|------------|-----|
| Definition | =2 consecutive pregnancy losses | | https://www.asrm.org/RPL |
| **Genetic** | **Parental karyotype** (3-5% have balanced translocation) | **PGT-SR** (IVF with preimplantation genetic testing) | |
| **Anatomic** | **HSG**, **SIS**, **Hysteroscopy** | Surgical correction (septum resection, Myomectomy, Adhesiolysis) | |
| **Antiphospholipid Syndrome (APS)** | **Lupus anticoagulant**, **Anticardiolipin antibodies**, **Beta-2 glycoprotein I antibodies** (positive x2, 12 weeks apart) | **Lovenox + Low-dose aspirin** during pregnancy | |
| **Endocrine** | TSH, Prolactin, HbA1c, PCOS | Treat underlying disorder | |
| **Unexplained** | 50% | Reassurance; Prognosis good (70-80% subsequent live birth rate) | |
