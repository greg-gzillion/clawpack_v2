# Gynecology - Sexual Health

## Female Sexual Dysfunction - Classification
| Disorder | Definition | Prevalence | URL |
|----------|------------|------------|-----|
| **Hypoactive Sexual Desire Disorder (HSDD)** | Persistent lack of sexual interest/desire causing distress | 10-15% | https://www.issm.info |
| **Female Sexual Interest/Arousal Disorder (FSIAD)** | Lack of or significantly reduced sexual interest/arousal | 10-20% | |
| **Female Orgasmic Disorder** | Marked delay, Infrequency, or absence of orgasm | 10-15% | |
| **Genito-Pelvic Pain/Penetration Disorder** | Pain with penetration, Fear/anxiety, Pelvic floor tension | 10-15% | |

## Hypoactive Sexual Desire Disorder (HSDD) - Management
| Treatment | Mechanism | Indications | Considerations | URL |
|-----------|-----------|--------------|----------------|-----|
| **Flibanserin (Addyi)** | 5-HT1A agonist / 5-HT2A antagonist | **Premenopausal HSDD** | **Daily dosing**, **Alcohol contraindication**, Hypotension, Syncope | https://www.fda.gov/flibanserin |
| **Bremelanotide (Vyleesi)** | Melanocortin receptor agonist | **Premenopausal HSDD** | **PRN dosing** (45 min before sexual activity), Nausea (40%), Flushing, **No alcohol restriction** | https://www.fda.gov/bremelanotide |
| **Testosterone** | Androgen receptor | **Postmenopausal HSDD** (off-label) | **Not FDA-approved for women**; Monitor for hirsutism, Acne, Lipids | |
| **Bupropion** | Dopamine/Norepinephrine reuptake inhibitor | Off-label | Activating; Avoid in seizure disorder | |
| **Cognitive Behavioral Therapy (CBT)** | Address negative cognitions, Mindfulness | All | **First-line or adjunct** | |

## Genito-Pelvic Pain/Penetration Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| Causes | **Vulvodynia**, **Vaginismus**, **GSM**, **Endometriosis**, **Pelvic floor hypertonicity**, **Vaginal dryness**, **Infections**, **Dermatoses** | https://www.acog.org/pelvic-pain |
| Evaluation | Comprehensive history, Pelvic exam (map pain), Q-tip test, Pelvic floor assessment, Rule out underlying pathology | |
| Management | **Multidisciplinary**: **Pelvic floor physical therapy** (first-line), **Vaginal dilators**, **CBT**, **Topical lidocaine**, **Vaginal estrogen/DHEA**, **Treat underlying condition** | |

## LGBTQ+ Health in Gynecology
| Population | Considerations | URL |
|------------|----------------|-----|
| **Lesbian/Bisexual Women** | **Still need cervical cancer screening** (HPV can be transmitted between women), STI screening based on behaviors, Contraception may be needed for menstrual regulation or pregnancy prevention | https://www.lgbthealtheducation.org |
| **Transgender Men (FTM)** | **Cervical cancer screening** if cervix present (may be challenging due to testosterone-induced atrophy, Dysphoria); **Pregnancy possible** if uterus present and not on contraception; **Testosterone is teratogenic** | |
| **Transgender Women (MTF)** | **Neovagina** (penile inversion) does not require cervical screening; **Prostate cancer screening** per guidelines; **Breast cancer screening** per guidelines (estrogen use) | |

## Intimate Partner Violence (IPV)
| Component | Description | URL |
|-----------|-------------|-----|
| Screening | **Universal screening** recommended (all women of reproductive age) | https://www.acog.org/IPV |
| Questions | "Do you feel safe at home?" "Has your partner ever hurt or threatened you?" "Has your partner ever forced you to have sex?" | |
| Reproductive Coercion | Sabotaging contraception, Forcing pregnancy/abortion | |
| Management | **Validate**, **Assess safety**, **Provide resources** (hotline, Shelter, Counseling), **Document** (objective findings, Patient statements), **Mandatory reporting** (varies by state) | |
| National Hotline | **1-800-799-SAFE (7233)** | https://www.thehotline.org | |

## Female Genital Mutilation/Cutting (FGM/C)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | All procedures involving partial or total removal of external female genitalia for non-medical reasons | https://www.who.int/fgm |
| Types | Type 1 (Clitoridectomy), Type 2 (Excision), Type 3 (Infibulation), Type 4 (Other) | |
| Complications | Pain, Infection, Bleeding, Urinary retention, Dyspareunia, Childbirth complications, Psychological trauma | |
| Management | **Culturally sensitive care**, **Defibulation** (for Type 3 - before pregnancy or during labor), **Clitoral reconstruction** (controversial) | |
