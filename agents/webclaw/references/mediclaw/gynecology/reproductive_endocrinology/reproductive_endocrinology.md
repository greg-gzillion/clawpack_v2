# Gynecology - Reproductive Endocrinology

## Polycystic Ovary Syndrome (PCOS)
| Component | Description | URL |
|-----------|-------------|-----|
| Rotterdam Criteria (2 of 3) | **1. Oligo/anovulation**, **2. Hyperandrogenism** (clinical or biochemical), **3. Polycystic ovaries on ultrasound** (=20 follicles per ovary or ovarian volume =10 mL) | https://www.asrm.org/PCOS |
| Prevalence | 6-12% of reproductive-age women | |
| Pathophysiology | Insulin resistance ? Hyperinsulinemia ? Increased ovarian androgen production; Altered GnRH pulsatility ? Increased LH:FSH ratio | |
| Clinical Features | Irregular menses, Hirsutism, Acne, Androgenic alopecia, Obesity, Infertility, Acanthosis nigricans | |
| Metabolic Risks | **Impaired glucose tolerance (30-40%)**, **Type 2 diabetes (10-20%)**, Dyslipidemia, Hypertension, Obstructive sleep apnea, **Endometrial hyperplasia/cancer** | |

## PCOS - Diagnostic Evaluation
| Test | Finding | URL |
|------|---------|-----|
| **Total Testosterone** | Mildly elevated or high-normal | https://www.endocrine.org/PCOS |
| **Free Testosterone** | Elevated (more sensitive) | |
| **DHEA-S** | Elevated in 25-50% | |
| **LH:FSH Ratio** | >2:1 (not diagnostic) | |
| **17-OHP** | Rule out non-classic CAH (<200 ng/dL normal; 200-1000 ? ACTH stimulation) | |
| **TSH, Prolactin** | Rule out other causes of anovulation | |
| **2-hour OGTT** | **Screen for IGT/DM** (every 1-3 years) | |
| **Fasting Lipid Panel** | Screen for dyslipidemia | |

## PCOS - Management
| Goal | Treatment | URL |
|------|-----------|-----|
| **Menstrual Regulation / Endometrial Protection** | **Combined OCPs** (first-line), Cyclic progestins, LNg-IUD | https://www.acog.org/PCOS |
| **Hirsutism/Acne** | **Combined OCPs** (first-line), **Spironolactone** 50-100 mg BID, **Eflornithine cream** (topical), Laser hair removal | |
| **Infertility** | **Letrozole** (first-line), **Clomiphene citrate**, **Metformin** (adjunct), Gonadotropins, **IVF** | |
| **Metabolic / Weight Management** | **Lifestyle modification** (diet, Exercise - 5-10% weight loss restores ovulation in 50-75%), **Metformin** (modest benefit), **GLP-1 agonists** (weight loss) | |
| **Prevention of Complications** | Screen for IGT/DM, Dyslipidemia, OSA; Consider endometrial biopsy for prolonged amenorrhea | |

## Hyperprolactinemia
| Component | Description | URL |
|-----------|-------------|-----|
| Causes | **Prolactinoma** (most common pathologic cause), **Medications** (antipsychotics, Antidepressants, Metoclopramide, Verapamil), **Hypothyroidism**, **Pregnancy**, **Chest wall trauma**, **Renal failure**, **Idiopathic** | https://www.endocrine.org/hyperprolactinemia |
| Presentation | **Galactorrhea**, **Amenorrhea/Oligomenorrhea**, **Infertility**, Hypogonadism, Decreased libido, Headache, Visual changes (macroprolactinoma) | |
| Diagnosis | **Serum prolactin**; **Macroprolactin** (if asymptomatic hyperprolactinemia); **TSH**; **MRI pituitary** (if pathologic) | |
| Microprolactinoma (<10 mm) | **Cabergoline** 0.25-0.5 mg weekly; **Bromocriptine** (alternative) | |
| Macroprolactinoma (>10 mm) | **Cabergoline** 0.5-1 mg weekly; Visual field testing; MRI surveillance | |

## Hypothalamic Amenorrhea (Functional Hypothalamic Amenorrhea - FHA)
| Component | Description | URL |
|-----------|-------------|-----|
| Pathophysiology | **Suppression of GnRH pulsatility** ? Low LH/FSH ? Anovulation, Hypoestrogenism | https://www.endocrine.org/FHA |
| Triggers | **Energy deficit** (weight loss, Disordered eating, Excessive exercise), **Psychological stress** | |
| Diagnosis | **Rule out other causes** (pregnancy, PCOS, Hyperprolactinemia, Thyroid disease, POI) | |
| Clinical Features | Amenorrhea, Low/normal LH/FSH, Low estradiol, Normal prolactin/TSH | |
| Consequences | **Osteoporosis** (estrogen deficiency), Infertility, Cardiovascular risk | |
| Treatment | **Address underlying cause** (weight gain, Reduce exercise, Stress reduction, CBT); **Estrogen replacement** (if prolonged amenorrhea - transdermal preferred); **Avoid OCPs** (mask underlying issue, May worsen bone loss) | |

## Congenital Adrenal Hyperplasia (CAH)
| Type | Enzyme Deficiency | Presentation | Diagnosis | Treatment | URL |
|------|------------------|--------------|-----------|-----------|-----|
| **21-Hydroxylase (Classic)** | CYP21A2 | **Salt-wasting** (hyponatremia, Hyperkalemia, Hypotension) in infancy; **Simple virilizing** (ambiguous genitalia, Precocious puberty) | **Elevated 17-OHP** (ACTH stimulation if borderline) | **Glucocorticoids** (Hydrocortisone) + **Mineralocorticoids** (Fludrocortisone) | https://www.endocrine.org/CAH |
| **21-Hydroxylase (Non-Classic)** | CYP21A2 | Hirsutism, Acne, Oligomenorrhea, Infertility (mimics PCOS) | **Elevated 17-OHP** on ACTH stimulation | **Glucocorticoids** (if symptomatic); OCPs for menstrual regulation | |
| **11-beta-Hydroxylase** | CYP11B1 | Virilization + **Hypertension** | Elevated 11-deoxycortisol | Glucocorticoids | |
| **17-alpha-Hydroxylase** | CYP17A1 | **Hypertension**, **Hypokalemia**, Sexual infantilism | Elevated progesterone, DOC | Glucocorticoids + Estrogen replacement | |
