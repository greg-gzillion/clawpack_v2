## Evaluation of Abnormal Uterine Bleeding
| Component | Assessment | URL |
|-----------|------------|-----|
| History | Pattern (heavy, Irregular, Intermenstrual, Postcoital), Duration, Impact on quality of life, Associated symptoms | https://www.acog.org/AUB |
| Physical Exam | Pelvic exam, Speculum exam, Bimanual exam, BMI, Signs of PCOS/thyroid disease | |
| Laboratory | **CBC** (anemia), **TSH**, **Prolactin**, **Coagulation studies** (if indicated), **Pregnancy test** | |
| Imaging | **Transvaginal ultrasound** (first-line); Saline infusion sonohysterography (SIS); Hysteroscopy | |
| Endometrial Sampling | Indicated if **age =45**, **risk factors for endometrial cancer** (obesity, Diabetes, Chronic anovulation, Lynch syndrome), or **failed medical management** | |

## Heavy Menstrual Bleeding (HMB) - Management
| Treatment | Options | Considerations | URL |
|-----------|---------|----------------|-----|
| **Hormonal** | **Levonorgestrel IUD** (first-line), Combined OCPs, Oral progestins, DMPA | LNg-IUD: 71-95% reduction in blood loss | https://www.acog.org/HMB |
| **Non-Hormonal** | **Tranexamic acid** (1.3g TID x 5 days), NSAIDs (Ibuprofen, Naproxen, Mefenamic acid) | Tranexamic acid reduces blood loss by 40-50%; Avoid in VTE history | |
| **Surgical** | Endometrial ablation, Myomectomy, Hysterectomy | For completed childbearing, Failed medical management | |
| **Iron Supplementation** | Oral iron for anemia; IV iron for severe anemia or intolerance | | |

## Amenorrhea - Evaluation Algorithm
| Category | Causes | Evaluation | URL |
|----------|--------|------------|-----|
| **Primary Amenorrhea** | No menses by age 15 (with normal growth) or age 13 (without secondary sexual characteristics) | **FSH**, **Karyotype**, **Pelvic US**, **MRI** if indicated | https://www.acog.org/amenorrhea |
| **Secondary Amenorrhea** | Absence of menses for =3 months in previously menstruating woman | **Rule out pregnancy** ? **TSH, Prolactin** ? **Progesterone challenge** ? **FSH, Estradiol** | |
| **Hypogonadotropic Hypogonadism** | Low FSH, Low Estradiol | Functional hypothalamic amenorrhea (stress, Weight loss, Exercise), Pituitary disorders, Kallmann syndrome | |
| **Hypergonadotropic Hypogonadism** | High FSH, Low Estradiol | **Primary ovarian insufficiency**, Turner syndrome | |
| **Eugonadotropic** | Normal FSH, Normal Estradiol | **PCOS** (most common), Outflow tract obstruction (Asherman syndrome) | |

## Primary Ovarian Insufficiency (POI)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Menopause before age 40 | https://www.acog.org/POI |
| Diagnosis | **FSH >40 IU/L on two occasions =1 month apart** + Amenorrhea | |
| Causes | Autoimmune (20-30%), Genetic (Fragile X premutation, Turner), Iatrogenic (chemo/radiation, Surgery), Idiopathic | |
| Evaluation | **Karyotype**, **FMR1 premutation testing**, **Thyroid antibodies**, **Adrenal antibodies** | |
| Treatment | **Estrogen replacement** (until age 50-51 for bone/cardiovascular protection); **Calcium/Vitamin D**; Fertility options (donor eggs, Adoption) | |
| Complications | Osteoporosis, Cardiovascular disease, Infertility, Psychological distress | |

## Dysmenorrhea
| Type | Characteristics | Management | URL |
|------|-----------------|------------|-----|
| **Primary** | Painful menstruation without pelvic pathology; Onset within 6-12 months of menarche | **NSAIDs** (first-line), **Hormonal contraceptives**, Heat therapy, Exercise | https://www.acog.org/dysmenorrhea |
| **Secondary** | Painful menstruation due to underlying pathology (endometriosis, Adenomyosis, Fibroids, PID) | Treat underlying cause; NSAIDs, Hormonal suppression | |
