# Gynecology - Gynecologic Oncology

## Endometrial Cancer
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | **Most common gynecologic cancer**; Median age 62 | https://www.sgo.org/endometrial-cancer |
| Types | **Type I (Endometrioid)** 80%: Estrogen-dependent, Obesity, Good prognosis; **Type II (Serous, Clear cell)** 20%: Estrogen-independent, Poor prognosis | |
| Risk Factors | **Unopposed estrogen** (obesity, PCOS, Nulliparity, Early menarche, Late menopause, Tamoxifen), **Lynch syndrome**, Diabetes, Hypertension | |
| Protective Factors | OCP use, Progestin IUD, Smoking, Multiparity | |
| Presentation | **Postmenopausal bleeding** (90%), Abnormal uterine bleeding | |
| Diagnosis | **Endometrial biopsy** (Pipelle); D&C if nondiagnostic | |
| Staging | **Surgical** (Total hysterectomy + BSO + Lymphadenectomy ± Omentectomy) | |
| Treatment (Stage I-II) | Surgery ± Adjuvant radiation (if high-risk) | |
| Treatment (Stage III-IV) | Surgery + Chemotherapy (Carboplatin/Paclitaxel) ± Radiation | |
| Molecular Classification | **POLE** (excellent prognosis), **MSI-H** (intermediate, Immunotherapy responsive), **p53 abnormal** (serous-like, Poor), **NSMP** (no specific molecular profile) | |

## Endometrial Hyperplasia
| Type | Progression to Cancer | Management | URL |
|------|----------------------|------------|-----|
| **Hyperplasia Without Atypia** | <5% | **Progestin therapy** (oral, LNg-IUD, DMPA); Repeat biopsy in 3-6 months; Hysterectomy if fails or completed childbearing | https://www.acog.org/endometrial-hyperplasia |
| **Atypical Hyperplasia / EIN** | 25-40% | **Hysterectomy** (if completed childbearing); **Progestin therapy** (if fertility desired - requires close surveillance, Repeat biopsy q3-6 months) | |

## Uterine Sarcoma
| Type | Frequency | Prognosis | Treatment | URL |
|------|-----------|-----------|-----------|-----|
| **Leiomyosarcoma** | Most common | Poor | **Total hysterectomy** (no morcellation); Adjuvant chemotherapy | https://www.sgo.org/uterine-sarcoma |
| **Endometrial Stromal Sarcoma** | Less common | Variable (low-grade = good) | Hysterectomy + BSO; Hormonal therapy (aromatase inhibitors) | |
| **Adenosarcoma** | Rare | Intermediate | Hysterectomy + BSO | |
| **Carcinosarcoma (MMMT)** | Aggressive | Poor | Surgery + Chemotherapy (Carboplatin/Paclitaxel) | |

## Gestational Trophoblastic Disease (GTD)
| Type | Features | Management | URL |
|------|----------|------------|-----|
| **Complete Mole** | 46,XX (paternal), No fetal tissue, 15-20% risk of GTN | **Suction D&C**; Monitor hCG weekly until normal x 3, Then monthly x 6 months | https://www.sgo.org/GTD |
| **Partial Mole** | 69,XXX/XXY (triploid), Fetal tissue present, <5% risk of GTN | **Suction D&C**; Monitor hCG | |
| **Invasive Mole** | Myometrial invasion | **Chemotherapy** (Methotrexate or Actinomycin-D) | |
| **Choriocarcinoma** | Aggressive, Metastatic | **Chemotherapy** (EMA-CO regimen) | |
| **Placental Site Trophoblastic Tumor (PSTT)** | Rare, Intermediate trophoblast | **Hysterectomy** (chemoresistant) | |
| **hCG Monitoring** | **Critical**; Contraception required (avoid pregnancy for 6-12 months) | |
