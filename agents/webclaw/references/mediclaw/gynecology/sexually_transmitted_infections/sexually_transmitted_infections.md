# Gynecology - Sexually Transmitted Infections

## CDC STI Treatment Guidelines - Summary
| Infection | Pathogen | First-Line Treatment | URL |
|-----------|----------|---------------------|-----|
| **Chlamydia** | Chlamydia trachomatis | **Doxycycline 100 mg BID x 7 days** (preferred); Azithromycin 1g x 1 (alternative) | https://www.cdc.gov/std/treatment-guidelines |
| **Gonorrhea** | Neisseria gonorrhoeae | **Ceftriaxone 500 mg IM x 1** (weight-based) | |
| **Co-infection** | Chlamydia + Gonorrhea | Treat both; Doxycycline 100 mg BID x 7 days + Ceftriaxone 500 mg IM | |
| **Trichomoniasis** | Trichomonas vaginalis | **Metronidazole 500 mg BID x 7 days** (women); **Metronidazole 2g x 1** (alternative) | |
| **Bacterial Vaginosis** | Polymicrobial | **Metronidazole 500 mg BID x 7 days**; Clindamycin cream; Secnidazole 2g x 1 | |
| **Vulvovaginal Candidiasis** | Candida albicans | **Fluconazole 150 mg x 1**; Topical azoles | |
| **Syphilis (Primary/Secondary/Early Latent)** | Treponema pallidum | **Penicillin G benzathine 2.4 million units IM x 1** | |
| **Syphilis (Late Latent/Unknown Duration)** | Treponema pallidum | **Penicillin G benzathine 2.4 million units IM weekly x 3 weeks** | |
| **Neurosyphilis/Ocular Syphilis** | Treponema pallidum | **Aqueous penicillin G 18-24 million units/day IV x 10-14 days** | |
| **Genital Herpes (Initial)** | HSV-1, HSV-2 | **Acyclovir 400 mg TID x 7-10 days**; Valacyclovir 1g BID; Famciclovir 250 mg TID | |
| **Genital Herpes (Recurrent)** | HSV-1, HSV-2 | Acyclovir 800 mg BID x 5 days; Valacyclovir 500 mg BID x 3 days; Famciclovir 1g BID x 1 day | |
| **Genital Herpes (Suppression)** | HSV-1, HSV-2 | Acyclovir 400 mg BID; Valacyclovir 500-1000 mg daily | |
| **Chancroid** | Haemophilus ducreyi | **Ceftriaxone 250 mg IM x 1**; Azithromycin 1g x 1; Ciprofloxacin 500 mg BID x 3 days | |
| **Lymphogranuloma Venereum (LGV)** | Chlamydia trachomatis (L1-L3) | **Doxycycline 100 mg BID x 21 days** | |
| **Granuloma Inguinale** | Klebsiella granulomatis | **Azithromycin 1g weekly x 3 weeks**; Doxycycline 100 mg BID x 3 weeks | |
| **Mycoplasma genitalium** | Mycoplasma genitalium | **Doxycycline 100 mg BID x 7 days followed by Moxifloxacin 400 mg daily x 7 days** | |

## Pelvic Inflammatory Disease (PID)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Inflammation of upper genital tract (endometritis, Salpingitis, Tubo-ovarian abscess, Pelvic peritonitis) | https://www.cdc.gov/std/treatment-guidelines/pid |
| Pathogens | N. gonorrhoeae, C. trachomatis, Anaerobes, Mycoplasma genitalium | |
| Clinical Criteria | **Pelvic/lower abdominal pain** + **Cervical motion tenderness** OR **Uterine tenderness** OR **Adnexal tenderness** | |
| Additional Criteria | Fever >38.3C, Abnormal cervical discharge, Elevated ESR/CRP, WBCs on saline microscopy | |
| Outpatient Treatment | **Ceftriaxone 500 mg IM x 1 + Doxycycline 100 mg BID x 14 days + Metronidazole 500 mg BID x 14 days** | |
| Inpatient Treatment | IV antibiotics (Cefoxitin or Cefotetan + Doxycycline); Transition to oral after 24-48h improvement | |
| Admission Indications | Pregnancy, Severe illness, Failed outpatient, Tubo-ovarian abscess, Inability to tolerate PO | |
| Complications | Infertility (15-20% after one episode), Ectopic pregnancy, Chronic pelvic pain | |

## Vaginitis - Differential Diagnosis
| Feature | Bacterial Vaginosis | Candidiasis | Trichomoniasis | URL |
|---------|---------------------|-------------|----------------|-----|
| Discharge | Thin, Gray, Homogeneous | Thick, White, Curd-like | Frothy, Yellow-green | https://www.cdc.gov/std/treatment-guidelines/vaginitis |
| Odor | **Fishy** (amine) | None | Malodorous | |
| pH | **>4.5** | <4.5 | **>4.5** | |
| Microscopy | **Clue cells** (>20% of epithelial cells) | **Pseudohyphae/Budding yeast** | **Motile trichomonads**, WBCs | |
| Whiff Test | **Positive** (fishy odor with KOH) | Negative | Often positive | |
| Treatment | Metronidazole 500 mg BID x 7d | Fluconazole 150 mg x 1 | Metronidazole 500 mg BID x 7d | |

## Cervicitis
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Inflammation of the cervix (mucopurulent discharge, Friability) | https://www.cdc.gov/std/treatment-guidelines/cervicitis |
| Etiology | C. trachomatis, N. gonorrhoeae, M. genitalium, HSV, Trichomonas | |
| Diagnosis | NAAT for chlamydia/gonorrhea; Consider M. genitalium testing | |
| Treatment | **Empiric treatment for chlamydia and gonorrhea** while awaiting results | |
| Persistent Cervicitis | Consider M. genitalium, HSV, Trichomonas, Non-infectious causes | |

## STI Screening Recommendations
| Population | Screening | Frequency | URL |
|------------|-----------|-----------|-----|
| **All sexually active women <25** | Chlamydia, Gonorrhea | **Annual** | https://www.cdc.gov/std/screening |
| **Women =25 with risk factors** | Chlamydia, Gonorrhea | Annual | |
| **All pregnant women** | Chlamydia, Gonorrhea, Syphilis, HIV, Hepatitis B | Initial prenatal visit; Repeat in 3rd trimester if high risk | |
| **MSM** | HIV, Syphilis, Chlamydia, Gonorrhea (urethral, Rectal, Pharyngeal) | q3-6 months if high risk; Annual for all | |
| **HIV-infected** | Syphilis, Chlamydia, Gonorrhea, Trichomonas (women) | Initial and at least annual | |
| **Transgender** | Based on anatomy and sexual behaviors | | |

## Expedited Partner Therapy (EPT)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Providing treatment for sex partners of patients diagnosed with chlamydia or gonorrhea without clinical assessment | https://www.cdc.gov/std/ept |
| Legal Status | Permissible in most states (check local laws) | |
| Regimen | **Cefixime 800 mg PO x 1 + Doxycycline 100 mg BID x 7 days** (preferred) | |
| Contraindications | Pregnancy, Suspected complicated infection, MSM (high risk of co-infections) | |
