# Transplant Medicine - Organ Allocation

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **OPTN** | **Organ Procurement and Transplantation Network** (oversees US transplant system) | https://optn.transplant.hrsa.gov |
| **UNOS** | **United Network for Organ Sharing** (operates OPTN under contract) | https://unos.org |
| **Final Rule** | **Allocation based on medical urgency and benefit**, **Not geography** (though geography still influences) | https://optn.transplant.hrsa.gov/about/final-rule/ |

## Kidney Allocation System (KAS)
| Component | Description | URL |
|-----------|-------------|-----|
| **KDPI** | **Kidney Donor Profile Index** (0-100%, lower = better quality) | https://optn.transplant.hrsa.gov/data/allocation-calculators/kdpi-calculator/ |
| **EPTS** | **Estimated Post-Transplant Survival** (0-100%, lower = longer survival) | https://optn.transplant.hrsa.gov/data/allocation-calculators/epts-calculator/ |
| **Longevity Matching** | **Match high-KDPI kidneys to low-EPTS recipients** | https://optn.transplant.hrsa.gov/professionals/by-organ/kidney/ |
| **Points System** | **Waiting time**, **Sensitization** (cPRA), **Age** (pediatric priority), **Prior living donor** | https://optn.transplant.hrsa.gov/professionals/by-organ/kidney/ |

## Liver Allocation System (MELD/PELD)
| Score | Components | Range | URL |
|-------|------------|-------|-----|
| **MELD** | **Bilirubin**, **INR**, **Creatinine**, **Sodium** | **6-40** | https://optn.transplant.hrsa.gov/data/allocation-calculators/meld-calculator/ |
| **MELD-Na** | **MELD + Sodium** (better predicts mortality) | **6-40** | https://optn.transplant.hrsa.gov/data/allocation-calculators/meld-calculator/ |
| **PELD** | **Bilirubin**, **INR**, **Albumin**, **Growth failure**, **Age <12** | **Negative-40+** | https://optn.transplant.hrsa.gov/data/allocation-calculators/peld-calculator/ |
| **Exception Points** | **HCC**, **Hepatopulmonary syndrome**, **Portopulmonary hypertension**, **Cholangiocarcinoma** | **Assigned by review board** | https://optn.transplant.hrsa.gov/professionals/by-organ/liver/ |

## Heart Allocation System
| Status | Criteria | URL |
|--------|----------|-----|
| **Status 1** | **ECMO**, **Non-dischargeable BiVAD**, **Mechanical ventilation** | https://optn.transplant.hrsa.gov/professionals/by-organ/heart/ |
| **Status 2** | **Non-dischargeable LVAD**, **IABP**, **Refractory VT/VF**, **MCS with complication** | https://optn.transplant.hrsa.gov/professionals/by-organ/heart/ |
| **Status 3** | **Dischargeable LVAD**, **Inotropes without hemodynamic monitoring**, **Stable MCS** | https://optn.transplant.hrsa.gov/professionals/by-organ/heart/ |
| **Status 4** | **Stable inotropes**, **Restrictive cardiomyopathy**, **CHD**, **Re-transplant** | https://optn.transplant.hrsa.gov/professionals/by-organ/heart/ |
| **Status 5** | **Stable on no inotropes** (combined heart-lung) | https://optn.transplant.hrsa.gov/professionals/by-organ/heart/ |
| **Status 6** | **Stable on no inotropes** (heart only) | https://optn.transplant.hrsa.gov/professionals/by-organ/heart/ |
