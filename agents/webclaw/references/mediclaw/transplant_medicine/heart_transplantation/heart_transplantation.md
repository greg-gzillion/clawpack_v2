# Transplant Medicine - Heart Transplantation

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | **End-stage heart failure** (NYHA IV, ACC/AHA Stage D) despite optimal medical/device therapy | https://ishlt.org/ishlt-documents/guidelines |
| **Contraindications** | **Active malignancy** (<5 years), **Severe irreversible PH** (PVR >5 Wood units), **Severe COPD**, **Active infection**, **Non-adherence**, **Active substance use** | https://ishlt.org/ishlt-documents/guidelines |

## Pre-Transplant Evaluation
| Component | Assessment | URL |
|-----------|------------|-----|
| **Cardiopulmonary** | **Right heart catheterization** (PVR), **CPET** (VO2max <14 mL/kg/min) | https://ishlt.org/ishlt-documents/guidelines |
| **Immunologic** | **HLA typing**, **PRA** | https://ishlt.org/ishlt-documents/guidelines |
| **Infectious Disease** | **HIV**, **HBV**, **HCV**, **CMV**, **EBV**, **Toxoplasma**, **TB**, **VZV** | https://ishlt.org/ishlt-documents/guidelines |

## Surgical Considerations
| Component | Description | URL |
|-----------|-------------|-----|
| **Bicaval Technique** | **Preserves right atrial anatomy** (lower TR, Better hemodynamics) | https://ishlt.org/ishlt-documents/guidelines |
| **Total Orthotopic** | **Bicaval + Pulmonary venous anastomoses** | https://ishlt.org/ishlt-documents/guidelines |

## Post-Transplant Complications
| Complication | Timing | Management | URL |
|--------------|--------|------------|-----|
| **Primary Graft Dysfunction (PGD)** | **<24h** | **Inotropes**, **Mechanical support** (IABP, Impella, ECMO) | https://ishlt.org/ishlt-documents/guidelines |
| **Acute Rejection** | **Weeks-months** | **Pulse steroids**, **Thymoglobulin**, **Plasmapheresis/IVIG** (AMR) | https://ishlt.org/ishlt-documents/guidelines |
| **Cardiac Allograft Vasculopathy (CAV)** | **Chronic** | **Statin**, **Everolimus/Sirolimus**, **PCI**, **Re-transplantation** | https://ishlt.org/ishlt-documents/guidelines |
| **Infection** | **Variable** | **CMV**, **Toxoplasmosis**, **PJP** (prophylaxis) | https://ishlt.org/ishlt-documents/guidelines |
| **Malignancy** | **Chronic** | **Skin cancer**, **PTLD** | https://ishlt.org/ishlt-documents/guidelines |

## Rejection Surveillance (Endomyocardial Biopsy)
| Grade | ISHLT Classification | Management | URL |
|-------|---------------------|------------|-----|
| **0R** | **No rejection** | **Continue maintenance** | https://ishlt.org/ishlt-documents/guidelines |
| **1R** | **Mild** (interstitial/perivascular infiltrate, No myocyte damage) | **Optimize immunosuppression** | https://ishlt.org/ishlt-documents/guidelines |
| **2R** | **Moderate** (=2 foci with myocyte damage) | **Pulse steroids** | https://ishlt.org/ishlt-documents/guidelines |
| **3R** | **Severe** (diffuse infiltrate, Multifocal damage, ｱ Hemodynamic compromise) | **Thymoglobulin**, **Plasmapheresis/IVIG** (if AMR) | https://ishlt.org/ishlt-documents/guidelines |
