# Transplant Medicine - Post-Transplant Malignancy

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Increased Risk** | **2-4x** overall cancer risk compared to general population | https://www.myast.org/education/post-transplant-malignancy |
| **Most Common** | **Skin cancer** (SCC > BCC), **PTLD**, **Kaposi sarcoma**, **Anogenital** (HPV-related), **Renal cell carcinoma** | https://www.myast.org/education/post-transplant-malignancy |

## Skin Cancer
| Type | Risk | Management | URL |
|------|------|------------|-----|
| **Squamous Cell Carcinoma (SCC)** | **65-250x** increased | **Annual dermatology exam**, **Sun protection**, **mTOR inhibitor** (sirolimus/everolimus) | https://www.myast.org/education/post-transplant-malignancy |
| **Basal Cell Carcinoma (BCC)** | **10-16x** increased | **Annual dermatology exam** | https://www.myast.org/education/post-transplant-malignancy |
| **Melanoma** | **2-5x** increased | **Annual dermatology exam** | https://www.myast.org/education/post-transplant-malignancy |

## Post-Transplant Lymphoproliferative Disorder (PTLD)
| Component | Description | URL |
|-----------|-------------|-----|
| **Etiology** | **EBV-driven B-cell proliferation** (80-90%) | https://www.myast.org/education/post-transplant-malignancy |
| **Risk Factors** | **EBV D+/R-**, **Intensive immunosuppression**, **Pediatric**, **Intestinal/Multivisceral transplant** | https://www.myast.org/education/post-transplant-malignancy |
| **Classification (WHO)** | **Early lesions**, **Polymorphic**, **Monomorphic** (DLBCL, Burkitt), **Classic Hodgkin** | https://www.myast.org/education/post-transplant-malignancy |
| **Management** | **Reduce immunosuppression** (first-line), **Rituximab** (CD20+), **Chemotherapy** (high-grade), **Radiation**, **Surgery** | https://www.myast.org/education/post-transplant-malignancy |

| **Prostate** | **PSA** (individualized) | https://www.myast.org/education/post-transplant-malignancy |
| **Lung** | **Low-dose CT** (if meets smoking criteria) | https://www.myast.org/education/post-transplant-malignancy |
| **Renal** | **Ultrasound** (if acquired cystic disease or prior RCC) | https://www.myast.org/education/post-transplant-malignancy |
