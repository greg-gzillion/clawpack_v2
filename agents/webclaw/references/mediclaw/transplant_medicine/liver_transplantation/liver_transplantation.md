# Transplant Medicine - Liver Transplantation

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | **Cirrhosis** (viral, Alcohol, NASH), **HCC**, **Acute liver failure**, **Cholestatic** (PBC, PSC), **Metabolic** | https://www.aasld.org/practice-guidelines/liver-transplantation |
| **Contraindications** | **Severe CVD/COPD**, **Active malignancy** (outside Milan), **Uncontrolled infection**, **Active substance use**, **Non-adherence** | https://www.aasld.org/practice-guidelines/liver-transplantation |

## MELD Exceptions for HCC (Milan Criteria)
| Criterion | Description | URL |
|-----------|-------------|-----|
| **Single Tumor** | **=5 cm** | https://optn.transplant.hrsa.gov/professionals/by-organ/liver/ |
| **2-3 Tumors** | **Each =3 cm** | https://optn.transplant.hrsa.gov/professionals/by-organ/liver/ |
| **No Extrahepatic Spread** | **No vascular invasion**, **No metastases** | https://optn.transplant.hrsa.gov/professionals/by-organ/liver/ |
| **MELD Exception** | **28 points** (increases every 3 months) | https://optn.transplant.hrsa.gov/professionals/by-organ/liver/ |

## Surgical Considerations
| Component | Description | URL |
|-----------|-------------|-----|
| **Deceased Donor** | **Whole liver** or **Split liver** | https://asts.org/patient-information/liver-transplant |
| **Living Donor** | **Right lobe** (adult), **Left lateral segment** (pediatric) | https://asts.org/patient-information/liver-transplant |
| **Anastomoses** | **Hepatic artery**, **Portal vein**, **Bile duct** (duct-to-duct or Roux-en-Y) | https://asts.org/patient-information/liver-transplant |

## Post-Transplant Complications
| Complication | Timing | Management | URL |
|--------------|--------|------------|-----|
| **Primary Non-Function (PNF)** | **<7 days** | **Re-transplantation** (urgent) | https://www.aasld.org/practice-guidelines/liver-transplantation |
| **Hepatic Artery Thrombosis (HAT)** | **Early** | **Thrombectomy**, **Re-transplantation** | https://www.aasld.org/practice-guidelines/liver-transplantation |
| **Biliary Complications** | **Variable** | **ERCP** (stricture, leak), **PTC**, **Surgical revision** | https://www.aasld.org/practice-guidelines/liver-transplantation |
| **Acute Rejection** | **Weeks-months** | **Pulse steroids**, **Optimize CNI** | https://www.aasld.org/practice-guidelines/liver-transplantation |
| **Recurrent Disease** | **Months-years** | **HCV** (DAA), **HBV** (HBIG + antivirals), **NASH**, **PBC/PSC**, **AIH** | https://www.aasld.org/practice-guidelines/liver-transplantation |
| **CNI Nephrotoxicity** | **Chronic** | **Renal-sparing regimens** (MMF + low CNI, mTORi, Belatacept) | https://www.aasld.org/practice-guidelines/liver-transplantation |

## Recurrent HCV Management
| Component | Description | URL |
|-----------|-------------|-----|
| **Universal Recurrence** | **100%** without treatment | https://www.hcvguidelines.org/unique-populations/post-liver-transplant |
| **Treatment** | **Direct-acting antivirals (DAAs)** for 12 weeks | https://www.hcvguidelines.org/unique-populations/post-liver-transplant |
| **Timing** | **=3 months post-transplant** (stable) | https://www.hcvguidelines.org/unique-populations/post-liver-transplant |
| **Drug Interactions** | **Avoid protease inhibitors with cyclosporine** | https://www.hcvguidelines.org/unique-populations/post-liver-transplant |
