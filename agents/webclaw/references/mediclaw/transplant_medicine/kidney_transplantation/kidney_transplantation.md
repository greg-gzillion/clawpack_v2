# Transplant Medicine - Kidney Transplantation

|-----------|-------------|-----|

## Post-Transplant Complications
| Complication | Timing | Management | URL |
|--------------|--------|------------|-----|
| **Delayed Graft Function (DGF)** | **<1 week** | **Dialysis**, **Monitor**, **Biopsy** (if prolonged) | https://www.kidney.org/atoz/content/kidney-transplantation |
| **Acute Rejection** | **Weeks-months** | **Pulse steroids**, **Thymoglobulin**, **Plasmapheresis/IVIG** (AMR) | https://www.kidney.org/atoz/content/kidney-transplantation |
| **Surgical Complications** | **Early** | **Bleeding**, **Thrombosis**, **Ureteral leak/stenosis**, **Lymphocele** | https://www.kidney.org/atoz/content/kidney-transplantation |
| **Infection** | **Variable** | **CMV**, **BKV**, **UTI**, **PJP** (prophylaxis timing-specific) | https://www.kidney.org/atoz/content/kidney-transplantation |
| **Recurrent Disease** | **Months-years** | **FSGS**, **IgA nephropathy**, **Membranous**, **Diabetes** | https://www.kidney.org/atoz/content/kidney-transplantation |
| **CNI Nephrotoxicity** | **Chronic** | **Minimize CNI**, **Consider belatacept or mTORi** | https://www.kidney.org/atoz/content/kidney-transplantation |
| **Post-Transplant Diabetes Mellitus (PTDM)** | **Weeks-months** | **Screen**, **Treat**, **Consider steroid minimization** | https://www.kidney.org/atoz/content/kidney-transplantation |

## BK Virus (BKV) Nephropathy
| Component | Description | URL |
|-----------|-------------|-----|
| **Screening** | **Plasma BKV PCR** monthly x6 months ? q3 months x1 year ? annually | https://www.myast.org/education/infectious-disease |
| **Threshold** | **>10,000 copies/mL** (or rising) | https://www.myast.org/education/infectious-disease |
| **Management** | **Reduce immunosuppression** (CNI or MMF), **Consider leflunomide**, **Cidofovir**, **IVIG**, **Quinolones** (limited evidence) | https://www.myast.org/education/infectious-disease |
| **Outcome** | **Graft loss** in 30-50% if untreated | https://www.myast.org/education/infectious-disease |
