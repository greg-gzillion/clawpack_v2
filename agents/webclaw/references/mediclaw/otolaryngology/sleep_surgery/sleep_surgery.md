# Otolaryngology - Sleep Surgery

## Obstructive Sleep Apnea (OSA)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Recurrent upper airway collapse** during sleep | https://www.entnet.org/conditions/obstructive-sleep-apnea/ |
| **Diagnosis** | **Polysomnography (PSG)** - AHI =5 | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Severity** | **Mild**: AHI 5-15; **Moderate**: 15-30; **Severe**: >30 | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |

## Surgical Treatment of OSA
| Procedure | Indication | Description | URL |
|-----------|------------|-------------|-----|
| **Uvulopalatopharyngoplasty (UPPP)** | **Palatal obstruction** | **Resect uvula, tonsils, redundant soft palate** | https://www.entnet.org/conditions/sleep-apnea-surgery/ |
| **Expansion Sphincter Pharyngoplasty** | **Lateral pharyngeal wall collapse** | **Reposition palatopharyngeus muscle** | https://www.entnet.org/conditions/sleep-apnea-surgery/ |
| **Tonsillectomy** | **Tonsillar hypertrophy** (especially children) | **Remove tonsils** | https://www.entnet.org/conditions/sleep-apnea-surgery/ |
| **Lingual Tonsillectomy** | **Lingual tonsil hypertrophy** | **Remove lingual tonsils** (TORS, Coblation) | https://www.entnet.org/conditions/sleep-apnea-surgery/ |
| **Midline Glossectomy** | **Tongue base obstruction** | **Resect midline posterior tongue** | https://www.entnet.org/conditions/sleep-apnea-surgery/ |
| **Genioglossus Advancement** | **Tongue base obstruction** | **Advance genial tubercle** | https://www.entnet.org/conditions/sleep-apnea-surgery/ |
| **Hyoid Suspension** | **Hypopharyngeal collapse** | **Suspend hyoid to thyroid cartilage or mandible** | https://www.entnet.org/conditions/sleep-apnea-surgery/ |
| **Maxillomandibular Advancement (MMA)** | **Multilevel obstruction**, **Refractory OSA** | **Le Fort I + BSSO** to advance both jaws | https://www.entnet.org/conditions/sleep-apnea-surgery/ |
| **Hypoglossal Nerve Stimulation** | **Moderate-severe OSA**, **CPAP intolerant**, **BMI <32**, **No concentric collapse on DISE** | **Implanted device** stimulates hypoglossal nerve during sleep | https://www.entnet.org/conditions/inspire-therapy/ |
