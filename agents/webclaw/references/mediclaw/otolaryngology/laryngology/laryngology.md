# Otolaryngology - Laryngology

## Hoarseness (Dysphonia)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Abnormal voice quality** (raspy, strained, breathy) | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Acute (<3 weeks)** | **Viral laryngitis**, **Voice overuse**, **Allergies** | **Voice rest**, **Hydration**, **Avoid irritants** | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Chronic (>3 weeks)** | **Laryngopharyngeal reflux (LPR)** , **Vocal cord nodules/polyps**, **Muscle tension dysphonia**, **Reinke edema**, **Malignancy** | **Laryngoscopy**, **Voice therapy**, **Treat underlying cause** | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Red Flags** | **Smoking history**, **Age >50**, **Dysphagia**, **Odynophagia**, **Otalgia**, **Hemoptysis**, **Weight loss** | **Urgent laryngoscopy** to rule out malignancy | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |

## Vocal Cord Lesions
| Lesion | Description | Treatment | URL |
|--------|-------------|-----------|-----|
| **Vocal Cord Nodules** | **Bilateral**, **Mid-membranous**, **"Singer's nodules"** | **Voice therapy** (first-line), **Surgery** (refractory) | https://www.entnet.org/conditions/vocal-cord-nodules/ |
| **Vocal Cord Polyp** | **Unilateral**, **Hemorrhagic or non-hemorrhagic** | **Voice therapy**, **Microsurgical excision** | https://www.entnet.org/conditions/vocal-cord-polyps/ |
| **Reinke Edema** | **Polypoid degeneration**, **Smoking**, **LPR** | **Smoking cessation**, **Voice therapy**, **Surgery** | https://www.entnet.org/conditions/reinke-edema/ |
| **Vocal Cord Cyst** | **Submucosal**, **Epidermoid or mucous retention** | **Microsurgical excision** | https://www.entnet.org/conditions/vocal-cord-cysts/ |
| **Vocal Cord Granuloma** | **Posterior**, **Intubation**, **LPR**, **Voice abuse** | **Treat LPR**, **Voice therapy**, **Observation** (often resolve spontaneously) | https://www.entnet.org/conditions/vocal-cord-granuloma/ |
| **Laryngeal Papillomatosis** | **HPV 6, 11**, **Recurrent**, **Risk of airway obstruction**, **Malignant transformation** | **Serial debridement**, **Adjuvant therapies** (cidofovir, Bevacizumab, HPV vaccine) | https://www.entnet.org/conditions/recurrent-respiratory-papillomatosis/ |

## Vocal Cord Paralysis
| Component | Description | URL |
|-----------|-------------|-----|
| **Etiology** | **Iatrogenic** (thyroidectomy, Anterior cervical spine, Carotid, Cardiac surgery - most common), **Malignancy** (lung, thyroid, esophageal), **Idiopathic**, **Neurologic** (stroke, ALS, MS) | https://www.entnet.org/conditions/vocal-cord-paralysis/ |
| **Unilateral** | **Hoarseness**, **Breathy voice**, **Aspiration**, **Ineffective cough** | **Voice therapy**, **Injection laryngoplasty** (temporary), **Medialization thyroplasty** (permanent) | https://www.entnet.org/conditions/vocal-cord-paralysis/ |
| **Bilateral** | **Stridor**, **Airway obstruction** (often requires tracheostomy), **Normal or near-normal voice** | **Tracheostomy**, **Cordotomy/arytenoidectomy**, **Reinnervation** | https://www.entnet.org/conditions/vocal-cord-paralysis/ |

## Laryngopharyngeal Reflux (LPR)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Reflux of gastric contents** into laryngopharynx | https://www.entnet.org/conditions/laryngopharyngeal-reflux/ |
| **Symptoms** | **Globus sensation**, **Chronic throat clearing**, **Cough**, **Hoarseness**, **Postnasal drip sensation** | https://www.entnet.org/conditions/laryngopharyngeal-reflux/ |
| **Laryngoscopy Findings** | **Posterior commissure hypertrophy**, **Erythema/edema of arytenoids**, **Vocal cord edema**, **Pseudosulcus** | https://www.entnet.org/conditions/laryngopharyngeal-reflux/ |
| **Reflux Finding Score (RFS)** | **>7 suggests LPR** | https://www.entnet.org/conditions/laryngopharyngeal-reflux/ |
| **Reflux Symptom Index (RSI)** | **>13 suggests LPR** | https://www.entnet.org/conditions/laryngopharyngeal-reflux/ |
| **Treatment** | **Lifestyle modifications** (elevate HOB, Avoid triggers), **PPI** (high-dose, 2-3 months), **Alginate**, **H2 blockers** | https://www.entnet.org/conditions/laryngopharyngeal-reflux/ |

## Laryngeal Cancer
| Component | Description | URL |
|-----------|-------------|-----|
| **Risk Factors** | **Tobacco**, **Alcohol** (synergistic) | https://www.cancer.gov/types/head-and-neck |
| **Symptoms** | **Hoarseness >2 weeks** (glottic), **Dysphagia**, **Odynophagia**, **Otalgia**, **Airway obstruction** | https://www.entnet.org/conditions/laryngeal-cancer/ |
| **Glottic** | **Hoarseness early** ? **Better prognosis** (rich lymphatic drainage delayed) | https://www.entnet.org/conditions/laryngeal-cancer/ |
| **Supraglottic** | **Late symptoms**, **Early lymphatic spread** | https://www.entnet.org/conditions/laryngeal-cancer/ |
| **Subglottic** | **Rare**, **Airway obstruction** | https://www.entnet.org/conditions/laryngeal-cancer/ |
| **Treatment** | **Early**: Radiation or Transoral laser microsurgery; **Advanced**: Chemoradiation or Total laryngectomy | https://www.nccn.org/guidelines/head-and-neck |
