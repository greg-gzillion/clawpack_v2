# Otolaryngology - Pediatric Otolaryngology

## Acute Otitis Media (AOM)
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnosis** | **Bulging TM** (most specific), **Impaired mobility**, **Otorrhea**, **Erythema** | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Observation Criteria** | **=6 months**, **Unilateral**, **Mild symptoms**, **No otorrhea**, **Reliable follow-up** | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Treatment** | **Amoxicillin** 80-90 mg/kg/day (first-line), **Amoxicillin-clavulanate** (treatment failure) | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Tympanostomy Tube Indications** | **Recurrent AOM** (=3 in 6 months or =4 in 12 months), **Chronic OME with hearing loss** | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |

## Otitis Media with Effusion (OME)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Fluid in middle ear without signs of acute infection** | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Management** | **Observation** (resolves in 3 months), **Tympanostomy tubes** (if persistent >3 months + hearing loss) | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Avoid** | **Decongestants**, **Antihistamines**, **Corticosteroids** (not effective) | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |

## Pediatric Obstructive Sleep Apnea (OSA)
| Component | Description | URL |
|-----------|-------------|-----|
| **Etiology** | **Adenotonsillar hypertrophy** (most common), **Obesity**, **Craniofacial abnormalities** | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Symptoms** | **Snoring**, **Witnessed apneas**, **Restless sleep**, **Enuresis**, **Daytime hyperactivity** (not sleepiness) | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Diagnosis** | **Polysomnography (PSG)** - AHI =1 = OSA | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Treatment** | **Adenotonsillectomy** (first-line), **CPAP**, **Weight loss**, **Intranasal corticosteroids** (mild) | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |

## Pediatric Stridor
| Condition | Age | Features | Treatment | URL |
|-----------|-----|----------|-----------|-----|
| **Laryngomalacia** | **Birth - 6 months** | **Inspiratory stridor**, **Worse with feeding/agitation**, **Improves with prone** | **Observation** (resolves by 12-18 months), **Supraglottoplasty** (severe) | https://www.entnet.org/conditions/laryngomalacia/ |
| **Vocal Cord Paralysis** | **Any** | **Biphasic stridor**, **Weak cry** | **Observation** (unilateral), **Tracheostomy** (bilateral) | https://www.entnet.org/conditions/pediatric-vocal-cord-paralysis/ |
| **Subglottic Stenosis** | **Any** | **Biphasic stridor**, **History of intubation** | **Endoscopic dilation**, **Laryngotracheal reconstruction** | https://www.entnet.org/conditions/subglottic-stenosis/ |
| **Croup** | **6 months - 3 years** | **Barking cough**, **Inspiratory stridor**, **Steeple sign** on X-ray | **Dexamethasone**, **Nebulized epinephrine** (severe) | https://www.entnet.org/conditions/croup/ |

## Foreign Body Aspiration
| Component | Description | URL |
|-----------|-------------|-----|
