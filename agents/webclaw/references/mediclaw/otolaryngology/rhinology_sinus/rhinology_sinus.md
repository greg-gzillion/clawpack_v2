## Acute Rhinosinusitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Viral** | **<10 days**, **Improving** | **Supportive** (saline irrigation, Decongestants, Analgesics) | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Acute Bacterial** | **>10 days**, **Double sickening**, **Purulent discharge**, **Fever >39蚓**, **Facial pain** | **Amoxicillin-clavulanate** (first-line), **Doxycycline** (penicillin allergy) | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Complications** | **Orbital cellulitis/abscess**, **Intracranial** (meningitis, Brain abscess, Cavernous sinus thrombosis), **Pott puffy tumor** (frontal osteomyelitis) | **Emergent ENT consultation**, **IV antibiotics**, **Surgical drainage** | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |

## Chronic Rhinosinusitis (CRS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **=12 weeks** of nasal obstruction, Rhinorrhea, Facial pressure, Hyposmia | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **CRS with Nasal Polyps (CRSwNP)** | **Th2 inflammation**, **Eosinophilia**, **Aspirin-exacerbated respiratory disease (AERD)** | **Intranasal corticosteroids**, **Saline irrigation**, **Oral corticosteroids**, **Biologics** (dupilumab, omalizumab, mepolizumab), **Endoscopic sinus surgery** | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **CRS without Nasal Polyps (CRSsNP)** | **Neutrophilic inflammation** | **Intranasal corticosteroids**, **Saline irrigation**, **Prolonged antibiotics**, **Endoscopic sinus surgery** | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
