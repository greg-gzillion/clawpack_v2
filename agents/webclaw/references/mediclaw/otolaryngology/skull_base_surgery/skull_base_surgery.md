## CSF Leak
| Component | Description | URL |
|-----------|-------------|-----|
| **Etiology** | **Trauma** (most common), **Iatrogenic** (sinus/skull base surgery), **Spontaneous** (idiopathic intracranial hypertension) | https://www.entnet.org/conditions/csf-leak/ |
| **Diagnosis** | **Beta-2 transferrin** (gold standard), **High-resolution CT**, **MRI cisternography**, **Intrathecal fluorescein** | https://www.entnet.org/conditions/csf-leak/ |
| **Treatment** | **Endoscopic repair** (multilayer closure with mucosal graft, Fascia, Fat, Nasoseptal flap), **Lumbar drain** (controversial) | https://www.entnet.org/conditions/csf-leak/ |
