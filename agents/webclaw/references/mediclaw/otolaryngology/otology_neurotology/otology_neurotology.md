# Otolaryngology - Otology and Neurotology

## Hearing Loss
| Type | Location | Causes | Treatment | URL |
|------|----------|--------|-----------|-----|
| **Conductive** | **External or middle ear** | **Cerumen impaction**, **TM perforation**, **Otitis media with effusion**, **Otosclerosis**, **Cholesteatoma** | **Treat underlying cause**, **Hearing aids**, **Surgery** | https://www.entnet.org/conditions/hearing-loss/ |
| **Sensorineural** | **Cochlea or CN VIII** | **Presbycusis**, **Noise-induced**, **Ototoxicity**, **Meniere disease**, **Acoustic neuroma**, **Congenital** | **Hearing aids**, **Cochlear implant** | https://www.entnet.org/conditions/hearing-loss/ |
| **Mixed** | **Both conductive and sensorineural** | **Chronic otitis media**, **Trauma**, **Otosclerosis with presbycusis** | **Combination** of treatments | https://www.entnet.org/conditions/hearing-loss/ |

## Sudden Sensorineural Hearing Loss (SSNHL)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **=30 dB loss** over **=3 contiguous frequencies** within **=72 hours** | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Etiology** | **Idiopathic** (most common), **Viral**, **Vascular**, **Autoimmune** | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Diagnosis** | **Audiogram** (essential), **MRI** (rule out acoustic neuroma) | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Treatment** | **Corticosteroids** (oral or intratympanic), **ｱ Hyperbaric oxygen** | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |
| **Prognosis** | **50-65% spontaneous recovery**; **Better with early treatment** | https://www.entnet.org/quality-practice/clinical-practice-guidelines/ |

## Tinnitus
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Perception of sound without external source** | https://www.entnet.org/conditions/tinnitus/ |
| **Subjective** | **Audible only to patient** (most common) | https://www.entnet.org/conditions/tinnitus/ |
| **Objective** | **Audible to examiner** (vascular, Muscular) | https://www.entnet.org/conditions/tinnitus/ |
| **Causes** | **Hearing loss**, **Noise exposure**, **Meniere disease**, **Acoustic neuroma**, **Ototoxic medications**, **TMJ disorder** | https://www.entnet.org/conditions/tinnitus/ |
| **Treatment** | **Treat underlying cause**, **Sound therapy**, **CBT**, **Hearing aids**, **Tinnitus retraining therapy (TRT)** | https://www.entnet.org/conditions/tinnitus/ |

## Chronic Otitis Media
| Type | Description | Treatment | URL |
|------|-------------|-----------|-----|
| **Mucosal** | **TM perforation**, **Intermittent otorrhea**, **Conductive hearing loss** | **Aural toilet**, **Topical antibiotics**, **Tympanoplasty** | https://www.entnet.org/conditions/chronic-otitis-media/ |
| **Cholesteatoma** | **Keratinizing squamous epithelium** in middle ear ? **Bone erosion**, **Complications** (facial paralysis, Labyrinthitis, Meningitis, Brain abscess) | **Surgical excision** (tympanomastoidectomy) | https://www.entnet.org/conditions/cholesteatoma/ |

## Cholesteatoma
| Component | Description | URL |
|-----------|-------------|-----|
| **Types** | **Congenital** (behind intact TM), **Acquired** (TM retraction pocket or perforation) | https://www.entnet.org/conditions/cholesteatoma/ |
| **Complications** | **Conductive hearing loss**, **Facial nerve paralysis**, **Labyrinthine fistula**, **Sigmoid sinus thrombosis**, **Meningitis**, **Brain abscess** | https://www.entnet.org/conditions/cholesteatoma/ |
| **Diagnosis** | **Otoscopy** (pearly mass), **CT temporal bone** (bone erosion) | https://www.entnet.org/conditions/cholesteatoma/ |
| **Treatment** | **Surgical excision** (tympanomastoidectomy) | https://www.entnet.org/conditions/cholesteatoma/ |

## Otosclerosis
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Abnormal bone remodeling** of otic capsule ? **Stapes fixation** | https://www.entnet.org/conditions/otosclerosis/ |
| **Presentation** | **Progressive conductive hearing loss** (young adults), **Women > Men**, **Worsens in pregnancy**, **Family history** | https://www.entnet.org/conditions/otosclerosis/ |
| **Schwartze Sign** | **Reddish hue** over promontory (active otosclerosis) | https://www.entnet.org/conditions/otosclerosis/ |
| **Audiogram** | **Conductive hearing loss**, **Carhart notch** (2 kHz bone conduction dip) | https://www.entnet.org/conditions/otosclerosis/ |
| **Treatment** | **Stapedectomy** or **Stapedotomy**, **Hearing aids** | https://www.entnet.org/conditions/otosclerosis/ |

## Vestibular Schwannoma (Acoustic Neuroma)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Benign tumor** of **CN VIII vestibular schwann cells** | https://www.entnet.org/conditions/acoustic-neuroma/ |
| **Presentation** | **Unilateral progressive sensorineural hearing loss**, **Tinnitus**, **Imbalance** | https://www.entnet.org/conditions/acoustic-neuroma/ |

## Meniere Disease
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Endolymphatic hydrops** causing episodic vertigo, hearing loss, tinnitus, aural fullness | https://www.entnet.org/conditions/menieres-disease/ |
| **Diagnostic Criteria** | **=2 episodes** of vertigo (20 min - 12h), **Audiometrically documented** sensorineural hearing loss, **Fluctuating symptoms** | https://www.entnet.org/conditions/menieres-disease/ |
| **Treatment (Acute)** | **Vestibular suppressants** (meclizine, Diazepam), **Antiemetics** | https://www.entnet.org/conditions/menieres-disease/ |
| **Treatment (Prophylaxis)** | **Low-sodium diet**, **Diuretics** (HCTZ/triamterene), **Intratympanic corticosteroids**, **Intratympanic gentamicin**, **Endolymphatic sac surgery**, **Vestibular neurectomy**, **Labyrinthectomy** | https://www.entnet.org/conditions/menieres-disease/ |

## Benign Paroxysmal Positional Vertigo (BPPV)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Otoconia displaced** into semicircular canal (usually posterior) | https://www.entnet.org/conditions/benign-paroxysmal-positional-vertigo/ |
| **Symptoms** | **Brief vertigo (<1 min)** triggered by **head position changes** (rolling in bed, looking up) | https://www.entnet.org/conditions/benign-paroxysmal-positional-vertigo/ |
| **Diagnosis** | **Positive Dix-Hallpike maneuver** (torsional upbeating nystagmus) | https://www.entnet.org/conditions/benign-paroxysmal-positional-vertigo/ |
| **Treatment** | **Canalith repositioning maneuvers** (Epley for posterior canal, Barbecue for horizontal) | https://www.entnet.org/conditions/benign-paroxysmal-positional-vertigo/ |
