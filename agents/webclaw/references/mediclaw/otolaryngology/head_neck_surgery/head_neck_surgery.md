## Thyroglossal Duct Cyst
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Remnant of thyroglossal duct** (thyroid descent) | https://www.entnet.org/conditions/thyroglossal-duct-cyst/ |
| **Presentation** | **Midline neck mass**, **Moves with swallowing and tongue protrusion** | https://www.entnet.org/conditions/thyroglossal-duct-cyst/ |
| **Diagnosis** | **Ultrasound** (confirm presence of normal thyroid) | https://www.entnet.org/conditions/thyroglossal-duct-cyst/ |
| **Treatment** | **Sistrunk procedure** (excision of cyst + tract + mid-portion of hyoid bone) | https://www.entnet.org/conditions/thyroglossal-duct-cyst/ |

## Oral Cavity Cancer
| Component | Description | URL |
|-----------|-------------|-----|
| **Risk Factors** | **Tobacco**, **Alcohol**, **Betel nut** | https://www.cancer.gov/types/head-and-neck |
| **Premalignant** | **Leukoplakia**, **Erythroplakia** (higher risk) | https://www.entnet.org/conditions/oral-cancer/ |
| **Common Sites** | **Tongue** (most common), **Floor of mouth**, **Buccal mucosa**, **Gingiva**, **Hard palate** | https://www.entnet.org/conditions/oral-cancer/ |
| **Treatment** | **Surgical resection** Ý **Neck dissection** Ý **Radiation** Ý **Chemotherapy** | https://www.nccn.org/guidelines/head-and-neck |

## Oropharyngeal Cancer

## Unknown Primary (Cervical Metastasis)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Cervical lymph node metastasis** with **no identifiable primary tumor** | https://www.entnet.org/conditions/unknown-primary/ |
| **Workup** | **PET-CT**, **Panendoscopy** with **biopsies of high-risk sites** (tonsil, Base of tongue, Nasopharynx, Pyriform sinus), **Ipsilateral tonsillectomy** | https://www.entnet.org/conditions/unknown-primary/ |
| **Treatment** | **Neck dissection** Ý **Radiation** to pharyngeal axis | https://www.nccn.org/guidelines/head-and-neck |
