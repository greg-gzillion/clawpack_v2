# Vascular Surgery - Lymphedema

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Accumulation of protein-rich fluid** in interstitial space due to **lymphatic dysfunction** | https://vascular.org/patients/vascular-conditions/lymphedema |
| **Primary Lymphedema** | **Congenital** (Milroy disease), **Praecox** (Meige disease, onset <35), **Tarda** (onset >35) | https://vascular.org/patients/vascular-conditions/lymphedema |
| **Secondary Lymphedema** | **Most common**: **Cancer treatment** (lymphadenectomy, Radiation), **Trauma**, **Infection** (filariasis), **Chronic venous insufficiency** | https://vascular.org/patients/vascular-conditions/lymphedema |

## Lymphedema vs Venous Edema
| Feature | Lymphedema | Venous Edema | URL |
|---------|------------|--------------|-----|
| **Onset** | **Gradual** | **Acute or chronic** | https://vascular.org/patients/vascular-conditions/lymphedema |
| **Pitting** | **Non-pitting** (late) | **Pitting** (early) | https://vascular.org/patients/vascular-conditions/lymphedema |
| **Skin Changes** | **Hyperkeratosis**, **Papillomatosis**, **Peau d'orange** | **Hemosiderin deposition**, **Lipodermatosclerosis** | https://vascular.org/patients/vascular-conditions/lymphedema |
| **Stemmer Sign** | **Positive** (inability to pinch dorsal foot skin) | **Negative** | https://vascular.org/patients/vascular-conditions/lymphedema |
| **Ulceration** | **Rare** | **Common** (gaiter area) | https://vascular.org/patients/vascular-conditions/lymphedema |

## Staging of Lymphedema (ISL)
| Stage | Description | URL |
|-------|-------------|-----|
| **0** | **Latent** (no visible edema, Impaired lymph transport) | https://vascular.org/patients/vascular-conditions/lymphedema |
| **I** | **Pitting edema**, **Reversible with elevation** | https://vascular.org/patients/vascular-conditions/lymphedema |
| **II** | **Non-pitting edema**, **Irreversible**, **Fibrosis** | https://vascular.org/patients/vascular-conditions/lymphedema |
| **III** | **Elephantiasis**, **Severe skin changes** (hyperkeratosis, Papillomatosis) | https://vascular.org/patients/vascular-conditions/lymphedema |

## Treatment of Lymphedema
| Modality | Description | URL |
|----------|-------------|-----|
| **Complete Decongestive Therapy (CDT)** | **Manual lymphatic drainage**, **Compression bandaging**, **Exercise**, **Skin care** (gold standard) | https://vascular.org/patients/vascular-conditions/lymphedema |
| **Compression Garments** | **20-50 mmHg** (maintenance phase) | https://vascular.org/patients/vascular-conditions/lymphedema |
| **Intermittent Pneumatic Compression (IPC)** | **Pump device** (adjunctive) | https://vascular.org/patients/vascular-conditions/lymphedema |
| **Surgical** | **Lymphovenous bypass**, **Vascularized lymph node transfer**, **Debulking** (refractory cases) | https://vascular.org/patients/vascular-conditions/lymphedema |
