# Vascular Surgery - Vascular Diagnostics

## Non-Invasive Vascular Testing
| Test | Measures | Indications | URL |
|------|----------|-------------|-----|
| **Ankle-Brachial Index (ABI)** | **Ratio of ankle to brachial systolic pressure** | **PAD screening**, **Severity assessment** | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **Toe-Brachial Index (TBI)** | **Ratio of toe to brachial pressure** | **PAD with calcified vessels** (diabetes, CKD) | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **Segmental Pressures** | **Pressures at multiple levels** | **Localize level of stenosis** | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **Pulse Volume Recording (PVR)** | **Volume change with each pulse** | **Assess perfusion**, **Not affected by calcification** | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **Duplex Ultrasound** | **B-mode + Doppler** | **Stenosis**, **Occlusion**, **Aneurysm**, **DVT**, **Venous reflux** | https://www.svunet.org/clinical-practice/guidelines |

## ABI Interpretation
| ABI Value | Interpretation | URL |
|-----------|----------------|-----|
| **>1.4** | **Non-compressible** (calcified vessels) | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **1.0-1.4** | **Normal** | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **0.90-0.99** | **Borderline PAD** | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **0.50-0.89** | **Moderate PAD** (claudication) | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **<0.50** | **Severe PAD** (rest pain, Tissue loss) | https://www.heart.org/en/health-topics/peripheral-artery-disease |

## Invasive Vascular Testing
| Test | Description | Indications | URL |
|------|-------------|-------------|-----|
| **Catheter Angiography** | **Contrast injection** with fluoroscopy | **Gold standard** for arterial anatomy, **Allows intervention** | https://www.radiologyinfo.org/en/info/angiocath |
| **CT Angiography (CTA)** | **CT with IV contrast** | **Aneurysm**, **Dissection**, **PAD**, **Pre-operative planning** | https://www.radiologyinfo.org/en/info/angioct |
| **MR Angiography (MRA)** | **MRI with/without contrast** | **PAD**, **Renal artery stenosis**, **Avoids radiation** | https://www.radiologyinfo.org/en/info/angiomr |
| **Intravascular Ultrasound (IVUS)** | **Catheter-based ultrasound** | **Precise vessel measurements**, **Assess stent deployment**, **Dissection** | https://www.radiologyinfo.org/en/info/ivus |
