## Vascular Malformations
| Type | Description | Treatment | URL |
|------|-------------|-----------|-----|
| **Capillary Malformation** | **Port-wine stain** | **Pulsed dye laser** | https://www.issva.org/classification |
| **Venous Malformation** | **Soft, compressible**, **Bluish**, **Enlarges with dependency/Valsalva** | **Compression**, **Sclerotherapy**, **Surgical resection** | https://www.issva.org/classification |
| **Lymphatic Malformation** | **Macrocystic** or **Microcystic** | **Sclerotherapy** (doxycycline, OK-432), **Sirolimus**, **Surgical resection** | https://www.issva.org/classification |
| **Arteriovenous Malformation (AVM)** | **Warm**, **Palpable thrill/bruit**, **Schobinger staging** (I-IV) | **Embolization** ± **Surgical resection** | https://www.issva.org/classification |

## Schobinger Staging of AVM
| Stage | Description | URL |
|-------|-------------|-----|
| **I (Quiescence)** | **Warm**, **Pink-bluish stain**, **AV shunting on Doppler** | https://www.issva.org/classification |
| **II (Expansion)** | **Enlargement**, **Pulsations**, **Thrill**, **Bruit**, **Tortuous veins** | https://www.issva.org/classification |
| **III (Destruction)** | **Dystrophic skin changes**, **Ulceration**, **Bleeding**, **Pain** | https://www.issva.org/classification |
| **IV (Decompensation)** | **High-output cardiac failure** | https://www.issva.org/classification |
