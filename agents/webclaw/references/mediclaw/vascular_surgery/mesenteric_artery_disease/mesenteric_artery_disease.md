## Chronic Mesenteric Ischemia (CMI)
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathophysiology** | **=2 of 3 vessels** severely stenosed (collaterals usually compensate for single vessel disease) | https://vascular.org/patients/vascular-conditions/mesenteric-artery-disease |
| **Classic Triad** | **Postprandial abdominal pain**, **Weight loss**, **Food fear** | https://vascular.org/patients/vascular-conditions/mesenteric-artery-disease |
| **Diagnosis** | **Duplex ultrasound** (PSV >275 cm/s in SMA, >200 cm/s in celiac), **CTA** (definitive) | https://vascular.org/patients/vascular-conditions/mesenteric-artery-disease |
| **Treatment** | **Revascularization** (endovascular first-line: SMA stenting; Open bypass: SMA bypass) | https://vascular.org/patients/vascular-conditions/mesenteric-artery-disease |

## Acute Mesenteric Ischemia (AMI)
| Component | Description | URL |
|-----------|-------------|-----|
| **Etiology** | **Embolism** (50% - cardiac), **Thrombosis** (25% - underlying stenosis), **NOMI** (20% - vasospasm), **Venous** (5%) | https://vascular.org/patients/vascular-conditions/acute-mesenteric-ischemia |
| **Presentation** | **Pain out of proportion to exam** (early), **Peritonitis** (late - bowel necrosis) | https://vascular.org/patients/vascular-conditions/acute-mesenteric-ischemia |
| **Labs** | **Leukocytosis**, **? Lactate** (late), **Metabolic acidosis** | https://vascular.org/patients/vascular-conditions/acute-mesenteric-ischemia |
| **Imaging** | **CTA** (gold standard) | https://vascular.org/patients/vascular-conditions/acute-mesenteric-ischemia |
| **Treatment** | **Emergent revascularization** + **Bowel resection** (if necrosis) | https://vascular.org/patients/vascular-conditions/acute-mesenteric-ischemia |

## Non-Occlusive Mesenteric Ischemia (NOMI)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Mesenteric ischemia without arterial occlusion** (vasospasm) | https://vascular.org/patients/vascular-conditions/acute-mesenteric-ischemia |
| **Risk Factors** | **Critical illness**, **Shock**, **Vasopressors**, **Cardiac surgery**, **Dialysis** | https://vascular.org/patients/vascular-conditions/acute-mesenteric-ischemia |
| **Diagnosis** | **Angiography** (diffuse vasospasm, "pruned tree" appearance) | https://vascular.org/patients/vascular-conditions/acute-mesenteric-ischemia |
| **Treatment** | **Intra-arterial vasodilators** (papaverine), **Treat underlying cause** | https://vascular.org/patients/vascular-conditions/acute-mesenteric-ischemia |
