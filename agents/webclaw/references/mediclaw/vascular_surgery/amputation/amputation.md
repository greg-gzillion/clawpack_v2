# Vascular Surgery - Amputation

## Indications for Amputation
| Indication | Description | URL |
|------------|-------------|-----|
| **Irreversible Ischemia** | **Non-reconstructable PAD**, **Failed revascularization**, **Acute limb ischemia with non-viable tissue** | https://vascular.org/patients/vascular-conditions/amputation |
| **Uncontrolled Infection** | **Gas gangrene**, **Necrotizing fasciitis**, **Osteomyelitis** not amenable to debridement | https://vascular.org/patients/vascular-conditions/amputation |
| **Trauma** | **Mangled extremity** (MESS score =7) | https://vascular.org/patients/vascular-conditions/amputation |
| **Tumor** | **Primary bone/soft tissue malignancy** | https://vascular.org/patients/vascular-conditions/amputation |
| **Non-Functional Limb** | **Severe contracture**, **Paralysis**, **Chronic pain** | https://vascular.org/patients/vascular-conditions/amputation |

## Amputation Principles
| Principle | Description | URL |
|----------|-------------|-----|
| **Preoperative Assessment** | **Vascular evaluation** (optimize inflow), **Nutritional status** (albumin >3.5), **Infection control** | https://vascular.org/patients/vascular-conditions/amputation |
| **Skin Flaps** | **Posterior flap** (BKA - long posterior), **Equal flaps** (AKA - fishmouth), **Avoid tension** | https://vascular.org/patients/vascular-conditions/amputation |
| **Myodesis/Myoplasty** | **Stabilize muscle**, **Improve prosthetic function** | https://vascular.org/patients/vascular-conditions/amputation |
| **Nerve Management** | **Sharp transection under tension**, **Allow retraction** (prevent neuroma) | https://vascular.org/patients/vascular-conditions/amputation |
| **Bone Beveling** | **Smooth edges**, **Remove sharp prominences** | https://vascular.org/patients/vascular-conditions/amputation |
| **Hemostasis** | **Meticulous**, **Avoid hematoma** | https://vascular.org/patients/vascular-conditions/amputation |
| **Postoperative** | **Rigid dressing** (prevents edema, Contracture), **Early prosthetic fitting** | https://vascular.org/patients/vascular-conditions/amputation |
