# Vascular Surgery - Thoracic Outlet Syndrome (TOS)

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Compression of neurovascular bundle** (brachial plexus, subclavian artery/vein) at thoracic outlet | https://vascular.org/patients/vascular-conditions/thoracic-outlet-syndrome |
| **Types** | **Neurogenic** (95%), **Venous** (4%), **Arterial** (1%) | https://vascular.org/patients/vascular-conditions/thoracic-outlet-syndrome |
| **Compression Sites** | **Interscalene triangle**, **Costoclavicular space**, **Subcoracoid space** (pectoralis minor) | https://vascular.org/patients/vascular-conditions/thoracic-outlet-syndrome |

## Neurogenic TOS (nTOS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Presentation** | **Pain**, **Paresthesias** (C8-T1 distribution - ulnar forearm/hand), **Weakness**, **Neck/shoulder pain** | https://vascular.org/patients/vascular-conditions/thoracic-outlet-syndrome |
| **Provocative Tests** | **Elevated Arm Stress Test (EAST)** , **Adson**, **Wright**, **Roos** (low specificity) | https://vascular.org/patients/vascular-conditions/thoracic-outlet-syndrome |
| **Diagnosis** | **Clinical** (diagnosis of exclusion), **EMG/NCS** (rule out other causes), **Scalene block** (diagnostic) | https://vascular.org/patients/vascular-conditions/thoracic-outlet-syndrome |
| **Treatment** | **Physical therapy** (first-line), **Scalenectomy** ｱ **First rib resection** (refractory) | https://vascular.org/patients/vascular-conditions/thoracic-outlet-syndrome |

## Venous TOS (vTOS) - Paget-Schroetter Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Effort thrombosis** of **subclavian vein** (repetitive arm motion) | https://vascular.org/patients/vascular-conditions/venous-thoracic-outlet-syndrome |
| **Presentation** | **Acute arm swelling**, **Cyanosis**, **Pain**, **Visible collateral veins** (Urschel sign) | https://vascular.org/patients/vascular-conditions/venous-thoracic-outlet-syndrome |
| **Diagnosis** | **Duplex ultrasound**, **Venography** (gold standard) | https://vascular.org/patients/vascular-conditions/venous-thoracic-outlet-syndrome |
| **Treatment** | **Catheter-directed thrombolysis** ? **First rib resection** (within weeks) | https://vascular.org/patients/vascular-conditions/venous-thoracic-outlet-syndrome |

## Arterial TOS (aTOS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Presentation** | **Arm claudication**, **Pallor**, **Cold intolerance**, **Digital ischemia/embolization**, **Subclavian artery aneurysm** | https://vascular.org/patients/vascular-conditions/arterial-thoracic-outlet-syndrome |
| **Pathology** | **Cervical rib** or **Anomalous first rib** ? **Subclavian artery compression** ? **Post-stenotic dilation/Aneurysm** ? **Thrombus/Embolism** | https://vascular.org/patients/vascular-conditions/arterial-thoracic-outlet-syndrome |
| **Diagnosis** | **Duplex ultrasound** (with provocative maneuvers), **CTA/MRA** | https://vascular.org/patients/vascular-conditions/arterial-thoracic-outlet-syndrome |
| **Treatment** | **Cervical/first rib resection** + **Arterial reconstruction** (if aneurysm) ｱ **Distal embolectomy** | https://vascular.org/patients/vascular-conditions/arterial-thoracic-outlet-syndrome |
