# Vascular Surgery - Aortic Dissection

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Tear in aortic intima** ? **Blood enters media** ? **Creates true and false lumen** | https://vascular.org/patients/vascular-conditions/aortic-dissection |
| **Incidence** | **3-4 per 100,000** person-years | https://www.cdc.gov/heartdisease/aortic_dissection.htm |
| **Risk Factors** | **Hypertension** (most common), **Connective tissue disorders** (Marfan, Loeys-Dietz, Ehlers-Danlos), **Bicuspid aortic valve**, **Aortitis**, **Trauma**, **Cocaine** | https://vascular.org/patients/vascular-conditions/aortic-dissection |

## Stanford Classification
| Type | Location | Management | URL |
|------|----------|------------|-----|
| **Type A** | **Involves ascending aorta** (regardless of origin) | **Emergent surgical repair** | https://vascular.org/patients/vascular-conditions/aortic-dissection |
| **Type B** | **Does NOT involve ascending aorta** | **Medical management** (unless complicated) | https://vascular.org/patients/vascular-conditions/aortic-dissection |

## Clinical Presentation
| Feature | Type A | Type B | URL |
|---------|--------|--------|-----|

## Complicated Type B Dissection
| Complication | Definition | Management | URL |
|--------------|------------|------------|-----|
| **Malperfusion** | **End-organ ischemia** (renal, mesenteric, limb) | **TEVAR** ± **Fenestration/Stenting** | https://vascular.org/patients/vascular-conditions/aortic-dissection |
| **Rupture/Impending Rupture** | **Hemothorax**, **Hemodynamic instability** | **Emergent TEVAR** | https://vascular.org/patients/vascular-conditions/aortic-dissection |
| **Refractory Hypertension** | **Uncontrolled BP despite maximal medical therapy** | **TEVAR** | https://vascular.org/patients/vascular-conditions/aortic-dissection |
| **Refractory Pain** | **Persistent pain despite medical therapy** | **TEVAR** | https://vascular.org/patients/vascular-conditions/aortic-dissection |
| **Rapid Aortic Expansion** | **>10 mm/year** | **TEVAR** | https://vascular.org/patients/vascular-conditions/aortic-dissection |

## Medical Management of Type B Dissection
| Component | Target | URL |
|-----------|--------|-----|
| **Heart Rate** | **<60 bpm** | https://vascular.org/patients/vascular-conditions/aortic-dissection |
| **Blood Pressure** | **SBP 100-120 mmHg** (lowest tolerated without malperfusion) | https://vascular.org/patients/vascular-conditions/aortic-dissection |
| **First-Line** | **Esmolol** or **Labetalol** IV | https://vascular.org/patients/vascular-conditions/aortic-dissection |
| **Second-Line** | **Nicardipine**, **Nitroprusside** (after beta-blockade) | https://vascular.org/patients/vascular-conditions/aortic-dissection |
| **Long-Term** | **Beta-blocker** + **ARB/ACEi** | https://vascular.org/patients/vascular-conditions/aortic-dissection |
