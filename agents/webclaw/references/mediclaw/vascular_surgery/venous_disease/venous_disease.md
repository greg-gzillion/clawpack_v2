# Vascular Surgery - Venous Disease

## Chronic Venous Insufficiency (CVI)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Impaired venous return** due to **valvular incompetence** or **obstruction** | https://vascular.org/patients/vascular-conditions/chronic-venous-insufficiency |
| **Pathophysiology** | **Venous reflux** ? **Venous hypertension** ? **Edema**, **Skin changes**, **Ulceration** | https://vascular.org/patients/vascular-conditions/chronic-venous-insufficiency |
| **Risk Factors** | **Age**, **Female sex**, **Obesity**, **Pregnancy**, **Prolonged standing**, **Family history**, **Prior DVT** | https://vascular.org/patients/vascular-conditions/chronic-venous-insufficiency |

## Varicose Veins
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Dilated, tortuous superficial veins** =3 mm diameter | https://vascular.org/patients/vascular-conditions/varicose-veins |
| **Symptoms** | **Aching**, **Heaviness**, **Itching**, **Swelling**, **Cramping** (worse with prolonged standing, better with elevation) | https://vascular.org/patients/vascular-conditions/varicose-veins |
| **Diagnosis** | **Duplex ultrasound** (assess reflux, Rule out DVT) | https://vascular.org/patients/vascular-conditions/varicose-veins |
| **Conservative Treatment** | **Compression stockings** (20-30 mmHg), **Elevation**, **Exercise**, **Weight loss** | https://vascular.org/patients/vascular-conditions/varicose-veins |
| **Interventional Treatment** | **Endovenous ablation** (laser or radiofrequency), **Sclerotherapy**, **Phlebectomy** | https://vascular.org/patients/vascular-conditions/varicose-veins |

## Venous Ulcer
| Component | Description | URL |
|-----------|-------------|-----|
| **Location** | **Gaiter area** (medial malleolus most common) | https://vascular.org/patients/vascular-conditions/venous-ulcer |
| **Appearance** | **Irregular borders**, **Shallow**, **Granulating base**, **Surrounding skin changes** (pigmentation, Lipodermatosclerosis) | https://vascular.org/patients/vascular-conditions/venous-ulcer |
| **Treatment** | **Compression therapy** (multilayer), **Wound care**, **Treat superficial reflux** (improves healing, Reduces recurrence) | https://vascular.org/patients/vascular-conditions/venous-ulcer |
