# Hematology - Anemia

## Anemia - Definition and Classification
| Parameter | Definition | URL |
|-----------|------------|-----|
| **Anemia (WHO)** | Hb <13 g/dL (men), <12 g/dL (women), <11 g/dL (pregnant) | https://www.who.int/health-topics/anaemia |
| **MCV Classification** | Microcytic (<80 fL), Normocytic (80-100 fL), Macrocytic (>100 fL) | |
| **Reticulocyte Classification** | Hypoproliferative (RI <2), Hyperproliferative (RI >2) | |

## Microcytic Anemia - Differential Diagnosis
| Cause | MCV | Iron Studies | Other Features | URL |
|-------|-----|--------------|----------------|-----|
| **Iron Deficiency Anemia (IDA)** | Low | Low Ferritin, High TIBC, Low Iron | Most common cause worldwide | https://www.ash.org/iron-deficiency |
| **Anemia of Chronic Disease (ACD)** | Normal/Low | High Ferritin, Low TIBC, Low Iron | Underlying inflammation/malignancy | |
| **Thalassemia** | Very Low | Normal ferritin, Normal iron | Elevated RBC count, Target cells | |
| **Sideroblastic Anemia** | Variable | High Ferritin, High Iron | Ring sideroblasts on bone marrow | |
| **Lead Poisoning** | Low | Normal | Basophilic stippling | https://www.cdc.gov/lead |

## Macrocytic Anemia - Differential Diagnosis
| Cause | MCV | Diagnostic Test | URL |
|-------|-----|-----------------|-----|
| **Vitamin B12 Deficiency** | >110 fL | Low B12, High MMA, High Homocysteine | https://www.nhlbi.nih.gov/anemia |
| **Folate Deficiency** | >110 fL | Low Folate, Normal MMA, High Homocysteine | |
| **Alcohol Use** | 100-110 fL | History, LFTs | |
| **Liver Disease** | 100-110 fL | LFTs, Target cells | |
| **Hypothyroidism** | 90-110 fL | High TSH | |
| **Myelodysplastic Syndrome (MDS)** | Variable | Bone marrow biopsy, Cytogenetics | |

## Iron Deficiency Anemia (IDA) - Lab Findings
| Test | IDA | ACD | URL |
|------|-----|-----|-----|
| Serum Iron | Low | Low | https://www.ash.org/iron-deficiency |
| TIBC | High | Low | |
| Transferrin Saturation | <15% | >15% | |
| Ferritin | **<30 ng/mL** | **>100 ng/mL** | |

## IDA - Management
| Cause | Evaluation | Treatment | URL |
|-------|------------|-----------|-----|
| **GI Blood Loss** | EGD/Colonoscopy | Treat cause; Iron supplementation | https://gi.org |
| **Menorrhagia** | Pelvic US | Gynecologic evaluation; Iron | |
| **Malabsorption** | Celiac serologies | IV iron | |
| **Oral Iron** | Ferrous sulfate 325 mg daily | Take with vitamin C | |
| **IV Iron** | Iron sucrose, Ferric carboxymaltose | For intolerance, Malabsorption | https://www.kidney.org |

## Vitamin B12 Deficiency
| Component | Description | URL |
|-----------|-------------|-----|
| Causes | Pernicious anemia, PPI use, Metformin, Vegan diet, Gastric bypass | https://www.nhlbi.nih.gov/anemia |
| Clinical | Macrocytic anemia, Peripheral neuropathy, Subacute combined degeneration, Glossitis | |
| Diagnosis | B12 <200 pg/mL; Borderline ? Check MMA, Homocysteine | |
| Treatment | IM B12 1000 mcg monthly or High-dose oral B12 1000-2000 mcg daily | |

## Folate Deficiency
| Component | Description | URL |
|-----------|-------------|-----|
| Causes | Dietary inadequacy, Alcoholism, Malabsorption, Pregnancy, Methotrexate | https://www.nhlbi.nih.gov/anemia |
| Diagnosis | Serum folate <2 ng/mL or RBC folate <140 ng/mL | |
| Treatment | Folic acid 1 mg daily | |
