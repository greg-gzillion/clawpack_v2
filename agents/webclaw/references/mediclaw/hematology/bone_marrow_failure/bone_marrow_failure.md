# Hematology - Bone Marrow Failure Syndromes

## Aplastic Anemia (AA)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | **Pancytopenia** with **hypocellular bone marrow** (<30% cellularity) | https://www.aamds.org |
| Pathophysiology | **Immune-mediated destruction** of hematopoietic stem cells (T-cell mediated) | |
| Causes | **Idiopathic** (70%), **Drugs** (chloramphenicol, NSAIDs, Anticonvulsants), **Chemicals** (benzene), **Viral** (hepatitis, EBV, HIV, Parvovirus), **Inherited** (Fanconi anemia, Dyskeratosis congenita), **PNH**, **Pregnancy** | |
| Diagnostic Criteria (Camitta) | **Severe AA**: Bone marrow cellularity <25% + =2 of: ANC <500/µL, Platelets <20,000/µL, Reticulocytes <20,000/µL; **Very Severe AA**: ANC <200/µL | |
| Evaluation | **CBC**, **Reticulocyte count**, **Bone marrow biopsy**, **Cytogenetics**, **Chromosomal breakage studies** (Fanconi), **Telomere length** (Dyskeratosis), **PNH flow cytometry**, **Viral serologies** | |

## Aplastic Anemia - Management
| Treatment | Indications | Description | URL |
|-----------|-------------|-------------|-----|
| **Allogeneic HSCT** | **Age <40 with matched sibling donor** | **Curative**; Younger patients, Matched sibling donor | https://www.aamds.org |
| **Immunosuppressive Therapy (IST)** | Age >40, No donor, Not HSCT candidate | **ATG (horse) + Cyclosporine + Eltrombopag** | |
| **Eltrombopag** | Add to IST, Refractory AA | **Thrombopoietin receptor agonist**; Stimulates trilineage hematopoiesis | |
| **Supportive Care** | All patients | **Transfusions** (leukoreduced, Irradiated if HSCT candidate), **Antibiotics** for neutropenic fever, **Iron chelation** | |
| **Response to IST** | 60-70% response at 3-6 months | Monitor blood counts; Taper cyclosporine slowly over 1-2 years | |

## Pure Red Cell Aplasia (PRCA)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Isolated **anemia** with **reticulocytopenia** and **absent erythroid precursors** on bone marrow (normal myeloid and megakaryocytic lineages) | https://www.ash.org/PRCA |
| Causes | **Parvovirus B19** (immunocompromised, Sickle cell), **Thymoma** (10-15%), **Lymphoproliferative disorders** (CLL, LGL leukemia), **Autoimmune** (SLE, RA), **Medications** (EPO, Azathioprine, Phenytoin), **ABO-incompatible HSCT** | |
| Diagnosis | Bone marrow biopsy (absent erythroid precursors), Parvovirus PCR, CT chest (thymoma), LGL panel | |
| Management | **Treat underlying cause**; **IVIG** (Parvovirus, Immunocompromised); **Immunosuppression** (Prednisone ± Cyclosporine); **Thymectomy** (if thymoma) | |

## Fanconi Anemia (FA)
| Component | Description | URL |
|-----------|-------------|-----|
| Genetics | **Autosomal recessive** (most); X-linked (FANCB); Mutations in **FANC genes** (23 genes) ? Defective DNA repair | https://www.fanconi.org |
| Clinical Features | **Congenital anomalies** (radial anomalies, Short stature, Microcephaly, Café-au-lait spots, Renal anomalies), **Progressive bone marrow failure** (median age 7 years), **Cancer predisposition** (AML, MDS, SCC of head/neck, Anogenital) | |
| Diagnosis | **Chromosomal breakage study** (diepoxybutane/DEB or mitomycin C/MMC) - **increased chromosomal breaks** | |
| Management | **Androgens** (oxymetholone) - temporary response; **Allogeneic HSCT** (curative for bone marrow failure); **Cancer surveillance** | |

## Dyskeratosis Congenita (DC)
| Component | Description | URL |
|-----------|-------------|-----|
| Genetics | Mutations in telomere maintenance genes (DKC1, TERC, TERT, TINF2, etc.) ? **Short telomeres** | https://www.dyskeratosiscongenita.org |
| Inheritance | X-linked (most common), Autosomal dominant, Autosomal recessive | |
| Clinical Triad | **Oral leukoplakia**, **Nail dystrophy**, **Reticulate skin pigmentation** | |
| Other Features | Bone marrow failure, Pulmonary fibrosis, Liver cirrhosis, Cancer predisposition (SCC, AML/MDS) | |
| Diagnosis | **Telomere length testing** (Flow FISH) - **very short telomeres** (<1st percentile) | |
| Management | **Androgens** (danazol) - improves cytopenias; **Allogeneic HSCT** (curative for bone marrow failure); **Avoid smoking, Sun exposure** | |
