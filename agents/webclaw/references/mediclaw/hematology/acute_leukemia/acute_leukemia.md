# Hematology - Acute Leukemia

## Acute Leukemia - Overview
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Clonal proliferation of immature hematopoietic cells (blasts) with accumulation in bone marrow, Peripheral blood, and extramedullary sites | https://www.lls.org/leukemia |
| Classification | **Acute Myeloid Leukemia (AML)** - 80% of adult acute leukemias; **Acute Lymphoblastic Leukemia (ALL)** - 20% (more common in children) | |
| Diagnosis | **=20% blasts** in bone marrow or peripheral blood (or presence of specific genetic abnormalities regardless of blast count) | |
| Clinical Presentation | **Bone marrow failure** (anemia, Neutropenia, Thrombocytopenia), **Extramedullary disease** (CNS, Skin - leukemia cutis, Gums - monocytic), **Tumor lysis syndrome** | |

## AML - Treatment Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Induction Chemotherapy** | **"7+3"** (Cytarabine x 7 days + Daunorubicin/Idarubicin x 3 days) for fit patients | https://www.nccn.org/aml |
| **Targeted Therapy** | **Midostaurin, Quizartinib** (FLT3-mutated), **Gemtuzumab ozogamicin** (CD33+), **CPX-351** (t-AML or AML-MRC), **Venetoclax + Azacitidine** (unfit/elderly) | |
| **Consolidation** | **High-dose cytarabine (HiDAC)** for favorable/intermediate risk; **Allogeneic HSCT** for adverse risk or intermediate risk with MRD+ | |
| **Maintenance** | **Oral azacitidine (CC-486)** for AML in remission not proceeding to HSCT | |
| **Acute Promyelocytic Leukemia (APL)** | **ATRA + Arsenic trioxide** (low/intermediate risk); **ATRA + Chemotherapy** (high risk); **Coagulopathy management** (cryoprecipitate, Platelets) | |

## Acute Lymphoblastic Leukemia (ALL) - Overview
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | Bimodal distribution (children 2-5 years, Adults >50 years) | https://www.lls.org/leukemia/acute-lymphoblastic-leukemia |
| Immunophenotype | **B-ALL** (80-85%), **T-ALL** (15-20%) | |
| Cytogenetics | **Hyperdiploidy** (>50 chromosomes - favorable), **Hypodiploidy** (<44 chromosomes - adverse), **t(9;22)/BCR-ABL1** (Ph+ ALL), **t(12;21)/ETV6-RUNX1**, **t(1;19)/TCF3-PBX1**, **KMT2A rearrangements** | |

## ALL - Treatment Overview
| Phase | Duration | Description | URL |
|-------|----------|-------------|-----|
| **Induction** | 4-6 weeks | Multi-agent chemotherapy (vincristine, Steroids, Anthracycline, Asparaginase); Goal: **CR** (>90% in children, 80-90% in adults) | https://www.nccn.org/all |
| **Consolidation** | 4-6 months | Multi-agent chemotherapy; **CNS prophylaxis** (intrathecal chemotherapy) | |
| **Maintenance** | 2-3 years | **6-MP daily + Methotrexate weekly** + Vincristine/Steroid pulses | |
| **Allogeneic HSCT** | High-risk features | **Ph+ ALL**, **High-risk genetics**, **Persistent MRD**, **Relapsed/refractory** | |
| **Targeted Therapy** | Ph+ ALL | **Tyrosine kinase inhibitors** (Imatinib, Dasatinib, Ponatinib) + Chemotherapy or **Blinatumomab** (chemo-free) | |

## Tumor Lysis Syndrome (TLS)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Metabolic derangement from rapid tumor cell lysis ? Hyperuricemia, Hyperkalemia, Hyperphosphatemia, Hypocalcemia | https://www.ash.org/TLS |
| Risk Factors | **High tumor burden** (WBC >50-100k, LDH >2x ULN, Bulky disease), **Highly chemosensitive tumors** (Burkitt, ALL, AML), **Pre-existing renal impairment**, **Dehydration** | |
| Prophylaxis | **Aggressive hydration**, **Allopurinol** (low/intermediate risk), **Rasburicase** (high risk) | |
| Monitoring | Labs q4-6h (K+, Ca2+, PO4, Cr, Uric acid), Cardiac monitoring, Strict I/Os | |
| Treatment | **Rasburicase** (if uric acid >8 or rapidly rising), **Hydration**, **Manage hyperkalemia** (calcium gluconate, Insulin/glucose, Kayexalate), **Hyperphosphatemia** (phosphate binders), **Dialysis** if refractory | |
