# Hematology - Hematopoietic Stem Cell Transplantation (HSCT)

## Types of HSCT
| Type | Donor | Graft Source | Advantages | Disadvantages | URL |
|------|-------|--------------|------------|---------------|-----|
| **Autologous** | Self | Peripheral blood stem cells (PBSC), Bone marrow | **No GVHD**, **Lower TRM** | **No GVL effect**, **Higher relapse**, Risk of tumor contamination | https://www.cibmtr.org |
| **Allogeneic (Matched Sibling)** | HLA-identical sibling | PBSC, Bone marrow | **GVL effect**, **Lower relapse**, No tumor contamination | **GVHD**, **Higher TRM**, Donor availability (25%) | |
| **Allogeneic (Matched Unrelated)** | HLA-matched unrelated donor (MUD) | PBSC, Bone marrow | Available for most patients | **Higher GVHD**, **Higher TRM**, Longer time to transplant | |
| **Allogeneic (Haploidentical)** | HLA-half-matched family member | Bone marrow, PBSC | **Universal donor availability** | **Higher GVHD** (mitigated by post-transplant Cy) | |
| **Allogeneic (Umbilical Cord Blood)** | Cord blood unit | Umbilical cord blood | **Less stringent HLA matching**, **Lower GVHD**, Immediate availability | **Slower engraftment**, **Higher infection risk**, Limited cell dose | |

## Indications for HSCT
| Disease | Allogeneic | Autologous | URL |
|---------|------------|------------|-----|
| **AML** | **High-risk CR1**, **Relapsed/refractory** | **Consolidation** (select cases) | https://www.nccn.org |
| **ALL** | **High-risk CR1**, **Ph+ ALL**, **Relapsed/refractory** | Not indicated | |
| **MDS** | **High-risk**, **Intermediate-risk with poor features** | Not indicated | |
| **CML** | **TKI failure**, **Accelerated/Blast phase** | Not indicated | |
| **Myelofibrosis** | **Intermediate-2/High risk** (curative) | Not indicated | |
| **Lymphoma (DLBCL)** | **Relapsed/refractory** (after salvage) | **Relapsed/refractory** (chemosensitive) | |
| **Multiple Myeloma** | **High-risk** (selected patients) | **Standard of care** (first-line consolidation) | |
| **Aplastic Anemia** | **Severe AA** (age <40, Matched sibling) | Not indicated | |
| **Sickle Cell Disease** | **Severe disease** with matched sibling | **Gene therapy** | |

## HLA Matching
| Relationship | HLA Match | Probability | URL |
|--------------|-----------|-------------|-----|
| **Sibling** | 10/10 (HLA-A, B, C, DRB1, DQB1) | 25% | https://bethematch.org |
| **Parent/Child** | **Haploidentical** (5/10) | 100% | |
| **Unrelated Donor** | 8/8 or 10/10 | Varies by ethnicity | |
| **Mismatched Unrelated** | 7/8 or 9/10 | Acceptable if no 8/8 | |
| **Cord Blood** | =4/6 (HLA-A, B, DRB1) | | |

## Conditioning Regimens
| Intensity | Regimens | Purpose | Toxicity | URL |
|-----------|----------|---------|----------|-----|
| **Myeloablative (MAC)** | **Cy/TBI**, **Bu/Cy**, **Flu/Bu** | **Maximize tumor kill**, Create space for engraftment | **High TRM** (10-30%), Infertility, Mucositis, VOD/SOS | https://www.cibmtr.org |
| **Reduced Intensity (RIC)** | **Flu/Mel**, **Flu/Bu**, **Flu/Cy/TBI** | **Lower TRM**, Allow GVL effect | **Higher relapse**, Less mucositis, Preserves fertility (some) | |
| **Non-Myeloablative (NMA)** | **Flu/Cy**, **Flu/TBI (2 Gy)** | **Minimal toxicity**, Relies on GVL | **Highest relapse**, Outpatient possible | |

## Engraftment
| Cell Type | Time to Engraftment | Definition | URL |
|-----------|---------------------|------------|-----|
| **Neutrophils** | **Day 14-28** (PBSC), **Day 21-35** (BM), **Day 30-60** (Cord) | **ANC =500/µL x 3 consecutive days** | https://www.cibmtr.org |
| **Platelets** | **Day 14-40** | **Plt =20k x 7 days without transfusion** | |
| **Graft Failure** | Failure to achieve ANC >500 by day 28 (primary) | **Second HSCT**, **Donor lymphocyte infusion (DLI)** | |

## Graft-versus-Host Disease (GVHD)
| Feature | Acute GVHD | Chronic GVHD | URL |
|---------|------------|--------------|-----|
| **Timing** | <100 days post-HSCT | >100 days post-HSCT | https://www.cibmtr.org/GVHD |
| **Target Organs** | **Skin** (rash), **GI** (diarrhea, Nausea, Anorexia), **Liver** (cholestasis) | **Skin** (scleroderma, Lichen planus), **GI** (dysphagia, Strictures), **Liver**, **Lung** (BOS), **Eyes** (dry eyes), **Joints** (fasciitis) | |
| **Prophylaxis** | **Tacrolimus + Methotrexate**; **Post-transplant cyclophosphamide** (haploidentical) | Taper immunosuppression slowly | |
| **First-Line Treatment** | **Corticosteroids** (methylprednisolone 1-2 mg/kg/day) | **Corticosteroids** | |
| **Second-Line** | **Ruxolitinib**, **Extracorporeal photopheresis (ECP)**, **Anti-thymocyte globulin (ATG)**, **Sirolimus**, **Mycophenolate** | **Ruxolitinib**, **Ibrutinib**, **Belumosudil**, **ECP**, **Rituximab** | |

## Complications of HSCT
| Complication | Timing | Management | URL |
|--------------|--------|------------|-----|
| **Mucositis** | Days 7-14 | Pain control, **Palifermin** (keratinocyte growth factor), Nutritional support | https://www.cibmtr.org |
| **Veno-occlusive Disease/Sinusoidal Obstruction Syndrome (VOD/SOS)** | Days 10-30 | **Weight gain**, **Ascites**, **Painful hepatomegaly**, **Hyperbilirubinemia**; **Defibrotide** | |
| **Hemorrhagic Cystitis** | Days 30-90 | **BK virus**, **Adenovirus**, **Cyclophosphamide**; Hydration, Bladder irrigation, **Cidofovir** | |
| **CMV Reactivation** | Days 30-100 | **Weekly CMV PCR**; **Valganciclovir**, **Letermovir** (prophylaxis) | |
| **EBV Post-Transplant Lymphoproliferative Disorder (PTLD)** | Months 2-6 | **Rituximab**, Reduce immunosuppression | |
| **Idiopathic Pneumonia Syndrome (IPS)** | Days 14-90 | Diffuse alveolar damage; **Etanercept**, Steroids | |
| **Graft Failure** | Variable | Second HSCT, DLI | |
| **Secondary Malignancy** | Years | **MDS/AML**, Solid tumors; Lifelong surveillance | |
