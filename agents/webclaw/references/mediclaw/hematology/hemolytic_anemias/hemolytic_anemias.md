## Glucose-6-Phosphate Dehydrogenase (G6PD) Deficiency
| Component | Description | URL |
|-----------|-------------|-----|
| Inheritance | **X-linked recessive** | https://www.cdc.gov/ncbddd/g6pd |
| Epidemiology | 400 million worldwide; Mediterranean, African, Asian descent | |
| Pathophysiology | Defective pentose phosphate pathway ? ? NADPH ? Oxidative stress ? Hemolysis | |
| Triggers | **Infections** (most common), **Fava beans**, **Medications** (sulfa, Dapsone, Primaquine, Rasburicase, Nitrofurantoin, Methylene blue) | |
| Clinical Features | Acute hemolytic anemia (2-4 days after trigger), **Hemoglobinuria**, Back pain, Jaundice | |
| Diagnosis | **G6PD enzyme assay** (do NOT test during acute hemolysis - false normal due to reticulocytosis); **Peripheral smear**: Bite cells, Blister cells, Heinz bodies (supravital stain) | |
| Management | **Avoid triggers**, Supportive care (hydration), Transfusion if severe | |

## Warm AIHA - Management
| Component | Description | URL |
|-----------|-------------|-----|
| Causes | **Idiopathic** (50%), **Secondary** (CLL, Lymphoma, SLE, Medications) | https://www.ash.org/AIHA |
| First-Line | **Prednisone 1 mg/kg/day** (taper over 3-6 months); **Rituximab** (can be used upfront with steroids) | |
| Second-Line | **Rituximab** (if not used first-line); **Splenectomy** | |
| Third-Line | **Immunosuppressants** (Mycophenolate, Azathioprine, Cyclosporine); **Danazol** | |
| Transfusion | **Difficult crossmatch**; Transfuse if life-threatening anemia (least incompatible unit) | |
| Folic Acid | 1 mg daily for all patients | |

## Paroxysmal Nocturnal Hemoglobinuria (PNH)
| Component | Description | URL |
|-----------|-------------|-----|
| Pathophysiology | Acquired **PIGA mutation** ? Deficiency of GPI-anchored proteins (CD55, CD59) ? **Complement-mediated intravascular hemolysis** | https://www.pnhfoundation.org |
| Clinical Triad | **Hemolytic anemia**, **Thrombosis** (Budd-Chiari, Cerebral, Mesenteric), **Bone marrow failure** | |
| Diagnosis | **Flow cytometry** for GPI-anchored proteins (CD55, CD59 on RBCs; FLAER on WBCs) | |
| Complications | **Iron deficiency** (hemoglobinuria), **Pulmonary hypertension**, **CKD**, **AML/MDS** | |
| Treatment | **Eculizumab**, **Ravulizumab** (C5 inhibitors); **Pegcetacoplan** (C3 inhibitor); **Allogeneic HSCT** (curative for severe AA/PNH) | |
