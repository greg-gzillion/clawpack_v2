## Hypercalcemia of Malignancy
| Component | Description | URL |
|-----------|-------------|-----|
| Causes | **Multiple myeloma** (osteolytic lesions), **Solid tumors** (breast, Lung, Renal) with bone metastases, **Humoral hypercalcemia** (PTHrP - squamous cell, Renal), **Lymphoma** (calcitriol-mediated) | https://www.ash.org/hypercalcemia |
| Symptoms | **"Bones, Stones, Groans, Psychiatric Overtones"**: Bone pain, Kidney stones, Constipation, Confusion, Lethargy, Coma | |
| Diagnosis | **Calcium >10.5 mg/dL** (correct for albumin: Ca + 0.8 x (4 - Albumin)); **PTH** (low), **PTHrP** (elevated in humoral), **1,25(OH)2D** (elevated in lymphoma) | |
| Management | **Aggressive IVF** (200-300 mL/hr), **Calcitonin** 4 units/kg IM/SC q12h (rapid onset), **Bisphosphonates** (Zoledronic acid 4 mg IV, Pamidronate 60-90 mg IV), **Denosumab** (refractory), **Dialysis** (severe, Refractory) | |

## Febrile Neutropenia
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | **ANC <500/無** (or <1000 with expected decline) + **Single temperature =38.3蚓** OR **=38.0蚓 for =1 hour** | https://www.idsociety.org/febrile-neutropenia |
| Risk Stratification | **MASCC Score**: Low-risk (=21) ? Outpatient oral antibiotics; High-risk (<21) ? Inpatient IV antibiotics | |
| Empiric Antibiotics (High-Risk) | **Cefepime**, **Piperacillin-tazobactam**, **Meropenem**, **Ceftazidime**; Add **Vancomycin** if: Hemodynamic instability, Pneumonia, Suspected catheter infection, Known MRSA, Severe mucositis | |
| Empiric Antibiotics (Low-Risk) | **Ciprofloxacin + Amoxicillin-clavulanate** | |
| Persistent Fever | After 4-7 days: **Consider empiric antifungal** (echinocandin, Voriconazole, Amphotericin B) | |
| G-CSF | **Not routinely recommended**; Consider for high-risk, Delayed count recovery | |

## Hyperviscosity Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| Causes | **Waldenstr闣 macroglobulinemia** (IgM), **Multiple myeloma** (IgA, IgG3), **Polycythemia vera**, **Hyperleukocytosis** (AML, CML, ALL) | https://www.ash.org/hyperviscosity |
| Symptoms | **Mucosal bleeding** (epistaxis, Gingival), **Visual changes** (blurring, Retinal hemorrhages, Papilledema), **Neurologic** (headache, Dizziness, Confusion, Coma), **Cardiac** (CHF) | |
| Diagnosis | **Serum viscosity** (normal =1.8 cP; Symptoms usually >4-5 cP); **Fundoscopic exam** (retinal vein engorgement/"sausage links", Hemorrhages) | |
| Management | **Emergent plasmapheresis** (for Ig-mediated); **Phlebotomy** (for polycythemia); **Leukapheresis** (for hyperleukocytosis with symptoms); **Treat underlying disease** | |

## Hyperleukocytosis and Leukostasis
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | **WBC >100,000/無** (AML) or **>300,000/無** (ALL, CML) with symptoms of leukostasis | https://www.ash.org/leukostasis |
| Pathophysiology | Blasts (especially myeloblasts) are less deformable ? **Microvascular occlusion** ? Tissue ischemia | |
| Symptoms | **CNS** (confusion, Headache, Visual changes, Focal deficits, Intracranial hemorrhage), **Pulmonary** (hypoxia, Diffuse infiltrates) | |
| Management | **Emergent leukapheresis** (if symptomatic or WBC >100k in AML); **Hydroxyurea** (cytoreduction); **Avoid RBC transfusions** (increases viscosity); **TLS prophylaxis** | |

| Component | Description | URL |
|-----------|-------------|-----|
| Causes | **Multiple myeloma**, **Lymphoma**, **Metastatic solid tumors** | https://www.asco.org/spinal-cord-compression |
| Presentation | **Back pain** (90%), **Weakness**, **Sensory level**, **Bowel/bladder dysfunction** | |
| Diagnosis | **Emergent MRI spine** | |
| Management | **Dexamethasone 10 mg IV** (immediately), **Radiation** OR **Surgery** (if rapid progression, Instability), **Treat underlying malignancy** | |
