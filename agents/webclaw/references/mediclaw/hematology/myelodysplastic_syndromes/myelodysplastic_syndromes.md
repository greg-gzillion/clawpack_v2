# Hematology - Myelodysplastic Syndromes (MDS)

## MDS - Overview
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Clonal hematopoietic stem cell disorders characterized by **ineffective hematopoiesis**, **peripheral cytopenias**, **dysplastic morphology**, and **increased risk of AML transformation** | https://www.mds-foundation.org |
| Epidemiology | Median age 70-75 years; Incidence 4-5 per 100,000 | |
| Risk Factors | **Age**, **Prior chemotherapy/radiation** (t-MDS), **Environmental exposures** (benzene, Smoking), **Inherited syndromes** (Fanconi, DC, SDS) | |
| Clinical Features | **Symptoms of cytopenias**: Fatigue (anemia), Infections (neutropenia), Bleeding (thrombocytopenia); **Asymptomatic** (incidental CBC finding) | |

## MDS - Diagnosis and Evaluation
| Component | Description | URL |
|-----------|-------------|-----|
| **CBC** | Cytopenia(s) - anemia (most common), Neutropenia, Thrombocytopenia | https://www.nccn.org/mds |
| **Peripheral Smear** | Dysplastic features: Ovalomacrocytes, Pseudo-Pelger-Huët, Hypogranular neutrophils, Blasts | |
| **Bone Marrow Aspirate/Biopsy** | **Hypercellular** (most) or hypocellular; Dysplasia in =10% of cells in =1 lineage; **Blast percentage** (<20%) | |
| **Cytogenetics** | **Del(5q)** (most common), -7/del(7q), +8, Del(20q), -Y, Complex karyotype (=3 abnormalities) | |
| **Molecular Testing** | **SF3B1** (ring sideroblasts), **TP53** (poor prognosis, Complex karyotype), **TET2, ASXL1, DNMT3A, RUNX1, SRSF2, U2AF1** | |
| **Iron Studies, B12, Folate** | **Rule out nutritional deficiencies** | |
| **HIV, Viral Hepatitis** | Rule out infectious causes | |

| Subtype | Blasts (%) | Cytopenias | Cytogenetics | URL |
|---------|-------------|------------|--------------|-----|
| **MDS with low blasts (MDS-LB)** | <5% BM, <2% PB | 1-3 lineages | Any | https://www.nccn.org/mds |
| **MDS with increased blasts (MDS-IB1)** | 5-9% BM, 2-4% PB | Variable | Any | |
| **MDS with increased blasts (MDS-IB2)** | 10-19% BM, 5-19% PB | Variable | Any | |
| **MDS with SF3B1 mutation** | <5% BM | Anemia, Thrombocytosis | **SF3B1 mutation**, =15% ring sideroblasts | |
| **MDS with isolated del(5q)** | <5% BM | Anemia, Normal/increased platelets | **Isolated del(5q)** | |
| **MDS, unclassifiable** | <5% BM | 1 lineage cytopenia | No specific features | |

## MDS - Management by Risk Category
| Risk Category | Treatment Approach | Specific Therapies | URL |
|---------------|-------------------|-------------------|-----|
| **Very Low/Low Risk** | **Supportive care**, Improve cytopenias | **ESAs** (anemia, EPO <500), **Luspatercept** (RS+, SF3B1+), **Lenalidomide** (del(5q)), **G-CSF**, **TPO-RA** (thrombocytopenia) | https://www.nccn.org/mds |
| **Intermediate Risk** | Individualized; Consider clinical trials | Hypomethylating agents, Supportive care, Allogeneic HSCT | |
| **High/Very High Risk** | **Allogeneic HSCT** (curative), Hypomethylating agents, Clinical trials | **Azacitidine**, **Decitabine**, **Oral decitabine/cedazuridine** | |
| **Transfusion Support** | All patients | **Leukoreduced RBCs**, **Platelets**, **Iron chelation** (if ferritin >1000 or >20 transfusions) | |

## MDS - Specific Therapies
| Therapy | Indication | Mechanism | URL |
|---------|------------|-----------|-----|
| **Lenalidomide** | **Isolated del(5q)** | Immunomodulatory; Cytotoxic to del(5q) clone | https://www.fda.gov |
| **Luspatercept** | **RS+ MDS or SF3B1+** | TGF-ß ligand trap; Promotes late-stage erythropoiesis | |
| **Erythropoiesis-Stimulating Agents (ESAs)** | Low-risk MDS, EPO <500 | Stimulate erythropoiesis | |
| **Azacitidine** | High-risk MDS | **Hypomethylating agent**; Improves survival | |
| **Decitabine** | High-risk MDS | Hypomethylating agent | |
| **Allogeneic HSCT** | High-risk MDS, Young/fit patients | **Only curative option** | |

## MDS - Monitoring and Complications
| Component | Description | URL |
|-----------|-------------|-----|
| **Monitor for Progression** | CBC q1-3 months; Bone marrow if worsening cytopenias or suspected transformation | https://www.nccn.org/mds |
| **AML Transformation** | 30-40% of high-risk MDS; Median time 1-2 years | |
| **Iron Overload** | Monitor ferritin; **Iron chelation** (Deferasirox, Deferoxamine) | |
| **Infections** | Neutropenia ? Increased risk; Prompt antibiotics for fever | |
