# Hematology - Platelet Disorders

## Thrombocytopenia - Differential Diagnosis by Mechanism
| Mechanism | Causes | URL |
|-----------|--------|-----|
| **Decreased Production** | Bone marrow failure (AA, MDS, Leukemia), Megaloblastic anemia, Medications, Alcohol, Viral infections | https://www.ash.org/thrombocytopenia |
| **Increased Destruction** | **Immune (ITP)**, **Drug-induced**, **HIT**, **TTP/HUS**, **DIC**, **Post-transfusion purpura** | |
| **Sequestration** | **Hypersplenism** (cirrhosis, Portal hypertension, Myelofibrosis) | |
| **Dilutional** | Massive transfusion | |

## Immune Thrombocytopenia (ITP)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Isolated thrombocytopenia (<100,000/無) with normal bone marrow and no other cause | https://www.hematology.org/ITP |
| Pathophysiology | **Autoantibodies against platelet glycoproteins** (GPIIb/IIIa, GPIb/IX) ? **Splenic destruction** + Impaired production | |
| Primary ITP | No underlying cause | |
| Secondary ITP | **SLE**, **CLL**, **HIV**, **HCV**, **H. pylori**, **Medications**, **Lymphoma**, **APS** | |
| Diagnosis | **Isolated thrombocytopenia**; Normal Hb, WBC; **Peripheral smear**: Large platelets, No schistocytes; **Bone marrow**: Normal/increased megakaryocytes (not routinely needed) | |

## ITP - Management
| Platelet Count | Bleeding | Management | URL |
|----------------|----------|------------|-----|
| **<30,000** (or <50,000 + bleeding) | Active bleeding | **IVIG 1 g/kg x 1-2 days** + **Dexamethasone 40 mg daily x 4 days** + **Platelet transfusion** (if life-threatening) | https://www.ash.org/ITP-guidelines |
| **<30,000** | No bleeding | **Prednisone 0.5-2 mg/kg/day** OR **Dexamethasone 40 mg daily x 4 days** (repeat cycles) OR **Observation** (if asymptomatic) | |
| **30,000-50,000** | No bleeding | **Observation** (most); Consider treatment if high bleeding risk | |
| **Refractory ITP** | Failed steroids | **Rituximab**, **TPO-RA** (Eltrombopag, Romiplostim, Avatrombopag), **Fostamatinib** (Syk inhibitor), **Mycophenolate**, **Splenectomy** | |

## Heparin-Induced Thrombocytopenia (HIT)
| Component | Description | URL |
|-----------|-------------|-----|
| Pathophysiology | **Antibodies against PF4-heparin complex** ? Platelet activation ? **Thrombocytopenia + Thrombosis** | https://www.ash.org/HIT |
| Type I HIT | Non-immune, Mild (platelets >100k), Early (days 1-4), Benign | |
| Type II HIT | **Immune-mediated**, Severe (platelets <150k or >50% drop), Days 5-10 (or earlier if prior heparin exposure), **High thrombosis risk** | |
| 4T Score | **T**hrombocytopenia (0-2), **T**iming (0-2), **T**hrombosis (0-2), O**T**her causes (0-2); **Low (=3)**: Unlikely HIT; **Intermediate (4-5)**: Possible; **High (6-8)**: Probable | |
| Diagnosis | **ELISA** (anti-PF4/heparin antibodies) - sensitive; **Serotonin release assay (SRA)** - specific | |

## HIT - Management
| Component | Description | URL |
|-----------|-------------|-----|
| **Immediate Action** | **STOP ALL HEPARIN** (including flushes, Heparin-coated catheters) | https://www.ash.org/HIT |
| **Non-Heparin Anticoagulant** | **Argatroban** (IV, Monitor aPTT), **Bivalirudin** (IV, aPTT), **Fondaparinux** (SC, Anti-Xa), **DOACs** (Rivaroxaban, Apixaban) | |
| **Warfarin** | **CONTRAINDICATED** in acute HIT (risk of venous limb gangrene); Start only after platelet recovery (>150k) and overlap with non-heparin anticoagulant | |
| **Platelet Transfusion** | **CONTRAINDICATED** (increases thrombosis risk) unless life-threatening bleeding | |
| **Duration** | **4-6 weeks** if HIT without thrombosis; **3-6 months** if HITT (HIT with thrombosis) | |
| **Documentation** | **List HIT as allergy** in medical record | |

## Thrombotic Thrombocytopenic Purpura (TTP)
| Component | Description | URL |
|-----------|-------------|-----|
| Pathophysiology | **Severe ADAMTS13 deficiency** (<10%) due to autoantibodies (acquired) or mutations (congenital/Upshaw-Schulman) | https://www.hematology.org/TTP |
| Pentad (Historical) | **Thrombocytopenia**, **MAHA**, **Fever**, **Neurologic symptoms**, **Renal impairment** (only 5% have full pentad) | |
| PLASMIC Score | Platelets <30k, Hemolysis, No active cancer, No transplant, MCV <90, INR <1.5, Cr <2.0; **Score =6 = High risk** | |
| Diagnosis | **ADAMTS13 activity <10%** (send before plasma exchange) | |
| Treatment | **Emergent plasma exchange (PLEX)** - daily until platelets normal x 2 days; **Caplacizumab** (anti-VWF nanobody); **Corticosteroids**; **Rituximab** (reduce relapse) | |
| **CRITICAL** | **DO NOT DELAY PLEX** waiting for ADAMTS13 result | |
| **Avoid Platelet Transfusion** | Unless life-threatening bleeding | |
