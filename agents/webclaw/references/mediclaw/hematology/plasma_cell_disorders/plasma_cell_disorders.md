# Hematology - Plasma Cell Disorders

## Spectrum of Plasma Cell Disorders
| Disorder | Definition | Progression Risk | URL |
|----------|------------|------------------|-----|
| **Monoclonal Gammopathy of Undetermined Significance (MGUS)** | M-protein <3 g/dL, BM plasma cells <10%, No CRAB | 1% per year | https://www.myeloma.org |
| **Smoldering Multiple Myeloma (SMM)** | M-protein =3 g/dL or BM plasma cells 10-59%, No CRAB | 10% per year (first 5 years) | |
| **Multiple Myeloma (MM)** | BM plasma cells =10% + CRAB or SLiM-CRAB | N/A | |
| **Plasma Cell Leukemia** | Circulating plasma cells >2 x10^9/L or >20% of WBC | Aggressive variant | |
| **Solitary Plasmacytoma** | Single lesion (bone or extramedullary), No BM involvement | 50-60% progress to MM | |
| **AL Amyloidosis** | Amyloidogenic light chains ? Organ deposition | N/A | |

## Myeloma - Supportive Care
| Complication | Management | URL |
|--------------|------------|-----|
| **Bone Disease** | **Bisphosphonates** (Zoledronic acid, Pamidronate) OR **Denosumab**; Calcium/Vitamin D; **Avoid dental extractions** (ONJ risk) | https://www.myeloma.org |
| **Anemia** | Erythropoiesis-stimulating agents; Transfusions | |
| **Renal Impairment** | Hydration, Avoid NSAIDs, **Plasmapheresis** (cast nephropathy), **Bortezomib-based regimens** | |
| **Hypercalcemia** | **Aggressive hydration**, **Calcitonin**, **Bisphosphonates/Denosumab** | |
| **Infection** | **Antibiotic prophylaxis** (Levofloxacin, Acyclovir, TMP-SMX), **IVIG** (if recurrent infections), **Vaccinations** (influenza, Pneumococcal, **Avoid live vaccines**) | |
| **Thrombosis** | **Aspirin** (low-risk), **Warfarin/LMWH/DOAC** (high-risk) if on IMiDs (lenalidomide, Pomalidomide) | |
| **Peripheral Neuropathy** | Dose reduction, **Gabapentin/Pregabalin**, Duloxetine, Avoid bortezomib SC | |

## AL Amyloidosis
| Component | Description | URL |
|-----------|-------------|-----|
| Pathophysiology | Deposition of amyloidogenic light chains (usually lambda) in organs | https://www.amyloidosis.org |
| Organ Involvement | **Heart** (restrictive cardiomyopathy, Low voltage ECG, Thickened LV on echo), **Kidneys** (nephrotic syndrome), **Liver** (hepatomegaly, Elevated Alk Phos), **Nerves** (peripheral/autonomic neuropathy), **GI** (bleeding, Dysmotility), **Soft tissue** (macroglossia, Periorbital purpura) | |
| Diagnosis | **Tissue biopsy** (abdominal fat pad, Bone marrow, Rectal, Organ) - **Congo red stain** (apple-green birefringence); **Serum free light chains**, **SPEP/UPEP** | |
| Staging (Mayo 2012) | **NT-proBNP** (=1800 ng/L), **Troponin T** (=0.025 ng/mL), **dFLC** (=180 mg/L); Stage I (0), II (1), III (2), IV (3 factors) | |
| Treatment | **Dara-CyBorD** (Daratumumab + Cyclophosphamide, Bortezomib, Dexamethasone) - preferred; **Autologous HSCT** (selected patients); Supportive care (Diuretics, Midodrine, Compression stockings) | |

## Waldenstr—m Macroglobulinemia (WM)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Lymphoplasmacytic lymphoma with **IgM monoclonal gammopathy** | https://www.iwmf.com |
| Genetics | **MYD88 L265P** (90-95%), **CXCR4** mutations (30-40%) | |
| Clinical Features | **Hyperviscosity** (oromucosal bleeding, Visual changes, Headache), Anemia, Lymphadenopathy, Splenomegaly, **Peripheral neuropathy** (anti-MAG) | |
| Diagnosis | **IgM M-protein** + **BM biopsy** (lymphoplasmacytic infiltrate) + **MYD88 L265P mutation** | |
| Indications for Treatment | Hb <10, Platelets <100k, Symptomatic hyperviscosity, Neuropathy, Bulky disease, Constitutional symptoms, Cryoglobulinemia, Cold agglutinin disease | |
| First-Line | **Bendamustine + Rituximab (BR)**, **Dexamethasone + Rituximab + Cyclophosphamide (DRC)**, **Ibrutinib Ý Rituximab** (MYD88-mutated, CXCR4 wild-type), **Zanubrutinib** | |
| Hyperviscosity | **Emergent plasmapheresis** | |
