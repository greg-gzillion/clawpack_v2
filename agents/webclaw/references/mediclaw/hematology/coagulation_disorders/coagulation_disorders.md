## Hemophilia A (Factor VIII Deficiency)
| Component | Description | URL |
|-----------|-------------|-----|
| Inheritance | **X-linked recessive** | https://www.hemophilia.org |
| Genetics | **F8 gene**; 30% de novo mutations | |
| Severity | **Severe**: <1% factor activity (spontaneous bleeding, Hemarthroses); **Moderate**: 1-5% (bleeding with minor trauma); **Mild**: 6-40% (bleeding with major trauma/surgery) | |
| Clinical Features | **Hemarthroses** (knees, Ankles, Elbows), **Muscle hematomas**, **Intracranial hemorrhage**, **Post-surgical bleeding**, **Easy bruising** | |
| Diagnosis | **Prolonged aPTT**, **Low Factor VIII activity**, Normal VWF | |
| Treatment (Bleeding) | **Factor VIII concentrate** (recombinant or plasma-derived); **Desmopressin (DDAVP)** for mild hemophilia A | |
| Treatment (Prophylaxis) | **Factor VIII concentrate 2-3x/week** (severe hemophilia); **Emicizumab** (bispecific antibody - mimics FVIII) | |
| Complications | **Inhibitor development** (20-30% of severe hemophilia A) ? **Bypassing agents** (aPCC, rFVIIa) or **Immune tolerance induction (ITI)** | |
