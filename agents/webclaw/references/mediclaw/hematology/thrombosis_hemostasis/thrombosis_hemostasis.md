## Antiphospholipid Syndrome (APS)
| Component | Description | URL |
|-----------|-------------|-----|
| Clinical Criteria | **Vascular thrombosis** (arterial, Venous, Small vessel) OR **Pregnancy morbidity** (=3 consecutive losses <10 weeks, =1 fetal death =10 weeks, Severe preeclampsia/placental insufficiency <34 weeks) | https://www.rheumatology.org/APS |
| Laboratory Criteria (=12 weeks apart) | **Lupus anticoagulant**, **Anticardiolipin antibodies** (IgG/IgM, Medium-high titer), **Anti-ß2-glycoprotein I antibodies** (IgG/IgM) | |
| Catastrophic APS (CAPS) | **Widespread thrombosis** (=3 organs) within 1 week; High mortality (30-50%) | |
| Treatment (VTE) | **Warfarin** (INR 2-3); **DOACs inferior to warfarin** (especially arterial or triple-positive) | |
| Treatment (Obstetric) | **Lovenox + Low-dose aspirin** during pregnancy | |
| Treatment (CAPS) | **Anticoagulation + Steroids + Plasma exchange + IVIG** ± Rituximab/Eculizumab | |

## Venous Thromboembolism (VTE) - Diagnosis
| Test | Indication | Interpretation | URL |
|------|------------|----------------|-----|
| **D-Dimer** | Low pretest probability (Wells =1) | **Negative** (<500 ng/mL age-adjusted) **rules out VTE**; **Positive** requires imaging | https://www.chestnet.org/VTE |
| **Compression Ultrasound** | Suspected DVT | **Non-compressible vein** = DVT | |
| **CT Pulmonary Angiography** | Suspected PE | **Filling defect** = PE | |
| **V/Q Scan** | Suspected PE (renal insufficiency, Contrast allergy) | **High probability** = PE; Normal = excludes | |

## Warfarin Management
| Component | Description | URL |
|-----------|-------------|-----|
| **Initiation** | Overlap with heparin/LMWH until INR =2 for =24 hours (minimum 5 days overlap) | https://www.ash.org/warfarin |
| **Target INR** | **2.0-3.0** (most indications); **2.5-3.5** (mechanical mitral valve) | |
| **INR Monitoring** | Daily initially ? weekly ? monthly | |
| **Elevated INR** | <4.5: Hold dose, Reduce; 4.5-10 (no bleeding): **Vitamin K 1-2.5 mg PO**; >10 or bleeding: **Vitamin K 5-10 mg IV**; Life-threatening: **4F-PCC + Vitamin K** | |
| **Drug Interactions** | **Potentiate** (amiodarone, TMP-SMX, Metronidazole, Fluconazole), **Inhibit** (rifampin, Carbamazepine, Phenytoin) | |
| **Diet** | Consistent vitamin K intake | |
