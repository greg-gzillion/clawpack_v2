|-----------|--------------|----------------------|-----|

| **Hematocrit (Hct)** | M: 41-53%; F: 36-46% | Same as Hb | |
| **RBC Count** | M: 4.5-5.9 x10^6/ĄL; F: 4.0-5.2 | Thalassemia (elevated), Bone marrow failure (low) | |

| **RDW** | 11.5-14.5% | **Anisocytosis** (IDA, MDS, Hemolysis) | |

| **Platelet Count** | 150-450 x10^3/ĄL | Thrombocytopenia, Thrombocytosis | |
| **MPV** | 7.5-11.5 fL | ITP (large platelets), Wiskott-Aldrich (small) | |

## Peripheral Blood Smear - Red Cell Morphology
| Finding | Description | Associated Conditions | URL |
|---------|-------------|----------------------|-----|
| **Microcytes** | Small RBCs (<7 Ąm) | IDA, Thalassemia, ACD, Sideroblastic | https://www.ash.org/peripheral-smear |
| **Macrocytes** | Large RBCs (>8.5 Ąm) | B12/Folate deficiency, MDS, Liver disease, Reticulocytosis | |
| **Hypochromia** | Pale RBCs (central pallor >1/3) | IDA, Thalassemia, ACD | |
| **Spherocytes** | Dense, Hyperchromic, No central pallor | **Hereditary spherocytosis**, **Warm AIHA** | |
| **Schistocytes** | Fragmented RBCs (helmet cells) | **MAHA** (TTP, HUS, DIC), Mechanical heart valve | |
| **Target Cells** | Bull's-eye appearance | Thalassemia, Liver disease, HbC, Post-splenectomy | |
| **Teardrop Cells** | Pear-shaped RBCs | **Myelofibrosis**, MDS, Thalassemia | |
| **Acanthocytes** | Spur cells (irregular spicules) | Liver disease, Abetalipoproteinemia, Post-splenectomy | |
| **Echinocytes** | Burr cells (regular spicules) | **Pyruvate kinase deficiency**, Uremia, Artifact | |
| **Bite/Blister Cells** | Heinz body hemolysis | **G6PD deficiency**, Oxidative stress | |
| **Sickle Cells** | Crescent-shaped | **Sickle cell disease** | |
| **Howell-Jolly Bodies** | Nuclear remnants | **Post-splenectomy**, Functional asplenia | |
| **Basophilic Stippling** | Ribosomal precipitates | **Lead poisoning**, **Thalassemia**, MDS, Pyrimidine 5-nucleotidase deficiency | |
| **Pappenheimer Bodies** | Iron granules | Sideroblastic anemia, Post-splenectomy | |
| **Rouleaux Formation** | Stacked RBCs | **Multiple myeloma**, Waldenstr÷m, Inflammation | |
| **Agglutination** | Clumped RBCs | **Cold agglutinin disease** | |

## Bone Marrow Examination
| Component | Normal | Abnormal | URL |
|-----------|--------|----------|-----|
| **Cellularity** | 100 - age (%) | Hypocellular (AA), Hypercellular (MDS, MPN, Leukemia) | https://www.ash.org/bone-marrow |
| **M:E Ratio** | 2:1 to 4:1 | Increased (CML, Infection), Decreased (Anemia, Hemolysis) | |
| **Erythroid** | 15-25% | Decreased (PRCA), Dysplasia (MDS) | |
| **Myeloid** | 50-70% | Increased (CML, Infection), Decreased (AA, MDS) | |
| **Megakaryocytes** | 1-3 per 40x field | Decreased (AA, MDS), Increased (ITP, MPN), Dysplastic (MDS) | |
| **Lymphocytes** | 5-15% | Increased (CLL, Lymphoma) | |
| **Plasma Cells** | <5% | Increased (Multiple myeloma, MGUS) | |
| **Blasts** | <5% | Increased (MDS, AML) | |
| **Fibrosis** | None/minimal | Increased (PMF, MDS, Metastatic cancer) | |
| **Iron Stores** | 2-3+ | Absent (IDA), Increased (ACD, Hemochromatosis, MDS) | |
| **Ring Sideroblasts** | None | =15% of erythroid precursors (MDS with RS, Sideroblastic anemia) | |
