# Hematology - Chronic Leukemia

## Chronic Lymphocytic Leukemia (CLL) - Overview
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Clonal proliferation of mature **CD5+ B lymphocytes** | https://www.lls.org/leukemia/chronic-lymphocytic-leukemia |
| Epidemiology | **Most common leukemia in adults**; Median age 70; Male > Female | |
| Diagnosis | **=5000 clonal B lymphocytes/µL** for =3 months; Flow cytometry: CD5+, CD19+, CD20(dim), CD23+, **FMC7-**, Surface Ig(dim) | |
| Staging (Rai) | **0**: Lymphocytosis only; **I**: + Lymphadenopathy; **II**: + Splenomegaly; **III**: + Anemia (Hb <11); **IV**: + Thrombocytopenia (<100k) | |
| Staging (Binet) | **A**: <3 lymphoid areas; **B**: =3 areas; **C**: Anemia (Hb <10) or Thrombocytopenia (<100k) | |

## CLL - Indications for Treatment (iwCLL 2018)
| Indication | Description | URL |
|------------|-------------|-----|
| **Progressive Marrow Failure** | Worsening anemia or thrombocytopenia | https://www.nccn.org/cll |
| **Massive/Symptomatic Splenomegaly** | =6 cm below costal margin | |
| **Massive/Symptomatic Lymphadenopathy** | =10 cm or symptomatic | |
| **Progressive Lymphocytosis** | Doubling time <6 months or =50% increase in 2 months | |
| **Autoimmune Complications** | AIHA or ITP poorly responsive to steroids | |
| **Constitutional Symptoms** | Weight loss =10% in 6 months, Fever >38°C x 2 weeks, Night sweats, Fatigue | |
| **Extranodal Involvement** | Symptomatic or functional impairment | |

## CLL - First-Line Treatment
| Patient Population | Preferred Regimens | URL |
|--------------------|-------------------|-----|
| **Fit, No TP53 aberration** | **Acalabrutinib ± Obinutuzumab**, **Zanubrutinib**, **Venetoclax + Obinutuzumab** (fixed duration) | https://www.nccn.org/cll |
| **Unfit, No TP53 aberration** | **Acalabrutinib ± Obinutuzumab**, **Zanubrutinib**, **Venetoclax + Obinutuzumab** | |
| **TP53 Aberration (del(17p) or TP53 mutation)** | **Acalabrutinib**, **Zanubrutinib**, **Venetoclax + Obinutuzumab** | |
| **Note** | **Chemoimmunotherapy (FCR, BR) no longer preferred** in most settings | |

## Richter Transformation
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Transformation of CLL to **aggressive lymphoma** (DLBCL - 90%, Hodgkin - 10%) | https://www.lls.org/richter-syndrome |
| Incidence | 2-10% of CLL patients | |
| Presentation | Rapidly enlarging lymph nodes, B symptoms, Elevated LDH, Hypercalcemia | |
| Diagnosis | **Excisional biopsy** (not FNA) of largest lymph node; **PET-CT** (SUVmax >10 suggests transformation) | |
| Treatment | **R-CHOP** (DLBCL-like), **CAR-T**, **Allogeneic HSCT** (consolidation); Prognosis poor | |

## Chronic Myeloid Leukemia (CML)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Myeloproliferative neoplasm characterized by **BCR-ABL1 fusion gene** (Philadelphia chromosome) | https://www.lls.org/leukemia/chronic-myeloid-leukemia |
| Epidemiology | Median age 64; 15% of adult leukemias | |
| Phases | **Chronic Phase (CP)** - 85-90% at diagnosis; **Accelerated Phase (AP)**; **Blast Crisis (BC)** - myeloid (70%) or lymphoid (30%) | |
| Diagnosis | **CBC** (leukocytosis with left shift, Basophilia, Thrombocytosis), **Peripheral smear** (myeloid precursors), **Bone marrow** (hypercellular), **Cytogenetics** (t(9;22)), **FISH** (BCR-ABL1), **PCR** (BCR-ABL1 transcript) | |

## CML - Treatment
| Phase | Treatment | URL |
|-------|-----------|-----|
| **Chronic Phase** | **Tyrosine Kinase Inhibitors (TKIs)**: **Imatinib** 400 mg daily, **Dasatinib** 100 mg daily, **Nilotinib** 300 mg BID, **Bosutinib** 400 mg daily, **Asciminib** (if T315I or refractory) | https://www.nccn.org/cml |
| **Accelerated Phase** | **TKIs** (higher doses); Consider **Allogeneic HSCT** | |
| **Blast Crisis** | **TKIs + Chemotherapy** (AML-type for myeloid, ALL-type for lymphoid); **Allogeneic HSCT** (only curative option) | |
| **TKI Intolerance/Resistance** | Switch to alternative TKI; **BCR-ABL1 kinase domain mutation testing** (guide selection) | |
| **T315I Mutation** | Resistant to imatinib, Dasatinib, Nilotinib, Bosutinib; **Ponatinib**, **Asciminib**, **Allogeneic HSCT** | |

## CML - Monitoring Response
| Milestone | Time | Goal | URL |
|-----------|------|------|-----|
| **Complete Hematologic Response (CHR)** | 3 months | Normal CBC, No splenomegaly | https://www.nccn.org/cml |
| **Early Molecular Response** | 3 months | **BCR-ABL1 =10% (IS)** | |
| **Complete Cytogenetic Response (CCyR)** | 12 months | No Philadelphia chromosome on metaphase cytogenetics | |
| **Major Molecular Response (MMR)** | 12-18 months | **BCR-ABL1 =0.1% (IS)** | |
| **Deep Molecular Response (DMR)** | Variable | **BCR-ABL1 =0.01% (MR4) or =0.0032% (MR4.5)** | |
| **Treatment-Free Remission (TFR)** | After =2 years of DMR | **Discontinue TKI**; Monitor monthly x 6 months, then q2 months | |

## Hairy Cell Leukemia (HCL)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Indolent B-cell neoplasm involving **mature B cells with cytoplasmic projections** | https://www.lls.org/leukemia/hairy-cell-leukemia |
| Epidemiology | Rare (<2% of leukemias); Median age 55; Male > Female (4:1) | |
| Presentation | Pancytopenia, **Monocytopenia** (hallmark), Splenomegaly (80-90%), Lymphadenopathy rare | |
| Diagnosis | **Peripheral smear**: Hairy cells; **Bone marrow**: **Dry tap** (fibrosis), **Fried-egg appearance**; **Flow cytometry**: CD19+, CD20(bright), CD22+, **CD11c+, CD25+, CD103+**; **BRAF V600E mutation** (95-100%) | |
| Variant (HCL-v) | **CD25 negative**, **BRAF wild-type**, **MAP2K1 mutations**; Poorer response to purine analogs | |
| Treatment | **Cladribine** (single course - CR 80-90%) or **Pentostatin**; **Rituximab** ± Vemurafenib for refractory; **Moxetumomab pasudotox** (anti-CD22 immunotoxin) for relapsed | |
