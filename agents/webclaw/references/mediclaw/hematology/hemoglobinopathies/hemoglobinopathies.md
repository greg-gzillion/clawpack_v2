## Sickle Cell Disease - Acute Complications
| Complication | Presentation | Management | URL |
|--------------|--------------|------------|-----|
| **Vaso-occlusive Crisis (VOC)** | Severe pain (back, Chest, Extremities) | **IV fluids**, **Opioids**, NSAIDs, Incentive spirometry | https://www.ash.org/sickle-cell |
| **Acute Chest Syndrome (ACS)** | Fever, Chest pain, Hypoxia, New infiltrate on CXR | **Oxygen**, **Antibiotics** (Ceftriaxone + Azithromycin), **Exchange transfusion**, Incentive spirometry | |
| **Splenic Sequestration** | Rapid splenic enlargement, Severe anemia, Hypovolemic shock | **IV fluids**, **Transfusion**, Splenectomy | |
| **Aplastic Crisis** | Severe anemia, Reticulocytopenia | **Transfusion**; Often due to Parvovirus B19 | |
| **Stroke** | Focal neurologic deficits | **Emergent exchange transfusion** | |
| **Priapism** | Painful prolonged erection | Hydration, Analgesia, Aspiration ｱ Phenylephrine, Exchange transfusion | |

## Sickle Cell Disease - Disease-Modifying Therapies
| Therapy | Mechanism | Indications | URL |
|---------|-----------|-------------|-----|
| **Hydroxyurea** | Increases HbF, Reduces adhesion | **All patients with HbSS or HbSﾟ0** (age =9 months) | https://www.ash.org/sickle-cell |
| **L-Glutamine (Endari)** | Reduces oxidative stress | VOC prevention | |
| **Voxelotor (Oxbryta)** | Increases Hb oxygen affinity, Reduces hemolysis | VOC prevention, Anemia improvement | |
| **Crizanlizumab (Adakveo)** | P-selectin inhibitor (reduces adhesion) | VOC prevention | |
| **Hematopoietic Stem Cell Transplant** | Curative | Severe disease with matched sibling donor | |
| **Gene Therapy** | Curative | FDA-approved (Casgevy, Lyfgenia) | |

## Sickle Cell Trait (HbAS)
| Component | Description | URL |
|-----------|-------------|-----|
| Prevalence | 8-10% of African Americans; 1 in 13 | https://www.cdc.gov/ncbddd/sicklecell/trait |
| Clinical Significance | **Generally asymptomatic**; **Renal medullary carcinoma** (rare), **Hematuria** (renal papillary necrosis), **Splenic infarction at high altitude**, **Exertional rhabdomyolysis** | |
| Counseling | Autosomal recessive inheritance; Offer partner screening | |
