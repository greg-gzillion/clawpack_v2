# Hematology - Transfusion Medicine

## Blood Products
| Product | Volume | Indications | Storage | URL |
|---------|--------|-------------|---------|-----|
| **Packed Red Blood Cells (PRBCs)** | 250-300 mL | **Symptomatic anemia**, Active bleeding, Hb <7 g/dL (stable) | 42 days (4蚓) | https://www.aabb.org |
| **Platelets** | 50-70 mL (pooled) or 200-400 mL (apheresis) | **Bleeding + Plt <50k**, Plt <10k (prophylaxis) | 5-7 days (room temp) | |
| **Fresh Frozen Plasma (FFP)** | 200-250 mL | **Bleeding + coagulopathy** (INR >1.5-2), Warfarin reversal, TTP plasma exchange | 1 year (-18蚓) | |
| **Cryoprecipitate** | 10-15 mL (per unit) | **Hypofibrinogenemia** (<100-150 mg/dL), DIC, Massive transfusion | 1 year (-18蚓) | |
| **Prothrombin Complex Concentrate (4F-PCC)** | Small volume | **Warfarin reversal** (life-threatening bleeding), DOAC reversal (off-label) | Room temp | |

| Product | Trigger | Goal | URL |

| **Cryoprecipitate** | **Fibrinogen <100-150 mg/dL + Bleeding**; DIC; Massive transfusion | Fibrinogen >150 mg/dL | |

## Transfusion Reactions
| Reaction | Timing | Mechanism | Symptoms | Management | URL |
|----------|--------|-----------|----------|------------|-----|
| **Acute Hemolytic** | Immediate | **ABO incompatibility** | Fever, Chills, **Flank pain**, **Hemoglobinuria**, Hypotension, DIC | **STOP transfusion**, IV fluids, Diuretics, Pressors | https://www.cdc.gov/bloodsafety |
| **Febrile Non-Hemolytic** | 1-6 hours | Cytokines or WBC antibodies | Fever, Chills | Antipyretics, Slow transfusion, **Leukoreduced products** | |
| **Allergic (Urticarial)** | Immediate | Plasma proteins | Urticaria, Pruritus | Antihistamines, Resume transfusion if mild | |
| **Anaphylactic** | Immediate | **IgA deficiency + Anti-IgA** | Hypotension, Bronchospasm, Angioedema | **Epinephrine**, **STOP transfusion**, **Washed products** | |
| **Transfusion-Related Acute Lung Injury (TRALI)** | <6 hours | Donor anti-HLA or anti-HNA antibodies | **Hypoxia**, **Bilateral pulmonary infiltrates**, Hypotension, Fever | **Supportive care**, **Mechanical ventilation**, **Diuretics NOT indicated** | |
| **Transfusion-Associated Circulatory Overload (TACO)** | <6 hours | Volume overload | Dyspnea, Hypertension, JVD, Pulmonary edema | **Diuretics**, Oxygen, **Slow transfusion** | |
| **Delayed Hemolytic** | 3-14 days | Anamnestic antibody response | Falling Hb, Jaundice, Positive DAT | Supportive; Avoid further transfusion | |
| **Transfusion-Associated Graft-vs-Host Disease (TA-GVHD)** | 1-6 weeks | Engraftment of donor lymphocytes | **Pancytopenia**, Rash, Diarrhea, Liver injury | **Fatal**; **Irradiated products** for at-risk patients | |

## TRALI vs TACO
| Feature | TRALI | TACO | URL |
|---------|-------|------|-----|
| **Pathophysiology** | Donor antibodies ? Neutrophil activation ? Capillary leak | Volume overload | https://www.aabb.org |
| **Blood Pressure** | Hypotension | **Hypertension** | |
| **JVD** | Absent | **Present** | |
| **BNP** | Normal/low | **Elevated** | |
| **Response to Diuretics** | **No response** (may worsen) | **Improves** | |
| **CXR** | Bilateral infiltrates | Pulmonary edema, Cardiomegaly | |

## Massive Transfusion Protocol (MTP)
| Component | Ratio | URL |
|-----------|-------|-----|
| **PRBCs : FFP : Platelets** | **1:1:1** | https://www.facs.org/mtp |
| **Tranexamic Acid (TXA)** | 1g IV over 10 min ? 1g over 8 hours (within 3 hours of injury) | |
| **Calcium** | Calcium gluconate 1-2g or Calcium chloride 1g (citrate toxicity) | |
| **Cryoprecipitate** | For fibrinogen <100-150 mg/dL | |
| **Goal** | **Permissive hypotension** (SBP ~90) until bleeding controlled | |

## Iron Overload from Transfusions
| Component | Description | URL |
|-----------|-------------|-----|
| **Risk** | After **10-20 transfusions** or **ferritin >1000 ng/mL** | https://www.aamds.org |
| **Consequences** | Cardiac (cardiomyopathy, Arrhythmias), Hepatic (cirrhosis, HCC), Endocrine (diabetes, Hypogonadism, Hypothyroidism) | |
| **Monitoring** | Ferritin q3-6 months; **MRI T2*** (cardiac/liver iron) | |
| **Chelation** | **Deferasirox** (PO), **Deferoxamine** (IV/SQ), **Deferiprone** (PO) | |
