## Polycythemia Vera (PV) - Diagnosis (WHO 2022)
| Major Criteria | Minor Criteria | URL |
|----------------|----------------|-----|
| 1. **Hb >16.5 g/dL (men)** or **>16.0 g/dL (women)** OR **Hct >49% (men)** or **>48% (women)** OR **Increased RBC mass** | 1. Subnormal serum erythropoietin level | https://www.nccn.org/mpn |
| 2. **Bone marrow biopsy** showing hypercellularity with panmyelosis | | |
| 3. **JAK2 V617F or JAK2 exon 12 mutation** | | |
| Diagnosis: **All 3 major** OR **First 2 major + minor** | | |

## PV - Management
| Component | Description | URL |
|-----------|-------------|-----|
| **Phlebotomy** | Goal **Hct <45%**; Initial weekly until goal, then maintenance q1-3 months | https://www.nccn.org/mpn |
| **Aspirin** | **81-100 mg daily** (all patients without contraindication) | |
| **Cytoreductive Therapy** | **Hydroxyurea** (first-line); **Ruxolitinib** (second-line, Hydroxyurea-intolerant/resistant); **Pegylated interferon alfa-2a** (younger patients) | |
| **Symptom Management** | Pruritus: Antihistamines, Ruxolitinib; Aquagenic pruritus: Avoid hot showers, UVB phototherapy | |
| **Thrombosis Risk** | **High-risk** (age =60 or prior thrombosis): Cytoreductive therapy; **Low-risk** (no risk factors): Phlebotomy + Aspirin | |

## Essential Thrombocythemia (ET) - Diagnosis (WHO 2022)
| Major Criteria | Minor Criteria | URL |
|----------------|----------------|-----|
| 1. **Platelet count =450 x10^9/L** | 1. Presence of clonal marker or absence of evidence for reactive thrombocytosis | https://www.nccn.org/mpn |
| 2. **Bone marrow biopsy** showing proliferation of megakaryocytic lineage with increased large, Mature megakaryocytes with hyperlobulated nuclei | | |
| 3. **Not meeting criteria for other MPNs** (CML, PV, PMF, MDS) | | |
| 4. **JAK2, CALR, or MPL mutation** | | |
| Diagnosis: **All 4 major** OR **First 3 major + minor** | | |

## ET - Management
| Risk Category | Criteria | Management | URL |
|---------------|----------|------------|-----|
| **Very Low Risk** | Age =60, No prior thrombosis, **JAK2 wild-type** | **Observation** or **Aspirin** | https://www.nccn.org/mpn |
| **Low Risk** | Age =60, No prior thrombosis, **JAK2 V617F positive** | **Aspirin 81-100 mg daily** | |
| **Intermediate Risk** | Age >60, No prior thrombosis, JAK2 wild-type | Aspirin; Consider cytoreduction | |
| **High Risk** | **Age >60 + JAK2 V617F** OR **Prior thrombosis** | **Aspirin + Cytoreductive therapy** (Hydroxyurea, Pegylated interferon, Anagrelide) | |
| **Extreme Thrombocytosis** | Platelets >1,000,000 | **Acquired von Willebrand syndrome** risk; Monitor VWF activity; **Avoid aspirin** if VWF:RCo <30% | |

## Primary Myelofibrosis (PMF) - Diagnosis (WHO 2022)
| Major Criteria | Minor Criteria | URL |
|----------------|----------------|-----|
| 1. **Megakaryocyte proliferation and atypia** with **reticulin and/or collagen fibrosis** (grade 2-3) | 1. Anemia not attributed to other condition | https://www.nccn.org/mpn |
| 2. **Not meeting criteria for other MPNs** | 2. Leukocytosis =11 x10^9/L | |
| 3. **JAK2, CALR, or MPL mutation** OR other clonal marker | 3. Palpable splenomegaly | |
| | 4. LDH elevated above ULN | |
| | 5. Leukoerythroblastosis | |
| Diagnosis: **All 3 major + =1 minor** (confirmed on 2 consecutive occasions) | | |

## PMF - Management
| Risk Category | Management | URL |
|---------------|------------|-----|
| **Low/Intermediate-1 Risk** | **Observation**, **JAK inhibitor** (Ruxolitinib, Fedratinib) for symptomatic splenomegaly, **Hydroxyurea**, **Erythropoiesis-stimulating agents** | https://www.nccn.org/mpn |
| **Intermediate-2/High Risk** | **Allogeneic HSCT** (only curative option) for eligible patients; **JAK inhibitors** for symptom palliation | |
| **Anemia** | **Danazol**, **Lenalidomide** (del(5q)), **Luspatercept**, **Prednisone**, **Transfusions** | |
| **Splenomegaly** | **Ruxolitinib**, **Fedratinib**, **Pacritinib**, **Momelotinib** (anemia benefit), **Splenectomy** (high morbidity/mortality), **Splenic radiation** (temporary) | |

## JAK Inhibitors for Myelofibrosis
| Drug | Targets | Key Features | URL |
|------|---------|--------------|-----|
| **Ruxolitinib** | JAK1/2 | **First-line**; Improves symptoms, Splenomegaly; May worsen anemia/thrombocytopenia | https://www.fda.gov |
| **Fedratinib** | JAK2, FLT3 | Ruxolitinib-exposed; Risk of **Wernicke encephalopathy** (monitor thiamine) | |
| **Pacritinib** | JAK2, IRAK1 | **Severe thrombocytopenia** (platelets <50k) | |
| **Momelotinib** | JAK1/2, ACVR1 | **Anemia benefit**; FDA-approved for MF with anemia | |
