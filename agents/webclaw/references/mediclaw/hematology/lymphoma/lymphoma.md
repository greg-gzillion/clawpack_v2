## Hodgkin Lymphoma - Clinical Features and Staging
| Component | Description | URL |
|-----------|-------------|-----|
| Presentation | **Painless lymphadenopathy** (cervical/supraclavicular), **B symptoms** (fever, Night sweats, Weight loss), **Pruritus**, **Alcohol-induced pain** | https://www.nccn.org/hodgkin |
| Staging (Lugano) | **I**: Single node region; **II**: =2 regions same side of diaphragm; **III**: Both sides of diaphragm; **IV**: Diffuse extranodal involvement | |
| Risk Stratification (Early Stage) | **Favorable**: Age <50, ESR <50, =3 sites, No bulky disease; **Unfavorable**: Any risk factor | |
| IPS Score (Advanced Stage) | Albumin <4, Hb <10.5, Male, Age =45, Stage IV, WBC =15k, Lymphocyte <600 or <8% | |

## Hodgkin Lymphoma - Treatment
| Stage | Treatment | URL |
|-------|-----------|-----|
| **Early Stage Favorable** | **ABVD x 2 cycles + Involved-site radiation (ISRT)** | https://www.nccn.org/hodgkin |
| **Early Stage Unfavorable** | **ABVD x 4 cycles + ISRT** OR **ABVD x 2 + AVD x 4** (if PET-2 negative) | |
| **Advanced Stage** | **ABVD x 6 cycles** OR **A+AVD** (Brentuximab + AVD) x 6 cycles | |
| **Relapsed/Refractory** | **Salvage chemotherapy** (ICE, DHAP, GDP, Bendamustine) ? **High-dose chemotherapy + Autologous HSCT**; **Brentuximab vedotin**, **Pembrolizumab/Nivolumab** (checkpoint inhibitors) | |

## Diffuse Large B-Cell Lymphoma (DLBCL)
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | **Most common NHL** (30-40%); Median age 64 | https://www.lls.org/dlbcl |
| Presentation | Rapidly enlarging lymph nodes, B symptoms, Extranodal disease (GI, CNS, Bone) | |
| Immunophenotype | CD19+, CD20+, CD22+, CD79a+, PAX5+ | |
| Cell of Origin | **GCB** (Germinal Center B-cell) - better prognosis; **ABC** (Activated B-cell) - worse prognosis | |
| Double-Hit Lymphoma | **MYC + BCL2 and/or BCL6 rearrangements**; Aggressive, Poor prognosis | |
| Staging | **PET-CT** (mandatory); Bone marrow biopsy; **CNS prophylaxis** (high-risk: testicular, Breast, Sinus, Renal, =2 extranodal sites) | |

## DLBCL - Treatment
| Setting | Treatment | URL |
|---------|-----------|-----|
| **First-Line** | **R-CHOP** (Rituximab + Cyclophosphamide, Doxorubicin, Vincristine, Prednisone) x 6 cycles | https://www.nccn.org/b-cell |
| **Double-Hit Lymphoma** | **DA-EPOCH-R** (Dose-adjusted EPOCH-R) | |
| **Relapsed/Refractory** | **Salvage chemotherapy** (R-ICE, R-DHAP, R-GDP) ? **High-dose chemotherapy + Autologous HSCT** (if chemosensitive) | |
| **CAR T-Cell Therapy** | **Axicabtagene ciloleucel**, **Lisocabtagene maraleucel**, **Tisagenlecleucel** (relapsed/refractory after =2 lines) | |
| **Bispecific Antibodies** | **Epcoritamab**, **Glofitamab** (CD20 x CD3) | |
| **Antibody-Drug Conjugates** | **Polatuzumab vedotin** (anti-CD79b), **Loncastuximab tesirine** (anti-CD19) | |

## Follicular Lymphoma (FL)
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | Second most common NHL (20-30%); Median age 60 | https://www.lls.org/follicular-lymphoma |
| Genetics | **t(14;18)/IGH-BCL2** (85-90%) | |
| Immunophenotype | CD19+, CD20+, CD10+, BCL2+, BCL6+, CD5-, CD23- | |
| Grading | **Grade 1-2**: Indolent; **Grade 3A**: Aggressive?; **Grade 3B**: Treat like DLBCL | |

## Follicular Lymphoma - Treatment
| Stage/Burden | Treatment | URL |
|--------------|-----------|-----|
| **Low Tumor Burden** | **Observation** ("watch and wait") OR **Rituximab monotherapy** | https://www.nccn.org/b-cell |
| **High Tumor Burden** | **Bendamustine + Rituximab (BR)** OR **R-CHOP** OR **Obinutuzumab + Chemotherapy** | |
| **Maintenance** | **Rituximab q2 months x 2 years** (if initial response) OR **Obinutuzumab q2 months x 2 years** | |
| **Relapsed** | Repeat initial therapy if late relapse (>24 months); **Lenalidomide + Rituximab (R2)**; **PI3K inhibitors** (Idelalisib, Copanlisib, Umbralisib, Duvelisib); **Tazemetostat** (EZH2-mutated); **CAR T-Cell**; **Bispecific antibodies** | |

## Mantle Cell Lymphoma (MCL)
| Component | Description | URL |
|-----------|-------------|-----|
| Genetics | **t(11;14)/CCND1-IGH** (cyclin D1 overexpression) | https://www.lls.org/mantle-cell-lymphoma |
| Immunophenotype | CD19+, CD20+, **CD5+**, CD23-, **Cyclin D1+**, SOX11+ | |
| Clinical Variants | **Classical** (aggressive, Nodal), **Leukemic non-nodal** (indolent, Splenomegaly, PB involvement) | |
| First-Line (Transplant-Eligible) | **R-DHAP alternating with R-CHOP** ? **High-dose cytarabine + Autologous HSCT** ? **Rituximab maintenance** | |
| First-Line (Transplant-Ineligible) | **Bendamustine + Rituximab** OR **VR-CAP** (Bortezomib + R-CHOP) ? Rituximab maintenance | |
| Relapsed | **BTK inhibitors** (Acalabrutinib, Zanubrutinib, Ibrutinib, Pirtobrutinib); **Venetoclax**; **CAR T-Cell** (Brexucabtagene autoleucel); **Bispecific antibodies** | |

## Burkitt Lymphoma
| Component | Description | URL |
|-----------|-------------|-----|
| Genetics | **t(8;14)/MYC-IGH** (80%), t(2;8) or t(8;22) | https://www.lls.org/burkitt-lymphoma |
| Variants | **Endemic** (Africa, Jaw, EBV+), **Sporadic** (Abdomen, EBV-), **Immunodeficiency-associated** (HIV) | |
| Morphology | **Starry sky** appearance (tingible-body macrophages) | |
| Immunophenotype | CD19+, CD20+, CD10+, BCL6+, **Ki-67 ~100%**, BCL2-, TdT- | |
