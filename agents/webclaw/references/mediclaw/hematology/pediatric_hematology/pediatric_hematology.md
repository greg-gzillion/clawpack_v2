## Physiologic Anemia of Infancy
| Component | Description | URL |
|-----------|-------------|-----|
| Timing | **Nadir at 6-9 weeks** (term), **3-6 weeks** (preterm) | https://www.aap.org |
| Hb Nadir | **9.5-11 g/dL** (term), **7-9 g/dL** (preterm) | |
| Mechanism | Transition from HbF to HbA + Decreased erythropoietin + Rapid growth | |
| Management | **Observation** (term infants); **Iron supplementation**; Transfusion only if symptomatic or severe | |

## Neonatal Alloimmune Thrombocytopenia (NAIT)
| Component | Description | URL |
|-----------|-------------|-----|
| Pathophysiology | Maternal antibodies against **paternal platelet antigens** (most commonly **HPA-1a**) ? Fetal/neonatal thrombocytopenia | https://www.ash.org/NAIT |
| Presentation | **Petechiae, Purpura** at birth or within hours; **Intracranial hemorrhage** (10-20% - can occur in utero) | |
| Diagnosis | **Neonatal thrombocytopenia**, Maternal anti-platelet antibodies, Parental platelet antigen typing | |
| Management | **IVIG 1 g/kg/day x 1-2 days**; **Platelet transfusion** (HPA-compatible or random donor); **Cranial ultrasound** | |

## Hemolytic Disease of the Newborn (HDN)
| Component | Description | URL |
|-----------|-------------|-----|
| Pathophysiology | Maternal IgG antibodies cross placenta ? Attack fetal RBC antigens | https://www.acog.org/HDN |
| **Rh(D) HDN** | Rh-negative mother sensitized to Rh(D) ? **Hydrops fetalis**, Severe anemia, Jaundice | |
| **ABO HDN** | Mother O, Infant A or B ? Mild-moderate jaundice, Anemia | |
| **Prevention** | **RhoGAM** (anti-D immune globulin) at 28 weeks and within 72 hours of delivery | |
| **Management** | **Phototherapy**, **IVIG**, **Exchange transfusion** (if severe), **Intrauterine transfusion** (for hydrops) | |

## Immune Thrombocytopenia (ITP) in Children
| Component | Description | URL |
|-----------|-------------|-----|
| Epidemiology | Peak age 2-5 years; Often post-viral | https://www.ash.org/pediatric-ITP |
| Presentation | **Acute onset petechiae, Purpura**, Epistaxis, Gingival bleeding; **Well-appearing child** | |
| Diagnosis | **Isolated thrombocytopenia**; Normal Hb, WBC; Peripheral smear: Large platelets | |
| Natural History | **80% resolve spontaneously within 6 months** | |
| Management | **Observation** (if platelets >20k and no significant bleeding); **IVIG** 0.8-1 g/kg x 1; **Anti-D** (Rh+, Non-splenectomized); **Corticosteroids** (short course) | |
| Chronic ITP | >12 months duration; Manage similarly to adult ITP | |

## Transient Erythroblastopenia of Childhood (TEC)
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | **Self-limited pure red cell aplasia** in young children (6 months - 4 years) | https://www.ash.org/TEC |
| Pathophysiology | Post-viral suppression of erythropoiesis (often Parvovirus B19) | |
| Presentation | **Pallor**, Fatigue; Well-appearing | |
| Labs | **Normocytic anemia**, **Low reticulocytes**, Normal WBC and platelets | |
| Differential | **Diamond-Blackfan anemia** (younger age, Congenital anomalies, Elevated HbF, Elevated eADA) | |
| Management | **Observation** (most recover in 1-2 months); **Transfusion** if symptomatic | |

## Diamond-Blackfan Anemia (DBA)
| Component | Description | URL |
|-----------|-------------|-----|
| Genetics | **Ribosomal protein gene mutations** (RPS19 most common); Autosomal dominant | https://www.dbafoundation.org |
| Presentation | **Macrocytic anemia** presenting in first year of life, **Congenital anomalies** (craniofacial, Thumb, Cardiac, Renal), Short stature | |
| Labs | **Macrocytic anemia**, **Reticulocytopenia**, **? HbF**, **? eADA** | |
| Management | **Corticosteroids** (first-line); **Chronic transfusions** + Iron chelation; **Allogeneic HSCT** (curative) | |
