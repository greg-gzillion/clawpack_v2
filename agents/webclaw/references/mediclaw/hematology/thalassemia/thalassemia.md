# Hematology - Thalassemia

## Thalassemia - Overview
| Component | Description | URL |
|-----------|-------------|-----|
| Definition | Decreased or absent synthesis of globin chains (a or ▀) | https://www.thalassemia.org |
| Pathophysiology | Imbalanced globin synthesis ? Precipitation of excess chains ? Ineffective erythropoiesis + Hemolysis | |
| Epidemiology | Mediterranean, Middle East, Southeast Asia, Africa, India | |

## Alpha-Thalassemia
| Syndrome | Genotype | a-Globin Genes | Clinical Features | URL |
|----------|----------|----------------|-------------------|-----|
| **Silent Carrier** | -a/aa | 3 functional | Asymptomatic; Normal CBC | https://www.cdc.gov/ncbddd/thalassemia |
| **Alpha-Thalassemia Trait** | --/aa (cis) or -a/-a (trans) | 2 functional | Mild microcytic anemia; **Cis form important for genetic counseling** | |
| **HbH Disease** | --/-a | 1 functional | Moderate-severe hemolytic anemia, Splenomegaly, **HbH on electrophoresis** | |
| **Hb Bart's Hydrops Fetalis** | --/-- | 0 functional | **Lethal in utero or at birth**; Maternal mirror syndrome | |

## Beta-Thalassemia
| Syndrome | Genotype | ▀-Globin Synthesis | Clinical Features | URL |
|----------|----------|---------------------|-------------------|-----|
| **Silent Carrier** | ▀+/▀ | Mild reduction | Asymptomatic; Normal HbA2 | https://www.cdc.gov/ncbddd/thalassemia |
| **Beta-Thalassemia Trait (Minor)** | ▀0/▀ or ▀+/▀ | Moderate reduction | Mild microcytic anemia, **Elevated HbA2 (>3.5%)**, Elevated HbF (variable) | |
| **Beta-Thalassemia Intermedia** | ▀+/▀+ or ▀0/▀+ | Significant reduction | Moderate-severe anemia, May require occasional transfusions, Iron overload | |
| **Beta-Thalassemia Major (Cooley's Anemia)** | ▀0/▀0 or ▀0/▀+ | Severe reduction | **Transfusion-dependent**, Severe anemia, Growth failure, Bony deformities, Iron overload | |

## Beta-Thalassemia Major - Management
| Component | Description | URL |
|-----------|-------------|-----|
| **Transfusion** | **Chronic transfusion program** (goal Hb 9-10.5 g/dL pre-transfusion); Every 2-4 weeks | https://thalassemia.org |
| **Iron Chelation** | **Start after 10-20 transfusions or ferritin >1000**; Deferoxamine (IV/SQ), Deferasirox (PO), Deferiprone (PO) | |
| **Splenectomy** | Indicated for hypersplenism, Increased transfusion requirements | |
| **Endocrine Monitoring** | Hypogonadism, Hypothyroidism, Diabetes, Growth failure, Osteoporosis | |
| **Cardiac Monitoring** | Iron-induced cardiomyopathy (leading cause of death); MRI T2* | |
| **Hepatitis Monitoring** | HBV, HCV (transfusion-related) | |
| **Hematopoietic Stem Cell Transplant** | **Curative**; Best outcomes in young children with matched sibling donor | |
| **Gene Therapy** | **FDA-approved (Zynteglo)**; Autologous CD34+ cells transduced with functional ▀-globin gene | |

## Thalassemia Intermedia - Management
| Component | Description | URL |
|-----------|-------------|-----|
| Transfusion | Occasional or none; Avoid if possible (increases iron overload) | https://thalassemia.org |
| Hydroxyurea | Increases HbF; Reduces transfusion need | |
| Splenectomy | For hypersplenism | |
| Iron Chelation | May be needed even without transfusions (increased absorption) | |
| Complications | **Extramedullary hematopoiesis**, **Pulmonary hypertension**, **Leg ulcers**, **Thrombosis** | |

## Genetic Counseling for Thalassemia
| Scenario | Risk | Recommendation | URL |
|----------|------|----------------|-----|
| **Both parents ▀-thalassemia trait** | 25% ▀-thalassemia major; 50% trait; 25% normal | **Prenatal diagnosis**, PGD | https://www.acog.org |
| **One parent a-thalassemia trait (cis), One normal** | 50% a-thalassemia trait (cis); 50% normal | Counsel about HbH disease risk in offspring | |
| **Both parents a-thalassemia trait (cis)** | 25% Hb Bart's hydrops; 50% trait (cis); 25% normal | **High-risk pregnancy**; Prenatal diagnosis | |
