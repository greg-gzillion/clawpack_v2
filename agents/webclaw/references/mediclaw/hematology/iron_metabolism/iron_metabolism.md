## Hemochromatosis - Diagnosis and Management
| Component | Description | URL |
|-----------|-------------|-----|
| **Screening** | Transferrin saturation >45% + Elevated ferritin | https://www.aasld.org/hemochromatosis |
| **Genetic Testing** | HFE gene (C282Y, H63D) | |
| **Liver Biopsy** | If ferritin >1000, Elevated LFTs, Age >40 | |
| **Treatment** | Phlebotomy weekly until ferritin 50-100 ng/mL | |
| **Maintenance** | Phlebotomy q2-4 months | |
| **Family Screening** | First-degree relatives: HFE genotyping + Iron studies | |
