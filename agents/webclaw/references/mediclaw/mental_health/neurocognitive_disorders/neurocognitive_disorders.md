# Mental Health - Neurocognitive Disorders

## Delirium
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5-TR Criteria** | **A. Disturbance in attention and awareness**; **B. Develops over hours-days**, **Fluctuating course**; **C. Additional cognitive disturbance** (memory, Orientation, Language, Perception) | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Mnemonic (I WATCH DEATH)** | **I**nfection, **W**ithdrawal, **A**cute metabolic, **T**rauma, **C**NS pathology, **H**ypoxia, **D**eficiencies, **E**ndocrine, **A**cute vascular, **T**oxins/drugs, **H**eavy metals | https://www.ncbi.nlm.nih.gov/books/NBK470239/ |
| **Confusion Assessment Method (CAM)** | **1. Acute onset and fluctuating course** + **2. Inattention** + **3. Disorganized thinking** OR **4. Altered level of consciousness** | https://www.hospitalelderlifeprogram.org/delirium-instruments |
| **Treatment** | **Treat underlying cause**, **Environmental modifications**, **Antipsychotics** (haloperidol) for severe agitation | https://www.ncbi.nlm.nih.gov/books/NBK470239/ |

## Delirium vs Dementia
| Feature | Delirium | Dementia | URL |
|---------|----------|----------|-----|
| **Onset** | **Acute** (hours-days) | **Insidious** (months-years) | https://www.nia.nih.gov/health/delirium |
| **Course** | **Fluctuating** | **Progressive** | https://www.nia.nih.gov/health/delirium |
| **Attention** | **Impaired** | **Preserved** (until late) | https://www.nia.nih.gov/health/delirium |
| **Consciousness** | **Altered** | **Clear** (until late) | https://www.nia.nih.gov/health/delirium |
| **Reversibility** | **Reversible** | **Irreversible** | https://www.nia.nih.gov/health/delirium |

## Major Neurocognitive Disorder (Dementia)
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5-TR Criteria** | **A. Significant cognitive decline** in =1 domain; **B. Interferes with independence**; **C. Not exclusively during delirium**; **D. Not better explained by another disorder** | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Cognitive Domains** | **Complex attention**, **Executive function**, **Learning and memory**, **Language**, **Perceptual-motor**, **Social cognition** | https://www.psychiatry.org/psychiatrists/practice/dsm |

## Lewy Body Dementia
| Component | Description | URL |
|-----------|-------------|-----|
| **Core Features** | **Fluctuating cognition**, **Recurrent visual hallucinations**, **REM sleep behavior disorder**, **Parkinsonism** | https://www.ninds.nih.gov/health-information/disorders/lewy-body-dementia |
| **Supportive Features** | **Neuroleptic sensitivity**, **Falls/Syncope**, **Autonomic dysfunction**, **Depression**, **Delusions** | https://www.ninds.nih.gov/health-information/disorders/lewy-body-dementia |

## Frontotemporal Dementia (FTD)
| Variant | Key Features | URL |
|---------|--------------|-----|
| **Behavioral Variant (bvFTD)** | **Disinhibition**, **Apathy**, **Loss of empathy**, **Compulsive behaviors**, **Hyperorality** | https://www.ninds.nih.gov/health-information/disorders/frontotemporal-dementia |
| **Semantic Variant PPA** | **Fluent aphasia**, **Loss of word meaning**, **Impaired object recognition** | https://www.ninds.nih.gov/health-information/disorders/frontotemporal-dementia |
| **Nonfluent/Agrammatic PPA** | **Effortful, halting speech**, **Agrammatism**, **Apraxia of speech** | https://www.ninds.nih.gov/health-information/disorders/frontotemporal-dementia |
| **Treatment** | **SSRI** (behavioral symptoms), **No disease-modifying therapy** | https://www.theaftd.org |

## Mild Neurocognitive Disorder (Mild Cognitive Impairment - MCI)
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5-TR Criteria** | **A. Modest cognitive decline**; **B. Does NOT interfere with independence**; **C. Not exclusively during delirium** | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Progression** | **10-15% per year** progress to dementia | https://www.alz.org/alzheimers-dementia/what-is-dementia/related_conditions/mild-cognitive-impairment |
| **Treatment** | **No FDA-approved medications**; **Cognitive stimulation**, **Exercise**, **Risk factor modification** | https://www.alz.org/alzheimers-dementia/what-is-dementia/related_conditions/mild-cognitive-impairment |
