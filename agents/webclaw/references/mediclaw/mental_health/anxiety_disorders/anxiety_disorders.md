# Mental Health - Anxiety Disorders

## Generalized Anxiety Disorder (GAD)
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5-TR Criteria** | **Excessive anxiety and worry** occurring **more days than not for =6 months** + =3 of: Restlessness, Fatigue, Poor concentration, Irritability, Muscle tension, Sleep disturbance | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Prevalence** | Lifetime: 5-6% | https://www.nimh.nih.gov/health/statistics/generalized-anxiety-disorder |
| **Treatment** | **SSRI/SNRI** (first-line), **CBT**, **Buspirone**, **Pregabalin** | https://www.apa.org/anxiety-guideline |

## Panic Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5-TR Criteria** | **Recurrent unexpected panic attacks** + =1 month of **persistent worry about additional attacks** or **maladaptive behavioral changes** | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Panic Attack** | Abrupt surge of intense fear with =4: Palpitations, Sweating, Trembling, SOB, Choking, Chest pain, Nausea, Dizziness, Chills/heat, Paresthesias, Derealization, Fear of losing control/dying | https://www.nimh.nih.gov/health/topics/panic-disorder |
| **Prevalence** | Lifetime: 3-5% | https://www.nimh.nih.gov/health/statistics/panic-disorder |
| **Treatment** | **SSRI** (start low, go slow), **CBT**, **Benzodiazepines** (short-term only) | https://www.apa.org/panic-disorder-guideline |

## Agoraphobia
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5-TR Criteria** | **Fear or anxiety about =2 of**: Public transportation, Open spaces, Enclosed spaces, Standing in line/crowds, Being outside home alone | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Prevalence** | Lifetime: 1-2% | https://www.nimh.nih.gov/health/statistics/agoraphobia |
| **Treatment** | **Exposure therapy**, **CBT**, **SSRI** | https://www.apa.org/anxiety-guideline |

## Social Anxiety Disorder (Social Phobia)
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5-TR Criteria** | **Fear or anxiety about social situations** where individual may be scrutinized; Fear of acting in way that will be negatively evaluated | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Prevalence** | Lifetime: 7-12% | https://www.nimh.nih.gov/health/statistics/social-anxiety-disorder |
| **Treatment** | **SSRI/SNRI** (first-line), **CBT** (exposure), **Performance-only**: Beta-blocker or Benzodiazepine PRN | https://www.apa.org/anxiety-guideline |

## Specific Phobia
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5-TR Criteria** | **Marked fear or anxiety about specific object/situation** (animals, Natural environment, Blood-injection-injury, Situational, Other) | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Prevalence** | Lifetime: 10-13% | https://www.nimh.nih.gov/health/statistics/specific-phobia |
| **Treatment** | **Exposure therapy** (first-line), **CBT** | https://www.apa.org/anxiety-guideline |

## Separation Anxiety Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5-TR Criteria** | **Excessive fear or anxiety concerning separation from attachment figures**; Onset typically in childhood but can persist into adulthood | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Treatment** | **CBT**, **Parent-child interaction therapy**, **SSRI** (if severe) | https://www.aacap.org/AACAP/Families_and_Youth/Facts_for_Families/FFF-Guide/Children-Who-Won't-Go-To-School-(Separation-Anxiety)-007.aspx |

## Selective Mutism
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5-TR Criteria** | **Consistent failure to speak in specific social situations** despite speaking in other situations; Duration =1 month | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Treatment** | **Behavioral therapy**, **CBT**, **SSRI** (if severe) | https://www.nimh.nih.gov/health/topics/anxiety-disorders |
