# Mental Health - Schizophrenia Spectrum and Other Psychotic Disorders

## Schizophrenia
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5-TR Criteria** | **=2 symptoms for =1 month** (=6 months total) with **=1 core symptom**: **Delusions**, **Hallucinations**, **Disorganized speech** | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Positive Symptoms** | **Delusions** (persecutory most common), **Hallucinations** (auditory most common), **Disorganized speech/behavior** | https://www.nimh.nih.gov/health/topics/schizophrenia |
| **Negative Symptoms** | **Affective flattening**, **Alogia**, **Avolition**, **Anhedonia**, **Asociality** | https://www.nimh.nih.gov/health/topics/schizophrenia |
| **Cognitive Symptoms** | Impaired attention, Memory, Executive function, Processing speed | https://www.nimh.nih.gov/health/topics/schizophrenia |
| **Prevalence** | Lifetime: 0.3-0.7% | https://www.nimh.nih.gov/health/statistics/schizophrenia |
| **Age of Onset** | Men: 18-25 years; Women: 25-35 years (second peak post-menopause) | https://www.nimh.nih.gov/health/statistics/schizophrenia |

## Clozapine REMS Program
| Component | Requirement | URL |
|-----------|-------------|-----|
| **Baseline ANC** | **ANC =1500/無** (=1000/無 for benign ethnic neutropenia) | https://www.clozapinerems.com |
| **Monitoring Frequency** | **Weekly x6 months** ? **Every 2 weeks x6 months** ? **Monthly** thereafter | https://www.clozapinerems.com |
| **ANC 1000-1499** | Continue clozapine, **Monitor ANC 3x/week** until recovery | https://www.clozapinerems.com |
| **ANC <1000** | **STOP clozapine**, Monitor daily, Consult hematology | https://www.clozapinerems.com |
| **Indications for Clozapine** | **Treatment-resistant schizophrenia**, **Suicidality**, **Tardive dyskinesia**, **Aggression** | https://www.psychiatry.org/psychiatrists/practice/clinical-practice-guidelines |

## Schizoaffective Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5-TR Criteria** | **Schizophrenia criteria + Major mood episode** (depressive or manic); **Delusions/hallucinations for =2 weeks without mood symptoms** | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Treatment** | **Antipsychotic + Mood stabilizer** or **Antipsychotic + Antidepressant** | https://www.psychiatry.org/psychiatrists/practice/clinical-practice-guidelines |

## Delusional Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5-TR Criteria** | **=1 delusion for =1 month**; **No prominent hallucinations**; **Functioning not markedly impaired**; **Never met schizophrenia criteria** | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Types** | **Erotomanic**, **Grandiose**, **Jealous**, **Persecutory**, **Somatic**, **Mixed** | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Treatment** | **Antipsychotics** (often poor insight/adherence), **CBT for psychosis** | https://www.psychiatry.org/psychiatrists/practice/clinical-practice-guidelines |

## Brief Psychotic Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5-TR Criteria** | **=1 psychotic symptom**; **Duration =1 day but <1 month**; **Full return to premorbid functioning** | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Treatment** | **Antipsychotics** (short-term), **Benzodiazepines** (if catatonic), **Monitor for progression to schizophrenia** | https://www.psychiatry.org/psychiatrists/practice/clinical-practice-guidelines |

## Catatonia
| Component | Description | URL |
|-----------|-------------|-----|
