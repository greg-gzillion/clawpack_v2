# Mental Health - Sleep-Wake Disorders

## Insomnia Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5-TR Criteria** | **Dissatisfaction with sleep quantity/quality** with =1 of: **Difficulty initiating sleep**, **Difficulty maintaining sleep**, **Early-morning awakening**; **=3 nights/week for =3 months**; **Adequate opportunity for sleep** | https://www.psychiatry.org/psychiatrists/practice/dsm |
| **Prevalence** | 10-30% of adults (chronic insomnia 6-10%) | https://www.nhlbi.nih.gov/health/insomnia |
| **First-Line Treatment** | **CBT-I** (Cognitive Behavioral Therapy for Insomnia) | https://www.aasm.org/clinical-resources/practice-standards/practice-guidelines/ |

## CBT-I Components
| Component | Description | URL |
|-----------|-------------|-----|
| **Stimulus Control** | **Bed only for sleep and sex**; Get out of bed if unable to sleep; Maintain regular wake time | https://www.aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Sleep Restriction** | **Limit time in bed to average sleep time** (minimum 5-6 hours) | https://www.aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Cognitive Therapy** | Challenge dysfunctional beliefs about sleep | https://www.aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Relaxation Training** | Progressive muscle relaxation, Deep breathing, Mindfulness | https://www.aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Sleep Hygiene** | Consistent schedule, Dark/cool room, No screens before bed, Avoid caffeine/alcohol | https://www.cdc.gov/sleep/about_sleep/sleep_hygiene.html |

## Obstructive Sleep Apnea Hypopnea (OSA)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Recurrent upper airway collapse during sleep causing hypoxemia and arousals | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **Risk Factors** | **Obesity**, Male, Age >50, Craniofacial abnormalities | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **STOP-BANG Score** | **S**noring, **T**ired, **O**bserved apnea, **P**ressure (HTN), **B**MI >35, **A**ge >50, **N**eck >40cm (M) or >38cm (F), **G**ender (male) | https://www.mdcalc.com/calc/3992/stop-bang-score-obstructive-sleep-apnea |
| **Diagnosis** | **Polysomnography** (AHI =5) or **Home Sleep Apnea Test** | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **AHI Severity** | **Mild**: 5-15; **Moderate**: 15-30; **Severe**: >30 | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Treatment** | **CPAP** (first-line), **Weight loss**, **Oral appliance**, **Positional therapy** | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |

## Narcolepsy
| Component | Description | URL |
|-----------|-------------|-----|
| **Type 1** | **Excessive daytime sleepiness** + **Cataplexy** + **Low CSF hypocretin-1** | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Type 2** | **Excessive daytime sleepiness** without cataplexy | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Tetrad** | **Excessive daytime sleepiness**, **Cataplexy**, **Sleep paralysis**, **Hypnagogic/Hypnopompic hallucinations** | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Diagnosis** | **Polysomnography + Multiple Sleep Latency Test (MSLT)** (mean sleep latency <8 min, =2 SOREMPs) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Treatment** | **Modafinil/Armodafinil**, **Sodium oxybate**, **Stimulants**, **Pitolisant**, **Solriamfetol** | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/narcolepsy-medications |

## Restless Legs Syndrome (RLS)
| Component | Description | URL |
|-----------|-------------|-----|
| **URGE Criteria** | **U**rge to move legs, **R**est worsens, **G**ets better with activity, **E**vening/night predominance | https://www.ninds.nih.gov/health-information/disorders/restless-legs-syndrome |
| **Workup** | **Ferritin** (target >50-75 ng/mL) | https://www.ninds.nih.gov/health-information/disorders/restless-legs-syndrome |
| **Treatment** | **Iron replacement** if low ferritin; **Gabapentinoids** (first-line), **Dopamine agonists** (caution: augmentation) | https://www.ninds.nih.gov/health-information/disorders/restless-legs-syndrome |
