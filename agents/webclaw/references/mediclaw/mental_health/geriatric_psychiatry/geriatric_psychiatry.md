# Mental Health - Geriatric Psychiatry

## Geriatric-Specific Considerations
| Component | Description | URL |
|-----------|-------------|-----|
| **Medical Comorbidity** | **Rule out medical causes** of psychiatric symptoms | https://www.americangeriatrics.org |

## Depression in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| **Presentation** | **Often atypical**: Less sadness, More **anhedonia**, **Somatic complaints**, **Cognitive complaints**, **Irritability** | https://www.nia.nih.gov/health/depression-and-older-adults |
| **Screening** | **PHQ-2**, **PHQ-9**, **Geriatric Depression Scale (GDS)** | https://www.americangeriatrics.org |
| **Treatment** | **SSRI** (sertraline, escitalopram), **SNRI**, **Mirtazapine**, **CBT**, **IPT** | https://www.psychiatry.org/psychiatrists/practice/clinical-practice-guidelines |
| **ECT** | **Safe and effective** in older adults; Consider for severe/treatment-resistant depression | https://www.psychiatry.org/patients-families/ect |

## Anxiety in Older Adults
| Component | Description | URL |
|-----------|-------------|-----|
| **Common Disorders** | **GAD**, **Phobias** (especially agoraphobia), **PTSD** | https://www.nia.nih.gov/health/anxiety-disorders |
| **Medical Rule-Outs** | **Hyperthyroidism**, **Arrhythmias**, **COPD**, **Medication side effects** | https://www.americangeriatrics.org |
| **Treatment** | **SSRI** (start low, go slow), **CBT**, **Avoid benzodiazepines** | https://www.psychiatry.org/psychiatrists/practice/clinical-practice-guidelines |

## Psychosis in Older Adults
| Etiology | Key Features | Treatment | URL |
|----------|--------------|-----------|-----|
| **Delirium** | **Acute onset**, **Fluctuating course**, **Inattention** | **Treat underlying cause**, **Low-dose antipsychotics** | https://www.nia.nih.gov/health/delirium |
| **Dementia with Psychosis** | **Gradual onset**, **Visual hallucinations** (LBD), **Delusions** | **Cholinesterase inhibitors**, **Avoid antipsychotics** if possible; **Low-dose atypical** if necessary | https://www.alz.org |
| **Late-Onset Schizophrenia** | **Paranoid delusions**, **Auditory hallucinations** | **Low-dose atypical antipsychotics** | https://www.psychiatry.org/psychiatrists/practice/clinical-practice-guidelines |
| **Delusional Disorder** | **Non-bizarre delusions** | **Low-dose antipsychotics** (often poor insight) | https://www.psychiatry.org/psychiatrists/practice/clinical-practice-guidelines |
