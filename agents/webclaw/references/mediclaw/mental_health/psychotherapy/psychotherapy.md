# Mental Health - Psychotherapy

## Major Psychotherapy Modalities
| Modality | Description | Evidence-Based Indications | URL |
|----------|-------------|---------------------------|-----|
| **Cognitive Behavioral Therapy (CBT)** | Identify and change maladaptive thoughts and behaviors | **Depression**, **Anxiety**, **PTSD**, **OCD**, **Eating disorders**, **Insomnia** | https://www.apa.org/ptsd-guideline/patients-and-families/cognitive-behavioral |
| **Dialectical Behavior Therapy (DBT)** | Skills training in mindfulness, Distress tolerance, Emotion regulation, Interpersonal effectiveness | **Borderline personality disorder**, **Self-harm**, **Suicidality**, **Eating disorders** | https://behavioraltech.org/resources/faqs/dialectical-behavior-therapy-dbt/ |
| **Acceptance and Commitment Therapy (ACT)** | Accept difficult thoughts/feelings, Commit to values-based action | **Depression**, **Anxiety**, **Chronic pain**, **PTSD** | https://contextualscience.org/act |
| **Interpersonal Therapy (IPT)** | Focus on interpersonal relationships and social functioning | **Depression** (acute and maintenance), **Eating disorders** | https://iptinstitute.com |
| **Psychodynamic Therapy** | Explore unconscious conflicts and early life experiences | **Depression**, **Anxiety**, **Personality disorders** | https://www.apa.org/topics/psychotherapy/approaches |
| **Motivational Interviewing (MI)** | Enhance intrinsic motivation to change | **Substance use disorders**, **Health behavior change** | https://motivationalinterviewing.org |
| **Exposure and Response Prevention (ERP)** | Gradual exposure to feared stimuli without performing compulsions | **OCD**, **Phobias**, **PTSD** | https://iocdf.org/about-ocd/treatment/erp/ |
| **Eye Movement Desensitization and Reprocessing (EMDR)** | Bilateral stimulation while processing traumatic memories | **PTSD** | https://www.emdr.com |
| **Cognitive Processing Therapy (CPT)** | Challenge and modify trauma-related beliefs | **PTSD** | https://www.ptsd.va.gov/professional/treat/txessentials/cpt_manual.asp |
| **Prolonged Exposure (PE)** | Repeated exposure to trauma memories and reminders | **PTSD** | https://www.ptsd.va.gov/professional/treat/txessentials/prolonged_exposure_professionals.asp |
| **Behavioral Activation (BA)** | Increase engagement in rewarding activities | **Depression** | https://www.apa.org/depression-guideline |
| **Family-Focused Therapy (FFT)** | Psychoeducation, Communication skills, Problem-solving | **Bipolar disorder**, **Schizophrenia** | https://www.nami.org |

## Evidence-Based Psychotherapy by Disorder
| Disorder | First-Line Psychotherapy | URL |
|----------|-------------------------|-----|
| **Major Depressive Disorder** | **CBT**, **IPT**, **Behavioral Activation** | https://www.apa.org/depression-guideline |
| **Generalized Anxiety Disorder** | **CBT** | https://www.apa.org/anxiety-guideline |
| **Panic Disorder** | **CBT** (including interoceptive exposure) | https://www.apa.org/panic-disorder-guideline |
| **Social Anxiety Disorder** | **CBT** (exposure) | https://www.apa.org/anxiety-guideline |
| **PTSD** | **CPT**, **PE**, **EMDR**, **CBT** | https://www.healthquality.va.gov/guidelines/MH/ptsd/ |
| **OCD** | **ERP** | https://iocdf.org/about-ocd/treatment/erp/ |
| **Borderline Personality Disorder** | **DBT**, **MBT**, **TFP**, **Schema Therapy** | https://www.psychiatry.org/psychiatrists/practice/clinical-practice-guidelines |
| **Bipolar Disorder** | **CBT**, **IPSRT**, **FFT** (adjunctive to medication) | https://www.psychiatry.org/psychiatrists/practice/clinical-practice-guidelines |
| **Schizophrenia** | **CBTp**, **Social Skills Training**, **Family Psychoeducation** | https://www.psychiatry.org/psychiatrists/practice/clinical-practice-guidelines |
| **Eating Disorders** | **CBT-E**, **FBT** (adolescents) | https://www.psychiatry.org/psychiatrists/practice/clinical-practice-guidelines |
| **Insomnia** | **CBT-I** | https://www.aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Substance Use Disorders** | **CBT**, **MI**, **Contingency Management** | https://www.samhsa.gov |
