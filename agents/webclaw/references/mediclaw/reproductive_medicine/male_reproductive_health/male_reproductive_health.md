## Testosterone Deficiency (Hypogonadism)
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnosis** | **Symptoms** + **Low testosterone** (<300 ng/dL on 2 morning measurements) | https://www.auanet.org/guidelines/testosterone-deficiency |
| **Treatment** | **Testosterone replacement** (gel, injection, patch, pellets) | https://www.auanet.org/guidelines/testosterone-deficiency |
| **Fertility Considerations** | **Testosterone suppresses spermatogenesis**; **Use hCG or clomiphene** if fertility desired | https://www.auanet.org/guidelines/testosterone-deficiency |
