# Reproductive Medicine - Endometriosis

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Ectopic endometrial glands and stroma** outside uterus | https://www.asrm.org/topics/endometriosis/ |
| **Prevalence** | **~10%** of reproductive-age women; **30-50%** of infertile women | https://www.asrm.org/topics/endometriosis/ |
| **Symptoms** | **Dysmenorrhea**, **Dyspareunia**, **Chronic pelvic pain**, **Infertility** | https://www.asrm.org/topics/endometriosis/ |

## Treatment of Endometriosis-Associated Infertility
| Stage | Treatment | URL |
|-------|-----------|-----|
| **Stage I/II** | **Laparoscopic ablation/resection** + **Adhesiolysis** (improves pregnancy rates) | https://www.asrm.org/practice-guidance/practice-committee-documents/endometriosis-and-infertility/ |
| **Stage III/IV** | **Laparoscopic resection** (improves pregnancy rates), **IVF** (if unsuccessful or severe) | https://www.asrm.org/practice-guidance/practice-committee-documents/endometriosis-and-infertility/ |
| **Endometrioma** | **Cystectomy** (if >3-4 cm, Symptomatic, Prior to IVF) - **Caution: ovarian reserve damage** | https://www.asrm.org/practice-guidance/practice-committee-documents/endometriosis-and-infertility/ |
| **Medical Suppression** | **NOT effective** for infertility (delays pregnancy) | https://www.asrm.org/practice-guidance/practice-committee-documents/endometriosis-and-infertility/ |
