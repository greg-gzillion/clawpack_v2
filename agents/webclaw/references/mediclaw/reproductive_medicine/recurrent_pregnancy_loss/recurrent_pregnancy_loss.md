# Reproductive Medicine - Recurrent Pregnancy Loss (RPL)

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition (ASRM)** | **=2 pregnancy losses** (clinical pregnancies) | https://www.asrm.org/practice-guidance/practice-committee-documents/definitions-of-infertility-and-recurrent-pregnancy-loss/ |
| **Prevalence** | **~5%** of couples experience =2 losses; **~1%** experience =3 losses | https://www.asrm.org/topics/recurrent-pregnancy-loss/ |
| **Prognosis** | **>50-60%** have successful pregnancy with supportive care | https://www.asrm.org/topics/recurrent-pregnancy-loss/ |

## Etiology of RPL
| Category | Examples | Frequency | URL |
|----------|----------|-----------|-----|
| **Embryonic/Fetal** | **Aneuploidy** (most common cause of sporadic loss) | **50-60%** (sporadic), **Lower in RPL** | https://www.asrm.org/topics/recurrent-pregnancy-loss/ |
| **Uterine** | **Septate uterus**, **Fibroids**, **Polyps**, **Adhesions**, **Cervical insufficiency** | **10-15%** | https://www.asrm.org/topics/recurrent-pregnancy-loss/ |
| **Endocrine** | **Luteal phase defect**, **PCOS**, **Thyroid disease**, **Diabetes** | **5-10%** | https://www.asrm.org/topics/recurrent-pregnancy-loss/ |
| **Immunologic** | **Antiphospholipid syndrome (APS)** | **5-10%** | https://www.asrm.org/topics/recurrent-pregnancy-loss/ |
| **Genetic (Parental)** | **Balanced translocation** | **3-5%** | https://www.asrm.org/topics/recurrent-pregnancy-loss/ |
| **Unexplained** | **No identifiable cause** | **50%** | https://www.asrm.org/topics/recurrent-pregnancy-loss/ |

## RPL Evaluation (ASRM)
| Component | Tests | URL |
|-----------|-------|-----|
| **Uterine Cavity** | **Saline infusion sonography (SIS)** , **Hysteroscopy** | https://www.asrm.org/practice-guidance/practice-committee-documents/evaluation-and-treatment-of-recurrent-pregnancy-loss/ |
| **Parental Karyotype** | **Both partners** | https://www.asrm.org/practice-guidance/practice-committee-documents/evaluation-and-treatment-of-recurrent-pregnancy-loss/ |
| **Antiphospholipid Antibodies** | **Lupus anticoagulant**, **Anti-cardiolipin IgG/IgM**, **Anti-ß2-glycoprotein I IgG/IgM** | https://www.asrm.org/practice-guidance/practice-committee-documents/evaluation-and-treatment-of-recurrent-pregnancy-loss/ |
| **Thyroid Function** | **TSH**, **Thyroid antibodies** | https://www.asrm.org/practice-guidance/practice-committee-documents/evaluation-and-treatment-of-recurrent-pregnancy-loss/ |
| **NOT Routinely Recommended** | **Inherited thrombophilia**, **Immunologic testing** (except APS), **Endometrial biopsy**, **Sperm DNA fragmentation** | https://www.asrm.org/practice-guidance/practice-committee-documents/evaluation-and-treatment-of-recurrent-pregnancy-loss/ |

## Treatment of RPL
| Etiology | Treatment | URL |
|----------|-----------|-----|
| **Uterine Septum** | **Hysteroscopic metroplasty** | https://www.asrm.org/practice-guidance/practice-committee-documents/evaluation-and-treatment-of-recurrent-pregnancy-loss/ |
| **Antiphospholipid Syndrome** | **Low-dose aspirin + Prophylactic heparin** | https://www.asrm.org/practice-guidance/practice-committee-documents/evaluation-and-treatment-of-recurrent-pregnancy-loss/ |
| **Parental Balanced Translocation** | **PGT-SR** (preimplantation genetic testing for structural rearrangements) | https://www.asrm.org/practice-guidance/practice-committee-documents/evaluation-and-treatment-of-recurrent-pregnancy-loss/ |
| **Unexplained** | **Supportive care**, **Progesterone** (limited evidence), **Avoid aspirin/heparin** (no benefit) | https://www.asrm.org/practice-guidance/practice-committee-documents/evaluation-and-treatment-of-recurrent-pregnancy-loss/ |
