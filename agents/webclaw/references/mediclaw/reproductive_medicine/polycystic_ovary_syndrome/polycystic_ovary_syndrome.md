# Reproductive Medicine - Polycystic Ovary Syndrome (PCOS)

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Hyperandrogenism + Ovulatory dysfunction + Polycystic ovarian morphology** | https://www.asrm.org/topics/pcos/ |
| **Prevalence** | **~8-13%** of reproductive-age women | https://www.asrm.org/topics/pcos/ |
| **Most Common Cause** | **Anovulatory infertility** | https://www.asrm.org/topics/pcos/ |

## Diagnostic Criteria (Rotterdam, 2003)
| Criteria | Description | URL |
|----------|-------------|-----|
| **1. Oligo/Anovulation** | **<8 cycles/year** or **Cycle length >35 days** | https://www.asrm.org/topics/pcos/ |
| **2. Hyperandrogenism** | **Clinical** (hirsutism, acne, alopecia) or **Biochemical** (? Testosterone) | https://www.asrm.org/topics/pcos/ |
| **3. Polycystic Ovaries** | **=12 follicles 2-9 mm** and/or **Ovarian volume >10 mL** (on ultrasound) | https://www.asrm.org/topics/pcos/ |
| **Diagnosis** | **=2 of 3 criteria** (after excluding other causes) | https://www.asrm.org/topics/pcos/ |

## PCOS and Infertility
| Mechanism | Treatment | URL |
|-----------|-----------|-----|
| **Anovulation** | **Weight loss** (5-10%), **Letrozole** (first-line), **Clomiphene citrate**, **Gonadotropins** | https://www.asrm.org/practice-guidance/practice-committee-documents/role-of-metformin-for-ovulation-induction-in-infertile-patients-with-polycystic-ovary-syndrome/ |
| **Endometrial Dysfunction** | **Progesterone** (luteal support) | https://www.asrm.org/topics/pcos/ |
| **Obesity** | **Lifestyle modification**, **Bariatric surgery** | https://www.asrm.org/topics/pcos/ |

## Ovulation Induction in PCOS (ASRM Guidelines)
| Agent | Recommendation | URL |
|-------|----------------|-----|
| **Letrozole** | **First-line** (higher live birth rate than clomiphene) | https://www.asrm.org/practice-guidance/practice-committee-documents/role-of-metformin-for-ovulation-induction-in-infertile-patients-with-polycystic-ovary-syndrome/ |
| **Clomiphene Citrate** | **Alternative** (if letrozole unavailable or contraindicated) | https://www.asrm.org/practice-guidance/practice-committee-documents/role-of-metformin-for-ovulation-induction-in-infertile-patients-with-polycystic-ovary-syndrome/ |
| **Metformin** | **NOT first-line** alone; **May add to clomiphene** (limited benefit) | https://www.asrm.org/practice-guidance/practice-committee-documents/role-of-metformin-for-ovulation-induction-in-infertile-patients-with-polycystic-ovary-syndrome/ |
| **Gonadotropins** | **Second-line** (if oral agents fail) | https://www.asrm.org/topics/pcos/ |
| **Laparoscopic Ovarian Drilling** | **Second-line** (if clomiphene-resistant) | https://www.asrm.org/topics/ovarian-drilling/ |
| **IVF** | **Third-line** (if other treatments fail or other indications) | https://www.asrm.org/topics/pcos/ |
