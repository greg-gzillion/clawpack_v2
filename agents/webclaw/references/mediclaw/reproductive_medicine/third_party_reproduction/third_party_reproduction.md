# Reproductive Medicine - Third-Party Reproduction

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Use of gametes, embryos, or gestational carrier** from third party | https://www.asrm.org/topics/third-party-reproduction/ |
| **Types** | **Donor oocyte**, **Donor sperm**, **Donor embryo**, **Gestational carrier** | https://www.asrm.org/topics/third-party-reproduction/ |

## Donor Oocyte
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | **Diminished ovarian reserve**, **Premature ovarian insufficiency**, **Recurrent IVF failure**, **Genetic disease** | https://www.asrm.org/topics/donor-oocyte/ |
| **Donor Types** | **Known** (friend/family), **Anonymous**, **Semi-anonymous** | https://www.asrm.org/topics/donor-oocyte/ |
| **Screening (FDA)** | **Infectious disease** (HIV, HBV, HCV, Syphilis, CMV), **Genetic**, **Psychological** | https://www.fda.gov/vaccines-blood-biologics/tissue-tissue-products/donor-eligibility |
| **Success Rates** | **>50% live birth per transfer** (independent of recipient age) | https://www.sart.org/patients/success-rates/ |

## Donor Sperm
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | **Severe male factor**, **Azoospermia**, **Genetic disease**, **Single women**, **Same-sex couples** | https://www.asrm.org/topics/donor-sperm/ |
| **Donor Types** | **Known**, **Anonymous** (from sperm bank) | https://www.asrm.org/topics/donor-sperm/ |
| **Screening** | **Infectious disease**, **Genetic**, **Quarantine** (6 months) | https://www.fda.gov/vaccines-blood-biologics/tissue-tissue-products/donor-eligibility |

## Gestational Carrier (Surrogacy)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Woman carries pregnancy** for intended parents (no genetic link) | https://www.asrm.org/topics/gestational-carrier/ |
| **Indications** | **Absent uterus** (congenital, Hysterectomy), **Uterine factor**, **Medical contraindication to pregnancy** | https://www.asrm.org/topics/gestational-carrier/ |
| **Screening** | **Medical**, **Psychological**, **Obstetric history** (prior uncomplicated term delivery) | https://www.asrm.org/practice-guidance/practice-committee-documents/recommendations-for-practices-utilizing-gestational-carriers/ |
| **Legal** | **Contract**, **Parentage orders** (varies by state) | https://www.asrm.org/topics/gestational-carrier/ |

| Component | Description | URL |
