# Reproductive Medicine - Preimplantation Genetic Testing (PGT)

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Genetic testing of embryos** before transfer | https://www.asrm.org/topics/pgt/ |
| **Types** | **PGT-A** (aneuploidy), **PGT-M** (monogenic/single gene), **PGT-SR** (structural rearrangements) | https://www.asrm.org/topics/pgt/ |
| **Method** | **Trophectoderm biopsy** (day 5-6 blastocyst) | https://www.asrm.org/topics/pgt/ |

## Types of PGT
| Type | Indications | URL |
|------|-------------|-----|
| **PGT-A** (formerly PGS) | **Advanced maternal age**, **Recurrent implantation failure**, **Recurrent pregnancy loss**, **Severe male factor** | https://www.asrm.org/topics/pgt/ |
| **PGT-M** (formerly PGD) | **Single gene disorders** (CF, SMA, Huntington, BRCA), **HLA matching** | https://www.asrm.org/topics/pgt/ |
| **PGT-SR** | **Parental balanced translocation**, **Inversion**, **Deletion/Duplication** | https://www.asrm.org/topics/pgt/ |

## PGT-A Results
| Result | Interpretation | URL |
|--------|----------------|-----|
| **Euploid** | **Normal chromosome number** (46,XX or 46,XY) | https://www.asrm.org/topics/pgt/ |
| **Aneuploid** | **Abnormal chromosome number** (trisomy, monosomy) | https://www.asrm.org/topics/pgt/ |
| **Mosaic** | **Mixture of euploid and aneuploid cells** (low-level vs high-level) | https://www.asrm.org/practice-guidance/practice-committee-documents/clinical-management-of-mosaic-results-from-preimplantation-genetic-testing-for-aneuploidy-pgt-a/ |
| **Segmental** | **Partial chromosome abnormality** | https://www.asrm.org/topics/pgt/ |
| **No Result** | **Insufficient DNA** | https://www.asrm.org/topics/pgt/ |

## Mosaic Embryos
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Embryo with both normal and abnormal cells** | https://www.asrm.org/practice-guidance/practice-committee-documents/clinical-management-of-mosaic-results-from-preimplantation-genetic-testing-for-aneuploidy-pgt-a/ |
| **Low-Level Mosaic** | **<20-50%** abnormal cells (higher implantation potential) | https://www.asrm.org/practice-guidance/practice-committee-documents/clinical-management-of-mosaic-results-from-preimplantation-genetic-testing-for-aneuploidy-pgt-a/ |
| **High-Level Mosaic** | **>50-80%** abnormal cells (lower implantation potential) | https://www.asrm.org/practice-guidance/practice-committee-documents/clinical-management-of-mosaic-results-from-preimplantation-genetic-testing-for-aneuploidy-pgt-a/ |
| **Management** | **Genetic counseling**, **Consider transfer** if no euploid embryos, **Prioritize euploid** | https://www.asrm.org/practice-guidance/practice-committee-documents/clinical-management-of-mosaic-results-from-preimplantation-genetic-testing-for-aneuploidy-pgt-a/ |

## PGT Limitations
| Limitation | Description | URL |
|------------|-------------|-----|
| **Biopsy Risk** | **Small risk** of embryo damage | https://www.asrm.org/topics/pgt/ |
| **Mosaicism** | **May misclassify** mosaic embryos | https://www.asrm.org/topics/pgt/ |
| **No Result** | **~2-5%** of embryos | https://www.asrm.org/topics/pgt/ |
| **Cost** | **Adds significant cost** to IVF cycle | https://www.asrm.org/topics/pgt/ |
| **Does NOT Improve Live Birth** | **PGT-A does NOT increase cumulative live birth rate** per cycle (may reduce time to pregnancy) | https://www.asrm.org/topics/pgt/ |
