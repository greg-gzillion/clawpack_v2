# Reproductive Medicine - Female Infertility

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Prevalence** | **~10-15%** of couples | https://www.asrm.org/topics/female-infertility/ |
| **Causes** | **Ovulatory disorders** (40%), **Tubal/Pelvic factors** (30-40%), **Uterine factors** (10%), **Unexplained** (10-20%) | https://www.asrm.org/topics/female-infertility/ |

## Female Infertility Evaluation (ASRM)
| Component | Tests | URL |
|-----------|-------|-----|
| **Ovulatory Function** | **Menstrual history**, **Ovulation predictor kits**, **Mid-luteal progesterone**, **Basal body temperature** | https://www.asrm.org/practice-guidance/practice-committee-documents/optimal-evaluation-of-the-infertile-female/ |
| **Ovarian Reserve** | **AMH**, **AFC**, **Day 3 FSH/Estradiol** | https://www.asrm.org/practice-guidance/practice-committee-documents/optimal-evaluation-of-the-infertile-female/ |
| **Uterine Cavity** | **Saline infusion sonography (SIS)** , **Hysterosalpingography (HSG)** , **Hysteroscopy** | https://www.asrm.org/practice-guidance/practice-committee-documents/optimal-evaluation-of-the-infertile-female/ |
| **Tubal Patency** | **HSG**, **Laparoscopy with chromotubation** | https://www.asrm.org/practice-guidance/practice-committee-documents/optimal-evaluation-of-the-infertile-female/ |
| **Cervical Factor** | **Postcoital test** (rarely performed) | https://www.asrm.org/practice-guidance/practice-committee-documents/optimal-evaluation-of-the-infertile-female/ |

## Ovulatory Disorders (WHO Classification)
| Class | Description | Examples | URL |
|-------|-------------|----------|-----|
| **WHO I** | **Hypogonadotropic hypogonadal** (? FSH, ? LH, ? E2) | **Hypothalamic amenorrhea**, **Kallmann syndrome** | https://www.asrm.org/topics/ovulatory-disorders/ |
| **WHO II** | **Normogonadotropic normogonadal** (normal FSH, LH) | **PCOS** (most common) | https://www.asrm.org/topics/ovulatory-disorders/ |
| **WHO III** | **Hypergonadotropic hypogonadal** (? FSH, ? LH, ? E2) | **Premature ovarian insufficiency**, **Turner syndrome** | https://www.asrm.org/topics/ovulatory-disorders/ |

## Tubal Factor Infertility
| Cause | Evaluation | Treatment | URL |
|------|------------|-----------|-----|
| **Pelvic Inflammatory Disease (PID)** | **HSG**, **Laparoscopy** | **IVF** (proximal occlusion), **Tubal surgery** (distal occlusion) | https://www.asrm.org/topics/tubal-factor/ |
| **Endometriosis** | **Laparoscopy** | **Surgical resection**, **IVF** | https://www.asrm.org/topics/endometriosis/ |
| **Prior Tubal Ligation** | **HSG** (if reversal candidate) | **Tubal anastomosis** or **IVF** | https://www.asrm.org/topics/tubal-factor/ |

## Unexplained Infertility
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **No identifiable cause** after complete evaluation | https://www.asrm.org/topics/unexplained-infertility/ |
| **Treatment** | **Expectant management** (if age <35, Short duration), **Ovulation induction + IUI**, **IVF** | https://www.asrm.org/practice-guidance/practice-committee-documents/management-of-unexplained-infertility/ |
