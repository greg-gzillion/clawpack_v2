# Internal Medicine - Nephrology

## Acute Kidney Injury (AKI)
| Component | Description | URL |
|-----------|-------------|-----|
| **KDIGO Guidelines** | Acute Kidney Injury | https://kdigo.org/guidelines/acute-kidney-injury/ |
| **Definition** | ? Cr =0.3 mg/dL in 48h OR ? Cr =1.5x baseline OR UOP <0.5 mL/kg/h x6h | https://kdigo.org/guidelines/acute-kidney-injury/ |
| **Stages** | Stage 1: Cr 1.5-1.9x; Stage 2: 2.0-2.9x; Stage 3: =3.0x or RRT | https://kdigo.org/guidelines/acute-kidney-injury/ |
| **Prerenal vs ATN** | FeNa <1% vs >2%; UNa <20 vs >40 | https://www.kidney.org/atoz/content/acute-kidney-injury |

| Component | Description | URL |
|-----------|-------------|-----|

| **BP Goal** | **<130/80** (if albuminuria) | https://kdigo.org/guidelines/ckd-evaluation-and-management/ |
