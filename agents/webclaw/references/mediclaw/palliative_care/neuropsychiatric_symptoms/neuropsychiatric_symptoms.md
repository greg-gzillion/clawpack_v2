# Palliative Care - Neuropsychiatric Symptoms

## Delirium
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Acute**, **Fluctuating** disturbance in **attention** and **awareness** with **cognitive change** | https://www.capc.org/toolkits/delirium/ |
| **Subtypes** | **Hyperactive** (agitated), **Hypoactive** (lethargic - most common, Often missed), **Mixed** | https://www.capc.org/toolkits/delirium/ |
| **Etiology (DELIRIUM)** | **D**rugs (opioids, Anticholinergics, Benzodiazepines, Steroids), **E**lectrolytes, **L**iver failure, **I**nfection, **R**espiratory (hypoxia), **I**ntracranial, **U**remia, **M**etastases | https://www.capc.org/toolkits/delirium/ |
| **Reversible Causes** | **Urinary retention**, **Fecal impaction**, **Pain**, **Dehydration**, **Sensory deprivation** | https://www.capc.org/toolkits/delirium/ |
| **Non-Pharmacologic** | **Orientation** (clocks, Calendars), **Family presence**, **Sensory aids** (glasses, Hearing aids), **Sleep hygiene**, **Early mobilization** | https://www.capc.org/toolkits/delirium/ |
| **Pharmacologic (Hyperactive)** | **Haloperidol** 0.5-2 mg PO/IV/SC q1-4h PRN, **Quetiapine** 12.5-50 mg PO q12h, **Olanzapine** 2.5-5 mg PO/SL qHS | https://www.capc.org/toolkits/delirium/ |
| **Terminal Delirium** | **Haloperidol** or **Chlorpromazine**; **Benzodiazepines** if anxiety/restlessness; **Focus on comfort** | https://www.capc.org/toolkits/delirium/ |

## Depression
| Component | Description | URL |
|-----------|-------------|-----|
| **Screening** | **PHQ-2**: "Little interest or pleasure" + "Feeling down, Depressed, Hopeless" | https://www.capc.org/toolkits/depression/ |
| **Psychotherapy** | **Supportive counseling**, **CBT**, **Dignity therapy**, **Meaning-centered therapy** | https://www.capc.org/toolkits/depression/ |
| **Pharmacologic** | **Methylphenidate** 2.5-5 mg AM and noon (rapid onset, Days), **SSRIs** (sertraline, Escitalopram - weeks), **Mirtazapine** (sedating, Appetite stimulant) | https://www.capc.org/toolkits/depression/ |
| **Demoralization** | **Hopelessness**, **Loss of meaning**, **Desire for hastened death**; **Dignity therapy**, **Meaning-centered therapy** | https://www.capc.org/toolkits/depression/ |
