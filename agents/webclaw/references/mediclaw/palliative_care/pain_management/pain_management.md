## Opioid Prescribing in Palliative Care
| Principle | Description | URL |
|-----------|-------------|-----|
| **Route** | **Oral** preferred; **Subcutaneous** (alternative); **Transdermal** (stable pain); **Avoid IM** | https://www.capc.org/toolkits/pain-management/ |
| **Scheduled + PRN** | **Long-acting scheduled** + **Short-acting PRN** (10-20% of total daily dose) | https://www.capc.org/toolkits/pain-management/ |
| **Dose Escalation** | Increase by **25-50%** for uncontrolled pain | https://www.capc.org/toolkits/pain-management/ |
| **Opioid Rotation** | If toxicity or inadequate analgesia; Reduce new opioid by **25-50%** for incomplete cross-tolerance | https://www.capc.org/toolkits/pain-management/ |
| **Renal Impairment** | **Hydromorphone** or **Fentanyl** preferred (avoid morphine - metabolite accumulation) | https://www.capc.org/toolkits/pain-management/ |
| **Constipation** | **Scheduled stimulant laxative** (senna) + Stool softener (docusate) | https://www.capc.org/toolkits/pain-management/ |
| **Naloxone Co-Prescribing** | Consider for all patients on chronic opioids | https://www.capc.org/toolkits/pain-management/ |
