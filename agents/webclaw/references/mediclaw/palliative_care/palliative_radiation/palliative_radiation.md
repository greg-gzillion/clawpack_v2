# Palliative Care - Palliative Radiation

## Indications for Palliative Radiation
| Indication | Benefit | Typical Regimen | URL |
|------------|---------|-----------------|-----|
| **Bone Metastases** | **Pain relief** (60-80%) | **8 Gy x 1** or **20 Gy x 5** or **30 Gy x 10** | https://www.astro.org/Patient-Care/Radiation-Therapy-for-Palliative-Care |
| **Spinal Cord Compression** | **Preserve/restore function** | **30 Gy x 10** or **20 Gy x 5** | https://www.astro.org/Patient-Care/Radiation-Therapy-for-Palliative-Care |
| **Brain Metastases** | **Improve symptoms**, **Prevent decline** | **Whole brain**: 30 Gy x 10 or 20 Gy x 5; **SRS**: 15-24 Gy x 1 | https://www.astro.org/Patient-Care/Radiation-Therapy-for-Palliative-Care |
| **SVC Syndrome** | **Relieve obstruction** | **20-30 Gy x 5-10** | https://www.astro.org/Patient-Care/Radiation-Therapy-for-Palliative-Care |
| **Hemoptysis** | **Control bleeding** | **17-20 Gy x 2-5** | https://www.astro.org/Patient-Care/Radiation-Therapy-for-Palliative-Care |
| **Airway Obstruction** | **Relieve dyspnea** | **20-30 Gy x 5-10** | https://www.astro.org/Patient-Care/Radiation-Therapy-for-Palliative-Care |

## Single vs Multi-Fraction for Bone Metastases
| Regimen | Efficacy | Convenience | Cost | URL |
|---------|----------|-------------|------|-----|
| **8 Gy x 1** | **Equivalent pain relief** to multi-fraction | **Most convenient** | **Lowest cost** | https://www.astro.org/Patient-Care/Radiation-Therapy-for-Palliative-Care |
| **20 Gy x 5** | Equivalent pain relief | Less convenient | Higher cost | https://www.astro.org/Patient-Care/Radiation-Therapy-for-Palliative-Care |
| **30 Gy x 10** | Equivalent pain relief | Least convenient | Highest cost | https://www.astro.org/Patient-Care/Radiation-Therapy-for-Palliative-Care |
| **Re-Treatment** | More common after single fraction (20% vs 8%) | | | https://www.astro.org/Patient-Care/Radiation-Therapy-for-Palliative-Care |
