# Palliative Care - Hospice Care

## Hospice Eligibility (Medicare Hospice Benefit)
| Diagnosis | General Guidelines | URL |
|-----------|---------------------|-----|
| **Dementia** | **FAST 7C** (incontinence, Unable to ambulate, Minimal verbal communication) + One of: Aspiration pneumonia, Pyelonephritis, Sepsis, Stage 3-4 pressure injury, Weight loss >10% in 6 months | https://www.cms.gov/medicare/hospice |
| **Heart Failure** | **NYHA Class IV** + **EF =20%** (or preserved EF with symptoms) + Optimal medical therapy + Not candidate for advanced therapies | https://www.cms.gov/medicare/hospice |
| **COPD** | **FEV1 <30%** + Cor pulmonale + Hypoxemia (SpO2 =88%) + Hypercapnia (pCO2 =50) + Frequent hospitalizations | https://www.cms.gov/medicare/hospice |
| **Cancer** | Metastatic or locally advanced + Declining functional status + Refractory to treatment or not pursuing treatment | https://www.cms.gov/medicare/hospice |
| **Stroke** | **PPS =40%** + Poor nutritional status (albumin <2.5) + Aspiration pneumonia or pyelonephritis | https://www.cms.gov/medicare/hospice |
| **General Decline** | **PPS =40%** + Weight loss + Albumin <2.5 + Functional decline + Frequent infections/hospitalizations | https://www.cms.gov/medicare/hospice |

## Hospice Levels of Care
| Level | Description | Setting | URL |
|-------|-------------|---------|-----|
| **Routine Home Care** | Standard hospice care | **Home**, Nursing home, Assisted living | https://www.cms.gov/medicare/hospice |
| **Continuous Home Care** | **Skilled nursing care** for acute symptom crisis (=8 hours/day) | **Home** | https://www.cms.gov/medicare/hospice |
| **Inpatient Respite Care** | **Short-term** (up to 5 days) to relieve caregivers | **Hospice facility**, Nursing home | https://www.cms.gov/medicare/hospice |
| **General Inpatient Care** | **Acute symptom management** requiring hospital-level care | **Hospice facility**, Hospital | https://www.cms.gov/medicare/hospice |
