# Palliative Care - Communication Skills

## SPIKES Protocol (Breaking Bad News)
| Step | Action | Example | URL |
|------|--------|---------|-----|
| **S**etting | **Private**, **Quiet**, **Seated**, **Tissues**, **Family present** | "Let's sit down together." | https://www.vitaltalk.org/guides/breaking-bad-news/ |
| **P**erception | **Assess understanding** | "What have you been told so far?" | https://www.vitaltalk.org/guides/breaking-bad-news/ |
| **I**nvitation | **Ask permission** | "Would it be okay if I share the results with you?" | https://www.vitaltalk.org/guides/breaking-bad-news/ |
| **K**nowledge | **Warning shot + Information** | "I'm afraid I have some difficult news. The scan shows..." | https://www.vitaltalk.org/guides/breaking-bad-news/ |
| **E**mpathy | **Acknowledge emotion** | "I can see this is not what you were hoping to hear." | https://www.vitaltalk.org/guides/breaking-bad-news/ |
| **S**trategy | **Plan next steps** | "Let's talk about what happens next." | https://www.vitaltalk.org/guides/breaking-bad-news/ |

## Discussing Prognosis
| Step | Action | Example | URL |
|------|--------|---------|-----|
| **Ask permission** | - | "Would it be helpful to talk about what the future might look like?" | https://www.vitaltalk.org/guides/discussing-prognosis/ |
| **Use ranges** | Avoid specific numbers | "We're thinking months to a year, rather than years." | https://www.vitaltalk.org/guides/discussing-prognosis/ |
| **Acknowledge uncertainty** | - | "I wish I could be more exact, but it's hard to predict." | https://www.vitaltalk.org/guides/discussing-prognosis/ |
| **Hope for best, Prepare for worst** | - | "I hope you have many good months ahead, and I want to help you prepare for all possibilities." | https://www.vitaltalk.org/guides/discussing-prognosis/ |
