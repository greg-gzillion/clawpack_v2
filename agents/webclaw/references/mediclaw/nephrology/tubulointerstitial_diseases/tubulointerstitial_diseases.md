# Nephrology - Tubulointerstitial Diseases

## Acute Interstitial Nephritis (AIN)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Acute inflammatory infiltrate in renal interstitium causing AKI | https://www.ncbi.nlm.nih.gov/books/NBK559182/ |
| **Etiology** | **Drugs** (70-75%): NSAIDs, Penicillins, Cephalosporins, Sulfonamides, PPIs, Rifampin; **Infections** (10-15%): Legionella, Leptospira, EBV, CMV; **Autoimmune** (5-10%): Sarcoidosis, Sjögren, SLE, IgG4-RD | https://www.ncbi.nlm.nih.gov/books/NBK559182/ |
| **Classic Triad** | **Fever**, **Rash**, **Eosinophilia** (present in only 10-30%) | https://www.ncbi.nlm.nih.gov/books/NBK559182/ |
| **Laboratory Findings** | **Sterile pyuria**, **WBC casts**, **Eosinophiluria** (low sensitivity/specificity), **FeNa >2%** | https://www.ncbi.nlm.nih.gov/books/NBK559182/ |
| **Diagnosis** | **Kidney biopsy** (gold standard): Interstitial edema, Lymphoplasmacytic infiltrate, Tubulitis | https://www.ncbi.nlm.nih.gov/books/NBK559182/ |
| **Treatment** | **Discontinue offending drug**; **Corticosteroids** for severe AKI or no improvement after drug cessation | https://www.ncbi.nlm.nih.gov/books/NBK559182/ |

## Chronic Interstitial Nephritis (CIN)
| Cause | Key Features | Diagnosis | Treatment | URL |
|-------|--------------|-----------|-----------|-----|
| **Analgesic Nephropathy** | Chronic NSAID/acetaminophen use; **Papillary necrosis**, **CT: rim calcifications** | History, CT imaging | **Discontinue analgesics** | https://www.ncbi.nlm.nih.gov/books/NBK559182/ |
| **Heavy Metal Nephropathy** | Lead (hypertension, Gout, Anemia), Cadmium (proteinuria) | Blood/urine metal levels | **Chelation therapy** (lead) | https://www.ncbi.nlm.nih.gov/books/NBK559182/ |
| **Aristolochic Acid Nephropathy** | **Chinese herb nephropathy** (Balkan endemic nephropathy); Rapid progression to ESKD | History, Biopsy | **Supportive** | https://www.ncbi.nlm.nih.gov/books/NBK559182/ |
| **Lithium Nephropathy** | Chronic lithium use; **Nephrogenic diabetes insipidus**, **Chronic tubulointerstitial fibrosis** | History, Urine concentrating defect | **Amiloride** (for NDI), **Discontinue lithium** if possible | https://www.ncbi.nlm.nih.gov/books/NBK559182/ |
| **Sarcoidosis** | **Non-caseating granulomas**, **Hypercalcemia**, Elevated ACE level | Biopsy, Chest imaging | **Corticosteroids** | https://www.ncbi.nlm.nih.gov/books/NBK430968/ |
| **Sjögren Syndrome** | **Distal RTA** (Type 1), Hypokalemia, Sicca symptoms | Anti-Ro/SSA, Anti-La/SSB, Biopsy | **Potassium replacement**, **Alkali therapy**, Immunosuppression | https://www.ncbi.nlm.nih.gov/books/NBK431049/ |

## Papillary Necrosis
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Ischemic necrosis of renal papillae | https://www.ncbi.nlm.nih.gov/books/NBK559182/ |
| **Causes** | **Diabetes mellitus** (most common), **Analgesic nephropathy**, **Sickle cell disease/trait**, **Urinary tract obstruction**, **Pyelonephritis** | https://www.ncbi.nlm.nih.gov/books/NBK559182/ |
| **Clinical Features** | Flank pain, **Hematuria**, **Fever**, **Passage of necrotic tissue in urine** | https://www.ncbi.nlm.nih.gov/books/NBK559182/ |
| **Diagnosis** | **CT urogram**: "Ring shadow" or "egg in cup" appearance; **IVP**: Clubbed calyces | https://www.ncbi.nlm.nih.gov/books/NBK559182/ |
| **Treatment** | Treat underlying cause; **Ureteral stenting** if obstruction | https://www.ncbi.nlm.nih.gov/books/NBK559182/ |

## Acute Tubular Necrosis (ATN)
| Component | Description | URL |
|-----------|-------------|-----|
| **Ischemic ATN** | Hypotension, Shock, Sepsis, Surgery; **More severe**, Patchy necrosis | https://www.ncbi.nlm.nih.gov/books/NBK507815/ |
| **Nephrotoxic ATN** | **Aminoglycosides** (non-oliguric, Delayed onset), **Amphotericin B** (distal RTA, Hypokalemia), **Cisplatin** (hypomagnesemia), **Contrast** (osmotic injury), **Heme pigments** (rhabdomyolysis, Hemolysis) | https://www.ncbi.nlm.nih.gov/books/NBK507815/ |
| **Phases** | **Initiation** (hours-days), **Maintenance** (1-3 weeks - oliguric), **Recovery** (polyuric phase) | https://www.ncbi.nlm.nih.gov/books/NBK507815/ |
| **Diagnosis** | **Muddy brown granular casts**, **FeNa >2%**, **Urine Na >40 mEq/L**, **Urine osmolality <350 mOsm/kg** | https://www.ncbi.nlm.nih.gov/books/NBK507815/ |
| **Treatment** | **Supportive**; Avoid nephrotoxins; **Loop diuretics do NOT improve outcomes** | https://www.ncbi.nlm.nih.gov/books/NBK507815/ |

## Myoglobinuric AKI (Rhabdomyolysis)
| Component | Description | URL |
|-----------|-------------|-----|
| **Causes** | Trauma, Crush injury, Seizures, Prolonged immobilization, Statins, Cocaine, Heat stroke | https://www.ncbi.nlm.nih.gov/books/NBK507815/ |
| **Pathophysiology** | Myoglobin ? **Direct tubular toxicity**, **Intratubular cast formation**, **Renal vasoconstriction** | https://www.ncbi.nlm.nih.gov/books/NBK507815/ |
| **Diagnosis** | **Elevated CK** (>5x ULN, often >5,000), **Positive urine dipstick for blood with few RBCs**, **Hyperkalemia**, **Hyperphosphatemia**, **Hypocalcemia** (early) | https://www.ncbi.nlm.nih.gov/books/NBK507815/ |
| **Treatment** | **Aggressive IV fluids** (goal UOP 200-300 mL/hr), **Avoid calcium** (unless severe hyperkalemia or symptomatic hypocalcemia), **Consider urinary alkalinization** (controversial) | https://www.ncbi.nlm.nih.gov/books/NBK507815/ |
