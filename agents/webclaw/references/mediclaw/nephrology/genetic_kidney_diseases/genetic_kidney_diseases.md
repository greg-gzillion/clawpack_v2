# Nephrology - Genetic Kidney Diseases

## Alport Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| **Genetics** | **X-linked** (COL4A5 - 80%), **Autosomal recessive** (COL4A3, COL4A4 - 15%), **Autosomal dominant** (COL4A3, COL4A4 - 5%) | https://www.niddk.nih.gov/health-information/kidney-disease/alport-syndrome |
| **Pathophysiology** | **Defective type IV collagen** ? **Abnormal GBM** (lamellation, Thinning) | https://www.niddk.nih.gov/health-information/kidney-disease/alport-syndrome |
| **Clinical Features** | **Hematuria** (microscopic, Persistent), **Progressive CKD** ? **ESKD** (males by age 30-50), **Sensorineural hearing loss**, **Ocular abnormalities** (anterior lenticonus, Retinal flecks) | https://www.niddk.nih.gov/health-information/kidney-disease/alport-syndrome |
| **Diagnosis** | **Kidney biopsy** (GBM lamellation on EM, a5(IV) collagen staining), **Genetic testing** | https://www.niddk.nih.gov/health-information/kidney-disease/alport-syndrome |
| **Treatment** | **RAAS blockade** (slows progression), **Hearing aids**, **Kidney transplantation** | https://www.niddk.nih.gov/health-information/kidney-disease/alport-syndrome |

## Thin Basement Membrane Disease (TBMD)
| Component | Description | URL |
|-----------|-------------|-----|
| **Genetics** | **Heterozygous COL4A3/COL4A4 mutations** (carrier state for Alport) | https://www.niddk.nih.gov/health-information/kidney-disease/thin-basement-membrane-disease |
| **Clinical Features** | **Isolated microscopic hematuria**, **Normal GFR**, **Normal life expectancy** | https://www.niddk.nih.gov/health-information/kidney-disease/thin-basement-membrane-disease |
| **Diagnosis** | **Kidney biopsy** (diffuse GBM thinning <250 nm) | https://www.niddk.nih.gov/health-information/kidney-disease/thin-basement-membrane-disease |
| **Treatment** | **Reassurance**, **No specific therapy** | https://www.niddk.nih.gov/health-information/kidney-disease/thin-basement-membrane-disease |

## Fabry Disease
| Component | Description | URL |
|-----------|-------------|-----|
| **Genetics** | **X-linked** (GLA gene) | https://www.ninds.nih.gov/health-information/disorders/fabry-disease |
| **Pathophysiology** | **Alpha-galactosidase A deficiency** ? **Globotriaosylceramide (Gb3) accumulation** in lysosomes | https://www.ninds.nih.gov/health-information/disorders/fabry-disease |
| **Clinical Features** | **Acroparesthesias** (burning pain in hands/feet), **Angiokeratomas** (skin), **Hypohidrosis**, **Corneal verticillata**, **Cardiac** (LVH, Arrhythmias), **CKD** ? ESKD (males by age 40-50) | https://www.ninds.nih.gov/health-information/disorders/fabry-disease |
| **Diagnosis** | **Alpha-galactosidase A activity** (low in males), **Genetic testing**, **Gb3 levels** | https://www.ninds.nih.gov/health-information/disorders/fabry-disease |
| **Treatment** | **Enzyme replacement therapy** (agalsidase beta, Agalsidase alfa), **Migalastat** (chaperone for amenable mutations) | https://www.fda.gov/drugs |

## Gitelman Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| **Genetics** | **Autosomal recessive** (SLC12A3 gene - NCC transporter) | https://www.niddk.nih.gov/health-information/kidney-disease/gitelman-syndrome |
| **Pathophysiology** | **Defective Na-Cl cotransporter** in DCT ? **Salt wasting** ? **Volume contraction** ? **Secondary hyperaldosteronism** | https://www.niddk.nih.gov/health-information/kidney-disease/gitelman-syndrome |
| **Clinical Features** | **Hypokalemia**, **Hypomagnesemia**, **Metabolic alkalosis**, **Hypocalciuria**, **Mild symptoms** (muscle cramps, Fatigue) | https://www.niddk.nih.gov/health-information/kidney-disease/gitelman-syndrome |
| **Treatment** | **K+ replacement**, **Mg2+ replacement**, **K+-sparing diuretics** (spironolactone, Amiloride), **NSAIDs** (indomethacin) | https://www.niddk.nih.gov/health-information/kidney-disease/gitelman-syndrome |
