# Nephrology - Cardiorenal Syndrome (CRS)

## Cardiorenal Syndrome - Classification
| Type | Name | Description | URL |
|------|------|-------------|-----|
| **Type 1** | **Acute Cardiorenal** | **Acute cardiac dysfunction** (ADHF, ACS, Cardiogenic shock) ? **AKI** | https://www.heart.org/en/health-topics/heart-failure |
| **Type 2** | **Chronic Cardiorenal** | **Chronic cardiac dysfunction** (CHF) ? **Progressive CKD** | https://www.heart.org/en/health-topics/heart-failure |
| **Type 3** | **Acute Renocardiac** | **AKI** ? **Acute cardiac dysfunction** (Arrhythmias, CHF, Ischemia) | https://www.kidney.org/kidney-topics/heart-disease-and-kidney-disease |
| **Type 4** | **Chronic Renocardiac** | **CKD** ? **Chronic cardiac dysfunction** (LVH, Diastolic dysfunction, Accelerated atherosclerosis) | https://www.kidney.org/kidney-topics/heart-disease-and-kidney-disease |
| **Type 5** | **Secondary** | **Systemic disease** (Sepsis, Diabetes, Amyloidosis) ? **Simultaneous cardiac and renal dysfunction** | https://www.kidney.org/kidney-topics/heart-disease-and-kidney-disease |

## Type 1 CRS (Acute Cardiorenal) - Pathophysiology
| Mechanism | Description | URL |
|-----------|-------------|-----|
| **Hemodynamic** | **Decreased cardiac output** ? **Renal hypoperfusion** ? **Prerenal AKI** | https://www.heart.org/en/health-topics/heart-failure |
| **Venous Congestion** | **Increased CVP** ? **Renal venous hypertension** ? **Decreased GFR**, **Increased interstitial pressure** | https://www.heart.org/en/health-topics/heart-failure |
| **Neurohormonal Activation** | **RAAS activation**, **Sympathetic nervous system**, **Arginine vasopressin** ? **Sodium/water retention**, **Vasoconstriction** | https://www.heart.org/en/health-topics/heart-failure |
| **Inflammation** | **Pro-inflammatory cytokines** ? **Endothelial dysfunction**, **Tubular injury** | https://www.heart.org/en/health-topics/heart-failure |

## Type 1 CRS - Management
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **Diuretics** | **Loop diuretics** (IV), **Sequential nephron blockade** (add thiazide), **Diuretic resistance management** | https://www.heart.org/en/health-topics/heart-failure |
| **Vasodilators** | **Nitroglycerin**, **Nitroprusside** (if hypertensive) | https://www.heart.org/en/health-topics/heart-failure |
| **Inotropes** | **Dobutamine**, **Milrinone** (low output state) | https://www.heart.org/en/health-topics/heart-failure |
| **Vasopressin Antagonists** | **Tolvaptan** (hypervolemic hyponatremia) | https://www.fda.gov/drugs |
| **Ultrafiltration** | **Aquapheresis** for diuretic-resistant volume overload | https://www.heart.org/en/health-topics/heart-failure |
| **RAAS Blockade** | **Continue if possible**; May worsen renal function initially | https://www.heart.org/en/health-topics/heart-failure |

## Type 2 CRS (Chronic Cardiorenal) - Management
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **Guideline-Directed Medical Therapy (GDMT)** | **ACE-I/ARB/ARNI**, **Beta-blockers**, **MRA**, **SGLT2 inhibitors** | https://www.heart.org/en/health-topics/heart-failure |
| **SGLT2 Inhibitors** | **Empagliflozin**, **Dapagliflozin** (reduce HF hospitalizations, Slow CKD progression) | https://www.heart.org/en/health-topics/heart-failure |
| **Loop Diuretics** | For volume overload; **Lowest effective dose** | https://www.heart.org/en/health-topics/heart-failure |
| **Iron Repletion** | **IV iron** for iron deficiency (improves functional status) | https://www.heart.org/en/health-topics/heart-failure |

## Type 4 CRS (Chronic Renocardiac) - Management
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **Blood Pressure Control** | **Target <130/80 mmHg** | https://www.kidney.org/kidney-topics/heart-disease-and-kidney-disease |
| **RAAS Blockade** | **ACE-I/ARB** (first-line for proteinuria) | https://www.kidney.org/kidney-topics/heart-disease-and-kidney-disease |
| **SGLT2 Inhibitors** | **Reduce CV events and CKD progression** | https://www.kidney.org/kidney-topics/heart-disease-and-kidney-disease |
| **Statin Therapy** | **Moderate-high intensity** for all adults =50 with CKD | https://www.kidney.org/kidney-topics/heart-disease-and-kidney-disease |
| **Anemia Management** | **Target Hb 10-11.5 g/dL** (avoid >13) | https://www.kidney.org/kidney-topics/anemia-chronic-kidney-disease |
| **CKD-MBD Management** | **Phosphate binders**, **Vitamin D analogs**, **Calcimimetics** | https://www.kidney.org/kidney-topics/mineral-bone-disorder-chronic-kidney-disease |
