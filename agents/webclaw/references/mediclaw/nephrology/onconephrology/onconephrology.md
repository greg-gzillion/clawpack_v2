## Tumor Lysis Syndrome (TLS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Massive release of intracellular contents (K+, PO4, Uric acid) from lysed tumor cells | https://www.cancer.gov/about-cancer/treatment/side-effects/tumor-lysis-syndrome |
| **Risk Factors** | **High tumor burden**, **Highly chemosensitive tumors** (Burkitt lymphoma, ALL, AML), **Pre-existing renal impairment**, **Dehydration** | https://www.cancer.gov/about-cancer/treatment/side-effects/tumor-lysis-syndrome |
| **Laboratory TLS** | **Hyperuricemia** (>8 mg/dL), **Hyperkalemia** (>6 mEq/L), **Hyperphosphatemia** (>4.5 mg/dL), **Hypocalcemia** | https://www.cancer.gov/about-cancer/treatment/side-effects/tumor-lysis-syndrome |
| **Clinical TLS** | **AKI**, **Seizures**, **Arrhythmias**, **Sudden death** | https://www.cancer.gov/about-cancer/treatment/side-effects/tumor-lysis-syndrome |
| **Prevention** | **Aggressive hydration**, **Allopurinol** (low/intermediate risk), **Rasburicase** (high risk) | https://www.cancer.gov/about-cancer/treatment/side-effects/tumor-lysis-syndrome |
| **Treatment** | **Rasburicase**, **Manage hyperkalemia**, **Dialysis** if refractory | https://www.cancer.gov/about-cancer/treatment/side-effects/tumor-lysis-syndrome |

## Chemotherapy Nephrotoxicity
| Drug | Renal Manifestation | Prevention/Management | URL |
|------|---------------------|----------------------|-----|
| **Cisplatin** | **ATN**, **Hypomagnesemia**, **Fanconi syndrome** | **Aggressive hydration**, **Avoid other nephrotoxins** | https://www.cancer.gov/about-cancer/treatment/side-effects/kidney-problems |
| **Methotrexate (High-Dose)** | **Crystal nephropathy** (precipitation in tubules) | **Urinary alkalinization**, **Aggressive hydration**, **Leucovorin rescue** | https://www.cancer.gov/about-cancer/treatment/side-effects/kidney-problems |
| **Ifosfamide** | **Hemorrhagic cystitis**, **Fanconi syndrome**, **ATN** | **Mesna** (for cystitis), **Hydration** | https://www.cancer.gov/about-cancer/treatment/side-effects/kidney-problems |
| **Immune Checkpoint Inhibitors** | **AIN** (most common), **Minimal change disease**, **FSGS**, **IgA nephropathy** | **Corticosteroids**, **Hold ICI** | https://www.cancer.gov/about-cancer/treatment/types/immunotherapy |
| **VEGF Inhibitors** | **Thrombotic microangiopathy**, **Hypertension**, **Proteinuria** | **BP control**, **Hold drug**, **Plasmapheresis** (severe TMA) | https://www.cancer.gov/about-cancer/treatment/side-effects/kidney-problems |
| **Bisphosphonates** | **FSGS**, **Minimal change disease**, **ATN** | **Hydration**, **Monitor renal function** | https://www.cancer.gov/about-cancer/treatment/side-effects/kidney-problems |

## Radiation Nephropathy
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Kidney injury from radiation exposure (>20-25 Gy) | https://www.cancer.gov/about-cancer/treatment/side-effects/kidney-problems |
| **Timing** | **Acute** (6-12 months), **Chronic** (years) | https://www.cancer.gov/about-cancer/treatment/side-effects/kidney-problems |
| **Pathology** | **Thrombotic microangiopathy**, **Mesangiolysis**, **Glomerulosclerosis**, **Interstitial fibrosis** | https://www.cancer.gov/about-cancer/treatment/side-effects/kidney-problems |
| **Clinical Features** | **Hypertension**, **Proteinuria**, **Progressive CKD** | https://www.cancer.gov/about-cancer/treatment/side-effects/kidney-problems |
| **Management** | **RAAS blockade**, **BP control**, **Avoid nephrotoxins** | https://www.cancer.gov/about-cancer/treatment/side-effects/kidney-problems |
