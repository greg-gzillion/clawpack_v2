# Nephrology - Urinary Tract Infections (UTI)

## UTI - Classification
| Category | Definition | URL |
|----------|------------|-----|
| **Uncomplicated Cystitis** | Healthy, Non-pregnant, Premenopausal women; No structural/functional abnormalities | https://www.cdc.gov/uti/about/index.html |
| **Complicated UTI** | Men, Pregnant women, Structural/functional abnormalities, Indwelling catheters, Immunocompromised, CKD, Healthcare-associated | https://www.cdc.gov/uti/about/index.html |
| **Pyelonephritis** | Infection of renal parenchyma and collecting system | https://www.niddk.nih.gov/health-information/urologic-diseases/kidney-infection-pyelonephritis |

## Uncomplicated Cystitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathogens** | E. coli (75-95%), S. saprophyticus (5-15% - young women), Klebsiella, Proteus, Enterococcus | https://www.cdc.gov/uti/about/index.html |
| **Clinical Features** | Dysuria, Frequency, Urgency, Suprapubic pain, Hematuria; No fever | https://www.cdc.gov/uti/about/index.html |
| **Diagnosis** | Urinalysis (pyuria - WBC >5-10/hpf, Leukocyte esterase, Nitrites), Urine culture (=10^3 CFU/mL) | https://www.cdc.gov/uti/about/index.html |
| **First-Line Treatment** | Nitrofurantoin 100 mg BID x 5 days, TMP-SMX DS BID x 3 days, Fosfomycin 3g x 1 | https://www.ncbi.nlm.nih.gov/books/NBK482344/ |
| **Alternative Treatment** | Ciprofloxacin 250 mg BID x 3 days or Levofloxacin 250 mg daily x 3 days | https://www.fda.gov/drugs/information-drug-class/fluoroquinolone-antimicrobial-drugs |

## Catheter-Associated UTI (CAUTI)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | UTI in patient with indwelling catheter for >2 days + Symptoms + =10^3 CFU/mL | https://www.cdc.gov/hai/cauti/index.html |
| **Prevention** | Avoid unnecessary catheters, Remove as soon as possible, Aseptic insertion, Maintain closed drainage system | https://www.cdc.gov/hai/cauti/index.html |
| **Diagnosis** | Only culture if symptomatic (fever, Flank pain, Delirium); Do NOT culture asymptomatic patients | https://www.cdc.gov/hai/cauti/index.html |
| **Treatment** | Remove or replace catheter + Antibiotics; Duration: 7 days (prompt response), 10-14 days (delayed response) | https://www.ncbi.nlm.nih.gov/books/NBK482344/ |

## Asymptomatic Bacteriuria (ASB)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | =10^5 CFU/mL in asymptomatic patient | https://www.cdc.gov/uti/about/index.html |
| **When to Treat** | Pregnancy, Prior to urologic procedure with mucosal bleeding | https://www.uspreventiveservicestaskforce.org/uspstf/recommendation/asymptomatic-bacteriuria-in-adults-screening |
| **When NOT to Treat** | Elderly, Catheterized patients, Diabetes, Long-term care residents, Pre-menopausal non-pregnant women | https://www.uspreventiveservicestaskforce.org/uspstf/recommendation/asymptomatic-bacteriuria-in-adults-screening |
| **Rationale** | No benefit; Increases adverse effects, C. difficile, Antimicrobial resistance | https://www.cdc.gov/antibiotic-use/core-elements/index.html |

## Recurrent UTI
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | =2 UTIs in 6 months or =3 UTIs in 12 months | https://www.cdc.gov/uti/about/index.html |
| **Prevention Strategies** | Hydration, Postcoital voiding, Avoid spermicides, Vaginal estrogen (postmenopausal), Methenamine hippurate | https://www.ncbi.nlm.nih.gov/books/NBK482344/ |
| **Antibiotic Prophylaxis** | Continuous: TMP-SMX, Nitrofurantoin, Cephalexin daily; Postcoital: Single dose after intercourse | https://www.ncbi.nlm.nih.gov/books/NBK482344/ |
| **Duration of Prophylaxis** | 6-12 months (reassess after discontinuation) | https://www.ncbi.nlm.nih.gov/books/NBK482344/ |

## Acute Bacterial Prostatitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathogens** | E. coli (most common), Klebsiella, Proteus, Pseudomonas, Enterococcus | https://www.cdc.gov/uti/about/index.html |
| **Clinical Features** | Fever, Chills, Dysuria, Frequency, Perineal/suprapubic pain, Tender boggy prostate on DRE | https://www.niddk.nih.gov/health-information/urologic-diseases/prostate-problems/prostatitis |
| **Diagnosis** | Urinalysis/culture (pyuria, Bacteriuria); AVOID vigorous prostate massage (risk of bacteremia) | https://www.niddk.nih.gov/health-information/urologic-diseases/prostate-problems/prostatitis |
| **Treatment** | Ciprofloxacin 500 mg BID x 2-4 weeks or Levofloxacin 500 mg daily; TMP-SMX DS BID (alternative) | https://www.ncbi.nlm.nih.gov/books/NBK482344/ |
| **Complications** | Prostatic abscess (persistent fever) ? Transurethral or percutaneous drainage | https://www.niddk.nih.gov/health-information/urologic-diseases/prostate-problems/prostatitis |

## Chronic Bacterial Prostatitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Clinical Features** | Recurrent UTIs with same organism, Pelvic pain, Dysuria, Often asymptomatic between episodes | https://www.niddk.nih.gov/health-information/urologic-diseases/prostate-problems/prostatitis |
| **Diagnosis** | Urine culture before and after prostate massage, Expressed prostatic secretions | https://www.niddk.nih.gov/health-information/urologic-diseases/prostate-problems/prostatitis |
| **Treatment** | Fluoroquinolone x 4-6 weeks (Ciprofloxacin, Levofloxacin); TMP-SMX (alternative) | https://www.ncbi.nlm.nih.gov/books/NBK482344/ |

## Xanthogranulomatous Pyelonephritis (XGP)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Chronic destructive granulomatous inflammation of kidney, Usually associated with obstruction and infection | https://www.ncbi.nlm.nih.gov/books/NBK557525/ |
