# Nephrology - Hepatorenal Syndrome (HRS)

## Hepatorenal Syndrome - Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Functional AKI** in patients with **advanced cirrhosis** or **acute liver failure**; **No structural kidney damage** | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **Pathophysiology** | **Splanchnic vasodilation** ? **Decreased effective arterial blood volume** ? **RAAS/SNS/ADH activation** ? **Intense renal vasoconstriction** ? **Prerenal AKI** | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **Types** | **HRS-AKI** (formerly Type 1 - rapidly progressive), **HRS-CKD** (formerly Type 2 - slowly progressive), **HRS-NAKI** (not meeting AKI criteria) | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |

## HRS - Diagnostic Criteria (ICA 2015)
| Criterion | Description | URL |
|-----------|-------------|-----|
| **1. Cirrhosis with ascites** | Diagnosis of cirrhosis | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **2. AKI** | Cr increase =0.3 mg/dL within 48h or =50% from baseline | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **3. No improvement after diuretic withdrawal and volume expansion** | **Albumin 1 g/kg/day** (max 100g) x 2 days | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **4. Absence of shock** | No hypotension requiring vasopressors | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **5. No recent nephrotoxic drugs** | NSAIDs, Aminoglycosides, Contrast | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **6. No parenchymal kidney disease** | **No significant proteinuria** (<500 mg/day), **No hematuria** (<50 RBCs/hpf), **Normal renal US** | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |

## HRS - Differential Diagnosis
| Condition | Distinguishing Features | URL |
|-----------|-------------------------|-----|
| **Prerenal AKI** | **Improves with volume expansion** | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **Acute Tubular Necrosis (ATN)** | **FeNa >2%**, **Muddy brown casts**, **No response to vasoconstrictors** | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **Glomerulonephritis** | **Active urine sediment** (RBC casts, Dysmorphic RBCs), **Significant proteinuria** | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **IgA Nephropathy** | Common in cirrhosis; **Hematuria**, **Proteinuria** | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |

## HRS-AKI - Management
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **Vasoconstrictors + Albumin** | **Terlipressin** (preferred), **Norepinephrine**, **Midodrine + Octreotide**; **Albumin** 20-40 g/day | https://www.fda.gov/drugs |
| **Volume Management** | **Avoid over-diuresis**, **Large volume paracentesis with albumin** (6-8g per L removed) | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **Spontaneous Bacterial Peritonitis (SBP) Prophylaxis** | **Norfloxacin**, **TMP-SMX** (high-risk patients) | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **Renal Replacement Therapy** | **Bridge to liver transplant**; **Poor prognosis without transplant** | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **Liver Transplantation** | **Only definitive treatment** | https://optn.transplant.hrsa.gov |

## HRS - Prevention
| Strategy | Description | URL |
|----------|-------------|-----|
| **SBP Prophylaxis** | **Norfloxacin** or **TMP-SMX** in high-risk cirrhotics | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **Albumin After Large Volume Paracentesis** | **6-8g per liter** of ascites removed (>5L) | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **Albumin for SBP** | **1.5 g/kg day 1**, **1 g/kg day 3** | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **Avoid Nephrotoxins** | **NSAIDs**, **Aminoglycosides**, **IV contrast** | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
| **Judicious Diuretic Use** | Avoid excessive diuresis (goal weight loss 0.5 kg/day) | https://www.niddk.nih.gov/health-information/liver-disease/cirrhosis |
