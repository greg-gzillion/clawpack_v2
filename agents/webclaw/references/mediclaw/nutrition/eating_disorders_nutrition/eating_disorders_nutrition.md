# Nutrition - Eating Disorders Nutrition

## Anorexia Nervosa
| Component | Description | URL |
|-----------|-------------|-----|
| **Nutrition Goals** | **Weight restoration** (1-2 lb/week outpatient, 2-3 lb/week inpatient), **Normalize eating patterns**, **Correct nutritional deficiencies** | https://www.nationaleatingdisorders.org |
| **Initial Caloric Intake** | **Start 1,200-1,500 kcal/day**; Increase by **200-300 kcal every 2-3 days** to goal (3,000-4,000+ kcal/day for weight gain) | https://www.nationaleatingdisorders.org |
| **Refeeding Syndrome Prevention** | **Start low, go slow**, **Monitor K, PO4, Mg**, **Thiamin supplementation** | https://www.nice.org.uk/guidance/ng69 |
| **Meal Plan** | **3 meals + 2-3 snacks**, **Structured**, **Supervised** | https://www.nationaleatingdisorders.org |
| **Nutritional Deficiencies** | **Vitamin D**, **Calcium**, **Iron**, **Zinc**, **B vitamins** | https://www.nationaleatingdisorders.org |

## Bulimia Nervosa
| Component | Description | URL |
|-----------|-------------|-----|
| **Nutrition Goals** | **Normalize eating patterns**, **Reduce binge-purge episodes**, **Correct electrolyte imbalances** | https://www.nationaleatingdisorders.org |
| **Meal Plan** | **Structured 3 meals + 2-3 snacks**, **Avoid restrictive eating** (triggers binges) | https://www.nationaleatingdisorders.org |
| **Electrolyte Monitoring** | **Potassium** (hypokalemia from vomiting/laxatives), **Chloride**, **Bicarbonate** | https://www.nationaleatingdisorders.org |
| **CBT-E** | **Cognitive Behavioral Therapy for Eating Disorders** (addresses dietary restraint, Binge-purge cycle) | https://www.nationaleatingdisorders.org |

## Binge-Eating Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **Nutrition Goals** | **Normalize eating patterns**, **Reduce binge episodes**, **Achieve/maintain healthy weight** | https://www.nationaleatingdisorders.org |
| **Meal Plan** | **Regular meals/snacks**, **Avoid skipping meals** (prevents extreme hunger) | https://www.nationaleatingdisorders.org |
| **Mindful Eating** | **Pay attention to hunger/fullness cues**, **Eat without distraction** | https://www.nationaleatingdisorders.org |
| **CBT-E** | Addresses dietary restraint and binge eating | https://www.nationaleatingdisorders.org |

## Avoidant/Restrictive Food Intake Disorder (ARFID)
| Component | Description | URL |
|-----------|-------------|-----|
| **Nutrition Goals** | **Expand food variety**, **Meet nutritional needs**, **Weight gain/stabilization** | https://www.nationaleatingdisorders.org |
| **Approach** | **Gradual exposure** to new foods, **Sensory-based interventions**, **Address anxiety** | https://www.nationaleatingdisorders.org |
| **Supplements** | **Oral nutrition supplements** to meet energy/nutrient needs during food expansion | https://www.nationaleatingdisorders.org |

## Medical Complications of Eating Disorders Requiring Nutrition Intervention
| Complication | Nutrition Intervention | URL |
|--------------|----------------------|-----|
| **Hypokalemia** | **Potassium-rich foods** (bananas, potatoes, spinach), **Potassium supplements** | https://www.nationaleatingdisorders.org |
| **Hypophosphatemia** | **Phosphorus-rich foods** (dairy, meat, beans), **Phosphate supplements** (refeeding) | https://www.nationaleatingdisorders.org |
| **Constipation** | **Gradual fiber increase**, **Adequate fluid**, **Stool softeners** (avoid stimulant laxatives) | https://www.nationaleatingdisorders.org |
| **Delayed Gastric Emptying** | **Small, frequent meals**, **Liquid calories**, **Prokinetic agents** | https://www.nationaleatingdisorders.org |
| **Osteoporosis** | **Calcium + Vitamin D**, **Weight restoration** (resumption of menses) | https://www.nationaleatingdisorders.org |
