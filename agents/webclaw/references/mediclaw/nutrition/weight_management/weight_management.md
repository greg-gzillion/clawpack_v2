## Lifestyle Interventions for Weight Loss
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **Caloric Deficit** | **500-750 kcal/day** deficit for 1-2 lb/week weight loss | https://www.niddk.nih.gov/health-information/weight-management |
| **Dietary Pattern** | Any pattern that creates caloric deficit (Mediterranean, Low-carb, Low-fat, DASH) | https://www.niddk.nih.gov/health-information/weight-management |
| **Physical Activity** | **=150 min/week moderate** for health; **200-300 min/week** for weight maintenance | https://health.gov/physical-activity-guidelines |
| **Behavioral Strategies** | **Self-monitoring** (food/activity logs), **Goal setting**, **Stimulus control**, **Problem solving** | https://www.niddk.nih.gov/health-information/weight-management |
| **Intensive Behavioral Therapy** | **=14 sessions in 6 months** (Medicare-covered) | https://www.cms.gov/medicare-coverage-database |

## Post-Bariatric Surgery Nutrition
| Nutrient | Recommendation | URL |
|----------|----------------|-----|
| **Protein** | **60-80 g/day** (or 1.1-1.5 g/kg IBW) | https://asmbs.org/patients/life-after-bariatric-surgery |
| **Fluid** | **=64 oz/day** (sip slowly, between meals) | https://asmbs.org/patients/life-after-bariatric-surgery |
| **Vitamin B12** | **350-500 mcg/day** (sublingual or IM) | https://asmbs.org/patients/life-after-bariatric-surgery |
| **Iron** | **45-60 mg/day** (with vitamin C) | https://asmbs.org/patients/life-after-bariatric-surgery |
| **Calcium Citrate** | **1,200-1,500 mg/day** (divided doses) | https://asmbs.org/patients/life-after-bariatric-surgery |
| **Vitamin D** | **3,000 IU/day** (titrate to level =30 ng/mL) | https://asmbs.org/patients/life-after-bariatric-surgery |
| **Multivitamin** | **Complete bariatric formulation** (2 tablets daily) | https://asmbs.org/patients/life-after-bariatric-surgery |
| **Thiamin** | **=12 mg/day** (prevent Wernicke encephalopathy) | https://asmbs.org/patients/life-after-bariatric-surgery |
