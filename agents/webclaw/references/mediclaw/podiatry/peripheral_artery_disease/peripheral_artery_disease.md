# Podiatry - Peripheral Artery Disease (PAD)

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Atherosclerotic narrowing** of arteries supplying lower extremities | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **Prevalence** | **>8.5 million** US adults; **20-30%** of diabetics >50 years | https://www.cdc.gov/heartdisease/peripheral_artery_disease.htm |
| **Risk Factors** | **Smoking**, **Diabetes**, **Hypertension**, **Hyperlipidemia**, **Age >65**, **CKD** | https://www.heart.org/en/health-topics/peripheral-artery-disease |

## Clinical Presentation
| Stage | Description | URL |
|-------|-------------|-----|
| **Asymptomatic** | **No symptoms** (often incidental ABI finding) | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **Claudication** | **Pain with walking**, **Relieved by rest** (reproducible) | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **Critical Limb Ischemia (CLI)** | **Rest pain**, **Tissue loss** (ulcer, gangrene) | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **Acute Limb Ischemia** | **Sudden onset** (6 P's: Pain, Pallor, Pulselessness, Paresthesias, Paralysis, Poikilothermia) | https://www.heart.org/en/health-topics/peripheral-artery-disease |

## Diagnostic Testing
| Test | Interpretation | URL |
|------|----------------|-----|
| **Ankle-Brachial Index (ABI)** | **>1.4**: Non-compressible (calcified); **1.0-1.4**: Normal; **0.90-0.99**: Borderline; **0.50-0.89**: Moderate PAD; **<0.50**: Severe PAD | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **Toe-Brachial Index (TBI)** | **>0.7**: Normal; **<0.7**: PAD (useful in calcified vessels) | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **Segmental Pressures** | **Localizes** level of stenosis | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **Duplex Ultrasound** | Visualizes stenosis, Measures velocity | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **CT Angiography / MR Angiography** | **Pre-procedural planning** | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **Catheter Angiography** | **Gold standard**, Allows intervention | https://www.heart.org/en/health-topics/peripheral-artery-disease |

## Management of PAD
| Intervention | Recommendation | URL |
|--------------|----------------|-----|
| **Lifestyle** | **Smoking cessation**, **Supervised exercise therapy** (30-45 min, =3x/week) | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **Pharmacotherapy** | **Antiplatelet** (aspirin or clopidogrel), **Statin** (high-intensity) | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **Cilostazol** | **Improves claudication** (contraindicated in HF) | https://www.heart.org/en/health-topics/peripheral-artery-disease |
| **Revascularization** | **Angioplasty/Stenting** or **Bypass surgery** for lifestyle-limiting claudication or CLI | https://www.heart.org/en/health-topics/peripheral-artery-disease |
