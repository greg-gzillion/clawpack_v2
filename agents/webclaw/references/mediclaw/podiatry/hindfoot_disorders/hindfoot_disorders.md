## Plantar Fasciitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Degenerative condition of plantar fascia** (not primarily inflammatory) | https://orthoinfo.aaos.org/en/diseases--conditions/plantar-fasciitis-and-bone-spurs/ |
| **Risk Factors** | **Obesity**, **Prolonged standing**, **Pes planus/cavus**, **Equinus**, **Running** | https://orthoinfo.aaos.org/en/diseases--conditions/plantar-fasciitis-and-bone-spurs/ |
| **Symptoms** | **First-step pain**, **Tender medial calcaneal tubercle**, **Worse after rest** | https://orthoinfo.aaos.org/en/diseases--conditions/plantar-fasciitis-and-bone-spurs/ |
| **Conservative Treatment** | **Stretching** (Achilles, Plantar fascia), **Night splint**, **Orthotics**, **NSAIDs**, **CS injection**, **Shockwave therapy** | https://orthoinfo.aaos.org/en/diseases--conditions/plantar-fasciitis-and-bone-spurs/ |
| **Surgical Treatment** | **Plantar fascia release** (open or endoscopic), **Gastrocnemius recession** | https://www.acfas.org/footankleinfo/Plantar-Fasciitis.htm |

## Posterior Tibial Tendon Dysfunction (PTTD)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Progressive failure of posterior tibial tendon** leading to **acquired adult flatfoot deformity** | https://orthoinfo.aaos.org/en/diseases--conditions/posterior-tibial-tendon-dysfunction/ |
| **Stages** | **Stage 1**: Tenosynovitis (no deformity); **Stage 2**: Flexible flatfoot; **Stage 3**: Rigid flatfoot; **Stage 4**: Rigid flatfoot + ankle arthritis | https://orthoinfo.aaos.org/en/diseases--conditions/posterior-tibial-tendon-dysfunction/ |
| **Physical Exam** | **Too many toes sign**, **Single heel rise test** (inability to perform), **Tenderness along tendon** | https://orthoinfo.aaos.org/en/diseases--conditions/posterior-tibial-tendon-dysfunction/ |
| **Conservative Treatment** | **Orthotics** (medial arch support), **Ankle brace**, **PT** (strengthening) | https://orthoinfo.aaos.org/en/diseases--conditions/posterior-tibial-tendon-dysfunction/ |
| **Surgical Treatment** | **Stage 1**: Tenosynovectomy; **Stage 2**: FDL transfer, Medializing calcaneal osteotomy; **Stage 3**: Subtalar fusion; **Stage 4**: TTC fusion | https://www.acfas.org/footankleinfo/Posterior-Tibial-Tendon-Dysfunction.htm |

## Tarsal Tunnel Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Entrapment neuropathy of posterior tibial nerve** within tarsal tunnel | https://orthoinfo.aaos.org/en/diseases--conditions/tarsal-tunnel-syndrome/ |
| **Anatomy** | **Flexor retinaculum** forms roof; Contents: **Tom, Dick, And Harry** (Tibialis posterior, Flexor digitorum longus, Posterior tibial artery/nerve, Flexor hallucis longus) | https://orthoinfo.aaos.org/en/diseases--conditions/tarsal-tunnel-syndrome/ |
| **Symptoms** | **Burning/tingling pain** in **plantar foot**, **Radiating to toes**, **Worse with activity/prolonged standing** | https://orthoinfo.aaos.org/en/diseases--conditions/tarsal-tunnel-syndrome/ |
| **Physical Exam** | **Tinel sign** over tarsal tunnel, **Sensory loss** in plantar foot | https://orthoinfo.aaos.org/en/diseases--conditions/tarsal-tunnel-syndrome/ |
| **Diagnosis** | **EMG/NCS** | https://orthoinfo.aaos.org/en/diseases--conditions/tarsal-tunnel-syndrome/ |
| **Conservative Treatment** | **Orthotics**, **NSAIDs**, **CS injection** | https://orthoinfo.aaos.org/en/diseases--conditions/tarsal-tunnel-syndrome/ |
| **Surgical Treatment** | **Tarsal tunnel release** | https://www.acfas.org/footankleinfo/Tarsal-Tunnel-Syndrome.htm |

## Tarsal Coalition
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Abnormal connection between tarsal bones** (calcaneonavicular, Talocalcaneal most common) | https://orthoinfo.aaos.org/en/diseases--conditions/tarsal-coalition/ |
| **Demographics** | **Adolescents** (presents when coalition ossifies) | https://orthoinfo.aaos.org/en/diseases--conditions/tarsal-coalition/ |
| **Symptoms** | **Rigid flatfoot**, **Recurrent ankle sprains**, **Pain with activity** | https://orthoinfo.aaos.org/en/diseases--conditions/tarsal-coalition/ |
| **Imaging** | **X-ray** (anteater sign - calcaneonavicular), **CT** (best for talocalcaneal) | https://orthoinfo.aaos.org/en/diseases--conditions/tarsal-coalition/ |
| **Conservative Treatment** | **Immobilization**, **Orthotics**, **Activity modification** | https://orthoinfo.aaos.org/en/diseases--conditions/tarsal-coalition/ |
| **Surgical Treatment** | **Coalition resection** ± **Fat graft interposition**, **Subtalar fusion** (if arthritis) | https://www.acfas.org/footankleinfo/Tarsal-Coalition.htm |
