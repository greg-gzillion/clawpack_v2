## Key Radiographic Angles (Weight-Bearing)
| Angle | Normal Value | Pathologic | URL |
|-------|--------------|------------|-----|
| **Hallux Valgus Angle (HVA)** | **<15°** | **>15°** (mild), **>20°** (moderate), **>40°** (severe) | https://www.orthobullets.com/foot-and-ankle/7005/foot-anatomy |
| **Intermetatarsal Angle (IMA)** | **<9°** | **>9°** | https://www.orthobullets.com/foot-and-ankle/7005/foot-anatomy |
| **Calcaneal Inclination Angle** | **18-22°** | **<18°** (pes planus), **>22°** (pes cavus) | https://www.orthobullets.com/foot-and-ankle/7005/foot-anatomy |
| **Talar-First Metatarsal Angle (Meary)** | **0°** | **>4°** plantarflexion = pes planus | https://www.orthobullets.com/foot-and-ankle/7005/foot-anatomy |
| **Talar Declination Angle** | **21°** | **>30°** = pes planus | https://www.orthobullets.com/foot-and-ankle/7005/foot-anatomy |
| **Metatarsus Adductus Angle** | **<15°** | **>15°** | https://www.orthobullets.com/foot-and-ankle/7005/foot-anatomy |
