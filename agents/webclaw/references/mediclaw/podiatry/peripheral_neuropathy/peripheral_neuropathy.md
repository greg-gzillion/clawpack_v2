# Podiatry - Peripheral Neuropathy

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Damage to peripheral nerves** causing sensory, motor, or autonomic dysfunction | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Most Common Cause** | **Diabetes** (50% of all neuropathy) | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |

## Etiology
| Category | Causes | URL |
|----------|--------|-----|
| **Metabolic** | **Diabetes**, **Hypothyroidism**, **Vitamin B12 deficiency**, **Uremia** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Toxic** | **Alcohol**, **Chemotherapy** (vincristine, cisplatin, taxanes), **Heavy metals** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Infectious/Inflammatory** | **HIV**, **Lyme**, **Guillain-Barré**, **CIDP**, **Vasculitis** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Hereditary** | **Charcot-Marie-Tooth** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Compressive** | **Tarsal tunnel syndrome**, **Morton neuroma** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Idiopathic** | **No identifiable cause** (20-30%) | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |

## Clinical Patterns
| Pattern | Description | Common Causes | URL |
|---------|-------------|---------------|-----|
| **Distal Symmetric Polyneuropathy** | **"Stocking-glove"** distribution | **Diabetes**, **Alcohol**, **B12 deficiency** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Mononeuropathy** | **Single nerve** | **Entrapment** (tarsal tunnel, Morton neuroma), **Trauma** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Mononeuritis Multiplex** | **Multiple individual nerves** | **Vasculitis**, **Diabetes** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Radiculopathy** | **Nerve root** | **Disc herniation**, **Spinal stenosis** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |

## Diagnostic Workup
| Test | Purpose | URL |
|------|---------|-----|
| **Monofilament Testing (10g)** | **Screen for protective sensation loss** | https://www.apma.org/Patients/Diabetes.cfm |
| **Vibration Testing (128 Hz)** | **Screen for large fiber neuropathy** | https://www.apma.org/Patients/Diabetes.cfm |
| **Ankle Reflexes** | **Assess S1 nerve root** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **EMG/NCS** | **Confirm neuropathy**, **Localize**, **Assess severity** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
| **Laboratory** | **HbA1c**, **TSH**, **B12**, **SPEP/UPEP**, **ANA**, **RF**, **RPR** | https://www.ninds.nih.gov/health-information/disorders/peripheral-neuropathy |
