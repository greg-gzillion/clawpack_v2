# Podiatry - Diabetic Foot

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Epidemiology** | **15-25%** lifetime risk of foot ulcer in diabetics | https://www.cdc.gov/diabetes/foot-complications/index.html |
| **Pathophysiology** | **Neuropathy** (sensory, motor, autonomic) + **Peripheral artery disease** + **Immunopathy** | https://www.cdc.gov/diabetes/foot-complications/index.html |
| **Prevention** | **Annual foot exam**, **Daily self-inspection**, **Proper footwear**, **Glycemic control** | https://www.cdc.gov/diabetes/foot-complications/index.html |

## Offloading for Diabetic Foot Ulcers
| Method | Description | URL |
|--------|-------------|-----|
| **Total Contact Cast (TCC)** | **Gold standard** for plantar neuropathic ulcers | https://iwgdfguidelines.org |
| **Removable Cast Walker** | **Similar efficacy** if made **non-removable** | https://iwgdfguidelines.org |
| **Surgical Offloading** | **Achilles lengthening**, **Metatarsal osteotomy**, **Exostectomy** | https://iwgdfguidelines.org |
| **Felted Foam Padding** | Temporary offloading | https://iwgdfguidelines.org |
| **Therapeutic Footwear** | **Prevention** of recurrence | https://iwgdfguidelines.org |

## Osteomyelitis in Diabetic Foot
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnosis** | **Probe-to-bone test** (+ predictive value), **MRI** (gold standard), **Bone biopsy** (definitive) | https://iwgdfguidelines.org |
| **Antibiotic Duration** | **4-6 weeks** (if infected bone resected), **=6 weeks** (if dead bone remains) | https://iwgdfguidelines.org |
| **Surgical Management** | **Debridement**, **Resection of infected bone**, **Amputation** (if uncontrolled) | https://iwgdfguidelines.org |

## Charcot Neuroarthropathy
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Progressive destruction of bones and joints** in neuropathic foot | https://www.apma.org/Patients/Diabetes.cfm |
| **Pathophysiology** | **Neurotraumatic** (repetitive microtrauma) + **Neurovascular** (? Blood flow ? Osteopenia) | https://www.apma.org/Patients/Diabetes.cfm |
| **Acute Phase** | **Erythema**, **Edema**, **Warmth** (often misdiagnosed as infection) | https://www.apma.org/Patients/Diabetes.cfm |
| **Eichenholtz Classification** | **Stage 0**: Inflammatory; **Stage I**: Fragmentation; **Stage II**: Coalescence; **Stage III**: Consolidation | https://www.apma.org/Patients/Diabetes.cfm |
| **Treatment** | **Offloading** (total contact cast, CROW boot), **Immobilization** until consolidation (3-6 months) | https://www.apma.org/Patients/Diabetes.cfm |
