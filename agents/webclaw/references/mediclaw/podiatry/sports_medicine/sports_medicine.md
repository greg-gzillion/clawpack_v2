# Podiatry - Sports Medicine

## Common Sports-Related Foot and Ankle Injuries
| Injury | Mechanism | Sports | Treatment | URL |
|--------|-----------|--------|-----------|-----|
| **Ankle Sprain** | **Inversion** | All sports | **Functional rehabilitation**, **Bracing** | https://www.aapsm.org/ankle-sprain.html |
| **Achilles Tendinopathy** | **Overuse** | Running, Basketball | **Eccentric strengthening**, **Activity modification** | https://www.aapsm.org/achilles-tendinopathy.html |
| **Plantar Fasciitis** | **Overuse** | Running, Dance | **Stretching**, **Night splint**, **Orthotics** | https://www.aapsm.org/plantar-fasciitis.html |
| **Stress Fracture** | **Repetitive loading** | Running, Gymnastics | **Rest/Immobilization**, **Address risk factors** | https://www.aapsm.org/stress-fractures.html |
| **Turf Toe** | **Hyperextension of first MTP** | Football, Soccer | **Stiff-soled shoes**, **Taping**, **Carbon fiber plate** | https://orthoinfo.aaos.org/en/diseases--conditions/turf-toe/ |
| **Jones Fracture** | **Inversion**, **Repetitive stress** | Soccer, Basketball | **Surgical fixation** (athletes), **NWB cast** | https://orthoinfo.aaos.org/en/diseases--conditions/toe-and-forefoot-fractures/ |
| **Lisfranc Injury** | **Axial load**, **Twisting** | Football, Equestrian | **Surgical fixation** (unstable) | https://orthoinfo.aaos.org/en/diseases--conditions/lisfranc-midfoot-injury/ |
| **Navicular Stress Fracture** | **Overuse** | Running, Jumping | **NWB cast**, **Surgical fixation** (athletes) | https://orthoinfo.aaos.org/en/diseases--conditions/stress-fractures-of-the-foot-and-ankle/ |

## Turf Toe
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Hyperextension injury of first MTP joint** with **plantar plate/capsular injury** | https://orthoinfo.aaos.org/en/diseases--conditions/turf-toe/ |
| **Mechanism** | **Axial load** on **plantarflexed first ray** (football lineman pushing off) | https://orthoinfo.aaos.org/en/diseases--conditions/turf-toe/ |
| **Grading** | **Grade 1**: Stretch (microtears); **Grade 2**: Partial tear; **Grade 3**: Complete tear | https://orthoinfo.aaos.org/en/diseases--conditions/turf-toe/ |
| **Physical Exam** | **Pain with passive dorsiflexion**, **Swelling**, **Ecchymosis**, **Instability** (grade 3) | https://orthoinfo.aaos.org/en/diseases--conditions/turf-toe/ |
| **Imaging** | **X-ray** (rule out fracture), **MRI** (assess plantar plate) | https://orthoinfo.aaos.org/en/diseases--conditions/turf-toe/ |
| **Conservative Treatment** | **Stiff-soled shoes**, **Taping**, **Carbon fiber plate**, **Rest** | https://orthoinfo.aaos.org/en/diseases--conditions/turf-toe/ |
| **Surgical Treatment** | **Plantar plate repair** (grade 3, Athletes) | https://orthoinfo.aaos.org/en/diseases--conditions/turf-toe/ |

## Return to Play Guidelines
| Principle | Description | URL |
|-----------|-------------|-----|
| **Pain-Free** | **No pain** with sport-specific activities | https://www.aapsm.org/return-to-play.html |
| **Full ROM** | **Full pain-free range of motion** | https://www.aapsm.org/return-to-play.html |
| **Strength** | **=90% of contralateral strength** | https://www.aapsm.org/return-to-play.html |
| **Functional Testing** | **Successful completion of sport-specific drills** | https://www.aapsm.org/return-to-play.html |
| **Psychological Readiness** | **Confidence in injured limb** | https://www.aapsm.org/return-to-play.html |
