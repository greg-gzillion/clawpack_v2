# Podiatry - Midfoot Disorders

## Lisfranc Injury
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Injury to tarsometatarsal (TMT) joint complex** | https://www.orthobullets.com/foot-and-ankle/7005/foot-anatomy |
| **Mechanism** | **Axial load** on plantarflexed foot, **Twisting**, **Crush** | https://www.orthobullets.com/foot-and-ankle/7005/foot-anatomy |
| **Symptoms** | **Midfoot pain/swelling**, **Inability to bear weight**, **Plantar ecchymosis** (pathognomonic) | https://www.orthobullets.com/foot-and-ankle/7005/foot-anatomy |
| **Imaging** | **Weight-bearing X-ray** (AP, Lateral, Oblique): **Diastasis >2 mm** between 1st and 2nd metatarsal bases, **Fleck sign** | https://www.orthobullets.com/foot-and-ankle/7005/foot-anatomy |
| **Classification** | **Stable** (no displacement) vs **Unstable** (diastasis) | https://www.orthobullets.com/foot-and-ankle/7005/foot-anatomy |
| **Treatment** | **Stable**: Non-weight bearing cast x6-8 weeks; **Unstable**: **Surgical fixation** (ORIF or primary arthrodesis) | https://www.orthobullets.com/foot-and-ankle/7005/foot-anatomy |
