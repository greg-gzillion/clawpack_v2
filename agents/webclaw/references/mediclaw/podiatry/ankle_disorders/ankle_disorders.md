## Chronic Ankle Instability
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Recurrent ankle sprains**, **Persistent pain/swelling**, **Feeling of "giving way"** | https://orthoinfo.aaos.org/en/diseases--conditions/chronic-ankle-instability/ |
| **Etiology** | **Ligamentous laxity** (mechanical instability), **Proprioceptive deficit** (functional instability) | https://orthoinfo.aaos.org/en/diseases--conditions/chronic-ankle-instability/ |
| **Physical Exam** | **+ Anterior drawer**, **+ Talar tilt**, **Proprioceptive deficits** | https://orthoinfo.aaos.org/en/diseases--conditions/chronic-ankle-instability/ |
| **Imaging** | **Stress X-ray**, **MRI** (assess ligament integrity, Associated lesions) | https://orthoinfo.aaos.org/en/diseases--conditions/chronic-ankle-instability/ |
| **Conservative Treatment** | **PT** (proprioception, Strengthening), **Bracing** | https://orthoinfo.aaos.org/en/diseases--conditions/chronic-ankle-instability/ |
| **Surgical Treatment** | **Brostrom-Gould repair** (gold standard), **Anatomic reconstruction**, **Tenodesis** | https://www.acfas.org/footankleinfo/Chronic-Ankle-Instability.htm |
