# Urology - Erectile Dysfunction (ED)

## ED - Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Persistent inability to achieve or maintain erection** sufficient for satisfactory sexual performance | https://www.niddk.nih.gov/health-information/urologic-diseases/erectile-dysfunction |
| **Epidemiology** | Affects 30 million US men; Increases with age (40% at 40, 70% at 70) | https://www.niddk.nih.gov/health-information/urologic-diseases/erectile-dysfunction |
| **Physiology** | **NO ? cGMP ? Smooth muscle relaxation ? Vasodilation ? Venous compression** | https://www.niddk.nih.gov/health-information/urologic-diseases/erectile-dysfunction |

## ED - Etiology
| Category | Causes | URL |
|----------|--------|-----|
| **Vasculogenic** | **Atherosclerosis**, **Hypertension**, **Diabetes**, **PAD**, **Venous leak** | https://www.niddk.nih.gov/health-information/urologic-diseases/erectile-dysfunction |
| **Neurogenic** | **Spinal cord injury**, **Stroke**, **MS**, **Parkinson**, **Pelvic surgery/radiation**, **Diabetic neuropathy** | https://www.niddk.nih.gov/health-information/urologic-diseases/erectile-dysfunction |
| **Endocrinologic** | **Hypogonadism**, **Hyperprolactinemia**, **Thyroid disease** | https://www.niddk.nih.gov/health-information/urologic-diseases/erectile-dysfunction |
| **Medication-Induced** | **Antihypertensives** (beta-blockers, Thiazides), **Antidepressants** (SSRIs), **Antiandrogens**, **Opioids** | https://www.niddk.nih.gov/health-information/urologic-diseases/erectile-dysfunction |
| **Psychogenic** | **Depression**, **Anxiety**, **Performance anxiety**, **Relationship issues** | https://www.niddk.nih.gov/health-information/urologic-diseases/erectile-dysfunction |
