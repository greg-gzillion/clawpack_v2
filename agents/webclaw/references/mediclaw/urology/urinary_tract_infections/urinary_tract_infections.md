# Urology - Urinary Tract Infections (UTI)

## UTI - Classification
| Category | Definition | URL |
|----------|------------|-----|
| **Uncomplicated Cystitis** | Healthy, Non-pregnant, Premenopausal women; No structural abnormalities | https://www.cdc.gov/uti/about/index.html |
| **Complicated UTI** | Men, Pregnant women, Structural/functional abnormalities, Catheters, Immunocompromised, CKD | https://www.cdc.gov/uti/about/index.html |
| **Pyelonephritis** | Infection of renal parenchyma and collecting system | https://www.niddk.nih.gov/health-information/urologic-diseases/kidney-infection-pyelonephritis |
| **Catheter-Associated UTI (CAUTI)** | UTI in patient with indwelling catheter | https://www.cdc.gov/hai/cauti/index.html |
| **Asymptomatic Bacteriuria (ASB)** | Significant bacteriuria without symptoms | https://www.cdc.gov/uti/about/index.html |

## Uncomplicated Cystitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathogens** | **E. coli** (75-95%), **S. saprophyticus** (5-15%), Klebsiella, Proteus | https://www.cdc.gov/uti/about/index.html |
| **Clinical Features** | **Dysuria**, **Frequency**, **Urgency**, **Suprapubic pain**, **Hematuria** | https://www.cdc.gov/uti/about/index.html |
| **First-Line Treatment** | **Nitrofurantoin** 100 mg BID x 5 days, **TMP-SMX DS** BID x 3 days, **Fosfomycin** 3g x 1 | https://www.ncbi.nlm.nih.gov/books/NBK482344/ |
| **Alternative Treatment** | **Ciprofloxacin** 250 mg BID x 3 days or **Levofloxacin** 250 mg daily x 3 days | https://www.fda.gov/drugs/information-drug-class/fluoroquinolone-antimicrobial-drugs |

## Pyelonephritis
| Component | Description | URL |
|-----------|-------------|-----|
| **Clinical Features** | **Fever**, **Chills**, **Flank pain**, **CVA tenderness**, Nausea/Vomiting | https://www.niddk.nih.gov/health-information/urologic-diseases/kidney-infection-pyelonephritis |
| **Diagnosis** | **Urinalysis** (pyuria, WBC casts), **Urine culture**, **Blood cultures** (severe) | https://www.niddk.nih.gov/health-information/urologic-diseases/kidney-infection-pyelonephritis |
| **Outpatient Treatment** | **Ciprofloxacin** 500 mg BID x 7 days or **Levofloxacin** 750 mg daily x 5-7 days | https://www.ncbi.nlm.nih.gov/books/NBK482344/ |
| **Inpatient Treatment** | **Ceftriaxone** 1-2g IV daily, **Cefepime**, **Piperacillin-tazobactam** | https://www.ncbi.nlm.nih.gov/books/NBK482344/ |
| **Duration** | Fluoroquinolones: 5-7 days; Beta-lactams: 10-14 days | https://www.ncbi.nlm.nih.gov/books/NBK482344/ |

## Asymptomatic Bacteriuria (ASB)
| Component | Description | URL |
|-----------|-------------|-----|
| **When to Treat** | **Pregnancy**, **Prior to urologic procedure** with mucosal bleeding | https://www.uspreventiveservicestaskforce.org/uspstf/recommendation/asymptomatic-bacteriuria-in-adults-screening |
| **When NOT to Treat** | **Elderly**, **Catheterized**, **Diabetes**, **Long-term care**, **Non-pregnant women** | https://www.uspreventiveservicestaskforce.org/uspstf/recommendation/asymptomatic-bacteriuria-in-adults-screening |

## Catheter-Associated UTI (CAUTI)
| Component | Description | URL |
|-----------|-------------|-----|
| **Prevention** | **Avoid unnecessary catheters**, **Remove as soon as possible**, **Aseptic insertion**, **Closed system** | https://www.cdc.gov/hai/cauti/index.html |
| **Diagnosis** | Only culture if **symptomatic** (fever, Flank pain, Delirium) | https://www.cdc.gov/hai/cauti/index.html |
| **Treatment** | **Remove/replace catheter** + **Antibiotics** | https://www.ncbi.nlm.nih.gov/books/NBK482344/ |
