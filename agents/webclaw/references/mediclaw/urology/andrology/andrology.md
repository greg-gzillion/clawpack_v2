## Hypogonadism
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnosis** | **Symptoms** + **Low testosterone** (<300 ng/dL on 2 morning measurements) | https://www.auanet.org/guidelines/testosterone-deficiency |
| **Primary** | **High LH/FSH**, **Low T** (Klinefelter, Mumps orchitis, Trauma, Chemo/RT) | https://www.auanet.org/guidelines/testosterone-deficiency |
| **Secondary** | **Low/Normal LH/FSH**, **Low T** (Kallmann, Pituitary tumor, Opioids, Obesity) | https://www.auanet.org/guidelines/testosterone-deficiency |
| **Treatment** | **Testosterone replacement** (gel, Injection, Patch, Pellets), **hCG** (fertility), **Clomiphene** (off-label) | https://www.auanet.org/guidelines/testosterone-deficiency |
| **Contraindications** | **Prostate cancer**, **Breast cancer**, **Hct >50%**, **Severe BPH**, **Untreated OSA**, **Severe CHF** | https://www.auanet.org/guidelines/testosterone-deficiency |

## Peyronie Disease
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Acquired penile curvature** due to **fibrosis of tunica albuginea** | https://www.auanet.org/guidelines/peyronie-disease |
| **Active Phase** | **Pain**, **Progressive curvature** (6-12 months) | https://www.auanet.org/guidelines/peyronie-disease |
| **Stable Phase** | **No pain**, **Stable curvature** (>3-6 months) | https://www.auanet.org/guidelines/peyronie-disease |
| **Non-Surgical** | **Intralesional collagenase** (Xiaflex), **Traction therapy**, **Oral therapies** (limited evidence) | https://www.auanet.org/guidelines/peyronie-disease |
| **Surgical** | **Plication** (adequate length, Simple curvature), **Plaque incision/excision + grafting** (severe curvature, Hourglass, Hinge), **Penile prosthesis** (ED + Peyronie) | https://www.auanet.org/guidelines/peyronie-disease |
