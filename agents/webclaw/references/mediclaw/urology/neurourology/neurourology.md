# Urology - Neurourology

## Neurogenic Bladder - Classification
| Type | Lesion | Urodynamics | Management | URL |
|------|--------|-------------|------------|-----|
| **Upper Motor Neuron (UMN)** | **Above sacral micturition center** (spinal cord, Brainstem, Cerebral) | **Detrusor overactivity**, **Detrusor-sphincter dyssynergia (DSD)** | **Clean intermittent catheterization (CIC)** + **Anticholinergics/Beta-3 agonists** ｱ **Botulinum toxin** | https://www.niddk.nih.gov/health-information/urologic-diseases/neurogenic-bladder |
| **Lower Motor Neuron (LMN)** | **Sacral cord**, **Cauda equina**, **Peripheral nerves** | **Detrusor underactivity**, **Overflow incontinence** | **CIC**, **Valsalva/Crede maneuver** (caution), **Indwelling catheter** | https://www.niddk.nih.gov/health-information/urologic-diseases/neurogenic-bladder |

## Neurogenic Bladder - Management by Condition
| Condition | Bladder Type | Management | URL |
|-----------|--------------|------------|-----|
| **Spinal Cord Injury (Suprasacral)** | **UMN** (detrusor overactivity + DSD) | **CIC + Anticholinergics/Beta-3 agonists**, **Botulinum toxin**, **Augmentation cystoplasty**, **Urinary diversion** | https://www.niddk.nih.gov/health-information/urologic-diseases/neurogenic-bladder |
| **Spina Bifida** | **UMN or LMN** | **CIC + Anticholinergics**, **Monitor upper tracts** (renal US, Urodynamics) | https://www.niddk.nih.gov/health-information/urologic-diseases/neurogenic-bladder |
| **Multiple Sclerosis** | **Detrusor overactivity** (most common) | **Anticholinergics/Beta-3 agonists**, **CIC** (if incomplete emptying) | https://www.nationalmssociety.org |
| **Parkinson Disease** | **Detrusor overactivity** (most common) | **Beta-3 agonists** (preferred), **Anticholinergics** (caution: cognitive effects) | https://www.parkinson.org |
| **Stroke** | **Detrusor overactivity** (acute), May resolve | **Timed voiding**, **Anticholinergics/Beta-3 agonists**, **CIC** if retention | https://www.stroke.org |

## Autonomic Dysreflexia
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Life-threatening sympathetic surge** in response to **noxious stimuli below level of SCI** (usually =T6) | https://www.ninds.nih.gov/health-information/disorders/autonomic-dysreflexia |
| **Triggers** | **Bladder distention** (most common), **Bowel impaction**, **UTI**, **Pressure injury** | https://www.ninds.nih.gov/health-information/disorders/autonomic-dysreflexia |
| **Symptoms** | **Severe hypertension**, **Bradycardia** (reflex), **Headache**, **Flushing/sweating above lesion** | https://www.ninds.nih.gov/health-information/disorders/autonomic-dysreflexia |
| **Management** | **Sit upright**, **Remove trigger** (check Foley, Perform CIC), **Rapid-acting antihypertensive** if BP remains elevated | https://www.ninds.nih.gov/health-information/disorders/autonomic-dysreflexia |
