# Infectious Disease - Hepatitis

## Hepatitis A (HAV)
| Component | Description | URL |
|-----------|-------------|-----|
| **Transmission** | Fecal-oral (contaminated food/water) | https://www.cdc.gov/hepatitis/hav/index.htm |
| **Incubation** | 15-50 days (average 28 days) | https://www.cdc.gov/hepatitis/hav/index.htm |
| **Clinical Features** | Fever, Malaise, Nausea, Jaundice, Elevated transaminases, IgM anti-HAV positive | https://www.cdc.gov/hepatitis/hav/index.htm |
| **Chronicity** | None (acute only) | https://www.cdc.gov/hepatitis/hav/index.htm |
| **Treatment** | Supportive | https://www.cdc.gov/hepatitis/hav/index.htm |
| **Prevention** | HAV vaccine (2 doses), Immune globulin for post-exposure | https://www.cdc.gov/hepatitis/hav/index.htm |

## Hepatitis B (HBV)
| Component | Description | URL |
|-----------|-------------|-----|
| **Transmission** | Blood, Sexual, Vertical | https://www.cdc.gov/hepatitis/hbv/index.htm |
| **Chronicity Risk** | Infants: 90%; Adults: <5% | https://www.cdc.gov/hepatitis/hbv/index.htm |
| **Diagnosis** | HBsAg (active), Anti-HBs (immunity), Anti-HBc IgM (acute), HBV DNA | https://www.cdc.gov/hepatitis/hbv/index.htm |
| **Treatment** | Tenofovir (TAF or TDF), Entecavir | https://www.cdc.gov/hepatitis/hbv/index.htm |
| **Prevention** | HBV vaccine (3 doses), HBIG for post-exposure | https://www.cdc.gov/hepatitis/hbv/index.htm |

## Hepatitis C (HCV)
| Component | Description | URL |
|-----------|-------------|-----|
| **Transmission** | Blood (IVDU, Transfusion before 1992) | https://www.cdc.gov/hepatitis/hcv/index.htm |
| **Chronicity Risk** | 75-85% | https://www.cdc.gov/hepatitis/hcv/index.htm |
| **Diagnosis** | Anti-HCV (screening), HCV RNA PCR (confirm) | https://www.cdc.gov/hepatitis/hcv/index.htm |
| **Treatment** | Glecaprevir-pibrentasvir x 8 weeks, Sofosbuvir-velpatasvir x 12 weeks; >95% SVR | https://www.cdc.gov/hepatitis/hcv/index.htm |
| **Prevention** | No vaccine; Harm reduction | https://www.cdc.gov/hepatitis/hcv/index.htm |

## Hepatitis D (HDV)
| Component | Description | URL |
|-----------|-------------|-----|
| **Virology** | Defective RNA virus requiring HBsAg | https://www.cdc.gov/hepatitis/hdv/index.htm |
| **Coinfection** | Simultaneous HBV + HDV ? More severe acute hepatitis | https://www.cdc.gov/hepatitis/hdv/index.htm |
| **Superinfection** | HDV in chronic HBV ? Accelerated cirrhosis | https://www.cdc.gov/hepatitis/hdv/index.htm |
| **Treatment** | Pegylated IFN-a; Bulevirtide (EU) | https://www.cdc.gov/hepatitis/hdv/index.htm |

## Hepatitis E (HEV)
| Component | Description | URL |
|-----------|-------------|-----|
| **Transmission** | Fecal-oral, Zoonotic (undercooked pork) | https://www.cdc.gov/hepatitis/hev/index.htm |
| **Clinical Features** | Acute self-limited; Severe in pregnancy; Chronic in immunocompromised | https://www.cdc.gov/hepatitis/hev/index.htm |
| **Treatment** | Supportive (acute); Ribavirin (chronic) | https://www.cdc.gov/hepatitis/hev/index.htm |
