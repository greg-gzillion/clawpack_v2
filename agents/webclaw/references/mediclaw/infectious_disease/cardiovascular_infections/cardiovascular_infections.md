# Infectious Disease - Cardiovascular Infections

## Infective Endocarditis (IE)
| Component | Description | URL |
|-----------|-------------|-----|
| **Modified Duke Criteria** | Major: Positive blood cultures, Evidence of endocardial involvement; Minor: Predisposition, Fever, Vascular/Immunologic phenomena | https://www.ncbi.nlm.nih.gov/books/NBK557641/ |
| **Microbiology** | S. aureus, Viridans streptococci, Enterococcus, CoNS, HACEK, Culture-negative | https://www.ncbi.nlm.nih.gov/books/NBK557641/ |
| **Diagnosis** | Blood cultures (3 sets), Echocardiography (TTE first, TEE if indicated) | https://www.ncbi.nlm.nih.gov/books/NBK557641/ |
| **Indications for Surgery** | CHF, Uncontrolled infection, Prevention of embolism | https://www.ncbi.nlm.nih.gov/books/NBK557641/ |
| **Prophylaxis** | Only high-risk patients undergoing dental procedures: Amoxicillin 2g PO | https://www.heart.org/en/health-topics/infective-endocarditis |

## Infective Endocarditis - Treatment by Pathogen
| Pathogen | Native Valve | Prosthetic Valve | Duration | URL |
|----------|--------------|------------------|----------|-----|
| **MSSA** | Nafcillin/Oxacillin or Cefazolin | Nafcillin/Oxacillin + Rifampin + Gentamicin | 4-6 weeks (native), =6 weeks (prosthetic) | https://www.ncbi.nlm.nih.gov/books/NBK557641/ |
| **MRSA** | Vancomycin or Daptomycin | Vancomycin + Rifampin + Gentamicin | 4-6 weeks (native), =6 weeks (prosthetic) | https://www.ncbi.nlm.nih.gov/books/NBK557641/ |
| **Viridans Streptococci** | Penicillin G or Ceftriaxone ± Gentamicin | Same + Gentamicin | 4 weeks (native), 6 weeks (prosthetic) | https://www.ncbi.nlm.nih.gov/books/NBK557641/ |
| **Enterococcus** | Ampicillin + Ceftriaxone | Same | 4-6 weeks | https://www.ncbi.nlm.nih.gov/books/NBK557641/ |

## Pericarditis
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathogens** | Viral (most common), Bacterial (S. aureus, S. pneumoniae, M. tuberculosis), Fungal | https://www.ncbi.nlm.nih.gov/books/NBK431080/ |
| **Clinical Features** | Sharp chest pain (worse supine, Relieved leaning forward), Friction rub | https://www.ncbi.nlm.nih.gov/books/NBK431080/ |
| **Diagnosis** | Echo (effusion), ECG (diffuse ST elevation, PR depression), CRP/ESR | https://www.ncbi.nlm.nih.gov/books/NBK431080/ |
| **Treatment** | NSAIDs + Colchicine x 3 months; Corticosteroids (refractory) | https://www.ncbi.nlm.nih.gov/books/NBK431080/ |
