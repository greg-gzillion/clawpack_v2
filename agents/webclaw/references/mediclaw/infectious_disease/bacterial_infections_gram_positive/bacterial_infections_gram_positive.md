# Infectious Disease - Gram-Positive Bacterial Infections

## Staphylococcus aureus
| Syndrome | Clinical Features | Treatment | URL |
|----------|------------------|-----------|-----|
| **MSSA** | Skin/soft tissue, Pneumonia, Bacteremia, Endocarditis, Osteomyelitis | **Nafcillin/Oxacillin**, **Cefazolin** (non-CNS), **Vancomycin** (severe penicillin allergy) | https://www.idsociety.org/mrsa |
| **MRSA** | Same (healthcare or community-associated) | **Vancomycin**, **Daptomycin** (bacteremia/endocarditis), **Linezolid** (pneumonia), **Ceftaroline**, **TMP-SMX**, **Doxycycline**, **Clindamycin** | |
| **Toxic Shock Syndrome (TSS)** | Fever, Hypotension, Diffuse macular rash, Multi-organ failure | **Clindamycin** (toxin suppression) + **Vancomycin or Nafcillin** + **IVIG** | |
| **Staphylococcal Scalded Skin Syndrome (SSSS)** | Diffuse erythema, Superficial blistering, Nikolsky positive | **Nafcillin/Oxacillin** or **Cefazolin** | |
| **Food Poisoning** | Nausea, Vomiting, Diarrhea (2-6 hours after ingestion) | **Supportive care** (self-limited) | https://www.cdc.gov/foodsafety |

## Coagulase-Negative Staphylococci (CoNS)
| Species | Clinical Significance | Treatment | URL |
|---------|----------------------|-----------|-----|
| **S. epidermidis** | **Prosthetic device infections**, Catheter-related bacteremia, **Biofilm producer** | **Vancomycin** (empiric); Remove hardware if possible | https://www.idsociety.org |
| **S. lugdunensis** | **Aggressive** (similar to S. aureus); Endocarditis, Bacteremia | Treat as S. aureus | |
| **S. saprophyticus** | **UTI in young sexually active women** | TMP-SMX, Nitrofurantoin, Ciprofloxacin | |

## Streptococcus pyogenes (Group A Strep)
| Syndrome | Clinical Features | Treatment | URL |
|----------|------------------|-----------|-----|
| **Pharyngitis** | Sore throat, Fever, Exudates, **No cough** | **Penicillin VK** or **Amoxicillin** x 10 days; **Cephalexin** (non-anaphylactic allergy); **Azithromycin** (macrolide resistance >10%) | https://www.idsociety.org/pharyngitis |
| **Scarlet Fever** | Pharyngitis + **Sandpaper rash**, **Strawberry tongue**, Pastia lines | Same as pharyngitis | |
| **Impetigo** | Honey-crusted lesions | **Topical mupirocin** or **Oral cephalexin** | https://www.idsociety.org/ssti |
| **Erysipelas** | **Raised, Demarcated**, Facial, Painful cellulitis | **Penicillin** or **Ceftriaxone** | |
| **Cellulitis** | Non-raised, Non-demarcated | **Cephalexin**, **Clindamycin** | |
| **Necrotizing Fasciitis** | **Pain out of proportion**, Rapid spread, Crepitus, Systemic toxicity | **Emergent surgical debridement** + **Penicillin + Clindamycin** | |
| **Streptococcal Toxic Shock Syndrome** | Hypotension, Multi-organ failure, Often with necrotizing fasciitis | **Clindamycin** (toxin suppression) + **Penicillin** + **IVIG** | |
| **Rheumatic Fever** | **JONES criteria**: Joints (migratory polyarthritis), Carditis, Nodules, Erythema marginatum, Sydenham chorea | **Secondary prophylaxis**: Penicillin G benzathine IM q4 weeks | https://www.heart.org/rheumatic-fever |
| **Post-Streptococcal Glomerulonephritis** | Hematuria, Proteinuria, Hypertension, Edema (1-3 weeks after infection) | **Supportive care**; No role for antibiotics | |

## Streptococcus agalactiae (Group B Strep)
| Syndrome | Clinical Features | Treatment | URL |
|----------|------------------|-----------|-----|
| **Neonatal Sepsis** | Early-onset (<7 days): Pneumonia, Sepsis; Late-onset (7-90 days): Meningitis, Bacteremia | **Ampicillin + Gentamicin** (neonates); **Penicillin G** | https://www.cdc.gov/groupbstrep |
| **Pregnancy** | **Screen at 35-37 weeks**; Intrapartum antibiotic prophylaxis (IAP) if positive | **Penicillin G** (preferred) or **Ampicillin**; **Cefazolin** (non-anaphylactic allergy); **Clindamycin** (only if susceptible) or **Vancomycin** (anaphylaxis) | |
| **Adult Infections** | Bacteremia, Cellulitis, Osteomyelitis, Endocarditis | **Penicillin G** or **Ceftriaxone** | |

## Streptococcus pneumoniae (Pneumococcus)
| Syndrome | Clinical Features | Treatment | URL |
|----------|------------------|-----------|-----|
| **Pneumonia** | Acute onset, Fever, Productive cough, Lobar consolidation | **Ceftriaxone** + **Azithromycin** (empiric); **Ampicillin** or **Penicillin G** if susceptible | https://www.idsociety.org/pneumonia |
| **Meningitis** | Fever, Headache, Nuchal rigidity, AMS | **Ceftriaxone** + **Vancomycin** + **Dexamethasone** (before or with antibiotics) | https://www.idsociety.org/meningitis |
| **Otitis Media** | Ear pain, Bulging TM | **Amoxicillin** (high-dose: 90 mg/kg/day) | |
| **Sinusitis** | Facial pain, Purulent nasal discharge | **Amoxicillin-clavulanate** | |
| **Prevention** | **PCV15 or PCV20** vaccine; **PPSV23** for select adults | | https://www.cdc.gov/vaccines |

## Viridans Group Streptococci
| Species | Clinical Significance | Treatment | URL |
|---------|----------------------|-----------|-----|
| **S. anginosus group** | **Abscesses** (brain, Liver, Lung), **Empyema** | **Penicillin G** or **Ceftriaxone** + **Drainage** | https://www.idsociety.org |
| **S. mitis, S. oralis** | **Neutropenic bacteremia**, **Endocarditis** | **Vancomycin** (empiric); **Penicillin** if susceptible | |
| **S. bovis (S. gallolyticus)** | **Bacteremia/Endocarditis** ? **Colonoscopy** (colonic neoplasia association) | **Penicillin G** or **Ceftriaxone** | |
| **S. mutans** | **Dental caries**, **Endocarditis** | Amoxicillin (dental prophylaxis for high-risk patients) | |

## Enterococcus
| Species | Resistance | Clinical Significance | Treatment | URL |
|---------|------------|----------------------|-----------|-----|
| **E. faecalis** | Usually ampicillin-susceptible | UTI, Bacteremia, Endocarditis, Intra-abdominal | **Ampicillin**; **Vancomycin** (if resistant or allergy) | https://www.idsociety.org/enterococcus |
| **E. faecium** | **Often ampicillin-resistant** | Same | **Vancomycin** (if susceptible) | |
| **VRE (Vancomycin-Resistant Enterococcus)** | **Vancomycin-resistant** (vanA, vanB) | Same | **Linezolid**, **Daptomycin**, **Tigecycline**, **Oritavancin** | |
| **Endocarditis** | Requires **synergistic bactericidal therapy** | **Ampicillin + Ceftriaxone** (E. faecalis); **Ampicillin + Gentamicin** (if gentamicin-susceptible) | |
