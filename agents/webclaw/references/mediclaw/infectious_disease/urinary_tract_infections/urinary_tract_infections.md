# Infectious Disease - Urinary Tract Infections

## UTI - Classification
| Category | Definition | URL |
|----------|------------|-----|
| **Uncomplicated Cystitis** | Healthy, Non-pregnant, Premenopausal women | https://www.cdc.gov/uti/index.html |
| **Complicated UTI** | Men, Pregnant, Structural abnormalities, Catheters | https://www.cdc.gov/uti/index.html |
| **Pyelonephritis** | Infection of renal parenchyma | https://www.cdc.gov/uti/index.html |
| **Catheter-Associated UTI (CAUTI)** | UTI with indwelling catheter | https://www.cdc.gov/hai/cauti/index.html |

## Uncomplicated Cystitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathogens** | E. coli (75-95%), S. saprophyticus | https://www.cdc.gov/uti/index.html |
| **Clinical Features** | Dysuria, Frequency, Urgency, Suprapubic pain | https://www.cdc.gov/uti/index.html |
| **First-Line Treatment** | Nitrofurantoin 100 mg BID x 5 days, TMP-SMX DS BID x 3 days, Fosfomycin 3g x 1 | https://www.cdc.gov/uti/index.html |

## Complicated UTI and Pyelonephritis
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathogens** | E. coli, Klebsiella, Proteus, Pseudomonas, Enterococcus | https://www.cdc.gov/uti/index.html |
| **Clinical Features** | Fever, Chills, Flank pain, CVA tenderness | https://www.cdc.gov/uti/index.html |
| **Empiric Treatment (Outpatient)** | Ciprofloxacin 500 mg BID x 7 days or Levofloxacin 750 mg daily x 5-7 days | https://www.cdc.gov/uti/index.html |
| **Empiric Treatment (Inpatient)** | Ceftriaxone 1-2g IV daily, Cefepime, Piperacillin-tazobactam | https://www.cdc.gov/uti/index.html |
| **Duration** | Fluoroquinolones: 5-7 days; Beta-lactams: 10-14 days | https://www.cdc.gov/uti/index.html |

## Asymptomatic Bacteriuria (ASB)
| Component | Description | URL |
|-----------|-------------|-----|
| **When to Treat** | Pregnancy, Prior to urologic procedure with mucosal bleeding | https://www.cdc.gov/uti/index.html |
| **When NOT to Treat** | Elderly, Catheterized patients, Diabetes, Long-term care | https://www.cdc.gov/uti/index.html |

## Recurrent UTI
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | =2 UTIs in 6 months or =3 UTIs in 12 months | https://www.cdc.gov/uti/index.html |
| **Prevention** | Hydration, Postcoital voiding, Vaginal estrogen, Methenamine | https://www.cdc.gov/uti/index.html |
| **Antibiotic Prophylaxis** | Continuous or Postcoital | https://www.cdc.gov/uti/index.html |
