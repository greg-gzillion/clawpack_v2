# Infectious Disease - Central Nervous System Infections

## Acute Bacterial Meningitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Common Pathogens by Age** | Neonates: GBS, E. coli, Listeria; Children: S. pneumoniae, N. meningitidis; Adults: S. pneumoniae, N. meningitidis, Listeria (>50) | https://www.cdc.gov/meningitis/index.html |
| **Clinical Features** | Fever, Headache, Nuchal rigidity, Photophobia, Altered mental status | https://www.cdc.gov/meningitis/index.html |
| **Diagnosis** | LP: Opening pressure, WBC >1000, PMN predominance, Low glucose, Elevated protein | https://www.cdc.gov/meningitis/index.html |
| **Empiric Treatment** | Ceftriaxone 2g IV q12h + Vancomycin + Dexamethasone 10 mg IV q6h | https://www.cdc.gov/meningitis/index.html |
| **Dexamethasone** | Adults with suspected pneumococcal meningitis: 10 mg IV q6h x 4 days | https://www.cdc.gov/meningitis/index.html |

## Viral Meningitis/Aseptic Meningitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Common Pathogens** | Enteroviruses, HSV-2, VZV, Arboviruses, HIV | https://www.cdc.gov/meningitis/viral.html |
| **CSF Findings** | WBC 10-500 (lymphocyte predominant), Normal glucose, Normal/mildly elevated protein | https://www.cdc.gov/meningitis/viral.html |
| **Treatment** | Supportive; Acyclovir for HSV/VZV | https://www.cdc.gov/meningitis/viral.html |

## Encephalitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Common Pathogens** | HSV-1, VZV, West Nile Virus, EBV, Enteroviruses, Rabies, Autoimmune | https://www.cdc.gov/encephalitis/index.html |
| **Clinical Features** | Fever, Headache, Altered mental status, Seizures, Focal deficits | https://www.cdc.gov/encephalitis/index.html |
| **Diagnosis** | MRI brain (temporal lobe in HSV), CSF PCR, EEG | https://www.cdc.gov/encephalitis/index.html |
| **Empiric Treatment** | Acyclovir 10 mg/kg IV q8h (all suspected) | https://www.cdc.gov/encephalitis/index.html |

## Brain Abscess
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathogenesis** | Contiguous spread (sinusitis, Otitis, Dental), Hematogenous, Post-neurosurgery | https://www.ncbi.nlm.nih.gov/books/NBK441841/ |
| **Microbiology** | Streptococci, Oral anaerobes, S. aureus, Gram-negatives | https://www.ncbi.nlm.nih.gov/books/NBK441841/ |
| **Diagnosis** | MRI with contrast (ring-enhancing lesion, Restricted diffusion) | https://www.ncbi.nlm.nih.gov/books/NBK441841/ |
| **Empiric Treatment** | Ceftriaxone + Metronidazole; Add Vancomycin if post-neurosurgery | https://www.ncbi.nlm.nih.gov/books/NBK441841/ |
| **Duration** | 4-8 weeks IV antibiotics | https://www.ncbi.nlm.nih.gov/books/NBK441841/ |
