# Infectious Disease - Sepsis and Septic Shock

## Sepsis - Definitions (Sepsis-3)
| Term | Definition | URL |
|------|------------|-----|
| **Sepsis** | Life-threatening organ dysfunction; SOFA score increase =2 points | https://www.cdc.gov/sepsis/index.html |
| **Septic Shock** | Sepsis + Vasopressor requirement + Lactate >2 mmol/L | https://www.cdc.gov/sepsis/index.html |
| **qSOFA** | RR =22, Altered mentation, SBP =100 mmHg | https://www.cdc.gov/sepsis/index.html |

## Surviving Sepsis Campaign - Hour-1 Bundle
| Component | Action | URL |
|-----------|--------|-----|
| **1. Measure Lactate** | Serum lactate; Remeasure if initial >2 | https://www.sccm.org/SurvivingSepsisCampaign |
| **2. Obtain Blood Cultures** | At least 2 sets before antibiotics | https://www.sccm.org/SurvivingSepsisCampaign |
| **3. Administer Antibiotics** | Empiric broad-spectrum | https://www.sccm.org/SurvivingSepsisCampaign |
| **4. Fluid Resuscitation** | 30 mL/kg crystalloid | https://www.sccm.org/SurvivingSepsisCampaign |
| **5. Vasopressors** | Norepinephrine (first-line) | https://www.sccm.org/SurvivingSepsisCampaign |

## Sepsis - Vasopressors
| Agent | Dose | Notes | URL |
|-------|------|-------|-----|
| **Norepinephrine** | 0.05-2 mcg/kg/min | First-line | https://www.sccm.org/SurvivingSepsisCampaign |
| **Vasopressin** | 0.03-0.04 units/min | Add to norepinephrine | https://www.sccm.org/SurvivingSepsisCampaign |
| **Epinephrine** | 0.05-2 mcg/kg/min | Second-line | https://www.sccm.org/SurvivingSepsisCampaign |
