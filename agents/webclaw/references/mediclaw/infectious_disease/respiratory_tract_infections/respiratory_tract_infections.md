# Infectious Disease - Respiratory Tract Infections

## Upper Respiratory Tract Infections
| Syndrome | Common Pathogens | Treatment | URL |
|----------|------------------|-----------|-----|
| **Acute Pharyngitis** | Viral, S. pyogenes (GAS) | Penicillin or Amoxicillin (GAS) | https://www.cdc.gov/groupastrep/index.html |
| **Acute Otitis Media** | S. pneumoniae, H. influenzae, M. catarrhalis | Amoxicillin (high-dose) | https://www.cdc.gov/ear-infection/index.html |
| **Acute Sinusitis** | S. pneumoniae, H. influenzae, M. catarrhalis | Amoxicillin-clavulanate | https://www.cdc.gov/sinus-infection/index.html |
| **Epiglottitis** | H. influenzae type b | Secure airway + Ceftriaxone | https://www.cdc.gov/hi-disease/index.html |
| **Croup** | Parainfluenza, RSV | Dexamethasone; Nebulized epinephrine | https://www.cdc.gov/parainfluenza/index.html |

## Community-Acquired Pneumonia (CAP)
| Component | Description | URL |
|-----------|-------------|-----|
| **Common Pathogens** | S. pneumoniae, H. influenzae, M. pneumoniae, C. pneumoniae, Legionella, Viruses | https://www.cdc.gov/pneumonia/index.html |
| **Diagnosis** | CXR, Sputum culture, Blood cultures, Urine antigen, PCR | https://www.cdc.gov/pneumonia/index.html |
| **Outpatient Treatment** | Amoxicillin or Doxycycline or Macrolide | https://www.cdc.gov/pneumonia/index.html |
| **Inpatient Treatment** | Ceftriaxone + Azithromycin or Respiratory fluoroquinolone | https://www.cdc.gov/pneumonia/index.html |
| **Duration** | 5 days (minimum) | https://www.cdc.gov/pneumonia/index.html |

| Component | Description | URL |

## Aspiration Pneumonia/Pneumonitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Aspiration Pneumonitis** | Chemical injury; No antibiotics | https://www.cdc.gov/pneumonia/index.html |
| **Aspiration Pneumonia** | Oral anaerobes, S. pneumoniae, Gram-negatives | Ampicillin-sulbactam or Clindamycin | https://www.cdc.gov/pneumonia/index.html |

## Lung Abscess
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathogens** | Oral anaerobes, S. aureus, Klebsiella | https://www.cdc.gov/pneumonia/index.html |
| **Treatment** | Clindamycin or Ampicillin-sulbactam x 4-8 weeks + Drainage | https://www.cdc.gov/pneumonia/index.html |

## Empyema
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Pus in pleural space | https://www.cdc.gov/pneumonia/index.html |
| **Diagnosis** | Thoracentesis: pH <7.2, Glucose <60, LDH >1000 | https://www.cdc.gov/pneumonia/index.html |
| **Treatment** | Chest tube drainage + Intrapleural fibrinolytics + Antibiotics | https://www.cdc.gov/pneumonia/index.html |
