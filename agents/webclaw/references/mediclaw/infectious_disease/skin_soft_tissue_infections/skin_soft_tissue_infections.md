## Erysipelas
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathogen** | S. pyogenes | https://www.cdc.gov/groupastrep/index.html |

## Cellulitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathogens** | S. pyogenes, S. aureus (MSSA, MRSA) | https://www.cdc.gov/groupastrep/index.html |
| **Clinical Features** | Non-raised, Non-demarcated, Erythema, Warmth, Tenderness | https://www.cdc.gov/groupastrep/index.html |
| **Treatment (Non-purulent)** | Cephalexin, Clindamycin | https://www.cdc.gov/groupastrep/index.html |
| **Treatment (Purulent)** | I&D + TMP-SMX, Doxycycline | https://www.cdc.gov/mrsa/index.html |
| **Duration** | 5-7 days | https://www.cdc.gov/groupastrep/index.html |

## Diabetic Foot Infection (DFI)
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathogens** | Polymicrobial (S. aureus, Streptococci, Gram-negatives, Anaerobes) | https://www.cdc.gov/diabetes/index.html |
| **Diagnosis** | Probe-to-bone test, X-ray, MRI | https://www.cdc.gov/diabetes/index.html |
| **Treatment (Mild)** | Cephalexin, Amoxicillin-clavulanate | https://www.cdc.gov/diabetes/index.html |
| **Treatment (Moderate-Severe)** | Piperacillin-tazobactam, Ceftriaxone + Metronidazole | https://www.cdc.gov/diabetes/index.html |
| **Duration** | 1-2 weeks (soft tissue), 4-6 weeks (osteomyelitis) | https://www.cdc.gov/diabetes/index.html |
