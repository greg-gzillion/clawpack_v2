## Pneumocystis jirovecii (PJP)
| Component | Description | URL |
|-----------|-------------|-----|
| **Risk Factors** | HIV (CD4 <200), Corticosteroids (>20 mg prednisone x >4 weeks) | https://www.cdc.gov/fungal/diseases/pneumocystis-pneumonia/index.html |
| **Clinical Features** | Progressive dyspnea, Dry cough, Hypoxia, Elevated LDH | https://www.cdc.gov/fungal/diseases/pneumocystis-pneumonia/index.html |
| **Diagnosis** | PCR (sputum, BAL), DFA, GMS stain | https://www.cdc.gov/fungal/diseases/pneumocystis-pneumonia/index.html |
| **Treatment** | TMP-SMX (high-dose) x 21 days + Corticosteroids if hypoxic | https://www.cdc.gov/fungal/diseases/pneumocystis-pneumonia/index.html |
| **Prophylaxis** | TMP-SMX DS daily | https://www.cdc.gov/fungal/diseases/pneumocystis-pneumonia/index.html |
