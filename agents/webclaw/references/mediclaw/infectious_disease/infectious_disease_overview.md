# Infectious Disease - Overview

## What is Infectious Disease?
Infectious Disease is the medical specialty concerned with the diagnosis, management, and prevention of infections caused by bacteria, viruses, fungi, parasites, and prions. It encompasses all organ systems and requires understanding of microbiology, antimicrobial therapy, epidemiology, and infection control.

## Host-Pathogen Interactions
| Concept | Description | URL |
|---------|-------------|-----|
| **Colonization** | Presence of microorganisms without tissue invasion or immune response | https://www.idsociety.org |
| **Infection** | Invasion and multiplication of pathogens with host response | |
| **Pathogenicity** | Ability of microorganism to cause disease | |
| **Virulence** | Degree of pathogenicity (virulence factors: toxins, Adhesins, Capsules, Enzymes) | |
| **Infectious Dose (ID50)** | Number of organisms required to cause infection in 50% of hosts | |
| **Incubation Period** | Time from exposure to symptom onset | |
| **Period of Communicability** | Time during which infected host can transmit to others | |
