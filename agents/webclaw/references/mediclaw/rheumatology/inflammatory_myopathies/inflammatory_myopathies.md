# Rheumatology - Inflammatory Myopathies

## Classification
| Type | Key Features | Autoantibodies | URL |
|------|--------------|----------------|-----|
| **Dermatomyositis** | **Proximal weakness** + **Characteristic rash** (heliotrope, Gottron papules) | **Anti-Mi-2**, **Anti-TIF1?**, **Anti-MDA5**, **Anti-NXP2** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Inflammatory-Myopathies |
| **Polymyositis** | **Proximal weakness** without rash | **Anti-Jo-1**, **Anti-SRP** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Inflammatory-Myopathies |
| **Inclusion Body Myositis** | **Asymmetric quadriceps/finger flexor weakness**, **Refractory to steroids** | **Anti-cN1A** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Inflammatory-Myopathies |
| **Immune-Mediated Necrotizing Myopathy** | **Severe proximal weakness**, **Very high CK** | **Anti-HMGCR** (statin-associated), **Anti-SRP** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Inflammatory-Myopathies |
| **Antisynthetase Syndrome** | **Myositis** + **ILD** + **Raynaud** + **Mechanic's hands** + **Arthritis** | **Anti-Jo-1**, **Anti-PL-7**, **Anti-PL-12**, **Anti-EJ**, **Anti-OJ** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Inflammatory-Myopathies |

## Clinical Features
| Feature | Description | URL |
|---------|-------------|-----|
| **Muscle Weakness** | **Proximal**, **Symmetric**, **Insidious onset**; Difficulty climbing stairs, Rising from chair, Combing hair | https://www.myositis.org/about-myositis/symptoms/ |
| **Skin (Dermatomyositis)** | **Heliotrope rash** (periorbital), **Gottron papules** (MCP/PIP extensor), **Shawl sign**, **V-sign**, **Mechanic's hands** | https://www.myositis.org/about-myositis/symptoms/ |
| **Pulmonary** | **ILD** (especially antisynthetase syndrome, Anti-MDA5) | https://www.myositis.org/about-myositis/symptoms/ |
| **Malignancy** | **? Risk in dermatomyositis** (especially anti-TIF1?, anti-NXP2); **Screen age-appropriate** | https://www.myositis.org/about-myositis/symptoms/ |

## Treatment of Inflammatory Myopathies
| Type | First-Line | Refractory | URL |
|------|------------|------------|-----|
| **Dermatomyositis/Polymyositis** | **High-dose corticosteroids** + **Methotrexate** or **Azathioprine** or **Mycophenolate** | **IVIG**, **Rituximab**, **Tacrolimus**, **Cyclophosphamide** | https://www.myositis.org/for-healthcare-professionals/treatment-guidelines/ |
| **Immune-Mediated Necrotizing Myopathy** | **Corticosteroids** + **IVIG** or **Rituximab** (especially anti-SRP) | Mycophenolate, Cyclophosphamide | https://www.myositis.org/for-healthcare-professionals/treatment-guidelines/ |
| **Inclusion Body Myositis** | **No proven immunosuppressive therapy**; **Exercise**, **Fall prevention**, **Treat dysphagia** | Supportive | https://www.myositis.org/for-healthcare-professionals/treatment-guidelines/ |
