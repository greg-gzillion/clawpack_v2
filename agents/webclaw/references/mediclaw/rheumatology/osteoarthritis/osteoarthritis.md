# Rheumatology - Osteoarthritis (OA)

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Degenerative joint disease** with **cartilage loss**, **bony changes** (osteophytes, sclerosis), and **variable inflammation** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Osteoarthritis |
| **Pathophysiology** | **Mechanical and inflammatory** factors ? **Chondrocyte dysfunction** ? **Cartilage degradation** | https://www.niams.nih.gov/health-topics/osteoarthritis |
| **Prevalence** | **Most common form of arthritis**; **>30 million US adults** | https://www.cdc.gov/arthritis/types/osteoarthritis.html |
| **Risk Factors** | **Age**, **Obesity**, **Female sex**, **Joint injury**, **Genetics**, **Occupation** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Osteoarthritis |

## Clinical Features
| Feature | Description | URL |
|---------|-------------|-----|
| **Symptoms** | **Use-related pain** (worse with activity, better with rest), **Morning stiffness <30 min**, **Crepitus**, **Limited range of motion** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Osteoarthritis |
| **Common Joints** | **Knees**, **Hips**, **DIP** (Heberden nodes), **PIP** (Bouchard nodes), **First CMC** (thumb base), **Spine** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Osteoarthritis |
| **Imaging** | **Joint space narrowing**, **Osteophytes**, **Subchondral sclerosis**, **Subchondral cysts** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Osteoarthritis |
| **Laboratory** | **Normal ESR/CRP** (unless erosive OA), **No specific biomarkers** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Osteoarthritis |

## Treatment of OA (ACR/AF 2019 Guidelines)
| Intervention | Recommendation | URL |
|--------------|----------------|-----|
| **Non-Pharmacologic** | **Exercise** (strongly recommended), **Weight loss** (if overweight/obese), **Tai Chi**, **Self-efficacy programs** | https://www.rheumatology.org/Portals/0/Files/Osteoarthritis-Guideline-2019.pdf |
| **Pharmacologic (Topical)** | **Topical NSAIDs** (strongly recommended for knee OA), **Topical capsaicin** | https://www.rheumatology.org/Portals/0/Files/Osteoarthritis-Guideline-2019.pdf |
| **Pharmacologic (Oral)** | **Oral NSAIDs** (lowest effective dose), **Acetaminophen** (conditional), **Duloxetine** (conditional) | https://www.rheumatology.org/Portals/0/Files/Osteoarthritis-Guideline-2019.pdf |
| **Intraarticular** | **Corticosteroids** (conditional), **Hyaluronic acid** (conditional - not recommended by ACR) | https://www.rheumatology.org/Portals/0/Files/Osteoarthritis-Guideline-2019.pdf |
| **NOT Recommended** | **Glucosamine**, **Chondroitin**, **Bisphosphonates**, **Hydroxychloroquine**, **Methotrexate**, **Stem cell injections**, **PRP** | https://www.rheumatology.org/Portals/0/Files/Osteoarthritis-Guideline-2019.pdf |
| **Surgical** | **Total joint arthroplasty** (for refractory pain, Functional limitation) | https://www.rheumatology.org/Portals/0/Files/Osteoarthritis-Guideline-2019.pdf |
