## Pre-Biologic Screening
| Test | Purpose | URL |
|------|---------|-----|
| **TB Screening** | **IGRA** (QuantiFERON, T-SPOT) or **TST**; Treat latent TB before starting TNFi, IL-6i, IL-17i, JAKi | https://www.cdc.gov/tb/topic/testing/igra.htm |
| **Hepatitis B** | **HBsAg**, **Anti-HBc**, **Anti-HBs**; Prophylaxis if chronic HBV | https://www.cdc.gov/hepatitis/hbv/index.htm |
| **Hepatitis C** | **Anti-HCV**; Treat before immunosuppression | https://www.cdc.gov/hepatitis/hcv/index.htm |
| **HIV** | **HIV Ag/Ab** | https://www.cdc.gov/hiv/testing/index.html |
| **Vaccinations** | **Influenza**, **Pneumococcal**, **Zoster (Shingrix)** , **COVID-19** (before starting if possible) | https://www.cdc.gov/vaccines/schedules/hcp/imz/adult.html |
