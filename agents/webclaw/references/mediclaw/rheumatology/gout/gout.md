## Clinical Stages of Gout
| Stage | Description | URL |
|-------|-------------|-----|
| **Asymptomatic Hyperuricemia** | **Uric acid >6.8 mg/dL** without symptoms | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Gout |
| **Acute Gout Flare** | **Sudden severe pain**, **Erythema**, **Swelling** (often **first MTP - podagra**) | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Gout |
| **Intercritical Gout** | **Asymptomatic period** between flares | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Gout |
| **Chronic Tophaceous Gout** | **Tophi** (MSU deposits), **Chronic arthritis**, **Erosions** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Gout |

## Treatment of Gout (ACR 2020 Guidelines)
| Phase | Recommendation | URL |
|-------|----------------|-----|
| **Acute Flare** | **NSAIDs** (first-line), **Colchicine** (low-dose: 1.2 mg ? 0.6 mg 1h later), **Corticosteroids** (oral, intraarticular, IM) | https://www.rheumatology.org/Portals/0/Files/Gout-Guideline-2020.pdf |
| **Urate-Lowering Therapy (ULT) Indications** | **=2 flares/year**, **Tophi**, **Erosive arthritis**, **CKD =Stage 3**, **Nephrolithiasis** | https://www.rheumatology.org/Portals/0/Files/Gout-Guideline-2020.pdf |
| **ULT First-Line** | **Allopurinol** (start =100 mg/day, titrate); **Febuxostat** (alternative) | https://www.rheumatology.org/Portals/0/Files/Gout-Guideline-2020.pdf |
| **ULT Target** | **Uric acid <6 mg/dL** (all patients); **<5 mg/dL** (tophi) | https://www.rheumatology.org/Portals/0/Files/Gout-Guideline-2020.pdf |
| **Flare Prophylaxis** | **Colchicine**, **NSAIDs**, or **Low-dose corticosteroids** for **3-6 months** after starting ULT | https://www.rheumatology.org/Portals/0/Files/Gout-Guideline-2020.pdf |
| **HLA-B*58:01 Screening** | Consider in **high-risk populations** (Southeast Asian, African American) before allopurinol | https://www.rheumatology.org/Portals/0/Files/Gout-Guideline-2020.pdf |

## Allopurinol Hypersensitivity Syndrome (AHS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Severe cutaneous adverse reaction** (SJS/TEN, DRESS) to allopurinol | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Gout |
| **Risk Factors** | **HLA-B*58:01**, **CKD**, **Higher starting dose**, **Thiazide use** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Gout |
| **Prevention** | **Start low, go slow** (=100 mg/day in CKD), **Consider HLA-B*58:01 screening** | https://www.rheumatology.org/Portals/0/Files/Gout-Guideline-2020.pdf |
