# Rheumatology - Calcium Pyrophosphate Deposition (CPPD)

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Deposition of calcium pyrophosphate crystals** in articular cartilage and soft tissues | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Calcium-Pyrophosphate-Deposition-CPPD |
| **Terminology** | **Pseudogout** (acute inflammatory arthritis), **Chondrocalcinosis** (radiographic finding), **CPPD arthropathy** (chronic) | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Calcium-Pyrophosphate-Deposition-CPPD |
| **Crystal Identification** | **Positively birefringent rhomboid crystals** on compensated polarized microscopy | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Calcium-Pyrophosphate-Deposition-CPPD |
| **Demographics** | **Age >60**, **Women = Men** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Calcium-Pyrophosphate-Deposition-CPPD |

## Clinical Presentations
| Presentation | Description | URL |
|--------------|-------------|-----|
| **Acute Pseudogout** | **Acute monoarthritis** (often **knee**, wrist, shoulder); Similar to gout but **less painful**, **Longer duration** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Calcium-Pyrophosphate-Deposition-CPPD |
| **Chronic CPPD Arthropathy** | **Chronic polyarthritis** resembling **OA** (often with **atypical joint involvement**: MCPs, wrists, elbows, shoulders) | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Calcium-Pyrophosphate-Deposition-CPPD |
| **Crowned Dens Syndrome** | **Acute neck pain**, **Fever**, **? ESR/CRP**; **Calcification around odontoid process** on CT | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Calcium-Pyrophosphate-Deposition-CPPD |
