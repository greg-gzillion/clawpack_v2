# Rheumatology - Overview

## What is Rheumatology?
Rheumatology is the medical specialty dealing with the diagnosis and management of autoimmune, inflammatory, and degenerative conditions affecting the joints, muscles, bones, and connective tissues.

| Core Concept | Description | URL |
|--------------|-------------|-----|
| **Autoimmunity** | Immune response against self-antigens | https://www.niams.nih.gov/health-topics/autoimmune-diseases |
| **Autoinflammatory** | Innate immune system dysfunction (not autoantibody-mediated) | https://www.niaid.nih.gov/diseases-conditions/autoinflammatory-diseases |
| **Inflammatory Arthritis** | Synovial inflammation (RA, PsA, AS) | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions |
| **Degenerative Arthritis** | Cartilage loss (Osteoarthritis) | https://www.niams.nih.gov/health-topics/osteoarthritis |
| **Connective Tissue Disease** | Multisystem autoimmune disorders (SLE, SSc, Myositis) | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions |
| **Vasculitis** | Inflammation of blood vessels | https://www.niams.nih.gov/health-topics/vasculitis |
