# Rheumatology - Fibromyalgia

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Chronic widespread pain** with **fatigue**, **sleep disturbance**, **cognitive dysfunction**, and **somatic symptoms** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Fibromyalgia |
| **Pathophysiology** | **Central sensitization** (amplified pain processing in CNS) | https://www.niams.nih.gov/health-topics/fibromyalgia |
| **Prevalence** | **2-4%** of population; **Women > Men (3-6:1)** | https://www.cdc.gov/arthritis/types/fibromyalgia.htm |
| **Age of Onset** | Peak: **30-50 years** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Fibromyalgia |

## 2016 ACR Diagnostic Criteria
| Criterion | Scoring | URL |
|-----------|---------|-----|
| **Widespread Pain Index (WPI)** | **Number of painful regions** (0-19) | https://www.rheumatology.org/Portals/0/Files/2016-Fibromyalgia-Diagnostic-Criteria.pdf |
| **Symptom Severity Scale (SSS)** | **Fatigue + Waking unrefreshed + Cognitive symptoms** (0-3 each) + **Somatic symptoms** (0-3) | https://www.rheumatology.org/Portals/0/Files/2016-Fibromyalgia-Diagnostic-Criteria.pdf |
| **Diagnosis** | **WPI =7 + SSS =5** OR **WPI 4-6 + SSS =9**; **Symptoms =3 months** | https://www.rheumatology.org/Portals/0/Files/2016-Fibromyalgia-Diagnostic-Criteria.pdf |

## Clinical Features
| Feature | Description | URL |
|---------|-------------|-----|
| **Widespread Pain** | **Pain in all four quadrants** and **axial skeleton** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Fibromyalgia |
| **Fatigue** | **Persistent**, **Not relieved by rest** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Fibromyalgia |
| **Sleep Disturbance** | **Non-restorative sleep**, **Frequent awakenings** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Fibromyalgia |
| **Cognitive Dysfunction** | **"Fibro fog"** (difficulty concentrating, Memory problems) | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Fibromyalgia |
| **Associated Conditions** | **IBS**, **TMJ disorder**, **Chronic headache**, **Interstitial cystitis**, **Depression/Anxiety** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Fibromyalgia |
| **Physical Exam** | **Diffuse tenderness**, **No synovitis**, **Normal strength/reflexes** | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Fibromyalgia |

## Treatment of Fibromyalgia (EULAR 2016 / ACR 2020)
| Intervention | Recommendation | URL |
|--------------|----------------|-----|
| **Non-Pharmacologic** | **Graded aerobic exercise** (strongly recommended), **CBT**, **Patient education**, **Mindfulness**, **Tai Chi/Yoga** | https://www.rheumatology.org/Portals/0/Files/Fibromyalgia-Guideline-2020.pdf |
| **Pharmacologic** | **Duloxetine** (SNRI), **Milnacipran** (SNRI), **Pregabalin** (gabapentinoid), **Amitriptyline** (TCA), **Cyclobenzaprine** (muscle relaxant) | https://www.rheumatology.org/Portals/0/Files/Fibromyalgia-Guideline-2020.pdf |
| **NOT Recommended** | **Opioids**, **Corticosteroids**, **NSAIDs**, **Benzodiazepines**, **Strong opioids** | https://www.rheumatology.org/Portals/0/Files/Fibromyalgia-Guideline-2020.pdf |
