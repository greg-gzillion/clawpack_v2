# Rheumatology - Systemic Lupus Erythematosus (SLE)

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Multisystem autoimmune disease** with autoantibodies against nuclear antigens | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Lupus |
| **Pathophysiology** | **Loss of self-tolerance** ? **ANA**, **Anti-dsDNA**, **Anti-Smith** ? **Immune complex deposition** | https://www.niams.nih.gov/health-topics/lupus |
| **Prevalence** | **~0.1%** of population; **Women > Men (9:1)** ; **African American > Caucasian** | https://www.cdc.gov/lupus/index.htm |
| **Age of Onset** | Peak: **15-45 years** | https://www.lupus.org/resources/what-is-lupus |

## Clinical Manifestations of SLE
| System | Manifestations | URL |
|--------|----------------|-----|
| **Constitutional** | **Fatigue**, **Fever**, **Weight loss** | https://www.lupus.org/resources/common-symptoms-of-lupus |
| **Mucocutaneous** | **Malar rash** (butterfly), **Discoid rash**, **Photosensitivity**, **Oral/nasal ulcers**, **Alopecia** | https://www.lupus.org/resources/common-symptoms-of-lupus |
| **Musculoskeletal** | **Non-erosive polyarthritis**, **Jaccoud arthropathy**, **Myalgias** | https://www.lupus.org/resources/common-symptoms-of-lupus |
| **Renal** | **Lupus nephritis** (hematuria, Proteinuria, ? Cr) | https://www.lupus.org/resources/lupus-nephritis |
| **Neuropsychiatric** | **Cognitive dysfunction**, **Seizures**, **Psychosis**, **Stroke**, **Peripheral neuropathy** | https://www.lupus.org/resources/neuropsychiatric-lupus |
| **Hematologic** | **Anemia** (AIHA, ACD), **Leukopenia**, **Thrombocytopenia** | https://www.lupus.org/resources/common-symptoms-of-lupus |
| **Cardiopulmonary** | **Pleuritis**, **Pericarditis**, **Pulmonary hypertension**, **Libman-Sacks endocarditis** | https://www.lupus.org/resources/common-symptoms-of-lupus |
| **Immunologic** | **ANA** (>95%), **Anti-dsDNA** (60-70%), **Anti-Smith** (30%), **? C3/C4** | https://www.lupus.org/resources/lab-tests-for-lupus |

## Treatment of SLE (EULAR 2019 / ACR 2020)
| Manifestation | Recommendation | URL |
|---------------|----------------|-----|
| **All Patients** | **Hydroxychloroquine** (unless contraindicated) | https://www.rheumatology.org/Portals/0/Files/SLE-Guideline-2020.pdf |
| **Mild (Skin/Joint)** | **NSAIDs**, **Topical CS**, **Hydroxychloroquine**, **Methotrexate** or **Azathioprine** | https://www.rheumatology.org/Portals/0/Files/SLE-Guideline-2020.pdf |
| **Moderate** | **Corticosteroids** (lowest dose, shortest duration), **Methotrexate**, **Azathioprine**, **Mycophenolate**, **Belimumab** | https://www.rheumatology.org/Portals/0/Files/SLE-Guideline-2020.pdf |
| **Severe (Major Organ)** | **High-dose CS**, **Cyclophosphamide** (induction), **Mycophenolate** (maintenance), **Rituximab**, **Belimumab**, **Voclosporin** (LN) | https://www.rheumatology.org/Portals/0/Files/SLE-Guideline-2020.pdf |

## Lupus Nephritis (ISN/RPS Classification)
| Class | Description | Treatment | URL |
|-------|-------------|-----------|-----|
| **I** | **Minimal mesangial** | **Treat extrarenal SLE** | https://www.kidney.org/atoz/content/lupus-nephritis |
| **II** | **Mesangial proliferative** | Treat extrarenal SLE | https://www.kidney.org/atoz/content/lupus-nephritis |
| **III** | **Focal proliferative** (<50% glomeruli) | **Induction**: CS + MMF or CYC; **Maintenance**: MMF or AZA | https://www.kidney.org/atoz/content/lupus-nephritis |
| **IV** | **Diffuse proliferative** (=50% glomeruli) | Same as Class III | https://www.kidney.org/atoz/content/lupus-nephritis |
| **V** | **Membranous** | CS + MMF or CNI (tacrolimus) | https://www.kidney.org/atoz/content/lupus-nephritis |
| **VI** | **Advanced sclerosing** | **Renal replacement therapy** | https://www.kidney.org/atoz/content/lupus-nephritis |
