# Genetics - Fundamentals

## DNA Structure and Function
| Component | Description |
|-----------|-------------|
| Nucleotides | Adenine (A), Thymine (T), Cytosine (C), Guanine (G) |
| Base Pairing | A-T (2 hydrogen bonds), C-G (3 hydrogen bonds) |
| Double Helix | Antiparallel strands (5' -> 3' and 3' -> 5') |
| Chromatin | DNA wrapped around histone proteins |
| Euchromatin | Loosely packed, Transcriptionally active |
| Heterochromatin | Tightly packed, Transcriptionally silent |

## The Human Genome
| Feature | Value |
|---------|-------|
| Total base pairs | ~3 billion |
| Protein-coding genes | ~20,000-25,000 |
| Average gene size | ~10,000-15,000 base pairs |
| Largest gene | DMD (Dystrophin) - 2.4 million base pairs |
| Chromosomes | 22 autosomes + X + Y |
| Mitochondrial DNA | 16,569 base pairs, 37 genes |

## Cell Cycle and Mitosis
| Phase | Description |
|-------|-------------|
| G1 | Cell growth, Protein synthesis |
| S | **DNA replication** |
| G2 | Preparation for mitosis |
| M | **Mitosis** (Prophase, Metaphase, Anaphase, Telophase) |
| G0 | Quiescent phase |

## Meiosis
| Stage | Key Events |
|-------|------------|
| Meiosis I | **Reductional division** (2n -> n); Homologous chromosomes separate |
| Prophase I | **Crossing over** (recombination) occurs |
| Metaphase I | Homologous pairs align at metaphase plate |
| Anaphase I | Homologous chromosomes separate |
| Meiosis II | **Equational division**; Sister chromatids separate |
| Result | 4 haploid gametes |

## Types of Mutations
| Mutation Type | Mechanism | Example |
|---------------|-----------|---------|
| Missense | Amino acid substitution | Sickle cell disease (HbS - Glu6Val) |
| Nonsense | Premature stop codon | DMD (nonsense mutations) |
| Frameshift | Insertion/deletion not multiple of 3 | CFTR (?F508) |
| In-frame | Insertion/deletion multiple of 3 | CFTR (some mutations) |
| Splice Site | Altered RNA splicing | Beta-thalassemia |
| Promoter | Altered gene expression | Hereditary persistence of fetal hemoglobin |
| Trinucleotide Repeat | Expanded repeats | Huntington disease (CAG), Fragile X (CGG) |

## Variant Nomenclature (HGVS)
| Type | Example |
|------|---------|
| Substitution | c.123A>G (coding DNA), p.Lys41Glu (protein) |
| Deletion | c.123_125del (coding), p.Lys41del (protein) |
| Duplication | c.123_125dup |
| Insertion | c.123_124insAGC |
| Intronic | c.123+1G>A (splice donor) |

## Hardy-Weinberg Equilibrium
| Genotype | Frequency |
|----------|-----------|
| p2 + 2pq + q2 = 1 | p = frequency of dominant allele, q = frequency of recessive allele |
| Assumptions | Large population, Random mating, No mutation, No migration, No selection |
| Carrier Frequency | 2pq (for rare recessive disorders, ~2q) |
