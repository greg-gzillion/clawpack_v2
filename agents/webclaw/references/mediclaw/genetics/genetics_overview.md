# Genetics - Overview

## What is Medical Genetics?
Medical Genetics is the specialty of medicine that involves the diagnosis and management of hereditary disorders. It encompasses the study of genes, genetic variation, and heredity in the context of human health and disease.

## Branches of Medical Genetics
| Branch | Focus |
|--------|-------|
| Clinical Genetics | Diagnosis and management of genetic disorders |
| Cytogenetics | Chromosomal abnormalities |
| Molecular Genetics | DNA/RNA level mutations |
| Biochemical Genetics | Inborn errors of metabolism |
| Cancer Genetics | Hereditary cancer syndromes |
| Pharmacogenomics | Genetic influence on drug response |
| Prenatal Genetics | Genetic disorders in pregnancy |
| Genetic Counseling | Risk assessment and patient education |

## Genetic Material Organization
| Level | Description |
|-------|-------------|
| Genome | Complete set of DNA (3 billion base pairs in humans) |
| Chromosome | Condensed DNA structure (46 total: 22 autosomal pairs + sex chromosomes) |
| Gene | Functional unit of heredity (~20,000-25,000 protein-coding genes) |
| Exon | Coding region of gene (~1.5% of genome) |
| Intron | Non-coding region (removed during splicing) |
| Codon | Three-nucleotide sequence coding for amino acid |

## Central Dogma of Molecular Biology
| Process | Description | Location |
|---------|-------------|----------|
| Replication | DNA -> DNA | Nucleus |
| Transcription | DNA -> RNA | Nucleus |
| Translation | RNA -> Protein | Ribosome (cytoplasm) |

## Types of Genetic Variation
| Type | Size | Examples |
|------|------|----------|
| Single Nucleotide Variant (SNV) | 1 base pair | Missense, Nonsense, Splice site |
| Insertion/Deletion (Indel) | 1-1000 base pairs | Frameshift, In-frame |
| Copy Number Variant (CNV) | >1000 base pairs | Deletions, Duplications |
| Structural Variant | Large scale | Translocations, Inversions |
| Aneuploidy | Whole chromosome | Trisomy, Monosomy |
