# Genetics - Cancer Genetics

## Hallmarks of Cancer
| Hallmark | Description |
|----------|-------------|
| Sustaining proliferative signaling | Oncogene activation |
| Evading growth suppressors | Tumor suppressor inactivation |
| Resisting cell death | Anti-apoptotic mechanisms |
| Enabling replicative immortality | Telomerase activation |
| Inducing angiogenesis | VEGF signaling |
| Activating invasion and metastasis | EMT, Matrix degradation |
| Genome instability and mutation | DNA repair defects |
| Tumor-promoting inflammation | Immune evasion |
| Deregulating cellular energetics | Warburg effect |
| Avoiding immune destruction | PD-L1 expression |

## Oncogenes vs Tumor Suppressor Genes
| Feature | Oncogenes | Tumor Suppressor Genes |
|---------|-----------|----------------------|
| Normal Function | Promote cell growth | Inhibit cell growth |
| Mutation Type | **Gain-of-function** | **Loss-of-function** |
| Inheritance | Somatic (sporadic cancer) | Can be germline (hereditary cancer) |
| "Two-Hit" Model | One hit sufficient | **Two hits required** (Knudson) |
| Examples | **RAS, MYC, HER2, EGFR, BRAF** | **TP53, RB1, BRCA1, APC, PTEN** |

## DNA Repair Pathways
| Pathway | Genes | Associated Syndromes | PARP Inhibitor Sensitive |
|---------|-------|----------------------|--------------------------|
| Homologous Recombination (HR) | **BRCA1, BRCA2, PALB2, RAD51C/D** | HBOC, Fanconi anemia | **Yes** |
| Mismatch Repair (MMR) | **MLH1, MSH2, MSH6, PMS2, EPCAM** | Lynch syndrome | Limited |
| Nucleotide Excision Repair (NER) | XPA-XPG, ERCC | Xeroderma pigmentosum, Cockayne | No |
| Base Excision Repair (BER) | MUTYH, NTHL1 | MAP, NTHL1-associated polyposis | No |
| Fanconi Anemia Pathway | FANC genes | Fanconi anemia | Yes (some) |

## Hereditary Cancer Syndromes
| Syndrome | Gene(s) | Key Cancers | Screening/Management |
|----------|---------|-------------|----------------------|
| **Hereditary Breast and Ovarian Cancer (HBOC)** | BRCA1, BRCA2, PALB2 | Breast, Ovarian, Pancreatic, Prostate | MRI/mammography (age 25-30), RRSO (age 35-40), Pancreatic screening |
| **Lynch Syndrome** | MLH1, MSH2, MSH6, PMS2, EPCAM | Colorectal, Endometrial, Ovarian, Gastric, Urothelial | **Colonoscopy q1-2 years** (age 20-25), Endometrial biopsy, Aspirin |
| **Familial Adenomatous Polyposis (FAP)** | APC | Colorectal (>100 adenomas), Duodenal, Desmoid | Sigmoidoscopy annually (age 10-12), **Prophylactic colectomy** |
| **Li-Fraumeni Syndrome** | TP53 | Breast, Sarcoma, Brain, Adrenocortical, Leukemia | **Whole-body MRI annually**, Breast MRI, Avoid radiation |
| **Cowden Syndrome** | PTEN | Breast, Thyroid, Endometrial, Renal | Breast MRI, Thyroid US, Endometrial biopsy, Renal imaging |
| **Peutz-Jeghers Syndrome** | STK11 | GI, Breast, Pancreatic, Ovarian, Cervical, Testicular | EGD/Colonoscopy q2-3y, Breast MRI, Pancreatic imaging |
| **von Hippel-Lindau** | VHL | RCC, Hemangioblastoma, Pheochromocytoma | Annual MRI brain/spine, Abdominal imaging, Ophthalmology |
| **Multiple Endocrine Neoplasia Type 1 (MEN1)** | MEN1 | Parathyroid, Pancreatic NET, Pituitary | Annual biochemical screening, Imaging |
| **Multiple Endocrine Neoplasia Type 2 (MEN2)** | RET | Medullary thyroid cancer, Pheochromocytoma, Parathyroid | **Prophylactic thyroidectomy** in childhood |
| **Hereditary Diffuse Gastric Cancer** | CDH1 | Diffuse gastric cancer, Lobular breast cancer | **Prophylactic gastrectomy** (age 20-30), Breast MRI |

## Somatic Genetic Testing in Cancer
| Test | Purpose | Examples |
|------|---------|----------|
| Tumor Sequencing | Identify actionable mutations | EGFR, BRAF, KRAS, HER2, NTRK |
| MSI/MMR Testing | Immunotherapy eligibility | MSI-H/dMMR -> **Pembrolizumab** |
| TMB (Tumor Mutational Burden) | Immunotherapy response prediction | High TMB -> Immunotherapy |
| ctDNA (Liquid Biopsy) | Monitor response, Detect resistance | EGFR T790M, RAS mutations |
| Cytogenetics/FISH | Translocations, Amplifications | BCR-ABL, HER2, ALK |
| HRD Testing | PARP inhibitor eligibility | BRCA1/2, Genomic instability scores |
