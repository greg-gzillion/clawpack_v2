# Genetics - Single Gene Disorders

## Autosomal Dominant Disorders
| Disorder | Gene | Key Features |
|----------|------|--------------|
| Huntington Disease | **HTT** (CAG repeat) | Progressive chorea, Dementia, Psychiatric symptoms, Age 30-50 onset |
| Marfan Syndrome | **FBN1** | Tall stature, Arachnodactyly, Lens dislocation, **Aortic root dilation** |
| Neurofibromatosis Type 1 | **NF1** | Café-au-lait spots, Neurofibromas, Lisch nodules, Optic glioma |
| Neurofibromatosis Type 2 | **NF2** | Bilateral vestibular schwannomas, Meningiomas, Juvenile cataracts |
| Tuberous Sclerosis | **TSC1, TSC2** | Seizures, ID, Angiofibromas, Cardiac rhabdomyomas, Renal angiomyolipomas |
| Familial Adenomatous Polyposis (FAP) | **APC** | >100 colonic adenomas, CRC by age 40, CHRPE |
| von Hippel-Lindau | **VHL** | Hemangioblastomas (retina, cerebellum), RCC, Pheochromocytoma |
| Achondroplasia | **FGFR3** | Short-limbed dwarfism, Frontal bossing, Foramen magnum stenosis |
| Osteogenesis Imperfecta (Type I) | **COL1A1, COL1A2** | Blue sclerae, Bone fragility, Hearing loss, Dentinogenesis imperfecta |
| Myotonic Dystrophy Type 1 | **DMPK** (CTG repeat) | Myotonia, Muscle weakness, Cataracts, Cardiac conduction defects |
| Familial Hypercholesterolemia | **LDLR, APOB, PCSK9** | Severe hypercholesterolemia, Xanthomas, Premature CAD |
| Acute Intermittent Porphyria | **HMBS** | Abdominal pain, Neuropsychiatric symptoms, Photosensitivity |

## Autosomal Recessive Disorders
| Disorder | Gene | Key Features | Carrier Frequency |
|----------|------|--------------|-------------------|
| Cystic Fibrosis | **CFTR** | Chronic lung disease, Pancreatic insufficiency, Male infertility | 1:25 (Caucasian) |
| Sickle Cell Disease | **HBB** (Glu6Val) | Hemolytic anemia, Vaso-occlusive crises, Autosplenectomy | 1:12 (African American) |
| Tay-Sachs Disease | **HEXA** | Progressive neurodegeneration, Cherry-red spot, Death by age 4 | 1:30 (Ashkenazi Jewish) |
| Phenylketonuria (PKU) | **PAH** | Intellectual disability, Musty odor, Eczema, Fair skin | 1:50 |
| Spinal Muscular Atrophy (SMA) | **SMN1** | Progressive proximal muscle weakness, Areflexia | 1:50 |
| Gaucher Disease | **GBA** | Hepatosplenomegaly, Bone pain, Pancytopenia | 1:15 (Ashkenazi Jewish) |
| Alpha-1 Antitrypsin Deficiency | **SERPINA1** | Emphysema, Liver cirrhosis, Panniculitis | 1:25 |
| Wilson Disease | **ATP7B** | Liver disease, Neuropsychiatric symptoms, Kayser-Fleischer rings | 1:90 |
| Hemochromatosis | **HFE** | Iron overload, Cirrhosis, Diabetes, Bronze skin | 1:10 (Northern European) |
| Congenital Adrenal Hyperplasia (21-OH) | **CYP21A2** | Salt-wasting, Virilization, Hypotension | 1:60 |
| Friedreich Ataxia | **FXN** (GAA repeat) | Progressive ataxia, Cardiomyopathy, Diabetes | 1:100 |
| Canavan Disease | **ASPA** | Macrocephaly, Hypotonia, Spongy degeneration | 1:40 (Ashkenazi Jewish) |

## X-Linked Recessive Disorders
| Disorder | Gene | Key Features |
|----------|------|--------------|
| Duchenne Muscular Dystrophy | **DMD** | Progressive proximal weakness, **Gowers sign**, Calf pseudohypertrophy, Cardiomyopathy |
| Becker Muscular Dystrophy | **DMD** | Milder, Later onset than Duchenne |
| Hemophilia A | **F8** | Factor VIII deficiency, Hemarthroses, Easy bruising |
| Hemophilia B | **F9** | Factor IX deficiency, Similar to Hemophilia A |
| Red-Green Color Blindness | **OPN1LW, OPN1MW** | Inability to distinguish red and green |
| G6PD Deficiency | **G6PD** | Hemolytic anemia with oxidative stress (fava beans, sulfa drugs) |
| X-Linked Agammaglobulinemia | **BTK** | Recurrent bacterial infections, Absent B cells |
| Chronic Granulomatous Disease (X-linked) | **CYBB** | Recurrent catalase-positive infections |
| Fabry Disease | **GLA** | Acroparesthesias, Angiokeratomas, Renal failure, Cardiac hypertrophy |
| Hunter Syndrome (MPS II) | **IDS** | Coarse facies, ID, Hepatosplenomegaly, Skeletal abnormalities |
| Lesch-Nyhan Syndrome | **HPRT1** | Self-mutilation, Gout, ID, Choreoathetosis |
| Ornithine Transcarbamylase Deficiency | **OTC** | Hyperammonemia, Protein aversion, Seizures, Coma |

## X-Linked Dominant Disorders
| Disorder | Gene | Key Features |
|----------|------|--------------|
| Rett Syndrome | **MECP2** | Normal development -> Regression, Hand-wringing, Seizures, **Lethal in males** |
| Fragile X Syndrome | **FMR1** (CGG repeat) | ID, Macroorchidism, Long face, Large ears, Autism |
| Incontinentia Pigmenti | **IKBKG** | Skin lesions (Blaschko lines), Dental anomalies, Seizures, **Lethal in males** |
| Hypophosphatemic Rickets | **PHEX** | Short stature, Bowing of legs, Dental abscesses |
| Alport Syndrome | **COL4A5** | Hematuria, Sensorineural hearing loss, Ocular abnormalities |
