# Genetics - Mitochondrial Disorders

## Mitochondrial Genetics
| Feature | Description |
|---------|-------------|
| mtDNA Size | 16,569 base pairs |
| Genes | 37 (13 proteins, 22 tRNAs, 2 rRNAs) |
| Inheritance | **Maternal inheritance** |
| Heteroplasmy | Mix of mutant and wild-type mtDNA |
| Threshold Effect | Phenotype depends on % mutant mtDNA |
| Mitotic Segregation | Proportion of mutant mtDNA can change during cell division |
| Bottleneck | Reduction in mtDNA copy number during oogenesis |

## Mitochondrial Inheritance Pattern
| Feature | Description |
|---------|-------------|
| Affected Mother | All offspring at risk (both males and females) |
| Affected Father | **No transmission** to offspring |
| Variable Expression | Due to heteroplasmy and tissue distribution |
| Age of Onset | Often later onset (accumulation of mutations) |

## Common Mitochondrial Disorders
| Disorder | Gene/Mutation | Key Features |
|----------|---------------|--------------|
| Leber Hereditary Optic Neuropathy (LHON) | **MT-ND1, MT-ND4, MT-ND6** | Subacute bilateral vision loss, Young adult males |
| MELAS | **MT-TL1** (m.3243A>G) | **M**itochondrial **E**ncephalomyopathy, **L**actic **A**cidosis, **S**troke-like episodes |
| MERRF | **MT-TK** (m.8344A>G) | **M**yoclonic **E**pilepsy with **R**agged **R**ed **F**ibers |
| NARP | **MT-ATP6** | **N**europathy, **A**taxia, **R**etinitis **P**igmentosa |
| Leigh Syndrome | Multiple mtDNA and nDNA genes | Progressive neurodegeneration, Bilateral basal ganglia lesions |
| Kearns-Sayre Syndrome | mtDNA deletions | Progressive external ophthalmoplegia (PEO), Retinitis pigmentosa, Heart block |
| Pearson Syndrome | mtDNA deletions | Sideroblastic anemia, Pancreatic insufficiency, Pancytopenia |
| Chronic Progressive External Ophthalmoplegia (CPEO) | mtDNA deletions | Ptosis, Ophthalmoplegia |
| Mitochondrial Diabetes and Deafness | **MT-TL1** (m.3243A>G) | Diabetes mellitus, Sensorineural hearing loss |

## Nuclear Genes Affecting Mitochondrial Function
| Disorder | Gene | Inheritance | Key Features |
|----------|------|-------------|--------------|
| Alpers Syndrome | **POLG** | AR | Developmental regression, Seizures, Liver failure, **Valproate contraindicated** |
| Mitochondrial DNA Depletion Syndrome | **TK2, DGUOK, SUCLA2, RRM2B** | AR | Tissue-specific mtDNA depletion (myopathic, hepatocerebral) |
| Leigh Syndrome (Nuclear) | **SURF1, LRPPRC** | AR, X-linked | Same as mtDNA Leigh syndrome |
| Coenzyme Q10 Deficiency | **COQ2, COQ8A** | AR | Encephalopathy, Ataxia, Nephrotic syndrome |
| Barth Syndrome | **TAZ** | X-linked | Cardiomyopathy, Neutropenia, Growth delay, 3-methylglutaconic aciduria |

## Mitochondrial Disorder - Clinical Features
| System | Manifestations |
|--------|----------------|
| CNS | Seizures, Stroke-like episodes, Ataxia, Dementia, Migraine |
| Muscle | Weakness, Exercise intolerance, Ptosis, Ophthalmoplegia |
| Heart | Cardiomyopathy, Conduction defects |
| Eye | Optic atrophy, Retinitis pigmentosa, Cataracts |
| Ear | Sensorineural hearing loss |
| Endocrine | Diabetes mellitus, Short stature, Hypoparathyroidism |
| GI | Dysmotility, Pseudo-obstruction, Pancreatic insufficiency |
| Renal | Fanconi syndrome, Focal segmental glomerulosclerosis |
| Hematologic | Sideroblastic anemia, Pancytopenia |

## Diagnostic Approach to Mitochondrial Disorders
| Step | Test |
|------|------|
| 1. Clinical Suspicion | Multi-system involvement, Maternal inheritance pattern |
| 2. Initial Labs | Lactate, Pyruvate, CK, Amino acids, Organic acids |
| 3. Muscle Biopsy | **Ragged red fibers** (Gomori trichrome), **COX-negative fibers**, Electron microscopy (abnormal mitochondria) |
| 4. Genetic Testing | mtDNA sequencing, mtDNA deletion/duplication analysis, Nuclear gene panel |
| 5. Functional Studies | Respiratory chain enzyme analysis |

## Management of Mitochondrial Disorders
| Component | Recommendations |
|-----------|-----------------|
| Supplements | **Coenzyme Q10**, L-carnitine, Riboflavin, Thiamine, Vitamin C, Vitamin E, Lipoic acid |
| Avoid | **Valproic acid** (hepatotoxicity), **Aminoglycosides** (hearing loss), Fasting, Strenuous exercise |
| Metabolic Stress | Avoid triggers (infection, surgery, fasting) |
| Monitoring | Cardiac (ECG, Echo), Ophthalmology, Audiology, Endocrine |
| Genetic Counseling | Maternal inheritance for mtDNA mutations; Autosomal/X-linked for nuclear mutations |
