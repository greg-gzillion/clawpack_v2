# Plastic Surgery - Breast Reduction (Reduction Mammaplasty)

## Indications for Breast Reduction
| Indication | Description | URL |
|------------|-------------|-----|
| **Macromastia Symptoms** | **Neck pain**, **Back pain**, **Shoulder grooving**, **Inframammary fold rash**, **Headaches**, **Upper extremity paresthesias** | https://www.plasticsurgery.org/cosmetic-procedures/breast-reduction |
| **Functional Impairment** | **Difficulty with exercise**, **Poor posture**, **Difficulty finding clothing** | https://www.plasticsurgery.org/cosmetic-procedures/breast-reduction |
| **Insurance Criteria** | **Estimated resection weight** (often =500g per breast), **Documented symptoms**, **Failed conservative measures** | https://www.plasticsurgery.org/cosmetic-procedures/breast-reduction |

## Breast Reduction Techniques
| Technique | Pedicle | Indications | URL |
|-----------|---------|-------------|-----|
| **Wise Pattern (Inverted-T)** | **Inferior**, **Superomedial**, **Medial** | **Most common**, **Large reductions** | https://www.plasticsurgery.org/cosmetic-procedures/breast-reduction |
| **Vertical (Lejour)** | **Superior**, **Superomedial** | **Moderate reductions**, **Better long-term shape** | https://www.plasticsurgery.org/cosmetic-procedures/breast-reduction |
| **Periareolar (Benelli)** | **Inferior** | **Small reductions**, **Mild ptosis** | https://www.plasticsurgery.org/cosmetic-procedures/breast-reduction |
| **Liposuction Only** | N/A | **Fatty breasts**, **Good skin elasticity**, **No ptosis** | https://www.plasticsurgery.org/cosmetic-procedures/breast-reduction |

## Breast Reduction - Complications
| Complication | Incidence | Management | URL |
|--------------|-----------|------------|-----|
| **Delayed Wound Healing** | 10-15% | **Local wound care**, **Secondary intention** | https://www.plasticsurgery.org/cosmetic-procedures/breast-reduction |
| **Hematoma** | 1-3% | **Evacuation**, **Hemostasis** | https://www.plasticsurgery.org/cosmetic-procedures/breast-reduction |
| **Seroma** | 5-10% | **Aspiration**, **Compression** | https://www.plasticsurgery.org/cosmetic-procedures/breast-reduction |
| **Nipple Necrosis** | <1% (pedicled), 5-10% (free graft) | **Observation**, **Debridement**, **Nipple reconstruction** | https://www.plasticsurgery.org/cosmetic-procedures/breast-reduction |
| **Fat Necrosis** | 5-10% | **Observation**, **Excision** if symptomatic | https://www.plasticsurgery.org/cosmetic-procedures/breast-reduction |
| **Altered Nipple Sensation** | Common | **Reassurance** | https://www.plasticsurgery.org/cosmetic-procedures/breast-reduction |
| **Breastfeeding Inability** | Variable | **Preoperative counseling** | https://www.plasticsurgery.org/cosmetic-procedures/breast-reduction |
