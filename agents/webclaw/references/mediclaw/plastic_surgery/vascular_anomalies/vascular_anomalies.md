## Infantile Hemangioma
| Component | Description | URL |
|-----------|-------------|-----|
| **Natural History** | **Appears in first weeks**, **Proliferates** (first year), **Involutes** (years) | https://www.plasticsurgery.org/reconstructive-procedures/infantile-hemangioma |
| **Indications for Treatment** | **Vision compromise**, **Airway compromise**, **Ulceration**, **Disfigurement**, **Hepatic** (multiple cutaneous) | https://www.plasticsurgery.org/reconstructive-procedures/infantile-hemangioma |
| **Treatment** | **Oral propranolol** (first-line), **Topical timolol** (superficial), **Corticosteroids**, **Surgery** | https://www.plasticsurgery.org/reconstructive-procedures/infantile-hemangioma |

| Type | Characteristics | Treatment | URL |
