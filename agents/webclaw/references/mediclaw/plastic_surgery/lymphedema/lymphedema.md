# Plastic Surgery - Lymphedema

## Lymphedema - Classification
| Type | Etiology | URL |
|------|----------|-----|
| **Primary** | **Congenital** (Milroy disease), **Lymphedema praecox** (Meige disease), **Lymphedema tarda** | https://www.cdc.gov/cancer/lymphedema/index.htm |
| **Secondary** | **Cancer treatment** (lymph node dissection, Radiation), **Trauma**, **Infection**, **Filariasis** | https://www.cdc.gov/cancer/lymphedema/index.htm |

## Lymphedema - Staging (International Society of Lymphology)
| Stage | Description | URL |
|-------|-------------|-----|
| **Stage 0 (Latent)** | **No visible edema**, Abnormal lymphoscintigraphy | https://www.cdc.gov/cancer/lymphedema/index.htm |
| **Stage I (Reversible)** | **Pitting edema**, **Elevates** | https://www.cdc.gov/cancer/lymphedema/index.htm |
| **Stage II (Irreversible)** | **Non-pitting**, **Fibrosis**, **Does not elevate** | https://www.cdc.gov/cancer/lymphedema/index.htm |
| **Stage III (Elephantiasis)** | **Severe**, **Acanthosis**, **Papillomatosis**, **Fungal infections** | https://www.cdc.gov/cancer/lymphedema/index.htm |

## Lymphedema - Surgical Management
| Procedure | Description | Indications | URL |
|-----------|-------------|-------------|-----|
| **Lymphaticovenous Anastomosis (LVA)** | **Supermicrosurgical anastomosis** of lymphatics to venules | **Early-stage** (Stage I-II) | https://www.plasticsurgery.org/reconstructive-procedures/lymphedema-surgery |
| **Vascularized Lymph Node Transfer (VLNT)** | **Transfer of lymph nodes** (groin, Supraclavicular, Submental) | **Moderate-stage** (Stage II-III) | https://www.plasticsurgery.org/reconstructive-procedures/lymphedema-surgery |
| **Suction-Assisted Lipectomy (SAL)** | **Liposuction** of fibrofatty tissue | **Late-stage** (Stage III) | https://www.plasticsurgery.org/reconstructive-procedures/lymphedema-surgery |
| **Charles Procedure** | **Radical excision** of skin and subcutaneous tissue + **Skin grafting** | **Severe elephantiasis** | https://www.plasticsurgery.org/reconstructive-procedures/lymphedema-surgery |
