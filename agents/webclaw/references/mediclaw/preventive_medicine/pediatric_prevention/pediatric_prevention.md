# Preventive Medicine - Pediatric Prevention

## Bright Futures/AAP Periodicity Schedule
| Age | Key Components | URL |
|-----|----------------|-----|
| **Newborn** | **Newborn screening**, **Hearing screen**, **Hepatitis B vaccine**, **Breastfeeding support** | https://www.aap.org/en/practice-management/bright-futures |
| **2 months** | **Growth**, **Development**, **Immunizations** (DTaP, Hib, IPV, PCV13, Rotavirus, HepB) | https://www.aap.org/en/practice-management/bright-futures |
| **4 months** | Growth, Development, Immunizations (DTaP, Hib, IPV, PCV13, Rotavirus) | https://www.aap.org/en/practice-management/bright-futures |
| **6 months** | Growth, Development, Immunizations (DTaP, Hib, PCV13, Rotavirus, **Influenza**, COVID-19) | https://www.aap.org/en/practice-management/bright-futures |
| **9 months** | Growth, Development, **Iron/Lead screening** (if risk) | https://www.aap.org/en/practice-management/bright-futures |
| **12 months** | Growth, Development, Immunizations (MMR, Varicella, HepA, Hib, PCV13), **Anemia screening**, **Lead screening** | https://www.aap.org/en/practice-management/bright-futures |
| **15 months** | Growth, Development, Immunizations (DTaP) | https://www.aap.org/en/practice-management/bright-futures |
| **18 months** | Growth, Development, Immunizations (HepA), **Autism screening** (M-CHAT-R/F) | https://www.aap.org/en/practice-management/bright-futures |
| **24 months** | Growth, Development, **Autism screening**, **Lead screening** | https://www.aap.org/en/practice-management/bright-futures |
| **30 months** | Growth, Development | https://www.aap.org/en/practice-management/bright-futures |
| **3 years** | Growth, Development, **Vision/Hearing**, **Blood pressure** | https://www.aap.org/en/practice-management/bright-futures |
| **4-6 years** | Growth, Development, Vision/Hearing, Immunizations (**DTaP**, **IPV**, **MMR**, **Varicella**) | https://www.aap.org/en/practice-management/bright-futures |
| **7-10 years** | Growth, Development, Vision/Hearing, BP | https://www.aap.org/en/practice-management/bright-futures |
| **11-12 years** | Growth, Development, Vision/Hearing, BP, Immunizations (**Tdap**, **MenACWY**, **HPV**), **Lipid screening** (once 9-11) | https://www.aap.org/en/practice-management/bright-futures |
| **13-21 years** | **Annually**: Growth, Development, Vision/Hearing, BP, Immunizations, **Depression/Substance use screening**, **Lipid screening** (once 17-21) | https://www.aap.org/en/practice-management/bright-futures |
