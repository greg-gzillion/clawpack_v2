# Preventive Medicine - Metabolic Screening

## Diabetes and Prediabetes Screening
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **USPSTF Grade** | **B** (Adults 35-70 with overweight/obesity) | https://www.uspreventiveservicestaskforce.org/uspstf/recommendation/screening-for-prediabetes-and-type-2-diabetes |
| **Screening Tests** | **Fasting glucose**, **HbA1c**, **2-hour OGTT** | https://www.uspreventiveservicestaskforce.org/uspstf/recommendation/screening-for-prediabetes-and-type-2-diabetes |
| **Frequency** | **Every 3 years** (more often if risk factors) | https://www.uspreventiveservicestaskforce.org/uspstf/recommendation/screening-for-prediabetes-and-type-2-diabetes |
| **Diagnostic Criteria** | **Diabetes**: FPG =126, 2h OGTT =200, A1c =6.5%, Random =200 + symptoms | https://professional.diabetes.org/standards-of-care |
| **Prediabetes Criteria** | **FPG 100-125**, **2h OGTT 140-199**, **A1c 5.7-6.4%** | https://professional.diabetes.org/standards-of-care |

## Diabetes Prevention
| Intervention | Recommendation | URL |
|--------------|----------------|-----|
| **Lifestyle Intervention** | **=7% weight loss** + **=150 min/week moderate activity** | https://www.cdc.gov/diabetes-prevention/index.html |
| **National DPP** | **CDC-recognized Diabetes Prevention Program** (reduces progression by 58%) | https://www.cdc.gov/diabetes-prevention/index.html |
| **Metformin** | Consider for **prediabetes** (especially BMI =35, Age <60, Prior GDM) | https://professional.diabetes.org/standards-of-care |
