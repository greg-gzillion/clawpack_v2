# Preventive Medicine - Screening Principles

## WHO Principles of Screening (Wilson and Jungner, 1968)
| Principle | Description | URL |
|-----------|-------------|-----|
| **1** | The condition should be an **important health problem** | https://www.who.int/publications/i/item/WHO-PHP-34 |
| **2** | There should be an **accepted treatment** for patients with recognized disease | https://www.who.int/publications/i/item/WHO-PHP-34 |
| **3** | Facilities for **diagnosis and treatment** should be available | https://www.who.int/publications/i/item/WHO-PHP-34 |
| **4** | There should be a **recognizable latent or early symptomatic stage** | https://www.who.int/publications/i/item/WHO-PHP-34 |
| **5** | There should be a **suitable test or examination** | https://www.who.int/publications/i/item/WHO-PHP-34 |
| **6** | The test should be **acceptable to the population** | https://www.who.int/publications/i/item/WHO-PHP-34 |
| **7** | The **natural history** of the condition should be adequately understood | https://www.who.int/publications/i/item/WHO-PHP-34 |
| **8** | There should be an **agreed policy on whom to treat** | https://www.who.int/publications/i/item/WHO-PHP-34 |
| **9** | The **cost** of case-finding should be economically balanced | https://www.who.int/publications/i/item/WHO-PHP-34 |
| **10** | Case-finding should be a **continuing process**, not a "once and for all" project | https://www.who.int/publications/i/item/WHO-PHP-34 |

## Screening Test Characteristics
| Measure | Formula | Description | URL |
|---------|---------|-------------|-----|
| **Sensitivity** | **TP / (TP + FN)** | **Ability to detect disease** when present | https://www.cdc.gov/csels/dsepd/ss1978/lesson3/section1.html |
| **Specificity** | **TN / (TN + FP)** | **Ability to exclude disease** when absent | https://www.cdc.gov/csels/dsepd/ss1978/lesson3/section1.html |
| **Positive Predictive Value (PPV)** | **TP / (TP + FP)** | **Probability that positive test indicates disease** (depends on prevalence) | https://www.cdc.gov/csels/dsepd/ss1978/lesson3/section1.html |
| **Negative Predictive Value (NPV)** | **TN / (TN + FN)** | **Probability that negative test excludes disease** | https://www.cdc.gov/csels/dsepd/ss1978/lesson3/section1.html |
| **Likelihood Ratio Positive (LR+)** | **Sensitivity / (1 - Specificity)** | **How much to increase probability of disease** with positive test | https://www.cebm.ox.ac.uk/resources/ebm-tools/likelihood-ratios |
| **Likelihood Ratio Negative (LR-)** | **(1 - Sensitivity) / Specificity** | **How much to decrease probability of disease** with negative test | https://www.cebm.ox.ac.uk/resources/ebm-tools/likelihood-ratios |

## Lead-Time Bias and Length-Time Bias
| Bias | Definition | Impact | URL |
|------|------------|--------|-----|
| **Lead-Time Bias** | **Earlier detection** makes survival appear longer, even if death occurs at same time | **Overestimates survival benefit** of screening | https://www.cancer.gov/publications/dictionaries/cancer-terms/def/lead-time-bias |
| **Length-Time Bias** | Screening **disproportionately detects slower-growing tumors** (longer preclinical phase) | **Overestimates survival benefit**; Detected cancers appear less aggressive | https://www.cancer.gov/publications/dictionaries/cancer-terms/def/length-bias |
| **Overdiagnosis** | Detection of disease that **would never cause symptoms or death** | **Unnecessary treatment**, **Anxiety**, **Costs** | https://www.cancer.gov/publications/dictionaries/cancer-terms/def/overdiagnosis |

## Shared Decision Making
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Collaborative process where clinician and patient **make decisions together** based on best evidence and patient values | https://www.uspreventiveservicestaskforce.org/uspstf/shared-decision-making |
| **When to Use** | **Preference-sensitive decisions** (Grade C recommendations, Multiple reasonable options) | https://www.uspreventiveservicestaskforce.org/uspstf/shared-decision-making |
| **Key Elements** | **Discuss benefits/harms**, **Elicit patient values**, **Reach mutual decision** | https://www.uspreventiveservicestaskforce.org/uspstf/shared-decision-making |
| **Examples** | **Prostate cancer screening** (PSA), **Breast cancer screening** (age 40-49), **Lung cancer screening** | https://www.uspreventiveservicestaskforce.org/uspstf/shared-decision-making |
