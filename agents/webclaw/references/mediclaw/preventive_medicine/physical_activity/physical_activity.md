# Preventive Medicine - Physical Activity

## Physical Activity Guidelines for Americans (2nd Edition, 2018)
| Age Group | Aerobic Activity | Muscle Strengthening | URL |
|-----------|-----------------|---------------------|-----|
| **Children 6-17** | **=60 min/day** (mostly moderate-vigorous) | **=3 days/week** | https://health.gov/physical-activity-guidelines |
| **Adults 18-64** | **=150 min/week moderate** OR **=75 min/week vigorous** OR **equivalent combination** | **=2 days/week** | https://health.gov/physical-activity-guidelines |
| **Older Adults =65** | Same as adults + **Balance training** | =2 days/week | https://health.gov/physical-activity-guidelines |
| **Pregnancy/Postpartum** | =150 min/week moderate (if no contraindications) | =2 days/week | https://health.gov/physical-activity-guidelines |
| **Adults with Chronic Conditions** | **Be as active as able**; Follow adult guidelines if possible | As able | https://health.gov/physical-activity-guidelines |

## Exercise Prescription (FITT Principle)
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **Frequency** | **3-5 days/week** (aerobic), **2-3 days/week** (resistance) | https://www.acsm.org/read-research/trending-topics-resource-pages/physical-activity-guidelines |
| **Intensity** | **Moderate** (can talk but not sing) or **Vigorous** (cannot say more than few words) | https://www.acsm.org/read-research/trending-topics-resource-pages/physical-activity-guidelines |
| **Time** | **30-60 min/session** (moderate), **20-30 min/session** (vigorous) | https://www.acsm.org/read-research/trending-topics-resource-pages/physical-activity-guidelines |
| **Type** | **Aerobic** (walking, cycling, swimming), **Resistance** (weights, bands, body weight), **Flexibility** (stretching, yoga), **Balance** (tai chi, standing on one leg) | https://www.acsm.org/read-research/trending-topics-resource-pages/physical-activity-guidelines |
