# Integrative Medicine - Manual Therapies

## Overview
Manual therapies involve hands-on techniques to manipulate or mobilize soft tissues and joints to improve health and relieve pain.

| Therapy | Description | Evidence | URL |
|---------|-------------|----------|-----|
| **Chiropractic** | Focus on spine and nervous system relationship | **Strong**: Low back pain, Neck pain, Headache | https://www.nccih.nih.gov/health/chiropractic-in-depth |
| **Osteopathic Manipulative Treatment (OMT)** | Hands-on techniques to diagnose and treat | **Moderate**: Low back pain | https://osteopathic.org/what-is-osteopathic-medicine/osteopathic-manipulative-treatment/ |
| **Massage Therapy** | Manipulation of soft tissues | **Moderate**: Pain, Anxiety, Depression, Fibromyalgia | https://www.nccih.nih.gov/health/massage-therapy-what-you-need-to-know |
| **Physical Therapy** | Evidence-based rehabilitation | **Strong**: Musculoskeletal conditions, Post-op rehab | https://www.apta.org |
| **Craniosacral Therapy** | Gentle manipulation of skull and sacrum | **Limited** | https://www.nccih.nih.gov/health/craniosacral-therapy |

## Chiropractic
| Component | Description | URL |
|-----------|-------------|-----|
| **Spinal Manipulation** | High-velocity, low-amplitude (HVLA) thrust | https://www.nccih.nih.gov/health/spinal-manipulation-what-you-need-to-know |
| **Evidence for Low Back Pain** | **Strong**: As effective as conventional treatments | https://pubmed.ncbi.nlm.nih.gov/29198932/ |
| **Evidence for Neck Pain** | **Moderate-Strong**: Effective for acute and chronic neck pain | https://www.nccih.nih.gov/health/pain-considering-complementary-approaches-ebook |
| **Evidence for Headache** | **Moderate**: Effective for cervicogenic headache, migraine | https://www.nccih.nih.gov/health/headaches-and-complementary-health-approaches |
| **Safety** | Generally safe; Rare: Vertebral artery dissection (controversial association) | https://www.nccih.nih.gov/health/spinal-manipulation-what-you-need-to-know |

## Massage Therapy
| Type | Description | Evidence | URL |
|------|-------------|----------|-----|
