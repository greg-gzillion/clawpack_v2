# Travel Medicine - Pre-Travel Consultation

## Timing and Components
| Component | Description | URL |
|-----------|-------------|-----|
| **Ideal Timing** | **4-6 weeks before departure** (allows time for vaccine series) | https://wwwnc.cdc.gov/travel/page/traveler-information-center |
| **Itinerary Review** | **Destinations** (urban vs rural), **Duration**, **Season**, **Activities**, **Accommodations** | https://wwwnc.cdc.gov/travel/page/traveler-information-center |
| **Medical History** | **Chronic conditions**, **Medications**, **Allergies**, **Immunocompromise**, **Pregnancy**, **Prior vaccinations** | https://wwwnc.cdc.gov/travel/page/traveler-information-center |
| **Vaccinations** | **Routine** (update), **Required** (yellow fever, meningococcal), **Recommended** (based on risk) | https://wwwnc.cdc.gov/travel/page/vaccines |
| **Malaria Prophylaxis** | If traveling to **endemic area** | https://www.cdc.gov/malaria/travelers/index.html |
| **Traveler's Diarrhea** | **Prevention counseling** + **Self-treatment prescription** | https://wwwnc.cdc.gov/travel/page/travelers-diarrhea |
| **General Advice** | **Food/water safety**, **Insect precautions**, **Altitude illness**, **Motion sickness**, **Jet lag**, **Safety/security** | https://wwwnc.cdc.gov/travel/page/traveler-information-center |

## Risk Stratification by Travel Type
| Risk Level | Travel Type | URL |
|------------|-------------|-----|
| **Low** | **Resort/urban travel**, **Short duration**, **Good accommodations** | https://wwwnc.cdc.gov/travel/page/traveler-information-center |
| **Moderate** | **Rural travel**, **Longer duration**, **Basic accommodations** | https://wwwnc.cdc.gov/travel/page/traveler-information-center |
| **High** | **Backpacking**, **Adventure travel**, **Humanitarian work**, **Visiting friends and relatives (VFR)** , **Long-term expatriates** | https://wwwnc.cdc.gov/travel/page/traveler-information-center |

## Visiting Friends and Relatives (VFR) Travelers
| Risk Factor | Description | URL |
|-------------|-------------|-----|
| **Higher Risk** | **VFR travelers have higher rates of preventable infections** (malaria, Typhoid, Hepatitis A) | https://wwwnc.cdc.gov/travel/page/vfr |
| **Reasons** | **Last-minute travel**, **Less pre-travel advice**, **Perception of lower risk**, **Travel to rural areas**, **Closer contact with local population** | https://wwwnc.cdc.gov/travel/page/vfr |
| **Key Interventions** | **Emphasize malaria prophylaxis**, **Vaccinations**, **Food/water precautions** | https://wwwnc.cdc.gov/travel/page/vfr |
