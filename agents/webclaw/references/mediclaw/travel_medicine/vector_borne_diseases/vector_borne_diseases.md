| **Dengue** | **Aedes** (day-biting) | **Tropics/subtropics worldwide** | **Dengvaxia** (only for seropositive, not for travelers) | https://www.cdc.gov/dengue/index.html |

## Dengue
| Component | Description | URL |
|-----------|-------------|-----|
| **Presentation** | **Fever**, **Severe headache**, **Retro-orbital pain**, **Myalgias/arthralgias** ("breakbone fever"), **Rash** | https://www.cdc.gov/dengue/symptoms/index.html |
| **Severe Dengue** | **Plasma leakage**, **Hemorrhage**, **Shock**, **Organ impairment** (occurs with **secondary infection** - antibody-dependent enhancement) | https://www.cdc.gov/dengue/symptoms/index.html |
| **Management** | **Supportive** (fluids, Acetaminophen), **AVOID NSAIDs/aspirin** (bleeding risk) | https://www.cdc.gov/dengue/treatment/index.html |
| **Prevention** | **Mosquito avoidance** (Aedes bites during day) | https://www.cdc.gov/dengue/prevention/index.html |
| **Vaccine** | **Dengvaxia** (only for **seropositive** individuals age 9-45 in endemic areas; **NOT for travelers**) | https://www.cdc.gov/dengue/vaccine/index.html |

## Zika Virus
| Component | Description | URL |
|-----------|-------------|-----|
| **Presentation** | **Often asymptomatic** (80%), **Fever**, **Rash**, **Conjunctivitis**, **Arthralgias** | https://www.cdc.gov/zika/symptoms/index.html |
| **Pregnancy Risk** | **Congenital Zika syndrome** (microcephaly, Brain abnormalities) | https://www.cdc.gov/zika/pregnancy/index.html |
| **Sexual Transmission** | **Can be transmitted sexually** (use condoms or abstain) | https://www.cdc.gov/zika/prevention/sexual-transmission.html |
| **Prevention** | **Mosquito avoidance**, **Condoms** (for pregnant women or those planning pregnancy) | https://www.cdc.gov/zika/prevention/index.html |
| **Testing** | **Only if symptomatic** with possible exposure (no routine screening) | https://www.cdc.gov/zika/testing/index.html |
