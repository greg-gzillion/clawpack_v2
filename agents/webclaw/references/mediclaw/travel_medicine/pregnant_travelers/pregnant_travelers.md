# Travel Medicine - Pregnant Travelers

## General Considerations
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **Safest Time to Travel** | **Second trimester** (14-28 weeks) - lowest risk of miscarriage/preterm labor | https://wwwnc.cdc.gov/travel/page/pregnant-travelers |
| **Air Travel** | **Most airlines allow until 36 weeks** (domestic) or **28-35 weeks** (international); **Check airline policy** | https://www.acog.org/clinical/clinical-guidance/committee-opinion/articles/2018/08/air-travel-during-pregnancy |
| **DVT Prophylaxis** | **Compression stockings**, **Ambulate q2h**, **Aisle seat**, **Hydration** | https://wwwnc.cdc.gov/travel/page/pregnant-travelers |
| **Travel Insurance** | **Ensure coverage for pregnancy complications**, **Neonatal care** | https://wwwnc.cdc.gov/travel/page/pregnant-travelers |

## Vaccines in Pregnancy
| Vaccine | Recommendation | URL |
|---------|----------------|-----|
| **Influenza (Inactivated)** | **Recommended** (any trimester) | https://www.cdc.gov/vaccines/pregnancy/hcp-toolkit/flu-vaccine-pregnancy.html |
| **Tdap** | **Recommended** (27-36 weeks, or any time if traveling) | https://www.cdc.gov/vaccines/pregnancy/hcp-toolkit/tdap-vaccine-pregnancy.html |
| **COVID-19** | **Recommended** (any trimester) | https://www.cdc.gov/coronavirus/2019-ncov/vaccines/recommendations/pregnancy.html |
| **Hepatitis A** | **Consider if high-risk** (risk of infection vs theoretical vaccine risk) | https://www.cdc.gov/vaccines/pregnancy/hcp-toolkit/hepatitis-a-vaccine-pregnancy.html |
| **Hepatitis B** | **Recommended if at risk** | https://www.cdc.gov/vaccines/pregnancy/hcp-toolkit/hepatitis-b-vaccine-pregnancy.html |
| **Typhoid (Vi polysaccharide)** | **Consider if high-risk** | https://www.cdc.gov/vaccines/pregnancy/hcp-toolkit/typhoid-vaccine-pregnancy.html |
| **Yellow Fever** | **Generally avoid** (live vaccine); **Consider if high-risk** (consultation) | https://www.cdc.gov/yellowfever/vaccine/pregnancy.html |
| **MMR, Varicella** | **Contraindicated** (live vaccines) | https://www.cdc.gov/vaccines/pregnancy/hcp-toolkit/mmr-vaccine-pregnancy.html |

## Malaria Prophylaxis in Pregnancy
| Drug | Recommendation | URL |
|------|----------------|-----|
| **Mefloquine** | **Safe** (Category B) | https://www.cdc.gov/malaria/travelers/drugs.html |
| **Chloroquine** | **Safe** (Category C - used safely) | https://www.cdc.gov/malaria/travelers/drugs.html |
| **Atovaquone-Proguanil** | **Limited data** (Category C) | https://www.cdc.gov/malaria/travelers/drugs.html |
| **Doxycycline** | **Contraindicated** (Category D - tooth discoloration) | https://www.cdc.gov/malaria/travelers/drugs.html |
| **Primaquine** | **Contraindicated** (fetal G6PD status unknown) | https://www.cdc.gov/malaria/travelers/drugs.html |
| **Mosquito Avoidance** | **Essential** (DEET safe in pregnancy) | https://www.cdc.gov/mosquitoes/mosquito-bites/prevent-mosquito-bites.html |

## Traveler's Diarrhea in Pregnancy
| Treatment | Recommendation | URL |
|-----------|----------------|-----|
| **Oral Rehydration** | **First-line** | https://wwwnc.cdc.gov/travel/page/pregnant-travelers |
| **Loperamide** | **Avoid** (Category C, limited data) | https://wwwnc.cdc.gov/travel/page/pregnant-travelers |
| **Azithromycin** | **Preferred antibiotic** if needed | https://wwwnc.cdc.gov/travel/page/pregnant-travelers |
| **Fluoroquinolones** | **Avoid** (Category C - arthropathy in animal studies) | https://wwwnc.cdc.gov/travel/page/pregnant-travelers |
| **Rifaximin** | **Consider** (Category C, Non-absorbed) | https://wwwnc.cdc.gov/travel/page/pregnant-travelers |

## Zika and Pregnancy
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **Travel** | **Avoid travel to areas with active Zika transmission** | https://www.cdc.gov/zika/pregnancy/index.html |
| **If Travel Unavoidable** | **Strict mosquito avoidance**, **Condoms** (or abstain) for entire pregnancy | https://www.cdc.gov/zika/pregnancy/index.html |
