## Malaria Prophylaxis Regimens
| Drug | Regimen | Notes | URL |
|------|---------|-------|-----|
| **Atovaquone-Proguanil (Malarone)** | **Daily**, Start **1-2 days before**, Continue **7 days after** | **Best tolerated**, **Daily dosing**, **Pregnancy Category C** | https://www.cdc.gov/malaria/travelers/drugs.html |
| **Doxycycline** | **Daily**, Start **1-2 days before**, Continue **4 weeks after** | **Photosensitivity**, **GI upset**, **Vaginal candidiasis**, **Contraindicated in pregnancy/children <8 years** | https://www.cdc.gov/malaria/travelers/drugs.html |
| **Mefloquine** | **Weekly**, Start **=2 weeks before**, Continue **4 weeks after** | **Neuropsychiatric effects** (boxed warning), **Contraindicated in psychiatric/seizure history**, **Safe in pregnancy** | https://www.cdc.gov/malaria/travelers/drugs.html |
| **Primaquine** | **Daily**, Start **1-2 days before**, Continue **7 days after** | **Requires G6PD testing** (hemolysis if deficient), **Contraindicated in pregnancy**, **Effective against P. vivax hypnozoites** | https://www.cdc.gov/malaria/travelers/drugs.html |
| **Tafenoquine (Arakoda)** | **Weekly**, Start **3 days before**, **Weekly during**, **Single dose week after** | **Requires G6PD testing**, **Contraindicated in pregnancy/children**, **Long half-life** | https://www.cdc.gov/malaria/travelers/drugs.html |
| **Chloroquine** | **Weekly**, Start **1-2 weeks before**, Continue **4 weeks after** | **Only for chloroquine-sensitive areas** (Haiti, Dominican Republic, Central America west of Panama Canal) | https://www.cdc.gov/malaria/travelers/drugs.html |

## G6PD Deficiency
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **X-linked enzyme deficiency** ? **Hemolytic anemia** with **oxidative stress** | https://www.cdc.gov/malaria/travelers/g6pd.html |
| **Prevalence** | **~400 million people** worldwide; **Higher in malaria-endemic areas** (Africa, Mediterranean, Middle East, Asia) | https://www.cdc.gov/malaria/travelers/g6pd.html |
| **Testing** | **Required before primaquine or tafenoquine** | https://www.cdc.gov/malaria/travelers/g6pd.html |
| **Drugs to Avoid** | **Primaquine**, **Tafenoquine**, **Dapsone**, **Sulfonamides**, **Nitrofurantoin**, **Rasburicase** | https://www.cdc.gov/malaria/travelers/g6pd.html |

## Mosquito Avoidance (Bite Prevention)
| Measure | Recommendation | URL |
|---------|----------------|-----|
| **Insect Repellent** | **DEET 20-50%** (safe in pregnancy/children), **Picaridin 20%**, **IR3535**, **Oil of lemon eucalyptus** (PMD) | https://www.cdc.gov/mosquitoes/mosquito-bites/prevent-mosquito-bites.html |
| **Permethrin-Treated Clothing** | **Treat clothing, shoes, gear** (or buy pre-treated) | https://www.cdc.gov/mosquitoes/mosquito-bites/prevent-mosquito-bites.html |
| **Bed Nets** | **Insecticide-treated nets (ITNs)** essential in endemic areas | https://www.cdc.gov/malaria/travelers/bed-nets.html |
| **Clothing** | **Long sleeves/pants**, **Light colors** | https://www.cdc.gov/mosquitoes/mosquito-bites/prevent-mosquito-bites.html |
| **Time of Day** | **Anopheles mosquitoes bite from dusk to dawn** | https://www.cdc.gov/malaria/about/biology/index.html |
