## Beers Criteria - Key Potentially Inappropriate Medications
| Drug Class | Examples | Rationale | Alternatives | URL |
|------------|----------|-----------|--------------|-----|
| **Anticholinergics** | Diphenhydramine, Oxybutynin, Amitriptyline | **Confusion**, **Constipation**, **Falls** | Cetirizine, Mirabegron, Duloxetine | https://www.americangeriatrics.org/clinical-practice/beers-criteria |
| **Benzodiazepines** | Lorazepam, Alprazolam, Clonazepam | **Falls**, **Cognitive impairment**, **Dependence** | CBT, SSRIs, Melatonin | https://www.americangeriatrics.org/clinical-practice/beers-criteria |
| **NSAIDs** | Ibuprofen, Naproxen, Indomethacin | **GI bleeding**, **AKI**, **Hypertension** | Acetaminophen, Topical NSAIDs | https://www.americangeriatrics.org/clinical-practice/beers-criteria |
| **PPIs >8 weeks** | Omeprazole, Pantoprazole | **C. diff**, **Fractures**, **B12 deficiency**, **CKD** | H2RAs, Deprescribe | https://www.americangeriatrics.org/clinical-practice/beers-criteria |
| **Sulfonylureas** | Glyburide, Glimepiride | **Severe prolonged hypoglycemia** | Metformin, DPP-4 inhibitors | https://www.americangeriatrics.org/clinical-practice/beers-criteria |
| **TCAs** | Amitriptyline, Nortriptyline | **Anticholinergic**, **Orthostasis**, **QTc prolongation** | SSRIs, SNRIs, Gabapentinoids | https://www.americangeriatrics.org/clinical-practice/beers-criteria |
| **Antipsychotics (for dementia)** | Risperidone, Olanzapine, Quetiapine | **Increased mortality**, **CVA**, **Falls** | Non-pharmacologic interventions | https://www.americangeriatrics.org/clinical-practice/beers-criteria |

## Geriatric Prescribing Principles
| Principle | Description | URL |
|-----------|-------------|-----|
| **Start Low, Go Slow** | **50% reduction** in starting dose; Titrate slowly | https://www.nia.nih.gov/health/safe-use-medicines-older-adults |
| **Avoid Polypharmacy** | Review medications regularly; **Deprescribe** when appropriate | https://deprescribing.org |
| **Renal Dose Adjustment** | Adjust for **declining GFR** | https://www.kidney.org/professionals/kdoqi/gfr_calculator |
| **Monitor for Falls** | Avoid CNS-active drugs | https://www.cdc.gov/steadi |
| **Simplify Regimens** | Once-daily dosing, Pill organizers | https://www.nia.nih.gov/health/safe-use-medicines-older-adults |
