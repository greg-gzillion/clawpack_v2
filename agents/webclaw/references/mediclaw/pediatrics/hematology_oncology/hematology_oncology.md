# Pediatrics - Hematology and Oncology

## Iron Deficiency Anemia
| Component | Description | URL |
|-----------|-------------|-----|
| **Etiology** | **Inadequate intake** (excessive cow's milk), **Blood loss**, **Malabsorption** | https://www.cdc.gov/ncbddd/irondeficiency/index.html |
| **Diagnosis** | **Microcytic**, **Hypochromic anemia**, **Low ferritin**, **High TIBC**, **Low iron** | https://www.cdc.gov/ncbddd/irondeficiency/index.html |
| **Treatment** | **Oral iron** 3-6 mg/kg/day elemental iron x 3 months; **Limit cow's milk** (<24 oz/day) | https://www.cdc.gov/ncbddd/irondeficiency/index.html |

## Sickle Cell Disease
| Component | Description | URL |
|-----------|-------------|-----|
| **Genetics** | **Autosomal recessive**, **HBB gene** (Glu6Val) | https://www.cdc.gov/ncbddd/sicklecell/index.html |
| **Complications** | **Vaso-occlusive crisis**, **Acute chest syndrome**, **Splenic sequestration**, **Aplastic crisis**, **Stroke**, **Infection** (functional asplenia) | https://www.cdc.gov/ncbddd/sicklecell/index.html |
| **Prophylaxis** | **Penicillin** (until age 5), **Pneumococcal vaccines**, **Hydroxyurea** (starting at 9 months) | https://www.cdc.gov/ncbddd/sicklecell/index.html |

## Immune Thrombocytopenia (ITP)
| Component | Description | URL |
|-----------|-------------|-----|
| **Presentation** | **Acute onset petechiae, Purpura**, Epistaxis, **Well-appearing child** | https://www.nhlbi.nih.gov/health/immune-thrombocytopenia |
| **Diagnosis** | **Isolated thrombocytopenia** (<100k), **Normal Hb/WBC**, **Large platelets** on smear | https://www.nhlbi.nih.gov/health/immune-thrombocytopenia |
| **Management** | **Observation** (platelets >20k, No bleeding); **IVIG**, **Anti-D**, **Corticosteroids** | https://www.nhlbi.nih.gov/health/immune-thrombocytopenia |
| **Prognosis** | **80% resolve spontaneously** within 6 months | https://www.nhlbi.nih.gov/health/immune-thrombocytopenia |
