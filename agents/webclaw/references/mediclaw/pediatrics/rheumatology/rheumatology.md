# Pediatrics - Rheumatology

## Juvenile Idiopathic Arthritis (JIA)
| Subtype | Frequency | Features | URL |
|---------|-----------|----------|-----|
| **Oligoarticular** | 50-60% | **=4 joints** in first 6 months; **Chronic anterior uveitis** risk (ANA+) | https://www.niams.nih.gov/health-topics/juvenile-arthritis |
| **Polyarticular (RF-)** | 20-30% | **=5 joints**; Similar to adult RA | https://www.niams.nih.gov/health-topics/juvenile-arthritis |
| **Polyarticular (RF+)** | 5-10% | **=5 joints**, **RF+**, **Anti-CCP+**; More erosive | https://www.niams.nih.gov/health-topics/juvenile-arthritis |
| **Systemic** | 5-10% | **Quotidian fever**, **Evanescent rash**, **Hepatosplenomegaly**, **Serositis** | https://www.niams.nih.gov/health-topics/juvenile-arthritis |
| **Enthesitis-Related** | 5-10% | **Enthesitis**, **Axial involvement**, **HLA-B27+** | https://www.niams.nih.gov/health-topics/juvenile-arthritis |
| **Psoriatic** | 5-10% | Arthritis + **Psoriasis** or **Nail pitting** or **Dactylitis** or **Family history** | https://www.niams.nih.gov/health-topics/juvenile-arthritis |

## JIA - Treatment
| Severity | Treatment | URL |
|----------|-----------|-----|
| **Mild** | **NSAIDs** (naproxen), **Intra-articular corticosteroids** | https://www.niams.nih.gov/health-topics/juvenile-arthritis |
| **Moderate** | **Methotrexate**, **Sulfasalazine** | https://www.niams.nih.gov/health-topics/juvenile-arthritis |
| **Severe** | **Biologics** (TNF inhibitors, IL-6R inhibitors, IL-1 inhibitors, CTLA4-Ig) | https://www.niams.nih.gov/health-topics/juvenile-arthritis |
| **Uveitis Screening** | **Ophthalmology exam** q3-12 months based on risk | https://www.niams.nih.gov/health-topics/juvenile-arthritis |

## Systemic Lupus Erythematosus (SLE)
| Component | Description | URL |
|-----------|-------------|-----|
| **Clinical Features** | **Malar rash**, **Discoid rash**, **Photosensitivity**, **Arthritis**, **Serositis**, **Renal** (nephritis), **Hematologic** (cytopenias), **ANA+** | https://www.niams.nih.gov/health-topics/lupus |
| **Diagnosis** | **SLICC criteria** or **ACR/EULAR criteria** | https://www.niams.nih.gov/health-topics/lupus |
| **Treatment** | **Hydroxychloroquine** (all), **Corticosteroids**, **Immunosuppressants** (mycophenolate, Cyclophosphamide), **Biologics** (belimumab) | https://www.niams.nih.gov/health-topics/lupus |

## Juvenile Dermatomyositis (JDM)
| Component | Description | URL |
|-----------|-------------|-----|
| **Clinical Features** | **Proximal muscle weakness**, **Heliotrope rash**, **Gottron papules**, **Calcinosis** | https://www.niams.nih.gov/health-topics/juvenile-dermatomyositis |
| **Diagnosis** | **Elevated CK/Aldolase**, **MRI** (muscle edema), **EMG**, **Muscle biopsy** | https://www.niams.nih.gov/health-topics/juvenile-dermatomyositis |
| **Treatment** | **Corticosteroids**, **Methotrexate**, **IVIG**, **Mycophenolate**, **Rituximab** | https://www.niams.nih.gov/health-topics/juvenile-dermatomyositis |
