# Pediatrics - Pediatric Procedures

## Lumbar Puncture
| Component | Description | URL |
|-----------|-------------|-----|
| **Positioning** | **Lateral decubitus** or **Sitting** (neck flexed, Legs extended) | https://www.aap.org/en/patient-care/lumbar-puncture/ |
| **Landmark** | **L3-L4** or **L4-L5** (iliac crest) | https://www.aap.org/en/patient-care/lumbar-puncture/ |
| **Needle Size** | **22 gauge**, **1.5 inch** (infant), **2.5-3.5 inch** (older child) | https://www.aap.org/en/patient-care/lumbar-puncture/ |
| **CSF Interpretation** | **WBC**: <1 (neonate), <5 (older); **Protein**: <170 (neonate), <45 (older); **Glucose**: >60% serum | https://www.aap.org/en/patient-care/lumbar-puncture/ |

## Intraosseous (IO) Access
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | **Emergent vascular access** when IV cannot be obtained | https://www.aap.org/en/patient-care/intraosseous-access/ |
| **Sites** | **Proximal tibia** (1-2 cm below and medial to tibial tuberosity), **Distal femur**, **Proximal humerus** | https://www.aap.org/en/patient-care/intraosseous-access/ |
| **Contraindications** | **Fracture** in target bone, **Prior IO attempt** in same bone, **Osteogenesis imperfecta** | https://www.aap.org/en/patient-care/intraosseous-access/ |
