# Pediatrics - Infectious Diseases

## Pediatric Fever Without Source
| Age Group | Management | URL |
|-----------|------------|-----|
| **0-28 days (Neonate)** | **Full sepsis workup** (CBC, Blood culture, UA/Urine culture, LP, CXR if respiratory) + **Admit** + **Empiric antibiotics** (Ampicillin + Gentamicin or Cefotaxime) ± **Acyclovir** | https://www.aap.org/en/patient-care/neonatal-fever/ |
| **29-60 days** | **Labs** (CBC, UA/Urine culture, Blood culture) ± LP; **Consider outpatient** if low-risk (normal labs, Well-appearing) | https://www.aap.org/en/patient-care/infant-fever/ |
| **61-90 days** | **Labs** (CBC, UA/Urine culture) ± Blood culture; **Outpatient management** if low-risk | https://www.aap.org/en/patient-care/infant-fever/ |
| **3-36 months** | **UA/Urine culture** (all girls <24 mo, Uncircumcised boys <12 mo, Circumcised boys <6 mo); **CXR** if respiratory symptoms or WBC >20k; **Blood culture** if ill-appearing | https://www.aap.org/en/patient-care/toddler-fever/ |

## Common Pediatric Exanthems
| Disease | Pathogen | Clinical Features | Treatment | URL |
|---------|----------|-------------------|-----------|-----|
| **Measles (Rubeola)** | Measles virus | **3 C's** (Cough, Coryza, Conjunctivitis), **Koplik spots**, **Maculopapular rash** (head ? trunk) | **Vitamin A**, Supportive | https://www.cdc.gov/measles/index.html |
| **Rubella** | Rubella virus | Fever, Lymphadenopathy, Maculopapular rash | Supportive | https://www.cdc.gov/rubella/index.html |
| **Erythema Infectiosum (Fifth Disease)** | Parvovirus B19 | **"Slapped cheeks"** ? Lacy rash | Supportive | https://www.cdc.gov/parvovirusb19/index.html |
| **Roseola Infantum** | HHV-6, HHV-7 | **High fever** ? Rash as fever defervesces | Supportive | https://www.cdc.gov/hhv-6/index.html |
| **Hand-Foot-Mouth Disease** | Coxsackievirus A16 | **Vesicles on hands, Feet, Mouth** | Supportive | https://www.cdc.gov/hand-foot-mouth/index.html |
| **Scarlet Fever** | S. pyogenes | Pharyngitis + **Sandpaper rash**, **Strawberry tongue** | Penicillin/Amoxicillin | https://www.cdc.gov/groupastrep/index.html |
| **Varicella (Chickenpox)** | VZV | **Pruritic vesicular rash** (crops) | Supportive; Acyclovir if high-risk | https://www.cdc.gov/chickenpox/index.html |

## Acute Otitis Media (AOM)
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Bulging TM** (most specific), **Decreased mobility**, **Erythema**, **Otorrhea** | https://www.aap.org/en/patient-care/otitis-media/ |
| **Observation Criteria** | **Age =6 mo**, **Unilateral**, **Mild symptoms**, **No otorrhea**, **Reliable follow-up** | https://www.aap.org/en/patient-care/otitis-media/ |
| **First-Line Antibiotic** | **Amoxicillin** 80-90 mg/kg/day divided BID x 10 days (5-7 days if =6 years) | https://www.aap.org/en/patient-care/otitis-media/ |
| **Treatment Failure** | **Amoxicillin-clavulanate**, **Ceftriaxone IM/IV** | https://www.aap.org/en/patient-care/otitis-media/ |

## Bronchiolitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Etiology** | **RSV** (most common), Rhinovirus, Human metapneumovirus | https://www.aap.org/en/patient-care/bronchiolitis/ |
| **Clinical Features** | **Wheezing**, **Crackles**, **Respiratory distress** (infants <2 years) | https://www.aap.org/en/patient-care/bronchiolitis/ |
| **Treatment** | **Supportive** (suctioning, Hydration, Oxygen); **AVOID** bronchodilators, Steroids, Antibiotics | https://www.aap.org/en/patient-care/bronchiolitis/ |
| **Prevention** | **Palivizumab** for high-risk infants (<29 weeks GA, CHD, CLD) | https://www.aap.org/en/patient-care/bronchiolitis/ |

## Croup (Laryngotracheobronchitis)
| Component | Description | URL |
|-----------|-------------|-----|
| **Etiology** | **Parainfluenza** (most common), RSV, Influenza | https://www.aap.org/en/patient-care/croup/ |
| **Clinical Features** | **Barking cough**, **Stridor**, Hoarseness, Fever | https://www.aap.org/en/patient-care/croup/ |
| **Treatment** | **Dexamethasone** 0.6 mg/kg PO/IM (max 10 mg); **Nebulized epinephrine** (moderate-severe) | https://www.aap.org/en/patient-care/croup/ |

## Pediatric UTI
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnosis** | **Catheterized specimen**: =50,000 CFU/mL; **Clean catch**: =100,000 CFU/mL | https://www.aap.org/en/patient-care/uti/ |
| **Treatment (Oral)** | Cephalexin, Cefixime, Amoxicillin-clavulanate, TMP-SMX | https://www.aap.org/en/patient-care/uti/ |
| **Treatment (IV)** | Ceftriaxone, Cefotaxime, Gentamicin | https://www.aap.org/en/patient-care/uti/ |
| **Duration** | 7-14 days | https://www.aap.org/en/patient-care/uti/ |
| **Imaging** | **Renal/bladder ultrasound** (all children <2 years after first UTI); **VCUG** (if abnormal US or recurrent) | https://www.aap.org/en/patient-care/uti/ |

## Kawasaki Disease
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Fever =5 days** + **=4 of**: Bilateral conjunctival injection, Oral mucous membrane changes, Polymorphous rash, Cervical lymphadenopathy (>1.5 cm), Extremity changes | https://www.heart.org/en/health-topics/kawasaki-disease |
| **Incomplete Kawasaki** | Fever =5 days + <4 criteria + **Lab/echo findings** | https://www.heart.org/en/health-topics/kawasaki-disease |
| **Treatment** | **IVIG 2 g/kg x 1** + **High-dose aspirin** (80-100 mg/kg/day) ? Low-dose aspirin (3-5 mg/kg/day) | https://www.heart.org/en/health-topics/kawasaki-disease |
| **Refractory** | **Second dose IVIG**, **Corticosteroids**, **Infliximab** | https://www.heart.org/en/health-topics/kawasaki-disease |
