| Component | Description | URL |
|-----------|-------------|-----|

## Nephrotic Syndrome in Children
| Component | Description | URL |
|-----------|-------------|-----|
| **Epidemiology** | Peak age 2-6 years; Male > Female | https://www.niddk.nih.gov/health-information/kidney-disease/children/nephrotic-syndrome-children |
| **Most Common Cause** | **Minimal Change Disease** (80-90%) | https://www.niddk.nih.gov/health-information/kidney-disease/children/nephrotic-syndrome-children |
| **Clinical Features** | **Edema** (periorbital ? generalized), **Proteinuria** (>40 mg/m2/h or UPCR >2), **Hypoalbuminemia** (<2.5), **Hyperlipidemia** | https://www.niddk.nih.gov/health-information/kidney-disease/children/nephrotic-syndrome-children |
| **Treatment** | **Prednisone** 2 mg/kg/day x 4-6 weeks ? taper | https://www.niddk.nih.gov/health-information/kidney-disease/children/nephrotic-syndrome-children |
| **Steroid Response** | **Complete remission** in 80-90%; **Relapse** common (60-80%) | https://www.niddk.nih.gov/health-information/kidney-disease/children/nephrotic-syndrome-children |

## Acute Post-Streptococcal Glomerulonephritis
| Component | Description | URL |
|-----------|-------------|-----|
| **Presentation** | **Hematuria** (cola-colored), **Proteinuria**, **Hypertension**, **Edema**, **Oliguria** (1-3 weeks after GAS pharyngitis/skin infection) | https://www.cdc.gov/groupastrep/diseases-hcp/post-streptococcal.html |
| **Diagnosis** | **Low C3** (normalizes in 8 weeks), **Elevated ASO/Anti-DNase B**, **UA** (RBC casts, Proteinuria) | https://www.cdc.gov/groupastrep/diseases-hcp/post-streptococcal.html |
| **Treatment** | **Supportive** (fluid/sodium restriction, Diuretics, Antihypertensives); **No role for antibiotics** | https://www.cdc.gov/groupastrep/diseases-hcp/post-streptococcal.html |

## Henoch-Schönlein Purpura (IgA Vasculitis)
| Component | Description | URL |
|-----------|-------------|-----|
| **Clinical Features** | **Palpable purpura** (lower extremities, Buttocks), **Arthralgias/arthritis**, **Abdominal pain**, **Hematuria** | https://www.niams.nih.gov/health-topics/henoch-schonlein-purpura |
| **Diagnosis** | **Clinical**; **UA** (hematuria, Proteinuria), **Normal platelets** | https://www.niams.nih.gov/health-topics/henoch-schonlein-purpura |
| **Treatment** | **Supportive** (most); **Corticosteroids** for severe GI or renal involvement | https://www.niams.nih.gov/health-topics/henoch-schonlein-purpura |

## Hemolytic Uremic Syndrome (HUS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Etiology** | **Shiga toxin-producing E. coli (STEC)** O157:H7 | https://www.cdc.gov/ecoli/index.html |
| **Clinical Triad** | **Microangiopathic hemolytic anemia** (schistocytes), **Thrombocytopenia**, **Acute kidney injury** | https://www.cdc.gov/ecoli/index.html |
