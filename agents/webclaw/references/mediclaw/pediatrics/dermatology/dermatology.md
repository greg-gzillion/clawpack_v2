## Diaper Dermatitis
| Type | Features | Treatment | URL |
|------|----------|-----------|-----|
| **Irritant** | **Erythema**, **Scaling**, **Spares creases** | **Barrier cream** (zinc oxide), **Frequent diaper changes**, **Air drying** | https://www.aap.org/en/patient-care/diaper-rash/ |
| **Candidal** | **Beefy red**, **Satellite lesions**, **Involves creases** | **Topical antifungal** (nystatin, Clotrimazole) | https://www.aap.org/en/patient-care/diaper-rash/ |

## Infantile Hemangioma
| Component | Description | URL |
|-----------|-------------|-----|
| **Natural History** | **Appears in first weeks**, **Proliferates** (first year), **Involutes** (years) | https://www.aap.org/en/patient-care/hemangioma/ |
| **Indications for Treatment** | **Vision compromise**, **Airway compromise**, **Ulceration**, **Disfigurement**, **Hepatic** (multiple cutaneous) | https://www.aap.org/en/patient-care/hemangioma/ |
| **Treatment** | **Oral propranolol** (first-line), **Topical timolol** (superficial), **Corticosteroids** | https://www.aap.org/en/patient-care/hemangioma/ |

## Acne Vulgaris
| Severity | Treatment | URL |
|----------|-----------|-----|
| **Mild (Comedonal)** | **Topical retinoids** (tretinoin, Adapalene), **Benzoyl peroxide** | https://www.aad.org/public/diseases/acne |
| **Moderate (Inflammatory)** | **Topical retinoid + Benzoyl peroxide + Topical antibiotic** or **Oral antibiotic** | https://www.aad.org/public/diseases/acne |
| **Severe (Nodulocystic)** | **Oral isotretinoin** | https://www.aad.org/public/diseases/acne |

## Common Pediatric Rashes
| Rash | Features | Treatment | URL |
|------|----------|-----------|-----|
| **Erythema Infectiosum** | **"Slapped cheeks"** ? Lacy rash | Supportive | https://www.cdc.gov/parvovirusb19/index.html |
| **Roseola** | High fever ? Rash as fever defervesces | Supportive | https://www.cdc.gov/hhv-6/index.html |
| **Hand-Foot-Mouth** | Vesicles on hands, Feet, Mouth | Supportive | https://www.cdc.gov/hand-foot-mouth/index.html |
| **Scarlet Fever** | Sandpaper rash, Strawberry tongue | Penicillin/Amoxicillin | https://www.cdc.gov/groupastrep/index.html |

| **Molluscum Contagiosum** | Umbilicated papules | Observation; Cryotherapy, Curettage | https://www.cdc.gov/poxvirus/molluscum-contagiosum/index.html |
