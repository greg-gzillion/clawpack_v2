# Pediatrics - Genetic and Congenital Disorders

## Down Syndrome (Trisomy 21)
| Component | Description | URL |
|-----------|-------------|-----|
| **Genetics** | **Trisomy 21** (95%), **Translocation** (4%), **Mosaicism** (1%) | https://www.cdc.gov/ncbddd/birthdefects/downsyndrome.html |
| **Clinical Features** | **Hypotonia**, **Flat facies**, **Upslanting palpebral fissures**, **Single transverse palmar crease**, **Brachycephaly**, **Epicanthal folds**, **Brushfield spots** | https://www.cdc.gov/ncbddd/birthdefects/downsyndrome.html |
| **Associated Conditions** | **Congenital heart disease** (AVSD, VSD, ASD, TOF), **Duodenal atresia**, **Hirschsprung disease**, **Hypothyroidism**, **Leukemia**, **Atlantoaxial instability**, **Alzheimer disease** (early onset) | https://www.cdc.gov/ncbddd/birthdefects/downsyndrome.html |
| **Health Supervision** | **Echocardiogram**, **Thyroid function** (newborn and annually), **Hearing/Vision screening**, **CBC** (newborn and annually), **Sleep study** (by age 4), **Cervical spine X-ray** (if symptomatic) | https://www.aap.org/en/patient-care/down-syndrome/ |

## Turner Syndrome (45,X)
| Component | Description | URL |
|-----------|-------------|-----|
| **Genetics** | **45,X** (50%), Mosaicism (45,X/46,XX), Isochromosome Xq | https://www.nichd.nih.gov/health/topics/turner |
| **Clinical Features** | **Short stature**, **Webbed neck**, **Broad chest with widely spaced nipples**, **Lymphedema** (hands/feet in infancy), **Cubitus valgus**, **Low posterior hairline** | https://www.nichd.nih.gov/health/topics/turner |
| **Associated Conditions** | **Coarctation of aorta**, **Bicuspid aortic valve**, **Horseshoe kidney**, **Hypothyroidism**, **Celiac disease**, **Ovarian dysgenesis** (infertility), **Learning disabilities** (math, Visuospatial) | https://www.nichd.nih.gov/health/topics/turner |
| **Health Supervision** | **Echocardiogram** (cardiology evaluation), **Renal ultrasound**, **Thyroid function** (annually), **Celiac screen**, **Hearing evaluation**, **Growth hormone therapy**, **Estrogen replacement** (puberty) | https://www.aap.org/en/patient-care/turner-syndrome/ |

## Klinefelter Syndrome (47,XXY)
| Component | Description | URL |
|-----------|-------------|-----|
| **Genetics** | **47,XXY** (most common), 48,XXXY, 49,XXXXY, Mosaicism | https://www.nichd.nih.gov/health/topics/klinefelter |
| **Clinical Features** | **Tall stature**, **Small testes**, **Gynecomastia**, **Infertility**, **Learning disabilities** (language), **Behavioral problems** | https://www.nichd.nih.gov/health/topics/klinefelter |
| **Health Supervision** | **Testosterone replacement** (puberty), **Speech therapy**, **Educational support**, **Fertility counseling** | https://www.aap.org/en/patient-care/klinefelter-syndrome/ |

## Fragile X Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| **Genetics** | **FMR1 gene** (CGG trinucleotide repeat >200); **X-linked dominant** | https://www.cdc.gov/ncbddd/fxs/index.html |
| **Clinical Features** | **Intellectual disability**, **Macroorchidism** (post-pubertal), **Long face**, **Large ears**, **Hyperextensible joints**, **Autism spectrum**, **ADHD** | https://www.cdc.gov/ncbddd/fxs/index.html |
| **Health Supervision** | **Genetic counseling**, **Developmental evaluation**, **Behavioral therapy**, **Educational support**, **Treat comorbidities** | https://www.aap.org/en/patient-care/fragile-x-syndrome/ |

## Prader-Willi Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| **Genetics** | **Deletion of paternal 15q11-q13** (70%), **Maternal uniparental disomy** (25%), Imprinting defect | https://www.nichd.nih.gov/health/topics/prader-willi |
| **Clinical Features** | **Neonatal hypotonia**, **Poor feeding** in infancy ? **Hyperphagia** and **Obesity** (age 2-4), **Short stature**, **Hypogonadism**, **Developmental delay**, **Behavioral problems** (temper tantrums, Skin picking) | https://www.nichd.nih.gov/health/topics/prader-willi |
| **Health Supervision** | **Growth hormone therapy**, **Strict dietary control**, **Behavioral therapy**, **Sex hormone replacement** (puberty) | https://www.aap.org/en/patient-care/prader-willi-syndrome/ |
