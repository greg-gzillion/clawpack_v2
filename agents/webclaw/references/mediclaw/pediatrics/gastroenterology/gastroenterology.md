# Pediatrics - Gastroenterology

## Infantile Hypertrophic Pyloric Stenosis
| Component | Description | URL |
|-----------|-------------|-----|
| **Presentation** | **Non-bilious projectile vomiting** at 3-6 weeks, **Palpable olive mass**, Visible peristalsis | https://www.aap.org/en/patient-care/pyloric-stenosis/ |
| **Risk Factors** | Firstborn male, Family history, Macrolide exposure | https://www.aap.org/en/patient-care/pyloric-stenosis/ |
| **Labs** | **Hypochloremic**, **Hypokalemic**, **Metabolic alkalosis** | https://www.aap.org/en/patient-care/pyloric-stenosis/ |
| **Diagnosis** | **Ultrasound** (pyloric muscle thickness >3 mm, Length >15 mm) | https://www.aap.org/en/patient-care/pyloric-stenosis/ |
| **Treatment** | **Fluid resuscitation**, **Correct electrolytes**, **Pyloromyotomy** | https://www.aap.org/en/patient-care/pyloric-stenosis/ |

## Intussusception
| Component | Description | URL |
|-----------|-------------|-----|
| **Presentation** | **Intermittent severe abdominal pain**, **Currant jelly stools**, Lethargy, Vomiting | https://www.aap.org/en/patient-care/intussusception/ |
| **Peak Age** | 3 months - 3 years | https://www.aap.org/en/patient-care/intussusception/ |
| **Diagnosis** | **Ultrasound** (target sign, Pseudokidney sign) | https://www.aap.org/en/patient-care/intussusception/ |
| **Treatment** | **Air or contrast enema reduction** (if stable); **Surgery** if failed or unstable | https://www.aap.org/en/patient-care/intussusception/ |

## Hirschsprung Disease
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathophysiology** | **Aganglionosis** of distal colon | https://www.niddk.nih.gov/health-information/digestive-diseases/hirschsprung-disease |
| **Presentation** | **Delayed passage of meconium** (>48 hours), Abdominal distention, Vomiting, **Enterocolitis** | https://www.niddk.nih.gov/health-information/digestive-diseases/hirschsprung-disease |
| **Diagnosis** | **Contrast enema** (transition zone), **Rectal suction biopsy** (absence of ganglion cells) | https://www.niddk.nih.gov/health-information/digestive-diseases/hirschsprung-disease |
| **Treatment** | **Surgical resection** (pull-through procedure) | https://www.niddk.nih.gov/health-information/digestive-diseases/hirschsprung-disease |

## Celiac Disease
| Component | Description | URL |
|-----------|-------------|-----|
| **Presentation** | **Diarrhea**, **Failure to thrive**, Abdominal distention, Irritability, Short stature, Iron deficiency | https://www.niddk.nih.gov/health-information/digestive-diseases/celiac-disease |

## Functional Constipation in Children
| Component | Description | URL |
|-----------|-------------|-----|
| **Rome IV Criteria** | <2 years: =1 month of symptoms; >2 years: =1 month, =2 criteria | https://www.niddk.nih.gov/health-information/digestive-diseases/constipation-children |
