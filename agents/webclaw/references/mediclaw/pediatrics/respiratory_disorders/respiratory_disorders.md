# Pediatrics - Respiratory Disorders

## Asthma
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnosis** | **Recurrent episodes** of wheezing, Cough, Chest tightness, Dyspnea; **Reversible airflow obstruction** on spirometry (=6 years) | https://www.nhlbi.nih.gov/health/asthma |
| **Classification** | **Intermittent**, **Mild Persistent**, **Moderate Persistent**, **Severe Persistent** | https://www.nhlbi.nih.gov/health/asthma |
| **Stepwise Management** | **Step 1**: PRN SABA; **Step 2**: Low-dose ICS; **Step 3**: Medium-dose ICS or Low-dose ICS/LABA; **Step 4**: Medium-dose ICS/LABA; **Step 5**: High-dose ICS/LABA + Referral | https://www.nhlbi.nih.gov/health/asthma |
| **Asthma Action Plan** | **Green** (well), **Yellow** (caution), **Red** (emergency) | https://www.nhlbi.nih.gov/health/asthma/action-plan |

## Cystic Fibrosis (CF)
| Component | Description | URL |
|-----------|-------------|-----|
| **Genetics** | **Autosomal recessive**, **CFTR gene** (most common ?F508) | https://www.cdc.gov/genomics/disease/cystic_fibrosis.htm |
| **Newborn Screening** | **Elevated immunoreactive trypsinogen (IRT)** ? **CFTR mutation panel** ? **Sweat chloride test** (=60 mmol/L diagnostic) | https://www.cdc.gov/newbornscreening/index.html |
| **Clinical Features** | **Chronic sinopulmonary infections**, **Pancreatic insufficiency**, **Failure to thrive**, **Meconium ileus**, **Nasal polyps**, **Clubbing**, **Male infertility** | https://www.nhlbi.nih.gov/health/cystic-fibrosis |
| **Treatment** | **Airway clearance**, **Mucolytics** (dornase alfa, Hypertonic saline), **Antibiotics**, **Pancreatic enzymes**, **CFTR modulators** (ivacaftor, Lumacaftor, Tezacaftor, Elexacaftor) | https://www.nhlbi.nih.gov/health/cystic-fibrosis |

## Epiglottitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Etiology** | **H. influenzae type b** (vaccinated), S. pneumoniae, S. pyogenes | https://www.aap.org/en/patient-care/epiglottitis/ |
| **Clinical Features** | **Tripod position**, **Drooling**, **Stridor**, **Muffled voice**, Fever | https://www.aap.org/en/patient-care/epiglottitis/ |
| **Diagnosis** | **Lateral neck X-ray** ("thumbprint sign"); **DO NOT examine oropharynx** | https://www.aap.org/en/patient-care/epiglottitis/ |
| **Treatment** | **Secure airway** (emergent), **Ceftriaxone** or **Cefotaxime** + **Vancomycin** | https://www.aap.org/en/patient-care/epiglottitis/ |

## Bacterial Tracheitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Etiology** | **S. aureus** (most common), S. pneumoniae, H. influenzae, M. catarrhalis | https://www.aap.org/en/patient-care/bacterial-tracheitis/ |
| **Clinical Features** | **Prodrome of viral URI** ? **High fever**, **Stridor**, **Toxic appearance**, **Copious purulent secretions** | https://www.aap.org/en/patient-care/bacterial-tracheitis/ |
| **Treatment** | **Secure airway** (often requires intubation), **IV antibiotics** (Ceftriaxone + Vancomycin) | https://www.aap.org/en/patient-care/bacterial-tracheitis/ |

## Retropharyngeal Abscess
| Component | Description | URL |
|-----------|-------------|-----|
| **Clinical Features** | Fever, Neck stiffness, Dysphagia, **Bulging posterior pharyngeal wall**, **Muffled voice** | https://www.aap.org/en/patient-care/retropharyngeal-abscess/ |
| **Diagnosis** | **CT neck with contrast** | https://www.aap.org/en/patient-care/retropharyngeal-abscess/ |
| **Treatment** | **Ampicillin-sulbactam** or **Clindamycin** + **Drainage** | https://www.aap.org/en/patient-care/retropharyngeal-abscess/ |
