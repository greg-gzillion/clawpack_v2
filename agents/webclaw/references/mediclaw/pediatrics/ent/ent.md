## Otitis Media with Effusion (OME)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Fluid in middle ear** without signs of acute infection | https://www.aap.org/en/patient-care/otitis-media-with-effusion/ |
| **Clinical Features** | **Hearing loss**, **Sensation of fullness**, **Asymptomatic** | https://www.aap.org/en/patient-care/otitis-media-with-effusion/ |
| **Management** | **Observation** (3 months), **Hearing test** if persistent >3 months or high-risk, **Tympanostomy tubes** if persistent and hearing loss | https://www.aap.org/en/patient-care/otitis-media-with-effusion/ |

## Acute Sinusitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Persistent** (>10 days) nasal discharge/daytime cough, **Worsening** ("double sickening"), **Severe** (fever =39蚓 + purulent discharge x 3 days) | https://www.aap.org/en/patient-care/sinusitis/ |
| **First-Line Antibiotic** | **Amoxicillin** 80-90 mg/kg/day divided BID x 10-14 days | https://www.aap.org/en/patient-care/sinusitis/ |
| **Treatment Failure** | **Amoxicillin-clavulanate**, **Ceftriaxone** | https://www.aap.org/en/patient-care/sinusitis/ |

## Pharyngitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Centor Criteria** | **Fever**, **Exudates**, **Tender anterior cervical LAD**, **No cough** | https://www.cdc.gov/groupastrep/diseases-hcp/strep-throat.html |
| **Diagnosis** | **Rapid strep test** or **Throat culture** | https://www.cdc.gov/groupastrep/diseases-hcp/strep-throat.html |
| **Treatment** | **Penicillin VK** or **Amoxicillin** x 10 days | https://www.cdc.gov/groupastrep/diseases-hcp/strep-throat.html |

## Obstructive Sleep Apnea (OSA)
| Component | Description | URL |
|-----------|-------------|-----|

| **Symptoms** | **Snoring**, **Witnessed apneas**, **Restless sleep**, **Daytime hyperactivity/behavioral problems**, **Enuresis** | https://www.aap.org/en/patient-care/sleep-apnea/ |
| **Diagnosis** | **Polysomnography (PSG)** | https://www.aap.org/en/patient-care/sleep-apnea/ |
