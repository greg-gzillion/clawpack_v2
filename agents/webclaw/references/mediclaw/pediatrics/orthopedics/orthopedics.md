# Pediatrics - Orthopedics

## Developmental Dysplasia of the Hip (DDH)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Spectrum of hip instability** (dysplasia, Subluxation, Dislocation) | https://www.aap.org/en/patient-care/ddh/ |
| **Risk Factors** | **Female**, **Breech presentation**, **Family history**, **Firstborn**, **Oligohydramnios** | https://www.aap.org/en/patient-care/ddh/ |
| **Examination** | **Barlow** (dislocates), **Ortolani** (reduces), **Galeazzi sign**, **Asymmetric thigh folds**, **Limited abduction** | https://www.aap.org/en/patient-care/ddh/ |
| **Diagnosis** | **Ultrasound** (<4-6 months), **X-ray** (>4-6 months) | https://www.aap.org/en/patient-care/ddh/ |
| **Treatment** | **Pavlik harness** (<6 months), **Closed reduction + Spica cast** (6-18 months), **Open reduction + Osteotomy** (>18 months) | https://www.aap.org/en/patient-care/ddh/ |

| Component | Description | URL |
|-----------|-------------|-----|

## Slipped Capital Femoral Epiphysis (SCFE)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Posterior and inferior displacement of femoral epiphysis** | https://www.aap.org/en/patient-care/scfe/ |
| **Demographics** | **Obese**, **Adolescent males** (10-16 years) | https://www.aap.org/en/patient-care/scfe/ |
| **Symptoms** | **Hip or knee pain**, **Limp**, **External rotation deformity** | https://www.aap.org/en/patient-care/scfe/ |
