# Pediatrics - Endocrinology

## Type 1 Diabetes Mellitus (T1DM)
| Component | Description | URL |
|-----------|-------------|-----|
| **Pathophysiology** | **Autoimmune destruction** of pancreatic beta cells | https://www.niddk.nih.gov/health-information/diabetes/overview/what-is-diabetes/type-1-diabetes |
| **Presentation** | **Polyuria**, **Polydipsia**, **Weight loss**, **DKA** (nausea, Vomiting, Abdominal pain, Kussmaul breathing, Fruity breath) | https://www.niddk.nih.gov/health-information/diabetes/overview/what-is-diabetes/type-1-diabetes |
| **Diagnosis** | **Fasting glucose =126**, **Random glucose =200 + symptoms**, **HbA1c =6.5%**, **Autoantibodies** (GAD65, IA-2, ZnT8, Insulin) | https://www.niddk.nih.gov/health-information/diabetes/overview/what-is-diabetes/type-1-diabetes |
| **Treatment** | **Basal-bolus insulin** (MDI or insulin pump), **Carbohydrate counting**, **CGM** | https://www.niddk.nih.gov/health-information/diabetes/overview/managing-diabetes |
| **Glycemic Targets** | **HbA1c <7.0%** (individualize), **Fasting 80-130**, **Postprandial <180** | https://www.niddk.nih.gov/health-information/diabetes/overview/managing-diabetes |

## Diabetic Ketoacidosis (DKA)
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnosis** | **Glucose >200**, **pH <7.3** or **HCO3 <15**, **Ketones positive** | https://www.niddk.nih.gov/health-information/diabetes/overview/what-is-diabetes/type-1-diabetes |
| **Fluids** | **10-20 mL/kg** bolus (if shock) ? **Maintenance + Deficit** over 48 hours | https://www.niddk.nih.gov/health-information/diabetes/overview/what-is-diabetes/type-1-diabetes |
| **Insulin** | **0.05-0.1 units/kg/h** IV infusion (start after initial fluids) | https://www.niddk.nih.gov/health-information/diabetes/overview/what-is-diabetes/type-1-diabetes |
| **Cerebral Edema** | **Headache**, **Altered mental status**, **Bradycardia**, **Hypertension**; **Mannitol** or **Hypertonic saline** | https://www.niddk.nih.gov/health-information/diabetes/overview/what-is-diabetes/type-1-diabetes |

## Congenital Hypothyroidism
| Component | Description | URL |
|-----------|-------------|-----|
| **Newborn Screening** | **Elevated TSH** or **Low T4** on heel stick | https://www.nichd.nih.gov/health/topics/congenital-hypothyroidism |
| **Confirmatory Testing** | **Serum TSH**, **Free T4** | https://www.nichd.nih.gov/health/topics/congenital-hypothyroidism |
| **Treatment** | **Levothyroxine** 10-15 mcg/kg/day (start immediately) | https://www.nichd.nih.gov/health/topics/congenital-hypothyroidism |
| **Goal** | **Normalize T4 in 2 weeks**, **TSH in 1 month** | https://www.nichd.nih.gov/health/topics/congenital-hypothyroidism |

## Growth Hormone Deficiency
| Component | Description | URL |
|-----------|-------------|-----|
| **Presentation** | **Short stature**, **Decreased growth velocity**, **Delayed bone age**, **Hypoglycemia** (infants) | https://www.nichd.nih.gov/health/topics/growth-hormone |
| **Diagnosis** | **Low IGF-1**, **IGFBP-3**, **Abnormal GH stimulation test** (peak <10 ng/mL) | https://www.nichd.nih.gov/health/topics/growth-hormone |
| **Treatment** | **Recombinant GH** 0.16-0.24 mg/kg/week divided daily | https://www.nichd.nih.gov/health/topics/growth-hormone |
