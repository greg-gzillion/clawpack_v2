# Psychiatry - Psychiatric Examination

## Components of the Psychiatric Interview
| Component | Description | URL |
|-----------|-------------|-----|
| **Chief Complaint** | Patient's primary concern in their own words | https://www.psychiatry.org/psychiatrists/practice/telepsychiatry/toolkit/clinical-standards |
| **History of Present Illness** | Chronological account of symptoms, Precipitants, Context | https://www.psychiatry.org/psychiatrists/practice/telepsychiatry/toolkit/clinical-standards |
| **Past Psychiatric History** | Prior diagnoses, Hospitalizations, Medications, Suicide attempts | https://www.psychiatry.org/psychiatrists/practice/telepsychiatry/toolkit/clinical-standards |
| **Substance Use History** | Alcohol, Tobacco, Illicit drugs, Prescription misuse | https://www.samhsa.gov |
| **Medical History** | Medical conditions, Medications, Allergies | https://www.psychiatry.org/psychiatrists/practice/telepsychiatry/toolkit/clinical-standards |
| **Family History** | Psychiatric disorders in relatives, Suicide | https://www.psychiatry.org/psychiatrists/practice/telepsychiatry/toolkit/clinical-standards |
| **Social History** | Relationships, Education, Occupation, Living situation, Trauma | https://www.psychiatry.org/psychiatrists/practice/telepsychiatry/toolkit/clinical-standards |

## Mental Status Examination (MSE)
| Domain | Assessment | Examples | URL |
|--------|------------|----------|-----|
| **Appearance** | Grooming, Attire, Eye contact, Posture | Disheveled, Well-groomed, Psychomotor agitation/retardation | https://www.psychiatry.org/psychiatrists/practice/dsm/educational-resources/assessment-measures |
| **Behavior** | Attitude, Cooperation, Activity level | Cooperative, Guarded, Agitated, Catatonic | https://www.psychiatry.org/psychiatrists/practice/dsm/educational-resources/assessment-measures |
| **Speech** | Rate, Rhythm, Volume, Articulation | Pressured, Slowed, Slurred, Monotone | https://www.psychiatry.org/psychiatrists/practice/dsm/educational-resources/assessment-measures |
| **Mood** | Subjective emotional state (patient's report) | "Depressed," "Anxious," "Angry," "Euphoric" | https://www.psychiatry.org/psychiatrists/practice/dsm/educational-resources/assessment-measures |
| **Affect** | Objective emotional expression | Euthymic, Dysthymic, Labile, Flat, Blunted, Inappropriate | https://www.psychiatry.org/psychiatrists/practice/dsm/educational-resources/assessment-measures |
| **Thought Process** | Flow and organization of thoughts | Linear, Circumstantial, Tangential, Flight of ideas, Loose associations, Word salad | https://www.psychiatry.org/psychiatrists/practice/dsm/educational-resources/assessment-measures |
| **Thought Content** | Themes and beliefs | Delusions, Obsessions, Suicidal/Homicidal ideation, Phobias | https://www.psychiatry.org/psychiatrists/practice/dsm/educational-resources/assessment-measures |
| **Perceptions** | Sensory experiences | Hallucinations (auditory, Visual, Olfactory, Tactile), Illusions, Depersonalization/Derealization | https://www.psychiatry.org/psychiatrists/practice/dsm/educational-resources/assessment-measures |
| **Cognition** | Orientation, Memory, Attention, Language | Alert, Oriented x3, Intact/Impaired | https://www.psychiatry.org/psychiatrists/practice/dsm/educational-resources/assessment-measures |
| **Insight** | Awareness of illness | Good, Fair, Poor, Absent | https://www.psychiatry.org/psychiatrists/practice/dsm/educational-resources/assessment-measures |
| **Judgment** | Decision-making capacity | Good, Fair, Poor, Impaired | https://www.psychiatry.org/psychiatrists/practice/dsm/educational-resources/assessment-measures |
