# Psychiatry - Neurodevelopmental Disorders

## Attention-Deficit/Hyperactivity Disorder (ADHD)
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Persistent pattern of inattention and/or hyperactivity-impulsivity**; Onset before **age 12**; Present in **=2 settings**; Duration =6 months | https://www.nimh.nih.gov/health/topics/attention-deficit-hyperactivity-disorder-adhd |
| **Inattention Symptoms** | =6 of (=5 for adults): Careless mistakes, Difficulty sustaining attention, Not listening, Failing to finish tasks, Disorganized, Avoids sustained effort, Loses things, Easily distracted, Forgetful | https://www.nimh.nih.gov/health/topics/attention-deficit-hyperactivity-disorder-adhd |
| **Hyperactivity-Impulsivity** | =6 of (=5 for adults): Fidgets, Leaves seat, Runs/climbs inappropriately, Unable to play quietly, "On the go," Talks excessively, Blurts out, Difficulty waiting turn, Interrupts | https://www.nimh.nih.gov/health/topics/attention-deficit-hyperactivity-disorder-adhd |
| **Presentation** | **Combined**, **Predominantly Inattentive**, **Predominantly Hyperactive-Impulsive** | https://www.nimh.nih.gov/health/topics/attention-deficit-hyperactivity-disorder-adhd |

## ADHD - Treatment
| Modality | Options | URL |
|----------|---------|-----|
| **Stimulants (First-Line)** | **Methylphenidate** (Ritalin, Concerta), **Amphetamine** (Adderall, Vyvanse), **Dexmethylphenidate** (Focalin) | https://www.nimh.nih.gov/health/topics/attention-deficit-hyperactivity-disorder-adhd |
| **Non-Stimulants** | **Atomoxetine**, **Guanfacine ER**, **Clonidine ER**, **Viloxazine**, **Bupropion** | https://www.nimh.nih.gov/health/topics/attention-deficit-hyperactivity-disorder-adhd |
| **Behavioral Therapy** | **Parent training**, **Classroom interventions**, **CBT** (adults) | https://www.nimh.nih.gov/health/topics/attention-deficit-hyperactivity-disorder-adhd |

## Autism Spectrum Disorder (ASD)
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Persistent deficits in social communication and interaction** + **Restricted, Repetitive patterns of behavior, Interests, or activities** | https://www.nimh.nih.gov/health/topics/autism-spectrum-disorders-asd |
| **Social Communication Deficits** | Deficits in **social-emotional reciprocity**, **Nonverbal communication**, **Developing/maintaining relationships** | https://www.nimh.nih.gov/health/topics/autism-spectrum-disorders-asd |
| **Restricted/Repetitive Behaviors** | =2 of: **Stereotyped movements/speech**, **Insistence on sameness**, **Restricted fixated interests**, **Hyper/Hyporeactivity to sensory input** | https://www.nimh.nih.gov/health/topics/autism-spectrum-disorders-asd |
| **Treatment** | **Applied Behavior Analysis (ABA)** , **Speech therapy**, **Occupational therapy**, **Social skills training**, **Risperidone/Aripiprazole** (irritability), **Treat comorbidities** | https://www.nimh.nih.gov/health/topics/autism-spectrum-disorders-asd |

## Intellectual Disability (ID)
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Deficits in intellectual functioning** (reasoning, Problem-solving, Learning) + **Deficits in adaptive functioning** + **Onset during developmental period** | https://www.psychiatry.org/patients-families/intellectual-disability |
| **Severity** | **Mild** (85%), **Moderate** (10%), **Severe** (3-4%), **Profound** (1-2%) | https://www.psychiatry.org/patients-families/intellectual-disability |
| **Treatment** | **Special education**, **Behavioral interventions**, **Family support**, **Treat comorbid psychiatric disorders** | https://www.psychiatry.org/patients-families/intellectual-disability |

## Specific Learning Disorder
| Component | Description | URL |
