# Psychiatry - Geriatric Psychiatry

## Late-Life Depression
| Component | Description | URL |
|-----------|-------------|-----|
| **Atypical Presentation** | **Somatic complaints**, **Cognitive complaints** ("pseudodementia"), **Anhedonia**, **Irritability**, **Social withdrawal** | https://www.nimh.nih.gov/health/topics/older-adults-and-mental-health |
| **Screening** | **Geriatric Depression Scale (GDS)** , **PHQ-9** | https://www.nimh.nih.gov/health/topics/older-adults-and-mental-health |
| **Treatment** | **SSRIs** (sertraline, Escitalopram - start low, Go slow), **CBT**, **Problem-solving therapy**, **ECT** (severe, Psychotic, Catatonic) | https://www.nimh.nih.gov/health/topics/older-adults-and-mental-health |

## Late-Life Anxiety
| Component | Description | URL |
|-----------|-------------|-----|
| **Common Disorders** | **GAD**, **Specific phobia** (fear of falling), **Agoraphobia** | https://www.nimh.nih.gov/health/topics/older-adults-and-mental-health |
| **Treatment** | **SSRIs**, **CBT**, **Avoid benzodiazepines** (falls, Cognitive impairment) | https://www.nimh.nih.gov/health/topics/older-adults-and-mental-health |

## Late-Life Psychosis
| Component | Description | URL |
|-----------|-------------|-----|
| **Causes** | **Dementia** (Alzheimer, Lewy body), **Delirium**, **Schizophrenia**, **Delusional disorder**, **Parkinson disease** | https://www.nimh.nih.gov/health/topics/older-adults-and-mental-health |
| **Treatment** | **Treat underlying cause**, **Low-dose atypical antipsychotics** (quetiapine, Risperidone - **caution: increased mortality in dementia**), **Pimavanserin** (Parkinson psychosis) | https://www.nimh.nih.gov/health/topics/older-adults-and-mental-health |
