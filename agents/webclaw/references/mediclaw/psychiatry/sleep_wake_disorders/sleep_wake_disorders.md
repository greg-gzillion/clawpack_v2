# Psychiatry - Sleep-Wake Disorders

## Insomnia Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Difficulty initiating or maintaining sleep** or **early morning awakening** for **=3 nights/week for =3 months** + **Daytime impairment** | https://www.nhlbi.nih.gov/health/insomnia |
| **Treatment (First-Line)** | **CBT-I** (Cognitive Behavioral Therapy for Insomnia): Sleep restriction, Stimulus control, Cognitive therapy, Relaxation, Sleep hygiene | https://www.nhlbi.nih.gov/health/insomnia/treatment |
| **Pharmacotherapy** | **Z-drugs** (zolpidem, Eszopiclone, Zaleplon), **Benzodiazepines** (temazepam), **Melatonin agonists** (ramelteon), **Orexin antagonists** (suvorexant, Lemborexant, Daridorexant), **Doxepin** (low-dose), **Trazodone** | https://www.nhlbi.nih.gov/health/insomnia/treatment |

## Hypersomnolence Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Excessive sleepiness** despite **=7 hours** of sleep for **=3 months** + **Daytime impairment** | https://www.ninds.nih.gov/health-information/disorders/hypersomnia |
| **Treatment** | **Modafinil**, **Armodafinil**, **Solriamfetol**, **Pitolisant**, **Traditional stimulants** (methylphenidate, Amphetamines) | https://www.ninds.nih.gov/health-information/disorders/hypersomnia |

## Narcolepsy
| Component | Description | URL |
|-----------|-------------|-----|
| **Type 1 (with Cataplexy)** | **Excessive daytime sleepiness** + **Cataplexy** + **Low/absent CSF hypocretin-1** | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Type 2 (without Cataplexy)** | Excessive daytime sleepiness, Normal CSF hypocretin-1 | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Treatment (Sleepiness)** | **Modafinil**, **Armodafinil**, **Solriamfetol**, **Pitolisant**, **Traditional stimulants** | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Treatment (Cataplexy)** | **Sodium oxybate**, **Pitolisant**, **Venlafaxine**, **Fluoxetine**, **Clomipramine** | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |

## Circadian Rhythm Sleep-Wake Disorders
| Disorder | Description | Treatment | URL |
|----------|-------------|-----------|-----|
| **Delayed Sleep Phase** | Late sleep onset/wake | **Melatonin** (evening), **Light therapy** (morning) | https://www.nhlbi.nih.gov/health/circadian-rhythm-disorders |
| **Advanced Sleep Phase** | Early sleep onset/wake | **Light therapy** (evening) | https://www.nhlbi.nih.gov/health/circadian-rhythm-disorders |
| **Shift Work Disorder** | Sleep disruption with night shifts | **Modafinil**, **Armodafinil**, **Caffeine**, **Melatonin** (before daytime sleep) | https://www.nhlbi.nih.gov/health/circadian-rhythm-disorders |
