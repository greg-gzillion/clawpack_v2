# Psychiatry - Substance Use Disorders

## Substance Use Disorder - General Criteria
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **=2 of 11 criteria** within 12 months: **Impaired control**, **Social impairment**, **Risky use**, **Pharmacologic** (tolerance, Withdrawal) | https://www.nimh.nih.gov/health/topics/substance-use-and-mental-health |
| **Severity** | **Mild** (2-3), **Moderate** (4-5), **Severe** (=6) | https://www.nimh.nih.gov/health/topics/substance-use-and-mental-health |

## Alcohol Use Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **Screening** | **AUDIT-C**, **CAGE** | https://www.aaap.org/clinical-resources/audit-c/ |
| **Withdrawal** | **Tremor**, **Anxiety**, **Diaphoresis**, **Tachycardia**, **Hypertension**, **Seizures** (24-48h), **Delirium tremens** (48-96h - confusion, Hallucinations, Autonomic instability) | https://www.niaaa.nih.gov/publications/clinical-guides-and-manuals |
| **Withdrawal Treatment** | **Benzodiazepines** (CIWA protocol: lorazepam, Diazepam, Chlordiazepoxide), **Thiamine**, **Folate**, **IV fluids**, **Electrolyte replacement** | https://www.niaaa.nih.gov/publications/clinical-guides-and-manuals |
| **Maintenance Treatment** | **Naltrexone** (oral or IM), **Acamprosate**, **Disulfiram**, **Topiramate**, **Gabapentin** | https://www.niaaa.nih.gov/publications/clinical-guides-and-manuals |

## Opioid Use Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **Withdrawal** | **Dysphoria**, **Nausea/Vomiting**, **Diarrhea**, **Myalgias**, **Lacrimation**, **Rhinorrhea**, **Yawning**, **Dilated pupils**, **Piloerection**, **Insomnia** | https://www.samhsa.gov/medications-substance-use-disorders |
| **Withdrawal Treatment** | **Clonidine**, **Lofexidine**, **Buprenorphine**, **Methadone**, **Symptomatic treatment** | https://www.samhsa.gov/medications-substance-use-disorders |
| **Maintenance Treatment** | **Buprenorphine-naloxone** (Suboxone), **Methadone**, **Naltrexone** (oral or IM) | https://www.samhsa.gov/medications-substance-use-disorders |
| **Overdose Reversal** | **Naloxone** (Narcan) | https://www.cdc.gov/stop-overdose/index.html |

## Stimulant Use Disorder (Cocaine, Methamphetamine)
| Component | Description | URL |
|-----------|-------------|-----|
| **Withdrawal** | **Dysphoria**, **Fatigue**, **Vivid dreams**, **Insomnia/Hypersomnia**, **Increased appetite**, **Psychomotor agitation/retardation** | https://www.samhsa.gov/find-help/atod/stimulants |
| **Treatment** | **Contingency management**, **CBT**, **Matrix Model**; **No FDA-approved medications** | https://www.samhsa.gov/find-help/atod/stimulants |
