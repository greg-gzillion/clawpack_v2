# Psychiatry - Sexual and Gender Disorders

## Gender Dysphoria
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Marked incongruence between experienced/expressed gender and assigned gender** for **=6 months** + **Distress or impairment** | https://www.psychiatry.org/patients-families/gender-dysphoria |
| **Treatment** | **Psychotherapy**, **Gender-affirming hormone therapy**, **Gender-affirming surgery**, **Social transition** | https://www.psychiatry.org/patients-families/gender-dysphoria |

## Sexual Dysfunctions
| Disorder | Description | Treatment | URL |
|----------|-------------|-----------|-----|
| **Erectile Disorder** | **Difficulty obtaining or maintaining erection** | **PDE5 inhibitors** (sildenafil, Tadalafil), **Psychotherapy**, **Treat underlying medical/psychiatric conditions** | https://www.niddk.nih.gov/health-information/urologic-diseases/erectile-dysfunction |
| **Female Sexual Interest/Arousal Disorder** | **Lack of/reduced sexual interest/arousal** | **Flibanserin**, **Bremelanotide**, **CBT**, **Treat underlying conditions** | https://www.womenshealth.gov/a-z-topics/sexual-health |
| **Delayed Ejaculation** | **Marked delay or inability to achieve ejaculation** | **Psychotherapy**, **Medication adjustment** (if SSRI-induced) | https://www.niddk.nih.gov/health-information/urologic-diseases/ejaculation-problems |
| **Premature Ejaculation** | **Ejaculation within 1 minute** of penetration, Before desired | **SSRIs** (paroxetine, Sertraline), **Clomipramine**, **Topical anesthetics**, **Behavioral techniques** (stop-start, Squeeze) | https://www.niddk.nih.gov/health-information/urologic-diseases/ejaculation-problems |
| **Genito-Pelvic Pain/Penetration Disorder** | **Pain with penetration**, **Fear/anxiety**, **Pelvic floor tension** | **Pelvic floor physical therapy**, **CBT**, **Vaginal dilators**, **Treat underlying conditions** | https://www.womenshealth.gov/a-z-topics/sexual-health |

## Paraphilic Disorders
| Disorder | Description | URL |
|----------|-------------|-----|
| **General Criteria** | **Recurrent intense sexually arousing fantasies/urges/behaviors** involving non-consenting persons, Suffering/humiliation, or non-human objects for **=6 months** + **Acted on or cause distress/impairment** | https://www.psychiatry.org/patients-families/paraphilic-disorders |
| **Types** | **Voyeuristic**, **Exhibitionistic**, **Frotteuristic**, **Sexual masochism**, **Sexual sadism**, **Pedophilic**, **Fetishistic**, **Transvestic** | https://www.psychiatry.org/patients-families/paraphilic-disorders |
| **Treatment** | **CBT**, **SSRIs**, **Antiandrogens** (medroxyprogesterone, Cyproterone), **GnRH agonists** (leuprolide) | https://www.psychiatry.org/patients-families/paraphilic-disorders |
