# Psychiatry - Feeding and Eating Disorders

## Anorexia Nervosa
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Restriction of energy intake** leading to **significantly low body weight**; **Intense fear of gaining weight**; **Disturbance in self-perceived weight/shape** | https://www.nimh.nih.gov/health/topics/eating-disorders |
| **Subtypes** | **Restricting type** (no binge/purge), **Binge-eating/purging type** | https://www.nimh.nih.gov/health/topics/eating-disorders |
| **Severity** | Based on BMI: **Mild** (=17), **Moderate** (16-16.99), **Severe** (15-15.99), **Extreme** (<15) | https://www.nimh.nih.gov/health/topics/eating-disorders |
| **Medical Complications** | **Bradycardia**, **Hypotension**, **Hypothermia**, **Amenorrhea**, **Osteoporosis**, **Lanugo**, **Electrolyte abnormalities**, **Cardiac arrhythmias** | https://www.nimh.nih.gov/health/topics/eating-disorders |
| **Treatment** | **Nutritional rehabilitation**, **Family-Based Treatment (FBT)** for adolescents, **CBT-E**, **SSRIs** (limited efficacy, Treat comorbidities) | https://www.nimh.nih.gov/health/topics/eating-disorders |

## Bulimia Nervosa
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Recurrent binge eating** + **Inappropriate compensatory behaviors** (vomiting, Laxatives, Diuretics, Fasting, Excessive exercise) for **=1x/week for 3 months**; **Self-evaluation unduly influenced by weight/shape** | https://www.nimh.nih.gov/health/topics/eating-disorders |
| **Medical Complications** | **Electrolyte abnormalities** (hypokalemia, Hypochloremia, Metabolic alkalosis), **Dental erosion**, **Parotid gland enlargement**, **Esophageal tears**, **Russell sign** (calluses on knuckles) | https://www.nimh.nih.gov/health/topics/eating-disorders |
| **Treatment** | **CBT-E** (first-line), **Fluoxetine** (FDA-approved, 60 mg), **IPT**, **Nutritional counseling** | https://www.nimh.nih.gov/health/topics/eating-disorders |

## Binge-Eating Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Recurrent binge eating** (=1x/week for 3 months) **without compensatory behaviors** + =3 of: Eating rapidly, Uncomfortably full, Eating when not hungry, Eating alone due to embarrassment, Feeling disgusted/depressed/guilty | https://www.nimh.nih.gov/health/topics/eating-disorders |
| **Treatment** | **CBT**, **IPT**, **Lisdexamfetamine** (FDA-approved), **SSRIs**, **Topiramate** | https://www.nimh.nih.gov/health/topics/eating-disorders |

## Avoidant/Restrictive Food Intake Disorder (ARFID)
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Eating disturbance** leading to **weight loss, Nutritional deficiency, Dependence on enteral feeding, or Psychosocial impairment**; **Not due to body image disturbance** | https://www.nimh.nih.gov/health/topics/eating-disorders |
| **Treatment** | **CBT-AR**, **Family-based treatment**, **Nutritional rehabilitation** | https://www.nimh.nih.gov/health/topics/eating-disorders |
