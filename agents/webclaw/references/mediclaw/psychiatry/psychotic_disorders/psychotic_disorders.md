# Psychiatry - Psychotic Disorders

## Schizophrenia
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **=2 of the following for =1 month** (with at least one being delusions, Hallucinations, or Disorganized speech): **Delusions**, **Hallucinations**, **Disorganized speech**, **Grossly disorganized or catatonic behavior**, **Negative symptoms**; **Continuous signs for =6 months** | https://www.nimh.nih.gov/health/topics/schizophrenia |
| **Positive Symptoms** | **Delusions** (persecutory, Referential, Grandiose, Somatic, Control), **Hallucinations** (auditory most common), **Disorganized thinking/speech** | https://www.nimh.nih.gov/health/topics/schizophrenia |
| **Negative Symptoms** | **Diminished emotional expression**, **Avolition**, **Alogia**, **Anhedonia**, **Asociality** | https://www.nimh.nih.gov/health/topics/schizophrenia |
| **Cognitive Symptoms** | Impaired attention, Executive function, Working memory, Verbal learning | https://www.nimh.nih.gov/health/topics/schizophrenia |

## Schizophrenia - Treatment
| Modality | Options | URL |
|----------|---------|-----|
| **Antipsychotics (First-Line)** | **Risperidone**, **Olanzapine**, **Quetiapine**, **Aripiprazole**, **Paliperidone**, **Lurasidone**, **Brexpiprazole**, **Cariprazine**, **Asenapine**, **Iloperidone** | https://www.nimh.nih.gov/health/topics/mental-health-medications |
| **Antipsychotics (Second-Line)** | **Clozapine** (treatment-resistant), **Haloperidol**, **Perphenazine**, **Fluphenazine**, **Chlorpromazine** | https://www.nimh.nih.gov/health/topics/mental-health-medications |
| **Long-Acting Injectables (LAIs)** | **Paliperidone palmitate**, **Aripiprazole lauroxil**, **Risperidone microspheres**, **Haloperidol decanoate**, **Fluphenazine decanoate** | https://www.nimh.nih.gov/health/topics/mental-health-medications |
| **Psychosocial Interventions** | **CBT for psychosis (CBTp)** , **Social skills training**, **Family psychoeducation**, **Supported employment**, **Assertive community treatment (ACT)** | https://www.nimh.nih.gov/health/topics/schizophrenia |

## Schizoaffective Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Uninterrupted period of illness** with **major mood episode** + **Criterion A of schizophrenia**; **Delusions or hallucinations for =2 weeks** in absence of major mood episode | https://www.nimh.nih.gov/health/topics/schizophrenia |
| **Treatment** | **Antipsychotics** + **Mood stabilizer** or **Antidepressant**; **Paliperidone** FDA-approved | https://www.nimh.nih.gov/health/topics/schizophrenia |

## Schizophreniform Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | Meets criteria for schizophrenia but **duration 1-6 months** | https://www.nimh.nih.gov/health/topics/schizophrenia |
| **Treatment** | Same as schizophrenia; **1/3 remit**, **2/3 progress to schizophrenia/schizoaffective** | https://www.nimh.nih.gov/health/topics/schizophrenia |

## Brief Psychotic Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **=1 psychotic symptom** (delusions, Hallucinations, Disorganized speech, Grossly disorganized/catatonic behavior) lasting **>1 day but <1 month**; **Full return to baseline** | https://www.nimh.nih.gov/health/topics/schizophrenia |
| **Treatment** | **Antipsychotics** (short-term), **Benzodiazepines**, **Supportive therapy** | https://www.nimh.nih.gov/health/topics/schizophrenia |

## Delusional Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **=1 delusion for =1 month**; Never met Criterion A for schizophrenia; **Functioning not markedly impaired** | https://www.nimh.nih.gov/health/topics/schizophrenia |
| **Types** | **Erotomanic**, **Grandiose**, **Jealous**, **Persecutory**, **Somatic**, **Mixed**, **Unspecified** | https://www.nimh.nih.gov/health/topics/schizophrenia |
| **Treatment** | **Antipsychotics**, **CBT** | https://www.nimh.nih.gov/health/topics/schizophrenia |

## Antipsychotic Adverse Effects
| Effect | High-Risk Agents | Management | URL |
|--------|------------------|------------|-----|
| **Extrapyramidal Symptoms (EPS)** | Haloperidol, Fluphenazine, Risperidone, Paliperidone | **Anticholinergics** (benztropine), **Amantadine**, **Switch to SGA** | https://www.nimh.nih.gov/health/topics/mental-health-medications |
| **Tardive Dyskinesia (TD)** | All antipsychotics (FGA > SGA) | **Valbenazine**, **Deutetrabenazine**, **Switch to clozapine** | https://www.ninds.nih.gov/health-information/disorders/tardive-dyskinesia |
| **Metabolic Syndrome** | Olanzapine, Clozapine, Quetiapine, Risperidone | **Monitor weight**, **Glucose**, **Lipids**, **Lifestyle**, **Switch to lower-risk agent** (lurasidone, Aripiprazole) | https://www.nimh.nih.gov/health/topics/mental-health-medications |
| **Hyperprolactinemia** | Risperidone, Paliperidone, Haloperidol | **Switch to aripiprazole**, **Add aripiprazole** | https://www.nimh.nih.gov/health/topics/mental-health-medications |
| **QTc Prolongation** | Haloperidol, Pimozide, Ziprasidone, Iloperidone | **Baseline and periodic ECG**, **Avoid other QT-prolonging drugs** | https://www.nimh.nih.gov/health/topics/mental-health-medications |
| **Neuroleptic Malignant Syndrome (NMS)** | All antipsychotics | **Stop antipsychotic**, **Supportive care**, **Dantrolene**, **Bromocriptine** | https://www.ninds.nih.gov/health-information/disorders/neuroleptic-malignant-syndrome |
| **Agranulocytosis** | **Clozapine** (1-2%) | **Weekly CBC x 6 months**, Then **biweekly x 6 months**, Then **monthly** | https://www.fda.gov/drugs |
