# Psychiatry - Consultation-Liaison Psychiatry

## Common Consult Questions
| Question | Approach | URL |
|----------|----------|-----|
| **Capacity Evaluation** | Assess ability to **communicate choice**, **Understand information**, **Appreciate situation**, **Reason about options** | https://www.psychiatry.org/psychiatrists/practice/medical-ethics |
| **Depression in Medically Ill** | **Treat underlying medical condition**, **SSRIs** (watch for drug interactions), **CBT**, **Avoid medications that worsen medical condition** | https://www.psychiatry.org/psychiatrists/practice/professional-interests/consultation-liaison |
| **Delirium** | **Identify and treat underlying cause**, **Non-pharmacologic interventions**, **Low-dose antipsychotics** if severe agitation | https://www.psychiatry.org/psychiatrists/practice/professional-interests/consultation-liaison |
| **Catatonia** | **Lorazepam challenge**, **ECT** | https://www.psychiatry.org/psychiatrists/practice/professional-interests/consultation-liaison |
| **Adjustment Disorder** | **Supportive psychotherapy**, **Treat symptoms**, **Avoid unnecessary medications** | https://www.psychiatry.org/psychiatrists/practice/professional-interests/consultation-liaison |

|-----------------|---------------------------|-----|

| **Multiple Sclerosis** | **Depression**, **Anxiety**, **Cognitive impairment**, **Pseudobulbar affect** | https://www.ninds.nih.gov/health-information/disorders/multiple-sclerosis |
| **Parkinson Disease** | **Depression**, **Anxiety**, **Apathy**, **Psychosis**, **Impulse control disorders** | https://www.ninds.nih.gov/health-information/disorders/parkinsons-disease |
| **HIV/AIDS** | **Depression**, **Anxiety**, **Cognitive impairment** (HAND), **Mania** | https://www.hiv.gov |
