# Psychiatry - Mood Disorders (Depressive Disorders)

## Major Depressive Disorder (MDD)
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **=5 of SIGECAPS** for =2 weeks, with at least one being **depressed mood** or **anhedonia**: **S**leep, **I**nterest, **G**uilt, **E**nergy, **C**oncentration, **A**ppetite, **P**sychomotor, **S**uicidality | https://www.nimh.nih.gov/health/topics/depression |
| **Epidemiology** | Lifetime prevalence 15-20%; Female > Male (2:1); Mean onset 30s | https://www.nimh.nih.gov/health/statistics/major-depression |
| **Specifiers** | Anxious distress, Mixed features, Melancholic, Atypical, Psychotic, Catatonic, Peripartum, Seasonal | https://www.psychiatry.org/psychiatrists/practice/dsm |

## MDD - Treatment
| Modality | Options | URL |
|----------|---------|-----|
| **Psychotherapy** | **CBT**, **IPT**, Behavioral activation, Problem-solving therapy | https://www.nimh.nih.gov/health/topics/psychotherapies |
| **Pharmacotherapy (First-Line)** | **SSRIs** (sertraline, Escitalopram, Fluoxetine, Paroxetine, Citalopram), **SNRIs** (venlafaxine, Duloxetine), **Bupropion**, **Mirtazapine** | https://www.nimh.nih.gov/health/topics/mental-health-medications |
| **Pharmacotherapy (Second-Line)** | **TCAs** (nortriptyline, Amitriptyline), **MAOIs** (phenelzine, Tranylcypromine), **Vortioxetine**, **Vilazodone** | https://www.nimh.nih.gov/health/topics/mental-health-medications |
| **Neuromodulation** | **ECT** (severe, Psychotic, Catatonic, Treatment-resistant), **TMS**, **VNS** | https://www.nimh.nih.gov/health/topics/brain-stimulation-therapies |
| **Adjunctive** | **Atypical antipsychotics** (aripiprazole, Brexpiprazole, Quetiapine, Olanzapine), **Lithium**, **Thyroid hormone**, **Ketamine/Esketamine** | https://www.nimh.nih.gov/health/topics/mental-health-medications |

## Persistent Depressive Disorder (Dysthymia)
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Depressed mood for =2 years** (1 year in children/adolescents) + =2 of: Appetite changes, Sleep changes, Low energy, Low self-esteem, Poor concentration, Hopelessness | https://www.nimh.nih.gov/health/topics/persistent-depressive-disorder-dysthymia |
| **Treatment** | **SSRIs**, **CBT**, **Combination therapy** | https://www.nimh.nih.gov/health/topics/persistent-depressive-disorder-dysthymia |

## Premenstrual Dysphoric Disorder (PMDD)
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Mood symptoms** (lability, Irritability, Depression, Anxiety) + **Physical/behavioral symptoms** in **luteal phase**, Resolving with menses | https://www.womenshealth.gov/menstrual-cycle/premenstrual-syndrome/premenstrual-dysphoric-disorder-pmdd |
| **Treatment** | **SSRIs** (continuous or luteal phase), **OCPs** (drospirenone/ethinyl estradiol), **CBT**, **Calcium**, **Vitamin B6** | https://www.womenshealth.gov/menstrual-cycle/premenstrual-syndrome/premenstrual-dysphoric-disorder-pmdd |

## Disruptive Mood Dysregulation Disorder (DMDD)
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Severe recurrent temper outbursts** + **Persistently irritable/angry mood** between outbursts; Onset <10 years; Duration =12 months | https://www.nimh.nih.gov/health/topics/disruptive-mood-dysregulation-disorder-dmdd |
| **Treatment** | **CBT**, **Parent training**, **SSRIs**, **Stimulants** (if comorbid ADHD) | https://www.nimh.nih.gov/health/topics/disruptive-mood-dysregulation-disorder-dmdd |
