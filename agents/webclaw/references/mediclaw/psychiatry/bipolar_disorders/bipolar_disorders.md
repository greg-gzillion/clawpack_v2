# Psychiatry - Bipolar and Related Disorders

## Bipolar I Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **=1 manic episode** (may be preceded or followed by hypomanic or major depressive episodes) | https://www.nimh.nih.gov/health/topics/bipolar-disorder |
| **Manic Episode** | **Distinct period of abnormally and persistently elevated, Expansive, or irritable mood** + **Increased goal-directed activity/energy** for **=1 week** + =3 of **DIGFAST** (4 if irritable): **D**istractibility, **I**mpulsivity, **G**randiosity, **F**light of ideas, **A**ctivity increase, **S**leep decreased, **T**alkativeness | https://www.nimh.nih.gov/health/topics/bipolar-disorder |

## Bipolar II Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **=1 hypomanic episode** + **=1 major depressive episode**; **Never a manic episode** | https://www.nimh.nih.gov/health/topics/bipolar-disorder |
| **Hypomanic Episode** | Same symptoms as mania but **duration =4 days**, **Not severe enough to cause marked impairment or psychosis**, **No hospitalization** | https://www.nimh.nih.gov/health/topics/bipolar-disorder |

## Cyclothymic Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Numerous periods of hypomanic and depressive symptoms** for **=2 years** (1 year in children/adolescents); **Never met criteria for manic, Hypomanic, or major depressive episode** | https://www.nimh.nih.gov/health/topics/bipolar-disorder |

## Bipolar Disorder - Treatment
| Phase | Options | URL |
|-------|---------|-----|
| **Acute Mania** | **Mood stabilizers** (lithium, Valproate, Carbamazepine), **Atypical antipsychotics** (olanzapine, Risperidone, Quetiapine, Aripiprazole, Cariprazine, Asenapine), **Combination therapy** | https://www.nimh.nih.gov/health/topics/bipolar-disorder |
| **Acute Depression** | **Lurasidone**, **Quetiapine**, **Cariprazine**, **Olanzapine-fluoxetine combination**, **Lamotrigine**, **Lithium**, **Valproate** | https://www.nimh.nih.gov/health/topics/bipolar-disorder |
| **Maintenance** | **Lithium** (gold standard - reduces suicide), **Valproate**, **Lamotrigine** (depression-predominant), **Atypical antipsychotics** | https://www.nimh.nih.gov/health/topics/bipolar-disorder |

## Lithium
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | **Bipolar disorder** (acute mania, Maintenance, Depression augmentation), **Schizoaffective disorder**, **Treatment-resistant depression** | https://www.nimh.nih.gov/health/topics/mental-health-medications |
| **Monitoring** | **Lithium level** (0.6-1.2 mEq/L acute, 0.6-0.8 mEq/L maintenance), **TSH**, **BUN/Cr**, **Electrolytes**, **Pregnancy test** | https://www.nimh.nih.gov/health/topics/mental-health-medications |
| **Adverse Effects** | **Tremor**, **Polyuria/Polydipsia** (nephrogenic DI), **Hypothyroidism**, **Weight gain**, **Acne**, **Teratogenicity** (Ebstein anomaly) | https://www.nimh.nih.gov/health/topics/mental-health-medications |
| **Toxicity** | **>1.5 mEq/L**: Nausea, Vomiting, Diarrhea, Ataxia, Slurred speech, Confusion; **>2.5 mEq/L**: Seizures, Coma, Death | https://www.nimh.nih.gov/health/topics/mental-health-medications |

## Valproate
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | **Acute mania**, **Maintenance**, **Epilepsy**, **Migraine prophylaxis** | https://www.nimh.nih.gov/health/topics/mental-health-medications |
| **Monitoring** | **Valproate level** (50-125 mcg/mL), **CBC** (thrombocytopenia), **LFTs**, **Ammonia**, **Pregnancy test** | https://www.nimh.nih.gov/health/topics/mental-health-medications |
| **Adverse Effects** | **Weight gain**, **Tremor**, **Thrombocytopenia**, **Hepatotoxicity**, **Pancreatitis**, **Teratogenicity** (neural tube defects), **PCOS** | https://www.nimh.nih.gov/health/topics/mental-health-medications |
