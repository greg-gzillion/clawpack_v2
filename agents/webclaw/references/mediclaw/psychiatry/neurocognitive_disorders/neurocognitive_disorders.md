# Psychiatry - Neurocognitive Disorders

## Delirium
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Acute disturbance in attention and awareness** + **Change in cognition** + **Fluctuating course** + **Evidence of medical cause** | https://www.nimh.nih.gov/health/topics/delirium |
| **Subtypes** | **Hyperactive** (agitated), **Hypoactive** (lethargic - most common, Often missed), **Mixed** | https://www.nimh.nih.gov/health/topics/delirium |
| **Causes (DELIRIUM)** | **D**rugs, **E**lectrolytes, **L**ack of drugs (withdrawal), **I**nfection, **R**educed sensory input, **I**ntracranial, **U**rinary/Fecal, **M**yocardial/Pulmonary | https://www.nimh.nih.gov/health/topics/delirium |
| **Treatment** | **Treat underlying cause**, **Non-pharmacologic** (orientation, Sleep hygiene, Sensory optimization), **Low-dose antipsychotics** (haloperidol, Quetiapine) if severe agitation | https://www.nimh.nih.gov/health/topics/delirium |

## Major Neurocognitive Disorder (Dementia)
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnostic Criteria** | **Significant cognitive decline** in =1 domain + **Interference with independence** + **Not due to delirium** | https://www.nia.nih.gov/health/alzheimers-and-dementia |
| **Etiologies** | **Alzheimer disease**, **Vascular**, **Lewy body**, **Frontotemporal**, **Parkinson disease**, **Huntington disease**, **Prion**, **HIV**, **TBI** | https://www.nia.nih.gov/health/alzheimers-and-dementia |

## Alzheimer Disease
| Component | Description | URL |
|-----------|-------------|-----|
| **Clinical Features** | **Progressive episodic memory loss**, **Anosognosia**, **Visuospatial deficits**, **Language impairment**, **Executive dysfunction** | https://www.nia.nih.gov/health/alzheimers-disease |
| **Treatment** | **Cholinesterase inhibitors** (donepezil, Rivastigmine, Galantamine), **Memantine**, **Anti-amyloid antibodies** (lecanemab, Donanemab) | https://www.nia.nih.gov/health/alzheimers-disease/treatment |

## Vascular Dementia
| Component | Description | URL |
|-----------|-------------|-----|
| **Clinical Features** | **Executive dysfunction**, **Psychomotor slowing**, **Gait disturbance**, **Urinary symptoms**, Focal neurologic deficits | https://www.nia.nih.gov/health/vascular-dementia |
| **Treatment** | **Risk factor control** (BP, Lipids, Glucose), **Antiplatelet therapy**, **Cholinesterase inhibitors** (modest benefit) | https://www.nia.nih.gov/health/vascular-dementia |

## Dementia with Lewy Bodies
| Component | Description | URL |
|-----------|-------------|-----|
| **Core Features** | **Fluctuating cognition**, **Recurrent visual hallucinations**, **REM sleep behavior disorder**, **Parkinsonism** | https://www.nia.nih.gov/health/lewy-body-dementia |
| **Treatment** | **Cholinesterase inhibitors** (rivastigmine), **AVOID antipsychotics** (neuroleptic sensitivity), **Treat RBD** (melatonin, Clonazepam) | https://www.nia.nih.gov/health/lewy-body-dementia |

## Frontotemporal Dementia
| Component | Description | URL |
|-----------|-------------|-----|
| **Behavioral Variant** | **Disinhibition**, **Apathy**, **Loss of empathy**, **Compulsive behaviors**, **Hyperorality** | https://www.nia.nih.gov/health/frontotemporal-dementia |
| **Language Variants** | **Semantic variant PPA** (loss of word meaning), **Nonfluent/Agrammatic variant PPA** (effortful speech) | https://www.nia.nih.gov/health/frontotemporal-dementia |
| **Treatment** | **SSRIs** (behavioral symptoms), **Avoid cholinesterase inhibitors** (may worsen) | https://www.nia.nih.gov/health/frontotemporal-dementia |
