# Psychiatry - Emergency Psychiatry

## Suicide Risk Assessment
| Component | Description | URL |
|-----------|-------------|-----|
| **Risk Factors** | **Prior attempts** (strongest), **Psychiatric illness** (depression, Bipolar, Schizophrenia, Substance use), **Hopelessness**, **Family history**, **Medical illness**, **Recent loss**, **Access to means**, **Male**, **Older age** | https://www.nimh.nih.gov/health/topics/suicide-prevention |
| **Assessment** | **ASK directly**: "Are you thinking about suicide?" **C-SSRS** (Columbia Suicide Severity Rating Scale) | https://www.cdc.gov/suicide/pdf/ssrs_clinicalcare.pdf |
| **Management** | **Safety planning**, **Means restriction**, **Treat underlying psychiatric illness**, **Consider hospitalization** | https://www.nimh.nih.gov/health/topics/suicide-prevention |

## Psychiatric Emergencies
| Emergency | Management | URL |
|-----------|------------|-----|
| **Agitation** | **Verbal de-escalation**, **Offer PO medications**, **IM medications** (midazolam, Lorazepam, Haloperidol, Olanzapine, Ziprasidone), **Physical restraints** (last resort, Continuous monitoring) | https://www.samhsa.gov/resource/dbhis/agitation-management |
| **Catatonia** | **Lorazepam challenge** (1-2 mg IV/IM), **ECT** (if benzodiazepine-refractory) | https://www.nimh.nih.gov/health/topics/catatonia |
| **Neuroleptic Malignant Syndrome (NMS)** | **Stop antipsychotic**, **Supportive care** (hydration, Cooling), **Dantrolene**, **Bromocriptine** | https://www.ninds.nih.gov/health-information/disorders/neuroleptic-malignant-syndrome |
| **Serotonin Syndrome** | **Stop serotonergic agents**, **Supportive care**, **Benzodiazepines**, **Cyproheptadine** | https://www.nimh.nih.gov/health/topics/mental-health-medications |
| **Acute Dystonia** | **Benztropine** 1-2 mg IM/IV or **Diphenhydramine** 25-50 mg IM/IV | https://www.nimh.nih.gov/health/topics/mental-health-medications |

## Involuntary Psychiatric Holds
| Criteria | Description | URL |
|----------|-------------|-----|
| **Danger to Self** | **Suicidal ideation** with plan/intent, **Self-harm** | Varies by state |
| **Danger to Others** | **Homicidal ideation**, **Threats**, **Aggressive behavior** | Varies by state |
| **Grave Disability** | **Inability to provide for basic needs** (food, Shelter, Clothing) due to mental illness | Varies by state |
