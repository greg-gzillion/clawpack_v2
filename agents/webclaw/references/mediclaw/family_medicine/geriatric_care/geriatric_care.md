# Family Medicine - Geriatric Care

## Comprehensive Geriatric Assessment
| Domain | Assessment Tools | URL |
|--------|-----------------|-----|
| **Functional Status** | **ADLs** (bathing, dressing, toileting, transferring, continence, feeding), **IADLs** (shopping, cooking, finances, medications, phone, transportation) | https://www.americangeriatrics.org |
| **Cognition** | **Mini-Cog**, **MMSE**, **MoCA**, **SLUMS** | https://www.alz.org |
| **Mood** | **PHQ-9**, **GDS** (Geriatric Depression Scale) | https://www.americangeriatrics.org |
| **Falls Risk** | **Timed Up and Go (TUG)** , **STEADI** algorithm | https://www.cdc.gov/steadi |
| **Nutrition** | **Mini Nutritional Assessment (MNA)** , Weight loss (>5% in 1 month or >10% in 6 months) | https://www.americangeriatrics.org |
| **Polypharmacy** | **Medication review**, **Beers Criteria**, **STOPP/START** | https://dcri.org/beers-criteria/ |
| **Social Support** | Living situation, Caregiver availability, Finances | https://www.americangeriatrics.org |
| **Advance Care Planning** | Code status, Healthcare proxy, Living will | https://www.nia.nih.gov/health/advance-care-planning |

## Cognitive Assessment
| Tool | Time | Score Interpretation | URL |
|------|------|---------------------|-----|
| **Mini-Cog** | 3 min | **=2/5 = Possible impairment** | https://mini-cog.com |
| **MMSE** | 10 min | **<24 = Abnormal** (education-adjusted) | https://www.parinc.com/Products/Pkey/237 |
| **MoCA** | 10-15 min | **<26 = Abnormal** | https://www.mocatest.org |
| **SLUMS** | 7-10 min | **<27 (high school), <25 (less than high school)** | https://www.slu.edu/medicine/internal-medicine/geriatric-medicine/aging-successfully/assessment-tools/mental-status-exam.php |

## Falls Prevention
| Component | Intervention | URL |
|-----------|--------------|-----|
| **STEADI Algorithm** | **S**topping **E**lderly **A**ccidents, **D**eaths & **I**njuries | https://www.cdc.gov/steadi |
| **Screening Questions** | "Have you fallen in the past year?" "Do you feel unsteady?" "Do you worry about falling?" | https://www.cdc.gov/steadi |
| **Timed Up and Go (TUG)** | **=12 seconds = Increased fall risk** | https://www.cdc.gov/steadi |
| **Interventions** | **Exercise** (balance, strength), **Medication review**, **Vitamin D** (800-1000 IU), **Home safety**, **Vision correction** | https://www.cdc.gov/steadi |

## Beers Criteria - Key Inappropriate Medications
| Drug Class | Rationale | Alternative | URL |
|------------|-----------|-------------|-----|
| **Anticholinergics** (diphenhydramine, oxybutynin) | Confusion, Constipation, Falls | Non-anticholinergic options | https://dcri.org/beers-criteria/ |
| **Benzodiazepines** | Falls, Cognitive impairment | Non-pharmacologic, SSRIs | https://dcri.org/beers-criteria/ |
| **NSAIDs** (long-term) | GI bleeding, AKI, ? BP | Acetaminophen, Topical NSAIDs | https://dcri.org/beers-criteria/ |
| **PPIs >8 weeks** | C. diff, Bone loss, B12 deficiency | H2 blocker, Deprescribe | https://dcri.org/beers-criteria/ |
| **Sliding scale insulin** | Hypoglycemia | Basal insulin | https://dcri.org/beers-criteria/ |
| **Glyburide** | Severe prolonged hypoglycemia | Glipizide, Metformin | https://dcri.org/beers-criteria/ |
| **Antipsychotics** (in dementia) | Stroke, Mortality | Non-pharmacologic | https://dcri.org/beers-criteria/ |

## Common Geriatric Syndromes
| Syndrome | Key Points | URL |
|----------|------------|-----|
| **Delirium** | **Acute confusion**, Inattention, Fluctuating course; **Medical emergency** | https://www.nia.nih.gov/health/delirium |
| **Dementia** | **Chronic progressive cognitive decline**; Alzheimer most common | https://www.alz.org |
| **Depression** | Often **atypical** (somatic complaints, anhedonia); Screen with PHQ-2/GDS | https://www.nimh.nih.gov/health/topics/older-adults-and-mental-health |
| **Urinary Incontinence** | Ask! Treatable; Types: Stress, Urge, Overflow, Functional | https://www.nia.nih.gov/health/urinary-incontinence |
| **Pressure Injuries** | Prevention: Repositioning q2h, Pressure-reducing surfaces, Skin care | https://www.nia.nih.gov/health/pressure-sores |
