# Family Medicine - Sleep Disorders

## Obstructive Sleep Apnea (OSA)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Recurrent upper airway collapse during sleep causing hypoxemia and arousals | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **Risk Factors** | **Obesity** (strongest), Male, Age >50, Craniofacial abnormalities | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **Symptoms** | **Loud snoring**, **Witnessed apneas**, **Daytime sleepiness**, Morning headache | https://www.nhlbi.nih.gov/health/sleep-apnea/symptoms |
| **Screening** | **STOP-BANG**: Snoring, Tired, Observed apnea, Pressure (HTN), BMI >35, Age >50, Neck circumference >40 cm (men)/ >38 cm (women), Gender (male) | https://www.mdcalc.com/calc/3992/stop-bang-score-obstructive-sleep-apnea |

## OSA Diagnosis and Severity
| AHI (events/hour) | Severity | URL |
|-------------------|----------|-----|
| **<5** | Normal | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **5-15** | Mild | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **15-30** | Moderate | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **>30** | Severe | https://www.nhlbi.nih.gov/health/sleep-apnea |
| **Diagnosis** | **Home Sleep Apnea Test (HSAT)** or **In-lab Polysomnography (PSG)** | https://www.nhlbi.nih.gov/health/sleep-apnea/diagnosis |

## OSA Treatment
| Treatment | Indication | URL |
|-----------|------------|-----|
| **CPAP** | **First-line** for all OSA | https://www.nhlbi.nih.gov/health/sleep-apnea/treatment |
| **Weight Loss** | All overweight/obese patients | https://www.nhlbi.nih.gov/health/sleep-apnea/treatment |
| **Positional Therapy** | Positional OSA (supine-predominant) | https://www.nhlbi.nih.gov/health/sleep-apnea/treatment |
| **Oral Appliance** | Mild-moderate OSA, CPAP intolerant | https://www.nhlbi.nih.gov/health/sleep-apnea/treatment |
| **Surgery** | Refractory OSA with anatomic obstruction | https://www.nhlbi.nih.gov/health/sleep-apnea/treatment |
| **Hypoglossal Nerve Stimulation** | Moderate-severe OSA, CPAP intolerant, BMI <32 | https://www.fda.gov/medical-devices |

## Insomnia
| Type | Features | Management | URL |
|------|----------|------------|-----|
| **Acute** | <3 months, Identifiable trigger | Sleep hygiene, Short-term hypnotics (if needed) | https://www.nhlbi.nih.gov/health/insomnia |

## Restless Legs Syndrome (RLS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Urge to move legs**, Worse at rest/evening, Relieved by movement | https://www.ninds.nih.gov/health-information/disorders/restless-legs-syndrome |
| **Diagnosis** | **Clinical** (URGE criteria) | https://www.ninds.nih.gov/health-information/disorders/restless-legs-syndrome |
| **Workup** | **Ferritin** (target >50-75 ng/mL) | https://www.ninds.nih.gov/health-information/disorders/restless-legs-syndrome |
| **Treatment** | **Iron replacement** if low ferritin; **Gabapentinoids**, **Dopamine agonists** (caution: augmentation) | https://www.ninds.nih.gov/health-information/disorders/restless-legs-syndrome |
