# Thoracic Surgery - Thoracic Trauma

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanisms** | **Blunt** (MVC, Falls), **Penetrating** (GSW, Stab) | https://www.sts.org/patients/thoracic-trauma |
| **ATLS Principles** | **ABCDE** with focus on **immediately life-threatening injuries** | https://www.facs.org/quality-programs/trauma/atls |

## Immediately Life-Threatening Chest Injuries (ATLS)
| Injury | Diagnosis | Management | URL |
|--------|-----------|------------|-----|
| **Airway Obstruction** | **Stridor**, **Respiratory distress** | **Airway management** (intubation, Cricothyroidotomy) | https://www.facs.org/quality-programs/trauma/atls |
| **Tension Pneumothorax** | **Hypotension**, **Distended neck veins**, **Absent breath sounds**, **Tracheal deviation** | **Emergent needle decompression** ? **Chest tube** | https://www.facs.org/quality-programs/trauma/atls |
| **Open Pneumothorax** | **Sucking chest wound** | **Three-sided occlusive dressing** ? **Chest tube** | https://www.facs.org/quality-programs/trauma/atls |
| **Massive Hemothorax** | **Shock**, **Absent breath sounds**, **Dull to percussion** | **Chest tube** (=1500 mL initial or =200 mL/h x2-4h ? **Thoracotomy**) | https://www.facs.org/quality-programs/trauma/atls |
| **Flail Chest** | **=3 consecutive ribs fractured in =2 places** ? **Paradoxical movement** | **Pain control**, **Pulmonary toilet**, **Positive pressure ventilation** (if respiratory failure) | https://www.facs.org/quality-programs/trauma/atls |
| **Cardiac Tamponade** | **Beck triad** (Hypotension, Muffled heart sounds, JVD) | **Pericardiocentesis**, **Emergent thoracotomy** | https://www.facs.org/quality-programs/trauma/atls |

## Rib Fractures
| Component | Description | URL |
|-----------|-------------|-----|
| **Morbidity** | **Pain**, **Pneumonia**, **Respiratory failure** (especially elderly, =3 fractures) | https://www.sts.org/patients/thoracic-trauma |
| **Management** | **Multimodal pain control**, **Incentive spirometry**, **Pulmonary toilet**, **Surgical stabilization of rib fractures (SSRF)** (flail chest, Severe pain, Non-union) | https://www.sts.org/patients/thoracic-trauma |

## Pulmonary Contusion
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Parenchymal lung injury** ? **Edema**, **Hemorrhage**, **V/Q mismatch** | https://www.sts.org/patients/thoracic-trauma |
| **Management** | **Supportive care**, **Judicious fluids**, **Low tidal volume ventilation** (if intubated) | https://www.sts.org/patients/thoracic-trauma |

## Aortic Injury
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **Rapid deceleration** (MVC, Fall) ? **Aortic isthmus** (ligamentum arteriosum) | https://www.sts.org/patients/thoracic-trauma |
| **Diagnosis** | **CT angiography** | https://www.sts.org/patients/thoracic-trauma |
| **Management** | **Blood pressure control** (beta-blocker), **TEVAR** (preferred), **Open repair** | https://www.sts.org/patients/thoracic-trauma |

## Esophageal Injury
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **Penetrating trauma** (neck/chest), **Blunt** (rare) | https://www.sts.org/patients/thoracic-trauma |
| **Diagnosis** | **Esophagram**, **CT**, **Endoscopy** | https://www.sts.org/patients/thoracic-trauma |
| **Management** | **Primary repair** (within 24h), **Drainage**, **Diversion** (delayed) | https://www.sts.org/patients/thoracic-trauma |

## Emergency Department Thoracotomy (EDT)
| Indication | URL |
|------------|-----|
| **Penetrating chest trauma** with **loss of vital signs** within **10-15 minutes** | https://www.facs.org/quality-programs/trauma/atls |
| **Blunt trauma** with **loss of vital signs in ED** (very low survival) | https://www.facs.org/quality-programs/trauma/atls |
| **Goal** | **Release tamponade**, **Control hemorrhage**, **Cross-clamp aorta**, **Open cardiac massage** | https://www.facs.org/quality-programs/trauma/atls |
