# Thoracic Surgery - Congenital Thoracic Disorders

## Congenital Pulmonary Airway Malformation (CPAM)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Hamartomatous lung malformation** (formerly CCAM) | https://www.sts.org/patients/congenital-thoracic-disorders |
| **Types** | **Type 1** (large cysts, most common), **Type 2** (small cysts), **Type 3** (solid), **Type 4** (peripheral cysts) | https://www.sts.org/patients/congenital-thoracic-disorders |
| **Treatment** | **Elective resection** (3-6 months) due to risk of infection and malignancy (pleuropulmonary blastoma) | https://www.sts.org/patients/congenital-thoracic-disorders |

## Bronchopulmonary Sequestration
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Non-functioning lung tissue** with **systemic arterial supply** | https://www.sts.org/patients/congenital-thoracic-disorders |
| **Intralobar** | **Within visceral pleura**, **Venous drainage to pulmonary veins**, **Presents later** (recurrent infection) | https://www.sts.org/patients/congenital-thoracic-disorders |
| **Extralobar** | **Separate pleural investment**, **Venous drainage to systemic veins**, **Presents earlier** (respiratory distress) | https://www.sts.org/patients/congenital-thoracic-disorders |
| **Treatment** | **Surgical resection** (sequestrectomy or lobectomy) | https://www.sts.org/patients/congenital-thoracic-disorders |

## Congenital Lobar Emphysema (CLE)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Progressive hyperinflation of lobe** due to **bronchial obstruction** (cartilage deficiency) | https://www.sts.org/patients/congenital-thoracic-disorders |
| **Presentation** | **Neonatal respiratory distress** | https://www.sts.org/patients/congenital-thoracic-disorders |
| **Treatment** | **Lobectomy** (symptomatic), **Observation** (mild) | https://www.sts.org/patients/congenital-thoracic-disorders |

| Type | Description | URL |
|------|-------------|-----|
| **Type A** | **EA without TEF** (pure esophageal atresia) | https://www.sts.org/patients/congenital-thoracic-disorders |
| **Type B** | **EA with proximal TEF** | https://www.sts.org/patients/congenital-thoracic-disorders |
| **Type C** | **EA with distal TEF** (most common, 85%) | https://www.sts.org/patients/congenital-thoracic-disorders |
| **Type D** | **EA with proximal and distal TEF** | https://www.sts.org/patients/congenital-thoracic-disorders |
