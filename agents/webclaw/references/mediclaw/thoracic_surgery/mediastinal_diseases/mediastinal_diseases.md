## Thymoma
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Neoplasm of thymic epithelial cells** | https://www.sts.org/patients/mediastinal-tumors |
| **Associations** | **Myasthenia gravis** (30-50%), **Hypogammaglobulinemia**, **Pure red cell aplasia** | https://www.sts.org/patients/mediastinal-tumors |
| **WHO Classification** | **Type A** (spindle cell), **Type AB** (mixed), **Type B1** (lymphocyte-rich), **Type B2** (cortical), **Type B3** (epithelial), **Type C** (thymic carcinoma) | https://www.nccn.org/guidelines/thymic |
| **Masaoka-Koga Staging** | **I**: Encapsulated; **II**: Microscopic capsular invasion; **III**: Macroscopic invasion; **IVA**: Pleural/pericardial dissemination; **IVB**: Hematogenous metastases | https://www.nccn.org/guidelines/thymic |
| **Treatment** | **Surgical resection** (thymectomy), **Radiation** (Stage II-IV), **Chemotherapy** (advanced) | https://www.nccn.org/guidelines/thymic |
