# Thoracic Surgery - Esophageal Cancer

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Types** | **Adenocarcinoma** (80% in US, Distal/GEJ), **Squamous cell carcinoma** (20%, Mid/proximal) | https://www.cancer.gov/types/esophageal |
| **Risk Factors** | **Adenocarcinoma**: **GERD**, **Barrett esophagus**, **Obesity**; **SCC**: **Smoking**, **Alcohol**, **Achalasia** | https://www.cancer.gov/types/esophageal |

## Staging and Treatment (NCCN)
| Stage | Treatment | URL |
|-------|-----------|-----|
| **T1a** (mucosal) | **Endoscopic resection** (EMR/ESD) ± **Ablation** | https://www.nccn.org/guidelines/esophageal |
| **T1b** (submucosal) | **Esophagectomy** (or endoscopic resection for low-risk) | https://www.nccn.org/guidelines/esophageal |
| **T2-T4a, N0-N+, M0** | **Neoadjuvant chemoradiation** (CROSS regimen) ? **Esophagectomy** | https://www.nccn.org/guidelines/esophageal |
| **T4b** (unresectable) | **Definitive chemoradiation** | https://www.nccn.org/guidelines/esophageal |
| **M1** | **Systemic therapy** ± **Palliative radiation** | https://www.nccn.org/guidelines/esophageal |

## CROSS Trial (Neoadjuvant Chemoradiotherapy)
| Component | Description | URL |
|-----------|-------------|-----|
| **Regimen** | **Carboplatin + Paclitaxel** weekly x5 + **Concurrent radiation** (41.4 Gy) | https://www.nejm.org/doi/full/10.1056/NEJMoa1112088 |
| **Outcome** | **Improved OS** (median 49 vs 24 months), **Improved R0 resection** (92% vs 69%) | https://www.nejm.org/doi/full/10.1056/NEJMoa1112088 |

## Barrett Esophagus
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Intestinal metaplasia** in distal esophagus | https://gi.org/guideline/barretts-esophagus/ |
| **Surveillance** | **No dysplasia**: q3-5y; **LGD**: q6-12m or ablation; **HGD**: **Endoscopic eradication** | https://gi.org/guideline/barretts-esophagus/ |
| **Treatment** | **Radiofrequency ablation (RFA)** or **Cryoablation** | https://gi.org/guideline/barretts-esophagus/ |
