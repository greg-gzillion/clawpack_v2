# Thoracic Surgery - Esophageal Surgery

## Esophagectomy Approaches
| Approach | Description | Indications | URL |
|----------|-------------|-------------|-----|
| **Transhiatal** | **Abdominal + cervical incision**, **Blunt dissection** | **Distal esophageal/GEJ tumors**, **Avoid thoracotomy** | https://www.sts.org/patients/esophageal-surgery |
| **Ivor Lewis** | **Abdominal + Right thoracotomy** | **Mid-distal esophageal tumors**, **Better lymphadenectomy** | https://www.sts.org/patients/esophageal-surgery |
| **McKeown (3-field)** | **Abdominal + Right thoracotomy + Cervical** | **Mid-proximal esophageal tumors**, **Cervical anastomosis** | https://www.sts.org/patients/esophageal-surgery |
| **Minimally Invasive (MIE)** | **Laparoscopic/Thoracoscopic** or **Robotic** | **Reduced morbidity**, **Similar oncologic outcomes** | https://www.sts.org/patients/esophageal-surgery |

## Esophagectomy Complications
| Complication | Management | URL |
|--------------|------------|-----|
| **Anastomotic Leak** | **Cervical**: Open wound, Pack; **Intrathoracic**: **Drainage**, **Stent**, **Reoperation** | https://www.sts.org/patients/esophageal-surgery |
| **Anastomotic Stricture** | **Endoscopic dilation** | https://www.sts.org/patients/esophageal-surgery |
| **Chylothorax** | **Low-fat diet/MCT oil**, **TPN**, **Thoracic duct ligation** (if >1L/day) | https://www.sts.org/patients/esophageal-surgery |
| **Recurrent Laryngeal Nerve Injury** | **Vocal cord medialization**, **Speech therapy** | https://www.sts.org/patients/esophageal-surgery |
| **Delayed Gastric Emptying** | **Pyloroplasty/pyloromyotomy**, **Prokinetics** | https://www.sts.org/patients/esophageal-surgery |
| **Atrial Fibrillation** | **Rate/rhythm control**, **Anticoagulation** | https://www.sts.org/patients/esophageal-surgery |
