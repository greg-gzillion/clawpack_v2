# Thoracic Surgery - Tracheal Surgery

## Tracheal Stenosis
| Component | Description | URL |
|-----------|-------------|-----|
| **Etiology** | **Post-intubation** (most common), **Post-tracheostomy**, **Idiopathic**, **Trauma**, **Infection**, **Neoplasm** | https://www.sts.org/patients/tracheal-surgery |
| **Symptoms** | **Dyspnea**, **Stridor**, **Wheezing**, **Cough** | https://www.sts.org/patients/tracheal-surgery |
| **Diagnosis** | **CT neck/chest**, **Bronchoscopy** | https://www.sts.org/patients/tracheal-surgery |
| **Treatment** | **Endoscopic dilation** (temporary), **Laser excision**, **Tracheal resection and reconstruction** (definitive) | https://www.sts.org/patients/tracheal-surgery |

## Tracheal Resection and Reconstruction
| Component | Description | URL |
|-----------|-------------|-----|
| **Maximum Resectable Length** | **~50% of trachea** (~6 cm) | https://www.sts.org/patients/tracheal-surgery |
| **Release Maneuvers** | **Anterior**: **Suprahyoid** (Montgomery), **Infrahyoid**; **Posterior**: **Hilar release** | https://www.sts.org/patients/tracheal-surgery |
| **Complications** | **Anastomotic dehiscence**, **Restenosis**, **Recurrent laryngeal nerve injury** | https://www.sts.org/patients/tracheal-surgery |

## Tracheomalacia
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Dynamic airway collapse** during expiration | https://www.sts.org/patients/tracheal-surgery |
| **Etiology** | **Congenital**, **Acquired** (post-intubation, COPD, Relapsing polychondritis) | https://www.sts.org/patients/tracheal-surgery |
| **Treatment** | **CPAP**, **Tracheobronchoplasty** (posterior splinting with mesh), **Tracheostomy**, **Stenting** | https://www.sts.org/patients/tracheal-surgery |

## Tracheoesophageal Fistula (TEF)
| Component | Description | URL |
|-----------|-------------|-----|
