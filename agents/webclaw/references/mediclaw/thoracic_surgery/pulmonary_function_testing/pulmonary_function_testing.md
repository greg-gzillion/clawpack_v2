# Thoracic Surgery - Pulmonary Function Testing

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Purpose** | **Assess operative risk** for lung resection, **Predict postoperative function** | https://www.sts.org/education/thoracic-surgery-pft |

## Spirometry
| Parameter | Normal | Significance | URL |
|-----------|--------|--------------|-----|
| **FEV1** | **>80% predicted** | **Obstruction** if ?; **<30% = severe** | https://www.thoracic.org/professionals/clinical-resources/pulmonary-function-testing |
| **FVC** | **>80% predicted** | **Restriction** if ? | https://www.thoracic.org/professionals/clinical-resources/pulmonary-function-testing |
| **FEV1/FVC** | **>0.70** | **<0.70 = Obstruction** | https://www.thoracic.org/professionals/clinical-resources/pulmonary-function-testing |

## Lung Volumes (Plethysmography)
| Parameter | Normal | Significance | URL |
|-----------|--------|--------------|-----|

| **RV** | **80-120% predicted** | **? in air trapping** (COPD) | https://www.thoracic.org/professionals/clinical-resources/pulmonary-function-testing |

| Parameter | Normal | Significance | URL |

## Preoperative Risk Assessment for Lung Resection (ACCP/STS Guidelines)
| Test | Acceptable for Pneumonectomy | Acceptable for Lobectomy | URL |
|------|----------------------------|-------------------------|-----|
| **FEV1** | **>2 L** or **>80% predicted** | **>1.5 L** | https://journal.chestnet.org/guidelines |
| **DLCO** | **>60% predicted** | **>60% predicted** | https://journal.chestnet.org/guidelines |
| **If below thresholds** | **Calculate predicted postoperative (ppo) values** | **Calculate ppo values** | https://journal.chestnet.org/guidelines |

## Predicted Postoperative (ppo) Values
| Calculation | Acceptable Threshold | URL |
|-------------|---------------------|-----|
| **ppoFEV1 = Preoperative FEV1 ū (1 - Fraction of lung to be resected)** | **>40%** | https://journal.chestnet.org/guidelines |
| **ppoDLCO = Preoperative DLCO ū (1 - Fraction of lung to be resected)** | **>40%** | https://journal.chestnet.org/guidelines |
| **Quantitative Perfusion Scan** | **Preferred for pneumonectomy** | https://journal.chestnet.org/guidelines |

## Additional Preoperative Testing
| Test | Indication | Threshold | URL |

| **Cardiopulmonary Exercise Testing (CPET)** | **Borderline PFTs** | **VO2max >15-20 mL/kg/min** = Acceptable risk | https://journal.chestnet.org/guidelines |
