## Hypertrophic Cardiomyopathy (HCM)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Unexplained left ventricular hypertrophy** (=15 mm) | https://www.heart.org/en/health-topics/cardiomyopathy/what-is-cardiomyopathy-in-adults/hypertrophic-cardiomyopathy |
| **Genetics** | **Autosomal dominant** (sarcomeric protein mutations) | https://www.heart.org/en/health-topics/cardiomyopathy/what-is-cardiomyopathy-in-adults/hypertrophic-cardiomyopathy |
| **Symptoms** | **Chest pain**, **Dyspnea**, **Syncope** (especially with exertion), **Palpitations** | https://www.heart.org/en/health-topics/cardiomyopathy/what-is-cardiomyopathy-in-adults/hypertrophic-cardiomyopathy |
| **Physical Exam** | **Systolic murmur** (? with Valsalva/standing, ? with squatting), **S4** | https://www.heart.org/en/health-topics/cardiomyopathy/what-is-cardiomyopathy-in-adults/hypertrophic-cardiomyopathy |
| **ECG** | **LVH**, **Deep septal Q waves**, **T-wave inversions** | https://www.heart.org/en/health-topics/cardiomyopathy/what-is-cardiomyopathy-in-adults/hypertrophic-cardiomyopathy |
| **Sports Participation** | **Generally restricted** from competitive sports (except low-intensity) | https://www.acc.org/guidelines/hubs/sports-participation |

## Commotio Cordis
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Sudden cardiac death** from **blunt chest wall impact** during **vulnerable period of repolarization** (10-30 ms before T-wave peak) | https://www.heart.org/en/health-topics/cardiac-arrest/what-is-commotio-cordis |
| **Mechanism** | **Induction of ventricular fibrillation** | https://www.heart.org/en/health-topics/cardiac-arrest/what-is-commotio-cordis |
| **Demographics** | **Young athletes** (compliant chest wall), **Sports with projectiles** (baseball, lacrosse, hockey) | https://www.heart.org/en/health-topics/cardiac-arrest/what-is-commotio-cordis |
| **Prevention** | **Protective equipment** (chest protectors), **Age-appropriate safety baseballs** | https://www.heart.org/en/health-topics/cardiac-arrest/what-is-commotio-cordis |
| **Survival** | **<25%** (requires **immediate CPR + AED**) | https://www.heart.org/en/health-topics/cardiac-arrest/what-is-commotio-cordis |
