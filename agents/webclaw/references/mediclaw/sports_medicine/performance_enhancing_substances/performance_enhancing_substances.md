## Therapeutic Use Exemption (TUE)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Permission to use prohibited substance** for **legitimate medical condition** | https://www.wada-ama.org/en/what-we-do/science-medical/therapeutic-use-exemptions |
| **Criteria** | **1. Significant health impairment without substance**, **2. No performance enhancement beyond return to normal health**, **3. No reasonable therapeutic alternative**, **4. Not consequence of prior non-TUE use** | https://www.wada-ama.org/en/what-we-do/science-medical/therapeutic-use-exemptions |
| **Application** | **Retroactive** (emergency) or **Prospective** (=30 days before competition) | https://www.wada-ama.org/en/what-we-do/science-medical/therapeutic-use-exemptions |
