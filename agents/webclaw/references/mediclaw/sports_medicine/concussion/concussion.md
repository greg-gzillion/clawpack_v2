# Sports Medicine - Concussion

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Traumatic brain injury** induced by **biomechanical forces**, resulting in **rapid onset of short-lived neurologic impairment** that **resolves spontaneously** | https://www.cdc.gov/headsup/basics/concussion_whatis.html |
| **Pathophysiology** | **Neurometabolic cascade**: **Ionic flux**, **Energy crisis**, **Impaired neurotransmission**, **Axonal injury** | https://www.cdc.gov/headsup/basics/concussion_whatis.html |
| **Mechanism** | **Direct blow** to head, face, neck, or **elsewhere with impulsive force transmitted to head** | https://www.cdc.gov/headsup/basics/concussion_whatis.html |
| **Key Point** | **Loss of consciousness NOT required** (<10% of concussions) | https://www.cdc.gov/headsup/basics/concussion_whatis.html |

|-----------|------------|-----|
| **Red Flags (Emergent)** | **Loss of consciousness**, **Seizure**, **Focal neurologic deficit**, **Worsening headache**, **Repeated vomiting**, **Significant altered mental status**, **Suspected skull fracture** | https://www.cdc.gov/headsup/basics/concussion_danger_signs.html |
| **Sideline Assessment** | **SCAT6** (Sport Concussion Assessment Tool), **SAC** (Standardized Assessment of Concussion), **BESS** (Balance Error Scoring System) | https://www.cdc.gov/headsup/basics/concussion_recovery.html |
| **Office Assessment** | **Symptom checklist**, **Cognitive testing**, **Neurologic exam**, **Balance testing**, **Oculomotor exam** | https://www.cdc.gov/headsup/basics/concussion_recovery.html |

## Concussion Management (Return to Learn and Return to Play)
| Phase | Activity | Goal | URL |
|-------|----------|------|-----|
| **1. Relative Rest** | **24-48 hours** of reduced physical and cognitive activity | Symptom stabilization | https://www.cdc.gov/headsup/basics/concussion_recovery.html |
| **2. Return to Learn** | **Gradual increase in cognitive activity** (school, work) | **Full return to academic/work activities** without accommodations | https://www.cdc.gov/headsup/basics/concussion_recovery.html |
| **3. Light Aerobic Activity** | Walking, Stationary bike (no resistance) | **? HR** without symptoms | https://www.cdc.gov/headsup/basics/concussion_recovery.html |
| **4. Sport-Specific Exercise** | Running drills (no head impact) | **Add movement** | https://www.cdc.gov/headsup/basics/concussion_recovery.html |
| **5. Non-Contact Training Drills** | Complex drills, Resistance training | **Add coordination and cognition** | https://www.cdc.gov/headsup/basics/concussion_recovery.html |
| **6. Full Contact Practice** | Normal training activities | **Restore confidence**, **Assess functional skills** | https://www.cdc.gov/headsup/basics/concussion_recovery.html |
| **7. Return to Sport** | Normal game play | **Medical clearance** | https://www.cdc.gov/headsup/basics/concussion_recovery.html |

## Return to Play Guidelines
| Principle | Description | URL |
|-----------|-------------|-----|
| **Symptom-Limited** | **Progress to next stage only if asymptomatic** at current stage | https://www.cdc.gov/headsup/basics/concussion_recovery.html |
| **=24 Hours per Stage** | **Minimum 1 week** for full return-to-play progression | https://www.cdc.gov/headsup/basics/concussion_recovery.html |
| **Return to Baseline** | **If symptoms recur**, **Return to previous asymptomatic stage** | https://www.cdc.gov/headsup/basics/concussion_recovery.html |
| **Medical Clearance** | **Required before return to contact activities** | https://www.cdc.gov/headsup/basics/concussion_recovery.html |

## Prolonged Recovery and Post-Concussion Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Symptoms persisting >4 weeks** (adults) or **>2-4 weeks** (children/adolescents) | https://www.cdc.gov/headsup/basics/concussion_recovery.html |
| **Risk Factors** | **Prior concussion**, **Migraine history**, **Mental health history**, **Female sex**, **Younger age** | https://www.cdc.gov/headsup/basics/concussion_recovery.html |
| **Management** | **Multidisciplinary**: Vestibular therapy, Vision therapy, Cognitive rehabilitation, Exertional therapy, Psychology | https://www.cdc.gov/headsup/basics/concussion_recovery.html |

## Second Impact Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Catastrophic cerebral edema** occurring when **second concussion sustained before resolution of first** | https://www.cdc.gov/headsup/basics/concussion_whatis.html |
| **Pathophysiology** | **Loss of cerebral autoregulation** ? **Malignant cerebral edema** ? **Brain herniation** | https://www.cdc.gov/headsup/basics/concussion_whatis.html |
| **Outcome** | **High mortality** (50%) or **severe disability** | https://www.cdc.gov/headsup/basics/concussion_whatis.html |
| **Prevention** | **Strict adherence to return-to-play guidelines**; **No same-day return** | https://www.cdc.gov/headsup/basics/concussion_whatis.html |
