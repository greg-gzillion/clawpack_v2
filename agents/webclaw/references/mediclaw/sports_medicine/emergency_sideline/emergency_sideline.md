## Spine Injury Management
| Principle | Description | URL |
|-----------|-------------|-----|
| **Assume Cervical Spine Injury** | In **unconscious athlete** or **mechanism concerning for spine injury** | https://www.nata.org/practice-patient-care/health-issues/spine-injury |
| **Primary Survey** | **Airway** (jaw thrust, not head tilt), **Breathing**, **Circulation** | https://www.nata.org/practice-patient-care/health-issues/spine-injury |
| **Stabilization** | **Manual in-line stabilization**, **Rigid cervical collar**, **Spine board** | https://www.nata.org/practice-patient-care/health-issues/spine-injury |
| **Equipment Removal** | **Helmet and shoulder pads removed together** (or left in place if inadequate personnel) | https://www.nata.org/practice-patient-care/health-issues/spine-injury |
| **Transport** | **Trauma center** with spine surgery capabilities | https://www.nata.org/practice-patient-care/health-issues/spine-injury |

## Sudden Cardiac Arrest
| Principle | Description | URL |
|-----------|-------------|-----|
| **Recognition** | **Unresponsive**, **No normal breathing**, **No pulse** | https://cpr.heart.org/en/resuscitation-science/cpr-and-ecc-guidelines |
| **Activate EAP** | **Call 911**, **Retrieve AED** | https://cpr.heart.org/en/resuscitation-science/cpr-and-ecc-guidelines |
| **High-Quality CPR** | **100-120 compressions/min**, **2-2.4 inches depth**, **Full chest recoil** | https://cpr.heart.org/en/resuscitation-science/cpr-and-ecc-guidelines |
| **AED** | **Apply as soon as available**, **Follow prompts** | https://cpr.heart.org/en/resuscitation-science/cpr-and-ecc-guidelines |
| **Continue Until** | **Return of spontaneous circulation**, **EMS arrives**, **Exhaustion** | https://cpr.heart.org/en/resuscitation-science/cpr-and-ecc-guidelines |

## Exertional Heat Stroke
| Principle | Description | URL |
|-----------|-------------|-----|
| **Recognition** | **CNS dysfunction** (confusion, Collapse), **Core temp >40°C (104°F)** | https://www.nata.org/practice-patient-care/health-issues/heat-illness |
| **Immediate Cooling** | **Cold water immersion** (35-59°F) **within 30 minutes** | https://www.nata.org/practice-patient-care/health-issues/heat-illness |
| **Cool First, Transport Second** | **On-site cooling** takes priority over transport | https://www.nata.org/practice-patient-care/health-issues/heat-illness |
| **Continue Cooling** | Until **core temp <39°C (102°F)** or **mental status improves** | https://www.nata.org/practice-patient-care/health-issues/heat-illness |
