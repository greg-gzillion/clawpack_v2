# Sports Medicine - Rehabilitation Principles

## Phases of Rehabilitation
| Phase | Goal | Interventions | URL |
|-------|------|---------------|-----|
| **1. Acute/Protection** | **Control pain/inflammation**, **Protect injured tissue**, **Maintain ROM** | **RICE** (Rest, Ice, Compression, Elevation), **Protected weight-bearing**, **Gentle ROM** | https://www.aapmr.org/about-physiatry/conditions-treatments/sports-medicine |
| **2. Recovery/Subacute** | **Restore ROM**, **Improve strength/endurance**, **Correct neuromuscular deficits** | **Progressive strengthening**, **Proprioception**, **Cardiovascular conditioning** | https://www.aapmr.org/about-physiatry/conditions-treatments/sports-medicine |
| **3. Functional/Sport-Specific** | **Restore sport-specific skills**, **Improve power/agility**, **Prepare for return to play** | **Plyometrics**, **Agility drills**, **Sport-specific drills** | https://www.aapmr.org/about-physiatry/conditions-treatments/sports-medicine |
| **4. Return to Play** | **Safe return to competition**, **Prevent reinjury** | **Functional testing**, **Gradual return to full activity**, **Maintenance program** | https://www.aapmr.org/about-physiatry/conditions-treatments/sports-medicine |

## RICE vs POLICE vs PEACE & LOVE
| Acronym | Components | URL |
|---------|------------|-----|
| **RICE** | **R**est, **I**ce, **C**ompression, **E**levation (traditional acute management) | https://www.aapmr.org/about-physiatry/conditions-treatments/sports-medicine |
| **POLICE** | **P**rotection, **O**ptimal **L**oading, **I**ce, **C**ompression, **E**levation (emphasizes early controlled loading) | https://www.aapmr.org/about-physiatry/conditions-treatments/sports-medicine |
| **PEACE & LOVE** | **P**rotection, **E**levation, **A**void anti-inflammatories, **C**ompression, **E**ducation (acute) + **L**oad, **O**ptimism, **V**ascularization, **E**xercise (subacute/chronic) | https://www.aapmr.org/about-physiatry/conditions-treatments/sports-medicine |

## Return to Play Criteria
| Criterion | Description | URL |
|-----------|-------------|-----|
| **Pain-Free** | **No pain** with sport-specific activities | https://www.amssm.org/Content/Resources-1/Return-to-Play.htm |
| **Full ROM** | **Full pain-free range of motion** | https://www.amssm.org/Content/Resources-1/Return-to-Play.htm |
| **Strength** | **=90% of contralateral strength** | https://www.amssm.org/Content/Resources-1/Return-to-Play.htm |
| **Functional Testing** | **Successful completion of sport-specific drills** (e.g., hop tests, Agility drills) | https://www.amssm.org/Content/Resources-1/Return-to-Play.htm |
| **Psychological Readiness** | **Confidence in injured limb**, **No fear of reinjury** | https://www.amssm.org/Content/Resources-1/Return-to-Play.htm |
