# Sports Medicine - Preparticipation Physical Evaluation (PPE)

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Purpose** | **Screen for life-threatening or disabling conditions**, **Identify conditions requiring further evaluation**, **Meet legal/insurance requirements**, **Counsel on injury prevention** | https://www.aap.org/en/patient-care/sports-medicine-and-fitness/preparticipation-physical-evaluation/ |
| **Timing** | **6-8 weeks before season** (allows time for evaluation/rehabilitation) | https://www.aap.org/en/patient-care/sports-medicine-and-fitness/preparticipation-physical-evaluation/ |
| **Frequency** | **Annually** (comprehensive), **Interim** (interval history, Vital signs) | https://www.aap.org/en/patient-care/sports-medicine-and-fitness/preparticipation-physical-evaluation/ |
| **Setting** | **Individual office visit** (preferred) or **Station-based group** | https://www.aap.org/en/patient-care/sports-medicine-and-fitness/preparticipation-physical-evaluation/ |

## PPE Components (AAP/ACSM/AMSSM)
| Component | Key Elements | URL |
|-----------|--------------|-----|
| **Medical History** | **Cardiac symptoms** (chest pain, Syncope, Palpitations), **Family history** (sudden death <50, Cardiomyopathy, Marfan), **Prior injuries/surgeries**, **Concussion history**, **Seizure disorder**, **Heat illness**, **Sickle cell trait** | https://www.aap.org/en/patient-care/sports-medicine-and-fitness/preparticipation-physical-evaluation/ |
| **Physical Exam** | **Vital signs** (BP, HR), **Cardiac** (murmurs, Marfan stigmata), **Musculoskeletal** (general, Sport-specific), **Neurologic** | https://www.aap.org/en/patient-care/sports-medicine-and-fitness/preparticipation-physical-evaluation/ |
| **Clearance** | **Cleared**, **Cleared with limitations**, **Not cleared** (pending further evaluation) | https://www.aap.org/en/patient-care/sports-medicine-and-fitness/preparticipation-physical-evaluation/ |

## Cardiac Screening in PPE (AHA/ACC 14-Element Checklist)
| Category | Questions | URL |
|----------|-----------|-----|
| **Personal History** | Chest pain/discomfort with exertion, Unexplained syncope/near-syncope, Excessive exertional dyspnea/fatigue, Prior heart murmur, Elevated systemic BP | https://www.heart.org/en/health-topics/congenital-heart-defects/symptoms--diagnosis-of-congenital-heart-defects/pre-participation-screening |
| **Family History** | Premature death (<50 years) due to heart disease, Disability from heart disease in close relative <50, Hypertrophic cardiomyopathy, Dilated cardiomyopathy, Long QT syndrome, Marfan syndrome, Arrhythmogenic right ventricular cardiomyopathy | https://www.heart.org/en/health-topics/congenital-heart-defects/symptoms--diagnosis-of-congenital-heart-defects/pre-participation-screening |
| **Physical Exam** | Heart murmur (supine/standing/Valsalva), Femoral pulses, Stigmata of Marfan syndrome, Brachial artery BP (sitting) | https://www.heart.org/en/health-topics/congenital-heart-defects/symptoms--diagnosis-of-congenital-heart-defects/pre-participation-screening |

## PPE Clearance Decisions
| Finding | Recommendation | URL |
|---------|----------------|-----|
| **Positive cardiac findings** | **Further evaluation** (ECG, Echocardiogram, Cardiology referral) before clearance | https://www.aap.org/en/patient-care/sports-medicine-and-fitness/preparticipation-physical-evaluation/ |
| **Single paired organ** (eye, kidney, testicle) | **Clearance with protective equipment**; Counsel on risks | https://www.aap.org/en/patient-care/sports-medicine-and-fitness/preparticipation-physical-evaluation/ |
| **Seizure disorder** | **Cleared if well-controlled**; Avoid swimming, climbing alone | https://www.aap.org/en/patient-care/sports-medicine-and-fitness/preparticipation-physical-evaluation/ |
| **Sickle cell trait** | **Cleared** with precautions (hydration, Acclimatization, Avoid extreme exertion) | https://www.aap.org/en/patient-care/sports-medicine-and-fitness/preparticipation-physical-evaluation/ |
| **Prior concussion** | **Cleared** if asymptomatic with full return-to-learn/play | https://www.aap.org/en/patient-care/sports-medicine-and-fitness/preparticipation-physical-evaluation/ |
