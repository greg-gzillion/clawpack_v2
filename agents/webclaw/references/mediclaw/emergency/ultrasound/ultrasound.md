# Emergency Medicine - Point-of-Care Ultrasound (POCUS)

## eFAST Exam
| View | Assesses | Positive Finding |
|------|----------|------------------|
| Right Upper Quadrant | Hepatorenal recess (Morison's pouch) | Anechoic stripe (free fluid) |
| Left Upper Quadrant | Splenorenal recess | Anechoic stripe (free fluid) |
| Subxiphoid | Pericardium | Pericardial effusion |
| Suprapubic | Pelvis (Pouch of Douglas) | Anechoic stripe (free fluid) |
| Anterior Chest (eFAST) | Pleural space | Absent lung sliding, Barcode sign |

## Cardiac POCUS Views
| View | Assesses | Key Findings |
|------|----------|--------------|
| Parasternal Long Axis | LV, RV, MV, AV, Pericardium | Effusion, LV function, RV size |
| Parasternal Short Axis | LV (mid-papillary) | LV function, Septal motion, Effusion |
| Apical 4-Chamber | All 4 chambers | RV size, LV function, Effusion |
| Subxiphoid | Pericardium | **Best view for tamponade** |
| IVC | Volume status | IVC diameter, Collapsibility |

## IVC Assessment
| Finding | Interpretation |
|---------|----------------|
| IVC <2 cm, >50% collapse | Hypovolemia |
| IVC >2 cm, <50% collapse | Hypervolemia |
| Plethoric IVC | Fluid overload, Tamponade, Tension PTX |

## Lung Ultrasound
| Finding | Interpretation | Sign |
|---------|----------------|-------|
| Lung Sliding | Normal pleural movement | Present |
| A-lines | Normal aeration | Horizontal reverberation |
| B-lines | Interstitial edema | Vertical comet-tail artifacts |
| Absent Lung Sliding | Pneumothorax | Barcode sign on M-mode |
| Lung Point | Transition point of pneumothorax | Diagnostic of PTX |
| Hepatization | Consolidation | Tissue-like appearance |
| Pleural Effusion | Anechoic space above diaphragm | Spine sign |

## Abdominal POCUS
| Application | Findings |
|-------------|----------|
| Gallbladder | Gallstones, Wall thickening, Pericholecystic fluid, Sonographic Murphy's |
| Biliary | CBD dilation (>6 mm) |
| Renal | Hydronephrosis, Stones (shadowing) |
| Bladder | Volume, Foley placement |
| Aorta | AAA (>3 cm) |
| Appendix | Non-compressible, >6 mm, Target sign |
| Small Bowel | Obstruction (dilated loops, Back-and-forth peristalsis) |

## OB/GYN POCUS
| Application | Findings |
|-------------|----------|
| Intrauterine Pregnancy | Gestational sac, Yolk sac, Fetal pole, Cardiac activity |
| Ectopic Pregnancy | Empty uterus + Positive hCG + Adnexal mass/fluid |
| Fetal Heart Rate | Normal 110-160 bpm |
| Gestational Age | CRL (most accurate) |

## Vascular POCUS
| Application | Findings |
|-------------|----------|
| DVT (Lower Extremity) | Non-compressible vein; Echogenic thrombus |
| DVT (Upper Extremity) | Non-compressible vein |
| AAA Screening | Aorta >3 cm (ectasia), >5.5 cm (repair threshold) |
| Carotid | Plaque, Stenosis (PSV >125 cm/s = >50%) |

## Soft Tissue and Musculoskeletal
| Application | Findings |
|-------------|----------|
| Abscess | Anechoic/hypoechoic collection; Swirl sign |
| Cellulitis | Cobblestoning; No discrete collection |
| Foreign Body | Hyperechoic with shadowing |
| Tendon | Fibrillar pattern; Rupture (gap) |
| Fracture | Cortical disruption |
| Joint Effusion | Anechoic fluid in joint space |

## Ocular POCUS
| Finding | Interpretation |
|---------|----------------|
| Retinal Detachment | Hyperechoic membrane floating in vitreous |
| Vitreous Hemorrhage | Echoic material in vitreous |
| Lens Dislocation | Lens in abnormal position |
| Papilledema | Elevated optic disc (>1 mm) |
| ONSD >5 mm | Elevated ICP |

## RUSH Exam (Rapid Ultrasound for Shock and Hypotension)
| Component | Assesses |
|-----------|----------|
| **H**eart | Effusion, LV function, RV size |
| **I**VC | Volume status |
| **M**orison's/Fast | Free fluid |
| **A**orta | AAA |
| **P**neumothorax | Lung sliding |
| **T**horax | Pleural effusion |

## Ultrasound-Guided Procedures
| Procedure | Technique |
|-----------|-----------|
| Peripheral IV | In-plane or out-of-plane |
| Central Line | Transverse or longitudinal |
| Paracentesis | Identify pocket; Avoid vessels |
| Thoracentesis | Identify effusion; Above rib |
| Pericardiocentesis | Subxiphoid; Visualize needle |
| Nerve Block | Identify nerve; Visualize spread |
| Lumbar Puncture | Identify interspace; Mark |
| Arthrocentesis | Identify effusion |
| Abscess Drainage | Identify collection |
