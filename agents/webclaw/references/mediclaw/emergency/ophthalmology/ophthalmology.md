# Emergency Medicine - Ophthalmologic Emergencies

## The Red Eye - Differential
| Condition | Key Features | Management |
|-----------|--------------|------------|
| Conjunctivitis | Discharge, Bilateral | Antibiotics if bacterial; Supportive if viral |
| Subconjunctival Hemorrhage | Painless, Flat | Reassurance; Check BP |
| Corneal Abrasion | Pain, Photophobia, Fluorescein uptake | Antibiotic ointment; Cycloplegics |
| Corneal Ulcer | White spot on cornea, Pain | **Emergent ophthalmology consult**; Antibiotics |
| Keratitis | Contact lens wearer, Pain | **Stop contacts**; Antibiotics (Pseudomonas coverage) |
| Acute Angle-Closure Glaucoma | Severe pain, N/V, Mid-fixed pupil, Cloudy cornea | **Emergent**; Timolol, Pilocarpine, Acetazolamide |
| Anterior Uveitis | Pain, Photophobia, Ciliary flush | Steroids; Cycloplegics |
| Scleritis | Severe boring pain, Violet hue | Systemic steroids; Rheumatology workup |
| Episcleritis | Mild discomfort, Sectoral redness | Topical NSAIDs; Self-limited |
| Hyphema | Blood in anterior chamber | Eye shield; Head elevation; Check for sickle cell |
| Orbital Cellulitis | Proptosis, EOM pain, Vision changes | **Emergent**; IV antibiotics; CT orbit |

## Acute Angle-Closure Glaucoma
| Finding | Management |
|---------|------------|
| Presentation | Severe eye pain, Headache, N/V, Halos around lights |
| Exam | Mid-dilated fixed pupil, Cloudy cornea, Shallow anterior chamber |
| Treatment | Timolol 0.5% (1 drop); Pilocarpine 2% (1 drop); Acetazolamide 500 mg IV; Mannitol |
| Definitive | Laser peripheral iridotomy |

## Central Retinal Artery Occlusion (CRAO)
| Finding | Management |
|---------|------------|
| Presentation | **Sudden, painless, monocular vision loss** |
| Exam | Cherry-red spot on fundoscopy; Afferent pupillary defect |
| Treatment | Ocular massage; Lower IOP; Hyperbaric oxygen |
| Workup | **Stroke workup** (CRAO is a stroke equivalent) |

## Retinal Detachment
| Finding | Management |
|---------|------------|
| Presentation | Flashes, Floaters, Curtain over vision |
| Exam | Gray elevated retina on fundoscopy |
| Treatment | **Emergent ophthalmology consult**; Surgical repair |

## Temporal Arteritis (Giant Cell Arteritis)
| Finding | Management |
|---------|------------|
| Presentation | Headache, Jaw claudication, Vision loss, Age >50 |
| Labs | Elevated ESR, CRP |
| Treatment | **High-dose steroids (Methylprednisolone 1g IV)**; Temporal artery biopsy |

## Chemical Ocular Burns
| Type | Management |
|------|------------|
| Alkali (worse) | **Immediate copious irrigation**; Check pH; Cycloplegics; Ophthalmology consult |
| Acid | Copious irrigation; Check pH; Ophthalmology consult |
| Irrigation | Minimum 1-2 L; Continue until pH 7.0-7.4 |

## Orbital Compartment Syndrome
| Finding | Management |
|---------|------------|
| Presentation | Proptosis, Decreased vision, Increased IOP, EOM restriction |
| Treatment | **Lateral canthotomy and cantholysis**; Emergent ophthalmology |

## Ocular Trauma
| Injury | Management |
|--------|------------|
| Corneal Foreign Body | Remove; Antibiotics; Cycloplegics |
| Globe Rupture | **Eye shield; Do NOT examine or patch**; IV antibiotics; Surgery |
| Eyelid Laceration | Repair; Check for canalicular injury |
| Blowout Fracture | EOM entrapment, Enophthalmos | Antibiotics; Surgery if entrapment |

## Sudden Vision Loss - Differential
| Etiology | Key Feature |
|----------|-------------|
| CRAO | Cherry-red spot |
| CRVO | Blood and thunder fundus |
| Retinal Detachment | Flashes, floaters, curtain |
| Vitreous Hemorrhage | Red haze, No red reflex |
| Temporal Arteritis | Headache, Elevated ESR/CRP |
| Optic Neuritis | Pain with EOM, Afferent pupillary defect |
| Amaurosis Fugax | Transient, Carotid disease |
