# Emergency Medicine - Orthopedic Emergencies

## Fracture Description Terminology
| Term | Definition |
|------|------------|
| Closed | Skin intact |
| Open | Skin breached (communicates with fracture) |
| Transverse | Perpendicular to bone axis |
| Oblique | Angled across bone |
| Spiral | Rotational force |
| Comminuted | >2 fragments |
| Segmental | Two separate fractures |
| Greenstick | Incomplete (pediatric) |
| Pathologic | Through abnormal bone |
| Displaced | Malalignment of fragments |
| Angulated | Deviation from normal axis |

## Open Fractures
| Gustilo-Anderson Classification | Management |
|--------------------------------|------------|
| Type I | <1 cm wound; Cefazolin |
| Type II | 1-10 cm wound; Cefazolin |
| Type IIIA | >10 cm; Adequate soft tissue coverage; Cefazolin + Gentamicin |
| Type IIIB | Inadequate soft tissue coverage; Cefazolin + Gentamicin |
| Type IIIC | Vascular injury requiring repair; Cefazolin + Gentamicin + Penicillin |

## Compartment Syndrome
| Finding | Management |
|---------|------------|
| 5 P's | Pain out of proportion, Pain with passive stretch, Paresthesia, Pallor, Pulselessness (late) |
| Diagnosis | Compartment pressure >30 mmHg OR Delta P <30 mmHg |
| Treatment | **Emergent fasciotomy** |

## Upper Extremity Injuries
| Injury | Key Finding | Management |
|--------|-------------|------------|
| Clavicle Fracture | Tenting of skin | Sling; Surgery if open/displaced |
| Shoulder Dislocation (Anterior) | Arm abducted/externally rotated | Reduction; Post-reduction x-ray |
| Shoulder Dislocation (Posterior) | Arm adducted/internally rotated | Reduction (associated with seizures) |
| Proximal Humerus Fracture | Pain, Swelling | Sling; Ortho referral |
| Humeral Shaft Fracture | Deformity | Coaptation splint; Monitor radial nerve |
| Supracondylar Fracture | Pediatric; Fall on outstretched hand | **Emergent ortho** (risk of compartment syndrome) |
| Radial Head Fracture | Pain with supination/pronation | Sling; Early ROM |
| Olecranon Fracture | Inability to extend elbow | Surgery if displaced |
| Elbow Dislocation | Posterior deformity | Reduction; Neurovascular exam |
| Monteggia Fracture | Ulna fracture + Radial head dislocation | Surgery |
| Galeazzi Fracture | Radius fracture + DRUJ dislocation | Surgery |
| Colles Fracture | Distal radius; Dorsal angulation | Reduction; Splint |
| Smith Fracture | Distal radius; Volar angulation | Reduction; Splint |
| Scaphoid Fracture | Snuffbox tenderness | Thumb spica cast (even if x-ray negative) |
| Lunate/Perilunate Dislocation | Wrist pain; X-ray findings | Emergent ortho (risk of AVN) |
| Boxer's Fracture | 5th metacarpal neck | Reduction if angulated; Ulnar gutter splint |

## Lower Extremity Injuries
| Injury | Key Finding | Management |
|--------|-------------|------------|
| Hip Dislocation | Posterior: Flexed/adducted/internally rotated | **Emergent reduction** (<6 hrs to prevent AVN) |
| Femoral Neck Fracture | Shortened/externally rotated | Surgery; Ortho admission |
| Intertrochanteric Fracture | Similar presentation | Surgery; Ortho admission |
| Femoral Shaft Fracture | High-energy; Blood loss | Traction splint; Surgery |
| Knee Dislocation | Gross deformity; Vascular injury | **Vascular emergency**; Reduction; ABI/CTA |
| Patellar Dislocation | Lateral displacement | Reduction; Knee immobilizer |
| Patellar Fracture | Unable to extend knee | Surgery if displaced |
| Tibial Plateau Fracture | Axial load; Ligament injury | CT; Ortho consultation |
| Ankle Fracture | Weber classification | Splint; Ortho referral |
| Maisonneuve Fracture | Proximal fibula fracture + Ankle injury | **Look for ankle injury**; Ortho referral |
| Calcaneus Fracture | Fall from height | **Evaluate lumbar spine**; Ortho referral |
| Lisfranc Injury | Midfoot pain; Widening on x-ray | Non-weight bearing; Surgery |
| Jones Fracture | 5th metatarsal base (zone 2) | Non-weight bearing; Surgery (risk of nonunion) |
| Achilles Tendon Rupture | Positive Thompson test | Splint in plantarflexion; Ortho referral |

## Pediatric Orthopedic Considerations
| Injury | Key Finding | Management |
|--------|-------------|------------|
| Salter-Harris Classification | Growth plate fractures | Type I-II: Splint; Type III-V: Surgery |
| Nursemaid's Elbow | Pulled elbow; Refuses to use arm | Hyperpronation reduction |
| Toddler's Fracture | Spiral tibia fracture | Cast; Ortho follow-up |
| SCFE | Obese adolescent; Hip/knee pain | Non-weight bearing; **Emergent ortho** |
| Legg-Calve-Perthes | Hip pain; Limp | Non-weight bearing; Ortho referral |

## Septic Arthritis
| Finding | Management |
|---------|------------|
| Presentation | Joint pain, Fever, Limited ROM |
| Diagnosis | Arthrocentesis (WBC >50,000, PMN >75%) |
| Treatment | **Emergent** IV antibiotics; Ortho consultation for washout |
