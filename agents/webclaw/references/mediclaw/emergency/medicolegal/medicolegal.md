# Emergency Medicine - Medicolegal Issues

## EMTALA (Emergency Medical Treatment and Labor Act)
| Requirement | Description |
|-------------|-------------|
| Medical Screening Exam | All patients presenting to ED must receive MSE |
| Stabilization | Stabilize emergency medical conditions before transfer |
| Transfer | Only after stabilization or if benefits outweigh risks |
| No Dumping | Cannot transfer for financial reasons |

## Informed Consent
| Component | Description |
|-----------|-------------|
| Capacity | Patient understands and can communicate decision |
| Voluntariness | Free from coercion |
| Information | Risks, Benefits, Alternatives explained |
| Exceptions | Emergency, Waiver, Therapeutic privilege |

## Mandatory Reporting
| Condition | Report To |
|-----------|-----------|
| Child Abuse | Child Protective Services |
| Elder Abuse | Adult Protective Services |
| Domestic Violence | Law enforcement (varies by state) |
| Gunshot/Stab Wounds | Law enforcement |
| Communicable Diseases | Public Health Department |
| Impaired Drivers | DMV (varies by state) |

## Restraints
| Requirement | Description |
|-------------|-------------|
| Indication | Imminent danger to self or others |
| Orders | Physician order within 1 hour |
| Reassessment | Every 1-2 hours (varies) |
| Documentation | Reason, Type, Duration, Reassessment |

## AMA (Against Medical Advice)
| Component | Action |
|-----------|--------|
| Capacity Assessment | Document patient has capacity |
| Risks Explained | Document discussion of risks |
| Alternatives | Offer reasonable alternatives |
| Documentation | AMA form; Document refusal |

## HIPAA (Health Insurance Portability and Accountability Act)
| Component | Description |
|-----------|-------------|
| Privacy Rule | Protect patient health information |
| Permitted Disclosures | Treatment, Payment, Operations |
| Mandatory Disclosures | Patient request, HHS investigation |
| Breach Notification | Notify patients of breaches |

## Malpractice Elements
| Element | Description |
|---------|-------------|
| Duty | Provider-patient relationship established |
| Breach | Care fell below standard of care |
| Causation | Breach caused harm |
| Damages | Patient suffered injury |
