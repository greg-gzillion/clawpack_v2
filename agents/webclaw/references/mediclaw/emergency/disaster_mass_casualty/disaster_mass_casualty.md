# Emergency Medicine - Disaster and Mass Casualty Incidents

## Disaster Triage (SALT)
| Category | Color | Description | Priority |
|----------|-------|-------------|----------|
| Immediate | Red | Life-threatening; Can be stabilized | 1 |
| Delayed | Yellow | Serious; Can wait hours | 2 |
| Minimal | Green | Walking wounded | 3 |
| Expectant | Black | Unlikely to survive | 4 |

## START Triage (Simple Triage and Rapid Treatment)
| Assessment | Action |
|------------|--------|
| Able to walk? | Green (Minor) |
| Breathing? | If no, open airway -> If still no, Black |
| RR >30 or <10? | Red (Immediate) |
| Cap refill >2 sec or no radial pulse? | Red (Immediate) |
| Can follow commands? | If no, Red; If yes, Yellow (Delayed) |

## HAZMAT Principles
| Component | Description |
|-----------|-------------|
| Recognition | Identify hazard (placards, symptoms) |
| PPE Levels | A (full encapsulation), B (chemical splash), C (air-purifying), D (standard) |
| Decontamination | Remove clothing, Water irrigation |
| Zones | Hot (contaminated), Warm (decon), Cold (clean) |
