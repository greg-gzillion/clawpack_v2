# Emergency Medicine - Critical Care and Transport

## ICU Admission Criteria
| System | Criteria |
|--------|----------|
| Respiratory | Mechanical ventilation, PaO2/FiO2 <300, PaCO2 >50 with pH <7.30 |
| Cardiovascular | Vasopressor requirement, MAP <65 despite fluids |
| Neurologic | GCS <9, Status epilepticus, Increased ICP |
| Other | Severe metabolic derangements, Active bleeding requiring transfusion |

## Interfacility Transport
| Component | Considerations |
|-----------|----------------|
| Pre-transport | Stabilize patient, Secure airway, Lines/tubes secured |
| During Transport | Continuous monitoring, Medications available |
| Handoff | SBAR communication, Documentation |

## Ventilator Management During Transport
| Setting | Recommendation |
|---------|----------------|
| Mode | Assist-Control or SIMV |
| FiO2 | Same or higher than baseline |
| PEEP | Maintain same level |
| Alarms | Set appropriate limits |
