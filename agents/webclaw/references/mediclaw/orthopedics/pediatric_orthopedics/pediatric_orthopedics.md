# Orthopedics - Pediatric Orthopedics

## Developmental Dysplasia of the Hip (DDH)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Spectrum of hip instability** (dysplasia, Subluxation, Dislocation) | https://orthoinfo.aaos.org/en/diseases--conditions/developmental-dislocation-dysplasia-of-the-hip-ddh/ |
| **Risk Factors** | **Female**, **Breech presentation**, **Family history**, **Firstborn**, **Oligohydramnios** | https://orthoinfo.aaos.org/en/diseases--conditions/developmental-dislocation-dysplasia-of-the-hip-ddh/ |
| **Examination** | **Barlow** (dislocates), **Ortolani** (reduces), **Galeazzi sign** (unequal knee heights), **Asymmetric thigh folds**, **Limited abduction** | https://orthoinfo.aaos.org/en/diseases--conditions/developmental-dislocation-dysplasia-of-the-hip-ddh/ |
| **Diagnosis** | **Ultrasound** (<4-6 months), **X-ray** (>4-6 months) | https://orthoinfo.aaos.org/en/diseases--conditions/developmental-dislocation-dysplasia-of-the-hip-ddh/ |
| **Treatment (<6 months)** | **Pavlik harness** | https://orthoinfo.aaos.org/en/diseases--conditions/developmental-dislocation-dysplasia-of-the-hip-ddh/ |
| **Treatment (6-18 months)** | **Closed reduction + Spica cast** | https://orthoinfo.aaos.org/en/diseases--conditions/developmental-dislocation-dysplasia-of-the-hip-ddh/ |
| **Treatment (>18 months)** | **Open reduction + Femoral/Pelvic osteotomy** | https://orthoinfo.aaos.org/en/diseases--conditions/developmental-dislocation-dysplasia-of-the-hip-ddh/ |

## Legg-Calvé-Perthes Disease
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Idiopathic AVN of femoral head** in children (age 4-10) | https://orthoinfo.aaos.org/en/diseases--conditions/legg-calve-perthes-disease/ |
| **Symptoms** | **Limp**, **Hip or knee pain**, **Decreased ROM** (especially abduction and internal rotation) | https://orthoinfo.aaos.org/en/diseases--conditions/legg-calve-perthes-disease/ |
| **Diagnosis** | **X-ray** (may be normal early), **MRI** | https://orthoinfo.aaos.org/en/diseases--conditions/legg-calve-perthes-disease/ |
| **Treatment** | **Observation** (age <6, <50% involvement), **Bracing/Surgery** (age >8, >50% involvement) | https://orthoinfo.aaos.org/en/diseases--conditions/legg-calve-perthes-disease/ |

## Slipped Capital Femoral Epiphysis (SCFE)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Posterior and inferior displacement of femoral epiphysis** relative to metaphysis | https://orthoinfo.aaos.org/en/diseases--conditions/slipped-capital-femoral-epiphysis-scfe/ |
| **Demographics** | **Obese**, **Adolescent males** (10-16 years) | https://orthoinfo.aaos.org/en/diseases--conditions/slipped-capital-femoral-epiphysis-scfe/ |
| **Symptoms** | **Hip or knee pain**, **Limp**, **External rotation deformity**, **Decreased internal rotation** | https://orthoinfo.aaos.org/en/diseases--conditions/slipped-capital-femoral-epiphysis-scfe/ |
| **Diagnosis** | **X-ray** (AP and frog-leg lateral) - **"Ice cream slipping off cone"** | https://orthoinfo.aaos.org/en/diseases--conditions/slipped-capital-femoral-epiphysis-scfe/ |
| **Treatment** | **Emergent** (unstable) or **Urgent** (stable) **in situ screw fixation** | https://orthoinfo.aaos.org/en/diseases--conditions/slipped-capital-femoral-epiphysis-scfe/ |

|-----------|-------------|-----|
