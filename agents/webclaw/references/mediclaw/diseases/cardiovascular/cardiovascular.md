# Diseases - Cardiovascular Diseases

## Coronary Artery Disease (CAD)
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Atherosclerotic narrowing of coronary arteries causing myocardial ischemia | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Risk Factors | Hypertension, dyslipidemia, diabetes, smoking, obesity, family history | https://www.acc.org/guidelines |
| Stable Angina | Predictable chest pain with exertion; Relieved by rest/nitroglycerin | https://www.acc.org/guidelines |
| Acute Coronary Syndrome | STEMI, NSTEMI, unstable angina | https://www.acc.org/guidelines |
| Diagnosis | ECG, stress testing, coronary CTA, invasive angiography | https://www.acc.org/guidelines |
| Management | Antiplatelet, statin, beta-blocker, ACE-I/ARB; PCI or CABG | https://www.acc.org/guidelines |

| Feature | Description | URL |
|---------|-------------|-----|

| HFrEF | LVEF =40% | https://www.acc.org/guidelines |

## Hypertension
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | SBP =130 mmHg or DBP =80 mmHg (ACC/AHA) | https://www.acc.org/guidelines |
| Primary (Essential) | 90-95% of cases; Multifactorial | https://www.acc.org/guidelines |
| Secondary | Renal artery stenosis, primary aldosteronism, pheochromocytoma, OSA | https://www.acc.org/guidelines |

| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Supraventricular tachyarrhythmia with uncoordinated atrial activation | https://www.acc.org/guidelines |
| Classification | Paroxysmal (<7 days), Persistent (>7 days), Long-standing persistent (>12 months), Permanent | https://www.acc.org/guidelines |

## Peripheral Artery Disease (PAD)
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Atherosclerotic narrowing of non-coronary arteries | https://www.acc.org/guidelines |
| Symptoms | Claudication, critical limb ischemia (rest pain, tissue loss) | https://www.acc.org/guidelines |
| Diagnosis | ABI =0.90; Duplex ultrasound; CTA/MRA | https://www.acc.org/guidelines |
| Management | Antiplatelet (aspirin or clopidogrel), statin, smoking cessation, exercise | https://www.acc.org/guidelines |
| Revascularization | Endovascular or surgical for disabling claudication or CLI | https://www.acc.org/guidelines |

## Venous Thromboembolism (VTE)
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Deep vein thrombosis (DVT) and/or pulmonary embolism (PE) | https://www.acc.org/guidelines |
| Risk Factors | Virchow's triad: Stasis, hypercoagulability, endothelial injury | https://www.ncbi.nlm.nih.gov/books/NBK470371/ |
| DVT Symptoms | Unilateral leg swelling, pain, erythema, warmth | https://www.acc.org/guidelines |
| PE Symptoms | Dyspnea, pleuritic chest pain, hemoptysis, syncope | https://www.acc.org/guidelines |
| Diagnosis | D-dimer, duplex ultrasound (DVT), CTPA (PE) | https://www.acc.org/guidelines |
| Treatment | Anticoagulation (DOAC, warfarin, LMWH); Thrombolysis for massive PE | https://www.acc.org/guidelines |
