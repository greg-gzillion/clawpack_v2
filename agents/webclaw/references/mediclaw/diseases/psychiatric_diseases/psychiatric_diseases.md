# Diseases - Psychiatric Diseases

## Major Depressive Disorder (MDD)
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5 Criteria** | =5 symptoms for =2 weeks, including depressed mood OR anhedonia | https://www.nimh.nih.gov/health/topics/depression |
| **Symptoms** | **SIGECAPS**: Sleep, Interest, Guilt, Energy, Concentration, Appetite, Psychomotor, Suicidality | https://www.nimh.nih.gov/health/topics/depression |
| **Treatment** | **Psychotherapy** (CBT, IPT), **Antidepressants** (SSRIs first-line), Neuromodulation (ECT, TMS) | https://www.nimh.nih.gov/health/topics/depression |
| **Treatment-Resistant** | Failure of =2 adequate trials | Augmentation (atypical antipsychotics, lithium), ECT, Ketamine/Esketamine | https://www.nimh.nih.gov/health/topics/depression |

## Schizophrenia
| Component | Description | URL |
|-----------|-------------|-----|
| **DSM-5 Criteria** | =2 symptoms for =1 month (=6 months total), including =1 core symptom | https://www.nimh.nih.gov/health/topics/schizophrenia |
| **Positive Symptoms** | **Hallucinations** (auditory most common), **Delusions**, Disorganized speech/behavior | https://www.nimh.nih.gov/health/topics/schizophrenia |
| **Negative Symptoms** | **Affective flattening**, **Alogia**, **Avolition**, **Anhedonia**, **Asociality** | https://www.nimh.nih.gov/health/topics/schizophrenia |
| **Cognitive Symptoms** | Impaired attention, Memory, Executive function | https://www.nimh.nih.gov/health/topics/schizophrenia |
| **Treatment** | **Antipsychotics** (atypical preferred), Psychosocial interventions | https://www.nimh.nih.gov/health/topics/schizophrenia |
