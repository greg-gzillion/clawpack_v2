# Diseases - Renal Diseases

## Acute Kidney Injury (AKI)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Abrupt decrease in kidney function (hours to days) | https://www.niddk.nih.gov/health-information/kidney-disease/acute-kidney-injury |
| **KDIGO Criteria** | ? Cr =0.3 mg/dL in 48h OR ? Cr =1.5x baseline in 7d OR UOP <0.5 mL/kg/h x6h | https://kdigo.org/guidelines/acute-kidney-injury/ |
| **Stages** | Stage 1: Cr 1.5-1.9x; Stage 2: Cr 2.0-2.9x; Stage 3: Cr =3.0x or RRT | https://kdigo.org/guidelines/acute-kidney-injury/ |

## Chronic Kidney Disease (CKD)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Kidney damage or GFR <60 mL/min/1.73m˛ for =3 months | https://www.niddk.nih.gov/health-information/kidney-disease/chronic-kidney-disease-ckd |
| **Etiology** | **Diabetes** (40%), **Hypertension** (30%), Glomerulonephritis | https://www.cdc.gov/kidneydisease/basics.html |
| **Staging (KDIGO)** | G1: =90; G2: 60-89; G3a: 45-59; G3b: 30-44; G4: 15-29; G5: <15 | https://kdigo.org/guidelines/ckd-evaluation-and-management/ |
| **Complications** | Anemia, Bone-mineral disorder, Hyperkalemia, Acidosis, CVD | https://www.niddk.nih.gov/health-information/kidney-disease/chronic-kidney-disease-ckd |
| **Management** | **BP control** (<130/80), **RAAS blockade**, **SGLT2i**, Treat complications | https://kdigo.org/guidelines/ckd-evaluation-and-management/ |
