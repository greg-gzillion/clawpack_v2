# Diseases - Respiratory Diseases

## Chronic Obstructive Pulmonary Disease (COPD)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Progressive airflow limitation, not fully reversible | https://www.nhlbi.nih.gov/health/copd |
| **Etiology** | **Smoking** (80-90%), Alpha-1 antitrypsin deficiency | https://www.cdc.gov/copd/index.html |
| **Types** | **Chronic bronchitis** (productive cough =3 months/year x2 years), **Emphysema** (alveolar destruction) | https://www.nhlbi.nih.gov/health/copd |
| **Diagnosis** | **FEV1/FVC <0.70** post-bronchodilator | https://goldcopd.org |
| **GOLD Classification** | GOLD 1: FEV1 =80%, GOLD 2: 50-79%, GOLD 3: 30-49%, GOLD 4: <30% | https://goldcopd.org |
| **Treatment** | Bronchodilators (LAMA/LABA), ICS (if eosinophilic), Pulmonary rehab, Oxygen | https://goldcopd.org |

## Asthma
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Chronic inflammatory airway disease with variable airflow obstruction | https://www.nhlbi.nih.gov/health/asthma |
| **Triggers** | Allergens, Infections, Exercise, Cold air, Irritants | https://www.cdc.gov/asthma/triggers.html |
| **Diagnosis** | Symptoms + **=12% and =200 mL reversibility** or positive bronchial challenge | https://www.nhlbi.nih.gov/health/asthma/diagnosis |
| **Treatment** | Stepwise: ICS (controller), SABA (reliever); Biologics for severe | https://ginasthma.org |
