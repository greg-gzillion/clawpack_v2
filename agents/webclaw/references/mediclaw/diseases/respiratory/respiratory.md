# Diseases - Respiratory Diseases

## Chronic Obstructive Pulmonary Disease (COPD)
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Progressive, irreversible airflow limitation | https://goldcopd.org |
| Etiology | Tobacco smoking (80-90%); Alpha-1 antitrypsin deficiency; Biomass fuel | https://goldcopd.org |
| Pathophysiology | Chronic bronchitis (mucus hypersecretion) + Emphysema (alveolar destruction) | https://www.ncbi.nlm.nih.gov/books/NBK470357/ |
| Diagnosis | FEV1/FVC <0.70 post-bronchodilator | https://goldcopd.org |
| GOLD Stages | 1: FEV1 =80%; 2: 50-79%; 3: 30-49%; 4: <30% | https://goldcopd.org |
| GOLD Groups | A (low symptoms, low risk), B (high symptoms, low risk), E (high risk) | https://goldcopd.org |
| Pharmacotherapy | LAMA, LABA, ICS (for eosinophils =300 or hospitalizations) | https://goldcopd.org |
| Non-Pharmacologic | Smoking cessation, pulmonary rehabilitation, oxygen therapy | https://goldcopd.org |

## Asthma
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Chronic inflammatory airway disease with variable expiratory airflow limitation | https://ginasthma.org |
| Pathophysiology | Airway inflammation, bronchial hyperresponsiveness, reversible obstruction | https://ginasthma.org |
| Symptoms | Wheezing, dyspnea, chest tightness, cough (worse at night/early morning) | https://ginasthma.org |
| Diagnosis | Spirometry with bronchodilator reversibility (FEV1 increase =12% and =200 mL) | https://ginasthma.org |
| Severity | Intermittent, Mild persistent, Moderate persistent, Severe persistent | https://ginasthma.org |
| Stepwise Therapy | Step 1: PRN SABA; Step 2: Low-dose ICS; Step 3: Low-dose ICS-LABA; Step 4: Medium-dose ICS-LABA; Step 5: Add-on biologics | https://ginasthma.org |
| Biologics | Omalizumab (anti-IgE), Mepolizumab/Reslizumab/Benralizumab (anti-IL-5), Dupilumab (anti-IL-4/13), Tezepelumab (anti-TSLP) | https://ginasthma.org |

## Community-Acquired Pneumonia (CAP)
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Acute infection of lung parenchyma acquired outside healthcare setting | https://www.idsociety.org/practice-guideline/community-acquired-pneumonia-cap/ |
| Common Pathogens | S. pneumoniae, H. influenzae, M. pneumoniae, C. pneumoniae, Respiratory viruses | https://www.idsociety.org |
| Symptoms | Fever, cough, sputum, dyspnea, pleuritic chest pain | https://www.idsociety.org |
| Diagnosis | CXR (infiltrate), CBC, procalcitonin, sputum/blood cultures | https://www.idsociety.org |
| Risk Stratification | CURB-65, PSI/PORT score | https://www.mdcalc.com/curb-65-score-pneumonia-severity |
| Outpatient Treatment | Amoxicillin, Doxycycline, Macrolide (low-risk); Amoxicillin-clavulanate + macrolide or Respiratory FQ (comorbidities) | https://www.idsociety.org |
| Inpatient Treatment | Beta-lactam + macrolide or Respiratory FQ | https://www.idsociety.org |
| Duration | 5-7 days (clinical stability) | https://www.idsociety.org |

## Hospital-Acquired/Ventilator-Associated Pneumonia (HAP/VAP)
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Pneumonia occurring =48 hours after hospital admission or intubation | https://www.idsociety.org/practice-guideline/hap-vap/ |
| Common Pathogens | MRSA, Pseudomonas, Acinetobacter, Enterobacteriaceae | https://www.idsociety.org |
| Risk Factors for MDR | Prior IV antibiotics within 90 days, septic shock, ARDS, renal replacement therapy | https://www.idsociety.org |
| Treatment | Anti-MRSA + Anti-pseudomonal beta-lactam; De-escalate based on cultures | https://www.idsociety.org |
