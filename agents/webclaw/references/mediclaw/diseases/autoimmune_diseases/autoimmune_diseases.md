## Systemic Lupus Erythematosus (SLE)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Multisystem autoimmune disease with autoantibodies against nuclear antigens | https://www.niams.nih.gov/health-topics/lupus |
| **Clinical Features** | **Malar rash**, Discoid rash, Photosensitivity, Oral ulcers, Arthritis, Serositis, Renal, CNS, Hematologic | https://www.lupus.org |
| **Diagnosis** | **ANA** (sensitive, not specific), **Anti-dsDNA**, **Anti-Smith**, ? C3/C4 | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Lupus |
| **SLICC/ACR Criteria** | =4 criteria (=1 clinical + =1 immunologic) OR **Biopsy-proven lupus nephritis + ANA or anti-dsDNA** | https://www.rheumatology.org/Portals/0/Files/Classification-Criteria-Systemic-Lupus-Erythematosus.pdf |
| **Treatment** | **Hydroxychloroquine** (all patients), Corticosteroids, Immunosuppressants (MMF, AZA, CYC), Belimumab | https://www.rheumatology.org/Practice-Quality/Clinical-Support/Clinical-Practice-Guidelines |

## Antiphospholipid Syndrome (APS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Autoantibodies against phospholipid-binding proteins causing thrombosis and/or pregnancy morbidity | https://www.nhlbi.nih.gov/health/antiphospholipid-syndrome |
| **Clinical Criteria** | **Vascular thrombosis** (arterial, venous, small vessel), **Pregnancy morbidity** (=3 early losses, =1 late loss, Preterm for severe preeclampsia) | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Antiphospholipid-Syndrome |
| **Laboratory Criteria** | **Lupus anticoagulant**, **Anti-cardiolipin** (IgG/IgM), **Anti-ß2-glycoprotein I** (IgG/IgM); Positive x2 =12 weeks apart | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Antiphospholipid-Syndrome |
| **Treatment** | **Anticoagulation** (warfarin, DOACs less effective), Aspirin ± Anticoagulation in pregnancy | https://www.rheumatology.org/Practice-Quality/Clinical-Support/Clinical-Practice-Guidelines |

## Sjögren Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Autoimmune exocrinopathy causing sicca symptoms | https://www.niams.nih.gov/health-topics/sjogrens-syndrome |
| **Clinical Features** | **Dry eyes** (keratoconjunctivitis sicca), **Dry mouth** (xerostomia), Parotid enlargement, Extraglandular manifestations | https://www.sjogrens.org |
| **Diagnosis** | **Anti-SSA/Ro**, **Anti-SSB/La**, **Schirmer test**, Salivary gland biopsy (focal lymphocytic sialadenitis) | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Sjogrens-Syndrome |
| **Treatment** | Symptomatic (artificial tears, pilocarpine), Hydroxychloroquine, Immunosuppression for systemic disease | https://www.sjogrens.org/understanding-sjogrens/treatment |
