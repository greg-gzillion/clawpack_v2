# Diseases - Musculoskeletal Diseases

## Osteoarthritis (OA)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Degenerative joint disease with cartilage loss and bony changes | https://www.niams.nih.gov/health-topics/osteoarthritis |
| **Risk Factors** | **Age**, **Obesity**, Female, Joint injury, Genetics | https://www.cdc.gov/arthritis/basics/osteoarthritis.htm |
| **Clinical Features** | **Use-related pain**, **Morning stiffness <30 min**, Crepitus, Bony enlargement | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Osteoarthritis |
| **Common Joints** | **DIP (Heberden nodes)** , **PIP (Bouchard nodes)** , **First CMC**, Knees, Hips, Spine | https://www.niams.nih.gov/health-topics/osteoarthritis |

## Gout
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Inflammatory arthritis due to monosodium urate crystal deposition | https://www.niams.nih.gov/health-topics/gout |
| **Pathophysiology** | **Hyperuricemia** ? Urate crystal deposition ? Acute inflammation | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Gout |
| **Clinical Features** | **Acute monoarthritis** (often **first MTP - podagra**), Severe pain, Erythema, Warmth | https://www.niams.nih.gov/health-topics/gout |
| **Diagnosis** | **Joint aspiration** (negatively birefringent needle-shaped crystals), Clinical + Hyperuricemia | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Gout |
| **Acute Treatment** | **NSAIDs**, **Colchicine**, **Corticosteroids** | https://www.rheumatology.org/Practice-Quality/Clinical-Support/Clinical-Practice-Guidelines |
| **Chronic Treatment** | **Allopurinol** (first-line), Febuxostat; **Target uric acid <6 mg/dL** (<5 if tophi) | https://www.rheumatology.org/Practice-Quality/Clinical-Support/Clinical-Practice-Guidelines |

## Osteoporosis
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Reduced bone mass and microarchitectural deterioration leading to fragility fractures | https://www.niams.nih.gov/health-topics/osteoporosis |
| **Risk Factors** | **Postmenopausal**, Age, **Corticosteroids**, Low BMI, Smoking, Alcohol | https://www.bonehealthandosteoporosis.org |
| **Screening** | **DXA scan**: Women =65, Men =70, Younger with risk factors | https://www.uspreventiveservicestaskforce.org/uspstf/recommendation/osteoporosis-screening |
| **DXA Interpretation** | **T-score**: Normal (>-1), Osteopenia (-1 to -2.5), Osteoporosis (<-2.5) | https://www.bonehealthandosteoporosis.org |
| **FRAX Tool** | Estimates 10-year fracture risk | https://frax.shef.ac.uk/FRAX/ |
| **Treatment** | **Bisphosphonates** (alendronate, risedronate, zoledronic acid), **Denosumab**, Teriparatide, Romosozumab | https://www.endocrine.org/clinical-practice-guidelines/osteoporosis |

## Fibromyalgia
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Chronic widespread pain with fatigue, sleep, and cognitive disturbances | https://www.niams.nih.gov/health-topics/fibromyalgia |
| **Pathophysiology** | **Central sensitization** (amplified pain processing) | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Fibromyalgia |
| **Diagnosis (ACR 2016)** | **Widespread Pain Index (WPI)** =7 and **Symptom Severity Scale (SSS)** =5 OR WPI 4-6 and SSS =9; Symptoms =3 months | https://www.rheumatology.org/Portals/0/Files/2016-Fibromyalgia-Diagnostic-Criteria.pdf |
| **Treatment** | **Exercise** (graded aerobic), **CBT**, **Medications** (duloxetine, milnacipran, pregabalin, amitriptyline) | https://www.rheumatology.org/Practice-Quality/Clinical-Support/Clinical-Practice-Guidelines |

## Ankylosing Spondylitis (Axial Spondyloarthritis)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Chronic inflammatory arthritis primarily affecting the axial skeleton | https://www.niams.nih.gov/health-topics/ankylosing-spondylitis |
| **Clinical Features** | **Inflammatory back pain** (improves with activity, worsens with rest), Morning stiffness, Enthesitis, Uveitis | https://spondylitis.org |
| **Diagnosis** | **Sacroliitis on MRI/X-ray**, **HLA-B27** (90% positive) | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Spondyloarthritis |
| **Treatment** | **NSAIDs**, **TNF inhibitors** (adalimumab, etanercept, infliximab), **IL-17 inhibitors** (secukinumab, ixekizumab) | https://www.rheumatology.org/Practice-Quality/Clinical-Support/Clinical-Practice-Guidelines |

## Psoriatic Arthritis (PsA)

## Septic Arthritis
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Bacterial infection of a joint | https://www.niams.nih.gov/health-topics/infectious-arthritis |
| **Risk Factors** | Prosthetic joint, RA, Diabetes, Immunocompromise, IVDU | https://www.rheumatology.org/I-Am-A/Patient-Caregiver/Diseases-Conditions/Infectious-Arthritis |

| **Treatment** | **Emergent joint drainage** (serial aspiration or arthroscopy), **IV antibiotics** | https://www.idsociety.org/practice-guideline/native-joint-infections/ |
