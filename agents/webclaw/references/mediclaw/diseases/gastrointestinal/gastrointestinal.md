# Diseases - Gastrointestinal Diseases

## Gastroesophageal Reflux Disease (GERD)
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Reflux of gastric contents causing troublesome symptoms or complications | https://www.gastro.org/guidelines |
| Symptoms | Heartburn, regurgitation, dysphagia, chest pain | https://www.gastro.org/guidelines |
| Alarm Features | Dysphagia, odynophagia, weight loss, anemia, GI bleeding | https://www.gastro.org/guidelines |
| Diagnosis | Clinical; EGD if alarm features or refractory to PPI | https://www.gastro.org/guidelines |
| Treatment | Lifestyle modifications; PPI (8 weeks); H2RA if PPI contraindicated | https://www.gastro.org/guidelines |
| Complications | Erosive esophagitis, stricture, Barrett's esophagus, adenocarcinoma | https://www.gastro.org/guidelines |

## Barrett's Esophagus
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Metaplastic change of distal esophageal epithelium from squamous to columnar | https://www.gastro.org/guidelines |
| Risk Factors | Chronic GERD, male, age >50, obesity, smoking | https://www.gastro.org/guidelines |
| Diagnosis | EGD with biopsy; =1 cm of salmon-colored mucosa | https://www.gastro.org/guidelines |
| Dysplasia Risk | Non-dysplastic (0.3%/year), Low-grade (0.5%/year), High-grade (7%/year) | https://www.gastro.org/guidelines |
| Surveillance | EGD q3-5 years (non-dysplastic); q6-12 months (LGD); EMR/ablation (HGD) | https://www.gastro.org/guidelines |

## Peptic Ulcer Disease (PUD)
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Mucosal defect >5 mm in stomach or duodenum | https://www.gastro.org/guidelines |
| Etiology | H. pylori (70%), NSAIDs (25%), idiopathic (5%) | https://www.gastro.org/guidelines |
| Symptoms | Epigastric pain (duodenal: relieved by food; gastric: worsened by food) | https://www.gastro.org/guidelines |
| Diagnosis | EGD (gold standard); H. pylori testing | https://www.gastro.org/guidelines |
| Treatment | PPI; H. pylori eradication; Discontinue NSAIDs | https://www.gastro.org/guidelines |
| Complications | Bleeding, perforation, gastric outlet obstruction | https://www.gastro.org/guidelines |

## Irritable Bowel Syndrome (IBS)
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Functional GI disorder with abdominal pain and altered bowel habits | https://www.gastro.org/guidelines |
| Rome IV Criteria | Recurrent abdominal pain =1 day/week for 3 months + =2 of: related to defecation, change in stool frequency, change in stool form | https://www.gastro.org/guidelines |
| Subtypes | IBS-C (constipation), IBS-D (diarrhea), IBS-M (mixed), IBS-U (unspecified) | https://www.gastro.org/guidelines |
| IBS-C Treatment | Fiber, PEG, Lubiprostone, Linaclotide, Plecanatide, Tenapanor | https://www.gastro.org/guidelines |
| IBS-D Treatment | Loperamide, Rifaximin, Eluxadoline, Alosetron (severe, women) | https://www.gastro.org/guidelines |
| Abdominal Pain | Antispasmodics, TCA, SSRI | https://www.gastro.org/guidelines |

## Celiac Disease
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Immune-mediated enteropathy triggered by dietary gluten | https://www.gastro.org/guidelines |
| Genetics | HLA-DQ2 (95%), HLA-DQ8 (5%) | https://www.gastro.org/guidelines |
| Symptoms | Diarrhea, steatorrhea, weight loss, abdominal pain, iron deficiency anemia | https://www.gastro.org/guidelines |
| Diagnosis | IgA anti-tTG antibody (screening); Duodenal biopsy (villous atrophy, crypt hyperplasia, IELs) | https://www.gastro.org/guidelines |
| Treatment | Strict gluten-free diet (lifelong) | https://www.gastro.org/guidelines |
| Complications | Refractory celiac disease, enteropathy-associated T-cell lymphoma | https://www.gastro.org/guidelines |
