# Allergy Medicine - Asthma

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Chronic inflammatory airway disease with variable airflow obstruction | https://www.nhlbi.nih.gov/health/asthma |
| **Prevalence** | 25 million Americans; 8% of adults, 7% of children | https://www.cdc.gov/asthma/most_recent_national_asthma_data.htm |

## Diagnosis
| Criteria | Description | URL |
|----------|-------------|-----|
| **Symptoms** | Wheeze, dyspnea, chest tightness, cough (worse at night/early morning) | https://www.nhlbi.nih.gov/health/asthma/diagnosis |
| **Spirometry** | **FEV1/FVC < LLN** with **=12% and =200 mL reversibility** post-bronchodilator | https://www.nhlbi.nih.gov/health/asthma/diagnosis |
| **Peak Flow Variability** | **>10% diurnal variability** | https://www.nhlbi.nih.gov/health/asthma/diagnosis |
| **Bronchial Challenge** | Methacholine (PC20 <8 mg/mL) or exercise challenge | https://www.nhlbi.nih.gov/health/asthma/diagnosis |
