# Allergy Medicine - Allergic Rhinitis

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | IgE-mediated inflammation of nasal mucosa | https://www.aaaai.org/conditions-treatments/allergies/rhinitis |
| **Prevalence** | Affects 10-30% of adults, up to 40% of children | https://www.cdc.gov/nchs/fastats/allergies.htm |

## Classification (ARIA Guidelines)
| Type | Definition | URL |
|------|------------|-----|
| **Intermittent** | Symptoms <4 days/week OR <4 weeks | https://www.euforea.eu/aria |
| **Persistent** | Symptoms >4 days/week AND >4 weeks | https://www.euforea.eu/aria |
| **Mild** | Normal sleep, daily activities, work/school | https://www.euforea.eu/aria |
| **Moderate-Severe** | Impaired sleep, activities, or work/school | https://www.euforea.eu/aria |

## Clinical Features
| Symptom | Characteristics | URL |
|---------|-----------------|-----|
| **Sneezing** | Paroxysmal, especially in morning | https://www.mayoclinic.org/diseases-conditions/hay-fever/symptoms-causes/syc-20373039 |
| **Rhinorrhea** | Clear, watery discharge | https://www.mayoclinic.org/diseases-conditions/hay-fever/symptoms-causes/syc-20373039 |
| **Nasal Congestion** | Alternating, worse at night | https://www.mayoclinic.org/diseases-conditions/hay-fever/symptoms-causes/syc-20373039 |
| **Nasal Itching** | Often involves palate and ears | https://www.mayoclinic.org/diseases-conditions/hay-fever/symptoms-causes/syc-20373039 |
| **Ocular Symptoms** | Itching, redness, tearing (allergic conjunctivitis) | https://www.mayoclinic.org/diseases-conditions/hay-fever/symptoms-causes/syc-20373039 |

| Finding | Description | URL |
|---------|-------------|-----|

## Treatment - Stepwise Approach
| Step | Treatment | URL |
|------|-----------|-----|
| **1. Allergen Avoidance** | Environmental controls (HEPA filters, dust mite covers) | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/indoor-allergens |
| **2. Oral Antihistamines** | Second-generation: Cetirizine, Loratadine, Fexofenadine | https://www.fda.gov/drugs/information-drug-class/antihistamines |
| **3. Intranasal Corticosteroids** | **First-line**: Fluticasone, Mometasone, Budesonide | https://www.aaaai.org/conditions-treatments/allergies/rhinitis |
| **4. Intranasal Antihistamines** | Azelastine, Olopatadine | https://www.aaaai.org/conditions-treatments/allergies/rhinitis |
| **5. Leukotriene Receptor Antagonists** | Montelukast (especially with asthma) | https://www.fda.gov/drugs |
| **6. Immunotherapy** | SCIT or SLIT | https://www.aaaai.org/conditions-treatments/allergies/allergy-immunotherapy |

## Allergen Immunotherapy
| Type | Description | URL |
|------|-------------|-----|
| **Subcutaneous Immunotherapy (SCIT)** | Injections in clinic, 3-5 years | https://www.aaaai.org/conditions-treatments/allergies/allergy-immunotherapy |
| **Sublingual Immunotherapy (SLIT)** | Daily tablets at home | https://www.fda.gov/vaccines-blood-biologics/allergenics |
| **SLIT Tablets Available** | Grass (Oralair, Grastek), Ragweed (Ragwitek), Dust mite (Odactra) | https://www.fda.gov/vaccines-blood-biologics/allergenics |
