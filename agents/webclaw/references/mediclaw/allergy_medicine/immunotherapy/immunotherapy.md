# Allergy Medicine - Immunotherapy

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | Administration of increasing doses of allergen to induce tolerance | https://www.aaaai.org/conditions-treatments/allergies/allergy-immunotherapy |
| **Mechanism** | Shift from Th2 ? Th1/Treg, Induction of blocking IgG4 antibodies | https://www.aaaai.org/conditions-treatments/allergies/allergy-immunotherapy |
| **Indications** | Allergic rhinitis, Allergic asthma, Hymenoptera venom allergy | https://www.aaaai.org/conditions-treatments/allergies/allergy-immunotherapy |

## Subcutaneous Immunotherapy (SCIT)
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | Moderate-severe allergic rhinitis, Allergic asthma, Venom allergy | https://www.aaaai.org/conditions-treatments/allergies/allergy-immunotherapy |
| **Build-Up Phase** | Weekly injections, Increasing doses (4-8 months) | https://www.aaaai.org/conditions-treatments/allergies/allergy-immunotherapy |
| **Maintenance Phase** | Monthly injections, 3-5 years | https://www.aaaai.org/conditions-treatments/allergies/allergy-immunotherapy |
| **Efficacy** | 80-90% improvement in symptoms | https://www.aaaai.org/conditions-treatments/allergies/allergy-immunotherapy |
| **Contraindications** | Severe/uncontrolled asthma, Beta-blocker use, Pregnancy (initiation) | https://www.aaaai.org/conditions-treatments/allergies/allergy-immunotherapy |

## SCIT Safety
| Component | Description | URL |
|-----------|-------------|-----|
| **Pre-Injection** | Assess current asthma control, Recent illness, Prior reactions | https://www.aaaai.org/practice-resources/running-a-practice/immunotherapy-safety |
| **30-Minute Observation** | Required post-injection | https://www.aaaai.org/practice-resources/running-a-practice/immunotherapy-safety |
| **Systemic Reaction Rate** | 0.1% of injections (mostly mild) | https://www.aaaai.org/practice-resources/running-a-practice/immunotherapy-safety |
| **Fatalities** | Extremely rare (1 per 2.5 million injections) | https://www.aaaai.org/practice-resources/running-a-practice/immunotherapy-safety |
| **Emergency Equipment** | Epinephrine, Antihistamines, Oxygen, Resuscitation equipment | https://www.aaaai.org/practice-resources/running-a-practice/immunotherapy-safety |

## Sublingual Immunotherapy (SLIT)
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | Oral mucosal tolerance induction | https://www.fda.gov/vaccines-blood-biologics/allergenics |
| **FDA-Approved Tablets** | Oralair (5-grass), Grastek (Timothy grass), Ragwitek (Ragweed), Odactra (Dust mite) | https://www.fda.gov/vaccines-blood-biologics/allergenics |
| **Administration** | Daily at home, First dose in clinic | https://www.fda.gov/vaccines-blood-biologics/allergenics |
| **Duration** | Pre-seasonal (grass), Year-round (dust mite) | https://www.fda.gov/vaccines-blood-biologics/allergenics |
| **Efficacy** | 20-40% improvement in symptoms | https://www.fda.gov/vaccines-blood-biologics/allergenics |
| **Safety** | Local oral pruritus common, Systemic reactions rare | https://www.fda.gov/vaccines-blood-biologics/allergenics |

## Venom Immunotherapy (VIT)
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | Systemic reaction to Hymenoptera sting + Positive venom IgE | https://www.aaaai.org/conditions-treatments/allergies/stinging-insect-allergy |
| **Venoms** | Honey bee, Yellow jacket, Yellow hornet, White-faced hornet, Wasp, Fire ant | https://www.aaaai.org/conditions-treatments/allergies/stinging-insect-allergy |
| **Duration** | 3-5 years (lifelong if severe reaction or high risk) | https://www.aaaai.org/conditions-treatments/allergies/stinging-insect-allergy |
| **Efficacy** | **95-98%** protection from systemic reactions | https://www.aaaai.org/conditions-treatments/allergies/stinging-insect-allergy |
| **Discontinuation** | May consider after 5 years if no reaction to sting | https://www.aaaai.org/conditions-treatments/allergies/stinging-insect-allergy |

## Food Oral Immunotherapy (OIT)
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | IgE-mediated food allergy (peanut, milk, egg) | https://www.fda.gov/vaccines-blood-biologics/allergenics/palforzia |
| **FDA-Approved** | **Palforzia** (peanut OIT) for ages 4-17 | https://www.fda.gov/vaccines-blood-biologics/allergenics/palforzia |
| **Goal** | Protection from accidental ingestion (NOT cure) | https://www.fda.gov/vaccines-blood-biologics/allergenics/palforzia |
| **Risks** | Anaphylaxis during dosing, Eosinophilic esophagitis | https://www.fda.gov/vaccines-blood-biologics/allergenics/palforzia |
