# Allergy Medicine - Diagnostic Testing

## Skin Prick Testing (SPT)
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | Suspected IgE-mediated allergy (inhalants, foods, venoms, drugs) | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |
| **Method** | Prick epidermis with allergen extract | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |
| **Positive Result** | **Wheal =3 mm > negative control** | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |
| **Sensitivity** | High (85-95%) | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |
| **Specificity** | Moderate (70-85%) | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |
| **Limitations** | Antihistamines, Severe eczema, Anaphylaxis risk | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |

## Intradermal Testing
| Component | Description | URL |
|-----------|-------------|-----|
| **Indications** | Hymenoptera venom, Drug allergy (penicillin), Inhalants (when SPT negative) | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |
| **Method** | Inject small volume intradermally | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |
| **Positive Result** | Wheal and flare at 15-20 minutes | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |
| **Sensitivity** | Higher than SPT | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |
| **Specificity** | Lower than SPT (more false positives) | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |

## Serum-Specific IgE Testing
| Component | Description | URL |
|-----------|-------------|-----|
| **Method** | ImmunoCAP, ELISA | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |
| **Positive Result** | **=0.35 kU/L** (class 1) | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |
| **Sensitivity** | 70-90% (compared to SPT) | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |
| **Advantages** | No antihistamine interference, No anaphylaxis risk, Quantitative | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |
| **Disadvantages** | Higher cost, Delayed results | https://www.aaaai.org/tools-for-the-public/conditions-library/allergies/allergy-testing |
