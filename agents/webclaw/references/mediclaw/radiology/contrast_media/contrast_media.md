# Radiology - Contrast Media

## Iodinated Contrast Media
| Component | Description | URL |
|-----------|-------------|-----|
| **Types** | **High-osmolar** (ionic), **Low-osmolar** (non-ionic - preferred), **Iso-osmolar** | https://www.acr.org/Clinical-Resources/Contrast-Manual |
| **Adverse Reactions** | **Allergic-like** (urticaria, Angioedema, Anaphylaxis), **Physiologic** (nausea, Vomiting, Flushing, Vasovagal) | https://www.acr.org/Clinical-Resources/Contrast-Manual |
| **Contrast-Induced Nephropathy (CIN)** | **AKI** within 48-72 hours; Risk: **eGFR <30** (or <45 with risk factors) | https://www.acr.org/Clinical-Resources/Contrast-Manual |
| **CIN Prevention** | **IV fluids** (preferred), **Hold nephrotoxins**, **Minimize contrast volume** | https://www.acr.org/Clinical-Resources/Contrast-Manual |
| **Metformin** | **Hold** if eGFR <30 (or <45 with risk factors); Resume after 48 hours if stable renal function | https://www.acr.org/Clinical-Resources/Contrast-Manual |

## Gadolinium-Based Contrast Agents (GBCAs)
| Component | Description | URL |
|-----------|-------------|-----|
| **Types** | **Linear** (higher NSF risk), **Macrocyclic** (lower NSF risk - preferred) | https://www.acr.org/Clinical-Resources/Contrast-Manual |
| **Nephrogenic Systemic Fibrosis (NSF)** | **Fibrosis of skin and organs** in patients with **severe renal impairment** (eGFR <30, Dialysis) | https://www.acr.org/Clinical-Resources/Contrast-Manual |
| **NSF Prevention** | **Avoid Group I (linear) agents** in high-risk patients; Use **Group II (macrocyclic)** agents with caution | https://www.acr.org/Clinical-Resources/Contrast-Manual |
| **Gadolinium Deposition** | **Deposition in brain** (dentate nucleus, Globus pallidus); **No known clinical sequelae** | https://www.fda.gov/drugs/drug-safety-and-availability/fda-drug-safety-communication-fda-warns-gadolinium-based-contrast-agents-gbcas-are-retained-body |

| Severity | Symptoms | Management | URL |
|----------|----------|------------|-----|
