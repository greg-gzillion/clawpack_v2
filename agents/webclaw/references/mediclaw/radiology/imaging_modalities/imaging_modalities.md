# Radiology - Imaging Modalities

## X-Ray (Radiography)
| Component | Description | URL |
|-----------|-------------|-----|
| **Principle** | **Differential absorption of X-rays** by tissues (attenuation) | https://www.nibib.nih.gov/science-education/science-topics/x-rays |
| **Densities** | **Air** (black), **Fat** (dark gray), **Soft tissue/Water** (gray), **Bone** (white), **Metal** (bright white) | https://www.nibib.nih.gov/science-education/science-topics/x-rays |

## Computed Tomography (CT)
| Component | Description | URL |
|-----------|-------------|-----|
| **Principle** | **Rotating X-ray tube** produces **cross-sectional images**; Measured in **Hounsfield Units (HU)** | https://www.nibib.nih.gov/science-education/science-topics/computed-tomography-ct |
| **HU Values** | **Air**: -1000, **Fat**: -100 to -50, **Water**: 0, **Soft tissue**: 20-100, **Bone**: >400, **Metal**: >1000 | https://www.nibib.nih.gov/science-education/science-topics/computed-tomography-ct |
| **Advantages** | **Excellent anatomic detail**, **Fast**, **Multiplanar reconstruction** | https://www.nibib.nih.gov/science-education/science-topics/computed-tomography-ct |
| **Disadvantages** | **Higher radiation**, **Iodinated contrast risks** (allergy, Nephrotoxicity) | https://www.nibib.nih.gov/science-education/science-topics/computed-tomography-ct |

## Magnetic Resonance Imaging (MRI)
| Component | Description | URL |
|-----------|-------------|-----|
| **Principle** | **Alignment of protons in magnetic field** ? **Radiofrequency pulse** ? **Relaxation** (T1, T2) | https://www.nibib.nih.gov/science-education/science-topics/magnetic-resonance-imaging-mri |
| **T1-Weighted** | **Fat bright**, **Water dark**; **Good for anatomy** | https://www.nibib.nih.gov/science-education/science-topics/magnetic-resonance-imaging-mri |
| **T2-Weighted** | **Water bright**, **Fat intermediate**; **Good for pathology** (edema, Inflammation) | https://www.nibib.nih.gov/science-education/science-topics/magnetic-resonance-imaging-mri |
| **FLAIR** | **T2 with CSF suppression**; **Good for periventricular lesions** | https://www.nibib.nih.gov/science-education/science-topics/magnetic-resonance-imaging-mri |
| **DWI/ADC** | **Water diffusion**; **Hyperacute stroke**, **Abscess**, **Cellular tumors** | https://www.nibib.nih.gov/science-education/science-topics/magnetic-resonance-imaging-mri |
| **Advantages** | **Excellent soft tissue contrast**, **No ionizing radiation** | https://www.nibib.nih.gov/science-education/science-topics/magnetic-resonance-imaging-mri |
| **Disadvantages** | **Longer scan time**, **Expensive**, **Contraindications** (pacemaker, Metal) | https://www.nibib.nih.gov/science-education/science-topics/magnetic-resonance-imaging-mri |
