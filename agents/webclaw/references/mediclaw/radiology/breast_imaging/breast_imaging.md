## Mammography - Key Findings
| Finding | Benign Features | Malignant Features | URL |
|---------|-----------------|---------------------|-----|
| **Mass** | **Circumscribed**, **Round/oval** | **Spiculated**, **Irregular**, **Microlobulated** | https://radiopaedia.org |
| **Calcifications** | **Coarse**, **Popcorn**, **Round**, **Vascular**, **Milk of calcium** | **Pleomorphic**, **Fine linear branching**, **Amorphous** | https://radiopaedia.org |
| **Architectural Distortion** | **Post-surgical**, **Radial scar** | **No prior surgery**, **Developing** | https://radiopaedia.org |
| **Asymmetry** | **Stable**, **Benign features** | **Developing**, **Focal**, **Global** | https://radiopaedia.org |
