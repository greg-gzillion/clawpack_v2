# Radiology - Pediatric Imaging

## Pediatric Imaging - Principles
| Principle | Description | URL |
|-----------|-------------|-----|
| **ALARA** | **As Low As Reasonably Achievable** - Minimize radiation exposure | https://www.imagegently.org |
| **Ultrasound First** | **Preferred initial modality** for many pediatric conditions | https://www.acr.org/Clinical-Resources/ACR-Appropriateness-Criteria |
| **MRI** | **No radiation**; Excellent for CNS, MSK, Abdomen | https://www.acr.org/Clinical-Resources/ACR-Appropriateness-Criteria |
| **Sedation** | **Minimize movement**, **Ensure safety** | https://www.pedrad.org |

## Common Pediatric Conditions - Imaging
| Condition | Preferred Imaging | Key Findings | URL |
|-----------|-------------------|--------------|-----|
| **Hypertrophic Pyloric Stenosis** | **Ultrasound** | **Pyloric muscle thickness >3 mm**, **Length >15 mm** | https://radiopaedia.org |
| **Intussusception** | **Ultrasound** | **Target sign**, **Pseudokidney sign** | https://radiopaedia.org |
| **Appendicitis** | **Ultrasound** (first-line), **CT** (equivocal) | **Non-compressible**, **>6 mm**, **Hyperemia**, **Appendicolith** | https://www.acr.org/Clinical-Resources/ACR-Appropriateness-Criteria |
| **Developmental Dysplasia of Hip (DDH)** | **Ultrasound** (<6 months), **X-ray** (>6 months) | **Alpha angle <60ｰ**, **Dislocated** | https://radiopaedia.org |
| **Bronchiolitis** | **CXR** (not routinely indicated) | **Hyperinflation**, **Peribronchial thickening** | https://www.acr.org/Clinical-Resources/ACR-Appropriateness-Criteria |
| **Croup** | **Lateral neck X-ray** | **Steeple sign** (subglottic narrowing) | https://radiopaedia.org |
| **Epiglottitis** | **Lateral neck X-ray** (DO NOT delay airway management) | **Thumbprint sign** | https://radiopaedia.org |
