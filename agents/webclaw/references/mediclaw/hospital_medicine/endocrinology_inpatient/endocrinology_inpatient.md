# Hospital Medicine - Inpatient Endocrinology

## Diabetic Ketoacidosis (DKA)
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnosis** | **Glucose >250 mg/dL**, **pH <7.3**, **Bicarbonate <18 mEq/L**, **Anion gap >12**, **Ketones positive** | https://professional.diabetes.org/standards-of-care |
| **Management** | **IV fluids** (NS), **Insulin infusion** (0.1 units/kg/h), **Potassium replacement**, **Transition to subcutaneous when anion gap closed** | https://professional.diabetes.org/standards-of-care |

## Hyperosmolar Hyperglycemic State (HHS)
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnosis** | **Glucose >600 mg/dL**, **pH >7.3**, **Bicarbonate >18**, **Osmolality >320 mOsm/kg**, **No significant ketones** | https://professional.diabetes.org/standards-of-care |
| **Management** | **IV fluids** (more aggressive than DKA), **Insulin** (if glucose not falling with fluids alone), **Electrolyte replacement** | https://professional.diabetes.org/standards-of-care |

## Inpatient Glycemic Management
| Setting | Target Glucose | URL |
|---------|----------------|-----|
| **Non-Critically Ill** | **140-180 mg/dL** | https://professional.diabetes.org/standards-of-care |
| **Critically Ill** | **140-180 mg/dL** (avoid <110 and >180) | https://professional.diabetes.org/standards-of-care |
| **Hypoglycemia** | **<70 mg/dL** ? **Give 15-20g carbohydrate** (PO if alert, D50W IV if NPO/altered) | https://professional.diabetes.org/standards-of-care |

## Thyroid Emergencies
| Condition | Management | URL |
|-----------|------------|-----|
| **Myxedema Coma** | **Levothyroxine IV** (load 200-400 mcg ? 1.6 mcg/kg/day), **Hydrocortisone** (until adrenal insufficiency ruled out), **Supportive care** | https://www.thyroid.org/professionals/ata-professional-guidelines/ |
| **Thyroid Storm** | **PTU** (load), **Iodine** (after PTU), **Beta-blocker** (propranolol), **Corticosteroids**, **Supportive care** | https://www.thyroid.org/professionals/ata-professional-guidelines/ |

## Adrenal Insufficiency
| Component | Description | URL |
|-----------|-------------|-----|
| **Acute Adrenal Crisis** | **Hypotension**, **Hyponatremia**, **Hyperkalemia**, **Hypoglycemia** | https://www.endocrine.org/clinical-practice-guidelines/adrenal-insufficiency |
| **Management** | **Hydrocortisone** 100 mg IV bolus ? 50 mg q6h, **IV fluids**, **Treat precipitant** | https://www.endocrine.org/clinical-practice-guidelines/adrenal-insufficiency |
