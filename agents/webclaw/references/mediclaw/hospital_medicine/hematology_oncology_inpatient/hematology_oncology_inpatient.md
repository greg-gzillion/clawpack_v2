## Febrile Neutropenia
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **ANC <500** (or <1000 with expected nadir <500) + **Temp =38.3蚓 once** or **=38.0蚓 x1h** | https://www.nccn.org/guidelines/febrile-neutropenia |
| **Risk Stratification** | **MASCC Score** (=21 = low risk) | https://www.mdcalc.com/calc/10460/mascc-score-febrile-neutropenia |
| **Empiric Antibiotics** | **Anti-pseudomonal beta-lactam** (cefepime, piperacillin-tazobactam, meropenem) | https://www.nccn.org/guidelines/febrile-neutropenia |
| **Duration** | **Until afebrile =48h and ANC =500** | https://www.nccn.org/guidelines/febrile-neutropenia |
