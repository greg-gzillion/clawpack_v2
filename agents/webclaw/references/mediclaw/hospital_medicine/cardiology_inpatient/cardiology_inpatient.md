# Hospital Medicine - Inpatient Cardiology

## Acute Coronary Syndrome (ACS)
| Component | Description | URL |
|-----------|-------------|-----|
| **STEMI** | **ST elevation** or **new LBBB** | **Emergent reperfusion** (PCI <90 min or thrombolytics) | https://www.acc.org/guidelines/hubs/acute-coronary-syndromes |
| **NSTEMI** | **? Troponin**, **No ST elevation** | **Antiplatelets**, **Anticoagulation**, **Early PCI** (<24h for high-risk) | https://www.acc.org/guidelines/hubs/acute-coronary-syndromes |
| **Unstable Angina** | **Normal troponin**, **Ischemic symptoms** | **Antiplatelets**, **Anticoagulation**, **Stress testing/PCI** | https://www.acc.org/guidelines/hubs/acute-coronary-syndromes |
| **HEART Score** | **Risk stratification** for chest pain (0-3: low risk; 4-6: moderate; =7: high) | https://www.mdcalc.com/calc/10456/heart-score-major-cardiac-events |

## Acute Heart Failure
| Component | Description | URL |
|-----------|-------------|-----|
| **Classification** | **Warm/Dry** (normal), **Warm/Wet** (most common - treat diuretics), **Cold/Wet** (cardiogenic shock), **Cold/Dry** (hypoperfusion) | https://www.acc.org/guidelines/hubs/heart-failure |
| **Initial Management** | **IV loop diuretics**, **Vasodilators** (nitroglycerin), **Oxygen**, **Non-invasive ventilation** (if respiratory distress) | https://www.acc.org/guidelines/hubs/heart-failure |
| **GDMT** | **ARNI/ACEi/ARB**, **Beta-blocker**, **MRA**, **SGLT2i** (initiate/continue when stable) | https://www.acc.org/guidelines/hubs/heart-failure |

## Atrial Fibrillation - Inpatient Management
| Component | Description | URL |
|-----------|-------------|-----|
| **Rate Control** | **Beta-blocker** (metoprolol) or **CCB** (diltiazem) - first-line; **Digoxin** (adjunct) | https://www.acc.org/guidelines/hubs/atrial-fibrillation |
| **Rhythm Control** | **Cardioversion** (if unstable), **Antiarrhythmics** (amiodarone, flecainide), **Ablation** | https://www.acc.org/guidelines/hubs/atrial-fibrillation |
| **Anticoagulation** | **CHA2DS2-VASc Score**: =2 men or =3 women ? **Anticoagulate** | https://www.mdcalc.com/calc/801/cha2ds2-vasc-score-atrial-fibrillation-stroke-risk |
| **HAS-BLED Score** | **Bleeding risk** assessment (=3 = high risk) | https://www.mdcalc.com/calc/10457/has-bled-score-bleeding-risk |

## Hypertension - Inpatient Management
| Scenario | Management | URL |
|----------|------------|-----|
| **Asymptomatic HTN** | **No acute treatment** (restart home medications, Address barriers) | https://www.acc.org/guidelines/hubs/high-blood-pressure |
| **Hypertensive Emergency** | **End-organ damage** (encephalopathy, Aortic dissection, AKI, Pulmonary edema, Eclampsia) ? **IV antihypertensives**, **ICU** | https://www.acc.org/guidelines/hubs/high-blood-pressure |
| **Hypertensive Urgency** | **SBP >180 or DBP >120** without end-organ damage ? **Oral agents**, **Outpatient follow-up** | https://www.acc.org/guidelines/hubs/high-blood-pressure |
