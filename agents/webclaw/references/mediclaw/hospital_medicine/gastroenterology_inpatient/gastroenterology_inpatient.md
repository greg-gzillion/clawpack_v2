# Hospital Medicine - Inpatient Gastroenterology

## Gastrointestinal Bleeding
| Component | Description | URL |
|-----------|-------------|-----|
| **Initial Management** | **2 large-bore IVs**, **IV fluids**, **Transfusion** (target Hb >7 g/dL, >8 g/dL if cardiac), **PPI** (high-dose if suspected variceal) | https://gi.org/guideline/management-of-acute-gastrointestinal-bleeding/ |
| **Risk Stratification** | **Glasgow-Blatchford Score** (predicts need for intervention) | https://www.mdcalc.com/calc/10458/glasgow-blatchford-score-gbs-upper-gi-bleed |
| **Upper GI Bleed** | **EGD within 24h** (urgent if hemodynamically unstable) | https://gi.org/guideline/management-of-acute-gastrointestinal-bleeding/ |
| **Variceal Bleed** | **Octreotide**, **Antibiotics** (ceftriaxone), **EGD with banding** | https://www.aasld.org/practice-guidelines/portal-hypertension |
| **Lower GI Bleed** | **Colonoscopy** (after rapid prep), **CTA** (if massive) | https://gi.org/guideline/management-of-acute-lower-gastrointestinal-bleeding/ |

## Acute Pancreatitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnosis** | **=2 of**: **Epigastric pain**, **Lipase/Amylase >3x ULN**, **Imaging findings** | https://gi.org/guideline/acute-pancreatitis/ |
| **Severity** | **BISAP Score**, **Ranson Criteria** | https://www.mdcalc.com/calc/10425/bisap-score-pancreatitis-mortality |
| **Management** | **Aggressive IV fluids** (LR preferred), **Early enteral nutrition**, **Pain control** | https://gi.org/guideline/acute-pancreatitis/ |
| **Antibiotics** | **NOT for prophylaxis**; Only for **infected necrosis** | https://gi.org/guideline/acute-pancreatitis/ |

## Acute Cholangitis
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnosis** | **Charcot triad** (Fever, Jaundice, RUQ pain) or **Reynolds pentad** (+ Hypotension, AMS) | https://www.idsociety.org/practice-guideline/intra-abdominal-infection/ |
| **Tokyo Guidelines** | **Diagnostic criteria** and **severity grading** | https://www.jshbps.jp/modules/en/index.php?content_id=47 |
| **Management** | **Antibiotics**, **IV fluids**, **ERCP** (urgent for severe) | https://www.idsociety.org/practice-guideline/intra-abdominal-infection/ |

## Clostridioides difficile Infection
| Severity | Definition | Treatment | URL |
|----------|------------|-----------|-----|
| **Non-Severe** | **WBC =15**, **Cr <1.5x baseline** | **Fidaxomicin** (preferred) or **Vancomycin PO** x10d | https://www.idsociety.org/practice-guideline/c-difficile/ |
| **Severe** | **WBC =15** or **Cr =1.5x baseline** | **Fidaxomicin** or **Vancomycin PO** x10d | https://www.idsociety.org/practice-guideline/c-difficile/ |
| **Fulminant** | **Hypotension**, **Shock**, **Ileus**, **Megacolon** | **Vancomycin PO/NG + Metronidazole IV**; **Consider surgery** | https://www.idsociety.org/practice-guideline/c-difficile/ |
