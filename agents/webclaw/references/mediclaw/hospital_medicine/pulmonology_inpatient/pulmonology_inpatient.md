# Hospital Medicine - Inpatient Pulmonology

## Community-Acquired Pneumonia (CAP)
| Component | Description | URL |
|-----------|-------------|-----|
| **CURB-65 Score** | **C**onfusion, **U**remia (>7 mmol/L), **R**R =30, **B**P <90/60, **Age =65** (score =2 = inpatient) | https://www.mdcalc.com/calc/324/curb-65-score-pneumonia-severity |
| **PSI/PORT Score** | **Pneumonia Severity Index** (risk stratification) | https://www.mdcalc.com/calc/33/psi-port-score-pneumonia-severity-index-cap |
| **Empiric Therapy** | **Ceftriaxone + Azithromycin** OR **Respiratory fluoroquinolone** (levofloxacin) | https://www.idsociety.org/practice-guideline/community-acquired-pneumonia/ |
| **Duration** | **5 days** (if clinically stable and afebrile x48h) | https://www.idsociety.org/practice-guideline/community-acquired-pneumonia/ |

## Hospital-Acquired/Ventilator-Associated Pneumonia (HAP/VAP)
| Component | Description | URL |
|-----------|-------------|-----|
| **Risk Factors** | **=48h after admission**, **Mechanical ventilation**, **Aspiration** | https://www.idsociety.org/practice-guideline/hap-vap/ |
| **Empiric Therapy** | **Anti-MRSA** (vancomycin or linezolid) + **Anti-Pseudomonal** (piperacillin-tazobactam, cefepime, meropenem) | https://www.idsociety.org/practice-guideline/hap-vap/ |
| **Duration** | **7 days** | https://www.idsociety.org/practice-guideline/hap-vap/ |

## COPD Exacerbation
| Component | Description | URL |
|-----------|-------------|-----|
| **Diagnosis** | **? Dyspnea**, **? Sputum volume/purulence** | https://goldcopd.org |
| **Treatment** | **Oxygen** (target SpO2 88-92%), **SABA/SAMA** (nebulized), **Corticosteroids** (prednisone 40 mg x5 days), **Antibiotics** (if purulent sputum or ventilated) | https://goldcopd.org |
| **Non-Invasive Ventilation (NIV)** | **Indicated for acute hypercapnic respiratory failure** (pH <7.35, PaCO2 >45) | https://goldcopd.org |

## Acute Asthma Exacerbation
| Component | Description | URL |
|-----------|-------------|-----|
| **Treatment** | **SABA** (nebulized), **Ipratropium**, **Corticosteroids** (IV methylprednisolone or oral prednisone), **Magnesium sulfate** (severe) | https://ginasthma.org |
| **NIV** | **Consider for severe exacerbation** | https://ginasthma.org |
