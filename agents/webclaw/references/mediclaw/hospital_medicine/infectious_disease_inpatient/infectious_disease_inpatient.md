# Hospital Medicine - Inpatient Infectious Disease

## Sepsis (See Sepsis Management section)

## Bacteremia
| Component | Description | URL |
|-----------|-------------|-----|
| **Gram-Positive** | **S. aureus** (MSSA/MRSA), **Streptococcus**, **Enterococcus** | https://www.idsociety.org/practice-guideline/bacteremia/ |
| **Gram-Negative** | **E. coli**, **Klebsiella**, **Pseudomonas**, **Enterobacter** | https://www.idsociety.org/practice-guideline/bacteremia/ |
| **Management** | **Source control**, **Targeted antibiotics**, **Echocardiography** (if S. aureus), **Repeat blood cultures** | https://www.idsociety.org/practice-guideline/bacteremia/ |
| **Duration** | **7-14 days** (uncomplicated), **4-6 weeks** (endocarditis, osteomyelitis) | https://www.idsociety.org/practice-guideline/bacteremia/ |

## Urinary Tract Infection (UTI)
| Type | Management | URL |
|------|------------|-----|
| **Uncomplicated Cystitis** | **Nitrofurantoin**, **TMP-SMX**, **Fosfomycin** x3-5 days | https://www.idsociety.org/practice-guideline/uti/ |
| **Complicated UTI** | **Ciprofloxacin**, **Levofloxacin**, **Ceftriaxone** x7-14 days | https://www.idsociety.org/practice-guideline/uti/ |
| **Pyelonephritis** | **Ciprofloxacin**, **Levofloxacin**, **Ceftriaxone** x7-14 days | https://www.idsociety.org/practice-guideline/uti/ |
| **Catheter-Associated UTI (CAUTI)** | **Remove/replace catheter**, **Antibiotics**, **Avoid treating asymptomatic bacteriuria** | https://www.cdc.gov/infectioncontrol/guidelines/cauti/ |
