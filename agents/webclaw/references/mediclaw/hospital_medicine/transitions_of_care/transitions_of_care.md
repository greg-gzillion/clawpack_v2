# Hospital Medicine - Transitions of Care

## Components of Safe Transitions
| Component | Description | URL |
|-----------|-------------|-----|
| **Medication Reconciliation** | **Review and reconcile** at admission, transfer, discharge | https://www.hospitalmedicine.org/clinical-topics/transitions-of-care/ |
| **Discharge Summary** | **Timely** (within 48h), **Accurate**, **Complete** | https://www.hospitalmedicine.org/clinical-topics/transitions-of-care/ |
| **Patient/Caregiver Education** | **Teach-back**, **Written instructions**, **Medication list** | https://www.hospitalmedicine.org/clinical-topics/transitions-of-care/ |
| **Follow-Up Appointment** | **Scheduled before discharge**, **Within 7-14 days** | https://www.hospitalmedicine.org/clinical-topics/transitions-of-care/ |
| **Communication with Outpatient Providers** | **Direct communication** (phone call, Secure message), **Discharge summary** | https://www.hospitalmedicine.org/clinical-topics/transitions-of-care/ |
| **Post-Discharge Follow-Up** | **Phone call within 48-72 hours** | https://www.hospitalmedicine.org/clinical-topics/transitions-of-care/ |
