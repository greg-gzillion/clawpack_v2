# Hospital Medicine - Glycemic Control

## Inpatient Glycemic Targets
| Setting | Target Glucose | URL |
|---------|----------------|-----|
| **Non-Critically Ill** | **140-180 mg/dL** | https://professional.diabetes.org/standards-of-care |
| **Critically Ill** | **140-180 mg/dL** | https://professional.diabetes.org/standards-of-care |
| **Hypoglycemia** | **<70 mg/dL** | https://professional.diabetes.org/standards-of-care |

## Subcutaneous Insulin Regimens (Non-ICU)
| Component | Description | URL |
|-----------|-------------|-----|
| **Basal Insulin** | **Long-acting** (glargine, detemir, degludec) - **50% of total daily dose** | https://professional.diabetes.org/standards-of-care |
| **Prandial Insulin** | **Rapid-acting** (lispro, aspart) before meals - **50% of total daily dose** (divided) | https://professional.diabetes.org/standards-of-care |
| **Correction Insulin** | **Rapid-acting** for hyperglycemia (based on correction factor) | https://professional.diabetes.org/standards-of-care |
| **Avoid Sliding Scale Alone** | **Reactive**, **Ineffective**, **? Hypoglycemia risk** | https://professional.diabetes.org/standards-of-care |

## Insulin Dosing Calculations
| Component | Formula | URL |
|-----------|---------|-----|
| **Total Daily Dose (TDD)** | **0.3-0.5 units/kg/day** (insulin-naive) | https://professional.diabetes.org/standards-of-care |
| **Correction Factor** | **1800 / TDD** (for rapid-acting) | https://professional.diabetes.org/standards-of-care |
| **Carbohydrate Ratio** | **500 / TDD** | https://professional.diabetes.org/standards-of-care |

## Hypoglycemia Management
| Severity | Treatment | URL |
|----------|-----------|-----|
| **Mild** (70-54 mg/dL) | **15g carbohydrate** PO (4 oz juice, 3-4 glucose tablets) | https://professional.diabetes.org/standards-of-care |
| **Severe** (<54 mg/dL) | **15-20g carbohydrate** PO if alert; **D50W IV** if NPO/altered | https://professional.diabetes.org/standards-of-care |
| **Recheck** | **15 minutes** after treatment | https://professional.diabetes.org/standards-of-care |

## Transition from IV to Subcutaneous Insulin
| Step | Action | URL |
|------|--------|-----|
| **1** | **Calculate TDD** based on **stable IV insulin requirement** over preceding 6-8h | https://professional.diabetes.org/standards-of-care |
| **2** | **Administer basal insulin** 2-4 hours **before stopping** IV insulin | https://professional.diabetes.org/standards-of-care |
| **3** | **Stop IV insulin** 1-2 hours after basal insulin | https://professional.diabetes.org/standards-of-care |
