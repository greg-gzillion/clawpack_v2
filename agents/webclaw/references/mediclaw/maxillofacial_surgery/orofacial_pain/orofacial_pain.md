## Overview
| Component | Description | URL |
|-----------|-------------|-----|

## Trigeminal Neuralgia (Tic Douloureux)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Paroxysmal, lancinating facial pain** in **V2/V3 distribution**, **Triggered by light touch** | https://www.ninds.nih.gov/health-information/disorders/trigeminal-neuralgia |
| **Etiology** | **Neurovascular compression** of trigeminal root by **superior cerebellar artery** | https://www.ninds.nih.gov/health-information/disorders/trigeminal-neuralgia |
| **Diagnosis** | **Clinical** (history), **MRI** (rule out MS, tumor) | https://www.ninds.nih.gov/health-information/disorders/trigeminal-neuralgia |
| **Medical Treatment** | **Carbamazepine** (first-line), **Oxcarbazepine**, **Gabapentin**, **Baclofen** | https://www.ninds.nih.gov/health-information/disorders/trigeminal-neuralgia |
| **Surgical Treatment** | **Microvascular decompression (MVD)** (gold standard), **Stereotactic radiosurgery** (Gamma Knife), **Percutaneous rhizotomy** (balloon, glycerol, Radiofrequency) | https://www.ninds.nih.gov/health-information/disorders/trigeminal-neuralgia |

| **Burning Mouth Syndrome** | **Burning pain** with **normal mucosa**, **Postmenopausal women** | **Clonazepam** (topical/systemic), **CBT**, **Alpha-lipoic acid** | https://www.nidcr.nih.gov/health-info/burning-mouth |

| **Atypical Odontalgia** | **Persistent tooth pain** without dental cause | **TCAs**, **SNRIs**, **Gabapentinoids** | https://www.aaop.org |
