# Oral and Maxillofacial Surgery - Dental Implantology

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Surgical placement of titanium fixtures** into jawbone to **support dental prostheses** | https://www.aaoms.org/procedures/dental-implants |
| **Osseointegration** | **Direct structural and functional connection** between bone and implant surface | https://www.aaoms.org/procedures/dental-implants |
| **Success Rate** | **>95%** (mandible), **>90%** (maxilla) at 10 years | https://www.aaoms.org/procedures/dental-implants |

## Implant Placement Protocols
| Protocol | Description | Indications | URL |
|----------|-------------|-------------|-----|
| **Immediate Placement** | **Implant placed at time of extraction** | **Intact socket walls**, **No infection**, **Adequate primary stability** | https://www.aaoms.org/procedures/dental-implants |
| **Early Placement** | **4-8 weeks post-extraction** (soft tissue healing) | **Mild socket deficiency** | https://www.aaoms.org/procedures/dental-implants |
| **Delayed Placement** | **3-6 months post-extraction** (bone healing) | **Significant bone loss**, **Infection**, **Need for grafting** | https://www.aaoms.org/procedures/dental-implants |

## Bone Grafting for Implants
| Graft Type | Source | Indications | URL |
|------------|--------|-------------|-----|
| **Autograft** | **Patient's own bone** (iliac crest, Ramus, Chin) | **Gold standard**, **Large defects** | https://www.aaoms.org/procedures/bone-grafting |
| **Allograft** | **Cadaveric bone** (FDBA, DFDBA) | **Moderate defects**, **Avoid donor site morbidity** | https://www.aaoms.org/procedures/bone-grafting |
| **Xenograft** | **Bovine/porcine bone** | **Sinus lift**, **Socket preservation** | https://www.aaoms.org/procedures/bone-grafting |
| **Alloplast** | **Synthetic** (hydroxyapatite, Beta-TCP) | **Small defects** | https://www.aaoms.org/procedures/bone-grafting |

## Sinus Lift (Sinus Augmentation)
| Component | Description | URL |
|-----------|-------------|-----|
| **Indication** | **Inadequate maxillary posterior bone height** (<8-10 mm) | https://www.aaoms.org/procedures/sinus-lift |
| **Lateral Window Approach** | **Access via lateral maxillary wall**, **Elevate Schneiderian membrane**, **Place graft** | https://www.aaoms.org/procedures/sinus-lift |
| **Osteotome Approach** | **Crestal approach**, **Osteotome to elevate sinus floor** (requires =5-6 mm residual bone) | https://www.aaoms.org/procedures/sinus-lift |
| **Complications** | **Membrane perforation**, **Sinusitis**, **Implant failure** | https://www.aaoms.org/procedures/sinus-lift |
