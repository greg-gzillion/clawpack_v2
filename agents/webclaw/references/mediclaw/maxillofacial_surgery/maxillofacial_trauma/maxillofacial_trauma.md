## Mandible Fractures
| Component | Description | URL |
|-----------|-------------|-----|
| **Common Sites** | **Angle** (most common), **Body**, **Condyle**, **Symphysis**, **Parasymphysis**, **Ramus**, **Coronoid** | https://www.aaoms.org/procedures/facial-trauma |
| **Favorable vs Unfavorable** | **Favorable**: Fracture line resists displacement by muscles; **Unfavorable**: Muscles displace segments | https://www.aaoms.org/procedures/facial-trauma |
| **Treatment Principles** | **Reduction**, **Fixation** (ORIF with plates/screws), **Maxillomandibular fixation (MMF)** | https://www.aaoms.org/procedures/facial-trauma |
| **Condylar Fractures** | **Closed treatment** (MMF 2-3 weeks, Early mobilization) for most; **ORIF** for displaced, Bilateral, Malocclusion | https://www.aaoms.org/procedures/facial-trauma |

## Midface Fractures (Le Fort Classification)
| Type | Fracture Line | Clinical Features | URL |
|------|---------------|-------------------|-----|
| **Le Fort I** | **Transverse** above maxillary teeth (piriform rim ? pterygoid plates) | **Mobile maxilla**, **Malocclusion** | https://www.aaoms.org/procedures/facial-trauma |
| **Le Fort II** | **Pyramidal** (nasofrontal suture ? infraorbital rim ? zygomaticomaxillary buttress ? pterygoid plates) | **Mobile midface**, **Step-off at infraorbital rim**, **CSF rhinorrhea** | https://www.aaoms.org/procedures/facial-trauma |
| **Le Fort III** | **Craniofacial dysjunction** (nasofrontal ? zygomaticofrontal ? zygomatic arch ? pterygoid plates) | **Mobile entire face**, **CSF rhinorrhea**, **"Dish face" deformity** | https://www.aaoms.org/procedures/facial-trauma |

## Orbital Fractures
| Type | Description | URL |

## Nasal Fractures
| Component | Description | URL |
|-----------|-------------|-----|
| **Presentation** | **Nasal deformity**, **Epistaxis**, **Septal hematoma** | https://www.aaoms.org/procedures/facial-trauma |
| **Septal Hematoma** | **Drain immediately** (prevents septal necrosis, Saddle nose deformity) | https://www.aaoms.org/procedures/facial-trauma |
| **Closed Reduction** | **Within 2 weeks** (before fibrous union) | https://www.aaoms.org/procedures/facial-trauma |
| **Open Reduction (Rhinoplasty)** | **Delayed** (3-6 months) for residual deformity | https://www.aaoms.org/procedures/facial-trauma |
