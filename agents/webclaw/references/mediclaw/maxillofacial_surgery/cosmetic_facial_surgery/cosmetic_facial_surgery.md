## Rhinoplasty
| Component | Description | URL |
|-----------|-------------|-----|

| **Techniques** | **Dorsal hump reduction**, **Osteotomies** (narrowing), **Tip refinement**, **Spreader grafts** (internal valve) | https://www.aaoms.org/procedures/rhinoplasty |

## Blepharoplasty
| Component | Description | URL |
|-----------|-------------|-----|
| **Upper Blepharoplasty** | **Remove excess skin/fat** from upper eyelid (may be functional if obstructs vision) | https://www.aaoms.org/procedures/blepharoplasty |
| **Lower Blepharoplasty** | **Remove/reposition fat**, **Tighten skin** (transcutaneous or transconjunctival) | https://www.aaoms.org/procedures/blepharoplasty |
| **Complications** | **Lagophthalmos**, **Ectropion**, **Dry eye**, **Retrobulbar hematoma** (emergency) | https://www.aaoms.org/procedures/blepharoplasty |
