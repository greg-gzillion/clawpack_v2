## Overview
| Component | Description | URL |
|-----------|-------------|-----|

## Idiopathic Hypersomnia
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **EDS** with **prolonged sleep duration** (>10-11 hours) or **sleep inertia** (difficulty waking) | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Diagnostic Criteria** | **Daily EDS =3 months**, **MSLT** (mean sleep latency =8 min, **<2 SOREMPs**), **No cataplexy**, **CSF hypocretin-1 normal** (if tested) | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Treatment** | **Stimulants** (modafinil, armodafinil, methylphenidate), **Sodium oxybate** (FDA-approved), **Wake-promoting agents** (pitolisant, solriamfetol) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |

## Kleine-Levin Syndrome (Recurrent Hypersomnia)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Recurrent episodes of hypersomnia** (12-24h sleep/day) lasting **days to weeks**, with **cognitive impairment**, **Hyperphagia**, **Hypersexuality**, **Irritability** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Demographics** | **Adolescent males** (peak onset 15 years) | https://rarediseases.org/rare-diseases/kleine-levin-syndrome/ |
| **Diagnosis** | **Clinical** (episodes =2/year, Normal between episodes) | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Treatment** | **Supportive** during episodes; **Lithium** (prevention), **Stimulants**, **Amantadine** | https://rarediseases.org/rare-diseases/kleine-levin-syndrome/ |

## Insufficient Sleep Syndrome
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **EDS due to volitional sleep restriction** (<7 hours/night) | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Diagnosis** | **Sleep diary/actigraphy** showing inadequate sleep; **Symptoms resolve with sleep extension** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Treatment** | **Sleep extension** (=8-9 hours/night) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
