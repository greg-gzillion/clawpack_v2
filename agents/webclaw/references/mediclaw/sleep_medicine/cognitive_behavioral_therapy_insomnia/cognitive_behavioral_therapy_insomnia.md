# Sleep Medicine - Cognitive Behavioral Therapy for Insomnia (CBT-I)

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Structured, evidence-based psychotherapy** targeting maladaptive thoughts and behaviors that perpetuate insomnia | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Efficacy** | **First-line treatment** for chronic insomnia; **Sustained improvement** without medication risks | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Format** | **4-8 sessions** (individual, group, or digital) | https://www.sleepfoundation.org/insomnia/treatment/cognitive-behavioral-therapy-insomnia |
| **Components** | **Stimulus control**, **Sleep restriction**, **Cognitive therapy**, **Relaxation training**, **Sleep hygiene** | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |

## Core Components of CBT-I
| Component | Description | URL |
|-----------|-------------|-----|
| **Stimulus Control** | **Re-associate bed/bedroom with sleep**: Go to bed only when sleepy, Get out of bed if unable to sleep (after 15-20 min), Use bed only for sleep and sex, Maintain consistent wake time | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Sleep Restriction** | **Limit time in bed to average sleep time** (minimum 5-6h), Gradually increase as sleep efficiency improves (>85-90%) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Cognitive Therapy** | **Challenge dysfunctional beliefs** about sleep (catastrophizing, Unrealistic expectations, Misattributions) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Relaxation Training** | **Progressive muscle relaxation**, **Deep breathing**, **Mindfulness**, **Guided imagery** | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Sleep Hygiene** | **Consistent schedule**, **Avoid caffeine/alcohol**, **Dark/quiet/cool bedroom**, **Avoid screens before bed** | https://www.cdc.gov/sleep/about_sleep/sleep_hygiene.html |

## CBT-I vs Pharmacotherapy
| Feature | CBT-I | Pharmacotherapy | URL |
|---------|-------|-----------------|-----|
| **Onset of Effect** | **Slower** (2-4 weeks) | **Rapid** (night 1) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Durability** | **Long-term** (months-years) | **Short-term** (effects stop with discontinuation) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Side Effects** | **None** (may have temporary sleepiness) | **Multiple** (dependence, Falls, Cognitive impairment) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Contraindications** | **None** (may be challenging with untreated OSA, Bipolar) | **Multiple** (pregnancy, Substance use, Elderly) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Cost** | **Higher upfront**, Lower long-term | **Lower upfront**, Higher long-term | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
