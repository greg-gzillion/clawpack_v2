# Sleep Medicine - Insomnia

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Persistent difficulty with sleep initiation, duration, consolidation, or quality** despite adequate opportunity for sleep, resulting in **daytime impairment** | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **ICSD-3 Criteria** | Symptoms **=3 nights/week** for **=3 months** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Prevalence** | **10-30%** of adults (chronic insomnia **6-10%**) | https://www.cdc.gov/sleep/data-and-statistics/adults.html |
| **Risk Factors** | **Female sex**, **Older age**, **Psychiatric disorders**, **Chronic pain**, **Shift work** | https://www.nhlbi.nih.gov/health/insomnia |

## Types of Insomnia
| Type | Description | URL |
|------|-------------|-----|
| **Sleep-Onset Insomnia** | Difficulty **falling asleep** (>30 min) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Sleep-Maintenance Insomnia** | Difficulty **staying asleep** (frequent or prolonged awakenings) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Early-Morning Awakening** | Waking up **earlier than desired** with inability to return to sleep | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Chronic Insomnia** | =3 nights/week for **=3 months** | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Short-Term Insomnia** | <3 months (often triggered by stressor) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |

## Treatment of Insomnia (AASM Guidelines)
| Intervention | Recommendation | URL |
|--------------|----------------|-----|
| **Cognitive Behavioral Therapy for Insomnia (CBT-I)** | **First-line** (strong recommendation) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Sleep Hygiene** | Adjunctive (not sufficient alone) | https://www.cdc.gov/sleep/about_sleep/sleep_hygiene.html |
| **Pharmacotherapy** | **Short-term** use only (=4 weeks); Consider when CBT-I unavailable or insufficient | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |

## Sleep Hygiene Recommendations
| Recommendation | Rationale | URL |
|----------------|-----------|-----|
| **Maintain consistent sleep-wake schedule** | Reinforces circadian rhythm | https://www.cdc.gov/sleep/about_sleep/sleep_hygiene.html |
| **Create relaxing bedtime routine** | Conditions brain for sleep | https://www.cdc.gov/sleep/about_sleep/sleep_hygiene.html |
| **Keep bedroom dark, quiet, cool** | Optimal sleep environment | https://www.cdc.gov/sleep/about_sleep/sleep_hygiene.html |
| **Avoid screens 1-2 hours before bed** | Blue light suppresses melatonin | https://www.cdc.gov/sleep/about_sleep/sleep_hygiene.html |
| **Avoid caffeine after noon** | Caffeine half-life 3-5 hours | https://www.cdc.gov/sleep/about_sleep/sleep_hygiene.html |
| **Avoid alcohol before bed** | Fragments sleep, Suppresses REM | https://www.cdc.gov/sleep/about_sleep/sleep_hygiene.html |
| **Exercise regularly** (not within 2-3h of bed) | Improves sleep quality | https://www.cdc.gov/sleep/about_sleep/sleep_hygiene.html |
| **Get out of bed if unable to sleep** | Avoid associating bed with wakefulness | https://www.cdc.gov/sleep/about_sleep/sleep_hygiene.html |
| **Use bed only for sleep and sex** | Strengthens bed-sleep association | https://www.cdc.gov/sleep/about_sleep/sleep_hygiene.html |
