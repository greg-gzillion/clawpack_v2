# Sleep Medicine - Narcolepsy

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Chronic neurologic disorder** characterized by **excessive daytime sleepiness** and **dysregulation of REM sleep** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Pathophysiology** | **Loss of hypocretin (orexin)-producing neurons** in lateral hypothalamus (autoimmune) | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Prevalence** | **~1 in 2,000** | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Age of Onset** | Peak: **15-25 years** (bimodal with second peak ~35 years) | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |

## Types of Narcolepsy
| Type | Key Features | URL |
|------|--------------|-----|
| **Narcolepsy Type 1 (NT1)** | **Excessive daytime sleepiness** + **Cataplexy** + **Low CSF hypocretin-1** (=110 pg/mL) | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Narcolepsy Type 2 (NT2)** | **Excessive daytime sleepiness** + **No cataplexy** + **Normal CSF hypocretin-1** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |

## Clinical Features (Classic Tetrad)
| Feature | Description | URL |
|---------|-------------|-----|
| **Excessive Daytime Sleepiness (EDS)** | **Universal**, **Sleep attacks**, **Refreshing naps** (5-15 min) | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Cataplexy** | **Sudden loss of muscle tone** triggered by **strong emotions** (laughter, Surprise, Anger); **Pathognomonic for NT1** | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Sleep Paralysis** | **Inability to move** during sleep-wake transitions | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Hypnagogic/Hypnopompic Hallucinations** | **Vivid dream-like experiences** during sleep onset or awakening | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |
| **Disrupted Nocturnal Sleep** | **Fragmented sleep** with frequent awakenings | https://www.ninds.nih.gov/health-information/disorders/narcolepsy |

## Diagnostic Criteria (ICSD-3)
| Criterion | NT1 | NT2 | URL |
|-----------|-----|-----|-----|
| **Daily EDS =3 months** | **Required** | **Required** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Cataplexy** | **Required** | **Absent** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **MSLT** | Mean sleep latency **=8 min** + **=2 SOREMPs** | Mean sleep latency **=8 min** + **=2 SOREMPs** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **CSF Hypocretin-1** | **=110 pg/mL** (or <1/3 of normal) | **>110 pg/mL** (or not tested) | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **HLA-DQB1*06:02** | **Positive in >95%** (not diagnostic) | Positive in ~40% | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |

## Multiple Sleep Latency Test (MSLT)
| Component | Description | URL |
|-----------|-------------|-----|
| **Protocol** | **5 nap opportunities** at 2-hour intervals following **overnight PSG** | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Normal** | **Mean sleep latency >10-15 min**, **=1 SOREMP** | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Narcolepsy Criteria** | **Mean sleep latency =8 min** + **=2 SOREMPs** | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **SOREMP** | **Sleep-Onset REM Period** (REM within 15 min of sleep onset) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Prerequisites** | **Adequate sleep** prior to testing (=6 hours), **Drug washout**, **No untreated OSA** | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |

## Treatment of Narcolepsy (AASM Guidelines)
| Symptom | Treatment | URL |
|---------|-----------|-----|
| **Excessive Daytime Sleepiness** | **Modafinil/Armodafinil** (first-line), **Sodium oxybate**, **Pitolisant**, **Solriamfetol**, **Stimulants** (methylphenidate, amphetamines) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Cataplexy** | **Sodium oxybate** (first-line), **Pitolisant**, **Venlafaxine**, **Fluoxetine**, **Clomipramine** | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Disrupted Nocturnal Sleep** | **Sodium oxybate**, **Baclofen** | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |
| **Behavioral** | **Scheduled naps**, **Sleep hygiene** | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |

## Sodium Oxybate (Xyrem, Xywav)
| Component | Description | URL |
|-----------|-------------|-----|
| **Mechanism** | **GABA-B agonist** (consolidates sleep, ? Cataplexy, Improves EDS) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/sodium-oxybate |
| **Indications** | **Narcolepsy** (EDS and cataplexy), **Idiopathic hypersomnia** | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/sodium-oxybate |
| **Dosing** | **Twice nightly** (bedtime and 2.5-4h later) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/sodium-oxybate |
| **REM Program** | **Risk Evaluation and Mitigation Strategy** (due to abuse potential, CNS depression) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/sodium-oxybate |
| **Contraindications** | **Succinic semialdehyde dehydrogenase deficiency**, **Concomitant alcohol/CNS depressants** | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/sodium-oxybate |
