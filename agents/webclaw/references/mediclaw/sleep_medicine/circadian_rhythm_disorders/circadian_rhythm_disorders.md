# Sleep Medicine - Circadian Rhythm Sleep-Wake Disorders

## Overview
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Misalignment between endogenous circadian rhythm and desired sleep-wake schedule** resulting in **insomnia**, **excessive sleepiness**, or **both** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Pathophysiology** | **Altered circadian timing** (phase advance/delay), **Weak circadian signal**, **Irregular circadian rhythm** | https://www.nhlbi.nih.gov/health/circadian-rhythm-disorders |

## Types of Circadian Rhythm Disorders
| Type | Key Features | URL |
|------|--------------|-----|
| **Delayed Sleep-Wake Phase Disorder (DSWPD)** | **Sleep onset delayed =2 hours** relative to desired time; **Difficulty waking**; **"Night owl"** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Advanced Sleep-Wake Phase Disorder (ASWPD)** | **Sleep onset advanced =2 hours**; **Early morning awakening**; **"Morning lark"** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Irregular Sleep-Wake Rhythm Disorder** | **No clear circadian pattern**; **Multiple naps throughout 24h**; **Associated with neurodegenerative disorders** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Non-24-Hour Sleep-Wake Rhythm Disorder** | **Free-running circadian rhythm** (typically >24h); **Most common in blind individuals** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Shift Work Disorder** | **Insomnia and/or EDS** related to **work schedule during usual sleep period** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Jet Lag Disorder** | **Transient** misalignment after **travel across =2 time zones** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |

## Delayed Sleep-Wake Phase Disorder (DSWPD)
| Component | Description | URL |
|-----------|-------------|-----|
| **Demographics** | **Adolescents and young adults** (peak prevalence 7-16%) | https://www.sleepfoundation.org/delayed-sleep-phase-syndrome |
| **Diagnosis** | **Sleep diary/actigraphy** (=7 days) showing **consistent delay** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Treatment** | **Chronotherapy** (progressively delay sleep), **Bright light therapy** (morning), **Melatonin** (low-dose, 4-6h before desired bedtime), **Strict sleep-wake schedule** | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |

## Advanced Sleep-Wake Phase Disorder (ASWPD)
| Component | Description | URL |
|-----------|-------------|-----|
| **Demographics** | **Older adults** (prevalence ~1%) | https://www.sleepfoundation.org/advanced-sleep-phase-disorder |
| **Genetics** | **Autosomal dominant** (mutations in **PER2**, **CKId**) | https://www.sleepfoundation.org/advanced-sleep-phase-disorder |
| **Treatment** | **Bright light therapy** (evening), **Chronotherapy** (progressively advance sleep) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |

## Shift Work Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Insomnia and/or EDS** temporally associated with **work schedule during usual sleep period** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Prevalence** | **10-40%** of shift workers | https://www.cdc.gov/niosh/work-hour-training-for-nurses/longhours/mod2/05.html |
| **Health Consequences** | **? CVD**, **? Metabolic syndrome**, **? Cancer risk** (IARC Group 2A), **? Accidents**, **? Cognitive function** | https://www.cdc.gov/niosh/work-hour-training-for-nurses/longhours/mod2/05.html |
| **Treatment** | **Scheduled naps**, **Bright light therapy** (during night shift), **Darkness** (during daytime sleep), **Modafinil/Armodafinil** (FDA-approved), **Caffeine**, **Melatonin** (before daytime sleep) | https://aasm.org/clinical-resources/practice-standards/practice-guidelines/ |

## Jet Lag Disorder
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Transient insomnia/EDS** after travel across **=2 time zones** | https://aasm.org/clinical-resources/international-classification-sleep-disorders/ |
| **Eastward Travel** | **Harder to adapt** (phase advance required) | https://wwwnc.cdc.gov/travel/page/jet-lag |
| **Westward Travel** | **Easier to adapt** (phase delay required) | https://wwwnc.cdc.gov/travel/page/jet-lag |
| **Treatment** | **Melatonin** (0.5-5 mg at target bedtime), **Bright light exposure** (strategic timing), **Hydration**, **Avoid alcohol/caffeine** | https://wwwnc.cdc.gov/travel/page/jet-lag |
