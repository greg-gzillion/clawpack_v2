# Gastroenterology - Gastrointestinal Procedures

## Esophagogastroduodenoscopy (EGD)
| Indications | Contraindications | Complications |
|-------------|-------------------|---------------|
| Dysphagia, Odynophagia | Suspected perforation | Perforation (<0.1%) |
| GERD refractory to PPI | Severe coagulopathy | Bleeding |
| UGIB | Hemodynamic instability | Aspiration |
| Anemia (iron deficiency) | Acute myocardial infarction | Infection |
| Chronic nausea/vomiting | | Cardiopulmonary events |
| Abnormal imaging | | |
| Surveillance (Barrett's) | | |

## EGD - Therapeutic Interventions
| Intervention | Indications | Technique |
|--------------|-------------|-----------|
| Variceal Band Ligation | Esophageal varices, Bleeding prophylaxis | Suction varix into banding cap; Deploy band |
| Hemostasis | UGIB (ulcers, Mallory-Weiss, Dieulafoy) | Epinephrine injection, BICAP, Hemoclips |
| Dilation | Strictures (peptic, eosinophilic, radiation, anastomotic) | Savary (wire-guided) or Balloon dilation |
| PEG Placement | Long-term enteral nutrition | Transillumination, Finger indentation, Trocar insertion |
| Foreign Body Removal | Food impaction, Ingested objects | Overtube, Retrieval net, Rat-tooth forceps |
| EMR/ESD | Early neoplasia (Barrett's, Gastric adenoma) | Submucosal injection, Snare resection |
| RFA/Cryotherapy | Barrett's esophagus with dysplasia | Balloon-based or focal ablation |
| POEM | Achalasia | Submucosal tunneling, Myotomy |

## Colonoscopy - Quality Indicators
| Indicator | Target |
|-----------|--------|
| Cecal Intubation Rate | >=95% (>=90% with photo documentation) |
| Adenoma Detection Rate (ADR) | >=30% (men), >=20% (women) |
| Withdrawal Time | >=6 minutes |
| Bowel Preparation Adequacy | >=85% (excellent/good) |
| Post-Polypectomy Bleeding Rate | <1% |
| Perforation Rate | <1:500 (screening), <1:1000 (overall) |

## Colonoscopy - Bowel Preparations
| Preparation | Volume | Split-Dose | Special Considerations |

| PEG-3350 (GoLYTELY) | 4 L | Yes (half night before, half morning of) | Gold standard; Safe in renal failure |
| PEG-3350 (MiraLAX) | 238g + 64 oz Gatorade | Yes | Better tolerated; Avoid in renal failure |
| Sodium Picosulfate/Magnesium Citrate (Prepopik) | Low volume (10 oz) | Yes | Avoid in renal failure |

| Magnesium Citrate | 300 mL | Yes | Avoid in renal failure |

## Endoscopic Retrograde Cholangiopancreatography (ERCP)
| Indications | Contraindications | Complications |
|-------------|-------------------|---------------|
| Choledocholithiasis | Uncorrected coagulopathy | **Post-ERCP pancreatitis** (3-10%) |
| Biliary obstruction (strictures, tumors) | Recent MI | Bleeding (1-2%) |
| Cholangitis | | Perforation (<1%) |
| Bile leak | | Cholangitis (1-3%) |
| Sphincter of Oddi dysfunction (Type I) | | Cardiopulmonary events |
| Pancreatic duct stones/strictures | | |

## ERCP - Interventions
| Intervention | Indications | Technique |
|--------------|-------------|-----------|
| Sphincterotomy | CBD stones, SOD Type I | Cut biliary or pancreatic sphincter |
| Stone Extraction | CBD stones | Balloon or basket after sphincterotomy |
| Biliary Stenting | Biliary obstruction (stricture, tumor) | Plastic (short-term) or Metal (long-term) |
| Pancreatic Stenting | Pancreatic duct stricture, Prophylaxis (post-ERCP pancreatitis) | Small caliber (3-5 Fr) |
| Balloon Dilation | Biliary/pancreatic strictures | Controlled radial expansion |
| Cholangioscopy | Direct visualization of bile duct | SpyGlass system |
| Papillectomy | Ampullary adenoma | Snare resection ｱ pancreatic stent |

## Endoscopic Ultrasound (EUS)
| Indications | Advantages |
|-------------|------------|
| Pancreatic mass evaluation | **Tissue acquisition (FNA/FNB)** |
| Submucosal lesions (GIST, carcinoid) | Layer of origin |
| Bile duct stones (when MRCP contraindicated) | High sensitivity for CBD stones |
| Mediastinal lymphadenopathy | Lung cancer staging |
| Celiac plexus block/neurolysis | Pain management (pancreatic cancer, chronic pancreatitis) |
| Pancreatic cyst evaluation | Cyst fluid analysis (CEA, amylase, cytology) |
| TNM staging (esophageal, gastric, rectal, pancreatic) | Local staging |

| Indications | Contraindications | Complications |
|-------------|-------------------|---------------|
| **Obscure GI bleeding** (most common) | **Known or suspected GI obstruction** | **Capsule retention** (1-2%) |
| Suspected Crohn's disease | Strictures | |
| Celiac disease (refractory) | Swallowing disorders | |
| Polyposis syndromes | Pregnancy | |
| Small bowel tumors | Cardiac pacemakers (relative) | |

## Deep Enteroscopy (Double-Balloon / Single-Balloon)
| Indications | Advantages |
|-------------|------------|
| Obscure GI bleeding with positive VCE | **Therapeutic intervention** (APC, Hemoclip, Polypectomy) |
| Small bowel polyps (PJS, FAP) | Polypectomy |
| Stricture dilation | Balloon dilation |
| Foreign body retrieval | Retrieval |
| ERCP in altered anatomy (Roux-en-Y) | Access to biliary tree |

## Liver Biopsy
| Approach | Indications | Complications |
|----------|-------------|---------------|
| Percutaneous | Unexplained LFT abnormalities, Fibrosis staging, Mass | Bleeding (0.5-1%), Pain, Pneumothorax |
| Transjugular | Coagulopathy, Ascites, Obesity | Less bleeding risk; Adequacy may be limited |
| EUS-Guided | Focal liver lesions | Diagnostic accuracy |

## Paracentesis
| Indications | Contraindications | Complications |
|-------------|-------------------|---------------|
| **New-onset ascites** (diagnostic) | Severe coagulopathy (relative) | Bleeding |
| **Suspected SBP** | Overlying infection | Infection |
| **Large volume paracentesis** (therapeutic) | Bowel obstruction | Bowel perforation |
| | | Hypotension (post-paracentesis) |

## Paracentesis - Fluid Analysis
| Test | Indication | Interpretation |
|------|------------|----------------|
| Cell count and differential | All diagnostic taps | **PMN >=250 cells/mm3 = SBP** |
| Albumin | All taps | **SAAG** = Serum albumin - Ascites albumin |
| Total protein | All taps | Exudate vs Transudate |
| Culture (blood culture bottles) | Suspected SBP | Identifies pathogen |
| Cytology | Suspected malignancy | Malignant cells |
| Amylase | Suspected pancreatic ascites | Elevated |
| Triglycerides | Chylous ascites | Elevated |
| Adenosine deaminase (ADA) | Suspected TB peritonitis | Elevated |
