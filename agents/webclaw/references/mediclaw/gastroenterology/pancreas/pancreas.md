# Gastroenterology - Pancreatic Disorders

## Acute Pancreatitis
| Component | Management |
|-----------|------------|
| Etiology | **Gallstones** (40-70%), **Alcohol** (25-35%), Idiopathic (10-25%), Hypertriglyceridemia (>1000), Post-ERCP, Drugs, Trauma |
| Diagnosis | 2 of 3: (1) Epigastric pain radiating to back, (2) Lipase/Amylase >3x ULN, (3) CT findings |
| Severity Scoring | **Ranson Criteria**, **BISAP Score**, **APACHE II**, **CT Severity Index** |
| Fluid Resuscitation | **Aggressive IVF** (LR preferred); 250-500 mL/hr initially |
| Nutrition | **Early enteral nutrition** (within 24-48 hours); Avoid TPN if possible |
| Antibiotics | **Not routinely indicated**; Only for infected necrosis or extrapancreatic infection |
| ERCP | **Within 24 hours** for cholangitis; Not routine for gallstone pancreatitis without obstruction |
| Cholecystectomy | **During same admission** for mild gallstone pancreatitis |

## Acute Pancreatitis - Complications
| Complication | Timing | Management |
|--------------|--------|------------|
| Acute Peripancreatic Fluid Collection (APFC) | <4 weeks | Observation; Usually resolves |
| Pancreatic Pseudocyst | >4 weeks | Drain if symptomatic, infected, or enlarging |
| Acute Necrotic Collection (ANC) | <4 weeks | Supportive; Antibiotics only if infected |
| Walled-Off Necrosis (WON) | >4 weeks | **Endoscopic transmural drainage** ｱ necrosectomy |
| Infected Necrosis | Variable | **Antibiotics** (Carbapenems); **Step-up approach**: Endoscopic drainage -> Necrosectomy |
| Splenic Vein Thrombosis | Variable | Anticoagulation controversial; Monitor for varices |

## BISAP Score (Bedside Index of Severity in Acute Pancreatitis)
| Criterion | Points |
|-----------|--------|
| BUN >25 mg/dL | 1 |
| Impaired mental status | 1 |
| SIRS >=2 | 1 |
| Age >60 years | 1 |
| Pleural effusion | 1 |
| **Interpretation** | 0-1: Low mortality; 2: Moderate; 3-5: High mortality |

## Chronic Pancreatitis
| Component | Management |
|-----------|------------|
| Etiology | **Alcohol** (most common), Idiopathic, Genetic (PRSS1, SPINK1, CFTR), Autoimmune, Obstructive |
| Presentation | **Chronic abdominal pain**, Exocrine insufficiency (steatorrhea), Endocrine insufficiency (diabetes) |
| Diagnosis | CT (calcifications, dilated duct), MRCP, EUS (Rosemont criteria), Pancreatic function tests |
| Pain Management | Alcohol cessation; Smoking cessation; Analgesics; Antioxidants; **Celiac plexus block** |
| Endoscopic Therapy | **ERCP with pancreatic sphincterotomy, Stone extraction, Stent placement**; ESWL for large stones |
| Surgical Therapy | **Lateral pancreaticojejunostomy (Puestow)**, Pancreaticoduodenectomy (Whipple), Total pancreatectomy with islet autotransplantation |
| Exocrine Insufficiency | **Pancreatic enzyme replacement therapy (PERT)** with meals |

## Autoimmune Pancreatitis (AIP)
| Type | Features | IgG4 | Other Organ Involvement | Treatment |
|------|----------|------|------------------------|-----------|
| Type 1 (IgG4-related) | Older men, Obstructive jaundice, Sausage-shaped pancreas | Elevated | **IgG4-related disease** (bile ducts, salivary glands, retroperitoneum, kidneys) | **Steroids**; Rituximab |
| Type 2 (Idiopathic Duct-centric) | Younger, Associated with IBD | Normal | IBD (UC > Crohn's) | Steroids |
| Diagnosis | Histology (lymphoplasmacytic infiltrate, storiform fibrosis, obliterative phlebitis) | | | |
| Response to Steroids | **Dramatic improvement** (diagnostic and therapeutic) | | | |

## Pancreatic Cystic Lesions
| Type | Malignant Potential | Management |
|------|---------------------|------------|
| Serous Cystadenoma | Very low (<1%) | Observation if asymptomatic |
| Mucinous Cystic Neoplasm (MCN) | Moderate (10-15%) | **Resection** if >4 cm, mural nodule, or symptomatic |
| Intraductal Papillary Mucinous Neoplasm (IPMN) | Variable (Main duct > Branch duct) | **Main duct**: Resection; **Branch duct**: Surveillance (Fukuoka guidelines) |
| Solid Pseudopapillary Neoplasm | Low (10-15%) | **Resection** |
| Pseudocyst | None | Drain if symptomatic |
| **Worrisome Features** | Size >3 cm, Mural nodule, Thickened/enhancing wall, Main pancreatic duct dilation 5-9 mm, Abrupt change in duct caliber, Lymphadenopathy, Elevated CA 19-9, Rapid growth | **EUS with FNA**; Consider resection |

## Pancreatic Ductal Adenocarcinoma (PDAC)
| Component | Management |
|-----------|------------|
| Epidemiology | 4th leading cause of cancer death; 5-year survival <10% |
| Risk Factors | Smoking, Obesity, Diabetes, Chronic pancreatitis, Family history, BRCA2, PALB2, p16, STK11, Lynch syndrome |
| Presentation | Painless jaundice, Weight loss, Epigastric pain, New-onset diabetes, Courvoisier's sign |
| Diagnosis | **CT pancreas protocol** (preferred); EUS with FNA for tissue diagnosis; CA 19-9 (monitoring, not diagnosis) |
| Staging | Resectable, Borderline resectable, Locally advanced, Metastatic |
| Treatment (Resectable) | **Neoadjuvant chemotherapy** (FOLFIRINOX or Gemcitabine/nab-paclitaxel) -> **Pancreaticoduodenectomy (Whipple)** -> Adjuvant chemotherapy |
| Treatment (Borderline/Locally Advanced) | Neoadjuvant therapy -> Re-stage -> Surgery or continued systemic therapy |
| Treatment (Metastatic) | Palliative chemotherapy; **Germline genetic testing** (BRCA, PALB2 -> PARP inhibitors) |
| Palliation | Biliary stenting, Celiac plexus block, Pain management |

## Pancreatic Neuroendocrine Tumors (PNETs)
| Type | Hormone | Clinical Syndrome | Malignant Potential | Treatment |
|------|---------|-------------------|---------------------|-----------|
| Insulinoma | Insulin | Hypoglycemia (Whipple's triad) | Low (10%) | Surgical enucleation |
| Gastrinoma | Gastrin | Zollinger-Ellison syndrome | Moderate (60%) | PPI; Surgical resection |
| Glucagonoma | Glucagon | Necrolytic migratory erythema, DM, DVT | High (80%) | Surgery; Octreotide |
| VIPoma | VIP | WDHA syndrome | High | Surgery; Octreotide |
| Somatostatinoma | Somatostatin | DM, Gallstones, Steatorrhea | High | Surgery |
| Non-Functioning PNET | None | Mass effect, Metastases | Variable | Surgery if >2 cm or symptomatic |
