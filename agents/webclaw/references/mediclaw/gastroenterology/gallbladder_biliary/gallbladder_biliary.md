# Gastroenterology - Gallbladder and Biliary Disorders

## Gallstones (Cholelithiasis)
| Component | Management |
|-----------|------------|
| Prevalence | 10-15% of adults; 80% asymptomatic |
| Types | Cholesterol stones (80%), Pigment stones (20%) |
| Risk Factors | "5 F's": Female, Fertile, Forty, Fat, Family history; Hemolytic disorders (pigment stones) |
| Asymptomatic | No treatment; Cholecystectomy NOT indicated |
| Symptomatic | Biliary colic -> **Elective cholecystectomy** |
| Complications | Cholecystitis, Choledocholithiasis, Cholangitis, Gallstone pancreatitis |

## Acute Cholecystitis
| Component | Management |
|-----------|------------|
| Pathophysiology | Cystic duct obstruction by gallstone (90-95%), Acalculous (5-10%) |
| Presentation | RUQ pain, Fever, Nausea/Vomiting, **Murphy's sign** |
| Diagnosis | Ultrasound (gallstones, wall thickening >4mm, pericholecystic fluid, sonographic Murphy's) |
| HIDA Scan | If US equivocal; Non-filling of gallbladder = acute cholecystitis |
| Tokyo Guidelines | Diagnostic criteria and severity grading |
| Treatment | **IV antibiotics** + **Early cholecystectomy** (within 24-72 hours) |
| High-Risk Surgical Candidates | Percutaneous cholecystostomy tube |

## Acute Cholangitis
| Component | Management |
|-----------|------------|
| Pathophysiology | Biliary obstruction + Infection |
| Charcot's Triad | Fever, Jaundice, RUQ pain (present in 50-70%) |
| Reynolds' Pentad | Charcot's triad + Hypotension + AMS (severe cholangitis) |
| Diagnosis | Clinical + Labs (elevated WBC, Alk Phos, Bilirubin) + Imaging (CBD dilation, stones) |
| Treatment | **IV antibiotics** + **IV fluids** + **Emergent ERCP with biliary drainage** (within 24 hours) |
| Severe Cholangitis | **ERCP within 6-12 hours**; ICU admission |

## Choledocholithiasis
| Component | Management |
|-----------|------------|
| Presentation | Biliary colic, Jaundice, Cholangitis, Pancreatitis |
| Diagnosis | **MRCP** (diagnostic); **EUS** (high sensitivity); **ERCP** (therapeutic) |
| Predictors of CBD Stone | CBD dilation >6mm, Elevated bilirubin, Elevated Alk Phos, Stone on US |
| ASGE Criteria | High probability: ERCP; Intermediate: EUS or MRCP first; Low: Cholecystectomy with IOC |
| Treatment | **ERCP with sphincterotomy and stone extraction** |
| Timing | Pre-cholecystectomy, Intra-operative (IOC with CBD exploration), Post-cholecystectomy |

## Sphincter of Oddi Dysfunction (SOD)
| Type | Description | Management |
|------|-------------|------------|
| Type I | Biliary pain + Abnormal LFTs + Dilated CBD + Delayed drainage | **Sphincterotomy** (no manometry needed) |
| Type II | Biliary pain + One or two criteria | Sphincter of Oddi manometry -> Sphincterotomy if elevated pressure |
| Type III | Biliary pain only | **Avoid ERCP/manometry** (high complication rate); Medical management |
| Note | SOD Type III is often functional abdominal pain; ERCP NOT recommended |

## Gallbladder Cancer
| Component | Management |
|-----------|------------|
| Risk Factors | Gallstones (especially large, long-standing), Porcelain gallbladder, Choledochal cysts, PSC |
| Presentation | Often incidental on cholecystectomy; Advanced: Jaundice, Weight loss, Mass |
| Diagnosis | CT/MRI, EUS for staging |
| Treatment | **Surgical resection** (extended cholecystectomy + lymphadenectomy) |
| Incidental Finding | Re-resection if T1b or greater |
| Prognosis | Poor (often presents late); 5-year survival <5% for advanced disease |

## Cholangiocarcinoma
| Type | Location | Risk Factors | Treatment |
|------|----------|--------------|-----------|
| Intrahepatic | Within liver | Cirrhosis, HBV/HCV, NAFLD, Liver flukes | Resection, Liver transplant |
| Perihilar (Klatskin) | Bifurcation | PSC, Choledochal cysts, Liver flukes | Resection ｱ Liver transplant |
| Distal | Common bile duct | PSC, Choledochal cysts | Pancreaticoduodenectomy (Whipple) |
| Bismuth-Corlette | Classification of perihilar tumors | I-IV (proximal extent) |
| Diagnosis | MRCP, ERCP (brush cytology, FISH), EUS with FNA |
| Palliation | **ERCP with stent placement** (plastic or metal) |
