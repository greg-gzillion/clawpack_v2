# Gastroenterology - Liver Disorders

## Liver Function Tests Interpretation
| Pattern | AST/ALT | Alk Phos | Bilirubin | Differential |
|---------|---------|----------|-----------|--------------|
| Hepatocellular | Elevated (>500) | Normal/Mild | Variable | Viral hepatitis, AIH, Ischemia, Drugs |
| Cholestatic | Normal/Mild | Elevated (>2x ULN) | Elevated | Biliary obstruction, PBC, PSC, Drugs |
| Mixed | Elevated | Elevated | Variable | Drugs, AIH with cholestasis |
| Isolated Hyperbilirubinemia | Normal | Normal | Elevated | Gilbert's, Hemolysis, Crigler-Najjar, Dubin-Johnson |

## Viral Hepatitis
| Virus | Transmission | Chronicity | Treatment | Prevention |
|-------|--------------|------------|-----------|------------|
| HAV | Fecal-oral | None | Supportive | Vaccine (2 doses) |
| HBV | Blood, Sexual, Vertical | 5-10% (adults), 90% (infants) | Entecavir, Tenofovir | Vaccine (3 doses) |
| HCV | Blood (IVDU, Transfusion) | 75-85% | **DAAs** (Sofosbuvir/Velpatasvir) | No vaccine |
| HDV | Blood, Sexual (requires HBV) | High | Peg-IFN (Bulevirtide in EU) | HBV vaccine |
| HEV | Fecal-oral | Rare (immunocompromised) | Supportive; Ribavirin | Vaccine (China) |

## Hepatitis B Serologies
| Stage | HBsAg | Anti-HBs | Anti-HBc (IgM) | Anti-HBc (Total) | HBeAg | Anti-HBe | HBV DNA |
|-------|-------|----------|----------------|------------------|-------|----------|---------|
| Acute Infection | + | - | + | + | + | - | High |
| Chronic Infection | + | - | - | + | +/- | +/- | Variable |
| Resolved Infection | - | + | - | + | - | +/- | Undetectable |
| Vaccinated | - | + | - | - | - | - | - |

## Hepatitis B Treatment Indications
| Criteria | Threshold |
|----------|-----------|
| HBV DNA | >2000 IU/mL (HBeAg-negative) or >20,000 IU/mL (HBeAg-positive) |
| ALT | >2x ULN (or >ULN with significant fibrosis) |
| Cirrhosis | Treat regardless of DNA/ALT |
| Prevention | Mother-to-child transmission, Immunosuppression |

## Hepatitis C Treatment
| Component | Management |
|-----------|------------|
| Goal | **Sustained virologic response (SVR)** = Undetectable HCV RNA 12 weeks post-treatment |
| Pan-genotypic Regimens | **Sofosbuvir/Velpatasvir** x 12 weeks; **Glecaprevir/Pibrentasvir** x 8-12 weeks |
| Decompensated Cirrhosis | Avoid protease inhibitors; Use Sofosbuvir/Velpatasvir + Ribavirin |
| Post-SVR | Monitor for HCC if cirrhosis (ultrasound q6 months) |
| Reinfection | Treat again (same as initial infection) |

## Autoimmune Hepatitis (AIH)
| Component | Management |
|-----------|------------|
| Presentation | Female predominance, Elevated transaminases, Hypergammaglobulinemia |
| Autoantibodies | **ANA** (80%), **Anti-SMA** (60-80%), Anti-LKM1 (5-10%), Anti-SLA/LP |
| Diagnosis | Compatible histology + Autoantibodies + Elevated IgG + Exclusion of other causes |
| Treatment | **Prednisone** (0.5-1 mg/kg) + **Azathioprine** (1-2 mg/kg) |
| Maintenance | Azathioprine ｱ low-dose prednisone; Lifelong therapy often needed |
| Relapse | Common with steroid withdrawal |

## Primary Biliary Cholangitis (PBC)
| Component | Management |
|-----------|------------|
| Presentation | Middle-aged women, Fatigue, Pruritus, Elevated Alk Phos |
| Autoantibody | **Anti-Mitochondrial Antibody (AMA)** positive in 95% |
| Diagnosis | AMA + Elevated Alk Phos (biopsy not always required) |
| Treatment | **Ursodeoxycholic acid (UDCA)** 13-15 mg/kg/day |
| UDCA Response | Alk Phos <1.67x ULN and bilirubin normal at 1 year |
| Second-Line | **Obeticholic acid** (FXR agonist) or **Fibrates** (Bezafibrate, Fenofibrate) |

## Primary Sclerosing Cholangitis (PSC)
| Component | Management |
|-----------|------------|
| Association | **Ulcerative colitis** (70-80%), Male predominance |
| Presentation | Asymptomatic (50%), Fatigue, Pruritus, Cholangitis, Elevated Alk Phos |
| Diagnosis | **MRCP** (multifocal strictures, beading) or ERCP |
| Autoantibodies | **p-ANCA** (atypical pattern) in 50-80% |
| Complications | **Cholangiocarcinoma** (lifetime risk 10-20%), Gallbladder cancer, Colorectal cancer |
| Treatment | No proven medical therapy; UDCA may improve labs but not outcomes; **Liver transplant** for end-stage disease |

## Nonalcoholic Fatty Liver Disease (NAFLD) / MASLD
| Component | Management |
|-----------|------------|
| Spectrum | Simple steatosis -> NASH (steatohepatitis) -> Fibrosis -> Cirrhosis -> HCC |
| Risk Factors | Obesity, Type 2 diabetes, Dyslipidemia, Metabolic syndrome |
| Diagnosis | Imaging (US, CT, MRI); **VCTE (FibroScan)** or MRE for fibrosis; Liver biopsy for NASH |
| Non-Invasive Fibrosis Scores | **FIB-4**, **NAFLD Fibrosis Score**, **ELF** |
| Treatment | **Weight loss** (7-10% body weight); Vitamin E (800 IU/day) for non-diabetic NASH; Pioglitazone; **Resmetirom** (THR-beta agonist) |
| Bariatric Surgery | Improves NASH and fibrosis |

## Alcoholic Liver Disease
| Spectrum | Features | Management |
|----------|----------|------------|
| Alcoholic Fatty Liver | Steatosis, Normal LFTs or mild elevation | Abstinence; Reversible |
| Alcoholic Hepatitis | Fever, Jaundice, Tender hepatomegaly, AST:ALT >2:1 | **Maddrey's DF** >32: **Prednisolone** 40 mg daily x 28 days |
| Alcoholic Cirrhosis | Complications of portal hypertension | Abstinence; Liver transplant evaluation |
| Maddrey's Discriminant Function | 4.6 x (PT - control) + Bilirubin (mg/dL) | >32 = Severe alcoholic hepatitis |

## Cirrhosis
| Component | Management |
|-----------|------------|
| Compensated | No ascites, No variceal bleeding, No encephalopathy | Treat underlying cause; HCC surveillance (US q6 months); EGD for varices |
| Decompensated | Ascites, Variceal bleeding, Encephalopathy, Jaundice | Manage complications; **Refer for liver transplant** |
| Child-Pugh Score | Bilirubin, Albumin, INR, Ascites, Encephalopathy | Class A (5-6), B (7-9), C (10-15) |
| MELD-Na Score | Bilirubin, INR, Cr, Na, Dialysis | Predicts 3-month mortality; Determines transplant priority |

## Complications of Cirrhosis
| Complication | Management |
|--------------|------------|
| Ascites | Sodium restriction (<2g/day); **Spironolactone** + **Furosemide**; Large volume paracentesis; TIPS |
| Spontaneous Bacterial Peritonitis (SBP) | **Ceftriaxone** or **Cefotaxime**; **Albumin** 1.5 g/kg day 1, 1 g/kg day 3; Prophylaxis (Norfloxacin, TMP-SMX) |
| Variceal Bleeding | **Octreotide**; **Antibiotics** (Ceftriaxone); **EGD with band ligation**; TIPS if refractory |
| Hepatic Encephalopathy | **Lactulose** (titrate to 2-3 BM/day); **Rifaximin** 550 mg BID |
| Hepatorenal Syndrome (HRS-AKI) | **Terlipressin** + Albumin; Midodrine + Octreotide + Albumin; **Liver transplant** |
| Hepatocellular Carcinoma (HCC) | **LI-RADS** classification; Resection, Ablation, TACE, TARE, Systemic therapy, Transplant |

## Hereditary Hemochromatosis
| Component | Management |
|-----------|------------|
| Genetics | Autosomal recessive; **HFE gene** (C282Y/C282Y or C282Y/H63D) |
| Presentation | Fatigue, Arthralgias, Bronze diabetes, Cardiomyopathy, Hypogonadism |
| Labs | **Transferrin saturation >45%**, **Ferritin elevated** |
| Diagnosis | HFE gene testing; Liver biopsy if ferritin >1000 or elevated LFTs |
| Treatment | **Phlebotomy** (weekly until ferritin 50-100); Then maintenance |
| Screening | First-degree relatives (HFE gene testing, Iron studies) |

## Wilson's Disease
| Component | Management |
|-----------|------------|
| Genetics | Autosomal recessive; **ATP7B gene** mutation |
| Presentation | Liver (hepatitis, cirrhosis), Neurologic (dystonia, tremor, dysarthria), Psychiatric, Kayser-Fleischer rings |
| Diagnosis | **Low ceruloplasmin**, High 24h urine copper, Kayser-Fleischer rings, Liver copper >250 mcg/g |
| Treatment | **D-Penicillamine** or **Trientine** (chelation); **Zinc** (maintenance) |
| Monitoring | 24h urine copper, Neurologic exam, CBC, Urinalysis |
| Screening | First-degree relatives |

## Alpha-1 Antitrypsin Deficiency
| Component | Management |
|-----------|------------|
| Genetics | Autosomal co-dominant; **SERPINA1 gene** (ZZ, SZ, MZ) |
| Presentation | Lung (COPD, emphysema), Liver (neonatal hepatitis, cirrhosis, HCC) |
| Diagnosis | **Low alpha-1 antitrypsin level**, Phenotype (ZZ, SZ) |
| Liver Histology | **PAS-positive, diastase-resistant globules** in hepatocytes |
| Treatment | **Liver transplant** for end-stage liver disease; **Augmentation therapy** for lung disease (not liver) |
