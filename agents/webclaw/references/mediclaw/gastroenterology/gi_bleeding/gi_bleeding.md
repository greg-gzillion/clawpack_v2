# Gastroenterology - Gastrointestinal Bleeding

## Upper GI Bleeding (UGIB)
| Component | Management |
|-----------|------------|
| Presentation | **Hematemesis** (bright red or coffee-ground), **Melena**, Hematochezia (if brisk), Elevated BUN:Cr ratio |
| Causes | PUD (40-50%), Varices (10-20%), Mallory-Weiss tear (5-10%), Erosive esophagitis/gastritis, Malignancy, Angiodysplasia |
| Initial Management | **ABCs**, **2 large-bore IVs**, **IV PPI** (Pantoprazole 80 mg bolus + 8 mg/hr infusion), **Octreotide** if variceal suspected |
| Risk Stratification | **Glasgow-Blatchford Score** (pre-endoscopy), **Rockall Score** (post-endoscopy) |
| Transfusion Threshold | **Restrictive strategy**: Hb <7 g/dL (or <8 g/dL if cardiovascular disease) |
| Timing of EGD | **Within 24 hours** for most; **Within 12 hours** for high-risk (hemodynamic instability, active bleeding) |
| Anticoagulation Reversal | Warfarin: 4F-PCC + Vitamin K; DOACs: Andexanet alfa (Xa) or Idarucizumab (Dabigatran) |

## Glasgow-Blatchford Score (GBS)
| Risk Marker | Points |
|-------------|--------|
| BUN (mmol/L) | 2-6 points |
| Hemoglobin (g/dL) | 1-6 points |
| Systolic BP (mmHg) | 1-3 points |
| Heart Rate >=100 | 1 point |
| Melena | 1 point |
| Syncope | 2 points |
| Hepatic Disease | 2 points |
| Cardiac Failure | 2 points |
| **Interpretation** | **0-1**: Low risk (outpatient management possible); **>=2**: High risk (admit) |

## Endoscopic Management of UGIB
| Lesion | Endoscopic Therapy |
|--------|---------------------|
| Clean-based ulcer | No endoscopic therapy |
| Flat pigmented spot | No endoscopic therapy |
| Adherent clot | Consider irrigation ± therapy |
| Visible vessel | **Dual therapy** (Epinephrine injection + Thermal/BICAP or Hemoclip) |
| Active bleeding | **Dual therapy** |
| Varices | **Band ligation** (esophageal); **Cyanoacrylate glue** (gastric) |
| Mallory-Weiss tear | Hemoclips or band ligation |
| Dieulafoy's lesion | Hemoclips or band ligation |

## Post-EGD Management of UGIB
| Lesion | PPI Therapy | Diet | Disposition |
|--------|-------------|------|-------------|
| Low-risk (clean base, flat spot) | Oral PPI daily | Regular diet | Outpatient (if GBS 0-1) |
| High-risk (visible vessel, active bleeding) | **IV PPI infusion x 72 hours** | NPO initially; Advance after 24-48 hrs | Inpatient observation |
| H. pylori positive | **Test and treat** | | |
| Aspirin for secondary prevention | Resume after bleeding controlled (1-3 days) | | |
| Aspirin for primary prevention | Discontinue permanently | | |
| Dual antiplatelet therapy | Resume as soon as possible (with PPI) | | |

## Variceal Bleeding
| Component | Management |
|-----------|------------|
| Presentation | **Massive hematemesis**, Melena, History of cirrhosis, Stigmata of chronic liver disease |
| Initial Management | **Octreotide** 50 mcg bolus + 50 mcg/hr infusion; **Ceftriaxone** 1g IV daily x 7 days |
| Transfusion | **Restrictive strategy**: Hb target 7-8 g/dL (over-transfusion increases portal pressure) |
| EGD Timing | **Within 12 hours** |
| Endoscopic Therapy | **Esophageal varices**: Band ligation; **Gastric varices**: Cyanoacrylate glue injection |
| Refractory Bleeding | **Balloon tamponade (Blakemore/Minnesota tube)** as bridge to TIPS; **Emergent TIPS** |
| Secondary Prophylaxis | **Non-selective beta-blocker** (Propranolol, Nadolol) + **Serial band ligation** q2-4 weeks until obliteration |
| TIPS Indications | Refractory acute bleeding, Recurrent bleeding despite secondary prophylaxis, Refractory ascites |

## Lower GI Bleeding (LGIB)
| Component | Management |
|-----------|------------|
| Presentation | **Hematochezia** (bright red blood per rectum), Maroon stools |
| Causes | Diverticulosis (30-40%), Angiodysplasia (10-20%), Ischemic colitis, Post-polypectomy, IBD, Hemorrhoids, Malignancy |
| Initial Management | **ABCs**, IV access, Resuscitation, Labs (CBC, coags, type and cross) |
| Colonoscopy Timing | After **rapid bowel prep** (4-6 L PEG); Within 24 hours |
| If Hemodynamically Unstable | **CTA abdomen/pelvis** (localizes bleeding); **Angiography with embolization**; **Surgery** if refractory |
| Tagged RBC Scan | Localizes slow/intermittent bleeding; Does NOT provide therapy |

## Endoscopic Management of LGIB
| Lesion | Endoscopic Therapy |
|--------|---------------------|
| Diverticular bleed | **Hemoclip** or Band ligation; Epinephrine injection; Avoid thermal (risk of perforation) |
| Angiodysplasia | **Argon plasma coagulation (APC)** |
| Post-polypectomy bleed | Hemoclip; Epinephrine injection |
| Radiation proctitis | **APC**; Sucralfate enemas |
| Hemorrhoids | **Band ligation** (internal); Surgical referral (external) |

## Obscure GI Bleeding
| Component | Management |
|-----------|------------|
| Definition | Recurrent or persistent GI bleeding of unknown origin after negative EGD and colonoscopy |
| Evaluation | **Video capsule endoscopy (VCE)** (first-line for small bowel); **CT enterography** |
| If VCE Positive | **Deep enteroscopy** (Double-balloon or Single-balloon enteroscopy) for therapy |
| If VCE Negative but Ongoing | Repeat EGD/Colonoscopy (missed lesions), Meckel's scan, Angiography, Intraoperative enteroscopy |
| Causes | Angiodysplasia (most common), Small bowel tumors, Meckel's diverticulum, Crohn's disease, Celiac disease |
