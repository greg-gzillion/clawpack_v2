## Beers Criteria - Key Potentially Inappropriate Medications
| Drug Class | Rationale | Alternative | URL |
|------------|-----------|-------------|-----|
| **Anticholinergics** (diphenhydramine, oxybutynin) | **Confusion, Constipation, Falls** | **Avoid** | https://dcri.org/beers-criteria/ |
| **Benzodiazepines** | **Falls, Fractures, Cognitive impairment** | **Avoid** | https://dcri.org/beers-criteria/ |
| **Nonbenzodiazepine hypnotics** (zolpidem, eszopiclone) | Falls, Delirium | **Avoid** | https://dcri.org/beers-criteria/ |
| **NSAIDs** | GI bleeding, AKI, ? BP | **Avoid long-term** | https://dcri.org/beers-criteria/ |
| **PPIs >8 weeks** | C. diff, Bone loss, B12 deficiency | Deprescribe | https://dcri.org/beers-criteria/ |
| **Sliding scale insulin** | Hypoglycemia, No benefit | Avoid | https://dcri.org/beers-criteria/ |
| **Glyburide** | **Severe prolonged hypoglycemia** | Avoid | https://dcri.org/beers-criteria/ |
| **Peripheral alpha-blockers** | Orthostasis | Avoid for HTN | https://dcri.org/beers-criteria/ |
| **TCAs** | Anticholinergic, Orthostasis, QT prolongation | **Avoid** | https://dcri.org/beers-criteria/ |
| **Antipsychotics** (in dementia) | Stroke, Mortality | Avoid unless psychosis/agitation refractory | https://dcri.org/beers-criteria/ |
