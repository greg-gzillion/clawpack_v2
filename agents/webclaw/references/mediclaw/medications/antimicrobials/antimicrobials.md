# Medications - Antimicrobials

## Beta-Lactam Antibiotics
| Class | Examples | Spectrum | Key Adverse Effects | URL |
|-------|----------|----------|---------------------|-----|
| **Penicillins** | Amoxicillin, Ampicillin | Streptococcus, Enterococcus, Listeria | **Rash**, **Diarrhea**, **Anaphylaxis** | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/penicillins |
| **Penicillinase-Resistant** | Nafcillin, Oxacillin, Dicloxacillin | **MSSA** | Hepatotoxicity (oxacillin), Interstitial nephritis | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/penicillins |
| **Aminopenicillins + BLI** | Amoxicillin-clavulanate, Ampicillin-sulbactam | MSSA, Anaerobes, Gram-negatives | **Diarrhea**, **Hepatotoxicity** (amox-clav) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/penicillins |
| **Anti-Pseudomonal + BLI** | Piperacillin-tazobactam | **Pseudomonas**, Anaerobes, Gram-negatives | **Thrombocytopenia**, **Nephrotoxicity** | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/penicillins |
| **Cephalosporins (1st gen)** | Cefazolin, Cephalexin | MSSA, Streptococcus, E. coli, Klebsiella | **Cross-reactivity** with penicillin (<2%) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/cephalosporins |
| **Cephalosporins (2nd gen)** | Cefuroxime, Cefoxitin | H. influenzae, M. catarrhalis, Anaerobes (cefoxitin) | Cross-reactivity with penicillin | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/cephalosporins |
| **Cephalosporins (3rd gen)** | **Ceftriaxone**, Cefotaxime, Ceftazidime | **Meningitis**, Gonorrhea, Pseudomonas (ceftazidime) | **Biliary sludging** (ceftriaxone) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/cephalosporins |
| **Cephalosporins (4th gen)** | **Cefepime** | Pseudomonas, Enterobacteriaceae | **Neurotoxicity** (elderly, renal failure) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/cephalosporins |
| **Cephalosporins (5th gen)** | **Ceftaroline** | **MRSA**, MSSA, Streptococcus | Cross-reactivity with penicillin | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/cephalosporins |
| **Carbapenems** | Meropenem, Imipenem, Ertapenem | **Broadest spectrum** (except MRSA, VRE) | **Seizures** (imipenem), Cross-reactivity <1% | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/carbapenems |
| **Monobactams** | **Aztreonam** | Aerobic Gram-negatives (Pseudomonas) | **Safe in penicillin allergy** (except ceftazidime cross-reactivity) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/aztreonam |

## Non-Beta-Lactam Antibiotics
| Class | Examples | Spectrum | Key Adverse Effects | URL |
|-------|----------|----------|---------------------|-----|
| **Macrolides** | Azithromycin, Clarithromycin | Atypicals, Streptococcus | **QTc prolongation**, **GI upset** | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/macrolides |
| **Fluoroquinolones** | Ciprofloxacin, Levofloxacin, Moxifloxacin | Gram-negatives, Atypicals, Pseudomonas | **Tendon rupture**, **QTc prolongation**, **C. diff** | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/fluoroquinolones |
| **Tetracyclines** | Doxycycline, Minocycline | Atypicals, Rickettsia, Lyme, MRSA | **Photosensitivity**, **Tooth discoloration** (<8 years) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/tetracyclines |
| **Glycylcyclines** | **Tigecycline** | MRSA, VRE, MDR Gram-negatives | **Nausea/Vomiting**, **? Mortality** (monotherapy) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/tigecycline |
| **Aminoglycosides** | Gentamicin, Tobramycin, Amikacin | Aerobic Gram-negatives, Synergy for Gram-positives | **Nephrotoxicity**, **Ototoxicity** | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/aminoglycosides |
| **Lincosamides** | **Clindamycin** | Anaerobes, Streptococcus, MSSA | **C. diff** (high risk), Bitter taste | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/clindamycin |
| **Oxazolidinones** | **Linezolid**, Tedizolid | **MRSA**, **VRE** | **Thrombocytopenia**, **Serotonin syndrome** (with SSRIs) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/linezolid |
| **Glycopeptides** | **Vancomycin** | **MRSA**, Enterococcus, C. diff (PO) | **Red Man Syndrome**, **Nephrotoxicity** | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/vancomycin |
| **Lipoglycopeptides** | **Daptomycin** | **MRSA**, **VRE** | **CPK elevation**, **Eosinophilic pneumonia** (avoid in pneumonia) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/daptomycin |
| **Sulfonamides** | **TMP-SMX** | Gram-negatives, PJP, Nocardia, Stenotrophomonas | **Hyperkalemia**, **SJS/TEN**, **? INR** (with warfarin) | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/sulfonamides |
| **Nitroimidazoles** | **Metronidazole** | **Anaerobes**, Protozoa | **Disulfiram-like reaction**, **Metallic taste**, **Peripheral neuropathy** | https://www.fda.gov/drugs/postmarket-drug-safety-information-patients-and-providers/metronidazole |

## Vancomycin Dosing and Monitoring (IDSA 2020)
| Component | Recommendation | URL |
|-----------|----------------|-----|
| **Loading Dose** | **20-35 mg/kg** (actual body weight) | https://www.idsociety.org/practice-guideline/vancomycin/ |
| **Preferred Monitoring** | **AUC/MIC-guided** (target 400-600) | https://www.idsociety.org/practice-guideline/vancomycin/ |
| **Alternative Monitoring** | Trough-guided (15-20 mg/L) for non-severe infections | https://www.idsociety.org/practice-guideline/vancomycin/ |
| **Timing** | Trough before 4th dose, then weekly if stable | https://www.idsociety.org/practice-guideline/vancomycin/ |
| **Nephrotoxicity Risk** | AUC >600 or Trough >20 mg/L | https://www.idsociety.org/practice-guideline/vancomycin/ |
