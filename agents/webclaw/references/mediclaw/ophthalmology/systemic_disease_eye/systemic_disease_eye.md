## Thyroid Eye Disease (TED)
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Autoimmune orbital inflammation** associated with **Graves disease** | https://www.nei.nih.gov/learn-about-eye-health/eye-conditions-and-diseases/graves-eye-disease |
| **Clinical Features** | **Proptosis**, **Lid retraction** (Dalrymple sign), **Lid lag** (von Graefe sign), **Diplopia**, **Optic neuropathy** (apical crowding) | https://www.nei.nih.gov/learn-about-eye-health/eye-conditions-and-diseases/graves-eye-disease |
| **Treatment (Active Phase)** | **Selenium**, **Corticosteroids**, **Teprotumumab** (IGF-1R antibody), **Orbital radiation**, **Decompression surgery** (urgent if optic neuropathy) | https://www.nei.nih.gov/learn-about-eye-health/eye-conditions-and-diseases/graves-eye-disease |
| **Treatment (Inactive Phase)** | **Rehabilitative surgery**: **Orbital decompression**, **Strabismus surgery**, **Eyelid surgery** | https://www.nei.nih.gov/learn-about-eye-health/eye-conditions-and-diseases/graves-eye-disease |
