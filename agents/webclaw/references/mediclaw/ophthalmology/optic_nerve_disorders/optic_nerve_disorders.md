# Ophthalmology - Optic Nerve Disorders

## Optic Neuritis
| Component | Description | URL |
|-----------|-------------|-----|
| **Clinical Features** | **Monocular vision loss** (hours-days), **Pain with eye movement**, **Afferent pupillary defect (APD)** , **Desaturation of red**, **Central/cecocentral scotoma** | https://www.nei.nih.gov/learn-about-eye-health/eye-conditions-and-diseases/optic-neuritis |
| **Associations** | **Multiple sclerosis** (most common), **NMOSD** (anti-AQP4), **MOGAD** (anti-MOG) | https://www.nei.nih.gov/learn-about-eye-health/eye-conditions-and-diseases/optic-neuritis |
| **Diagnosis** | **MRI orbits with contrast** (optic nerve enhancement), **MRI brain** (white matter lesions ? MS risk) | https://www.nei.nih.gov/learn-about-eye-health/eye-conditions-and-diseases/optic-neuritis |
| **Treatment** | **IV Methylprednisolone 1000 mg x 3-5 days** (accelerates recovery); **Avoid oral prednisone alone** | https://www.nei.nih.gov/learn-about-eye-health/eye-conditions-and-diseases/optic-neuritis |

## Ischemic Optic Neuropathy
| Type | Clinical Features | ESR/CRP | Treatment | URL |
|------|-------------------|---------|-----------|-----|
| **Non-Arteritic Anterior Ischemic Optic Neuropathy (NAION)** | **Sudden**, Painless, **Segmental disc edema**, Altitudinal defect, **Age 50-70**, Risk factors: HTN, DM, Sleep apnea, **Disc at risk** (small cup-to-disc) | Normal | **No proven treatment**; **Control risk factors**, **Avoid PDE5 inhibitors** | https://www.aao.org/eye-health/diseases/naion |
| **Arteritic Anterior Ischemic Optic Neuropathy (AAION)** | **Sudden**, **Jaw claudication**, **Scalp tenderness**, **PMR symptoms**, **Pale chalky-white disc edema**, **Age >50** | **Markedly elevated** | **High-dose corticosteroids** (methylprednisolone 1g IV or prednisone 60-100 mg PO); **Prevents fellow eye involvement** | https://www.aao.org/eye-health/diseases/giant-cell-arteritis |

## Papilledema
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **Bilateral optic disc edema** due to **elevated intracranial pressure (ICP)** | https://www.aao.org/eye-health/diseases/papilledema |
| **Causes** | **Idiopathic intracranial hypertension (IIH)** , **Mass lesion**, **Cerebral venous thrombosis**, **Meningitis** | https://www.aao.org/eye-health/diseases/papilledema |
| **Clinical Features** | **Headache** (worse supine), **Pulsatile tinnitus**, **Transient visual obscurations**, **Diplopia** (CN VI palsy), **Enlarged blind spot** | https://www.aao.org/eye-health/diseases/papilledema |
| **Diagnosis** | **MRI/MRV brain**, **LP** (elevated opening pressure >25 cm H2O) | https://www.aao.org/eye-health/diseases/papilledema |
| **Treatment** | **Treat underlying cause**; **Weight loss**, **Acetazolamide** (IIH) | https://www.aao.org/eye-health/diseases/papilledema |

## Optic Atrophy
| Component | Description | URL |
|-----------|-------------|-----|
| **Definition** | **End-stage optic nerve damage** from any cause | https://www.aao.org/eye-health/diseases/optic-atrophy |
| **Causes** | **Glaucoma** (most common), **Ischemic** (NAION, AAION), **Inflammatory** (optic neuritis), **Compressive** (tumor, Aneurysm), **Toxic/Nutritional** (methanol, Ethambutol, B12 deficiency), **Traumatic**, **Hereditary** (Leber) | https://www.aao.org/eye-health/diseases/optic-atrophy |
| **Clinical Features** | **Pale**, **Flat optic disc**, **Decreased visual acuity**, **Visual field defects**, **APD** (if unilateral) | https://www.aao.org/eye-health/diseases/optic-atrophy |
| **Management** | **Treat underlying cause**, **Low vision rehabilitation** | https://www.aao.org/eye-health/diseases/optic-atrophy |

## Leber Hereditary Optic Neuropathy (LHON)
| Component | Description | URL |
|-----------|-------------|-----|
| **Genetics** | **Mitochondrial DNA** mutations (m.11778G>A, m.3460G>A, m.14484T>C); **Maternal inheritance** | https://www.nei.nih.gov/learn-about-eye-health/eye-conditions-and-diseases/leber-hereditary-optic-neuropathy |
| **Clinical Features** | **Young adult males** (80-90%), **Subacute**, **Painless**, **Sequential bilateral vision loss** (weeks-months), **Central/cecocentral scotoma** | https://www.nei.nih.gov/learn-about-eye-health/eye-conditions-and-diseases/leber-hereditary-optic-neuropathy |
| **Treatment** | **Idebenone** (approved in EU), **Supportive**, **Avoid triggers** (smoking, Alcohol) | https://www.nei.nih.gov/learn-about-eye-health/eye-conditions-and-diseases/leber-hereditary-optic-neuropathy |
