## Esotropia
| Type | Onset | Features | Treatment | URL |
|------|-------|----------|-----------|-----|
| **Infantile Esotropia** | <6 months | **Large angle**, **Cross-fixation**, **Normal refraction**, **Amblyopia** (if not alternating) | **Surgery** (bimedial recession) by 6-12 months; **Treat amblyopia** pre-op | https://www.aao.org/eye-health/diseases/infantile-esotropia |
| **Accommodative Esotropia** | 6 months - 7 years | **Hyperopia**, **Intermittent** (becomes constant), **Reduced with glasses** | **Full cycloplegic correction** (glasses); **Bifocals** if high AC/A; **Surgery** if residual | https://www.aao.org/eye-health/diseases/accommodative-esotropia |
| **Acquired Non-Accommodative Esotropia** | >6 months | **No significant hyperopia**, **Constant** | **Surgery** | https://www.aao.org/eye-health/diseases/esotropia |
| **Sensory Esotropia** | Any age | **Poor vision** in one eye (cataract, Optic atrophy, Anisometropia) | **Treat underlying cause**, **Surgery** | https://www.aao.org/eye-health/diseases/sensory-esotropia |

## Exotropia
| Type | Features | Treatment | URL |
|------|----------|-----------|-----|
| **Intermittent Exotropia** | **Most common**; Deviates with fatigue, Daydreaming, Bright light (**closure of one eye**) | **Observation** (if well-controlled), **Patching**, **Over-minus glasses**, **Surgery** (if poorly controlled) | https://www.aao.org/eye-health/diseases/intermittent-exotropia |
| **Constant Exotropia** | **Always deviated** | **Surgery** | https://www.aao.org/eye-health/diseases/exotropia |
| **Sensory Exotropia** | **Poor vision** in one eye | **Treat underlying cause**, **Surgery** | https://www.aao.org/eye-health/diseases/sensory-exotropia |

## Strabismus - Evaluation
| Test | Purpose | URL |
|------|---------|-----|
| **Cover-Uncover Test** | Detects **tropia** (manifest deviation) | https://www.aao.org/eye-health/diseases/strabismus-diagnosis |
| **Alternate Cover Test** | Detects **phoria** (latent deviation) + **Tropia** | https://www.aao.org/eye-health/diseases/strabismus-diagnosis |
| **Hirschberg Test** | Estimates deviation using corneal light reflex | https://www.aao.org/eye-health/diseases/strabismus-diagnosis |
| **Prism Cover Test** | **Quantifies** deviation in prism diopters | https://www.aao.org/eye-health/diseases/strabismus-diagnosis |
| **Stereopsis** | Tests **binocular vision** (Titmus, Randot) | https://www.aao.org/eye-health/diseases/strabismus-diagnosis |
