# Flowchart Best Practices

## Symbols
- Oval: Start/End
- Rectangle: Process
- Diamond: Decision
- Parallelogram: Input/Output
- Circle: Connector

## Layout Guidelines
- Top to bottom flow preferred
- Maximum 20 nodes per diagram
- Use consistent spacing
- Label all decision branches

## Color Coding
- Green: Start/End
- Blue: Processes
- Orange: Decisions
- Gray: Comments
