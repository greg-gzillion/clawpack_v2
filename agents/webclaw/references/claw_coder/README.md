# ClawCoder - Programming Language Reference Library

This directory contains reference materials for 38 programming languages.

## Available Languages

- Python, JavaScript, TypeScript, Java, Kotlin, Scala
- Go, Rust, C, C++, C#, Swift, Objective-C
- PHP, Ruby, Perl, Lua, R, Julia, MATLAB
- HTML/CSS, Sass/Less, Bash, PowerShell, SQL
- MongoDB, PostgreSQL, Redis, Elasticsearch
- Docker, Kubernetes, Terraform, Ansible
- GraphQL, WebAssembly, Zig

## Usage

Each language folder contains a markdown file with URLs to:
- Official documentation
- Learning resources
- Frameworks and libraries
- Tools and utilities
- Testing frameworks

These references are used by the claw_coder agent to provide accurate, up-to-date programming assistance.
