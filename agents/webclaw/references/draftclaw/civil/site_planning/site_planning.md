# Site Planning Standards - Reference
## All URLs verified May 2026

## Site Plan Elements
| Element | Standard | Source |
|---------|----------|--------|
| Property lines | Metes and bounds or lot/block description from recorded plat; show bearings and distances | https://en.wikipedia.org/wiki/Site_plan |
| Building footprint | Overall dimensions; location relative to property lines; show setbacks per zoning | https://en.wikipedia.org/wiki/Site_plan |
| Topography | Existing (dashed) and proposed (solid) contour lines at 1-foot or 2-foot intervals | https://en.wikipedia.org/wiki/Topography |
| Contour lines | Lines of equal elevation; index contours every 5th line, heavier weight; label with elevation | https://en.wikipedia.org/wiki/Contour_line |
| Spot elevations | X symbol with elevation to 0.01 foot at: building corners, drainage structures, pavement transitions | https://en.wikipedia.org/wiki/Topography |
| Easements | Cross-hatched or shaded area; width dimensioned; type noted (utility, drainage, access, conservation) | https://en.wikipedia.org/wiki/Easement |

## Parking Layout
| Element | Standard | Source |
|---------|----------|--------|
| Standard stall (90-degree) | 9'-0" x 18'-0" minimum; 24'-0" two-way drive aisle | https://en.wikipedia.org/wiki/Parking_space |
| Compact stall | 8'-6" x 16'-0" (where permitted by zoning); limited to percentage of total | https://en.wikipedia.org/wiki/Parking_space |
| Accessible (standard) | 8'-0" stall + 5'-0" access aisle = 13'-0" total width; 1 per 6 accessible spaces, minimum 1 | https://en.wikipedia.org/wiki/Parking_space |
| Accessible (van) | 8'-0" stall + 8'-0" access aisle = 16'-0" total width; 1 per 6 accessible spaces, minimum 1 | https://en.wikipedia.org/wiki/Parking_space |
| One-way drive aisle | 22'-0" minimum for 60-degree parking; 18'-0" for 45-degree | https://en.wikipedia.org/wiki/Parking_space |
| Two-way drive aisle | 24'-0" minimum | https://en.wikipedia.org/wiki/Parking_space |
| Compacted gravel parking | 6" aggregate base on geotextile fabric; 1% minimum slope for drainage | https://en.wikipedia.org/wiki/Parking_space |

## Right-of-Way and Access
| Element | Standard | Source |
|---------|----------|--------|
| Right-of-way (ROW) | Strip of land reserved for public use; typically 50'-100' wide for local roads, wider for arterials | https://en.wikipedia.org/wiki/Right-of-way_(transportation) |
| Access point spacing | Minimum distance between driveways per local DOT; typically 50'-100' on arterials | https://en.wikipedia.org/wiki/Right-of-way_(transportation) |
| Sight distance | Clear sight triangle at driveway/road intersection; per AASHTO Green Book based on road speed | https://en.wikipedia.org/wiki/Right-of-way_(transportation) |
| Fire access road | 20' clear width, 13'-6" vertical clearance; turnaround required for dead-ends over 150' | https://en.wikipedia.org/wiki/Right-of-way_(transportation) |
