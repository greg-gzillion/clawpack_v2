# Civil Engineering Drawing Standards

## Site Plan Standards
| Element | Standard | URL |
|---------|----------|-----|
| Property lines | Heavy dashed line with bearing and distance | https://www.nationalcadstandard.org/ |
| Building footprint | Solid line with dimensions | https://www.nationalcadstandard.org/ |
| Setbacks | Dimensioned from property line to building | https://codes.iccsafe.org/content/IBC2021 |
| Easements | Shaded or hatched area with width noted | https://en.wikipedia.org/wiki/Easement |
| Topography | Contour lines at 1' or 2' intervals | https://en.wikipedia.org/wiki/Topography |
| Spot elevations | X symbol with elevation noted | https://en.wikipedia.org/wiki/Topography |
| Utilities | Dashed lines with utility type noted | https://www.nationalcadstandard.org/ |
| Storm drainage | Flow arrows, inlet locations, pipe sizes | https://www.nationalcadstandard.org/ |
| Parking stalls | 9' x 18' standard, 8'-6" x 18' compact | https://www.nationalcadstandard.org/ |
| Drive aisles | 24' two-way, 22' one-way | https://www.nationalcadstandard.org/ |
| Fire lane | 20' minimum clear width | https://codes.iccsafe.org/content/IFC2021 |
| Landscaping | Tree symbols, planting areas labeled | https://www.nationalcadstandard.org/ |
| North arrow | Standard symbol in upper right | https://www.nationalcadstandard.org/ |
| Scale bar | Graphic scale in lower right | https://www.nationalcadstandard.org/ |
