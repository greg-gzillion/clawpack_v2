# Accessibility Standards (ADA / ICC A117.1) - Reference

## Clear Floor Space
| Space | Minimum | URL |
|-------|---------|-----|
| Wheelchair turning circle | 60" diameter | https://www.ada.gov/ |
| T-shaped turning space | 60" x 60" with 36" arms | https://www.ada.gov/ |
| Forward reach | 15" to 48" above floor | https://www.ada.gov/ |
| Side reach | 9" to 54" above floor | https://www.ada.gov/ |

## Doors and Corridors
| Element | Minimum | URL |
|---------|---------|-----|
| Door clear width | 32" minimum | https://www.ada.gov/ |
| Double door (active leaf) | 32" minimum | https://www.ada.gov/ |
| Corridor width | 36" minimum | https://www.ada.gov/ |
| Passing space | 60" x 60" every 200' | https://www.ada.gov/ |
| Door maneuvering clearance (front) | 48" x 60" | https://www.ada.gov/ |
| Door maneuvering clearance (latch side) | 42" x 60" | https://www.ada.gov/ |

## Restrooms
| Element | Minimum | URL |
|---------|---------|-----|
| Accessible stall width | 60" minimum | https://www.ada.gov/ |
| Accessible stall depth | 56" (wall-mounted WC) or 59" (floor-mounted) | https://www.ada.gov/ |
| Toilet centerline from side wall | 16" to 18" | https://www.ada.gov/ |
| Grab bar (side wall) | 42" long, 33"-36" above floor | https://www.ada.gov/ |
| Grab bar (rear wall) | 36" long | https://www.ada.gov/ |
| Sink height | 34" max rim height | https://www.ada.gov/ |
| Sink knee clearance | 27" high x 30" wide x 19" deep | https://www.ada.gov/ |

|-------------|-----------|-----|

| Landing length | 60" minimum | https://www.ada.gov/ |
