# Multi-Family Residential Standards - Reference

## Building Classification (IBC)
| Type | IBC Occupancy | Stories Above Grade | URL |
|------|--------------|-------------------|-----|
| Garden apartments | R-2, Type V-A | 2-3 stories maximum | https://codes.iccsafe.org/content/IBC2021P1 |
| Mid-rise apartments | R-2, Type III-A | 3-5 stories maximum | https://codes.iccsafe.org/content/IBC2021P1 |
| High-rise apartments | R-2, Type I-A or I-B | 6+ stories | https://codes.iccsafe.org/content/IBC2021P1 |
| Townhouses (row houses) | IRC (not IBC) | Up to 3 stories | https://codes.iccsafe.org/content/IRC2021P1 |
| Condominiums | R-2 | Same as apartments | https://codes.iccsafe.org/content/IBC2021P1 |

## Fire Protection (IBC / NFPA 13)
| Requirement | Standard | URL |
|-------------|----------|-----|
| Sprinkler system (NFPA 13 or 13R) | Required for all R occupancies >3 stories | https://www.nfpa.org/codes-and-standards/ |
| Fire alarm | Audible (15 dBA above ambient) + visual notification | https://www.nfpa.org/codes-and-standards/ |
| Fire separation (unit to unit) | 1-hour fire-resistance rating minimum | https://codes.iccsafe.org/content/IBC2021P1 |
| Fire separation (unit to corridor) | 1-hour minimum (no rating if sprinklered) | https://codes.iccsafe.org/content/IBC2021P1 |
| Corridor fire rating (non-sprinklered) | 1-hour | https://codes.iccsafe.org/content/IBC2021P1 |

## Accessibility (ADA / FHA)
| Requirement | Standard | URL |
|-------------|----------|-----|
| Accessible units (ADA) | 5% minimum of total units (ground floor for non-elevator) | https://www.ada.gov/law-and-regs/design-standards/2010-stds/ |
| Type A units (FHA) | All ground-floor units in non-elevator buildings (4+ units) | https://www.hud.gov/program_offices/fair_housing_equal_opp/disabilities/fhefhasp |
| Type B units (FHA) | All units in elevator buildings (4+ units) | https://www.hud.gov/program_offices/fair_housing_equal_opp/disabilities/fhefhasp |
| Accessible route | From public way to accessible entrance | https://www.ada.gov/law-and-regs/design-standards/2010-stds/ |
| Accessible parking | Per ADA Table 208.2 | https://www.ada.gov/law-and-regs/design-standards/2010-stds/ |
