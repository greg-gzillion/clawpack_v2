# Educational Facility Standards - Reference

## Classroom Standards (IES/ANSI)
| Space | Area per Student | URL |
|-------|-----------------|-----|
| Elementary classroom | 35-45 sq ft per student | https://www.ies.org/standards/ |
| Middle/High school classroom | 30-40 sq ft per student | https://www.ies.org/standards/ |
| Science lab | 50-60 sq ft per student | https://www.ies.org/standards/ |
| Computer lab | 40-50 sq ft per student | https://www.ies.org/standards/ |
| Lecture hall (tiered) | 15-20 sq ft per seat | https://www.ies.org/standards/ |
| Library/Media center | 25-35 sq ft per student | https://www.ies.org/standards/ |
| Cafeteria | 12-15 sq ft per seat | https://www.ies.org/standards/ |
| Gymnasium (elementary) | 50' x 74' minimum (3,700 sq ft) | https://www.ies.org/standards/ |
| Gymnasium (high school) | 60' x 90' minimum (5,400 sq ft) + seating | https://www.ies.org/standards/ |

## Egress (Educational, E Occupancy)
| Requirement | Standard | URL |
|-------------|----------|-----|
| Occupant load | 20 sq ft net per occupant (classrooms) | https://codes.iccsafe.org/content/IBC2021P1 |
| Corridor width | 72" minimum | https://codes.iccsafe.org/content/IBC2021P1 |
| Door swing | Must swing in direction of egress for 50+ | https://codes.iccsafe.org/content/IBC2021P1 |
| Exit signs | 8" letter height (schools require larger) | https://codes.iccsafe.org/content/IBC2021P1 |
| Fire drills | Monthly (elementary), quarterly (secondary) | https://www.nfpa.org/codes-and-standards/ |
