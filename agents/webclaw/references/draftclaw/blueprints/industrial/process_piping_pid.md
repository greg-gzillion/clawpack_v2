# Process Piping & Instrumentation Drawing Standards (P&ID)
## How to draw process flow diagrams per ANSI/ISA-5.1

## P&ID Symbol Standards (ANSI/ISA-5.1-2009)
| Symbol | Represents | Drawing Convention | Source |
|--------|-----------|-------------------|--------|
| Solid line | Process piping (primary flow) | Heavy line, label with pipe size, material, and spec (e.g., "4" CS Sch 40 - P001") | https://www.isa.org/standards-and-publications/isa-standards/isa-5-1 |
| Dashed line | Instrument signal (electrical, pneumatic, or data) | Label signal type (4-20mA, 3-15 psi, Ethernet) | https://www.isa.org/standards-and-publications/isa-standards/isa-5-1 |
| Circle with line through center | Valve (gate, globe, ball, butterfly) | Label valve type and size; show actuator (manual handwheel, pneumatic diaphragm, motor-operated) | https://www.isa.org/standards-and-publications/isa-standards/isa-5-1 |
| Circle with horizontal line (NO) | Normally open valve | Filled or open circle per ISA convention | https://www.isa.org/standards-and-publications/isa-standards/isa-5-1 |
| Circle with diagonal line (NC) | Normally closed valve | Filled circle with diagonal line | https://www.isa.org/standards-and-publications/isa-standards/isa-5-1 |
| Diamond with label | Instrument (field-mounted) | Label with ISA tag number (e.g., "PT-101" for pressure transmitter 101) | https://www.isa.org/standards-and-publications/isa-standards/isa-5-1 |
| Square with circle inside | Instrument (control room/DCS) | Label with tag; distinguish from field-mounted per ISA 5.1 | https://www.isa.org/standards-and-publications/isa-standards/isa-5-1 |
| Rectangle with rounded ends | Vessel / Tank | Show nozzles, manways, level connections | https://www.isa.org/standards-and-publications/isa-standards/isa-5-1 |
| Circle with impeller symbol | Pump | Show suction and discharge; label flow direction arrows | https://www.isa.org/standards-and-publications/isa-standards/isa-5-1 |
| Trapezoid | Centrifugal compressor | Show suction/discharge; label stages | https://www.isa.org/standards-and-publications/isa-standards/isa-5-1 |
| Rectangle with tube bank | Heat exchanger (shell and tube) | Show tube side, shell side, pass arrangement | https://www.isa.org/standards-and-publications/isa-standards/isa-5-1 |

## P&ID Line Identification (per ANSI/ISA-5.1)
| Line Type | Content Label | Format | Source |
|-----------|-------------|--------|--------|
| Process line tag | Pipe size, material, service, insulation | "4"-CS-P001-2H" (4 inch, Carbon Steel, Process line 001, 2 inch Hot insulation) | https://www.isa.org/standards-and-publications/isa-standards/isa-5-1 |
| Utility line tag | e.g., "2"-CS-CW-003" (Cooling Water line 003) | Per plant line numbering standard | https://www.isa.org/standards-and-publications/isa-standards/isa-5-1 |
| Instrument signal | Per ISA 5.1 Table 5.1 | "4-20mA" or "3-15 psi" or "Foundation Fieldbus" | https://www.isa.org/standards-and-publications/isa-standards/isa-5-1 |

## Industrial Ventilation Duct Drawing
| Duct Type | Symbol Convention | Source |
|-----------|------------------|--------|
| Supply duct | Solid parallel lines with flow arrows | https://www.ashrae.org/ |
| Return/exhaust duct | Dashed parallel lines with flow arrows (direction toward exhaust) | https://www.ashrae.org/ |
| Duct size label | Width x Height (or diameter for round) above duct line | https://www.ashrae.org/ |
| Volume damper | Triangle inside duct symbol with "VD" label | https://www.ashrae.org/ |
| Fire damper | Triangle with "FD" and rating (e.g., "1.5 HR") at fire barrier crossing | https://www.nfpa.org/codes-and-standards/ |
| Duct insulation | Note above or below duct: "INS 2" R-8" (2 inch, R-8) | https://www.ashrae.org/ |
