# Site Plan Drawing Standards
## How to draw commercial site plans

## Site Plan Sheet Contents (NCS C-001 through C-199)
| Element | Drawing Convention | Scale | Source |
|---------|-------------------|-------|--------|
| Property lines | Heavy dashed line with bearing and distance in degrees-minutes-seconds | 1" = 20'-0" typical | https://www.nationalcadstandard.org/ |
| Building footprint | Solid heavy line with overall dimensions | 1" = 20'-0" | https://www.nationalcadstandard.org/ |
| Setbacks | Dimensioned from property line to building face | Front, side, rear each called out | https://codes.iccsafe.org/content/IBC2021P1 |
| Easements | Cross-hatched area with width dimension and type noted (utility, drainage, access) | Per survey | https://www.nationalcadstandard.org/ |
| Parking layout | Stall dimensions (9' x 18'), access aisle (5' x 18'), drive aisle width (24' two-way) | 1" = 20'-0" | https://codes.iccsafe.org/content/IBC2021P1 |
| Accessible parking | Symbol with ISA (International Symbol of Accessibility); 1 per 25 spaces minimum (van-accessible per Table 208.2) | Note accessible route to entrance | https://www.ada.gov/law-and-regs/design-standards/2010-stds/ |
| Fire lane | Note "FIRE LANE - NO PARKING"; 20' minimum clear width, 13'-6" minimum height clearance | Show curb paint or signage | https://codes.iccsafe.org/content/IFC2021P1 |
| Loading zone | Heavy dashed outline with "LOADING" note; 10' x 50' minimum for delivery trucks | Show dock approach | https://codes.iccsafe.org/content/IBC2021P1 |

## Grading and Drainage Symbols
| Symbol | Represents | Source |
|--------|-----------|--------|
| Arrow with slope percentage | Drainage flow direction ("2% MIN") | https://www.nationalcadstandard.org/ |
| Contour line with elevation | Existing grade (dashed) or proposed grade (solid); label elevation | https://www.nationalcadstandard.org/ |
| Spot elevation (X with number) | Finished floor, top of curb, inlet rim, etc. | https://www.nationalcadstandard.org/ |
| Catch basin / area drain | Circle with grid lines; label rim elevation and pipe size | https://www.nationalcadstandard.org/ |
| Swale centerline | Heavy dash-dot line with flow arrows | https://www.nationalcadstandard.org/ |
| Retention/detention basin | Outline with hatch pattern; show overflow elevation and outlet structure | https://www.nationalcadstandard.org/ |

## Utility Symbols (per NCS Utility Module)
| Utility | Line Type | Source |
|---------|----------|--------|
| Sanitary sewer | Heavy dashed "S" or "SAN" label, size noted (e.g. "6" PVC SDR35 @ 1.0%") | https://www.nationalcadstandard.org/ |
| Storm sewer | Heavy dashed "STM" label, show pipe size and slope | https://www.nationalcadstandard.org/ |
| Water line | Heavy centerline with "W" or 3 parallel lines, size noted (e.g. "8" DIP CL52") | https://www.nationalcadstandard.org/ |
| Gas line | Heavy centerline with "G" noted; depth 18"-24" minimum cover | https://www.nationalcadstandard.org/ |
| Electric primary | Heavy line with "E" or lightning bolt symbol | https://www.nationalcadstandard.org/ |
| Communications (fiber/copper) | Line with "C" or "T" noted | https://www.nationalcadstandard.org/ |
| Fire hydrant | Circle with "FH"; maximum 400' spacing in commercial per IFC | https://codes.iccsafe.org/content/IFC2021P1 |

## Landscape Plan Symbols (per NCS Landscape Module)
| Element | Symbol Convention | Source |
|---------|------------------|--------|
| Deciduous tree | Open circle with radiating lines; label species and caliper size in schedule | https://www.asla.org/ |
| Evergreen tree | Filled circle with radiating lines; label species and height | https://www.asla.org/ |
| Shrub mass | Scalloped cloud outline; label species, spacing, and container size | https://www.asla.org/ |
| Ground cover | Dot pattern or stipple pattern with species noted | https://www.asla.org/ |
| Turf/lawn | "LAWN" or "TURF" text note with area designation | https://www.asla.org/ |
| Irrigation line | Dashed line with "I" noted; show controller location | https://www.asla.org/ |
