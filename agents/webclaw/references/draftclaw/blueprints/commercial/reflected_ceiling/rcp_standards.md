# Reflected Ceiling Plan (RCP) Standards
## How to draw ceiling plans for commercial spaces

## RCP Basics
| Element | Drawing Convention | Source |
|---------|-------------------|--------|
| Ceiling grid (2' x 2' or 2' x 4') | Thin dashed lines showing grid pattern; note ACT type and manufacturer in legend | https://www.nationalcadstandard.org/ |
| Ceiling height | Note above each room: "CLG. HT. = 9'-0" A.F.F." (Above Finished Floor) | https://www.nationalcadstandard.org/ |
| Ceiling material transitions | Heavy dashed line where material changes; label each area | https://www.nationalcadstandard.org/ |
| Soffit/bulkhead | Wide dashed outline with cross-hatching; note bottom elevation | https://www.nationalcadstandard.org/ |
| Cloud ceiling (suspended shapes) | Polyline outline with cross-hatching; dimension to walls | https://www.nationalcadstandard.org/ |

## Lighting Layout (per IES)
| Fixture Type | Symbol | Spacing Rule | Source |
|-------------|--------|-------------|--------|
| 2x4 recessed troffer (standard office) | Rectangle with cross | Space = 0.75 to 1.5 x mounting height above work plane for uniform illumination | https://www.ies.org/standards/ |
| 2x2 recessed (grid ceilings) | Square with cross | Typically 8' x 8' to 10' x 10' spacing for 30-40 fc at desk | https://www.ies.org/standards/ |
| Linear pendant (open office) | Long narrow rectangle | Mount 18-24" below ceiling; space per manufacturer photometrics | https://www.ies.org/standards/ |
| Wall washer (perimeter) | Rectangle with arrows toward wall | 2'-3' from wall, 4'-6' spacing typical | https://www.ies.org/standards/ |
| Downlight (accent/highlight) | Circle with dot center | Spacing per beam spread; 4'-8' typical for 6" aperture | https://www.ies.org/standards/ |
| Exit sign | Rectangle with arrows and "EXIT" | Per egress path requirements; 100' maximum visibility distance | https://www.nfpa.org/codes-and-standards/nfpa-101-standard-development/101 |

## HVAC Diffuser Layout
| Diffuser Type | Symbol | Location Rule | Source |
|-------------|--------|-------------|--------|
| Supply diffuser (ceiling) | Rectangle with X | Locate 18-24" from perimeter walls; space 6'-12' apart for 4-way throw | https://www.ashrae.org/ |
| Return grille (ceiling) | Rectangle with diagonal | Locate away from supply for air mixing; minimum 3' from supply | https://www.ashrae.org/ |
| Linear slot diffuser | Long narrow rectangle | Typically at perimeter; 1/2" to 1" slot width | https://www.ashrae.org/ |
| Exhaust grille (restroom) | Rectangle with EX | Locate in ceiling near odor source; minimum 10' from outside air intake | https://www.ashrae.org/ |

## RCP Coordination
| Element | Coordination Note | Source |
|---------|------------------|--------|
| Fire sprinkler heads | Show all heads; coordinate with grid layout (centered in tile); minimum 18" clearance below deflector | https://www.nfpa.org/codes-and-standards/nfpa-13-standard-development/13 |
| Smoke detectors | Show location per fire alarm drawings; typically at corridor intersections and within 15' of any point in corridor | https://www.nfpa.org/codes-and-standards/nfpa-72-standard-development/72 |
| Speakers/strobes | Show location per fire alarm; strobe coverage per NFPA 72 (typically 15' from end of corridor, 100' spacing) | https://www.nfpa.org/codes-and-standards/nfpa-72-standard-development/72 |
| Access panels | Note above ceiling access points; required at dampers, VAV boxes, and other serviceable equipment | https://codes.iccsafe.org/content/IMC2021P1 |
