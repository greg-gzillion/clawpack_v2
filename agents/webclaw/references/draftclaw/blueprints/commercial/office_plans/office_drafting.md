# Office Space Planning & Blueprint Standards
## How to lay out and draw commercial office spaces

## Space Programming Ratios
| Space Type | Percentage of Total | Drawing Notes | Source |
|-----------|-------------------|---------------|--------|
| Private offices | 15-25% | Draw with solid walls, door swing arc (1/4 circle), window placement per elevation | https://www.boma.org/BOMA/Research-Resources/Office-Standards.aspx |
| Open workstations | 30-50% | Draw as open area with furniture layout; show panel heights in elevation key; dimension workstation clusters | https://www.boma.org/BOMA/Research-Resources/Office-Standards.aspx |
| Conference/meeting rooms | 10-15% | Show table with seating count; AV equipment symbols; glass wall designation (dashed center line) | https://www.gsa.gov/real-estate/design-and-construction/facilities-standards |
| Circulation (corridors) | 20-25% | Hatch pattern per material; dimension width; show fire rating of walls where required | https://www.boma.org/BOMA/Research-Resources/Office-Standards.aspx |
| Support (copy, break, storage) | 10-15% | Label each room; show equipment layout in copy/break rooms | https://www.gsa.gov/real-estate/design-and-construction/facilities-standards |
| Restrooms | 3-5% | Per IBC fixture counts; show accessible stall dimensions per ADA; floor drain locations | https://codes.iccsafe.org/content/IBC2021P1 |

## Drawing Sheet Organization (per NCS Uniform Drawing System)
| Sheet Prefix | Content | Scale |
|-------------|---------|-------|
| A-001 | Cover sheet, code analysis, accessibility compliance | NTS |
| A-101 through A-199 | Floor plans (one sheet per floor) | 1/8" = 1'-0" typical |
| A-201 through A-299 | Reflected ceiling plans | 1/8" = 1'-0" |
| A-301 through A-399 | Building elevations | 1/8" = 1'-0" typical |
| A-401 through A-499 | Building sections | 1/4" or 1/2" = 1'-0" |
| A-501 through A-599 | Enlarged plans (restrooms, stairs, lobbies) | 1/4" or 1/2" = 1'-0" |
| A-601 through A-699 | Schedules (door, window, finish, hardware) | NTS |
| A-701 through A-799 | Interior elevations | 1/4" = 1'-0" typical |
| A-801 through A-899 | Details (wall sections, millwork, waterproofing) | 1-1/2" or 3" = 1'-0" |

## Symbol Legend (Standard Office Plan Symbols)
| Symbol | Represents | Drawing Standard | Source |
|--------|-----------|-----------------|--------|
| Dashed line with dot-dash pattern | Demising wall / fire-rated wall | Label with hourly rating | NCS 3.0, Module 4 |
| Circle with cross | Floor drain | Label size and slope direction | NCS 3.0, Module 5 |
| Rectangle with X inside | Supply air diffuser | Label CFM and size | NCS 3.0, Module 5 |
| Circle with diagonal line | Return air grille | Label grille size | NCS 3.0, Module 5 |
| Small rectangle on wall | Fire extinguisher | Label type and rating | NCS 3.0, Module 5 |
| Arrow with radius dimension | Door swing (90 degrees) | Show door panel and frame | https://www.nationalcadstandard.org/ |
| Hexagon | Electrical/data floor outlet | Label type (power, data, AV) | https://www.nationalcadstandard.org/ |
| Thin line with triangles at ends | Section cut reference | Label section number and sheet reference | https://www.nationalcadstandard.org/ |

## Dimensioning Standards for Office Plans
| Element | Dimension Convention | Source |
|---------|---------------------|--------|
| Overall building dimensions | Three dimension strings: overall, interior grid lines, partition centerlines | https://www.nationalcadstandard.org/ |
| Interior walls | Dimension to centerline or face (note which) | https://www.nationalcadstandard.org/ |
| Door openings | Width called out (e.g. "3'-0" x 7'-0" door"); frame type in schedule | https://www.nationalcadstandard.org/ |
| Window openings | Width and sill height called out; type referenced to window schedule | https://www.nationalcadstandard.org/ |
| Corridor width | Clear dimension between wall faces (not centerlines) | https://www.nationalcadstandard.org/ |
| Accessible clearances | Dimension and note per ADA (60" turning circle, clear floor spaces) | https://www.ada.gov/law-and-regs/design-standards/2010-stds/ |
