## Floor Plan Conventions
| Convention | Standard | URL |
|-----------|----------|-----|
| Exterior walls | 6" thick (2x6 + sheathing + finish) | https://www.nationalcadstandard.org/ |
| Interior walls | 4-1/2" thick (2x4 + 1/2" drywall each side) | https://www.nationalcadstandard.org/ |
| Door swing | Quarter-circle arc showing opening direction | https://en.wikipedia.org/wiki/Floor_plan |
| Door width | 2'-6" to 3'-0" standard residential | https://en.wikipedia.org/wiki/Floor_plan |
| Window symbol | Three parallel lines in wall | https://en.wikipedia.org/wiki/Floor_plan |
| Stair symbol | Parallel lines with arrow indicating UP | https://en.wikipedia.org/wiki/Floor_plan |
| Dimension lines | Extension lines, arrows/tick marks, text above line | https://en.wikipedia.org/wiki/Floor_plan |
| Room labels | Room name + square footage, centered | https://www.nationalcadstandard.org/ |
| Plumbing fixtures | Standard symbols: toilet, sink, tub, shower | https://en.wikipedia.org/wiki/Floor_plan |
| Electrical symbols | Circle with lines for outlets, cross for lights | https://en.wikipedia.org/wiki/Floor_plan |
