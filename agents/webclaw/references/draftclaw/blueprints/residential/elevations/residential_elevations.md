# Residential Elevation Drafting Standards
## How to draw front, rear, and side elevations of houses

## Elevation Types
| Elevation | What It Shows | Source |
|-----------|--------------|--------|
| Front elevation | Primary street-facing view; entry door, garage doors, main architectural features | https://www.nationalcadstandard.org/ |
| Rear elevation | Back of house; deck/patio doors, windows to yard, roofline from rear | https://www.nationalcadstandard.org/ |
| Right elevation | Side view showing window placement, wall height transitions, chimney if present | https://www.nationalcadstandard.org/ |
| Left elevation | Opposite side view; same conventions as right | https://www.nationalcadstandard.org/ |

## Elevation Drawing Layers (per NCS)
| Layer | Content | Line Weight | Source |
|-------|---------|------------|--------|
| A-ELEV-OUTL | Building outline, roof profile | 0.5mm (heavy) | https://www.nationalcadstandard.org/ |
| A-ELEV-GLAZ | Windows, glazing, mullions | 0.25mm (thin) | https://www.nationalcadstandard.org/ |
| A-ELEV-DOOR | Doors, door frames | 0.35mm (medium) | https://www.nationalcadstandard.org/ |
| A-ELEV-ROOF | Roof shingles/tiles, fascia, gutters, downspouts | 0.25mm (thin) | https://www.nationalcadstandard.org/ |
| A-ELEV-FNSH | Material hatching, siding lines, brick coursing | 0.18mm (fine) | https://www.nationalcadstandard.org/ |
| A-ELEV-GRAD | Grade line, foundation exposure, window wells | 0.5mm (heavy - grade line is thickest on drawing) | https://www.nationalcadstandard.org/ |

## Grade and Height Reference Lines
| Line | Convention | Source |
|------|-----------|--------|
| Finished grade line | Heavy continuous line (0.7mm) with short vertical dashes extending downward. Label "FIN. GRADE" with elevation where applicable. | https://www.nationalcadstandard.org/ |
| Finished floor line (main level) | Medium line with label "FFL EL. 0'-0"" or actual survey elevation if known | https://www.nationalcadstandard.org/ |
| Second floor line | Medium dashed line extending across elevation at second floor height. Label "2ND FLR EL. +9'-0"" | https://www.nationalcadstandard.org/ |
| Roof peak | Light centerline down from peak with label "RIDGE EL. ___" | https://www.nationalcadstandard.org/ |
| Height dimensions | Vertical dimension lines from grade to: floor level, window heads/sills, eave line, roof peak | https://www.nationalcadstandard.org/ |

## Material Indication (hatching patterns)
| Material | Hatch Pattern | Source |
|----------|-------------|--------|
| Brick veneer | Horizontal lines every 2-5/8" (brick coursing), light | https://www.bia.org/ |
| Stone veneer | Irregular polygonal outline pattern, no strict spacing | https://www.naturalstoneinstitute.org/ |
| Horizontal lap siding | Horizontal lines at 4", 5", 6", or 7" spacing per siding exposure | https://www.vinylsiding.org/ |
| Vertical board and batten | Vertical lines with alternating thick/thin at 12" spacing typical | https://www.awc.org/ |
| Stucco | Stipple or random dot pattern (drafted) or "STUCCO" text note | https://www.stuccomfgassoc.com/ |
| Concrete foundation (exposed) | Light diagonal hatching (45 degrees, wide spacing) | https://www.concrete.org/ |
| Roof shingle (asphalt) | Horizontal lines at 5" exposure with staggered vertical joints | https://www.asphaltroofing.org/ |
| Wood shingle/shake | Alternating horizontal lines suggesting shingle edges, no coursing pattern | https://www.cedarbureau.org/ |

## Window Representation in Elevation
| Element | How to Draw | Source |
|---------|------------|--------|
| Window frame | Outer rectangle (frame), inner rectangle (sash), parallel lines for glass indication | https://www.nationalcadstandard.org/ |
| Window mullions | Thin vertical and horizontal lines dividing the window into lites | https://www.nationalcadstandard.org/ |
| Window sill | Short horizontal projection below window frame, 1-1/2" to 2" typical | https://www.nationalcadstandard.org/ |
| Window head trim | Horizontal line above window with trim thickness shown | https://www.nationalcadstandard.org/ |
| Shutters (decorative) | Rectangles flanking window, width = half window width typical | https://www.nationalcadstandard.org/ |
| Window well (basement) | Dashed outline with vertical dimension from grade to sill; note corrugated metal or masonry | https://codes.iccsafe.org/content/IRC2021P1 |

## Roof Representation
| Element | Convention | Source |
|---------|-----------|--------|
| Roof pitch | Triangle symbol with rise and run (e.g., "8:12") | https://www.nationalcadstandard.org/ |
| Eave overhang | Dimension from wall face to fascia (12"-24" typical) | https://www.nationalcadstandard.org/ |
| Fascia board | 1x6 or 1x8 at eave edge; horizontal line at roof edge | https://www.nationalcadstandard.org/ |
| Soffit | Note material and vent type (continuous or individual vents every 6') | https://codes.iccsafe.org/content/IRC2021P1 |
| Gutter and downspout | Half-round or K-style profile at eave line; downspout shown as vertical rectangle at corners | https://www.nationalcadstandard.org/ |
| Roof vent (ridge) | Continuous ridge vent shown at peak with note "RIDGE VENT" | https://codes.iccsafe.org/content/IRC2021P1 |
| Roof vent (box/turtle) | Shown as small rectangle near ridge with note | https://codes.iccsafe.org/content/IRC2021P1 |
| Chimney (masonry) | Rectangle extending above ridge by 2' minimum; show cap detail and cricket if >30" wide | https://codes.iccsafe.org/content/IRC2021P1 |
