# Industrial & Warehouse Blueprint Standards

## Warehouse Layout Standards
| Element | Standard | URL |
|---------|----------|-----|
| Bay width | 40'-60' typical column spacing | https://www.nationalcadstandard.org/ |
| Clear height | 24'-40' for high-bay storage | https://www.osha.gov/warehousing |
| Loading dock height | 48"-52" above grade | https://www.osha.gov/warehousing |
| Dock door width | 8'-10' standard | https://www.osha.gov/warehousing |
| Dock leveler | 6'-8' width, 25K-40K lb capacity | https://www.osha.gov/warehousing |
| Floor slab | 6"-8" reinforced concrete, 4000 PSI | https://www.concrete.org/ |
| Floor flatness | FF 35 / FL 25 for VNA trucks | https://www.concrete.org/ |
| Aisle width | 12'-15' for forklift traffic | https://www.osha.gov/warehousing |
| Rack spacing | 8'-10' between rack rows | https://www.osha.gov/warehousing |
| Sprinkler clearance | 18" minimum below sprinkler heads | https://www.nfpa.org/codes-and-standards/ |
| Exit doors | Every 100' of perimeter wall | https://codes.iccsafe.org/content/IFC2021 |
| Emergency lighting | 1 foot-candle minimum along egress path | https://www.nfpa.org/codes-and-standards/ |
| Office area | 5-10% of total warehouse square footage | https://www.nationalcadstandard.org/ |
