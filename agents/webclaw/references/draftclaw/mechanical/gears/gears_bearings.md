# Gears and Power Transmission - Reference
## All URLs verified May 2026

## Gear Types
| Type | Characteristics | Typical Use | Source |
|------|----------------|-------------|--------|
| Spur gear | Straight teeth parallel to axis; simplest; no axial thrust | Parallel shafts, moderate speeds; most common gear type | https://en.wikipedia.org/wiki/Spur_gear |
| Helical gear | Angled teeth (15-30 degrees typical); smoother, quieter than spur; produces axial thrust | High-speed parallel shafts; automotive transmissions | https://en.wikipedia.org/wiki/Helical_gear |
| Double helical (herringbone) | Two opposing helical sections cancel axial thrust | High-power, high-speed applications; turbines, compressors | https://en.wikipedia.org/wiki/Helical_gear |
| Bevel gear (straight) | Conical shape; transmits power between intersecting shafts (typically 90 degrees) | Differential drives, hand drills | https://en.wikipedia.org/wiki/Bevel_gear |
| Spiral bevel gear | Curved teeth on conical surface; smoother than straight bevel | Automotive differentials, aerospace | https://en.wikipedia.org/wiki/Bevel_gear |
| Worm and worm gear | High reduction ratio in single stage (5:1 to 100:1); self-locking possible | Conveyors, elevators, tuning mechanisms | https://en.wikipedia.org/wiki/Worm_drive |
| Rack and pinion | Converts rotary to linear motion | Steering systems, linear actuators, CNC machines | https://en.wikipedia.org/wiki/Gear |

## Gear Terminology
| Term | Definition | Source |
|------|-----------|--------|
| Pitch circle | Theoretical circle on which gear calculations are based; two mating gears' pitch circles are tangent | https://en.wikipedia.org/wiki/Gear |
| Module (metric) | Pitch diameter / number of teeth; standard sizes: 0.5, 1, 1.5, 2, 3, 4, 5, 6, 8, 10mm | https://en.wikipedia.org/wiki/Gear |
| Diametral pitch (inch) | Number of teeth / pitch diameter (inches); standard: 2, 4, 6, 8, 10, 12, 16, 20, 24, 32, 48 | https://en.wikipedia.org/wiki/Gear |
| Pressure angle | Angle between tooth profile and radial line at pitch circle; standard: 20 degrees (14.5 and 25 also used) | https://en.wikipedia.org/wiki/Gear |
| Backlash | Clearance between mating teeth; allows for lubrication, thermal expansion, manufacturing tolerances | https://en.wikipedia.org/wiki/Gear |
| Gear ratio | Teeth on driven / teeth on driver; determines speed reduction/increase | https://en.wikipedia.org/wiki/Gear |

## Bearings
| Type | Characteristics | Source |
|------|----------------|--------|
| Plain bearing (bushing) | Simple sleeve; no rolling elements; shaft slides on bearing surface; lowest cost | https://en.wikipedia.org/wiki/Plain_bearing |
| Ball bearing | Rolling elements (balls) between inner and outer races; handles radial and axial loads | https://en.wikipedia.org/wiki/Rolling-element_bearing |
| Roller bearing (cylindrical) | Line contact (vs. point contact for balls); higher radial load capacity than ball bearings | https://en.wikipedia.org/wiki/Rolling-element_bearing |
| Tapered roller bearing | Conical rollers; handles combined radial and axial loads (e.g., vehicle wheel bearings) | https://en.wikipedia.org/wiki/Rolling-element_bearing |
| Thrust bearing | Designed for axial loads only; ball or roller type | https://en.wikipedia.org/wiki/Thrust_bearing |
| Needle roller bearing | Thin, long rollers; high capacity in limited radial space | https://en.wikipedia.org/wiki/Rolling-element_bearing |

## Shaft Connections
| Type | Function | Source |
|------|----------|--------|
| Key and keyseat | Square or rectangular bar inserted between shaft and hub to transmit torque | https://en.wikipedia.org/wiki/Key_(engineering) |
| Woodruff key | Half-moon shaped; self-aligning in tapered shaft applications | https://en.wikipedia.org/wiki/Key_(engineering) |
| Spline | Series of ridges on shaft matching grooves in hub; transmits high torque with axial sliding possible | https://en.wikipedia.org/wiki/Spline_(mechanical) |
| Set screw | Threaded fastener through hub bears against shaft; light duty only | https://en.wikipedia.org/wiki/Set_screw |
| Taper lock bushing | Split tapered bushing clamped between hub and shaft; high torque, easy removal | Manufacturer standard |
