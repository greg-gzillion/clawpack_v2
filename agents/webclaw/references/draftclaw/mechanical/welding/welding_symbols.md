# Welding Symbols - Reference (AWS A2.4)

## Basic Weld Types
| Symbol | Name | Description | URL |
|--------|------|-------------|-----|
| ││ | Fillet Weld | Triangular cross-section joining two surfaces at right angle | https://www.aws.org/ |
| V   | V-Groove Weld | V-shaped groove between joint members | https://www.aws.org/ |
| ᴗ  | J-Groove Weld | J-shaped groove, one member radiused | https://www.aws.org/ |
| ⏝  | U-Groove Weld | U-shaped groove, both members radiused | https://www.aws.org/ |
| ∩ | Flare-V Groove | Rounded members form V-groove | https://www.aws.org/ |
| ⊂ | Flare-Bevel Groove | One rounded member, one flat | https://www.aws.org/ |
| ∕ | Bevel Groove | Angled edge on one member | https://www.aws.org/ |
| Ξ | Square Groove | Butt joint with slight separation | https://www.aws.org/ |
| • | Spot Weld | Circular resistance weld | https://www.aws.org/ |
| ══  | Seam Weld | Continuous resistance weld along joint | https://www.aws.org/ |
| ◯  | Plug/Slot Weld | Circular hole filled with weld metal | https://www.aws.org/ |
