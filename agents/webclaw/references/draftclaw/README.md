# DraftClaw Reference Library
## Technical Drawing Knowledge Base

### Directory Structure
- **standards/** - ISO, ANSI/ASME, CAD layer standards
- **techniques/** - Floor plans, elevations, sections, dimensioning
- **blueprints/** - Architectural, warehouse, office layouts
- **cad/** - AutoCAD techniques, commands, blocks
- **circuits/** - Electronic symbols, circuit designs, PCB standards
- **mechanical/** - GD&T, fasteners, orthographic projection
- **architectural/** - Building codes, room dimensions, egress
- **civil/** - Site plans, grading, drainage, utilities
- **electrical/** - Power distribution, lighting, NEC code

- AIA Contract Documents: https://www.aiacontracts.org/

- OSHA Warehouse Safety: https://www.osha.gov/warehousing
- IPC PCB Design: https://www.ipc.org/
