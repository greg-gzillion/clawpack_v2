# Section Drawing Standards - Reference

## Section Types
| Type | Description | URL |
|------|-------------|-----|
| Full section | Cut through entire building/object | https://en.wikipedia.org/wiki/Cross_section_(geometry) |
| Half section | Symmetrical object, half shown internally | https://en.wikipedia.org/wiki/Cross_section_(geometry) |
| Offset section | Cutting plane stepped to show different features | https://en.wikipedia.org/wiki/Cross_section_(geometry) |
| Broken-out section | Partial cutaway showing internal feature | https://en.wikipedia.org/wiki/Cross_section_(geometry) |
| Revolved section | Cross-section rotated 90 degrees in place | https://en.wikipedia.org/wiki/Cross_section_(geometry) |
| Removed section | Cross-section shown elsewhere on drawing | https://en.wikipedia.org/wiki/Cross_section_(geometry) |

## Hatching Patterns (ANSI/ASME)
| Material | Pattern | URL |
|----------|---------|-----|
| Cast iron | Diagonal lines both directions, small spacing | https://www.asme.org/ |
| Steel | Diagonal lines one direction, medium spacing | https://www.asme.org/ |
| Bronze/Copper | Diagonal lines one direction, alternating dash | https://www.asme.org/ |
| Concrete | Small triangles and dots | https://www.asme.org/ |
| Earth | Horizontal lines with vertical breaks | https://www.asme.org/ |
| Wood (cross-grain) | Wavy/irregular parallel lines | https://www.asme.org/ |
| Wood (with-grain) | Tight parallel lines with knot symbols | https://www.asme.org/ |
| Insulation | Zigzag or "batting" pattern | https://www.asme.org/ |
| Glass | Diagonal lines, 45 degrees, wide spacing | https://www.asme.org/ |
| Brick/Stone | Irregular outlines with mortar lines | https://www.asme.org/ |
| Plastic/Rubber | Crosshatch with small circles | https://www.asme.org/ |

## Building Section Conventions
| Element | Convention | URL |
|---------|-----------|-----|
| Foundation | Footing, foundation wall, slab on grade shown | https://en.wikipedia.org/wiki/Architectural_drawing |
| Floor assembly | Joists, subfloor, underlayment, finish floor labeled | https://en.wikipedia.org/wiki/Architectural_drawing |
| Wall assembly | Studs, sheathing, weather barrier, insulation, drywall | https://en.wikipedia.org/wiki/Architectural_drawing |
| Ceiling | Joists or trusses, drywall, insulation above | https://en.wikipedia.org/wiki/Architectural_drawing |
| Roof assembly | Rafters/trusses, sheathing, underlayment, roofing | https://en.wikipedia.org/wiki/Architectural_drawing |
| Floor-to-floor height | Dimensioned with arrows | https://en.wikipedia.org/wiki/Architectural_drawing |
| Material labels | Each material called out with leader lines | https://en.wikipedia.org/wiki/Architectural_drawing |
| Section cut reference | Label on plan with arrow showing view direction | https://en.wikipedia.org/wiki/Cross_section_(geometry) |
