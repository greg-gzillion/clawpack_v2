# Drafting Techniques - Reference
## All URLs verified May 2026

## Drawing Process (General)
| Step | Technique | Source |
|------|-----------|--------|
| 1. Block out | Light construction lines for overall proportions and major features | https://en.wikipedia.org/wiki/Engineering_drawing |
| 2. Primary views | Establish front, top, and right side views per third-angle projection | https://en.wikipedia.org/wiki/Orthographic_projection |
| 3. Section details | Cut through complex areas; standard hatch patterns per material type | https://en.wikipedia.org/wiki/Cross_section_(geometry) |
| 4. Dimension | Outside dimensions first, then interior, then details; avoid redundancy | https://en.wikipedia.org/wiki/Engineering_drawing |
| 5. Notes/labels | All materials, finishes, and special instructions called out with leaders | https://en.wikipedia.org/wiki/Engineering_drawing |
| 6. Title block | Complete all fields: project, sheet, scale, date, initials, revisions | https://en.wikipedia.org/wiki/Architectural_drawing |
| 7. Review | Self-check dimensions, clearances, code compliance before release | https://en.wikipedia.org/wiki/Engineering_drawing |

## Isometric Drawing
| Element | Technique | Source |
|---------|-----------|--------|
| Axis setup | Vertical axis (height), 30 degrees left and right from horizontal (width and depth) | https://en.wikipedia.org/wiki/Isometric_projection |
| Circles in isometric | Appear as ellipses; use isometric ellipse template or four-center method | https://en.wikipedia.org/wiki/Isometric_projection |
| Dimensioning in isometric | Aligned with isometric planes; not true length unless on principal axis | https://en.wikipedia.org/wiki/Isometric_projection |

## Exploded Views (Assembly Drawings)
| Element | Convention | Source |
|---------|-----------|--------|
| Explosion axis | Parts separated along assembly axis, typically centerline of fasteners | https://en.wikipedia.org/wiki/Exploded-view_drawing |
| Alignment | Parts remain aligned on their assembly path; phantom lines show installed position | https://en.wikipedia.org/wiki/Exploded-view_drawing |
| Balloon callouts | Each part numbered with circle and leader; matches parts list/bill of materials | https://en.wikipedia.org/wiki/Exploded-view_drawing |

## Checking a Drawing (Quality Control)
| Check | What to Verify | Source |
|-------|---------------|--------|
| All views present | Front, top, right side minimum for mechanical; plan + elevations for architectural | https://en.wikipedia.org/wiki/Engineering_drawing |
| Dimensions closed | No missing dimensions; no redundant dimensions; check string totals | https://en.wikipedia.org/wiki/Engineering_drawing |
| Clearances verified | Moving parts have clearance; doors don't hit fixtures; maintenance access provided | https://en.wikipedia.org/wiki/Engineering_drawing |
| Materials called out | Every part has material specification or finish note | https://en.wikipedia.org/wiki/Engineering_drawing |
| Standards compliance | Latest adopted codes referenced; ASME/ISO/ANSI standards noted | https://en.wikipedia.org/wiki/Engineering_drawing |
| Scale correct | Viewport scale matches title block; bar scale verifies | https://en.wikipedia.org/wiki/Engineering_drawing |
