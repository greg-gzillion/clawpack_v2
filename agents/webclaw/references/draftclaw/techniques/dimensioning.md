# Dimensioning Standards - Reference (ASME Y14.5)

## Dimensioning Rules
| Rule | Description | URL |
|------|-------------|-----|
| Avoid duplicate dimensions | Each feature dimensioned once only | https://www.asme.org/codes-standards/find-codes-standards/y14-5-dimensioning-tolerancing |
| Dimensions outside view | Place outside object outline when practical | https://www.asme.org/codes-standards/find-codes-standards/y14-5-dimensioning-tolerancing |
| Avoid hidden lines | Do not dimension to hidden features | https://www.asme.org/codes-standards/find-codes-standards/y14-5-dimensioning-tolerancing |
| Stagger dimensions | Offset adjacent dimension lines for readability | https://www.asme.org/codes-standards/find-codes-standards/y14-5-dimensioning-tolerancing |
| Extension lines | Start 1/16" from object, extend 1/8" past dimension line | https://www.asme.org/codes-standards/find-codes-standards/y14-5-dimensioning-tolerancing |
| Dimension text | Readable from bottom or right side of drawing | https://www.asme.org/codes-standards/find-codes-standards/y14-5-dimensioning-tolerancing |
| Overall dimensions | Place outermost, intermediate dimensions inside | https://www.asme.org/codes-standards/find-codes-standards/y14-5-dimensioning-tolerancing |
| Angle dimensions | Degrees and minutes or decimal degrees | https://www.asme.org/codes-standards/find-codes-standards/y14-5-dimensioning-tolerancing |
| Reference dimensions | Enclosed in parentheses for information only | https://www.asme.org/codes-standards/find-codes-standards/y14-5-dimensioning-tolerancing |
| Basic dimensions | Enclosed in rectangle, theoretically exact | https://www.asme.org/codes-standards/find-codes-standards/y14-5-dimensioning-tolerancing |
