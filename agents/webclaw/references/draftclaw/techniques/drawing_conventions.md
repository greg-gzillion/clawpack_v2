# Blueprint Conventions - Reference

## Drawing Sheet Layout
| Element | Standard | URL |
|---------|----------|-----|
| Title block location | Bottom right corner of sheet | https://en.wikipedia.org/wiki/Title_block |
| Revision block | Top right or adjacent to title block | https://en.wikipedia.org/wiki/Title_block |
| North arrow | Upper right of plan unless rotated | https://en.wikipedia.org/wiki/Floor_plan |
| Graphic scale | Below title block or near north arrow | https://en.wikipedia.org/wiki/Floor_plan |
| Drawing index | First sheet of drawing set | https://en.wikipedia.org/wiki/Title_block |
| Key plan | Small location map on each sheet | https://en.wikipedia.org/wiki/Floor_plan |

## Elevation Drawing Conventions
| Convention | Standard | URL |
|-----------|----------|-----|
| Grade line | Thickest line (0.7mm), ground level reference | https://en.wikipedia.org/wiki/Architectural_drawing |
| Finish floor level (FFL) | Thin line with elevation noted | https://en.wikipedia.org/wiki/Architectural_drawing |
| Roof pitch | Triangle symbol with rise:run noted | https://en.wikipedia.org/wiki/Architectural_drawing |
| Window representation | Frame, sash, glass, sill shown | https://en.wikipedia.org/wiki/Architectural_drawing |
| Material hatching | Per ANSI/ASME standards | https://www.asme.org/ |
| Height dimensions | Vertical dimension lines with elevation | https://en.wikipedia.org/wiki/Architectural_drawing |
| Door swing in elevation | Dash line showing door swing arc | https://en.wikipedia.org/wiki/Architectural_drawing |
