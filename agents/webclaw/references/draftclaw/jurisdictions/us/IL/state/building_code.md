## Adopted Codes
| Code | Edition |
|------|---------|
| Building | IBC 2018 + IL Amendments |
| Residential | IRC 2018 |
| Electrical | NEC 2008 (IL adopted) |
| Plumbing | IPC 2018 |
| Mechanical | IMC 2018 |
| Fire | IFC 2018 |
| Energy | IECC 2018 + IL Amendments |
| Accessibility | IL Accessibility Code (stricter than ADA) |
