# Technical Drawing Standards - ANSI/ASME
## ASME Y14.5-2018 Dimensioning and Tolerancing
- The authoritative standard for geometric dimensioning
- https://www.asme.org/codes-standards/find-codes-standards/y14-5-dimensioning-tolerancing

## ASME Y14.1 Drawing Sheet Size and Format
- A size: 8.5 x 11 (letter)
- B size: 11 x 17 (tabloid)
- C size: 17 x 22
- D size: 22 x 34
- E size: 34 x 44

## AIA National CAD Standard (NCS)
- Uniform Drawing System (UDS) for organizing sheets
- https://www.nationalcadstandard.org/

## ANSI/ASME Y14.3 Multiview and Sectional Views
- Front, top, right side as primary views
- Section views with standardized hatching patterns
- Auxiliary views for inclined surfaces
