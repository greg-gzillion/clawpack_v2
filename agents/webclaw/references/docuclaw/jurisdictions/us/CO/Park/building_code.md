## Colorado Building Codes
| Code | Edition |
|------|---------|
| Building | IBC 2021 (Colorado Building Code) |
| Residential | IRC 2021 |
| Electrical | NEC 2023 (Colorado Electrical Code) |
| Plumbing | IPC 2021 |
| Mechanical | IMC 2021 |
| Fire | IFC 2021 |
| Energy | IECC 2021 + CO requirements |
| Radon | CRS 25-23-108 (mandatory mitigation) |
