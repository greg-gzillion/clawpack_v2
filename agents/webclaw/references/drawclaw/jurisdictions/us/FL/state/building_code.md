## Key FL Requirements
- HVHZ: Miami-Dade and Broward counties (strictest building codes in US)
- Wind-borne debris region: Impact glazing or shutters required
- Flood zones: Widespread per FEMA FIRM maps
- Energy: Florida-specific climate zones
- Product approval: FL-approved products required statewide
- 25% roof replacement rule triggers full code compliance
