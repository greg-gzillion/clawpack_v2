# Massachusetts State Building Codes
## MA Board of Building Regulations and Standards (BBRS)
## Address: 1000 Washington St., Boston, MA 02118
## Phone: (617) 701-5600

## Adopted Codes (MA State Building Code - 780 CMR)
| Code | Edition |
|------|---------|
| Building | IBC 2015 + MA Amendments (9th ed) |
| Residential | IRC 2015 + MA Amendments |
| Electrical | NEC 2020 + MA Amendments |
| Plumbing | IPC 2015 + MA Amendments |
| Mechanical | IMC 2015 |
| Fire | IFC 2015 + MA Fire Code |
| Energy | IECC 2015 + MA Stretch Code |
| Accessibility | MA Architectural Access Board (521 CMR) |

## Design Criteria
| Item | Western MA | Central MA | Eastern/Coastal MA |
|------|-----------|-----------|-------------------|
| Frost Depth | 42-48 in | 42 in | 36-42 in |
| Snow Load | 40-60 psf | 30-50 psf | 25-40 psf |
| Wind Speed | 110-120 mph | 115-130 mph | 130-145 mph |
| Seismic | SDC A-B | SDC A-B | SDC A-B |
