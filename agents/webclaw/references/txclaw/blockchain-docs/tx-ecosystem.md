# TX Blockchain Ecosystem

## Overview
TX is a purpose-built Layer-1 blockchain for Real-World Asset (RWA) tokenization.

## Key Features
- Smart Tokens: Native token management with built-in compliance
- WASM Smart Contracts: Code in multiple languages
- IBC Interoperability: Connect with Cosmos ecosystem
- XRPL Bridge: Cross-chain asset transfer
- Built-in DEX: Native decentralized exchange
