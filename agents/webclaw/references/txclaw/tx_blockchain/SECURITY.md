# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.3.x   | ✅ (current) |
| 0.2.x   | ❌ (legacy) |
| < 0.2   | ❌ |

## Reporting a Vulnerability

**Email:** gjf20842@gmail.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (if applicable)

**Response Time:** Within 48 hours

## Security Best Practices for Users

- ✅ Always verify you're on the official site: `phoenix-frontend-seven.vercel.app`
- ✅ Never share your private keys or seed phrase
- ✅ Double-check wallet addresses before sending funds
- ✅ Use hardware wallets for significant amounts
- ✅ Report suspicious activity immediately

## ⚠️ Important Note

PhoenixPME is currently in testnet/mock mode until March 6, 2026. No real funds are at risk. Please report any issues to help us improve security before mainnet.
