# TX Blockchain - Introduction

## Overview
TX is a 3rd-generation layer-1 enterprise-grade blockchain built to serve as a core infrastructure for decentralized applications with ISO20022 compatibility, IBC interoperability, and novel Smart Tokens.

## Key Characteristics
- **Type**: Layer-1 Enterprise Blockchain
- **Generation**: 3rd Generation
- **Compatibility**: ISO20022
- **Interoperability**: IBC (Inter-Blockchain Communication)
- **Token Standard**: Smart Tokens

## Network Capabilities
TX network guarantees:
- **Elevated throughput** - High transaction processing capacity
- **Cost-effective fees** - Low transaction costs
- **Unparalleled scalability** - Grows with demand
- **WASM-based smart contracts** - Flexible, powerful contract execution
- **Low-latency PoS network** - Fast block finality

## Native Token ($TX)
$TX is the native token for TX, used for:
- Interacting with blockchain features
- Consensus protocol participation
- Decentralized exchange trading
- Network fee payments
- Staking and validation

## Use Cases
An established use case for a network like TX is the **Sologenic Ecosystem**, which currently resides on the XRP Ledger, demonstrating the platform's capability for real-world asset tokenization.

## Technical Advantages
- **Energy Efficient**: Bonded Proof of Stake (BPoS) consensus eliminates high energy costs
- **Scalable**: High throughput without exorbitant fees
- **Environmentally Friendly**: Low carbon footprint compared to PoW chains
