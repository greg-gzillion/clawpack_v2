# TXclaw

**AI agent for TX Blockchain - transactions, smart contracts, and chain exploration**

## Overview

TXclaw is a specialized AI agent built specifically for the TX Blockchain ecosystem. It provides intelligent blockchain interactions through a natural language interface.

## Features

| Feature | Description |
|---------|-------------|
| 💸 **Transactions** | Create, sign, and broadcast TX transactions |
| 📜 **Smart Contracts** | Deploy and interact with smart contracts |
| 👛 **Wallet Management** | Manage wallets, keys, and balances |
| 🔍 **Chain Exploration** | Explore blocks, transactions, and addresses |
| ⛽ **Gas Estimation** | Real-time gas price tracking |
| 🌐 **Network Switching** | Seamless mainnet/testnet switching |
| 📊 **Analytics** | Blockchain statistics and metrics |
